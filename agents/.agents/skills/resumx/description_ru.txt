Работайте с Resumx, средством рендеринга резюме из Markdown в PDF. Используйте при создании, редактировании, преобразовании, составлении, стилизации или адаптации резюме. Охватывает синтаксис, интерфейс командной строки, параметры стиля, значки, теги, представления, переменные, многоязычность, подгонку страниц, проверку, пользовательский CSS, преобразование резюме JSON и написание резюме с помощью искусственного интеллекта.
