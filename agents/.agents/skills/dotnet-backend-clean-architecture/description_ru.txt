Чистая архитектура + CQRS (MediatR) для серверных API .NET. Создает и поддерживает API с вертикальным срезом (функции), командами/запросами, обработчиками, валидаторами и репозиториями. Используйте при создании или рефакторинге API .NET, добавлении конечных точек, команд, запросов, сущностей или когда пользователь упоминает чистую архитектуру, CQRS или MediatR.
