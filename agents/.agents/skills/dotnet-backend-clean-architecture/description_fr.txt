Clean Architecture + CQRS (MediatR) pour les API backend .NET. Construit et maintient des API avec une tranche verticale (fonctionnalités), des commandes/requêtes, des gestionnaires, des validateurs et des référentiels. À utiliser lors de la création ou de la refactorisation d'API .NET, de l'ajout de points de terminaison, de commandes, de requêtes, d'entités ou lorsque l'utilisateur mentionne Clean Architecture, CQRS ou MediatR.
