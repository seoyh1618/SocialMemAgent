بنية نظيفة + CQRS (MediatR) لواجهات برمجة تطبيقات الواجهة الخلفية لـ .NET. يبني ويحافظ على واجهات برمجة التطبيقات (APIs) مع الشريحة الرأسية (الميزات)، والأوامر/الاستعلامات، والمعالجات، وأدوات التحقق من الصحة، والمستودعات. يُستخدم عند إنشاء واجهات برمجة تطبيقات .NET أو إعادة هيكلتها، أو إضافة نقاط النهاية، أو الأوامر، أو الاستعلامات، أو الكيانات، أو عندما يذكر المستخدم Clean Architecture، أو CQRS، أو MediatR.
