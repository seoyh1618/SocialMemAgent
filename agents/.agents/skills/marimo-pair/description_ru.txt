Протокол совместной работы для сопряжения с пользователем через работающий ноутбук marimo с помощью встроенных сценариев или MCP. Используйте, когда пользователь просит вас работать, создавать, исследовать или изменять блокнот marimo — или когда вы обнаруживаете запущенный сеанс marimo, перечисляя сеансы. НЕ используйте для общих сценариев Python за пределами marimo или для разработки плагинов/пакетов marimo.
