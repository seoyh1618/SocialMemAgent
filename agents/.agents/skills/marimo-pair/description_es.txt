Protocolo de colaboración para emparejarse con un usuario a través de una computadora portátil marimo en ejecución a través de scripts incluidos o MCP. Úselo cuando el usuario le solicite que trabaje, construya, explore o modifique un cuaderno de marimo, o cuando detecte una sesión de marimo en ejecución al enumerar las sesiones. NO lo use para secuencias de comandos Python generales fuera de marimo o para el desarrollo de complementos/paquetes de marimo.
