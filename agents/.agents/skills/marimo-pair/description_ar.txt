بروتوكول تعاون للاقتران مع مستخدم من خلال كمبيوتر محمول ماريمو قيد التشغيل عبر البرامج النصية المجمعة أو MCP. يُستخدم عندما يطلب منك المستخدم العمل في دفتر ملاحظات ماريمو أو إنشائه أو استكشافه أو تعديله - أو عندما تكتشف جلسة ماريمو قيد التشغيل عن طريق سرد الجلسات. لا تستخدمه في البرمجة النصية العامة لـ Python خارج Marimo أو لتطوير حزمة/حزمة Marimo الإضافية.
