Protocollo di collaborazione per l'associazione con un utente tramite un notebook Marimo in esecuzione tramite script in bundle o MCP. Da utilizzare quando l'utente ti chiede di lavorare, costruire, esplorare o modificare un taccuino Marimo o quando rilevi una sessione Marimo in esecuzione elencando le sessioni. NON utilizzare per script Python generali al di fuori di Marimo o per lo sviluppo di plugin/pacchetti Marimo.
