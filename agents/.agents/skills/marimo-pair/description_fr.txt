Protocole de collaboration pour le couplage avec un utilisateur via un ordinateur portable Marimo en cours d'exécution via des scripts fournis ou MCP. À utiliser lorsque l'utilisateur vous demande de travailler, de créer, d'explorer ou de modifier un carnet Marimo — ou lorsque vous détectez une session Marimo en cours d'exécution en répertoriant les sessions. Ne PAS utiliser pour les scripts Python généraux en dehors de marimo ou pour le développement de plugins/packages marimo.
