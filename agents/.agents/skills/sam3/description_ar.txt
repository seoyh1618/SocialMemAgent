قم بإنشاء Meta SAM 3 (facebookresearch/sam3) والعمل معه لتجزئة الصور والفيديو ذات المفردات المفتوحة باستخدام مطالبات النص والنقطة والمربع والقناع. يُستخدم عند إعداد بيئات SAM3، أو طلب الوصول إلى نقطة تفتيش Hugging Face، أو إنشاء برامج نصية للاستدلال، أو دمج SAM3 في تطبيقات Python، أو الضبط الدقيق باستخدام تكوينات sam3/train، أو تشغيل SA-Co أو التقييمات المخصصة، أو تصحيح أخطاء CUDA/نقطة التفتيش/مشكلات خطوط الأنابيب السريعة.
