Управляет бесплатными моделями ИИ из OpenRouter для OpenClaw. Автоматически ранжирует модели по качеству, настраивает резервные варианты для обработки ограничений скорости и обновляет openclaw.json. Используйте, когда пользователь упоминает бесплатный ИИ, OpenRouter, переключение модели, ограничения скорости или хочет снизить затраты на ИИ.
