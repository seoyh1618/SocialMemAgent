Автоматически сгенерируйте конфигурацию рабочего процесса проекта (docs/workflow.md) из существующих PRD и CLAUDE.md без вопросов. Используйте, когда пользователь говорит «настроить рабочий процесс», «настроить TDD», «подключить рабочий процесс разработки» или после запуска /scaffold перед /plan. НЕ используйте для настройки основателя (используйте /init) или создания каркаса проекта (используйте /scaffold).
