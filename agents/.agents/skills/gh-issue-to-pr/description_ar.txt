تنفيذ مشكلات GitHub من البداية إلى النهاية باستخدام GitHub CLI وgit، بدءًا من استنساخ المستودع (والتفرع فقط عند الحاجة) إلى فتح طلب السحب. استخدمه عندما يُطلب منك التقاط مشكلة معينة من GitHub وتقديم علاقات عامة جاهزة للمراجعة مع سياق المشكلة المرتبط والالتزامات وملاحظات التحقق من الصحة وتصريح جودة ما قبل العلاقات العامة عبر $code-review.
