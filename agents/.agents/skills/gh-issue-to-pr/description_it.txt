Implementa i problemi di GitHub end-to-end utilizzando la CLI di GitHub e git, dalla clonazione di un repository (e il fork solo quando necessario) all'apertura di una richiesta pull. Da utilizzare quando viene richiesto di prelevare un problema specifico da GitHub e fornire un PR pronto per la revisione con contesto del problema collegato, commit, note di convalida e un pass di qualità pre-PR tramite $code-review.
