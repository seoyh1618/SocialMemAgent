Реализуйте проблемы GitHub комплексно, используя GitHub CLI и git, от клонирования репозитория (и разветвления только при необходимости) до открытия запроса на включение. Используйте, когда вас попросят выбрать конкретную проблему на GitHub и предоставить готовый к рассмотрению PR со связанным контекстом проблемы, коммитами, примечаниями по проверке и передачей качества перед PR через $code-review.
