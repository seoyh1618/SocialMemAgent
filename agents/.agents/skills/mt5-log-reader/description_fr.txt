Lit les fichiers journaux MetaTrader 5 pour valider l'exécution des indicateurs, les tests unitaires et les erreurs de compilation. Déclencheurs : vérifier les journaux, cela a-t-il fonctionné, volet Experts, sortie d'impression, erreurs d'exécution, sortie de débogage. (projet)
