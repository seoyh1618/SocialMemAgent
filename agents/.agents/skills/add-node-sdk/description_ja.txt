サーバー側分析、KV ストレージ、BLOB ストレージ、プラットフォーム管理のために Temps Node.js SDK を統合します。ユーザーが次のことを行う場合に使用します: (1) サーバー側のイベントを追跡する、(2) Temps KV (キーと値) ストレージを使用する、(3) ファイルに Temps Blob ストレージを使用する、(4) Node.js から Temps API にアクセスする、(5) サーバー側の分析統合、(6) バックエンド イベントの追跡。トリガー: 「temps ノード SDK」、「サーバー側分析」、「temps kv」、「temps blob」、「バックエンド統合」、「node.js temps」。
