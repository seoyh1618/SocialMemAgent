Mettez en place un projet de contrat intelligent Stylus avec OpenZeppelin Contracts pour Stylus sur Arbitrum. À utiliser lorsque les utilisateurs doivent : (1) installer la chaîne d'outils Rust et la cible WASM pour Stylus, (2) créer un nouveau projet Cargo Stylus, (3) ajouter les dépendances OpenZeppelin Stylus à Cargo.toml, ou (4) comprendre les conventions d'importation et les modèles de stockage de Stylus pour OpenZeppelin.
