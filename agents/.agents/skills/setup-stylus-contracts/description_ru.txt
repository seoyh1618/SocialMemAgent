Настройте проект смарт-контракта Stylus с помощью OpenZeppelin Contracts для Stylus на Arbitrum. Используйте, когда пользователям необходимо: (1) установить набор инструментов Rust и цель WASM для Stylus, (2) создать новый проект Cargo Stylus, (3) добавить зависимости OpenZeppelin Stylus в Cargo.toml или (4) понять правила импорта Stylus и шаблоны хранения для OpenZeppelin.
