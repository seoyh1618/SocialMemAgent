قم بإعداد مشروع عقد ذكي لـ Stylus باستخدام OpenZeppelin Contracts for Stylus على Arbitrum. يُستخدم عندما يحتاج المستخدمون إلى: (1) تثبيت سلسلة أدوات Rust وهدف WASM لـ Stylus، (2) إنشاء مشروع Cargo Stylus جديد، (3) إضافة تبعيات OpenZeppelin Stylus إلى Cargo.toml، أو (4) فهم اصطلاحات استيراد Stylus وأنماط التخزين لـ OpenZeppelin.
