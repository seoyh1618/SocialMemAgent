Configure un proyecto de contrato inteligente de Stylus con OpenZeppelin Contracts para Stylus en Arbitrum. Úselo cuando los usuarios necesiten: (1) instalar la cadena de herramientas Rust y el objetivo WASM para Stylus, (2) crear un nuevo proyecto de Cargo Stylus, (3) agregar dependencias de OpenZeppelin Stylus a Cargo.toml o (4) comprender las convenciones de importación de Stylus y los patrones de almacenamiento para OpenZeppelin.
