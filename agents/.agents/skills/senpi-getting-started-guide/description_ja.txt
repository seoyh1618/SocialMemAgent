Senpi/Hyperliquid での最初の取引についてユーザーをガイドします。ディスカバリー (トップトレーダー) を段階的に説明し、選択したトレーダーとのミラー戦略を作成し、モニタリングし、戦略を終了します。ユーザーが「取引してみましょう」、「初めて取引します」、「取引を教えてください」、「取引方法を教えてください」と言う場合、または状態が AWAITING_FIRST_TRADE の場合に使用します。状態が READY でない場合にも実行できます (例: エントリポイント ステップ 3 の後)。その後、必要に応じて開始する前にウォレットへの資金調達を求めるプロンプトが表示されます。 Senpi MCP を接続する必要があります。
