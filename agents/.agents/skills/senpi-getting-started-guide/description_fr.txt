Guide les utilisateurs lors de leur première transaction sur Senpi/Hyperliquide. Parcourt la découverte (meilleurs traders), crée une stratégie miroir avec un trader choisi, surveille et clôture la stratégie. À utiliser lorsque l'utilisateur dit « échangeons », « premier échange », « apprends-moi à échanger », « comment échanger » ou lorsque l'état est AWAITING_FIRST_TRADE. Peut également s'exécuter lorsque l'état n'est pas PRÊT (par exemple après l'étape 3 du point d'entrée) ; demande ensuite un financement de portefeuille avant de commencer en cas de besoin. Nécessite la connexion de Senpi MCP.
