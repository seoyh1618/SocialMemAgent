يرشد المستخدمين خلال تجارتهم الأولى على Senpi/Hyperliquid. يمشي من خلال الاكتشاف (كبار المتداولين)، وينشئ إستراتيجية مرآة مع المتداول المختار، ويراقب الإستراتيجية ويغلقها. يُستخدم عندما يقول المستخدم "دعونا نتداول"، أو "التداول الأول"، أو "علمني التداول"، أو "كيف أتداول"، أو عندما تكون الحالة AWAITING_FIRST_TRADE. يمكن تشغيله أيضًا عندما لا تكون الحالة جاهزة (على سبيل المثال، بعد نقطة الدخول الخطوة 3)؛ ثم يطالب بتمويل المحفظة قبل البدء عند الحاجة. يتطلب اتصال Senpi MCP.
