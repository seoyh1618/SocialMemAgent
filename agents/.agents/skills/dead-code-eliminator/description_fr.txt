Auditez une base de code pour détecter le code mort (fonctions inaccessibles, importations inutilisées, classes orphelines, branches inaccessibles et indicateurs de fonctionnalités obsolètes) en traçant les chemins d'appels en direct à partir de tous les points d'entrée. Vérifie également l’exactitude (les implémentations correspondent-elles à leur objectif revendiqué ?) et la solidité algorithmique. Produit un graphique d'appel ASCII et un rapport de code mort catégorisé. À utiliser lorsqu'une base de code a subi de nombreuses itérations, qu'un grand refactor a atterri, qu'une fonctionnalité a été supprimée mais que son code de support peut rester, ou que l'utilisateur demande de "trouver le code mort", "nettoyer le code inutilisé", "auditer ce qui est réellement utilisé" ou "trouver les fonctions orphelines".
