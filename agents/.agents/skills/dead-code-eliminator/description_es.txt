Audite una base de código en busca de código inactivo (funciones inalcanzables, importaciones no utilizadas, clases huérfanas, ramas inalcanzables e indicadores de funciones obsoletas) rastreando rutas de llamadas en vivo desde todos los puntos de entrada. También audita la corrección (¿las implementaciones coinciden con el propósito declarado?) y la solidez algorítmica. Produce un gráfico de llamadas ASCII y un informe de códigos muertos categorizados. Úselo cuando una base de código ha pasado por muchas iteraciones, ha llegado una gran refactorización, se eliminó una función pero su código de soporte puede permanecer, o el usuario solicita "encontrar código inactivo", "limpiar código no utilizado", "auditar lo que realmente se usa" o "encontrar funciones huérfanas".
