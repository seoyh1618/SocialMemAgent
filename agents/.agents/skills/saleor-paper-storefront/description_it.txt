Modelli specifici del progetto per la vetrina Saleor Paper creati con Next.js 16, TypeScript e Tailwind CSS. Da utilizzare quando si lavora con pagine di prodotto, flusso di pagamento, memorizzazione nella cache, selezione di varianti, filtri, SEO o componenti dell'interfaccia utente. Per i modelli API Saleor universali, consulta la dipendenza saleor-storefront.
