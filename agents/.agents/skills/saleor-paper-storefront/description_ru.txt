Шаблоны, специфичные для проекта, для витрины магазина Saleor Paper, созданные с помощью Next.js 16, TypeScript и Tailwind CSS. Используйте при работе со страницами продуктов, процессом оформления заказа, кэшированием, выбором варианта, фильтрацией, SEO или компонентами пользовательского интерфейса. Универсальные шаблоны API Saleor см. в разделе зависимость saleor-storefront.
