Guider les développeurs intégrant EUrouter dans leurs applications. EUrouter est une passerelle IA compatible OpenAI pour la conformité UE/RGPD. À utiliser lors de l'intégration d'EURouter, du passage d'OpenRouter ou d'OpenAI, de la configuration de la résidence des données dans l'UE, du routage des demandes d'IA vers des fournisseurs de l'UE, de la gestion des clés API ou des questions sur l'API d'EURouter pour les complétions de chat, les intégrations, le streaming, les appels d'outils, la vision, le routage de modèles ou les fonctionnalités de conformité au RGPD.
