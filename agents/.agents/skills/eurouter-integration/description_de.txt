Leitfaden für Entwickler bei der Integration von EURouter in ihre Anwendungen. EUrouter ist ein OpenAI-kompatibles KI-Gateway für die Einhaltung der EU/DSGVO. Verwenden Sie diese Option, wenn Sie EUrouter integrieren, von OpenRouter oder OpenAI wechseln, die EU-Datenresidenz konfigurieren, KI-Anfragen an EU-Anbieter weiterleiten, API-Schlüssel verwalten oder nach der API von EUrouter für Chat-Vervollständigungen, Einbettungen, Streaming, Tool-Aufrufe, Vision, Modellrouting oder DSGVO-Compliance-Funktionen fragen.
