Guida gli sviluppatori che integrano EUrouter nelle loro applicazioni. EUrouter è un gateway AI compatibile con OpenAI per la conformità UE/GDPR. Da utilizzare durante l'integrazione di EUrouter, il passaggio da OpenRouter o OpenAI, la configurazione della residenza dei dati nell'UE, l'instradamento delle richieste di intelligenza artificiale ai fornitori dell'UE, la gestione delle chiavi API o la richiesta di informazioni sull'API di EUrouter per completamenti di chat, incorporamenti, streaming, chiamate di strumenti, visione, routing di modelli o funzionalità di conformità GDPR.
