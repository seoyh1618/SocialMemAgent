Используйте этот навык, когда пользователям необходимо выполнять поиск на нескольких платформах, включая GitHub (репозитории, код, проблемы), Reddit (сообщения, субреддиты, пользователи), Xiaohongshu, Douyin, Bilibili, YouTube, Twitter, Google, Tavily, Mita Search, Volcano Engine и общие поисковые системы (DuckDuckGo, Brave, Yahoo, Bing, Wikipedia, Anna's Archive) или для загрузки изображений с 18 платформ изображений (Baidu, Bing, Google, Pixabay, Unsplash, Volcano Engine и т. д.). Он обеспечивает унифицированный интерфейс поиска со структурированным выводом, фильтрацией результатов, сортировкой, автоматическим архивированием ответов и пакетной загрузкой изображений (с сохранением метаданных).
