تدقيق منهجي للمخطوطة قبل النشر، مما يؤدي إلى إنتاج تقرير إعادة هيكلة منظم مع تشخيصات على مستوى القسم، وتحليل صحة الاقتباس، وتقييم الاستعداد للتقديم. استخدم هذه المهارة عندما يقوم المستخدم بتحميل مخطوطة أو ورقة بحثية أو فصل من الأطروحة أو تقديم مجلة أو ورقة مؤتمر ويطلب المراجعة أو التغذية الراجعة أو التحرير أو إعادة البناء أو فحص ما قبل التقديم أو التدقيق اللغوي أو تدقيق الجودة. يتم تشغيله أيضًا عندما يقول المستخدم "راجع ورقتي"، أو "تحقق قبل الإرسال"، أو "هل هذا جاهز للإرسال"، أو "قائمة مراجعة ما قبل النشر"، أو "مراجعة المخطوطة"، أو "أعد صياغة ورقتي"، أو يسأل عن اتساق الاقتباس، أو تماسك الوسيطة، أو الامتثال للتنسيق. يغطي الطلبات الجزئية مثل "التحقق من مراجعي" أو "هل يعمل الملخص" - مشكلات الأسطح التشخيصية الكاملة عبر جميع الجوانب حتى عندما يتم السؤال عن جانب واحد فقط.
