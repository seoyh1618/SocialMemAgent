قم بتوصيل Spice بمصادر البيانات والاستعلام عبرها باستخدام SQL الموحد. يُستخدم عند الاتصال بقواعد البيانات (Postgres، وMySQL، وDynamoDB)، أو بحيرات البيانات (S3، أو Delta Lake، أو Iceberg)، أو المستودعات (Snowflake، أو Databricks)، أو الملفات، أو واجهات برمجة التطبيقات، أو الكتالوجات؛ تكوين مجموعات البيانات؛ خلق وجهات النظر. كتابة البيانات؛ أو إعداد استعلامات عبر المصادر.
