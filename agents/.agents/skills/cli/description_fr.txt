Package R complet pour le style de l'interface de ligne de commande, la messagerie sémantique et la communication utilisateur. Utilisez cette compétence lorsque vous travaillez avec du code R qui doit : (1) Formater la sortie de la console avec un balisage et des couleurs en ligne, (2) Afficher des erreurs, des avertissements ou des messages avec cli_abort/cli_warn/cli_inform, (3) Afficher des indicateurs de progression pour les opérations de longue durée, (4) Créer des éléments CLI sémantiques (en-têtes, listes, alertes, blocs de code), (5) Appliquer des thèmes et personnaliser le style de sortie, (6) Gérer la pluralisation dans le texte destiné à l'utilisateur, (7) Travailler avec Chaînes ANSI, hyperliens ou conteneurs personnalisés. À utiliser également lors de la migration à partir d'un message/avertissement/arrêt de base R, du débogage du code cli ou de l'amélioration de l'utilisation de la cli existante.
