Pacchetto R completo per lo stile dell'interfaccia della riga di comando, la messaggistica semantica e la comunicazione con l'utente. Utilizzare questa competenza quando si lavora con codice R che deve: (1) formattare l'output della console con markup e colori incorporati, (2) visualizzare errori, avvisi o messaggi con cli_abort/cli_warn/cli_inform, (3) mostrare indicatori di avanzamento per operazioni di lunga durata, (4) creare elementi CLI semantici (intestazioni, elenchi, avvisi, blocchi di codice), (5) applicare temi e personalizzare lo stile dell'output, (6) gestire la pluralizzazione nel testo rivolto all'utente, (7) utilizzare ANSI stringhe, collegamenti ipertestuali o contenitori personalizzati. Utilizzare anche durante la migrazione dal messaggio/avviso/arresto di base R, per il debug del codice CLI o per il miglioramento dell'utilizzo della CLI esistente.
