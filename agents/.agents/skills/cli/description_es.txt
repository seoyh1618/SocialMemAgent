Paquete completo de R para diseño de interfaz de línea de comandos, mensajería semántica y comunicación con el usuario. Utilice esta habilidad cuando trabaje con código R que necesite: (1) Formatear la salida de la consola con marcas y colores en línea, (2) Mostrar errores, advertencias o mensajes con cli_abort/cli_warn/cli_inform, (3) Mostrar indicadores de progreso para operaciones de larga duración, (4) Crear elementos CLI semánticos (encabezados, listas, alertas, bloques de código), (5) Aplicar temas y personalizar el estilo de salida, (6) Manejar la pluralización en el texto de cara al usuario, (7) Trabajar con Cadenas ANSI, hipervínculos o contenedores personalizados. Úselo también al migrar desde un mensaje/advertencia/detención de R base, depurar código cli o mejorar el uso de cli existente.
