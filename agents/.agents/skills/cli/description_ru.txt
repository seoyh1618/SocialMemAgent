Комплексный пакет R для стилизации интерфейса командной строки, семантического обмена сообщениями и взаимодействия с пользователем. Используйте этот навык при работе с кодом R, который должен: (1) форматировать вывод консоли с помощью встроенной разметки и цветов, (2) отображать ошибки, предупреждения или сообщения с помощью cli_abort/cli_warn/cli_inform, (3) показывать индикаторы прогресса для длительных операций, (4) создавать семантические элементы CLI (заголовки, списки, оповещения, блоки кода), (5) применять темы и настраивать стиль вывода, (6) обрабатывать множественное число при взаимодействии с пользователем text, (7) Работа со строками ANSI, гиперссылками или пользовательскими контейнерами. Также используйте при переходе с базового сообщения/предупреждения/остановки R, отладки кода интерфейса командной строки или улучшения использования существующего интерфейса командной строки.
