GramIO — типобезопасная платформа TypeScript Telegram Bot API для Node.js, Bun и Deno. Используйте при создании ботов Telegram, обработке команд/обратных вызовов/встроенных запросов, создании клавиатур, форматировании сообщений, загрузке файлов, управлении сеансами/сценами, написании плагинов, настройке веб-перехватчиков или интеграции платежей Telegram.
