GramIO: marco API TypeScript Telegram Bot con seguridad de tipos para Node.js, Bun y Deno. Úselo al crear bots de Telegram, manejar comandos/devoluciones de llamadas/consultas en línea, crear teclados, formatear mensajes, cargar archivos, administrar sesiones/escenas, escribir complementos, configurar webhooks o integrar pagos de Telegram.
