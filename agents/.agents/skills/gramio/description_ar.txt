GramIO — إطار عمل TypeScript Telegram Bot API آمن النوع لـ Node.js وBun وDeno. يُستخدم عند إنشاء روبوتات Telegram، أو التعامل مع الأوامر/رد الاتصال/الاستعلامات المضمنة، أو إنشاء لوحات المفاتيح، أو تنسيق الرسائل، أو تحميل الملفات، أو إدارة الجلسات/المشاهد، أو كتابة المكونات الإضافية، أو إعداد خطافات الويب، أو دمج مدفوعات Telegram.
