Помогите пользователям превратить прочитанный/просмотренный материал в содержательные эссе для личных размышлений. Прочитайте исходный материал и пользовательские заметки из папки sparks/, углубите точку зрения пользователя посредством диалога и, наконец, выведите структурированный план письма (idea.md). Подходит для фазы формирования идей: рецензий на книги, размышлений о фильмах, учебных заметок и эссе, основанных на размышлениях.
