Автоматизация скрытого обнаружения браузера с помощью Camoufox (подмена отпечатков пальцев на уровне C++ на базе Firefox). Используйте этот навык для ЛЮБОЙ автоматизации браузера, которая должна избегать обнаружения ботов — вход в систему в социальных сетях, постоянные сеансы, гуманное взаимодействие или очистка защищенных сайтов. Охватывает API Camoufox версии 2026, известные предупреждения о нестабильности, шаблоны асинхронности/синхронизации, постоянный контекст и конфигурацию гуманизации. ВСЕГДА используйте это вместо необработанного Драматурга, когда требуется скрытность.
