Entwurfs- und Überprüfungsunterstützung für Projekte, die Clean Architecture einführen. Basierend auf einer vierschichtigen Struktur (Domänenschicht, Anwendungsfallschicht, Schnittstellenadapterschicht, Infrastrukturschicht). Insbesondere sollte die Infrastrukturschicht nur Querschnittsthemen (Protokollierung, Konfigurationsmanagement) enthalten, während Persistenz und RPC in die Schnittstellenadapterschicht gehören. Auslöser: Wird nur aktiviert, wenn Clean Architecture explizit erwähnt wird – z. B. „mit Clean Architecture“, „im Anschluss an Clean Architecture“, „Clean Architecture-Überprüfung“. Es ist nicht für die allgemeine „Entwurfsprüfung“ oder „Architekturberatung“ aktiviert.
