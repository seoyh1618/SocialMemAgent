Создавайте стили с помощью Panda CSS. Используйте при создании, редактировании или просмотре любого кода, использующего Panda CSS — css(), cva(), sva(), рецепты, шаблоны, токены, семантические токены, конфигурации, темы или компоненты в стиле JSX. Поддерживает React, Vue, Svelte, Solid и любую среду с PostCSS.
