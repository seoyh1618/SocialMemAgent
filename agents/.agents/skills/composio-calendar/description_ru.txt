Календарь Google через Composio API. Используйте, когда:
(1) Создание событий календаря с правильной продолжительностью.
(2) Поиск/поиск событий
(3) Обновление или удаление событий
(4) Поиск свободного времени
ВАЖНО: CREATE и UPDATE используют РАЗНЫЕ параметры длительности. Этот навык предотвращает распространенную 30-минутную ошибку по умолчанию.
