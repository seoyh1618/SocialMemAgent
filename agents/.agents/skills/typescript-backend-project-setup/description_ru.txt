Настраивает монорепозиторий NX для серверных проектов TypeScript, оптимизированных для разработки с помощью искусственного интеллекта. Делегирует команды NX, где это возможно, исправляет конфигурации в крайнем случае. Срабатывает при: «настроить серверный проект машинописного текста», «создать серверный проект», «инициализировать серверную часть машинописного текста», «создать монорепо» или при работе в пустой папке проекта.
