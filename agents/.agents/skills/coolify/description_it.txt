Coolify PaaS self-hosted per la distribuzione di applicazioni, database e servizi tramite API REST e integrazione Git. Copre l'autenticazione API, i trigger di distribuzione, i pacchetti di build (Nixpacks, Dockerfile, Compose), CI/CD con webhook GitHub Actions, gestione dell'ambiente e configurazione del proxy Traefik.

Da utilizzare durante la distribuzione di app su Coolify, l'attivazione di distribuzioni tramite API, la configurazione di pacchetti di build, l'impostazione di pipeline CI/CD di GitHub Actions, la gestione di variabili di ambiente o la risoluzione di errori di distribuzione.
