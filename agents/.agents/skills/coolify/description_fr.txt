Coolify PaaS auto-hébergé pour le déploiement d'applications, de bases de données et de services via l'API REST et l'intégration Git. Couvre l'authentification API, les déclencheurs de déploiement, les packs de build (Nixpacks, Dockerfile, Compose), CI/CD avec les webhooks GitHub Actions, la gestion de l'environnement et la configuration du proxy Traefik.

À utiliser lors du déploiement d'applications sur Coolify, du déclenchement de déploiements via l'API, de la configuration des packs de build, de la configuration des pipelines CI/CD GitHub Actions, de la gestion des variables d'environnement ou du dépannage des échecs de déploiement.
