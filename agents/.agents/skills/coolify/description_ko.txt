REST API 및 Git 통합을 통해 애플리케이션, 데이터베이스 및 서비스를 배포하기 위한 자체 호스팅 PaaS를 Coolify하세요. API 인증, 배포 트리거, 빌드 팩(Nixpacks, Dockerfile, Compose), GitHub Actions 웹후크가 포함된 CI/CD, 환경 관리 및 Traefik 프록시 구성을 다룹니다.
