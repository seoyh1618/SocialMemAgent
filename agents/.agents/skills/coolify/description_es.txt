Coolify PaaS autohospedado para implementar aplicaciones, bases de datos y servicios a través de API REST e integración de Git. Cubre la autenticación de API, activadores de implementación, paquetes de compilación (Nixpacks, Dockerfile, Compose), CI/CD con webhooks de GitHub Actions, administración del entorno y configuración del proxy Traefik.

Úselo al implementar aplicaciones en Coolify, activar implementaciones a través de API, configurar paquetes de compilación, configurar canalizaciones de CI/CD de GitHub Actions, administrar variables de entorno o solucionar fallas de implementación.
