Самостоятельное PaaS Coolify для развертывания приложений, баз данных и сервисов через REST API и интеграцию с Git. Охватывает аутентификацию API, триггеры развертывания, пакеты сборки (Nixpacks, Dockerfile, Compose), CI/CD с веб-перехватчиками GitHub Actions, управление средой и настройку прокси-сервера Traefik.

Используйте при развертывании приложений в Coolify, запуске развертываний через API, настройке пакетов сборки, настройке конвейеров CI/CD GitHub Actions, управлении переменными среды или устранении неполадок при развертывании.
