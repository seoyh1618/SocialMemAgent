Управление схемой ClickHouse с помощью chkit. Используйте при работе с командами CLI chkit, определениями таблиц/представлений/материализованных представлений ClickHouse, созданием миграции, обнаружением отклонений или файлами clickhouse.config.ts. Запуск по командам chkit, импорту @chkit/core или задачам определения схемы.
