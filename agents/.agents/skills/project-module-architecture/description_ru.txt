Руководство по архитектуре модуля управления проектами, охватывающее CRUD проекта, управление участниками, конфигурацию проекта, управление тегами и миграцию проекта. Используется, когда пользователи разрабатывают функции управления проектами, управляют участниками проекта, настраивают атрибуты проекта или реализуют логику, связанную с проектом.
