Eseguire il debug delle applicazioni HarmonyOS ArkWeb utilizzando il protocollo Chrome DevTools. Da utilizzare quando Claude deve: (1) avviare sessioni di debug di ArkWeb, (2) connettere Chrome DevTools alle app HarmonyOS tramite hdc, (3) ispezionare elementi di visualizzazione Web, console e rete, (4) testare le applicazioni ArkWeb con Chrome DevTools MCP, (5) risolvere problemi di debug di visualizzazione Web o (6) gestire sessioni di port forwarding e debug. Rileva automaticamente la configurazione del progetto, la connessione del dispositivo e i socket di debug. Funziona con la capacità ohos-app-build-debug per un flusso di lavoro completo.
