Отладка приложений HarmonyOS ArkWeb с использованием протокола Chrome DevTools. Используйте, когда Клоду необходимо: (1) запустить сеансы отладки ArkWeb, (2) подключить Chrome DevTools к приложениям HarmonyOS через hdc, (3) проверить элементы веб-просмотра, консоль и сеть, (4) протестировать приложения ArkWeb с помощью Chrome DevTools MCP, (5) устранить проблемы с отладкой веб-просмотра или (6) управлять сеансами переадресации портов и отладки. Автоматически определяет конфигурацию проекта, подключение устройств и сокеты отладки. Работает с навыком ohos-app-build-debug для полноценного рабочего процесса.
