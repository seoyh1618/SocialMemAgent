Déboguez les applications HarmonyOS ArkWeb à l'aide du protocole Chrome DevTools. À utiliser lorsque Claude doit : (1) démarrer des sessions de débogage ArkWeb, (2) connecter Chrome DevTools aux applications HarmonyOS via hdc, (3) inspecter les éléments, la console et le réseau de la vue Web, (4) tester les applications ArkWeb avec Chrome DevTools MCP, (5) résoudre les problèmes de débogage de la vue Web ou (6) gérer les sessions de redirection de port et de débogage. Détecte automatiquement la configuration du projet, la connexion de l'appareil et les sockets de débogage. Fonctionne avec la compétence ohos-app-build-debug pour un flux de travail complet.
