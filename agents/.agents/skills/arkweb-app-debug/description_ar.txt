تصحيح أخطاء تطبيقات HarmonyOS ArkWeb باستخدام بروتوكول Chrome DevTools. يُستخدم عندما يحتاج Claude إلى: (1) بدء جلسات تصحيح أخطاء ArkWeb، (2) توصيل Chrome DevTools بتطبيقات HarmonyOS عبر hdc، (3) فحص عناصر عرض الويب ووحدة التحكم والشبكة، (4) اختبار تطبيقات ArkWeb باستخدام Chrome DevTools MCP، (5) استكشاف مشكلات تصحيح أخطاء عرض الويب وإصلاحها، أو (6) إدارة جلسات إعادة توجيه المنفذ وتصحيح الأخطاء. يكتشف تلقائيًا تكوين المشروع واتصال الجهاز ومآخذ تصحيح الأخطاء. يعمل بمهارة ohos-app-build-debug لسير العمل الكامل.
