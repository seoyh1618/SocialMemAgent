Chrome DevTools プロトコルを使用して HarmonyOS ArkWeb アプリケーションをデバッグします。クロードが次のことを行う必要がある場合に使用します: (1) ArkWeb デバッグ セッションを開始する、(2) hdc 経由で Chrome DevTools を HarmonyOS アプリに接続する、(3) Web ビュー要素、コンソール、およびネットワークを検査する、(4) Chrome DevTools MCP を使用して ArkWeb アプリケーションをテストする、(5) Web ビューのデバッグの問題をトラブルシューティングする、または (6) ポート転送とデバッグ セッションを管理する。プロジェクト構成、デバイス接続、およびデバッグソケットを自動的に検出します。完全なワークフローのために ohos-app-build-debug スキルと連携します。
