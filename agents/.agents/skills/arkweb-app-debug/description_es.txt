Depure aplicaciones HarmonyOS ArkWeb utilizando el protocolo Chrome DevTools. Úselo cuando Claude necesite: (1) iniciar sesiones de depuración de ArkWeb, (2) conectar Chrome DevTools a aplicaciones HarmonyOS a través de hdc, (3) inspeccionar los elementos de la vista web, la consola y la red, (4) probar las aplicaciones ArkWeb con Chrome DevTools MCP, (5) solucionar problemas de depuración de la vista web o (6) administrar el reenvío de puertos y las sesiones de depuración. Detecta automáticamente la configuración del proyecto, la conexión del dispositivo y los sockets de depuración. Funciona con la habilidad ohos-app-build-debug para un flujo de trabajo completo.
