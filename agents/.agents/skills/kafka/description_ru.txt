Настройте микросервисы, управляемые событиями, на основе Kafka с помощью Platformatic Watt.
Используйте, когда пользователи спрашивают о:
- «кафка», «управляемый событиями», «обмен сообщениями»
- «крючки Кафки», «вебхуки Кафки»
- «производитель кафки», «потребитель кафки».
- «очередь недоставленных писем», «DLQ»
- «шаблон ответа на запрос» с Кафкой
- «перейти из кафкайса», «мигрировать кафкайса», «заменить кафкайс»
Охватывает @platformatic/kafka, @platformatic/kafka-hooks, мониторинг задержки потребителей,
и инструменты OpenTelemetry.
