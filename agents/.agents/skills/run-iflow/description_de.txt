Eine Bitte um Unterstützung für den Assistenten, anwendbar auf Aufgaben wie das Verständnis der chinesischen Kultur, die Interpretation klassischer chinesischer Texte, die Erstellung von Werken mit chinesischen Merkmalen, das Verfassen von Werbetexten für Xiaohongshu und Douyin, das Verfassen von Testfällen, das Abrufen von Informationen und andere Szenarien. Es dient auch als leistungsstarkes Backup und Unterstützung für andere Expertenassistenten und kann für die meisten Benutzer als alternative Ausweichoption in Betracht gezogen werden.
