Выполняет семантический анализ кода и оптимизацию токенов посредством разработки контекста и автоматической упаковки контекста. Используйте при сокращении накладных расходов на токены для больших баз кода, создании дайджестов репозитория с помощью Gitingest, упаковке контекста кода с помощью Repomix или отслеживании межфайловых зависимостей с помощью llm-tldr.
