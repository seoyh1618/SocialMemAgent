Effectue une intelligence sémantique du code et une optimisation des jetons grâce à l’ingénierie du contexte et à l’emballage automatisé du contexte. À utiliser pour réduire la surcharge des jetons pour les bases de code volumineuses, créer des résumés de référentiel avec Gitingest, empaqueter le contexte de code avec Repomix ou tracer les dépendances entre fichiers avec llm-tldr.
