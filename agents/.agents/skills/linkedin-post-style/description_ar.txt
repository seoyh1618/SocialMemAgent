اكتب منشورات على LinkedIn تتطابق مع صوت مؤلف تقني محدد - مباشر، وتحليلي، وذو روح الدعابة، ودقيق. استخدم هذه المهارة عندما يطلب المستخدم كتابة أو صياغة أو إعادة كتابة أو مراجعة أو تحسين أو تحسين منشور على LinkedIn أو منشور على وسائل التواصل الاجتماعي أو تعليق تقني أو أي كتابات قصيرة موجهة للعامة حول التكنولوجيا أو الذكاء الاصطناعي أو هندسة البرمجيات أو أدوات المطورين. يتم تشغيله أيضًا عندما يقول المستخدم "اكتب هذا بأسلوبي"، أو "أنشر هذا الأمر"، أو "أعد كتابة هذا لـ LinkedIn"، أو "قم بصياغة منشور بأسلوبي"، أو "هل يبدو هذا صحيحًا"، أو "كيف يمكنني صياغة هذا"، أو يقدم محتوى/ملاحظات أولية ويريد تشكيلها في منشور. يتضمن إرشادات مصاحبة مرئية لإقران المنشورات مع دوارات المستندات (عبر تحويل md إلى pdf مع مخططات حورية البحر)، أو الصور المخصصة (عبر تحويل المفهوم إلى صورة)، أو الرسوم المتحركة (عبر تحويل المفهوم إلى فيديو).
