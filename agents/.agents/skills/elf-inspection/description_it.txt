Abilità di ispezione binaria ELF per Linux. Da utilizzare quando si esaminano eseguibili ELF o librerie condivise con readelf, objdump, nm o ldd per comprendere la visibilità dei simboli, il layout della sezione, le dipendenze dinamiche, gli ID di build o le voci di riposizionamento. Si attiva su query su formato ELF, dipendenze di librerie condivise, tabelle di simboli, dimensioni di sezione, informazioni di debug DWARF nei file binari, analisi gonfiata dei file binari o errori di simboli non definiti.
