Compétence d'inspection binaire ELF pour Linux. À utiliser lors de l'examen des exécutables ELF ou des bibliothèques partagées avec readelf, objdump, nm ou ldd pour comprendre la visibilité des symboles, la disposition des sections, les dépendances dynamiques, les ID de build ou les entrées de relocalisation. S'active sur les requêtes concernant le format ELF, les dépendances de bibliothèques partagées, les tables de symboles, les tailles de section, les informations de débogage DWARF dans les binaires, l'analyse de ballonnement binaire ou les erreurs de symboles non définies.
