ELF مهارة الفحص الثنائي لنظام التشغيل Linux. يُستخدم عند فحص ملفات ELF التنفيذية أو المكتبات المشتركة باستخدام readelf أو objdump أو nm أو ldd لفهم رؤية الرمز أو تخطيط القسم أو التبعيات الديناميكية أو معرفات البناء أو إدخالات النقل. يتم التنشيط عند الاستعلامات حول تنسيق ELF، أو تبعيات المكتبة المشتركة، أو جداول الرموز، أو أحجام الأقسام، أو معلومات تصحيح DWARF في الثنائيات، أو تحليل الحجم الثنائي، أو أخطاء الرموز غير المحددة.
