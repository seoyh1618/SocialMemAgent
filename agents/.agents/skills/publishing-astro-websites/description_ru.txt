Комплексное руководство по созданию и развертыванию статических веб-сайтов с помощью платформы Astro.
Этот навык следует использовать, когда вас просят «создать космическую площадку», «развернуть космическую базу на огневой базе»,
«настроить коллекции контента», «добавить диаграммы русалок в astro», «настроить astro i18n»,
«создать статический блог» или «настроить astro markdown». Охватывает основы SSG, коллекции контента,
Markdown/MDX, частичная гидратация, островная архитектура и развертывание в Netlify, Vercel,
Страницы GitHub или GCP/Firebase.
