À utiliser lors de la définition de schémas de base de données, de la création de collections de données ou de la configuration d'un stockage de données structuré. Les déclencheurs incluent le CMS, la collection, la base de données, le schéma, le modèle de données, les champs, les relations, les autorisations, la structure des données, la définition de l'entité et l'extension des données.
