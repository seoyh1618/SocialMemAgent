قم بإنشاء سلاسل المستندات وإضافتها تلقائيًا إلى جميع التعليمات البرمجية التي تم تغييرها في git diff. استخدم هذه المهارة عندما يطلب المستخدم: إضافة سلاسل مستندات إلى تغييراته، أو توثيق git diff، أو إضافة تغطية وثائقية للالتزامات الأخيرة، أو كتابة سلاسل مستندات للوظائف المعدلة/الجديدة، أو تحسين الوثائق على الملفات التي تم تغييرها، أو أي شيء يتضمن "docstring" + "diff"/"changes"/"commit". يتم تشغيله أيضًا عندما يقول المستخدم أشياء مثل "توثيق تغييراتي" أو "إضافة مستندات إلى ما قمت بتغييره" أو "تغطية سلسلة المستندات للعلاقات العامة الخاصة بي".
