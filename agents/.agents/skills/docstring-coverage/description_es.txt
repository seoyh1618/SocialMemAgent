Genere y agregue automáticamente cadenas de documentación a todo el código modificado en un git diff. Utilice esta habilidad siempre que el usuario solicite: agregar cadenas de documentación a sus cambios, documentar su git diff, agregar cobertura de documentación para confirmaciones recientes, escribir cadenas de documentación para funciones nuevas/modificadas, mejorar la documentación de archivos modificados o cualquier cosa que involucre "cadena de documentación" + "diff"/"cambios"/"compromiso". También se activa cuando el usuario dice cosas como "documentar mis cambios", "agregar documentos a lo que cambié" o "cobertura de cadena de documentos para mi PR".
