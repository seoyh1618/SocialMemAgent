Genera e aggiungi automaticamente docstring a tutto il codice modificato in un git diff. Usa questa abilità ogni volta che l'utente chiede di: aggiungere docstring alle proprie modifiche, documentare il proprio git diff, aggiungere documentazione per commit recenti, scrivere docstring per funzioni modificate/nuove, migliorare la documentazione sui file modificati o qualsiasi cosa coinvolga "docstring" + "diff"/"changes"/"commit". Si attiva anche quando l'utente dice cose come "documenta le mie modifiche", "aggiungi documenti a ciò che ho modificato" o "copertura della stringa di documenti per il mio PR".
