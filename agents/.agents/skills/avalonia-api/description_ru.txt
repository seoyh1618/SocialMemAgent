Комплексный справочник по платформе пользовательского интерфейса Avalonia, включая синтаксис XAML, элементы управления, привязку данных, шаблоны MVVM, стили, настраиваемые элементы управления, систему макетов и лучшие практики. Охватывает интеграцию ReactiveUI, скомпилированные привязки, свойства зависимостей, прикрепленные свойства, шаблоны элементов управления и шаблоны кроссплатформенной разработки.
