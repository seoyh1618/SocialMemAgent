Analyse intelligente Core Web Vitals avec flux de travail et arbres de décision automatisés. Mesure LCP, CLS, INP avec débogage guidé qui détermine automatiquement l'analyse de suivi en fonction des résultats. Comprend des flux de travail pour l'analyse approfondie du LCP (5 phases), l'enquête CLS (chargement vs interaction), le débogage INP (répartition de la latence + attribution) et l'intégration inter-compétences avec le chargement, l'interaction et les compétences multimédias. À utiliser lorsque l'utilisateur pose des questions sur les Core Web Vitals, l'optimisation LCP, les changements de mise en page ou la réactivité des interactions. Compatible avec Chrome DevTools MCP.
