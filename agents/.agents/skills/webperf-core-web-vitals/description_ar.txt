التحليل الذكي لمؤشرات أداء الويب الأساسية باستخدام مسارات العمل الآلية وأشجار القرار. يقيس LCP وCLS وINP من خلال تصحيح الأخطاء الموجه الذي يحدد تحليل المتابعة تلقائيًا بناءً على النتائج. يتضمن سير العمل للتعمق العميق في LCP (5 مراحل)، وتحقيق CLS (التحميل مقابل التفاعل)، وتصحيح أخطاء INP (انهيار زمن الاستجابة + الإسناد)، والتكامل عبر المهارات مع مهارات التحميل والتفاعل والوسائط. يُستخدم عندما يسأل المستخدم عن Core Web Vitals، أو تحسين LCP، أو تغييرات التخطيط، أو استجابة التفاعل. متوافق مع Chrome DevTools MCP.
