Análisis inteligente de Core Web Vitals con flujos de trabajo automatizados y árboles de decisión. Mide LCP, CLS, INP con depuración guiada que determina automáticamente el análisis de seguimiento basado en los resultados. Incluye flujos de trabajo para profundización de LCP (5 fases), investigación de CLS (carga versus interacción), depuración de INP (desglose de latencia + atribución) e integración entre habilidades con carga, interacción y habilidades multimedia. Úselo cuando el usuario pregunte sobre Core Web Vitals, optimización de LCP, cambios de diseño o capacidad de respuesta de la interacción. Compatible con Chrome DevTools MCP.
