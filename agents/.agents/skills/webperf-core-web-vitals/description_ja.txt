自動化されたワークフローとデシジョン ツリーによるインテリジェントな Core Web Vitals 分析。結果に基づいて追跡分析を自動的に決定するガイド付きデバッグで LCP、CLS、INP を測定します。 LCP の詳細な調査 (5 フェーズ)、CLS 調査 (読み込みとインタラクション)、INP デバッグ (レイテンシの内訳 + 属性)、および読み込み、インタラクション、メディア スキルとのクロススキル統合のワークフローが含まれます。ユーザーが Core Web Vitals、LCP の最適化、レイアウトの変更、またはインタラクションの応答性について質問する場合に使用します。 Chrome DevTools MCP と互換性があります。
