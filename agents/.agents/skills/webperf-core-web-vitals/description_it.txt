Analisi intelligente dei Core Web Vitals con flussi di lavoro automatizzati e alberi decisionali. Misura LCP, CLS, INP con debug guidato che determina automaticamente l'analisi di follow-up in base ai risultati. Include flussi di lavoro per l'approfondimento LCP (5 fasi), l'indagine CLS (caricamento vs interazione), il debug INP (scomposizione della latenza + attribuzione) e l'integrazione di competenze incrociate con capacità di caricamento, interazione e multimediali. Da utilizzare quando l'utente chiede informazioni su Core Web Vitals, ottimizzazione LCP, cambiamenti di layout o reattività dell'interazione. Compatibile con Chrome DevTools MCP.
