Интеллектуальный анализ основных веб-показателей с автоматизированными рабочими процессами и деревьями решений. Измеряет LCP, CLS, INP с управляемой отладкой, которая автоматически определяет последующий анализ на основе результатов. Включает рабочие процессы для глубокого погружения в LCP (5 этапов), исследования CLS (загрузка и взаимодействие), отладки INP (разбивка по задержке + атрибуция) и интеграцию между навыками загрузки, взаимодействия и навыков работы с мультимедиа. Используйте, когда пользователь спрашивает об основных веб-показателях, оптимизации LCP, изменениях макета или скорости реагирования на взаимодействие. Совместим с Chrome DevTools MCP.
