Motor de sincronización de Postgres en tiempo real de ElectricSQL que utiliza Shapes para replicación parcial. Cubre la API ShapeStream, los enlaces de React, el filtrado de cláusulas Where, la selección de columnas, los patrones de proxy de autenticación y las estrategias de escritura progresiva desde solo en línea hasta a través de la base de datos.

Úselo al configurar ElectricSQL, configurar la sincronización de Postgres, implementar la carga de datos basada en formas, crear servidores proxy de autenticación para formas o elegir patrones de escritura para aplicaciones locales con Electric.
