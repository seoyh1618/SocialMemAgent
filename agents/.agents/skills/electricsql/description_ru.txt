Механизм синхронизации Postgres в реальном времени ElectricSQL с использованием фигур для частичной репликации. Охватывает API ShapeStream, перехватчики React, где фильтрация предложений, выбор столбцов, шаблоны прокси-сервера аутентификации и стратегии прогрессивной записи от онлайн-только до использования базы данных.

Используйте при настройке ElectricSQL, настройке синхронизации Postgres, реализации загрузки данных на основе фигур, создании прокси-серверов аутентификации для фигур или выборе шаблонов записи для локальных приложений с помощью Electric.
