Chargez de manière PROACTIVE lorsque la tâche implique une architecture CSS, des thèmes ou des systèmes de conception visuelle. À utiliser lorsque l'utilisateur dit « configurer Tailwind », « ajouter le mode sombre », « créer un système de conception », « le rendre réactif » ou « configurer le thème ». Couvre la configuration Tailwind et les plugins personnalisés, les systèmes de jetons de conception, les stratégies de points d'arrêt réactifs, la mise en œuvre du mode sombre, les modèles de variantes de composants (CVA/class-variance-authority), les alternatives CSS-in-JS et les modèles d'animation.
