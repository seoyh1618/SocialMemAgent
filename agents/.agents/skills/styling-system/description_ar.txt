قم بالتحميل بشكل استباقي عندما تتضمن المهمة بنية CSS أو السمات أو أنظمة التصميم المرئي. يُستخدم عندما يقول المستخدم "إعداد Tailwind"، أو "إضافة وضع مظلم"، أو "إنشاء نظام تصميم"، أو "جعله مستجيبًا"، أو "تكوين السمة". يغطي تكوين Tailwind والمكونات الإضافية المخصصة، وأنظمة الرموز المميزة للتصميم، واستراتيجيات نقاط التوقف سريعة الاستجابة، وتنفيذ الوضع المظلم، وأنماط متغيرات المكونات (CVA/class-variance-authority)، وبدائل CSS-in-JS، وأنماط الرسوم المتحركة.
