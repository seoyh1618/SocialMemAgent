タスクに CSS アーキテクチャ、テーマ、またはビジュアル デザイン システムが含まれる場合は、積極的に読み込みます。ユーザーが「Tailwind をセットアップする」、「ダーク モードを追加する」、「デザイン システムを作成する」、「レスポンシブにする」、または「テーマを構成する」と言うときに使用します。 Tailwind 構成とカスタム プラグイン、デザイン トークン システム、レスポンシブ ブレークポイント戦略、ダーク モード実装、コンポーネント バリアント パターン (CVA/クラスバリアンス オーソリティ)、CSS-in-JS の代替案、およびアニメーション パターンをカバーします。
