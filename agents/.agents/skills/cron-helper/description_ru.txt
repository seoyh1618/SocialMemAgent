Используйте интерфейс командной строки openclaw cron для управления запланированными задачами. Срабатывает, когда пользователям нужны периодические напоминания, запланированные задачи, проверки биографических данных или выполнение задач агента по времени. Ключевые слова: расписание, ежедневно/еженедельно/час/минута/секунда, напоминание, расписание, напоминать, каждые, cron, запланированная задача, выполнить позже, отложенное выполнение, расписание, план, будильник, обратный отсчет, X минут спустя, X секунд спустя, определенное время.
