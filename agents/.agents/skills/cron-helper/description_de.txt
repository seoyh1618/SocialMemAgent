Verwenden Sie die Openclaw-Cron-CLI, um geplante Aufgaben zu verwalten. Wird ausgelöst, wenn Benutzer regelmäßige Erinnerungen, geplante Aufgaben, Hintergrundüberprüfungen oder die Ausführung von Agentenaufgaben zu einem bestimmten Zeitpunkt benötigen. Schlüsselwörter: geplant, täglich/wöchentlich/stündlich/Minute/Sekunde, Erinnerung, Zeitplan, erinnern, alle, Cron, geplante Aufgabe, später ausführen, verzögerte Ausführung, Reserve, Plan, Alarm, Countdown, X Minuten später, X Sekunden später, HH:MM.
