Utilizza la CLI cron di openclaw per gestire le attività pianificate. Si attiva quando gli utenti necessitano di promemoria periodici, attività pianificate, controlli dei precedenti o attività dell'agente a tempo. Parole chiave: pianificazione, giornaliero/settimanale/orario/minuto/secondo, promemoria, pianificazione, promemoria, ogni, cron, attività pianificata, esegui più tardi, ritarda esecuzione, prenota, pianifica, allarme, conto alla rovescia, X minuti dopo, X secondi dopo, ora specifica.
