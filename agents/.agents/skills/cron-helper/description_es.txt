Utilice la CLI cron de openclaw para gestionar tareas programadas. Se activa cuando los usuarios necesitan recordatorios periódicos, tareas programadas, verificaciones de antecedentes o ejecución programada de tareas de agentes. Palabras clave: programado, diario/semanal/hora/minuto/cada segundo, recordatorio, programación, recordatorio, cada, cron, tarea programada, ejecutar más tarde, ejecución retrasada, programado, plan, alarma, cuenta regresiva, X minutos después, X segundos después, a qué hora.
