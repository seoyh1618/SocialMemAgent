Набор инструментов для управления хранилищем кода Alibaba Cloud Yunxiao Codeup. Варианты использования включают: (1) операции с хранилищем кода — управление ветвями, файловые операции, сравнение кода, управление мерж-реквестами/MR; (2) управление организацией — управление отделом, запросы участников, управление ролями; (3) работа с репозиториями Codeup, ветками, MR и мерж-реквестами; (4) опрос членов организации Yunxiao и списков отделов.
