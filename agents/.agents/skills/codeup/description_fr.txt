Ensemble d'outils de gestion du référentiel de codes Alibaba Cloud Yunxiao Codeup. Les cas d'utilisation incluent : (1) opérations de référentiel de code : gestion des branches, opérations sur les fichiers, comparaison de code, gestion des demandes de fusion/MR ; (2) gestion de l'organisation : gestion de département, requêtes des membres, gestion des rôles ; (3) exploiter les référentiels Codeup, les branches, les MR et les demandes de fusion ; (4) interroger les membres de l'organisation Yunxiao et les listes de départements.
