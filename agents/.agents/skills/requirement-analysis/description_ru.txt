Обеспечивает систематический рабочий процесс анализа требований для понимания требований, изучения базы кода, выяснения проблем, выполнения углубленного анализа и представления плана реализации перед кодированием. Подходит для разработки функций, проектирования API, проектирования баз данных, разработки модулей и модификации функций. Этот навык автоматически активируется, когда пользователи запрашивают разработку функций, проектирование API, проектирование базы данных и т. д. Взаимодействует в соответствии с настройками пользователя и языком ввода.
