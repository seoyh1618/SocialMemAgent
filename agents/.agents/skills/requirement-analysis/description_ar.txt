يوفر سير عمل تحليل المتطلبات المنهجي لفهم المتطلبات واستكشاف قاعدة التعليمات البرمجية وتوضيح المشكلات وإجراء تحليل متعمق وتقديم خطة التنفيذ قبل البرمجة. مناسب لتطوير الميزات وتصميم واجهة برمجة التطبيقات وتصميم قاعدة البيانات وتطوير الوحدات وتعديلات الميزات. يتم تشغيل هذه المهارة تلقائيًا عندما يطلب المستخدمون تطوير الميزات، وتصميم واجهة برمجة التطبيقات، وتصميم قاعدة البيانات، وما إلى ذلك. ويتفاعل وفقًا لإعدادات المستخدم ولغة الإدخال.
