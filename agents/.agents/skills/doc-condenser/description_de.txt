Verwandeln Sie ausführliche technische Dokumentation in prägnante, scanbare Spezifikationen. Nutzen Sie diese Fähigkeit, wenn Sie technische Dokumente, Spezifikationen oder READMEs komprimieren, zusammenfassen oder neu formatieren müssen – auch wenn ein Dokument zu ausführlich ist, wenn Sie eine technische Zusammenfassung wünschen oder wenn Sie mit langen Spezifikationsdokumenten arbeiten. Wird bei Ausdrücken wie „Machen Sie dies prägnant“, „zu ausführlich“, „Komprimieren Sie dies“, „technische Zusammenfassung“, „Entfernen Sie den Flaum“ oder „Formatieren Sie diese Spezifikation neu“ ausgelöst. Die Standardausgabestruktur finden Sie unter „assets/template.md“.
