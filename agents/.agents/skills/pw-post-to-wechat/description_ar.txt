أداة آلية لنشر المحتوى على حسابات WeChat الرسمية.

الميزات الأساسية:
- مشاركة نصية للصورة: صور متعددة مع نص، يدعم ما يصل إلى 9 صور
- منشور المقالة: تنسيق Markdown الكامل، والحفاظ على الأنماط والتخطيط
- سير العمل الآلي: يستخدم Chrome CDP لتسجيل الدخول وملء النماذج والنشر
- الضغط الذكي: يقوم بضغط العنوان والمحتوى تلقائيًا عند تجاوز الحدود
- دعم السمات: تدعم منشورات المقالات السمات الافتراضية والنعمة والبسيطة

متى تستخدم:
- يقول المستخدم صراحةً "النشر إلى حساب WeChat الرسمي" أو "الإرسال إلى الحساب الرسمي"
- يقدم المستخدم ملف Markdown أو محتوى يحتاج إلى النشر
- يحتاج المستخدم إلى نشر محتوى نصي للصور دفعة واحدة

لا ينطبق:
- يسأل المستخدم فقط عن كيفية النشر (فقط قدم النصيحة)
- يحتاج المستخدم إلى تعديل مقالة الحساب الرسمي الموجودة (التشغيل اليدوي)
- يحتاج المستخدم إلى النشر على منصات أخرى (استخدم أدوات النظام الأساسي المقابلة)
