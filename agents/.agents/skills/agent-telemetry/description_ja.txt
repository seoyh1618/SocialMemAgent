構造化されたログとテレメトリを公開することで、コーディング エージェントがアプリケーションの動作を可視化します。 「テレメトリの追加」、「エージェントがログにアクセスできるようにする」、「可観測性を追加する」、「ログを使用してデバッグする」ように求められた場合、またはエージェントが実行時の動作を理解する必要があるが、ログをクエリする方法がない場合に使用します。また、構造化されたログがないためにデバッグが難しい場合、エージェントのドキュメント (CLAUDE.md、AGENTS.md) にアプリケーション ログをクエリするための指示がない場合、または新規または既存の Web アプリケーションのログ インフラストラクチャを設定する場合にも使用します。
