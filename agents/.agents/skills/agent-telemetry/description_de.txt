Machen Sie das Anwendungsverhalten für Programmierer sichtbar, indem Sie strukturierte Protokolle und Telemetriedaten verfügbar machen. Verwenden Sie diese Option, wenn Sie aufgefordert werden, „Telemetrie hinzuzufügen“, „Protokolle für Agenten zugänglich zu machen“, „Beobachtbarkeit hinzuzufügen“, „mit Protokollen zu debuggen“ oder wenn ein Agent das Laufzeitverhalten verstehen muss, aber keine Möglichkeit hat, Protokolle abzufragen. Wird auch verwendet, wenn das Debuggen schwierig ist, weil keine strukturierten Protokolle vorhanden sind, wenn in Agentendokumenten (CLAUDE.md, AGENTS.md) Anweisungen zum Abfragen von Anwendungsprotokollen fehlen oder wenn eine Protokollierungsinfrastruktur für eine neue oder vorhandene Webanwendung eingerichtet wird.
