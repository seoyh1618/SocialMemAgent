Rendez le comportement des applications visible aux agents de codage en exposant les journaux structurés et la télémétrie. À utiliser lorsqu'on lui demande « d'ajouter de la télémétrie », « de rendre les journaux accessibles aux agents », « d'ajouter de l'observabilité », de « déboguer avec les journaux » ou lorsqu'un agent a besoin de comprendre le comportement d'exécution mais n'a aucun moyen d'interroger les journaux. À utiliser également lorsque le débogage est difficile car il n'y a pas de journaux structurés, lorsque les documents d'agent (CLAUDE.md, AGENTS.md) manquent d'instructions pour interroger les journaux d'application, ou lors de la configuration de l'infrastructure de journalisation pour une application Web nouvelle ou existante.
