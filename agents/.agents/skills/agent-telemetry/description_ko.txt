구조화된 로그와 원격 측정을 노출하여 코딩 에이전트에 애플리케이션 동작이 표시되도록 합니다. "원격 측정 추가", "에이전트가 로그에 액세스할 수 있도록 설정", "관찰 가능성 추가", "로그로 디버그"하라는 요청을 받거나 에이전트가 런타임 동작을 이해해야 하지만 로그를 쿼리할 방법이 없는 경우에 사용하세요. 또한 구조화된 로그가 없어 디버깅이 어려운 경우, 에이전트 문서(CLAUDE.md, AGENTS.md)에 애플리케이션 로그 쿼리에 대한 지침이 부족한 경우, 신규 또는 기존 웹 애플리케이션에 대한 로깅 인프라를 설정할 때 사용합니다.
