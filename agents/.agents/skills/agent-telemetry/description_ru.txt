Сделайте поведение приложения видимым для агентов кодирования, предоставляя структурированные журналы и данные телеметрии. Используйте, когда вас просят «добавить телеметрию», «сделать журналы доступными для агентов», «добавить возможность наблюдения», «отладить с помощью журналов» или когда агенту необходимо понять поведение во время выполнения, но у него нет возможности запрашивать журналы. Также используйте, когда отладка затруднена из-за отсутствия структурированных журналов, когда в документах агента (CLAUDE.md, AGENTS.md) отсутствуют инструкции по запросу журналов приложений или при настройке инфраструктуры журналирования для нового или существующего веб-приложения.
