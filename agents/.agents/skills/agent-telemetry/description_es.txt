Haga que el comportamiento de la aplicación sea visible para los agentes de codificación exponiendo registros estructurados y telemetría. Úselo cuando se le solicite "agregar telemetría", "hacer que los registros sean accesibles para los agentes", "agregar observabilidad", "depurar con registros" o cuando un agente necesita comprender el comportamiento del tiempo de ejecución pero no tiene forma de consultar los registros. Utilícelo también cuando la depuración sea difícil porque no hay registros estructurados, cuando los documentos del agente (CLAUDE.md, AGENTS.md) carezcan de instrucciones para consultar los registros de la aplicación o cuando se configure la infraestructura de registro para una aplicación web nueva o existente.
