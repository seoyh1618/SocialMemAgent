Rendi visibile il comportamento dell'applicazione agli agenti di codifica esponendo log strutturati e telemetria. Da utilizzare quando viene richiesto di "aggiungere telemetria", "rendere i log accessibili agli agenti", "aggiungere osservabilità", "debug con log" o quando un agente deve comprendere il comportamento di runtime ma non ha modo di eseguire query sui log. Da utilizzare anche quando il debugging è difficile perché non sono presenti log strutturati, quando i documenti dell'agente (CLAUDE.md, AGENTS.md) mancano di istruzioni per interrogare i log dell'applicazione o quando si configura l'infrastruttura di registrazione per un'applicazione web nuova o esistente.
