Prüfen Sie Django-Codebasen auf Sicherheit, Leistung, Korrektheit und Architektur-Antimuster. Führen Sie Systemprüfungen, Migrationsabweichungsprüfungen und statische Regelscans durch und geben Sie dann einen Integritätswert von 0 bis 100 mit umsetzbaren Korrekturen aus. Wird verwendet, wenn Benutzer ein Django-Backend scannen, eine Django-Integritätsprüfung durchführen, die Qualität des Backend-Codes überprüfen oder eine Prüfung vor der Bereitstellung durchführen möchten.
