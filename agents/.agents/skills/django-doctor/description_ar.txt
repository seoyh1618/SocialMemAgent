تدقيق قواعد بيانات Django للأمان والأداء والصحة والأنماط المضادة للهندسة المعمارية. قم بإجراء فحوصات النظام، وفحوصات انحراف الترحيل، وفحص القواعد الثابتة، ثم قم بإخراج درجة صحة من 0 إلى 100 مع إصلاحات قابلة للتنفيذ. يُستخدم عندما يطلب المستخدمون فحص واجهة Django الخلفية، أو تشغيل فحص سلامة Django، أو مراجعة جودة كود الواجهة الخلفية، أو إجراء تدقيق ما قبل النشر.
