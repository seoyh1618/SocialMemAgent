Auditez les bases de code Django pour vérifier la sécurité, les performances, l’exactitude et les anti-modèles d’architecture. Exécutez des vérifications du système, des contrôles de dérive de migration et des analyses de règles statiques, puis générez un score d'intégrité de 0 à 100 avec des correctifs exploitables. À utiliser lorsque les utilisateurs demandent à analyser un backend Django, à exécuter une vérification de l'état de Django, à examiner la qualité du code backend ou à effectuer un audit de pré-déploiement.
