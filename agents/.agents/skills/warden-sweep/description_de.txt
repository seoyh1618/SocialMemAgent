Vollständiger Repository-Code-Sweep. Scannt jede Datei mit dem Aufseher, verifiziert die Ergebnisse durch Deep Tracing und erstellt PR-Entwürfe für validierte Probleme. Verwenden Sie diese Option, wenn Sie aufgefordert werden, „das Repository zu durchsuchen“, „alles zu scannen“, „alle Fehler zu finden“, „vollständige Überprüfung der Codebasis“, „Batch-Code-Analyse“ durchzuführen oder „Warden“ über das gesamte Repository auszuführen.
