Competenza di configurazione del frontend React SPA (integrazione PocketBase). Vite + TanStack Router (routing basato su file) + TanStack Query + Tailwind CSS / Shadcn UI + Biome. Da utilizzare durante la creazione di un frontend React, una configurazione SPA, un'interfaccia utente PocketBase, l'inizializzazione di un progetto frontend, un'app React o la creazione di dashboard.
