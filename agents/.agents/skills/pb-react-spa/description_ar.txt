مهارة إعداد الواجهة الأمامية لـ SPA (تكامل PocketBase). Vite + TanStack Router (التوجيه القائم على الملفات) + TanStack Query + Tailwind CSS / Shadcn UI + Biome. يُستخدم عند إنشاء واجهة React الأمامية، أو إعداد SPA، أو PocketBase UI، أو تهيئة مشروع الواجهة الأمامية، أو تطبيق React، أو إنشاء لوحة المعلومات.
