Compétence de configuration du frontend React SPA (intégration PocketBase). Vite + TanStack Router (routage basé sur des fichiers) + TanStack Query + Tailwind CSS / Shadcn UI + Biome. À utiliser lors de la création d'une interface React, d'une configuration SPA, d'une interface utilisateur PocketBase, de l'initialisation d'un projet frontal, d'une application React ou de la création d'un tableau de bord.
