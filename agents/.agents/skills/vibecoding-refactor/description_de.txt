Vibe Coding Engineering Refactoring-Methodik. Refaktorieren Sie KI-generierten Code systematisch durch eine Tiefenanalyse auf Architektur- und Modulebene und entdecken Sie doppelte Implementierungen, ungenutzte Ressourcen, inkonsistente Muster und andere typische Probleme. Unterstützt die dauerhafte Aufgabenverwaltung für die inkrementelle Ausführung. Verwenden Sie es, wenn Sie aufgefordert werden, Code umzugestalten, Patches zu bereinigen, die Codequalität zu verbessern, die Architektur zu analysieren oder die Codebasis zu optimieren.
