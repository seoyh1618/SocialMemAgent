Développement d'un plugin Java pour les serveurs Hytale. Couvre la configuration de Java 25, les builds Gradle, la configuration Antigravity/IDE, le cycle de vie du plugin (installation/démarrage/arrêt), l'enregistrement des commandes, la gestion des événements, l'architecture ECS et le déploiement. À utiliser lors de la création de plugins de serveur, de l'extension des fonctionnalités du jeu ou de la mise en œuvre de mécanismes de jeu personnalisés.
