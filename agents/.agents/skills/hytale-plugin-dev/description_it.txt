Sviluppo plugin Java per server Hytale. Copre la configurazione di Java 25, le build di Gradle, la configurazione di Antigravity/IDE, il ciclo di vita del plug-in (configurazione/avvio/spegnimento), la registrazione dei comandi, la gestione degli eventi, l'architettura ECS e la distribuzione. Da utilizzare durante la creazione di plug-in del server, l'estensione delle funzionalità di gioco o l'implementazione di meccaniche di gioco personalizzate.
