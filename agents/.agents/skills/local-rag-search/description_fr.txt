Effectuez efficacement des recherches sur le Web à l'aide du serveur mcp-local-rag avec classement de similarité sémantique. Utilisez cette compétence lorsque vous devez rechercher sur le Web des informations actuelles, rechercher des sujets dans plusieurs sources ou recueillir du contexte à partir d'Internet sans utiliser d'API externes. Cette compétence enseigne l'utilisation efficace de la recherche Web basée sur RAG avec DuckDuckGo, Google et les capacités de recherche approfondie multimoteurs.
