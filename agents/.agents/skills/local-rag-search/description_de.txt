Führen Sie Websuchen effizient durch, indem Sie den mcp-local-rag-Server mit semantischem Ähnlichkeitsranking verwenden. Nutzen Sie diese Fähigkeit, wenn Sie das Web nach aktuellen Informationen durchsuchen, Themen aus mehreren Quellen recherchieren oder Kontext aus dem Internet sammeln müssen, ohne externe APIs zu verwenden.
