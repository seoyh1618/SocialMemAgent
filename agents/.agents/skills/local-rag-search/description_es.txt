Realice búsquedas web de manera eficiente utilizando el servidor mcp-local-rag con clasificación de similitud semántica. Utilice esta habilidad cuando necesite buscar en la web información actual, investigar temas en múltiples fuentes o recopilar contexto de Internet sin utilizar API externas. Esta habilidad enseña el uso eficaz de la búsqueda web basada en RAG con DuckDuckGo, Google y capacidades de investigación profunda de múltiples motores.
