Esegui in modo efficiente ricerche sul Web utilizzando il server mcp-local-rag con classificazione della somiglianza semantica. Utilizza questa competenza quando devi cercare sul Web informazioni attuali, ricercare argomenti da più fonti o raccogliere contesto da Internet senza utilizzare API esterne. Questa competenza insegna l'uso efficace della ricerca web basata su RAG con DuckDuckGo, Google e funzionalità di ricerca approfondita multi-motore.
