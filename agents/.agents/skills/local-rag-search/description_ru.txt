Эффективно выполняйте поиск в Интернете с помощью сервера mcp-local-rag с рейтингом семантического сходства. Используйте этот навык, когда вам нужно искать в Интернете актуальную информацию, исследовать темы в нескольких источниках или собирать контекст из Интернета без использования внешних API. Этот навык учит эффективному использованию веб-поиска на основе RAG с DuckDuckGo, Google и возможностями глубокого многоядерного исследования.
