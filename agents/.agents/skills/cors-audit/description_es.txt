Esta habilidad realiza una auditoría CORS (intercambio de recursos entre orígenes) integral en proyectos web. Debe usarse al diagnosticar errores de CORS, configurar CORS para nuevos proyectos, revisar la configuración de CORS después de problemas de implementación o validar que CORS se maneja correctamente en todas las capas (puerta de enlace, backend, frontend). Cubre configuraciones estándar de frontend-backend, arquitecturas de microaplicaciones (Qiankun, single-spa) y escenarios de origen dinámico de múltiples orígenes.
