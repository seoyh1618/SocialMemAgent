Cette compétence effectue un audit CORS (Cross-Origin Resource Sharing) complet sur les projets Web. Il doit être utilisé lors du diagnostic des erreurs CORS, de la configuration de CORS pour de nouveaux projets, de l'examen de la configuration CORS après des problèmes de déploiement ou de la validation que CORS est géré correctement sur toutes les couches (passerelle, backend, frontend). Couvre les configurations frontend-backend standard, les architectures de micro-applications (Qiankun, spa unique) et les scénarios d'origine dynamique multi-origines.
