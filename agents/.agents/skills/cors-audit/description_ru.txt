Этот навык выполняет комплексный аудит CORS (совместное использование ресурсов между источниками) в веб-проектах. Его следует использовать при диагностике ошибок CORS, настройке CORS для новых проектов, проверке конфигурации CORS после проблем с развертыванием или проверке правильности обработки CORS на всех уровнях (шлюз, серверная часть, внешний интерфейс). Охватывает стандартные настройки внешнего и внутреннего интерфейса, архитектуры микроприложений (Qiankun, single-spa) и сценарии динамического происхождения с несколькими источниками.
