معرفة حديثة بـ Microsoft Graph API لوكلاء الذكاء الاصطناعي. ابحث على الفور في 27,700 واجهة برمجة تطبيقات للرسم البياني وعينات مجتمعية منسقة - كل ذلك محليًا، دون الحاجة إلى مكالمات الشبكة. سد الفجوة بين فترات التدريب على LLM وواجهة برمجة تطبيقات Microsoft Graph التي يتم تحديثها أسبوعيًا. يعمل جنبًا إلى جنب مع خوادم Graph MCP (مثل lokka.dev) للتنفيذ، أو يصادق ويستدعي واجهة برمجة تطبيقات Microsoft Graph مباشرة عند الحاجة.
