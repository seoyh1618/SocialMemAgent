Актуальные знания API Microsoft Graph для агентов ИИ. Мгновенный поиск по 27 700 API-интерфейсов Graph и образцам, выбранным сообществом — все локально, без необходимости сетевых вызовов. Устраняет разрыв между ограничениями на обучение LLM и еженедельно обновляемым API Microsoft Graph. Работает вместе с серверами Graph MCP (например, lokka.dev) для выполнения или проверяет подлинность и вызывает API Microsoft Graph напрямую, когда это необходимо.
