Überwachen Sie installierte Fähigkeiten auf Projekt-, globaler und Plugin-Ebene. Listet Fähigkeiten mit Zeilenanzahl auf und identifiziert Verbesserungsmöglichkeiten (Prägnanz, Klarheit, Überschneidung, symbolische Verschwendung). Verwenden Sie diese Funktion, wenn Sie die Qualität von Fertigkeiten überprüfen, aufgeblähte Fertigkeiten finden oder Token-Budgets optimieren möchten.
