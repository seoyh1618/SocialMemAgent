Crea e crea presentazioni di diapositive utilizzando il framework PromptSlide (Vite + React 19 + Tailwind v4 + Framer Motion). Utilizzare quando l'utente desidera creare una nuova raccolta di diapositive, aggiungere o modificare diapositive, personalizzare temi o branding, lavorare con animazioni e transizioni di diapositive, acquisire diapositive come immagini o verificare visivamente l'aspetto delle diapositive. Si attiva quando vengono menzionate diapositive, presentazioni, presentazioni, PromptSlide, schermate di diapositive o attività correlate alle diapositive.
