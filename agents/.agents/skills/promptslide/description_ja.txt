PromptSlide フレームワーク (Vite + React 19 + Tailwind v4 + Framer Motion) を使用して、スライド デッキ プレゼンテーションを作成および作成します。ユーザーが新しいスライド デッキの作成、スライドの追加または編集、テーマやブランディングのカスタマイズ、スライド アニメーションとトランジションの操作、スライドを画像としてキャプチャするか、スライドの外観を視覚的に確認する場合に使用します。スライド、デッキ、プレゼンテーション、PromptSlide、スライドのスクリーンショット、またはスライド関連のタスクについてのメンションがトリガーされます。
