إنشاء عروض تقديمية لمجموعة الشرائح وتأليفها باستخدام إطار عمل PromptSlide (Vite + React 19 + Tailwind v4 + Framer Motion). يُستخدم عندما يريد المستخدم إنشاء مجموعة شرائح جديدة، أو إضافة شرائح أو تحريرها، أو تخصيص السمات أو العلامة التجارية، أو العمل مع الرسوم المتحركة للشرائح والانتقالات، أو التقاط الشرائح كصور، أو التحقق من مظهر الشريحة بصريًا. يتم تشغيله عند الإشارة إلى الشرائح أو المجموعات أو العروض التقديمية أو PromptSlide أو لقطات شاشة الشرائح أو المهام المتعلقة بالشرائح.
