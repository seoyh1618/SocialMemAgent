Оптимизация производительности Vue 3 и рекомендации по передовому опыту для современных интерфейсных приложений. Этот навык следует использовать при написании, проверке или рефакторинге кода Vue 3, чтобы обеспечить оптимальные шаблоны производительности, правильное использование Composition API и современные методы разработки. Запускается по задачам, включающим компоненты Vue 3, Composition API, реактивность, управление состоянием или оптимизацию производительности.
