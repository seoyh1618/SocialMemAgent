Vue 3-Leistungsoptimierungs- und Best-Practice-Richtlinien für moderne Frontend-Anwendungen. Diese Fähigkeit sollte beim Schreiben, Überprüfen oder Refactoring von Vue 3-Code verwendet werden, um optimale Leistungsmuster, die ordnungsgemäße Verwendung der Composition API und moderne Entwicklungspraktiken sicherzustellen. Wird bei Aufgaben ausgelöst, die Vue 3-Komponenten, Composition API, Reaktivität, Statusverwaltung oder Leistungsoptimierung betreffen.
