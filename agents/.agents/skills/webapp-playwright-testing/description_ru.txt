Набор инструментов для автоматизации браузера с использованием Playwright MCP для тестирования веб-приложений. Используйте, когда вас просят перемещаться по страницам, щелкать элементы, заполнять формы, делать снимки экрана, проверять компоненты пользовательского интерфейса, проверять журналы консоли, отлаживать проблемы внешнего интерфейса или проверять адаптивный дизайн. Поддерживает интерактивное взаимодействие с браузером и снимки доступности.
