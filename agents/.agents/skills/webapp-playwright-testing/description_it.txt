Toolkit di automazione del browser che utilizza Playwright MCP per testare le applicazioni web. Da utilizzare quando viene richiesto di navigare nelle pagine, fare clic sugli elementi, compilare moduli, acquisire screenshot, verificare i componenti dell'interfaccia utente, controllare i log della console, eseguire il debug di problemi di frontend o convalidare il responsive design. Supporta l'interazione live del browser e le istantanee di accessibilità.
