اقرأ هذه المهارة عند تصميم أو تخطيط أي بنية لنظام ألعاب - بما في ذلك أنظمة القتال أو المهارات أو الذكاء الاصطناعي أو واجهة المستخدم أو تعدد اللاعبين أو السرد أو المشهد. يحتوي على أدلة اختيار النموذج (DDD / المستندة إلى البيانات / النموذج الأولي)، ومراجع التصميم الخاصة بالنظام، واستراتيجيات المزج. يعمل كمكون إضافي للمعرفة بالمجال جنبًا إلى جنب مع مهارات سير العمل (OpenSpec، SpecKit) أو وضع التخطيط للوكيل.
