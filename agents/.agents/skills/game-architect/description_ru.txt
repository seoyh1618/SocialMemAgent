ПРОЧИТАЙТЕ этот навык при проектировании или планировании любой архитектуры игровой системы, включая системы боя, навыков, искусственного интеллекта, пользовательского интерфейса, многопользовательской игры, повествования или сцен. Содержит руководства по выбору парадигмы (DDD / Data-Driven / Prototype), рекомендации по проектированию для конкретной системы и стратегии смешивания. Работает как плагин знаний предметной области вместе с навыками рабочего процесса (OpenSpec, SpecKit) или режимом планирования агента.
