LEA esta habilidad cuando diseñe o planifique cualquier arquitectura de sistema de juego, incluidos sistemas de combate, habilidades, IA, interfaz de usuario, multijugador, narración o escena. Contiene guías de selección de paradigmas (DDD/Data-Driven/Prototype), referencias de diseño específicas del sistema y estrategias de combinación. Funciona como un complemento de conocimiento de dominio junto con las habilidades de flujo de trabajo (OpenSpec, SpecKit) o ​​el modo de plan de un agente.
