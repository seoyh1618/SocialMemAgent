Automatisiert die Erstellung von Komponententests für C++-Projekte mithilfe des GoogleTest-Frameworks (GTest) mit konsistenten Softwaretestmustern, einschließlich In-Got-Want-, tabellengesteuerten Tests und AAA-Mustern. Verwenden Sie diese Option, wenn Sie Komponententests erstellen, ändern oder überprüfen oder wenn der Benutzer Komponententests, Testabdeckung oder GTest erwähnt.
