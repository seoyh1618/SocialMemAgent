Automatiza la creación de pruebas unitarias para proyectos de C++ utilizando el marco GoogleTest (GTest) con patrones de prueba de software consistentes, incluidos los patrones In-Got-Want, Table-Driven Testing y AAA. Úselo al crear, modificar o revisar pruebas unitarias, o cuando el usuario mencione pruebas unitarias, cobertura de pruebas o GTest.
