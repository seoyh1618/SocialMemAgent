Intelligent media optimization with automated workflows for images, videos, and SVGs.包括检测 LCP 图像（触发格式/延迟加载/优先级分析）、识别布局转换风险（缺失尺寸）以及标记延迟加载问题（上方折叠延迟或下方急切）的决策树。具有用于完整媒体审核、LCP 图像调查、视频性能（海报优化）和 SVG 嵌入位图检测的工作流程。与 Core Web Vitals（LCP/CLS 影响）和加载（优先级提示、资源预加载）的跨技能集成。根据内容类型提供性能预算和格式建议。当用户询问图像优化、LCP 是图像/视频、媒体布局转变或媒体加载策略时使用。与 Chrome DevTools MCP 兼容。
