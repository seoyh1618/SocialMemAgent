تحسين ذكي للوسائط من خلال سير العمل الآلي للصور ومقاطع الفيديو وملفات SVG. يتضمن أشجار القرار التي تكتشف صور LCP (تنسيق المشغلات/التحميل البطيء/تحليل الأولويات)، وتحديد مخاطر تغيير التخطيط (الأبعاد المفقودة)، ووضع علامة على مشكلات التحميل البطيء (البطء فوق الطي البطيء أو التحمس السفلي الطي). يتميز بسير العمل لتدقيق الوسائط الكامل، والتحقيق في صور LCP، وأداء الفيديو (تحسين الملصق)، واكتشاف الصور النقطية المضمنة بتنسيق SVG. التكامل عبر المهارات مع Core Web Vitals (تأثير LCP/CLS) والتحميل (تلميحات الأولوية، والتحميل المسبق للموارد). يوفر ميزانيات الأداء وتوصيات التنسيق بناءً على نوع المحتوى. يُستخدم عندما يسأل المستخدم عن تحسين الصورة، LCP عبارة عن صورة/فيديو، أو تحولات التخطيط من الوسائط، أو استراتيجية تحميل الوسائط. متوافق مع Chrome DevTools MCP.
