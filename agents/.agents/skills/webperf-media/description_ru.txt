Интеллектуальная оптимизация мультимедиа с автоматизированными рабочими процессами для изображений, видео и SVG. Включает деревья решений, которые обнаруживают изображения LCP (триггеры формата/ленивой загрузки/анализа приоритетов), выявляют риски смещения макета (отсутствующие измерения) и отмечают проблемы с ленивой загрузкой (ленивый верхний сгиб или активный нижний сгиб). Содержит рабочие процессы для полного аудита мультимедиа, исследования изображений LCP, производительности видео (оптимизация плакатов) и обнаружения встроенных растровых изображений SVG. Межквалифицированная интеграция с Core Web Vitals (влияние LCP/CLS) и загрузкой (подсказки по приоритетам, предварительная загрузка ресурсов). Предоставляет бюджеты производительности и рекомендации по формату в зависимости от типа контента. Используйте, когда пользователь спрашивает об оптимизации изображения, LCP — это изображение/видео, изменения макета от медиа или стратегия загрузки мультимедиа. Совместим с Chrome DevTools MCP.
