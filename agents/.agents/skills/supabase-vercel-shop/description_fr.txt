Créez une plateforme e-commerce complète Supabase + Vercel avec CMS et panneau d'administration. À utiliser pour la configuration d'une boutique, la création d'une boutique en ligne, l'intégration de Stripe, un catalogue de produits, un tableau de bord d'administration ou des projets CMS. Comprend un schéma de base de données, l'authentification RBAC, la gestion de contenu, le flux de paiement et une architecture sans codage en dur avec tests obligatoires.
