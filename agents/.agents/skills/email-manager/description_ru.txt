Единое управление несколькими электронными письмами с помощью интеллектуального помощника. Поддерживает Gmail, QQ Mail и другие учетные записи IMAP; проверяет электронную почту по расписанию; использует ИИ для создания резюме и черновиков ответов; требует подтверждения пользователя перед отправкой.
