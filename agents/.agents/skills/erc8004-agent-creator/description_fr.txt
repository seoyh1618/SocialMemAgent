Utilisation et installation de create-8004-agent (échafaudage d'agent ERC-8004/8004 AI). À utiliser lorsque l'utilisateur demande comment créer un agent ERC-8004 ou 8004, comment installer ou exécuter create-8004-agent, ce que signifient les options de l'assistant, quelles chaînes sont prises en charge, ce qui est généré ou comment configurer et enregistrer le projet généré (env, npm run register, start:a2a, start:mcp). Couvre EVM (Ethereum, Base, Polygon, Monad) et Solana.
