Habilidades para la realización de revisiones documentales. Se utiliza al revisar documentos como propuestas, documentos de diseño, PRD y especificaciones técnicas. Activado por menciones de tareas de revisión de documentos como "revisar el documento", "verificar la propuesta", "verificar el documento de diseño", "revisar este documento" o "evaluar la calidad del documento". Evalúa desde múltiples ángulos (perspectiva empresarial/estratégica, estructura lógica, validez técnica, gestión de riesgos y precisión de expresión) y proporciona sugerencias de mejora constructivas.
