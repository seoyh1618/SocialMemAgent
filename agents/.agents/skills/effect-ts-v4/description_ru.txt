Комплексная документация по эффекту 4.0 (TypeScript) для агентов. Используйте при написании, проверке или рефакторинге кода, который импортируется из «эффекта». Охватывает тип эффекта, сервисы/уровни, схему, поток, параллелизм, транзакции и нестабильную экосистему (HTTP, RPC, SQL, CLI, AI, кластер). Запускается по: effect typescript, effect 4, effect-ts, Effect.gen, ServiceMap, Layer, Schema.
