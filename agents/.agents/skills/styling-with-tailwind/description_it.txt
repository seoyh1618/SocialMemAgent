Crea interfacce utente utilizzando classi di utilità CSS Tailwind e modelli shadcn/ui. Copre le variabili CSS con colori OKLCH, varianti di componenti con CVA, design reattivo, modalità oscura e funzionalità Tailwind v4.2. Supporta le primitive Radix UI e Base UI, CLI 3.0 e stili visivi. Da utilizzare durante la creazione di interfacce con Tailwind, lo styling di componenti shadcn/ui, l'implementazione di temi o l'utilizzo di CSS basati sull'utilità.
