ينشئ واجهات مستخدم باستخدام فئات الأداة المساعدة Tailwind CSS وأنماط shadcn/ui. يغطي متغيرات CSS بألوان OKLCH، ومتغيرات المكونات مع CVA، والتصميم سريع الاستجابة، والوضع الداكن، وميزات Tailwind v4.2. يدعم أساسيات Radix UI وBase UI وCLI 3.0 والأنماط المرئية. يُستخدم عند إنشاء واجهات باستخدام Tailwind، أو تصميم مكونات shadcn/ui، أو تنفيذ السمات، أو العمل باستخدام الأداة المساعدة CSS أولاً.
