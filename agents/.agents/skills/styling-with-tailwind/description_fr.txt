Crée des interfaces utilisateur à l'aide des classes utilitaires CSS Tailwind et des modèles shadcn/ui. Couvre les variables CSS avec les couleurs OKLCH, les variantes de composants avec CVA, la conception réactive, le mode sombre et les fonctionnalités Tailwind v4.2. Prend en charge les primitives Radix UI et Base UI, CLI 3.0 et les styles visuels. À utiliser lors de la création d'interfaces avec Tailwind, du style des composants shadcn/ui, de l'implémentation de thèmes ou de l'utilisation de CSS axés sur les utilitaires.
