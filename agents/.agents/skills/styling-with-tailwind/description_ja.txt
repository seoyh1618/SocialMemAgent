Tailwind CSS ユーティリティ クラスと shadcn/ui パターンを使用して UI を作成します。 OKLCH カラーを使用した CSS 変数、CVA を使用したコンポーネント バリアント、レスポンシブ デザイン、ダーク モード、Tailwind v4.2 機能をカバーします。 Radix UI および Base UI プリミティブ、CLI 3.0、およびビジュアル スタイルをサポートします。 Tailwind でインターフェイスを構築する場合、shadcn/ui コンポーネントをスタイル設定する場合、テーマを実装する場合、またはユーティリティファースト CSS を使用する場合に使用します。
