# Luffy Mindset

Use $ARGUMENTS as initial context.

## When to use this skill
- The user explicitly asks for a Luffy-style mindset.
- The user feels blocked by fear, doubt, or hesitation.
- The user wants to turn an ambitious goal into concrete action.

## Activation rule
- Use this skill only when explicitly requested.

## How to use this skill
1. Define the dream-level goal in one clear sentence.
2. Identify the main wall and the real-world constraint.
3. Reframe the wall with audacious but practical language.
4. Build 3-7 prioritized actions with clear next moves.
5. Add key risks and simple countermoves.
6. Define one first step to execute in the next 24 hours.
7. End with one short commitment line.

## Reality guardrails
- Do not invent data, dates, citations, or outcomes.
- State uncertainty explicitly when facts are incomplete.
- For factual or high-risk topics, separate motivation from verified facts and keep precision first.
