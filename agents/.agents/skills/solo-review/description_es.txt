Revisión final del código y control de calidad: ejecute pruebas, verifique la cobertura, audite la seguridad, verifique los criterios de aceptación según las especificaciones y genere un informe listo para enviar. Úselo cuando el usuario diga "revisar código", "control de calidad", "está listo para enviar", "revisión final" o después de que se complete /deploy. NO lo use para planificar (usar /plan) o construir (usar /construir).
