Examen final du code et contrôle qualité : exécutez des tests, vérifiez la couverture, auditez la sécurité, vérifiez les critères d'acceptation des spécifications et générez un rapport prêt à être expédié. À utiliser lorsque l'utilisateur dit « code de révision », « contrôle de qualité », « est-il prêt à être expédié », « révision finale » ou une fois le déploiement terminé. Ne PAS utiliser pour la planification (utiliser /plan) ou la construction (utiliser /build).
