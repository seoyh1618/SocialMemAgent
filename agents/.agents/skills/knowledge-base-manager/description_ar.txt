اختيار بنية قاعدة المعرفة وتنظيمها وإدارتها. يُستخدم عند الاختيار بين بنيات قاعدة المعارف المستندة إلى المستند (RAG) أو المستندة إلى الكيان (الرسم البياني) أو قاعدة المعارف المختلطة، أو إنشاء سير عمل تنظيم المحتوى، أو تنفيذ الإصدارات والحوكمة، أو تقييم مقاييس الجودة. لبناء خطوط أنابيب الاسترجاع، استخدم مهارة تنفيذ القماش. لبناء الرسوم البيانية المعرفية، استخدم مهارة بناء الرسم البياني المعرفي.
