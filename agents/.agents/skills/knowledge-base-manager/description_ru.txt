Выбор, курирование и управление архитектуры базы знаний. Используйте при выборе между архитектурой на основе документов (RAG), сущностей (граф) или гибридной архитектурой базы знаний, при создании рабочих процессов управления контентом, реализации управления версиями и управлением или оценке показателей качества. Для построения конвейеров поиска используйте навык rag-implementer. Для построения графов знаний используйте навык построения графов знаний.
