Auswahl, Kuration und Governance der Wissensdatenbankarchitektur. Verwenden Sie diese Option, wenn Sie zwischen dokumentbasierten (RAG), entitätsbasierten (Graph) oder hybriden KB-Architekturen wählen, Arbeitsabläufe zur Inhaltskuration einrichten, Versionierung und Governance implementieren oder Qualitätsmetriken bewerten. Verwenden Sie zum Erstellen von Abrufpipelines die Rag-Implementer-Fähigkeit. Verwenden Sie zum Erstellen von Wissensgraphen die Fähigkeit „Knowledge-Graph-Builder“.
