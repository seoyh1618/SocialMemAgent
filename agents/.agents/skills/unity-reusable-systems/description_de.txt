Verwenden Sie es, wenn Sie wiederverwendbare Unity-Spielsysteme erstellen, UPM-Pakete erstellen oder eine ScriptableObject-basierte modulare Architektur für Gameplay-Systeme wie Inventar, Kampf, Dialog, Quests oder Speichern/Laden entwerfen. Wird auch verwendet, wenn mehrere unabhängige Unity-Pakete verbunden werden, die ohne direkte Abhängigkeiten kommunizieren müssen.
