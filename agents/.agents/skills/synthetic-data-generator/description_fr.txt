Compétence en génération de données synthétiques pour les tests. Génère des données utilisateur, des transactions, des journaux, des réponses API réalistes, etc. Prend en charge la conformité au RGPD, l'intégration de contrefaçons, la génération basée sur un schéma, les données de séries chronologiques et la génération de données d'anomalies.
