Animations interactives basées sur l'état avec machines à états intégrées. À utiliser lorsque les animations doivent RÉAGIR aux entrées de l'utilisateur (survoler, cliquer, faire glisser), avoir plusieurs états/transitions ou répondre aux valeurs des données (barres de progression, compteurs). Idéal pour les boutons animés, les bascules, les cases à cocher, les personnages. Pour des animations de lecture/boucle simples, utilisez plutôt Lottie.
