Управление воспроизведением музыки Spotify + база данных 5947 жанров. Поиск, воспроизведение, пауза, управление очередью, рекомендации на основе сцен, поиск и открытие жанров.

ИСПОЛЬЗУЙТЕ ЭТОТ НАВЫК, когда пользователь упоминает:
- Воспроизведение: «включи музыку», «включи песню», «включи музыку», «играй», «я хочу услышать XX», «немного музыки XX».
- Элементы управления: «пауза», «возобновить», «следующий», «предыдущий», «громкость», «что играет»
- Поиск по жанру: /genre, «поиск музыкального жанра», «рекомендовать типы музыки», «какие поджанры существуют в XX».
- Сцена/настроение: «музыка для позднего вечера», «энергичная», «неземная», «порекомендуйте жанры с ХХ-атмосферой».
- Интеграция Suno: при использовании Suno для создания музыки без указания жанра.
