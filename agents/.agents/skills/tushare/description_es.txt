tushare es un paquete de interfaz de datos financieros que proporciona una rica variedad de contenido (datos de mercado para acciones, fondos, futuros, monedas digitales, etc.) y datos fundamentales como finanzas de empresas y administradores de fondos. El módulo unifica la entrega externa de estos activos de datos a través de API estandarizadas, lo que permite a los usuarios técnicos que los necesitan acceder a la información de una manera más concisa y liviana en tiempo real.
