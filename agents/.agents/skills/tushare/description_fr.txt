tushare est un package d'interface de données financières qui fournit une riche gamme de contenus (données de marché pour les actions, les fonds, les contrats à terme, les monnaies numériques, etc.) et des données fondamentales telles que les données financières des entreprises et les gestionnaires de fonds. Le module unifie la livraison externe de ces actifs de données via des API standardisées, permettant aux utilisateurs techniques qui en ont besoin d'accéder aux informations de manière plus concise et plus légère en temps réel.
