tushare è un pacchetto di interfaccia di dati finanziari che fornisce una ricca gamma di contenuti (dati di mercato per azioni, fondi, futures, valute digitali, ecc.) e dati fondamentali come dati finanziari aziendali e gestori di fondi. Il modulo unifica la distribuzione esterna di queste risorse di dati tramite API standardizzate, consentendo agli utenti tecnici che ne hanno bisogno di accedere alle informazioni in modo più semplice, conciso e in tempo reale.
