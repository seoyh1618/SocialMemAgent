Центральный центр для Claude Agent SDK (SDK TypeScript и Python). Охватывает установку SDK, аутентификацию (Anthropic key, Bedrock, Vertex), сеансы и возобновление, разветвление сеансов, потоковую передачу и одиночный режим, пользовательские инструменты, разрешения (allowedTools, disallowedTools, PermissionMode), интеграцию MCP, системные подсказки (CLAUDE.md, AppendSystemPrompt, OutputStyle), отслеживание затрат, отслеживание задач, структурированные выходные данные, шаблоны хостинга, плагины и рекомендации по брендингу SDK. Помогает создавать собственные агенты, настраивать параметры SDK и устранять проблемы с SDK. Делегирует 100% навыков управления официальной документацией.
