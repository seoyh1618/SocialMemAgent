Zentrale Autorität für das Claude Agent SDK (TypeScript- und Python-SDKs). Behandelt SDK-Installation, Authentifizierung (Anthropic Key, Bedrock, Vertex), Sitzungen und Wiederaufnahme, Forking-Sitzungen, Streaming vs. Single-Modus, benutzerdefinierte Tools, Berechtigungen (allowedTools, disallowedTools, freedomMode), MCP-Integration, Systemeingabeaufforderungen (CLAUDE.md, appendSystemPrompt, OutputStyle), Kostenverfolgung, Aufgabenverfolgung, strukturierte Ausgaben, Hosting-Muster, Plugins und SDK-Branding-Richtlinien. Hilft beim Erstellen benutzerdefinierter Agents, beim Konfigurieren von SDK-Optionen und beim Beheben von SDK-Problemen. Delegiert 100 % der Fähigkeiten zur Dokumentenverwaltung für die offizielle Dokumentation.
