Autorité centrale pour le SDK Claude Agent (SDK TypeScript et Python). Couvre l'installation du SDK, l'authentification (Anthropic key, Bedrock, Vertex), les sessions et la reprise, les sessions forking, le streaming ou le mode unique, les outils personnalisés, les autorisations (allowedTools, disallowedTools, permissionMode), l'intégration MCP, les invites système (CLAUDE.md, appendSystemPrompt, outputStyle), le suivi des coûts, le suivi des tâches, les sorties structurées, les modèles d'hébergement, les plugins et les directives de marque du SDK. Aide à créer des agents personnalisés, à configurer les options du SDK et à résoudre les problèmes liés au SDK. Délégue à 100 % la compétence de gestion des documents pour la documentation officielle.
