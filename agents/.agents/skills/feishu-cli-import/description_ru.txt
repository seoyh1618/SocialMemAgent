Импортируйте и создайте документ Feishu из файла Markdown, поддерживая вложенные списки, автоматическое преобразование диаграмм Mermaid/PlantUML в виртуальные доски и автоматическое разделение больших таблиц. Используется, когда пользователь запрашивает «импортировать Markdown», «создать документ из md» или «загрузить Markdown».
