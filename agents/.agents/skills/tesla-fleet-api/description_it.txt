Da utilizzare durante l'integrazione con l'API Fleet ufficiale di Tesla per leggere i dati del veicolo/dispositivo energetico o emettere comandi remoti (ad esempio avviare il precondizionamento HVAC, riattivare il veicolo, controllare la carica). Copre l'onboarding (registrazione dell'app per sviluppatori, regioni/URL di base), flussi di token OAuth (token di terze parti + partner, rotazione di aggiornamento), hosting di dominio/chiave pubblica richiesto e utilizzo del comando veicolo ufficiale/proxy http tesla di Tesla per i comandi del veicolo firmati.
