يُستخدم عند التكامل مع واجهة برمجة تطبيقات الأسطول الرسمية لشركة Tesla لقراءة بيانات السيارة/جهاز الطاقة أو إصدار أوامر عن بعد (على سبيل المثال، بدء التهيئة المسبقة لنظام التدفئة والتهوية وتكييف الهواء (HVAC)، وتنشيط السيارة، وأدوات التحكم في الشحن). يغطي الإعداد (تسجيل تطبيق المطور، المناطق/عناوين URL الأساسية)، وتدفقات رمز OAuth المميز (الرموز المميزة للطرف الثالث + الشريك، وتناوب التحديث)، واستضافة المجال/المفتاح العام المطلوبة، واستخدام أمر السيارة/وكيل tesla-http الرسمي لشركة Tesla لأوامر السيارة الموقعة.
