À utiliser lors de l'intégration avec l'API de flotte officielle de Tesla pour lire les données du véhicule/dispositif énergétique ou émettre des commandes à distance (par exemple, démarrer le préconditionnement CVC, réveiller le véhicule, contrôler la charge). Couvre l'intégration (enregistrement de l'application du développeur, régions/URL de base), les flux de jetons OAuth (jetons tiers + partenaires, rotation d'actualisation), l'hébergement de domaine/clé publique requis et l'utilisation du véhicule-commande/tesla-http-proxy officiel de Tesla pour les commandes de véhicules signées.
