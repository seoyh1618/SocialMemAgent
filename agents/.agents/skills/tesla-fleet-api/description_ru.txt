Используйте при интеграции с официальным API-интерфейсом Tesla Fleet для считывания данных транспортного средства/энергетического устройства или подачи удаленных команд (например, запуск предварительного кондиционирования HVAC, пробуждение автомобиля, управление зарядкой). Охватывает регистрацию (регистрация приложения разработчика, регионы/базовые URL-адреса), потоки токенов OAuth (токены третьих сторон и партнеров, ротация обновлений), необходимый хостинг домена/открытого ключа и использование официальной команды автомобиля Tesla/tesla-http-прокси для подписанных команд автомобиля.
