Используйте этот навык, когда пользователям нужно найти наиболее подходящие инструменты из **локально установленных навыков**. Подходит для «рекомендовать навыки для этапа Y», «выбрать навыки на основе моего проекта», «есть ли местные навыки, которые могут сделать X». Этот навык полностью учитывает контекст разговора, предысторию проекта и ход его выполнения и предоставляет точные рекомендации посредством многомерной классификации (этапы рабочего процесса, домены, типы файлов и т. д.). Для поиска и установки новых внешних навыков используйте find-skills.
