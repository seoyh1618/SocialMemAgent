Инструменты оптимизации Google. Программный пакет с открытым исходным кодом для оптимизации, специализирующийся на маршрутизации транспортных средств, потоках, целочисленном и линейном программировании, а также программировании в ограничениях. Имеет решатель CP-SAT мирового класса. Используется для решения задач маршрутизации транспортных средств (VRP), планирования, упаковки контейнеров, задач о рюкзаке, линейного программирования (LP), целочисленного программирования (MIP), сетевых потоков, программирования с ограничениями, комбинаторной оптимизации, распределения ресурсов, планирования смен, планирования цехов и задач дискретной оптимизации.
