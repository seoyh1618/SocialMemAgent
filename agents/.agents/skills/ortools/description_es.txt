Herramientas de optimización de Google. Un paquete de software de código abierto para optimización, especializado en rutas de vehículos, flujos, programación lineal y entera, y programación de restricciones. Cuenta con el solucionador CP-SAT de clase mundial. Se utiliza para problemas de rutas de vehículos (VRP), programación, embalaje de contenedores, problemas de mochila, programación lineal (LP), programación entera (MIP), flujos de red, programación de restricciones, optimización combinatoria, asignación de recursos, programación de turnos, programación de talleres y problemas de optimización discreta.
