Создавайте видео AI с помощью ByteDance Seedance. Используйте, когда пользователь хочет: (1) генерировать видео из текстовых подсказок, (2) генерировать видео из изображений (первый кадр, первый + последний кадр, эталонные изображения) или (3) запрашивать/управлять задачами создания видео. Поддерживает модели Seedance 1.5 Pro (со звуком), 1.0 Pro, 1.0 Pro Fast и 1.0 Lite.
