프로젝트 문서화 구조 생성 도우미. 숙련된 제품 관리자로서 다각적인 대화를 통해 프로젝트 요구 사항에 대한 깊은 통찰력을 얻고, 합리성을 분석하고 모범 사례 권장 사항을 제공하여 궁극적으로 완전한 기술 문서 시스템을 생성합니다.

**사용 시기:**
- 문서화 시스템 설정이 필요한 새로운 프로젝트 시작(첫 번째 단계 권장)
- 표준 문서가 부족하고 완료가 필요한 기존 프로젝트
- 코드는 존재하지만 문서가 누락되었습니다. 코드에서 문서를 다시 채워야 함
- 전문적인 제품 및 아키텍처 분석 조언이 필요합니다.

**지원되는 두 가지 모드:**
- 모드 A: 새 프로젝트 문서 작성(요구사항에서 문서까지)
- 모드 B: 코드에서 문서로 백필(코드에서 문서로)

**프로젝트 워크플로와의 관계:**
- 이 스킬은 완전한 문서(PRD, SAD, 개발 가이드 등)를 생성합니다.
- project-workflow는 해당 문서를 기반으로 개발 작업을 실행합니다.
- 추천 흐름: project-docs-setup → project-workflow

**트리거 문구:**
- "프로젝트 문서 생성" / "프로젝트 문서 초기화" / "프로젝트 문서 설정"
- "코드에서 문서 생성" / "프로젝트 문서 채우기" / "프로젝트 문서 보완"
- 프로젝트 요구 사항을 직접 설명합니다. "온라인 교육 플랫폼용 문서를 만드는 데 도움을 주세요."
