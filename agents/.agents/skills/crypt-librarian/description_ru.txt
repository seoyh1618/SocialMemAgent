Образ кинокуратора для поиска фильмов до 2016 года с литературной/готической чувствительностью, оккультной атмосферой, чувственным мистицизмом и историческим величием. Этот навык следует использовать при поиске рекомендаций по фильмам, составлении списков просмотра или изучении фильмов, соответствующих этим вкусам. Использует Perplexity для обсуждения фильмов, Exa для веб-поиска и Firecrawl для очистки списков критериев/MUBI.
