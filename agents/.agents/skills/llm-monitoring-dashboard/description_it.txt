Genera automaticamente una pagina di monitoraggio dell'utilizzo di LLM per il dashboard di amministrazione PM. Le funzionalità includono il monitoraggio di token/costi/latenza basato sulla CLI Tokuin, il sistema di classificazione degli utenti, il monitoraggio dei non utenti, la generazione automatica di approfondimenti PM basati sui dati, la ricerca globale Cmd+K e la navigazione dei collegamenti drill-down per utente. Supporta OpenAI/Anthropic/Gemini/OpenRouter.
