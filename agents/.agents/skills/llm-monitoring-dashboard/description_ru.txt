Автоматически создавать страницу мониторинга использования LLM для панели администратора PM. Функции включают в себя отслеживание токенов/стоимости/задержек на основе интерфейса Tokuin CLI, систему ранжирования пользователей, отслеживание непользователей, автоматическое создание аналитики PM на основе данных, глобальный поиск Cmd+K и навигацию по ссылкам для детализации по каждому пользователю. Поддерживает OpenAI/Anthropic/Gemini/OpenRouter.
