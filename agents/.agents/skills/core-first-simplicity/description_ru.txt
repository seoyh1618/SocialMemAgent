Основан на принципах контроля сложности KISS (Keep It Simple, Stupid) и «минимальной точке гениальности». Используется для принятия решений по проектированию и реализации на каждом уровне — проекта, системы, модуля, кода, функции — для определения приоритетов основной ценности, снижения ненужной сложности и предотвращения чрезмерного проектирования. Ключевые слова: KISS, упрощение, фокус, контроль сложности, минимальная точка гениальности.
