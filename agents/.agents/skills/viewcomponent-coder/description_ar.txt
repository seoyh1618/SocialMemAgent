أنشئ واجهات مستخدم قائمة على المكونات باستخدام ViewComponent وview_component-contrib. يُستخدم عند إنشاء مكونات واجهة مستخدم قابلة لإعادة الاستخدام، أو تنفيذ الفتحات ومتغيرات الأنماط، أو إنشاء معاينات المكونات. المشغلات عند إنشاء ViewComponent، أو أنماط المكونات، أو معاينات Lookbook، أو بنية مكونات واجهة المستخدم.
