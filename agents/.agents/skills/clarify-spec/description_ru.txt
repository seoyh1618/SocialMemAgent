Включается автоматически при расплывчатых запросах. Лучше уточнить, чем реализовать неверно. Признаки: запрос <25 слов, нет конкретных файлов/путей, нечёткие глаголы, неуверенные формулировки. Вывод — структурированный JSON для prompt-architect. Escape: «сделай просто», «без уточнений», «решай сам» пропускают уточнение.
