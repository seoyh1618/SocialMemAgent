قم بكتابة واجهات برمجة تطبيقات Swift أو مراجعتها أو تحسينها باستخدام إرشادات تصميم واجهة برمجة تطبيقات Swift للتسمية وتسميات الوسائط وتعليقات التوثيق والمصطلحات والاصطلاحات العامة. يُستخدم عند تصميم واجهات برمجة التطبيقات الجديدة، أو إعادة بناء الواجهات الحالية، أو مراجعة وضوح واجهة برمجة التطبيقات وطلاقتها.
