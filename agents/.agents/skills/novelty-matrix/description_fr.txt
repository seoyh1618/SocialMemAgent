Créez une matrice de nouveauté/travaux antérieurs qui compare les contributions de la soumission avec les travaux connexes (chevauchements ou deltas).
**Déclencheur** : matrice de nouveauté, matrice de travaux antérieurs, chevauchement/delta, comparaison de travaux liés, matrice de nouveauté.
**À utiliser lorsque** : évaluation de la nouveauté/du positionnement dans le cadre de l'examen par les pairs ; Il est nécessaire d’aligner les contributions sur les tâches connexes, élément par élément, et de fournir des preuves des différences.
**Ignorer si** : des revendications sont manquantes (exécutez d'abord `claims-extractor`) ou vous n'avez pas l'intention d'effectuer une analyse de positionnement de nouveauté.
**Réseau** : aucun (la récupération de travaux connexes supplémentaires est hors de portée, sauf si elle est fournie).
**Garde-corps** : indiquez clairement le chevauchement et le delta ; fournir des sources de preuves traçables (à partir de manuscrits/citations/déclarations d’auteurs) dans la mesure du possible.
