Recorta páginas web en Notion. Obtiene cualquier URL a través de Chrome CDP, convierte HTML a Markdown, luego a bloques de Notion y lo guarda en una base de datos o página de Notion especificada por el usuario. Úselo cuando el usuario quiera guardar/recortar una página web en Notion, o mencione "recortar en notion", "guardar página en notion", "网页剪藏到Notion".
