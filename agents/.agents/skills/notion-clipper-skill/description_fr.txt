Découpez les pages Web sur Notion. Récupère n'importe quelle URL via Chrome CDP, convertit le HTML en Markdown, puis en blocs Notion et enregistre dans la base de données ou la page Notion spécifiée par l'utilisateur. À utiliser lorsque l'utilisateur souhaite enregistrer/découper une page Web sur Notion, ou mentionne "découper sur Notion", "enregistrer la page sur Notion", "网页剪藏到Notion".
