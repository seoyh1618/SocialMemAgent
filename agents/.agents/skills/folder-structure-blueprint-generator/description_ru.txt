Комплексная, не зависящая от технологии подсказка для анализа и документирования структур папок проекта. Автоматически определяет типы проектов (.NET, Java, React, Angular, Python, Node.js, Flutter), создает подробные чертежи с параметрами визуализации, соглашениями об именах, шаблонами размещения файлов и шаблонами расширений для поддержания согласованной организации кода в различных стеках технологий.
