Prompt completo e indipendente dalla tecnologia per l'analisi e la documentazione delle strutture delle cartelle di progetto. Rileva automaticamente i tipi di progetto (.NET, Java, React, Angular, Python, Node.js, Flutter), genera progetti dettagliati con opzioni di visualizzazione, convenzioni di denominazione, modelli di posizionamento dei file e modelli di estensione per mantenere un'organizzazione coerente del codice tra diversi stack tecnologici.
