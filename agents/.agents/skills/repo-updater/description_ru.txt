Repo Updater — синхронизация нескольких репозиториев с оркестровкой проверки с помощью искусственного интеллекта. Параллельная синхронизация, очистка агентом грязных репозиториев, интеграция с ntm, git сантехника. 17K LOC Bash CLI. Используйте при синхронизации нескольких репозиториев GitHub, запуске агентской проверки незафиксированных изменений или организации проверки кода с помощью искусственного интеллекта в репозиториях.
