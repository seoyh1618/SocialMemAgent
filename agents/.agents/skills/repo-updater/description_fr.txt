Repo Updater - Synchronisation multi-dépôt avec orchestration des révisions assistée par l'IA. Synchronisation parallèle, balayage d'agent pour les dépôts sales, intégration ntm, plomberie git. 17K CLI Bash LOC. À utiliser lors de la synchronisation de plusieurs référentiels GitHub, de l'exécution d'un balayage d'agent sur des modifications non validées ou de l'orchestration de la révision de code assistée par l'IA dans les dépôts.
