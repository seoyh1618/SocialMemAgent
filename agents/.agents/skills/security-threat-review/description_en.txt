Evaluate the overall security of the application comprehensively from two perspectives: attackers (red team) and defenders (blue team). Run two agents in parallel → output a unified report via review-aggregator. Use it when you want to "know the overall security posture," "identify weaknesses from an attacker's viewpoint," or "verify that there are no gaps in defensive measures." For addressing specific vulnerabilities, use security-hardening; for high-speed detection of known patterns, use security-audit-quick.
