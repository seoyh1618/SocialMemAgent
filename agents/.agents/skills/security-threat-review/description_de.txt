Bewerten Sie umfassend die Gesamtsicherheit der Anwendung aus zwei Perspektiven: Angreifer (rotes Team) und Verteidiger (blaues Team). Führen Sie zwei Agenten parallel aus und lassen Sie den Review-Aggregator einen einheitlichen Bericht ausgeben. Verwenden Sie es, wenn Sie die allgemeine Sicherheitslage kennen, Schwachstellen aus der Sicht eines Angreifers identifizieren oder bestätigen möchten, dass Ihre Verteidigung keine Lücken aufweist. Um bestimmte Schwachstellen zu beheben, nutzen Sie die Sicherheitshärtung; Zur schnellen Erkennung bekannter Muster verwenden Sie security-audit-quick.
