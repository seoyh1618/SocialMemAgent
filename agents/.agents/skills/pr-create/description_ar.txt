ينشئ طلب سحب من التغييرات الحالية، ويراقب GitHub CI، ويصحح أخطاء أي فشل حتى يمر CI. استخدم هذا عندما يقول المستخدم "إنشاء علاقات عامة" أو "إجراء علاقات عامة" أو "طلب سحب مفتوح" أو "إرسال علاقات عامة" أو "علاقات عامة لهذه التغييرات" أو يريد تحويل عمله الحالي إلى علاقات عامة قابلة للمراجعة. بافتراض أن المشروع يستخدم git، ويتم استضافته على GitHub، ويحتوي على GitHub Actions CI مع عمليات التحقق التلقائية (الفحص، والبناء، والاختبارات، وما إلى ذلك). لا يتم الدمج - يتوقف عند مرور CI ويوفر رابط PR.
