Создает запрос на включение на основе текущих изменений, отслеживает CI GitHub и отлаживает любые сбои до тех пор, пока CI не пройдет. Используйте это, когда пользователь говорит «создать PR», «сделать PR», «открыть запрос на включение», «отправить запрос», «PR за эти изменения» или хочет превратить свою текущую работу в проверяемый PR. Предполагается, что проект использует git, размещен на GitHub и имеет GitHub Actions CI с автоматическими проверками (lint, сборка, тесты и т. д.). НЕ объединяется — останавливается, когда CI проходит и предоставляет ссылку PR.
