Comando combinado de compilación y ejecución de OpenHarmony XTS. Compilación y ejecución integrales del proyecto de prueba XTS. Utilice esta habilidad cuando necesite un flujo de prueba XTS completo: (1) cree el proyecto XTS para generar archivos HAP, (2) ejecute pruebas XTS y muestre los resultados. Admite pasar parámetros --package y --api a la etapa de ejecución.
