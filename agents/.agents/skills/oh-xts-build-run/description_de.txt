OpenHarmony XTS-Kombinationsbefehl zum Erstellen und Ausführen. Kompilieren und Ausführen des XTS-Testprojekts aus einer Hand. Verwenden Sie diese Fähigkeit, wenn ein vollständiger XTS-Testablauf erforderlich ist: (1) Erstellen Sie das XTS-Projekt, um HAP-Dateien zu generieren, (2) führen Sie XTS-Tests aus und zeigen Sie Ergebnisse an. Unterstützt die Übergabe der Parameter „--package“ und „--api“ an die Ausführungsphase.
