Комбинированная команда сборки и запуска OpenHarmony XTS. Комплексная компиляция и запуск тестового проекта XTS. Используйте этот навык, когда необходим полный поток тестов XTS: (1) создайте проект XTS для создания файлов HAP, (2) запустите тесты XTS и отобразите результаты. Поддерживает передачу параметров --package и --api на этап выполнения.
