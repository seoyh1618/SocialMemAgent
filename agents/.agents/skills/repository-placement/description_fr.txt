Un guide sur l'endroit où placer les interfaces du référentiel. Explique pourquoi, dans Clean Architecture, les interfaces du référentiel doivent être placées dans la couche de cas d'utilisation plutôt que dans la couche de domaine. Se concentre sur la prévention du couplage entre les modèles de domaine et les référentiels et sur l’application de l’intention de conception via la structure. Déclenché par des requêtes liées au placement du référentiel telles que « où placer le référentiel », « placement de l'interface du référentiel », « référentiel dans la couche de domaine » ou « référentiel dans une architecture propre ».
