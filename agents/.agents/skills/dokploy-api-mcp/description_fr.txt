Déployez et gérez des applications sur Dokploy (PaaS auto-hébergé). À utiliser lors du déploiement d'applications, de la gestion de services (PostgreSQL, Redis), de la configuration de domaines, de l'exécution de migrations ou du dépannage des déploiements Dokploy. Couvre l'API (tRPC), la CLI, le serveur MCP et les pièges courants avec Next.js, Docker et les pilotes de base de données.
