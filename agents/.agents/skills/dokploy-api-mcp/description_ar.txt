نشر التطبيقات وإدارتها على Dokploy (PaaS مستضاف ذاتيًا). يُستخدم عند نشر التطبيقات، أو إدارة الخدمات (PostgreSQL، أو Redis)، أو تكوين المجالات، أو تشغيل عمليات الترحيل، أو استكشاف أخطاء عمليات نشر Dokploy وإصلاحها. يغطي واجهة برمجة التطبيقات (tRPC) وCLI وخادم MCP والمزالق الشائعة مع Next.js وDocker وبرامج تشغيل قاعدة البيانات.
