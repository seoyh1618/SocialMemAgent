مهارة الخبراء في بناء محركات الألعاب والألعاب على الويب باستخدام HTML5 وCanvas وWebGL وJavaScript. يُستخدم عندما يُطلب منك إنشاء ألعاب، أو إنشاء محركات ألعاب، أو تنفيذ فيزياء اللعبة، أو التعامل مع اكتشاف التصادم، أو إعداد حلقات اللعبة، أو إدارة الكائنات، أو إضافة عناصر تحكم في اللعبة، أو العمل مع العرض ثنائي الأبعاد/ثلاثي الأبعاد. يغطي تقنيات منصات الألعاب، والألعاب بأسلوب الاختراق، وألعاب المتاهة، وخرائط البلاط، والصوت، ومتعددة اللاعبين عبر WebRTC، ونشر الألعاب.
