يجب القراءة قبل نشر أي وكيل ADK. دليل نشر ADK — Agent Engine، وCloud Run، وGKE، وCI/CD، والأسرار، وإمكانية المراقبة، وسير عمل الإنتاج. يُستخدم عند نشر الوكلاء على Google Cloud أو استكشاف أخطاء عمليات النشر وإصلاحها. لا تستخدم لأنماط كود واجهة برمجة التطبيقات (استخدم ورقة الغش adk)، أو التقييم (استخدم دليل تقييم adk)، أو سقالات المشروع (استخدم adk-scaffold).
