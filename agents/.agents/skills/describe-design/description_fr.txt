Recherchez une base de code et créez une documentation architecturale décrivant le fonctionnement des fonctionnalités ou des systèmes. À utiliser lorsque l'utilisateur demande à : (1) Documenter le fonctionnement d'une fonctionnalité, (2) Créer un aperçu de l'architecture, (3) Expliquer la structure du code pour l'intégration ou le transfert de connaissances, (4) Rechercher et décrire la conception d'un système. Produit des documents markdown avec des diagrammes Mermaid et des références de code stables adaptées aux humains et aux agents d'IA.
