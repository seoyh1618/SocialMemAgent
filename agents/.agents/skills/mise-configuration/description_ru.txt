Настройте переменные среды и параметры проекта, используя mise [env] как единственный источник достоверной информации. Используйте всякий раз, когда пользователю необходимо настроить mise.toml, централизовать переменные среды, настроить venvs Python с помощью Mise, использовать шаблоны Mise или структурировать конфигурации Hub-Spot и MonoRepo Mise с переопределением подпапок. НЕ используйте для оркестрации задач Mise (вместо этого используйте Mise-задачи) или для управления версиями во время выполнения, не связанным с конфигурацией среды.
