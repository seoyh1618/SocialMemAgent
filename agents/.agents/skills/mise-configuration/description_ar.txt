قم بتكوين متغيرات البيئة وإعدادات المشروع باستخدام mise [env] كمصدر وحيد للحقيقة. استخدمه عندما يحتاج المستخدم إلى إعداد mise.toml، أو مركزية متغيرات البيئة، أو تكوين Python venvs من خلال Mise، أو استخدام قوالب Mise، أو تكوين تكوينات Mise ذات المتحدث المحوري وmonorepo مع تجاوزات المجلد الفرعي. لا تستخدمه لتنسيق مهام Mise (استخدم مهام Mise بدلاً من ذلك) أو لإدارة إصدار وقت التشغيل غير المرتبط بتكوين البيئة.
