Configure las variables de entorno y los ajustes del proyecto utilizando mise [env] como única fuente de verdad. Úselo siempre que el usuario necesite configurar mise.toml, centralizar variables de entorno, configurar venvs de Python a través de mise, usar plantillas de mise o estructurar configuraciones de mise hub-spoke y monorepo con anulaciones de subcarpetas. NO lo use para la orquestación de tareas mise (use mise-tasks en su lugar) o para la administración de versiones en tiempo de ejecución no relacionadas con la configuración del entorno.
