在提交之前同步並驗證 Claude 文件（CLAUDE.md 文件和技能）。
透過根據實際程式碼庫狀態驗證所有聲明，確保文件準確性。

在以下情況下使用：(1) 在建立 PR 之前以確保文件準確，(2) 進行變更後
這可能會使文件宣告無效，(3) 當明確同步 Claude 文件時，
(4) 檢查文件是否陳舊時。

觸發器：「同步 claude」、「驗證 claude 文件」、「檢查文件」、「更新 CLAUDE.md」、
“提交之前”、“文檔不同步”、“/sync-claude”、“過時的文檔”

兩種模式：
- 完整：對儲存庫中所有克勞德文件進行詳盡驗證
- 變更：智慧型偵測受目前分支變更影響的文檔
