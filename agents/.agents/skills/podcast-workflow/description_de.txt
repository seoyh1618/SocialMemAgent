Podcast-Verarbeitungsworkflow aus einer Hand. Zwei Einstiegspunkte:
1) Direkte Bearbeitung: Stellen Sie einen YouTube-Link bereit und springen Sie direkt in den Workflow
2) Aktualisierungsprüfung: Rufen Sie die neuesten Episoden von verfolgten Erstellern ab und lassen Sie Benutzer auswählen, welche Episoden verarbeitet werden sollen
Funktionen: Untertitelextraktion → Content-Digest-Verarbeitung → automatische Speicherung in der Feishu-Wissensdatenbank.
Auslöserphrasen: „Podcast-Updates abrufen“, „Diesen Podcast verarbeiten“, „Podcast-Workflow“, „Alle neuen Podcasts“.
