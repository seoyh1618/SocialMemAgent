Высокоточный оптимизатор шаблонов LaTeX, поддерживающий выравнивание параметров стиля для любого шаблона LaTeX, выравнивание текста заголовка, сравнение формата заголовка (жирный шрифт), визуальный отчет HTML, предложения по автоматическому исправлению LaTeX и проверку сравнения PDF на уровне пикселей.
