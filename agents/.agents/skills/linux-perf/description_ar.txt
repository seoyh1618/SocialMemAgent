مهارة ملف تعريف Linux perf لتحليل أداء وحدة المعالجة المركزية. يُستخدم عند جمع ملفات تعريف العينات باستخدام سجل الأداء، أو إنشاء تقرير الأداء، أو قياس عدادات الأجهزة (أخطاء ذاكرة التخزين المؤقت، أو التنبؤات الخاطئة للفرع، أو IPC)، أو تحديد الوظائف الساخنة، أو إدخال بيانات الأداء في أدوات مخطط اللهب. يتم التنشيط عند الاستعلامات حول الأداء، أو عدادات أداء Linux، أو أحداث PMU، أو ملفات تعريف خارج وحدة المعالجة المركزية (CPU)، أو إحصائيات الأداء، أو التعليقات التوضيحية للأداء، أو ملفات التعريف المستندة إلى العينات على Linux.
