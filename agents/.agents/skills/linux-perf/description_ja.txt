CPU パフォーマンス分析のための Linux パフォーマンス プロファイラー スキル。パフォーマンス レコードを含むサンプリング プロファイルの収集、パフォーマンス レポートの生成、ハードウェア カウンタ (キャッシュ ミス、分岐予測ミス、IPC) の測定、ホット関数の特定、またはパフォーマンス データをフレームグラフ ツールに入力する場合に使用します。 Linux 上のパフォーマンス、Linux パフォーマンス カウンター、PMU イベント、オフ CPU プロファイリング、パフォーマンス統計、パフォーマンス アノテーション、またはサンプリング ベースのプロファイリングに関するクエリに対してアクティブ化されます。
