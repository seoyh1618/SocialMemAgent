Compétence de profileur de performances Linux pour l'analyse des performances du processeur. À utiliser pour collecter des profils d'échantillonnage avec enregistrement de performances, générer un rapport de performances, mesurer des compteurs matériels (manques de cache, erreurs de prédiction de branche, IPC), identifier des fonctions chaudes ou introduire des données de performances dans des outils flamegraph. S'active sur les requêtes concernant les performances, les compteurs de performances Linux, les événements PMU, le profilage hors CPU, les statistiques de performances, l'annotation des performances ou le profilage basé sur l'échantillonnage sous Linux.
