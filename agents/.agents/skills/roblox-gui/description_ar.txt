يُستخدم عند إنشاء عناصر Roblox GUI أو تحريكها أو تصحيح أخطاءها، بما في ذلك شاشات HUDs والقوائم وواجهة مستخدم الفضاء العالمي وتسميات المشغل. يتم التشغيل على: إعداد ScreenGui، أو موضع SurfaceGui أو BillboardGui، أو أسئلة تغيير حجم UDim2، أو الرسوم المتحركة لـ TweenService UI، أو القياس المستجيب، أو منطق واجهة المستخدم الرسومية LocalScript، أو مشكلات ResetOnSpawn، أو أي عمل تخطيط Frame/TextLabel/ImageButton.
