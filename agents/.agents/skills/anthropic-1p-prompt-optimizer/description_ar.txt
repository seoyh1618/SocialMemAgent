قم بتحسين المطالبات وإعادة كتابتها وتقييمها باستخدام أنماط Anthropic 1P التعليمية التفاعلية للهندسة السريعة (تعليمات واضحة/مباشرة، والمطالبة بالأدوار، وفصل علامات XML، وتنسيق المخرجات + التعبئة المسبقة، و"التعرف المسبق" خطوة بخطوة، وأمثلة قليلة اللقطات، وتقليل الهلوسة، وقوالب المطالبات المعقدة، وتسلسل المطالبات، وتنسيقات XML لاستخدام الأدوات). يُستخدم لـ 提示词优化/Prompt优化/الهندسة السريعة، أو إعادة كتابة النظام + مطالبات المستخدم، أو فرض المخرجات المنظمة (XML/JSON)، أو تقليل الهلوسة، أو إنشاء قوالب مطالبة متعددة الخطوات، أو إضافة أمثلة قليلة، أو تصميم سقالات تسلسل سريع/استدعاء الأدوات.
