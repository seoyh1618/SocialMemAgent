KI-Team-Rollenmanager für Multi-Agent-Entwicklungsworkflows. Verwenden Sie diese Option, wenn der Benutzer Teamrollen erstellen/löschen, Rollensitzungen in Terminal-Registerkarten öffnen, Rollen Aufgaben zuweisen, den Teamstatus überprüfen oder Rollenzweige zusammenführen möchte. Löst auf /agent-team-Befehle aus: „Eine Teamrolle erstellen“, „Rollensitzung öffnen“, „Aufgabe einer Rolle zuweisen“, „Teamstatus anzeigen“, „Rollenzweig zusammenführen“.
