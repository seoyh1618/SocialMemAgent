Ролевой менеджер группы ИИ для рабочих процессов многоагентной разработки. Используйте, когда пользователь хочет создавать/удалять роли в команде, открывать сеансы ролей на вкладках терминала, назначать задачи ролям, проверять статус команды или объединять ветки ролей. Запускается по командам /agent-team, «создать командную роль», «открыть сеанс роли», «назначить задачу роли», «показать статус команды», «объединить ветвь роли».
