Tutoriel interactif qui enseigne les tables dynamiques Snowflake. L'agent guide les utilisateurs étape par étape dans la création de pipelines de données avec actualisation automatique, traitement incrémentiel et modèles CDC. À utiliser lorsque l'utilisateur souhaite apprendre les tables dynamiques, créer un pipeline DT ou comprendre DT par rapport aux flux/tâches/vues matérialisées.
