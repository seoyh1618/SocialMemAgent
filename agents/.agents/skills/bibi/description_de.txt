BibiGPT-CLI zum Zusammenfassen von Videos, Audio und Podcasts direkt im Terminal. Verwenden Sie diese Option, wenn der Benutzer eine URL (YouTube, Bilibili, Podcast usw.) zusammenfassen oder seinen BibiGPT-Authentifizierungsstatus überprüfen möchte. Erfordert die Installation der BibiGPT-Desktop-App mit einer aktiven Anmeldesitzung oder eine festgelegte Umgebungsvariable BIBI_API_TOKEN.
