BibiGPT CLI pour résumer les vidéos, l'audio et les podcasts directement dans le terminal. À utiliser lorsque l'utilisateur souhaite résumer une URL (YouTube, Bilibili, podcast, etc.) ou vérifier son statut d'authentification BibiGPT. Nécessite l'application de bureau BibiGPT installée avec une session de connexion active ou un ensemble de variables d'environnement BIBI_API_TOKEN.
