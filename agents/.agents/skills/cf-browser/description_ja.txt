CloudflareのブラウザレンダリングREST APIを使用してWebサイトを閲覧およびスクレイピングします。エージェントがレンダリングされた Web コンテンツを取得したり、ページから構造化データを抽出したり、スクリーンショットを撮ったり、CSS セレクターを介して特定の要素を取得したりする必要がある場合に使用します。 「このサイトをスクレイピングする」、「このページからリストを取得する」、「この URL からデータを抽出する」、「このページのスクリーンショットを撮る」、「この Web サイトを閲覧する」などのタスク、またはライブ Web ページから情報を読み取り、クロールし、抽出するためにヘッドレス ブラウザー アクセスを必要とするタスクをトリガーします。 WebFetch が不十分な場合 (JS を多用するサイト、SPA、Cookie を必要とするページ、または構造化抽出が必要な場合) にも使用します。
