Explore y extraiga sitios web utilizando la API REST de renderizado del navegador de Cloudflare. Úselo cuando el agente necesite recuperar contenido web renderizado, extraer datos estructurados de páginas, tomar capturas de pantalla o extraer elementos específicos mediante selectores de CSS. Activa tareas como "extraer este sitio", "obtener listados de esta página", "extraer datos de esta URL", "tomar una captura de pantalla de esta página", "navegar este sitio web" o cualquier tarea que requiera acceso al navegador sin cabeza para leer, rastrear o extraer información de páginas web activas. Úselo también cuando WebFetch sea insuficiente (sitios con mucho JS, SPA, páginas que requieren cookies o cuando se necesita extracción estructurada).
