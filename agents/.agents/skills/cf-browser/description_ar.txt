تصفح واستخرج مواقع الويب باستخدام Browser Rendering REST API الخاص بـ Cloudflare. يُستخدم عندما يحتاج الوكيل إلى جلب محتوى الويب المعروض، أو استخراج البيانات المنظمة من الصفحات، أو التقاط لقطات شاشة، أو استخراج عناصر محددة عبر محددات CSS. مشغلات لمهام مثل "استخراج هذا الموقع"، أو "الحصول على قوائم من هذه الصفحة"، أو "استخراج البيانات من عنوان URL هذا"، أو "التقاط لقطة شاشة لهذه الصفحة"، أو "تصفح موقع الويب هذا"، أو أي مهمة تتطلب وصولاً بدون رأس إلى المتصفح لقراءة المعلومات أو الزحف إليها أو استخراجها من صفحات الويب المباشرة. يُستخدم أيضًا عندما يكون WebFetch غير كافٍ (مواقع JS الثقيلة، أو SPA، أو الصفحات التي تتطلب ملفات تعريف الارتباط، أو عند الحاجة إلى الاستخراج المنظم).
