Durchsuchen und scrapen Sie Websites mit der Browser Rendering REST API von Cloudflare. Verwenden Sie diese Option, wenn der Agent gerenderte Webinhalte abrufen, strukturierte Daten von Seiten extrahieren, Screenshots erstellen oder bestimmte Elemente über CSS-Selektoren durchsuchen muss. Wird bei Aufgaben wie „Diese Website durchsuchen“, „Einträge von dieser Seite abrufen“, „Daten von dieser URL extrahieren“, „Screenshot dieser Seite erstellen“, „Diese Website durchsuchen“ oder bei jeder Aufgabe ausgelöst, die kopflosen Browserzugriff zum Lesen, Crawlen oder Extrahieren von Informationen aus Live-Webseiten erfordert. Verwenden Sie es auch, wenn WebFetch nicht ausreicht (JS-lastige Websites, SPAs, Seiten, die Cookies erfordern, oder wenn eine strukturierte Extraktion erforderlich ist).
