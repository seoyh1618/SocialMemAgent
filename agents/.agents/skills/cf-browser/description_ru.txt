Просматривайте и очищайте веб-сайты с помощью REST API браузерного рендеринга Cloudflare. Используйте, когда агенту необходимо получить визуализированный веб-контент, извлечь структурированные данные со страниц, сделать снимки экрана или очистить определенные элементы с помощью селекторов CSS. Запускается при выполнении таких задач, как «очистить этот сайт», «получить списки с этой страницы», «извлечь данные из этого URL-адреса», «сделать снимок экрана этой страницы», «просмотреть этот веб-сайт» или любой задачи, требующей автономного доступа к браузеру для чтения, сканирования или извлечения информации с активных веб-страниц. Также используйте, когда WebFetch недостаточен (сайты с большим количеством JS, одностраничные приложения, страницы, требующие файлов cookie, или когда необходимо структурированное извлечение).
