Solution tout-en-un complète de navigation Amap : intègre des fonctionnalités cartographiques complètes (MapView / toutes les superpositions / cartes hors ligne / positionnement / calculs géométriques, aucun package de base requis), fournit une interface utilisateur de navigation officielle NaviView (voix / événements / trafic / boussole) ; prend en charge la planification d'itinéraires multimodes (conduite/marche/vélo/camion/moto/vélo électrique) avec des stratégies/restrictions/waypoints ; propose un routage indépendant avancé (estimation/aperçu/sélection/démarrage) ; inclut le plug-in de configuration pour la configuration automatique ; est en conflit avec le SDK expo-gaode-map, installez uniquement ce package.
