Gaode Navigation All-in-One-Lösung: Integriert vollständige Kartenfunktionen (MapView / alle Overlays / Offline-Karten / Standort / Geometrieberechnungen, kein Kernpaket erforderlich), bietet offizielle NaviView-Navigations-Benutzeroberfläche (Sprache / Ereignisse / Verkehr / Kompass); unterstützt multimodale Routenplanung (Autofahren / Gehen / Radfahren / LKW / Motorrad / E-Bike) mit Strategien / Einschränkungen / Wegpunkten; bietet erweitertes unabhängiges Routing (Schätzung/Vorschau/Auswahl/Start); enthält ein Config-Plugin für die automatische Konfiguration; Konflikte mit expo-gaode-map SDK, nur dieses Paket muss installiert werden.
