Полнофункциональное интегрированное решение Amap Navigation: включает в себя полные возможности картографирования (MapView/все наложения/автономные карты/позиционирование/геометрические расчеты — основной пакет не требуется), предоставляет официальный навигационный интерфейс NaviView (голос/события/трафик/компас); поддерживает мультимодальное планирование маршрута (автомобиль/ходьба/велосипед/грузовик/мотоцикл/электронный велосипед) с использованием стратегий/ограничений/путевых точек; предлагает расширенную независимую маршрутизацию (ETA/предварительный просмотр/выбор/запуск маршрута); включает плагин Config для автоматической настройки; конфликтует с SDK expo-gaode-map — установите только этот пакет.
