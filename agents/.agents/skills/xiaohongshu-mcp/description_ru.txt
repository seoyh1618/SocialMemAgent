Xiaohongshu MCP Skill — полное решение для автоматизации с исправлением входа в систему. Возможности: (1) Управление входом в систему, (2) Поиск, публикация, взаимодействие, (3) Полная поддержка протокола MCP (13 инструментов), (4) Стратегия комментариев. Встроенное уведомление Feishu, интеграция iflow. Триггеры: xiaohongshu, rednote, автоматизация 小红书.
