Guía de integración para WarpGrep de Morph (búsqueda rápida de código agente) y Fast Apply (edición de código de 10,500 tok/s). Úselo cuando cree agentes de codificación que necesiten una búsqueda de código rápida y precisa o que necesiten aplicar ediciones generadas por IA para codificar de manera eficiente. Particularmente útil para bases de código grandes, consultas lógicas profundas, seguimiento de errores y análisis de rutas de código.
