Guide d'intégration pour WarpGrep de Morph (recherche rapide de code agent) et Fast Apply (édition de code à 10 500 tok/s). À utiliser lors de la création d'agents de codage qui nécessitent une recherche de code rapide et précise ou qui doivent appliquer efficacement des modifications générées par l'IA au code. Particulièrement utile pour les bases de code volumineuses, les requêtes logiques approfondies, le suivi des bogues et l'analyse du chemin du code.
