Virtualización sin cabeza TanStack Virtual para React. Úselo al representar listas grandes (más de 100 elementos),
implementar desplazamiento virtual, crear feeds de desplazamiento infinito, virtualizar cuadrículas o tablas,
usando desplazamiento a nivel de ventana o implementando diseños de mampostería/carriles con @tanstack/react-virtual.
Activadores de: useVirtualizer, useWindowVirtualizer, lista virtual, desplazamiento virtual, virtualización de lista.
