Erstellen Sie Chrome-Erweiterungen mithilfe des WXT-Frameworks mit TypeScript, React, Vue oder Svelte. Verwenden Sie es, wenn Sie Browsererweiterungen erstellen, browserübergreifende Add-ons entwickeln oder mit Chrome Web Store-Projekten arbeiten. Wird durch Phrasen wie „Chrome-Erweiterung“, „Browser-Erweiterung“, „WXT-Framework“, „Manifest v3“ oder Dateimuster wie wxt.config.ts ausgelöst.
