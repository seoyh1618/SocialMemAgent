Créez des extensions Chrome à l'aide du framework WXT avec TypeScript, React, Vue ou Svelte. À utiliser lors de la création d'extensions de navigateur, du développement de modules complémentaires multi-navigateurs ou de l'utilisation de projets Chrome Web Store. Se déclenche sur des expressions telles que « extension Chrome », « extension de navigateur », « framework WXT », « manifeste v3 » ou des modèles de fichiers comme wxt.config.ts.
