المعايير الأساسية لجميع وكلاء سير عمل GitHub. يغطي المصادقة، والافتراضيات الذكية، واكتشاف المستودع، ومخرجات MD+HTML المزدوجة، ومعايير إمكانية الوصول إلى HTML المتوافقة مع قارئ الشاشة، وقواعد السلامة، وإعلانات التقدم، والتنفيذ المتوازي، وجودة المخرجات. تقدم بطلبك عند إنشاء أي وكيل لسير عمل GitHub - المشكلات، والعلاقات العامة، والإحاطات، والتحليلات، وتقارير المجتمع، وإدارة الفريق.
