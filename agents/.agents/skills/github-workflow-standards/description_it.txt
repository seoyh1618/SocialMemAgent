Standard fondamentali per tutti gli agenti del flusso di lavoro GitHub. Copre l'autenticazione, le impostazioni predefinite intelligenti, il rilevamento del repository, il doppio output MD+HTML, gli standard di accessibilità HTML compatibili con lo screen reader, le regole di sicurezza, gli annunci di avanzamento, l'esecuzione parallela e la qualità dell'output. Applicalo durante la creazione di qualsiasi agente del flusso di lavoro GitHub: problemi, PR, briefing, analisi, report della community, gestione del team.
