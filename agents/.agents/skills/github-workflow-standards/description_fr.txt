Normes de base pour tous les agents de workflow GitHub. Couvre l'authentification, les valeurs par défaut intelligentes, la découverte de référentiels, la double sortie MD + HTML, les normes d'accessibilité HTML compatibles avec les lecteurs d'écran, les règles de sécurité, les annonces de progression, l'exécution parallèle et la qualité de sortie. Postulez lors de la création d'un agent de workflow GitHub : problèmes, relations publiques, briefings, analyses, rapports de communauté, gestion d'équipe.
