Создавайте комплексные файлы ARCHITECTURE.md для любого репозитория, следуя каноническим рекомендациям matklad. Этот навык следует использовать при создании документации по базе кода, которая служит картой для разработчиков и агентов ИИ, при проверке существующих репозиториев на предмет понимания архитектуры или когда пользователи просят «задокументировать архитектуру», «создать архитектуру.md» или «сопоставить эту кодовую базу». Создает виды с высоты птичьего полета, диаграммы ASCII/Mermaid, кодовые карты, инварианты и границы слоев.
