Cree archivos ARCHITECTURE.md completos para cualquier repositorio siguiendo las pautas canónicas de matklad. Esta habilidad debe usarse al crear documentación de base de código que sirva como mapa para desarrolladores y agentes de IA, auditar repositorios existentes para comprender la arquitectura o cuando los usuarios soliciten "documentar la arquitectura", "crear un arquitectura.md" o "mapear esta base de código". Produce vistas panorámicas, diagramas ASCII/Sirena, mapas de códigos, invariantes y límites de capas.
