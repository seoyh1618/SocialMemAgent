البحث على الويب في الوقت الفعلي عبر الأنظمة الأساسية والبحث عبر نقطة نهاية Grok المتوافقة مع OpenAI، مما يؤدي إلى إرجاع JSON بالمحتوى والمصادر. يُستخدم للتحقق من الإصدار والتحقق من واجهة برمجة التطبيقات والمستندات واستكشاف الأخطاء وإصلاحها وأي حقائق حساسة للوقت على أنظمة التشغيل Windows أو macOS أو Linux.
