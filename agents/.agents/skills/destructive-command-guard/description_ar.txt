يحظر الأوامر الخطيرة قبل تنفيذها عبر خطاف Claude Code القائم على Rust. يُستخدم عند تكوين حراس أمان الوكيل، أو إعداد حظر الأوامر المدمر، أو تدقيق قواعد حماية واجهة سطر الأوامر (CLI). يُستخدم لحماية إعادة ضبط git، واعتراض rm -rf، وحظر الدفع بالقوة، وتصفية الأوامر المستندة إلى الحزمة، وسلامة الخطاف PreToolUse.
