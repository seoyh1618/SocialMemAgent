Блокирует опасные команды перед выполнением с помощью перехватчика Claude Code на основе Rust. Используйте при настройке средств защиты агентов, настройке деструктивной блокировки команд или аудите правил защиты CLI. Используйте для защиты от сброса git, перехвата rm -rf, принудительной блокировки, фильтрации команд на основе пакетов и безопасности перехвата PreToolUse.
