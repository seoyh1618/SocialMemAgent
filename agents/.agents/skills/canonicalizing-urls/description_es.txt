Úselo cuando trabaje con cualquier URL que pueda contener parámetros de seguimiento, envoltorios de redireccionamiento, prefijos locales o enlaces para compartir opacos, para producir una forma canónica y limpia. Se activa de forma proactiva cuando el agente usa una URL en un contexto donde la limpieza es importante (guardar en Notion, citar en un documento, crear un hipervínculo). También se activa explícitamente cuando el usuario dice "canonicalizar", "limpiar esta URL", "eliminar parámetros de seguimiento" o invoca /canonicalize.
