À utiliser lorsque vous travaillez avec une URL pouvant contenir des paramètres de suivi, des wrappers de redirection, des préfixes de paramètres régionaux ou des liens de partage opaques — pour produire un formulaire propre et canonique. Se déclenche de manière proactive lorsque l'agent utilise une URL dans un contexte où la propreté est importante (enregistrement dans Notion, citation dans un document, création d'un lien hypertexte). Se déclenche également explicitement lorsque l'utilisateur dit "canonicalize", "nettoyer cette URL", "supprimer les paramètres de suivi" ou invoque /canonicalize.
