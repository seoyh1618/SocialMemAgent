Verwenden Sie diese Option, wenn Sie mit einer URL arbeiten, die Tracking-Parameter, Redirect-Wrapper, Gebietsschema-Präfixe oder undurchsichtige Freigabe-Links enthalten kann – um ein sauberes, kanonisches Formular zu erstellen. Wird proaktiv ausgelöst, wenn der Agent eine URL in einem Kontext verwendet, in dem es auf Sauberkeit ankommt (Speichern in Notion, Zitieren in einem Dokument, Erstellen eines Hyperlinks). Wird auch explizit ausgelöst, wenn der Benutzer „kanonischisieren“, „diese URL bereinigen“, „Tracking-Parameter entfernen“ sagt oder /canonicalize aufruft.
