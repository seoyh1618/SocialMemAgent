Этот навык отслеживает проверки PR CI, опрашивая статус GitHub до завершения или истечения времени ожидания. Используйте, когда пользователь запрашивает проверку статуса CI, ожидание прохождения CI, мониторинг проверок PR или проверку статуса сборки. Применимо для таких запросов, как «проверьте мой CI», «подождите, пока CI пройдет», «зеленый ли мой PR» или «отслеживайте проверки CI».
