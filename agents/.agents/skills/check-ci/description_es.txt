Esta habilidad monitorea las comprobaciones de PR CI sondeando el estado de GitHub hasta su finalización o tiempo de espera. Úselo cuando el usuario solicite verificar el estado del CI, esperar a que pase el CI, monitorear las verificaciones de relaciones públicas o verificar el estado de la compilación. Aplicable para consultas como "verificar mi CI", "esperar a que pase el CI", "mi PR está en verde" o "supervisar las verificaciones de CI".
