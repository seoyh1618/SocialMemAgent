Этот навык следует использовать, когда пользователь спрашивает о написании торговых стратегий, бэктестировании, развертывании ботов Freqtrade, количественной торговле, оптимизации стратегии или любой другой операции, связанной с Freqtrade. Используйте, когда пользователь говорит: «написать стратегию», «создать стратегию», «тестирование на истории», «развертывание Freqtrade», «развертывание бота», «количественная торговля», «оптимизация стратегии», «гиперопт», «живой торговый бот», «写策略», «创建策略», «回测», «部署Freqtrade», «部署机器人», «量化交易», «量化策略», «策略优化», «超参数优化», «实盘机器人». ВАЖНО: ВСЕГДА используйте create_strategy для создания файлов стратегии. НИКОГДА не пишите код стратегии Python вручную. Для цен/графиков криптовалют используйте aicoin-market. Для биржевой торговли используйте aicoin-trading. Для Hyperliquid используйте aicoin-hyperliquid.
