Scannen Sie C/C++-Codebasen auf Codequalitätsprobleme, einschließlich besonders großer Dateien/Funktionen und zirkulärer Abhängigkeiten. Verwenden Sie es, wenn der Benutzer Dateigrößen überprüfen, übergroße Funktionen finden, zirkuläre Abhängigkeiten erkennen, die Codekomplexität analysieren, Codegerüche finden oder Wartbarkeitsprobleme in C/C++-Code identifizieren möchte. Unterstützt das Scannen einzelner Dateien oder ganzer Verzeichnisse mit konfigurierbaren Schwellenwerten.
