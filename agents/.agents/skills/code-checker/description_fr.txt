Analysez les bases de code C/C++ pour détecter les problèmes de qualité du code, notamment les fichiers/fonctions très volumineux et les dépendances circulaires. À utiliser lorsque l'utilisateur demande à vérifier la taille des fichiers, à rechercher des fonctions surdimensionnées, à détecter des dépendances circulaires, à analyser la complexité du code, à rechercher des odeurs de code ou à identifier des problèmes de maintenabilité dans le code C/C++. Prend en charge l'analyse de fichiers individuels ou de répertoires entiers avec des seuils configurables.
