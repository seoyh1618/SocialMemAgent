Esegui la scansione delle basi di codice C/C++ per individuare problemi di qualità del codice, inclusi file/funzioni di grandi dimensioni e dipendenze circolari. Da utilizzare quando l'utente chiede di controllare le dimensioni dei file, trovare funzioni sovradimensionate, rilevare dipendenze circolari, analizzare la complessità del codice, trovare odori di codice o identificare problemi di manutenibilità nel codice C/C++. Supporta la scansione di singoli file o intere directory con soglie configurabili.
