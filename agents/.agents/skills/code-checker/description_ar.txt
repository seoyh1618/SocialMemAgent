قم بمسح قواعد تعليمات برمجية C/C++ بحثًا عن مشكلات جودة التعليمات البرمجية بما في ذلك الملفات/الوظائف الكبيرة جدًا والتبعيات الدائرية. يُستخدم عندما يطلب المستخدم التحقق من أحجام الملفات، أو العثور على وظائف كبيرة الحجم، أو اكتشاف التبعيات الدائرية، أو تحليل تعقيد التعليمات البرمجية، أو العثور على روائح التعليمات البرمجية، أو تحديد مشكلات قابلية الصيانة في تعليمات برمجية C/C++. يدعم فحص الملفات الفردية أو الدلائل بأكملها بحدود قابلة للتكوين.
