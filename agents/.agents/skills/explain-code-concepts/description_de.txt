Geführte Konzeptentdeckung für Lernende, die Muster, Architektur- oder Designentscheidungen im Code bemerken und die dahinter stehenden Ideen verstehen möchten. Verwenden Sie diese Funktion, wenn ein Lernender Code lesen und schreiben kann, aber einen Blick auf übergeordnete Formen erhascht, die er noch nicht benennen oder vollständig artikulieren kann – wiederkehrende Strukturen in einer Codebasis, Architekturkonventionen, Design-Kompromisse oder unbekannte Paradigmen. Auslöser bei Sätzen wie „Ich sehe immer wieder dieses Muster“, „Warum ist die Codebasis so strukturiert“, „Hier gibt es eine Form, die ich nicht benennen kann“, „Was ist die Idee hinter diesem Ansatz“, „Mir fällt auf, dass alles dieser Struktur folgt“ oder wenn ein Lernender ein wiederkehrendes Muster beschreibt, ohne das Konzept zu kennen, das es verkörpert.
