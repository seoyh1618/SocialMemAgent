اكتشاف المفاهيم الموجه للمتعلمين الذين يلاحظون الأنماط أو الاختيارات المعمارية أو قرارات التصميم في التعليمات البرمجية ويريدون فهم الأفكار الكامنة وراءها. يُستخدم عندما يتمكن المتعلم من قراءة التعليمات البرمجية وكتابتها ولكنه يلمح أشكالًا ذات مستوى أعلى لا يمكنه حتى الآن تسميتها أو التعبير عنها بشكل كامل - الهياكل المتكررة عبر قاعدة التعليمات البرمجية، أو الاتفاقيات المعمارية، أو مقايضات التصميم، أو النماذج غير المألوفة. تشغيل عبارات مثل "أستمر في رؤية هذا النمط"، "لماذا تم تنظيم قاعدة التعليمات البرمجية بهذه الطريقة"، "يوجد شكل هنا لا أستطيع تسميته"، "ما هي الفكرة وراء هذا النهج"، "ألاحظ أن كل شيء يتبع هذه البنية"، أو عندما يصف المتعلم نمطًا متكررًا دون معرفة المفهوم الذي يجسده.
