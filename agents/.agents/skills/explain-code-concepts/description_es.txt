Descubrimiento de conceptos guiado para estudiantes que notan patrones, opciones arquitectónicas o decisiones de diseño en el código y desean comprender las ideas detrás de ellos. Úselo cuando un alumno puede leer y escribir código, pero vislumbra formas de nivel superior que aún no puede nombrar o articular completamente: estructuras recurrentes en una base de código, convenciones arquitectónicas, compensaciones de diseño o paradigmas desconocidos. Se activan frases como "Sigo viendo este patrón", "por qué el código base está estructurado de esta manera", "aquí hay una forma que no puedo nombrar", "cuál es la idea detrás de este enfoque", "Noto que todo sigue esta estructura" o cuando un alumno describe un patrón recurrente sin conocer el concepto que representa.
