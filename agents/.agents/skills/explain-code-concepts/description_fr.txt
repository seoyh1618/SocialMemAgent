Découverte guidée de concepts pour les apprenants qui remarquent des modèles, des choix architecturaux ou des décisions de conception dans le code et souhaitent comprendre les idées qui les sous-tendent. À utiliser lorsqu'un apprenant peut lire et écrire du code mais entrevoit des formes de niveau supérieur qu'il ne peut pas encore nommer ou articuler complètement : structures récurrentes dans une base de code, conventions architecturales, compromis de conception ou paradigmes inconnus. Se déclenche sur des phrases telles que « Je continue de voir ce modèle », « Pourquoi la base de code est-elle structurée de cette façon », « Il y a une forme ici que je ne peux pas nommer », « Quelle est l'idée derrière cette approche », « Je remarque que tout suit cette structure », ou lorsqu'un apprenant décrit un modèle récurrent sans connaître le concept qu'il incarne.
