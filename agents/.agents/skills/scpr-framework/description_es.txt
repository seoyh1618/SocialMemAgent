Marco SCPR (Situación-Complicación-Problema-Recomendación) para la resolución estructurada de problemas y la comunicación ejecutiva. Úselo cuando los usuarios necesiten estructurar argumentos estratégicos, analizar situaciones comerciales, crear resúmenes ejecutivos o desarrollar planteamientos claros de problemas utilizando la comunicación al estilo McKinsey. Aplicar al estructurar recomendaciones, escribir memorandos u organizar el pensamiento estratégico.
