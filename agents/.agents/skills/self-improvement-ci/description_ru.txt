Рабочий процесс самосовершенствования только для CI с использованием gh-aw (GitHub Agentic Workflows). Фиксирует повторяющиеся шаблоны сбоев и сигналы качества в результате проверок запросов на включение, выдает кандидатов для структурированного обучения и предлагает надежные правила предотвращения без интерактивных подсказок. Используйте, когда: вам нужен автоматический захват обучения в CI/безголовых конвейерах.
