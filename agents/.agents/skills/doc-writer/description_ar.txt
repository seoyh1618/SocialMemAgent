مهارة لدعم كتابة المستندات. يتم استخدامه عندما يريد المستخدم إنشاء المستندات أو تحريرها أو تحديثها بناءً على PRD. أثناء الرجوع إلى مستند متطلبات المنتج الموضح في docs/prd.md، تساعد هذه المهارة في كتابة المستندات الفنية ومستندات التصميم وأدلة المستخدم ووثائق API والمزيد. يتم تشغيله من خلال الإشارة إلى مهام إنشاء المستندات مثل "كتابة مستند" أو "إنشاء مواصفات" أو "إنشاء دليل".
