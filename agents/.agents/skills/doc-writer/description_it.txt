Un'abilità per supportare la scrittura di documenti. Viene utilizzato quando un utente desidera creare, modificare o aggiornare documenti basati su un PRD. Facendo riferimento al documento sui requisiti del prodotto descritto in docs/prd.md, questa competenza aiuta nella scrittura di documenti tecnici, documenti di progettazione, guide per l'utente, documentazione API e altro ancora. Viene attivato da menzioni di attività di creazione di documenti come "scrivere un documento", "creare una specifica" o "creare una guida".
