Una habilidad para apoyar la redacción de documentos. Se utiliza cuando un usuario desea crear, editar o actualizar documentos basados ​​en un PRD. Al hacer referencia al documento de requisitos del producto descrito en docs/prd.md, esta habilidad ayuda a escribir documentos técnicos, documentos de diseño, guías de usuario, documentación de API y más. Se desencadena por menciones de tareas de creación de documentos como "escribir un documento", "crear una especificación" o "hacer una guía".
