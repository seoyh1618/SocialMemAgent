Навыки поддержки написания документов. Он используется, когда пользователь хочет создавать, редактировать или обновлять документы на основе PRD. При обращении к документу с требованиями к продукту, описанному в docs/prd.md, этот навык помогает в написании технической документации, проектной документации, руководств пользователя, документации API и многого другого. Его вызывают упоминания о задачах создания документов, таких как «написать документ», «создать спецификацию» или «составить руководство».
