Eine Fähigkeit zur Unterstützung beim Schreiben von Dokumenten. Es wird verwendet, wenn ein Benutzer Dokumente basierend auf einem PRD erstellen, bearbeiten oder aktualisieren möchte. Diese Fähigkeit bezieht sich auf das in docs/prd.md beschriebene Produktanforderungsdokument und hilft beim Verfassen technischer Dokumente, Designdokumente, Benutzerhandbücher, API-Dokumentation und mehr. Es wird durch die Erwähnung von Dokumenterstellungsaufgaben wie „ein Dokument schreiben“, „eine Spezifikation erstellen“ oder „einen Leitfaden erstellen“ ausgelöst.
