يجب القراءة قبل إعداد إمكانية المراقبة لوكلاء ADK أو عند تحليل حركة الإنتاج أو تصحيح سلوك الوكيل أو تحسين أداء الوكيل. دليل إمكانية مراقبة ADK — التتبع السحابي، وتسجيل الاستجابة السريعة، وتحليلات وكيل BigQuery، وعمليات تكامل الجهات الخارجية، واستكشاف الأخطاء وإصلاحها. يُستخدم عند تكوين المراقبة أو التتبع أو التسجيل للوكلاء، أو عند فهم كيفية تعامل الوكيل المنشور مع حركة المرور الحقيقية.
