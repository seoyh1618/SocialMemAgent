MUSS LESEN, bevor Sie die Beobachtbarkeit für ADK-Agenten einrichten oder wenn Sie den Produktionsverkehr analysieren, das Agentenverhalten debuggen oder die Agentenleistung verbessern. ADK-Beobachtbarkeitsleitfaden – Cloud Trace, Prompt-Response-Protokollierung, BigQuery Agent Analytics, Integrationen von Drittanbietern und Fehlerbehebung. Verwenden Sie diese Option, wenn Sie die Überwachung, Ablaufverfolgung oder Protokollierung für Agents konfigurieren oder wenn Sie verstehen möchten, wie ein bereitgestellter Agent mit echtem Datenverkehr umgeht.
