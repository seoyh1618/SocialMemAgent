환각 방지 검증을 통해 포괄적이고 정확한 README.md 파일을 생성합니다.
사용 시기: 새 README 생성, 기존 README 업데이트, README 정확성 검증 또는 문서가 실제 코드베이스와 일치하는지 확인합니다.
기본 트리거: "README 생성" → 전체 생성 워크플로(스캔 → 생성 → 유효성 검사 → 테스트) "README 업데이트" → 유효성 검사를 통한 부분 업데이트 "README 유효성 검사" → 유효성 검사 전용 워크플로 "README 정확도 확인" → 환각 방지 감사
주요 기능: - 정확한 사실을 위한 코드베이스 스캐닝 - 5계층 환각 방지 검증 - 스크립트 실행 테스트 - 모든 주장에 대한 인용 추적 - 템플릿 기반 구조 선택
차별화 요소: 모든 주장은 실제 코드베이스와 비교하여 확인됩니다. 가정도 없고 환각도 없습니다. 스크립트 실행 테스트를 통해 예제가 작동하는지 확인합니다.
