Полная настройка Sentry SDK для Flutter и Dart. Используйте, когда вас просят «добавить Sentry во Flutter», «установить sendry_flutter», «настроить Sentry в Dart» или настроить мониторинг ошибок, трассировку, профилирование, воспроизведение сеанса или ведение журнала для приложений Flutter. Поддерживает Android, iOS, macOS, Linux, Windows и Интернет.
