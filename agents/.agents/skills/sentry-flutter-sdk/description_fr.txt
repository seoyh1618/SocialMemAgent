Configuration complète du SDK Sentry pour Flutter et Dart. À utiliser lorsqu'on vous demande d'« ajouter Sentry à Flutter », « d'installer sentry_flutter », de « configurer Sentry dans Dart » ou de configurer la surveillance des erreurs, le traçage, le profilage, la relecture de session ou la journalisation pour les applications Flutter. Prend en charge Android, iOS, macOS, Linux, Windows et Web.
