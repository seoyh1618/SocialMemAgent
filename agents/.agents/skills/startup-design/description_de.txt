Entwerfen, validieren und planen Sie ein Startup von Grund auf. Umfasst Marktforschung, Wettbewerbsanalyse, Geschäftsmodell, Markenidentität, Produktdefinition, Finanzprognosen und Validierungsexperimente. Wird ausgelöst, wenn der Benutzer eine Startup-Idee zum Erkunden hat, ein Geschäftskonzept validieren möchte, einen Geschäftsplan oder ein Lean Canvas benötigt, nach der Marktgröße oder dem Wettbewerbsumfeld fragt, eine Markenpositionierung oder eine Markteinführungsstrategie wünscht oder etwas sagt wie „Ich habe eine Idee für …“ oder „Lohnt es sich, diese Idee zu verfolgen“. Behandelt auch die Fortsetzung von einem vorherigen Prüfpunkt.
