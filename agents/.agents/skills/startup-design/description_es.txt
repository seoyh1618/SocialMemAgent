Diseñar, validar y planificar una startup desde cero. Cubre investigación de mercado, análisis competitivo, modelo de negocio, identidad de marca, definición de producto, proyecciones financieras y experimentos de validación. Se activa cuando el usuario tiene una idea de inicio para explorar, quiere validar un concepto de negocio, necesita un plan de negocios o un lienzo ajustado, solicita el tamaño del mercado o el panorama competitivo, quiere posicionamiento de marca o estrategia de comercialización, o dice algo como "Tengo una idea para..." o "vale la pena seguir esta idea". También maneja la reanudación desde un punto de control anterior.
