Оценивает и оптимизирует навыки агентов с помощью цикла GEPA (генерация/оценка/предложение/применение) на базе DSPy. Загружает YAML-файлы сценариев в виде наборов данных DSPy, оценивает выходные данные с помощью показателей соответствия шаблону и оптимизирует подсказки с помощью телесуфлеров BootstrapFewShot или MIPROv2. Также генерирует новые YAML-файлы сценариев на основе описаний навыков.
