Implementieren Sie die globale Statusverwaltung in React-Apps mit TanStack Store (@tanstack/react-store). Wird zum Erstellen von Stores, abgeleiteten Status, Abonnements, Batch-Updates oder zum Integrieren von TanStack Store mit React-Komponenten über den useStore-Hook verwendet. Behandelt createStore, useStore, flachen Vergleich, Batch, abgeleitete Stores, Atome und asynchrone Atome.
