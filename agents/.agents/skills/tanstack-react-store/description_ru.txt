Реализуйте глобальное управление состоянием в приложениях React с помощью TanStack Store (@tanstack/react-store). Используйте при создании хранилищ, производного состояния, подписок, пакетных обновлений или интеграции TanStack Store с компонентами React через перехватчик useStore. Охватывает createStore, useStore, поверхностное сравнение, пакетные операции, производные хранилища, атомы и асинхронные атомы.
