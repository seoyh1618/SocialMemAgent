تنفيذ إدارة الحالة العالمية في تطبيقات React باستخدام TanStack Store (@tanstack/react-store). يُستخدم عند إنشاء متاجر أو حالة مشتقة أو اشتراكات أو تحديثات مجمعة أو دمج TanStack Store مع مكونات React عبر خطاف useStore. يغطي createStore، وuseStore، والمقارنة الضحلة، والدُفعات، والمخازن المشتقة، والذرات، والذرات غير المتزامنة.
