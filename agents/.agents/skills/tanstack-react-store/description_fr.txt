Implémentez la gestion globale de l'état dans les applications React à l'aide de TanStack Store (@tanstack/react-store). À utiliser lors de la création de magasins, d'états dérivés, d'abonnements, de mises à jour par lots ou de l'intégration de TanStack Store avec des composants React via le hook useStore. Couvre createStore, useStore, la comparaison superficielle, le lot, les magasins dérivés, les atomes et les atomes asynchrones.
