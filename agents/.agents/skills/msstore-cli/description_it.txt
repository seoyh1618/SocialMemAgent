CLI per sviluppatori di Microsoft Store (msstore) per la pubblicazione di applicazioni Windows in Microsoft Store. Utilizzare quando richiesto per configurare le credenziali dello Store, elencare le app dello Store, controllare lo stato dell'invio, pubblicare gli invii, gestire i pacchetti in anteprima, configurare CI/CD per la pubblicazione dello Store o integrarsi con il Centro per i partner. Supporta le applicazioni Windows App SDK/WinUI, UWP, .NET MAUI, Flutter, Electron, React Native e PWA.
