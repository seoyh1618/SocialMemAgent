Microsoft Store Developer CLI (msstore) لنشر تطبيقات Windows على متجر Microsoft. استخدمه عندما يُطلب منك تكوين بيانات اعتماد المتجر، أو إدراج تطبيقات المتجر، أو التحقق من حالة الإرسال، أو نشر عمليات الإرسال، أو إدارة رحلات الحزمة، أو إعداد CI/CD لنشر المتجر، أو التكامل مع مركز الشركاء. يدعم تطبيقات Windows App SDK/WinUI وUWP و.NET MAUI وFlutter وElectron وReact Native وPWA.
