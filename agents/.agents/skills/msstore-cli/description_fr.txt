Microsoft Store Developer CLI (msstore) pour publier des applications Windows sur le Microsoft Store. À utiliser lorsque vous y êtes invité pour configurer les informations d'identification du Store, répertorier les applications du Store, vérifier l'état de la soumission, publier les soumissions, gérer les versions de package, configurer CI/CD pour la publication du Store ou intégrer l'Espace partenaires. Prend en charge les applications Windows App SDK/WinUI, UWP, .NET MAUI, Flutter, Electron, React Native et PWA.
