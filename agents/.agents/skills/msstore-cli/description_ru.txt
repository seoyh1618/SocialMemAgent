Интерфейс командной строки Microsoft Store Developer (msstore) для публикации приложений Windows в Microsoft Store. Используйте, когда вас попросят настроить учетные данные Магазина, составить список приложений Магазина, проверить статус отправки, опубликовать отправки, управлять тестами пакетов, настроить CI/CD для публикации в Магазине или интегрировать с Центром партнеров. Поддерживает приложения Windows App SDK/WinUI, UWP, .NET MAUI, Flutter, Electron, React Native и PWA.
