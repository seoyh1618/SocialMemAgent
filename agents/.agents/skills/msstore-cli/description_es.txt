CLI de desarrollador de Microsoft Store (msstore) para publicar aplicaciones de Windows en Microsoft Store. Úselo cuando se le solicite configurar las credenciales de la Tienda, enumerar las aplicaciones de la Tienda, verificar el estado de los envíos, publicar envíos, administrar vuelos de paquetes, configurar CI/CD para la publicación de la Tienda o integrar con el Centro de socios. Admite aplicaciones Windows App SDK/WinUI, UWP, .NET MAUI, Flutter, Electron, React Native y PWA.
