Dibuja 4 cartas del Tarot usando os.urandom() para inyectar entropía en la planificación cuando las indicaciones son vagas o poco especificadas. Interpreta la extensión para guiar los siguientes pasos. Úselo cuando el usuario se muestra indiferente, se siente afortunado, dice "deje que el destino decida", hace referencias a Yu-Gi-Oh ("corazón de las cartas"), demuestra indiferencia sobre el enfoque o dice "inténtelo de nuevo" en un sistema sin cambios. También se activa ante indicaciones suficientemente ambiguas donde múltiples enfoques son igualmente válidos.
