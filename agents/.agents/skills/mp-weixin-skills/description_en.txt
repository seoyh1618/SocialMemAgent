WeChat Official Account article management tool. Used for managing the publishing and updating of articles on WeChat Official Accounts. Use cases: when articles need to be published to the WeChat Official Account draft box; when existing draft content needs to be updated; when image assets need to be uploaded to the WeChat media library. Core functionality: uses AI to convert documents into WeChat Official Account HTML, then uses Python scripts to call the WeChat API to complete asset uploads and draft management.
