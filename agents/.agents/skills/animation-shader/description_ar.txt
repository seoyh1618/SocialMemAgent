اقرأ هذه المهارة عند تنفيذ أو تكوين تظليل نمط الرسوم المتحركة (Toon/Cel Shaders) - بما في ذلك الخطوط العريضة، أو إضاءة الحافة، أو تظليل Toon، أو MatCap، أو الانبعاث، أو الذوبان، أو التظليل، أو أي تأثير عرض منمق. يحتوي على أنماط محددة مسبقًا وتعيينات مرجعية لـ lilToon وPoiyomi وUTS2 وRToon وSToon وToonShadingCollection. يعمل كمكون إضافي للمعرفة بالمجال جنبًا إلى جنب مع مهارات سير العمل (OpenSpec، SpecKit) أو وضع التخطيط للوكيل.
