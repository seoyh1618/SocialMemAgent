Анализ тенденций и обнаружение культурных сигналов для новых новостей и поведения.
КОГДА ИСПОЛЬЗОВАТЬ: Исследование последних новостей (48 часов), выявление культурных/технологических/потребительских сдвигов до массового внедрения, анализ возникающих тенденций с помощью расширенного сбора информации.
ОСНОВНЫЕ ТРИГГЕРЫ: «coolhunt [тема]» = полный рабочий процесс исследования (5 шагов) «анализ тенденций» = глубокий анализ с методами выявления «сканирование новостей [тема]» = быстрый сбор новостей
РАБОЧИЙ ПРОЦЕСС: Запрос → Веб-исследование → Выбор информации → Анализ → Отчет.
ВЫХОД: отчет Markdown с заголовком, сводкой, проверкой фактов и поведенческим анализом, сохраненный в файле Coolhunter-output/report-{datetime}/{title}.md.
