새로운 뉴스와 행동에 대한 트렌드 인텔리전스와 문화적 신호를 감지합니다.
사용 시기: 최신 뉴스(48시간) 조사, 주류 채택 전 문화/기술/소비자 변화 식별, 고급 추출을 통해 새로운 트렌드 분석.
주요 트리거: "coolhunt [주제]" = 전체 연구 작업 흐름(5단계) "경향 분석" = 추출 방법을 사용한 심층 분석 "뉴스 스캔 [주제]" = 빠른 뉴스 수집
워크플로우: 요청 → 웹 조사 → 추출 선택 → 분석 → 보고서
출력: 제목, 요약, 사실 확인 및 행동 분석이 포함된 마크다운 보고서가 Coolhunter-output/report-{datetime}/{title}.md에 저장되었습니다.
