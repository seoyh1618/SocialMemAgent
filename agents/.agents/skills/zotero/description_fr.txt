Compétences en intégration API locale Zotero + API REST (API Web v3). Importation par lots de références depuis DOI, gestion de collection, recherche et listing d'articles. Via l'API REST, la création de collections, le CRUD d'éléments, la gestion des balises, le téléchargement de pièces jointes et les opérations de bibliothèque de groupes sont également possibles. À utiliser lorsque : (1) Enregistrez des références dans Zotero à partir d'une liste DOI, (2) Récupérez des listes de collections et d'éléments à partir de Zotero, (3) Importez des données BibTeX/RIS dans Zotero, (4) Recherchez des DOI de références à l'aide de l'API CrossRef, (5) Créez des collections, ajoutez des éléments et gérez des balises via l'API REST, (6) Exploitez des bibliothèques de groupe. Déclencheurs : "zotero", "文献登録", "DOI インポート", "コレクション", "論文追加", "文献管理"
