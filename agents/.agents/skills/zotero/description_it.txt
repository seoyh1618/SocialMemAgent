Competenze di integrazione API locale Zotero + API REST (API Web v3). Importazione batch di riferimenti da DOI, gestione delle collezioni, ricerca ed elenco degli articoli. Tramite l'API REST sono inoltre possibili la creazione di raccolte, l'elemento CRUD, la gestione dei tag, il caricamento di allegati di file e le operazioni di libreria di gruppo. Da utilizzare quando: (1) registrare i riferimenti in Zotero da un elenco DOI, (2) recuperare raccolte ed elenchi di elementi da Zotero, (3) importare dati BibTeX/RIS in Zotero, (4) cercare DOI di riferimenti utilizzando l'API CrossRef, (5) creare raccolte, aggiungere elementi e gestire tag tramite API REST, (6) utilizzare librerie di gruppo.
