Habilidades de integración de API local de Zotero + API REST (Web API v3). Importación por lotes de referencias desde DOI, gestión de colecciones, búsqueda y listado de artículos. A través de la API REST, también es posible la creación de colecciones, CRUD de elementos, administración de etiquetas, carga de archivos adjuntos y operaciones de biblioteca grupal. Úselo cuando: (1) Registre referencias en Zotero desde una lista de DOI, (2) Recuperar colecciones y listas de elementos de Zotero, (3) Importe datos BibTeX/RIS a Zotero, (4) Busque DOI de referencias usando CrossRef API, (5) Cree colecciones, agregue elementos y administre etiquetas a través de REST API, (6) Opera bibliotecas de grupo. Activadores: "zotero", "文献登録", "DOI インポート", "コレクション", "論文追加", "文献管理"
