مهارات التكامل Zotero Local API + REST API (Web API v3). استيراد دفعة من المراجع من DOI وإدارة المجموعة والبحث عن العناصر وإدراجها. من خلال REST API، من الممكن أيضًا إنشاء المجموعة، والعنصر CRUD، وإدارة العلامات، وتحميل مرفقات الملفات، وعمليات مكتبة المجموعة. يُستخدم عندما: (1) تسجيل المراجع في Zotero من قائمة DOI، (2) استرداد قوائم المجموعة والعناصر من Zotero، (3) استيراد بيانات BibTeX/RIS إلى Zotero، (4) البحث عن معرفات DOI للمراجع باستخدام CrossRef API، (5) إنشاء مجموعات وإضافة عناصر وإدارة العلامات عبر REST API، (6) تشغيل مكتبات المجموعة. المشغلات: "zotero"، "文献登録"، "DOI インポート"، "コレクション"، "論文追加"، "文献管理"
