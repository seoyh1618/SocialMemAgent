Zotero ローカル API + REST API (Web API v3) 統合スキル。 DOI からの参考文献のバッチインポート、コレクション管理、アイテムの検索とリスト。 REST API 経由で、コレクションの作成、アイテムの CRUD、タグ管理、添付ファイルのアップロード、グループ ライブラリの操作も可能です。 (1) DOI リストから Zotero に参考文献を登録する、(2) Zotero からコレクションとアイテムのリストを取得する、(3) BibTeX/RIS データを Zotero にインポートする、(4) CrossRef API を使用して参考文献の DOI を検索する、(5) REST API 経由でコレクションを作成、アイテムを追加、タグを管理する、(6) グループ ライブラリを操作する場合に使用します。トリガー: "zotero"、"文書登録"、"DOI インポート"、"コレクション"、"論文追加"、"文書管理"
