Zotero 本地 API + REST API（Web API v3）集成技能。批量导入 DOI 参考文献、馆藏管理、条目搜索和列表。通过 REST API，还可以进行集合创建、项目 CRUD、标签管理、文件附件上传和组库操作。在以下情况下使用：(1) 从 DOI 列表在 Zotero 中注册参考文献，(2) 从 Zotero 检索馆藏和项目列表，(3) 将 BibTeX/RIS 数据导入 Zotero，(4) 使用 CrossRef API 搜索参考文献的 DOI，(5) 通过 REST API 创建馆藏、添加项目和管理标签，(6) 操作组库。触发器: "zotero", "文献登录", "DOI インポート", "korekushon", "论文追加", "文献管理"
