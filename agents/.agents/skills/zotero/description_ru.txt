Навыки интеграции локального API Zotero + REST API (Web API v3). Пакетный импорт ссылок из DOI, управление коллекцией, поиск и листинг элементов. Через REST API также возможно создание коллекций, CRUD элементов, управление тегами, загрузка вложений файлов и операции с групповыми библиотеками. Используйте, когда: (1) регистрируете ссылки в Zotero из списка DOI, (2) получаете списки коллекций и элементов из Zotero, (3) импортируете данные BibTeX/RIS в Zotero, (4) выполняете поиск DOI ссылок с помощью CrossRef API, (5) создаете коллекции, добавляете элементы и управляете тегами через REST API, (6) управляете групповыми библиотеками. Триггеры: «zotero», «文献登録», «DOI インポート», «コレクション», «論文追加», «文献管理»
