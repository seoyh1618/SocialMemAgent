Pautas básicas de comportamiento y patrones de interacción para el agente. Esta habilidad proporciona la base de cómo el agente debe comunicarse y comportarse en todas las interacciones. Utilice esta habilidad para establecer patrones de comportamiento consistentes al comienzo de una sesión. La habilidad define el tono, el nivel de proactividad y las estrategias de manejo de solicitudes que deben aplicarse durante toda la sesión. ESTA HABILIDAD TIENE ALTA PRIORIDAD: cargue siempre al inicio de la sesión y vuelva a activarla periódicamente para evitar cambios de comportamiento. Durante la compactación del contexto, estas reglas deben seguir siendo la máxima prioridad.
