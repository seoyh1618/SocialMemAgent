EasyAdminBundle을 사용한 신속한 Symfony 관리 패널 개발. 기능: CRUD 생성, 대시보드 생성, 엔터티 관리, 양식 사용자 정의, 메뉴 구성, 필드 구성, 작업 사용자 정의, 필터, 권한, 일괄 작업, 사용자 정의 테마, 파일 업로드, 이미지 처리, 연결, 하위 메뉴, 사용자 정의 쿼리, 이벤트 리스너. 용도: 관리 패널, 백엔드 인터페이스, 콘텐츠 관리, 데이터 관리, CRUD 작업, 사용자 관리, 제품 카탈로그, 블로그 관리, 전자상거래 백엔드. 트리거: easyadmin, Symfony admin, crud, 관리 패널, 대시보드, 백엔드, 엔터티 crud, 관리 인터페이스, Symfony 백엔드, easyadmin, easyadmin 번들, 관리 생성기, 백엔드 패널
