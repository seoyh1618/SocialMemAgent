Stellen Sie Anwendungen mithilfe des bjw-s/app-template Helm-Diagramms bereit – einem flexiblen Diagramm zur Helmifizierung von Container-Images ohne dedizierte Diagramme.

Wird verwendet, wenn: (1) Container-Images bereitgestellt werden, denen offizielle Helm-Charts fehlen, (2) HelmRelease-Manifeste für Flux GitOps erstellt werden,
(3) Konfigurieren von Multi-Container-Pods mit Sidecars, (4) Einrichten von persistentem Speicher, Eingang und Diensten für benutzerdefinierte Anwendungen,
(5) Fragen zur Struktur oder Mustern von App-Template-Werten, (6) Bereitstellung eines benutzerdefinierten Containers in Kubernetes.

Auslöser: „Mit App-Vorlage bereitstellen“, „Dieses Image helmifizieren“, „Helm-Release erstellen für“, „App-Vorlagenwerte“,
