Развертывайте приложения с помощью bjw-s/app-template Helm Chart — гибкой диаграммы для объединения образов контейнеров без специальных диаграмм.

Используйте, когда: (1) развертывание образов контейнеров, в которых отсутствуют официальные диаграммы Helm, (2) создание манифестов HelmRelease для Flux GitOps,
(3) Настройка модулей с несколькими контейнерами с дополнительными модулями, (4) Настройка постоянного хранилища, входа и сервисов для пользовательских приложений,
(5) Вопросы о структуре или шаблонах значений шаблона приложения. (6) Развертывание любого пользовательского контейнера в Kubernetes.
