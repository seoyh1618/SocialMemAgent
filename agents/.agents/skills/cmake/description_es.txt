Habilidad del sistema de compilación CMake para proyectos C/C++. Úselo al escribir o refactorizar CMakeLists.txt, configurar compilaciones fuera del código fuente, seleccionar generadores (Ninja, Make, VS), administrar objetivos y dependencias con target_link_libraries, integrar paquetes externos a través de find_package o FetchContent, habilitar desinfectantes, configurar archivos de cadena de herramientas para compilación cruzada o exportar paquetes CMake. Se activa ante consultas sobre CMakeLists.txt, errores de configuración de cmake, propiedades de destino, reglas de instalación, CPack o ajustes preestablecidos de CMake.
