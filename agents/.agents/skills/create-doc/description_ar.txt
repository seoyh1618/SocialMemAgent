قم بإنشاء وثائق تتبع اصطلاحات المستندات كرمز. يتم تشغيله عندما: يطلب المستخدم إنشاء أو كتابة أو صياغة الوثائق أو الرسوم البيانية أو المستندات الفنية. أمثلة: "إنشاء مستند لـ..."، "كتابة وثائق الهندسة المعمارية"، "إضافة مخطط C4"، "توثيق نموذج المجال". لا تقم بالتشغيل عندما: يريد المستخدم ADR (استخدم create-adr)، أو المواصفات (استخدم create-spec)، أو يريد فقط قراءة المستندات الموجودة.
