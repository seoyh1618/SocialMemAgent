Cree documentación siguiendo las convenciones de documentos como código. DISPARAR cuando: el usuario solicita crear, escribir o redactar documentación, diagramas o documentos técnicos. Ejemplos: "crear un documento para...", "escribir documentación de arquitectura", "agregar un diagrama C4", "documentar el modelo de dominio". NO DISPARE cuando: el usuario quiera un ADR (use create-adr), una especificación (use create-spec) o simplemente quiera leer documentos existentes.
