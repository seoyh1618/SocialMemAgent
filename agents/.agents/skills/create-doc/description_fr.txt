Créez une documentation en suivant les conventions docs-as-code. DÉCLENCHEMENT lorsque : l'utilisateur demande à créer, rédiger ou rédiger une documentation, des diagrammes ou des documents techniques. Exemples : "créer un document pour...", "écrire la documentation de l'architecture", "ajouter un diagramme C4", "documenter le modèle de domaine". NE PAS DÉCLENCHER lorsque : l'utilisateur souhaite un ADR (utilisez create-adr), une spécification (utilisez create-spec) ou souhaite simplement lire des documents existants.
