Registrador JSON de alto rendimiento de Pino para Node.js con transportes de subprocesos de trabajo, registradores secundarios, redacción e integraciones de marcos. Úselo al configurar el registro estructurado, configurar el transporte de registros, agregar ID de correlación de solicitudes, redactar datos confidenciales o integrar con Fastify, Hono o Express. Úselo para pino, registro, registros estructurados, ID de solicitud, correlación, redacción, transportes, pino-http, pino-pretty.
