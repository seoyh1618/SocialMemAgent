مسجل JSON عالي الأداء من Pino لـ Node.js مع عمليات نقل سلاسل العمليات، ومسجلات البيانات الفرعية، والتنقيح، وتكامل إطار العمل. يُستخدم عند إعداد التسجيل المنظم، أو تكوين عمليات نقل السجل، أو إضافة معرفات ارتباط الطلب، أو تنقيح البيانات الحساسة، أو التكامل مع Fastify، أو Hono، أو Express. يستخدم لـ pino، والتسجيل، والسجلات المنظمة، ومعرف الطلب، والارتباط، والتنقيح، ووسائل النقل، وpino-http، وpino-pretty.
