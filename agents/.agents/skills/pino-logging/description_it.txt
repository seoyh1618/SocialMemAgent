Logger JSON ad alte prestazioni Pino per Node.js con trasporti di thread di lavoro, logger figlio, redazione e integrazioni di framework. Da utilizzare quando si imposta la registrazione strutturata, si configura il trasporto dei log, si aggiungono ID di correlazione delle richieste, si oscurano dati sensibili o si integra con Fastify, Hono o Express. Utilizzare per pino, registrazione, registri strutturati, ID richiesta, correlazione, redazione, trasporti, pino-http, pino-pretty.
