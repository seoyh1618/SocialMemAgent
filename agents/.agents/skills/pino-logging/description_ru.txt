Высокопроизводительный регистратор JSON Pino для Node.js с транспортом рабочих потоков, дочерними регистраторами, редактированием и интеграцией с инфраструктурой. Используйте при настройке структурированного ведения журналов, настройке транспорта журналов, добавлении идентификаторов корреляции запросов, редактировании конфиденциальных данных или интеграции с Fastify, Hono или Express. Используйте для пино, ведения журнала, структурированных журналов, идентификатора запроса, корреляции, редактирования, транспорта, пино-http, пино-довольно.
