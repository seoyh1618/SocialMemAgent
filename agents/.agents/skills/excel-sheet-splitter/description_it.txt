Dividere una cartella di lavoro Excel in file Excel separati per foglio di lavoro, in modo che ogni foglio di lavoro diventi un singolo file. Casi d'uso: (1) dividere un file Excel composto da più fogli in file separati, (2) estrarre un foglio di lavoro specifico come file autonomo, (3) distribuire fogli di lavoro da una cartella di lavoro unita, (4) creare copie di fogli di lavoro per un'elaborazione o distribuzione separata.
