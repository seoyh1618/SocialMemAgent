Teilen Sie eine Excel-Arbeitsmappe nach Arbeitsblatt in separate Excel-Dateien auf und erstellen Sie so eine einzelne Datei pro Arbeitsblatt. Geeignete Szenarien: (1) Aufteilen einer Excel-Datei mit mehreren Blättern in separate Dateien, (2) Extrahieren eines bestimmten Arbeitsblatts als eigenständige Datei, (3) Verteilen von Arbeitsblättern aus einer kombinierten Arbeitsmappe, (4) Erstellen von Arbeitsblattkopien für die separate Verarbeitung oder Verteilung.
