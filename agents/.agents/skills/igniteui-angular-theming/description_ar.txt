قم بتخصيص تصميم مكونات Ignite UI باستخدام خادم MCP الخاص بـ igniteui-theming من أجل التصميم بمساعدة الذكاء الاصطناعي. يُستخدم عندما يطلب المستخدمون تصميم سمة أو إعادة تصميم مكونات Ignite UI، أو تغيير لوحة الألوان، أو التبديل بين السمات الفاتحة والداكنة، أو تطبيق أو إنشاء سمة عامة، أو تخصيص الطباعة أو الارتفاعات، أو تكوين أنماط على مستوى المكونات.
