Personalice el estilo de los componentes de la interfaz de usuario de Ignite utilizando el servidor MCP de igniteui-theming para temas asistidos por IA. Úselo cuando los usuarios soliciten crear un tema o cambiar el estilo de los componentes de la interfaz de usuario de Ignite, cambiar la paleta de colores, cambiar entre temas claros y oscuros, aplicar o generar un tema global, personalizar la tipografía o las elevaciones, o configurar estilos a nivel de componente.
