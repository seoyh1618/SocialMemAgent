Используйте при запуске backtesting.py для результатов ClickHouse SQL, настройке конструктора Backtest(), обработке перекрывающихся сделок, многопозиционном режиме, обработке NaN скользящего квантиля, сортировке сделок или проверке ворот Oracle.
