Прозрачное, тщательное исследование с полной методологией, а не оболочка API «черного ящика». Проводит исчерпывающее расследование посредством обязательных двухцикловых исследований по каждой теме, 7-го цитирования APA, иерархии доказательств и 3 контрольных точек для пользователей. Автономный с использованием собственных инструментов OpenClaw (web_search, web_fetch, session_spawn). Используйте для обзоров литературы, конкурентной разведки или любых исследований, требующих академической строгости и воспроизводимости.
