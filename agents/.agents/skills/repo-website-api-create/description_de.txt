Erstellen Sie neue API-Referenzseiten für die Valibot-Website unter website/src/routes/api/. Verwenden Sie diese Option, wenn Sie Dokumentation für neue Schemata, Aktionen, Methoden oder Typen hinzufügen. Behandelt das Lesen von Quellcode, das Erstellen von Properties.ts- und index.mdx-Dateien, das Aktualisieren von menu.md und das Vergleichen verwandter APIs.
