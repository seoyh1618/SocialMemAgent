Genere o actualice PROJECT_MAP.md para una carpeta especificada por el usuario. Adecuado cuando el usuario solicita un mapa de directorio/mapa de proyecto/descripción general del repositorio/descripción a nivel de carpeta/actualización de un PROJECT_MAP.md existente. Primero debe preguntar qué carpeta(s) escanear; el análisis del repositorio completo está prohibido de forma predeterminada; admite directorios únicos o múltiples (generación fusionada o separada).
