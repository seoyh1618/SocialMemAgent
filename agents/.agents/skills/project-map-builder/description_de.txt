Generieren oder aktualisieren Sie PROJECT_MAP.md für einen vom Benutzer angegebenen Ordner. Geeignet, wenn der Benutzer eine Verzeichniszuordnung/Projektzuordnung/Repository-Übersicht/Beschreibung auf Ordnerebene/Aktualisierung einer vorhandenen PROJECT_MAP.md anfordert. Sie müssen zunächst fragen, welche Ordner gescannt werden sollen. Das Scannen des gesamten Repositorys ist standardmäßig verboten. unterstützt einzelne oder mehrere Verzeichnisse (zusammengeführte oder separate Generierung).
