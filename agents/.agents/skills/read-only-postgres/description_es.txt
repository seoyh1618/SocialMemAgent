Ejecute consultas SQL de solo lectura en bases de datos PostgreSQL. Úselo cuando: (1) consulte datos de PostgreSQL, (2) explore esquemas/tablas, (3) ejecute consultas SELECT para análisis, (4) verifique el contenido de la base de datos. Admite múltiples conexiones de bases de datos con descripciones para selección automática. Bloquea todas las operaciones de escritura (INSERT, UPDATE, DELETE, DROP, etc.) por seguridad.
