تنفيذ استعلامات SQL للقراءة فقط مقابل قواعد بيانات PostgreSQL. يُستخدم عندما: (1) الاستعلام عن بيانات PostgreSQL، (2) استكشاف المخططات/الجداول، (3) تشغيل استعلامات SELECT للتحليل، (4) التحقق من محتويات قاعدة البيانات. يدعم اتصالات قاعدة بيانات متعددة مع أوصاف للاختيار التلقائي. يحظر جميع عمليات الكتابة (INSERT، UPDATE، DELETE، DROP، وما إلى ذلك) من أجل السلامة.
