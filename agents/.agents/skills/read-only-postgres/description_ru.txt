Выполнение SQL-запросов только для чтения к базам данных PostgreSQL. Используйте, когда: (1) запрашиваете данные PostgreSQL, (2) изучаете схемы/таблицы, (3) выполняете запросы SELECT для анализа, (4) проверяете содержимое базы данных. Поддерживает несколько подключений к базе данных с описаниями для автоматического выбора. Блокирует все операции записи (INSERT, UPDATE, DELETE, DROP и т. д.) в целях безопасности.
