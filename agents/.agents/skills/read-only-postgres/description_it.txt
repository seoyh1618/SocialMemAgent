Esegui query SQL di sola lettura su database PostgreSQL. Da utilizzare quando: (1) si eseguono query su dati PostgreSQL, (2) si esplorano schemi/tabelle, (3) si eseguono query SELECT per l'analisi, (4) si controlla il contenuto del database. Supporta più connessioni al database con descrizioni per la selezione automatica. Blocca tutte le operazioni di scrittura (INSERT, UPDATE, DELETE, DROP, ecc.) per sicurezza.
