Führen Sie schreibgeschützte SQL-Abfragen für PostgreSQL-Datenbanken aus. Wird verwendet, wenn: (1) PostgreSQL-Daten abgefragt werden, (2) Schemata/Tabellen untersucht werden, (3) SELECT-Abfragen zur Analyse ausgeführt werden, (4) Datenbankinhalte überprüft werden. Unterstützt mehrere Datenbankverbindungen mit Beschreibungen für die automatische Auswahl. Blockiert aus Sicherheitsgründen alle Schreibvorgänge (INSERT, UPDATE, DELETE, DROP usw.).
