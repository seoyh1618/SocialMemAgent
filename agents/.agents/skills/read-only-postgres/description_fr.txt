Exécutez des requêtes SQL en lecture seule sur les bases de données PostgreSQL. À utiliser lorsque : (1) interroger des données PostgreSQL, (2) explorer des schémas/tables, (3) exécuter des requêtes SELECT pour l'analyse, (4) vérifier le contenu de la base de données. Prend en charge plusieurs connexions de bases de données avec des descriptions pour la sélection automatique. Bloque toutes les opérations d'écriture (INSERT, UPDATE, DELETE, DROP, etc.) pour des raisons de sécurité.
