Entwerfen und generieren Sie Amazon Connect-Kontaktfluss-JSON aus Anforderungen oder Diagrammen. Wird verwendet, wenn: (1) ein neuer Kontaktablauf von Grund auf entworfen wird, indem Anforderungen erfasst werden, (2) draw.io/Mermaid-Diagramme oder -Bilder in Fluss-JSON konvertiert werden, (3) Abläufe in AWS über CLI bereitgestellt werden, (4) vorhandene Abläufe aktualisiert werden. Deckt IVR-Menüs, Warteschlangenrouting, Lambda-Integration, Geschäftszeitenprüfungen und Flow-Module ab.
