Семинар по штурму событий, смоделированный с помощью искусственного интеллекта, с поддержкой нескольких человек. Используйте при обнаружении событий предметной области, команд, субъектов и ограниченных контекстов. Поддерживает три режима — полное моделирование (обсуждение пяти агентов), быстрый (однопроходной анализ) и управляемый (интерактивный с пользователем). Управляет персональными агентами и синтезирует результаты.
