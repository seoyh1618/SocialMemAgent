Создавайте децентрализованные приложения по протоколу NEAR. Используется для: (1) создания новых приложений NEAR dApp с помощью «create-near-app» (Vite+React, Next.js), (2) добавления подключения к кошельку NEAR к существующим приложениям с помощью «@hot-labs/near-connect» и «near-connect-hooks», (3) создания пользовательского интерфейса для смарт-контрактов NEAR, (4) интеграции входа/выхода из кошелька, вызовов контрактов и подписания транзакций в веб-приложения.
