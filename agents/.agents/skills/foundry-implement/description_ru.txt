Навыки реализации задач для рабочих процессов, основанных на спецификациях. Считывает спецификации, определяет следующие задачи, требующие действий, и создает подробные планы выполнения. Используйте, когда готовы реализовать задачу из существующей спецификации — устраняет разрыв между планированием и кодированием.
