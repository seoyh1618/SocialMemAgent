Операции с электронными таблицами Feishu: создание таблиц, чтение/запись ячеек, API форматированного текста V3, операции со строками/столбцами, настройки стиля, поиск и замена, изображения ячеек, защита рабочего листа. Поддерживает API простых данных V2 и API расширенного текста V3. Используйте, когда пользователям необходимо манипулировать таблицами Feishu, обрабатывать данные в пакетном режиме или устанавливать стили.
