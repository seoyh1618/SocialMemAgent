Feishu-Tabellenoperationen: Tabellen erstellen, Zellen lesen/schreiben, V3-Rich-Text-API, Zeilen- und Spaltenoperationen, Stileinstellungen, Suchen und Ersetzen, Zellbilder, Arbeitsblattschutz. Unterstützt sowohl einfache Daten-APIs der Version 2 als auch Rich-Text-APIs der Version 3. Wird verwendet, wenn Benutzer Feishu-Tabellen manipulieren, Daten stapelweise verarbeiten oder Stile festlegen müssen.
