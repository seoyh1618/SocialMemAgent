Opérations de feuille de calcul Feishu : création de tableaux, lecture/écriture de cellules, API de texte enrichi V3, opérations de ligne/colonne, paramètres de style, recherche et remplacement, images de cellules, protection des feuilles de calcul. Prend en charge les API de données simples V2 et de texte enrichi V3. À utiliser lorsque les utilisateurs doivent manipuler des feuilles Feishu, traiter des données par lots ou définir des styles.
