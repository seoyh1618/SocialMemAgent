Operazioni sui fogli di calcolo Feishu: creazione di tabelle, lettura/scrittura di celle, API rich text V3, operazioni su righe/colonne, impostazioni di stile, ricerca e sostituzione, immagini di celle, protezione del foglio di lavoro. Supporta sia API dati semplici V2 che API rich text V3. Utilizzare quando gli utenti devono manipolare fogli Feishu, elaborare dati in batch o impostare stili.
