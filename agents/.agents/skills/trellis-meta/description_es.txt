Metahabilidad para comprender y personalizar Mindfold Trellis: el sistema de flujo de trabajo de IA para Claude Code y Cursor. Esta habilidad documenta el diseño ORIGINAL del sistema Trellis. Cuando los usuarios personalizan su instalación de Trellis, las modificaciones deben registrarse en una habilidad "trellis-local" local del proyecto, NO en esta meta-habilidad. Utilice esta habilidad cuando: (1) comprenda la arquitectura de Trellis, (2) personalice los flujos de trabajo de Trellis, (3) agregue comandos/agentes/enganches, (4) solucione problemas o (5) adapte Trellis a proyectos específicos.
