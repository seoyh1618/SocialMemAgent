Méta-compétence pour comprendre et personnaliser Mindfold Trellis - le système de workflow d'IA pour Claude Code et Cursor. Cette compétence documente la conception ORIGINALE du système Trellis. Lorsque les utilisateurs personnalisent leur installation Trellis, les modifications doivent être enregistrées dans une compétence « trellis-local » locale du projet, PAS dans cette méta-compétence. Utilisez cette compétence pour : (1) comprendre l'architecture de Trellis, (2) personnaliser les flux de travail de Trellis, (3) ajouter des commandes/agents/hooks, (4) résoudre des problèmes ou (5) adapter Trellis à des projets spécifiques.
