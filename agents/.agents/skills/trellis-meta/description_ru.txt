Мета-навык для понимания и настройки Mindfold Trellis — системы рабочих процессов искусственного интеллекта для Claude Code и Cursor. Этот навык документирует ОРИГИНАЛЬНЫЙ дизайн системы Trellis. Когда пользователи настраивают свою установку Trellis, изменения должны быть записаны в локальном навыке `trellis-local`, а НЕ в этом мета-навыке. Используйте этот навык, когда: (1) понимаете архитектуру Trellis, (2) настраиваете рабочие процессы Trellis, (3) добавляете команды/агенты/перехватчики, (4) устраняете проблемы или (5) адаптируете Trellis к конкретным проектам.
