المهارات الفوقية لفهم وتخصيص Mindfold Trellis - نظام سير عمل الذكاء الاصطناعي لـ Claude Code وCursor. توثق هذه المهارة تصميم نظام التعريشة الأصلي. عندما يقوم المستخدمون بتخصيص تثبيت الشبكة الخاصة بهم، يجب تسجيل التعديلات في مهارة "التعريشة المحلية" الخاصة بالمشروع المحلي، وليس في هذه المهارة الوصفية. استخدم هذه المهارة عندما: (1) فهم بنية الشبكة، أو (2) تخصيص مسارات عمل الشبكة، أو (3) إضافة أوامر/وكلاء/خطافات، أو (4) استكشاف المشكلات وإصلاحها، أو (5) تكييف الشبكة مع مشاريع محددة.
