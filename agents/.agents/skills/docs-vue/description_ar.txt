ابحث عن المستندات والأدلة وصفحات التكوين المرجعية لـ CE.SDK Vue.js.

يُستخدم عندما يحتاج المستخدم إلى مستندات CE.SDK لـ Vue.js - التكوين، وتخصيص واجهة المستخدم،
خيارات التصدير أو أدلة الميزات أو تعليمات البدء. تم تشغيله أيضًا بواسطة "IMG.LY"، "CreativeEditor"،
"CE.SDK" أو "cesdk" عندما يحتاج المستخدم إلى صفحة مستند Vue.js موجودة.

ليس لكتابة التعليمات البرمجية (استخدم البناء) أو شرح المفاهيم (استخدم الشرح).

<مثال>
السياق: يسأل المستخدم عن تكوين Vue.js
المستخدم: "كيف أقوم بتكوين المحرر في Vue.js؟"
المساعد: "سأستخدم /cesdk:docs-vue للبحث عن خيارات التكوين."
</مثال>

<مثال>
السياق: يسأل المستخدم عن تسجيل البرنامج المساعد Vue
المستخدم: "كيف يمكنني تسجيل CE.SDK كمكون إضافي لـ Vue؟"
المساعد: "اسمح لي باستخدام /cesdk:docs-vue للعثور على الوثائق ذات الصلة."
</مثال>
