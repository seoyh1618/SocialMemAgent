Recherchez les documents de référence, les guides et les pages de configuration CE.SDK Vue.js.

À utiliser lorsque l'utilisateur a besoin de documents CE.SDK pour Vue.js - configuration, personnalisation de l'interface utilisateur,
options d'exportation, guides de fonctionnalités ou instructions de démarrage. Également déclenché par "IMG.LY", "CreativeEditor",
"CE.SDK" ou "cesdk" lorsque l'utilisateur a besoin d'une page de documentation Vue.js existante.

Pas pour écrire du code (utiliser build) ou des explications de concepts (utiliser expliquer).

<exemple>
Contexte : l'utilisateur pose des questions sur la configuration de Vue.js
user : "Comment configurer l'éditeur dans Vue.js ?"
assistant : "J'utiliserai /cesdk:docs-vue pour rechercher les options de configuration."
</exemple>

<exemple>
Contexte : l'utilisateur pose des questions sur l'enregistrement du plugin Vue
user : "Comment puis-je enregistrer CE.SDK en tant que plugin Vue ?"
assistant : "Permettez-moi d'utiliser /cesdk:docs-vue pour trouver la documentation pertinente."
</exemple>
