Найдите справочную документацию, руководства и страницы конфигурации CE.SDK Vue.js.

Используйте, когда пользователю нужна документация CE.SDK для Vue.js — настройка, настройка пользовательского интерфейса,
параметры экспорта, руководства по функциям или инструкции по началу работы. Также запускается "IMG.LY", "CreativeEditor",
«CE.SDK» или «cesdk», если пользователю нужна существующая страница документа Vue.js.

Не для написания кода (используйте сборку) или объяснения концепций (используйте объяснение).

<пример>
Контекст: пользователь спрашивает о конфигурации Vue.js.
пользователь: «Как мне настроить редактор в Vue.js?»
помощник: «Я буду использовать /cesdk:docs-vue для поиска параметров конфигурации».
</пример>

<пример>
Контекст: пользователь спрашивает о регистрации плагина Vue.
пользователь: «Как зарегистрировать CE.SDK как плагин Vue?»
помощник: «Позвольте мне использовать /cesdk:docs-vue, чтобы найти соответствующую документацию».
</пример>
