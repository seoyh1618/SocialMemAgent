CE.SDK Vue.js のリファレンス ドキュメント、ガイド、構成ページを検索してください。

ユーザーが Vue.js の CE.SDK ドキュメントを必要とする場合 (構成、UI カスタマイズ、
エクスポート オプション、機能ガイド、または開始手順。 「IMG.LY」、「CreativeEditor」によってもトリガーされます。
「CE.SDK」、またはユーザーが既存の Vue.js ドキュメント ページを必要とする場合は「cesdk」。

コードの作成 (build を使用) や概念の説明 (explain を使用) のためではありません。

<例>
コンテキスト: ユーザーが Vue.js の設定について尋ねる
ユーザー: 「Vue.js でエディタを設定するにはどうすればよいですか?」
アシスタント: 「/cesdk:docs-vue を使用して構成オプションを調べます。」
</例>

<例>
コンテキスト: ユーザーが Vue プラグインの登録について尋ねる
ユーザー: 「CE.SDK を Vue プラグインとして登録するにはどうすればよいですか?」
アシスタント: 「関連するドキュメントを見つけるには /cesdk:docs-vue を使用させてください。」
</例>
