opzioni di esportazione, guide alle funzionalità o istruzioni introduttive. Attivato anche da "IMG.LY", "CreativeEditor",
"CE.SDK" o "cesdk" quando l'utente necessita di una pagina di documenti Vue.js esistente.

Non per scrivere codice (usa build) o spiegazioni di concetti (usa spiega).
