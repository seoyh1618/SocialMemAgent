Руководства по использованию Gemini API в Google Cloud Vertex AI с Gen AI SDK. Используйте, когда пользователь спрашивает об использовании Gemini в корпоративной среде или явно упоминает Vertex AI. Охватывает использование SDK (Python, JS/TS, Go, Java, C#), такие возможности, как Live API, инструменты, генерацию мультимедиа, кэширование и пакетное прогнозирование.
