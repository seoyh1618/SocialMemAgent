Leitet die Verwendung der Gemini API auf Google Cloud Vertex AI mit dem Gen AI SDK. Verwenden Sie diese Option, wenn der Benutzer nach der Verwendung von Gemini in einer Unternehmensumgebung fragt oder Vertex AI ausdrücklich erwähnt. Behandelt die SDK-Nutzung (Python, JS/TS, Go, Java, C#), Funktionen wie Live API, Tools, Multimedia-Generierung, Caching und Batch-Vorhersage.
