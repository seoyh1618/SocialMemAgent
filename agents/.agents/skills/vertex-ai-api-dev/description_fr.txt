Guide l'utilisation de l'API Gemini sur Google Cloud Vertex AI avec le SDK Gen AI. À utiliser lorsque l'utilisateur demande comment utiliser Gemini dans un environnement d'entreprise ou mentionne explicitement Vertex AI. Couvre l'utilisation du SDK (Python, JS/TS, Go, Java, C#), des fonctionnalités telles que l'API Live, les outils, la génération multimédia, la mise en cache et la prédiction par lots.
