Создавайте, реорганизуйте и переносите сценарии Bash с помощью Octivi Bash Boilerplate (OBB), используя блоки маркеров (`# >>> OBB:BEGIN ...` / `# <<< OBB:END`). Используйте при выборе «заголовка» или «полной» интеграции, копировании готовых шаблонов, обновлении блоков маркеров и проверке качества сценария с помощью детерминированных проверок.
