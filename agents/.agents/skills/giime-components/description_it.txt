【Obbligatorio · Richiesto】 Linee guida di utilizzo obbligatorie per la libreria dei componenti Giime. È necessario leggere questa Skill prima di scrivere o modificare qualsiasi codice front-end che coinvolga componenti dell'interfaccia utente. Copre le funzionalità avanzate dei componenti gm-*, l'utilizzo di plugin come GmConfirmBox/GmMessage/GmCopy, modelli di codice comuni e come accedere alla documentazione MCP.
