Git erstellt einen MR. Verwenden Sie diese Fähigkeit, wenn der Benutzer die Erstellung eines MR, eine Zusammenführungsanforderung oder die Zusammenführung in einer beliebigen Umgebung/Zweig oder für einen MR-Erstellungsvorgang anfordert. Gilt auch, wenn der Benutzer „MR erstellen“, „Zusammenführungsanforderung senden“, „Zum Testen zusammenführen“, „Zur Entwicklung zusammenführen“, „Zur Produktion zusammenführen“, „Zur Testumgebung zusammenführen“, „Zur Entwicklungsumgebung zusammenführen“, „Zur Staging-Umgebung zusammenführen“ und ähnliche umgangssprachliche Ausdrücke sagt.
