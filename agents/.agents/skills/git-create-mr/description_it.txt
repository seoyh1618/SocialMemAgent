Git crea un MR. Utilizza questa abilità quando l'utente chiede di creare una MR, una richiesta di unione o qualsiasi operazione che coinvolga la creazione di MR. Si applica anche quando l'utente dice "crea un MR", "invia una richiesta di unione", "unisci per testare", "unisci per sviluppare", "unisci in produzione", "unisci in ambiente di test", "unisci in ambiente di sviluppo", "unisci in ambiente di staging" o espressioni colloquiali simili.
