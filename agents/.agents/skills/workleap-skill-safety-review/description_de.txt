Bewerten Sie die Fähigkeiten von Drittanbieter-Agenten vor der Einführung oder Aktualisierung auf Sicherheitsrisiken. Verwenden Sie diese Option, wenn: (1) eine Fertigkeit aus skills.sh, ClawHub oder einem anderen öffentlichen Register installiert oder aktualisiert wird, (2) Fertigkeiten auf Sicherheitsrisiken überprüft oder PRs überprüft werden, die Fertigkeitsabhängigkeiten hinzufügen/aktualisieren, (3) eine Team-/Organisationszulassungsliste mit genehmigten Fertigkeiten erstellt wird, (4) verdächtiges Verhalten einer Fertigkeit untersucht oder die Frage „Ist diese Fertigkeit sicher?“ beantwortet wird. / „Sollten wir diese Fähigkeit übernehmen?“
