Test de l'interface utilisateur Web et automatisation du navigateur avec le navigateur d'agent Vercel. À utiliser lorsque les tâches nécessitent d'ouvrir des pages, d'interagir avec des formulaires, de valider des flux d'interface utilisateur, de prendre des captures d'écran, d'extraire des données de page ou d'exécuter des vérifications répétables basées sur un navigateur localement ou dans CI.
