Recherchez et explorez les API de bureau Windows. À utiliser lors de la création de fonctionnalités nécessitant des fonctionnalités de plate-forme : caméra, accès aux fichiers, notifications, contrôles de l'interface utilisateur, IA/ML, capteurs, mise en réseau, etc. Découvrez l'API appropriée pour une tâche et récupérez les détails complets du type (méthodes, propriétés, événements, valeurs d'énumération).
