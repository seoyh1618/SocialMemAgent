ابحث عن واجهات برمجة تطبيقات سطح مكتب Windows واستكشفها. يُستخدم عند إنشاء ميزات تحتاج إلى إمكانات النظام الأساسي - الكاميرا، والوصول إلى الملفات، والإشعارات، وعناصر تحكم واجهة المستخدم، وAI/ML، وأجهزة الاستشعار، والشبكات، وما إلى ذلك. يكتشف واجهة برمجة التطبيقات المناسبة لمهمة ما ويسترجع تفاصيل النوع الكامل (الطرق، والخصائص، والأحداث، وقيم التعداد).
