Найдите и изучите API-интерфейсы настольных компьютеров Windows. Используйте при создании функций, требующих возможностей платформы — камеры, доступа к файлам, уведомлений, элементов управления пользовательского интерфейса, искусственного интеллекта и машинного обучения, датчиков, сетей и т. д. Находит подходящий API для задачи и получает полную информацию о типах (методы, свойства, события, значения перечисления).
