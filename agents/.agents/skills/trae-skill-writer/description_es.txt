Cree habilidades de Trae IDE (archivos SKILL.md) para capacidades de IA reutilizables. Úselo cuando el usuario quiera: crear una habilidad, crear un flujo de trabajo reutilizable, automatizar tareas repetitivas, convertir una conversación en una habilidad o encapsular un proceso para que lo siga la IA. Activadores en: '创建 habilidad', 'escribir un SKILL.md', 'hacer esto reutilizable', '.trae/skills/', 'Sigo haciendo lo mismo cada vez'. NO lo use para reglas (use trae-rules-writer) o agentes (use trae-agent-writer).
