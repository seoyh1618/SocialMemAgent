Erstellen Sie Trae-IDE-Fähigkeiten (SKILL.md-Dateien) für wiederverwendbare KI-Funktionen. Verwenden Sie diese Option, wenn der Benutzer Folgendes möchte: eine Fertigkeit erstellen, einen wiederverwendbaren Workflow erstellen, sich wiederholende Aufgaben automatisieren, eine Konversation in eine Fertigkeit umwandeln oder einen Prozess kapseln, dem die KI folgen soll. Wird ausgelöst durch: „Fähigkeit verbessern“, „SKILL.md schreiben“, „diese wiederverwendbar machen“, „.trae/skills/“, „Ich mache jedes Mal das Gleiche“. NICHT für Regeln (verwenden Sie trae-rules-writer) oder Agenten (verwenden Sie trae-agent-writer) verwenden.
