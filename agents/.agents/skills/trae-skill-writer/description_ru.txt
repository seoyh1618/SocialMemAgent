Создайте навыки Trae IDE (файлы SKILL.md) для многократного использования возможностей ИИ. Используйте, когда пользователь хочет: создать навык, создать многократно используемый рабочий процесс, автоматизировать повторяющиеся задачи, превратить разговор в навык или инкапсулировать процесс, которому будет следовать ИИ. Срабатывает при: «Навык 创建», «Написать SKILL.md», «сделать это многоразовым», «.trae/skills/», «Я продолжаю делать одно и то же каждый раз». НЕ используйте для правил (используйте trae-rules-writer) или агентов (используйте trae-agent-writer).
