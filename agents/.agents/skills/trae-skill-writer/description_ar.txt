قم بإنشاء مهارات Trae IDE (ملفات SKILL.md) لإمكانيات الذكاء الاصطناعي القابلة لإعادة الاستخدام. يُستخدم عندما يريد المستخدم: إنشاء مهارة، أو إنشاء سير عمل قابل لإعادة الاستخدام، أو أتمتة المهام المتكررة، أو تحويل محادثة إلى مهارة، أو تغليف عملية ليتبعها الذكاء الاصطناعي. مشغلات على: 'مهارة 创建'، 'اكتب SKILL.md'، 'اجعل هذا قابلاً لإعادة الاستخدام'، '.trae/skills/'، 'أستمر في فعل نفس الشيء في كل مرة'. لا تستخدم للقواعد (استخدم كاتب قواعد Trae) أو الوكلاء (استخدم كاتب Trae-Agent).
