Обогатите тему урока метаданными Teaching_guide, которые помогут учителям эффективно доставлять контент. Читает предыдущий урок для контекста прогресса. Работает с одним уроком, полной главой, несколькими главами или N конкретными уроками. Используйте при подготовке уроков для учителей, добавлении педагогических указаний или дополнении темы ключевыми моментами, заблуждениями, подсказками для обсуждения, советами по обучению и быстрой_проверкой.
