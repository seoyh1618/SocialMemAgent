Gestione lineare del progetto tramite API Composio. Utilizzare quando:
(1) Elencare, creare o aggiornare i problemi Linear
(2) Ottenere team, utenti o stati del flusso di lavoro
(3) Gestire progetti o aggiungere commenti
(4) Creazione di visualizzazioni personalizzate con filtri etichetta/assegnatario
(5) Esecuzione di query/mutazioni GraphQL personalizzate
NON utilizzare la CLI `lineare`: utilizzare solo l'API HTTP Composio.
