Gestión de proyectos lineales a través de Composio API. Úselo cuando:
(1) Listar, crear o actualizar problemas lineales
(2) Obtener equipos, usuarios o estados de flujo de trabajo
(3) Gestionar proyectos o agregar comentarios
(4) Crear vistas personalizadas con filtros de etiqueta/asignado
(5) Ejecutar consultas/mutaciones GraphQL personalizadas
NO utilice CLI `lineal`; utilice únicamente la API HTTP de Composio.
