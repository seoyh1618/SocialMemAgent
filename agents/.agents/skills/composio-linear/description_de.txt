Lineares Projektmanagement über Composio API. Verwenden Sie, wenn:
(1) Auflisten, Erstellen oder Aktualisieren linearer Probleme
(2) Abrufen von Teams, Benutzern oder Workflow-Status
(3) Projekte verwalten oder Kommentare hinzufügen
(4) Erstellen benutzerdefinierter Ansichten mit Label-/Zuweisungsfiltern
(5) Ausführen benutzerdefinierter GraphQL-Abfragen/Mutationen
Verwenden Sie KEINE „lineare“ CLI – verwenden Sie nur die Composio-HTTP-API.
