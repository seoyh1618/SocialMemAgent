Линейное управление проектами через Composio API. Используйте, когда:
(1) Перечисление, создание или обновление линейных задач
(2) Получение состояний команд, пользователей или рабочего процесса.
(3) Управление проектами или добавление комментариев.
(4) Создание пользовательских представлений с фильтрами меток/правопреемников.
(5) Запуск пользовательских запросов/мутаций GraphQL
НЕ используйте «линейный» CLI — используйте только Composio HTTP API.
