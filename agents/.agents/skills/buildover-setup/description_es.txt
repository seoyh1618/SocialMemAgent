Configure e integre BuildOver en cualquier proyecto web existente. BuildOver es una herramienta de desarrollo impulsada por IA que envuelve una aplicación web en ejecución con un widget de chat flotante a través de un proxy inverso, lo que le permite pedirle a Claude que modifique los archivos fuente en tiempo real con HMR.
Utilice esta habilidad cuando el usuario diga /buildover-setup, solicite "agregar BuildOver a mi proyecto", "integrar BuildOver", "configurar el asistente de codificación AI en mi servidor de desarrollo" o quiera conectar BuildOver a su aplicación en ejecución existente.
