Configurez et intégrez BuildOver dans n’importe quel projet Web existant. BuildOver est un outil de développement basé sur l'IA qui enveloppe une application Web en cours d'exécution avec un widget de discussion flottant via un proxy inverse, vous permettant de demander à Claude de modifier les fichiers sources en temps réel avec HMR.
Utilisez cette compétence lorsque l'utilisateur dit /buildover-setup, demande « d'ajouter BuildOver à mon projet », « d'intégrer BuildOver », de « configurer l'assistant de codage AI sur mon serveur de développement » ou souhaite connecter BuildOver à son application en cours d'exécution existante.
