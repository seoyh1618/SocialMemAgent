Configura e integra BuildOver in qualsiasi progetto web esistente. BuildOver è uno strumento di sviluppo basato sull'intelligenza artificiale che avvolge un'app Web in esecuzione con un widget di chat mobile tramite un proxy inverso, permettendoti di chiedere a Claude di modificare i file sorgente in tempo reale con HMR.
Utilizza questa abilità quando l'utente dice /buildover-setup, chiede di "aggiungere BuildOver al mio progetto", "integrare BuildOver", "configurare l'assistente di codifica AI sul mio server di sviluppo" o desidera connettere BuildOver all'applicazione in esecuzione esistente.
