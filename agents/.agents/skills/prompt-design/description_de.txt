Ein zeitnahes Design-, Optimierungs- und Qualitätsprüfungstool basierend auf den offiziellen Richtlinien von Anthropic. Anwendbare Szenarien:
(1) Neue Eingabeaufforderungen erstellen – Benutzer sagt: „Helfen Sie mir, eine XXX-Eingabeaufforderung zu erstellen/zu entwerfen.“
(2) Optimierung bestehender Eingabeaufforderungen – Benutzer sagt „Diese Eingabeaufforderung optimieren/verbessern“
(3) Überprüfung der Qualität der Eingabeaufforderung – Benutzer sagt „Überprüfen/überprüfen Sie diese Eingabeaufforderung“
(4) Verständnis der Eingabeaufforderungsstruktur – Benutzer fragt: „Wie gestalte ich eine Eingabeaufforderung?“ oder „Best Practices für Eingabeaufforderungen“
