Инструмент быстрого проектирования, оптимизации и проверки качества, основанный на официальных руководствах Anthropic. Применимые сценарии:
(1) Создание новых подсказок — пользователь говорит: «Помогите мне создать/разработать подсказку XXX».
(2) Оптимизация существующих подсказок — пользователь говорит: «Оптимизируйте/улучшите эту подсказку».
(3) Проверка качества подсказки — пользователь говорит: «Проверьте/просмотрите эту подсказку».
(4) Понимание структуры подсказки — пользователь спрашивает: «Как мне создать подсказку?» или «Рекомендации по использованию подсказок»
