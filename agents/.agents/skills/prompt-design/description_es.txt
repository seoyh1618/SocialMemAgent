Una herramienta rápida de diseño, optimización y control de calidad basada en las pautas oficiales de Anthropic. Escenarios aplicables:
(1) Creación de nuevos mensajes: el usuario dice "Ayúdame a crear/diseñar un mensaje XXX"
(2) Optimización de mensajes existentes: el usuario dice "Optimizar/mejorar este mensaje"
(3) Verificar la calidad del mensaje: el usuario dice "Verificar/revisar este mensaje"
(4) Comprender la estructura del mensaje: el usuario pregunta "¿Cómo diseño un mensaje?" o "Mejores prácticas para indicaciones"
