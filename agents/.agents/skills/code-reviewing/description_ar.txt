يقوم بإجراء مراجعة منهجية للتعليمات البرمجية مع أفضل الممارسات العالمية والمعايير الخاصة بالريبو. يتم التنشيط تلقائيًا بعد إجراء تغييرات كبيرة في التعليمات البرمجية. يُستخدم عند مراجعة التعليمات البرمجية، أو تدقيق الملفات، أو التحقق من العلاقات العامة، أو فحص التغييرات المرحلية، أو عندما يُطلب منك "مراجعة" التعليمات البرمجية أو "التحقق منها" أو "التدقيق" أو "فحصها". يفرض مبادئ التصميم (SOLID وDRY وKISS) والأمان (OWASP) والأداء وأمان التزامن والتوافق عبر الأنظمة الأساسية وأنماط قاعدة التعليمات البرمجية.
