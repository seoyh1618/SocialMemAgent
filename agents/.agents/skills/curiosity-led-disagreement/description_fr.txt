Un cadre pour gérer les désaccords professionnels intenses en dirigeant avec curiosité plutôt qu’avec défense. Utilisez-le lorsqu'un collègue propose une idée « complètement fausse », lorsque vous ressentez une réaction viscérale de « non » lors d'une réunion à enjeux élevés, ou lorsque vous devez combler un profond schisme dans les perspectives pour obtenir le meilleur résultat.
