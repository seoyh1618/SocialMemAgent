دليل بنية وحدة مستودع Artifactory Artifact، الذي يغطي تحميل/تنزيل العناصر، وتكييف الواجهة الخلفية للتخزين، والبيانات التعريفية للعناصر، وسياسات التنظيف، والتحكم في الأذونات. يُستخدم عندما يقوم المستخدمون بتطوير ميزات مستودع العناصر، أو التعامل مع تخزين العناصر، أو تكوين سياسات التنظيف، أو تنفيذ إدارة العناصر.
