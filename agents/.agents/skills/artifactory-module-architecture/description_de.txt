Artifactory Artifact Repository Module Architecture Guide, der das Hoch- und Herunterladen von Artefakten, die Anpassung des Speicher-Backends, Artefaktmetadaten, Bereinigungsrichtlinien und Berechtigungskontrolle behandelt. Wird verwendet, wenn Benutzer Artefakt-Repository-Funktionen entwickeln, die Artefaktspeicherung verwalten, Bereinigungsrichtlinien konfigurieren oder die Artefaktverwaltung implementieren.
