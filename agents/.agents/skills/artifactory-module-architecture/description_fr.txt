Guide d'architecture du module Artifactory Artifact Repository, couvrant le téléchargement/téléchargement d'artefacts, l'adaptation du backend de stockage, les métadonnées des artefacts, les politiques de nettoyage et le contrôle des autorisations. Utilisé lorsque les utilisateurs développent des fonctionnalités de référentiel d'artefacts, gèrent le stockage d'artefacts, configurent des politiques de nettoyage ou implémentent la gestion des artefacts.
