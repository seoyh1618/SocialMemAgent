Cree problemas de GitHub con combinación inteligente de plantillas y selección de herramientas. Detecta y aplica automáticamente plantillas de problemas del directorio .github/ISSUE_TEMPLATE mediante coincidencias inteligentes. Aprovecha las herramientas en orden de prioridad (GitHub MCP Server → GitHub CLI → REST API). Úselo cuando necesite crear una nueva incidencia de GitHub, opcionalmente con etiquetas, asignados, hitos o asignación de proyecto. La habilidad maneja la coincidencia de plantillas de manera inteligente: cuando existen varias plantillas, analiza la descripción del problema para sugerir la mejor plantilla y solicita aclaraciones solo cuando hay ambigüedades.
