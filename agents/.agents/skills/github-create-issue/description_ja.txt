インテリジェントなテンプレート マッチングとツール選択を使用して GitHub の課題を作成します。スマート マッチングを使用して、.github/ISSUE_TEMPLATE ディレクトリから課題テンプレートを自動的に検出して適用します。優先順位に従ってツールを活用します (GitHub MCP サーバー → GitHub CLI → REST API)。新しい GitHub の課題を作成する必要がある場合に使用します。必要に応じて、ラベル、担当者、マイルストーン、またはプロジェクトの割り当てを指定します。このスキルはテンプレート マッチングをインテリジェントに処理します。複数のテンプレートが存在する場合、問題の説明を分析して最適なテンプレートを提案し、あいまいな場合にのみ説明を求めます。
