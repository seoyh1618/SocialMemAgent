قم بإنشاء مشكلات GitHub من خلال مطابقة القالب الذكية واختيار الأداة. يكتشف ويطبق قوالب المشكلات تلقائيًا من دليل .github/ISSUE_TEMPLATE باستخدام المطابقة الذكية. الاستفادة من الأدوات حسب ترتيب الأسبقية (GitHub MCP Server → GitHub CLI → REST API). استخدمه عندما تحتاج إلى إنشاء مشكلة GitHub جديدة، بشكل اختياري مع التسميات أو المكلفين أو المعالم الرئيسية أو مهمة المشروع. تتعامل المهارة مع مطابقة القالب بذكاء - عند وجود قوالب متعددة، فإنها تحلل وصف المشكلة لاقتراح أفضل قالب، وتطلب التوضيح فقط عندما يكون هناك غموض.
