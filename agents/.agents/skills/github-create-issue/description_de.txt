Erstellen Sie GitHub-Probleme mit intelligentem Vorlagenabgleich und Toolauswahl. Erkennt mithilfe von Smart Matching automatisch Problemvorlagen aus dem Verzeichnis .github/ISSUE_TEMPLATE und wendet diese an. Nutzt Tools in der Reihenfolge ihrer Priorität (GitHub MCP Server → GitHub CLI → REST API). Verwenden Sie diese Option, wenn Sie ein neues GitHub-Problem erstellen müssen, optional mit Bezeichnungen, Beauftragten, Meilensteinen oder Projektzuweisungen. Der Skill verwaltet den Vorlagenabgleich auf intelligente Weise. Wenn mehrere Vorlagen vorhanden sind, analysiert er die Problembeschreibung, um die beste Vorlage vorzuschlagen, und bittet nur dann um Klärung, wenn Unklarheiten bestehen.
