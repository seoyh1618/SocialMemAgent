Создавайте задачи GitHub с помощью интеллектуального сопоставления шаблонов и выбора инструментов. Автоматически обнаруживает и применяет шаблоны задач из каталога .github/ISSUE_TEMPLATE с помощью интеллектуального сопоставления. Использует инструменты в порядке приоритета (GitHub MCP Server → GitHub CLI → REST API). Используйте, когда вам нужно создать новую задачу GitHub, при необходимости с метками, правопреемниками, контрольными точками или назначением проекта. Навык разумно обрабатывает сопоставление шаблонов: при наличии нескольких шаблонов он анализирует описание проблемы, чтобы предложить лучший шаблон, запрашивая разъяснения только в случае двусмысленности.
