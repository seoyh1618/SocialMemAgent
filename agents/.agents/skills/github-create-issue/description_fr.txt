Créez des problèmes GitHub avec une correspondance intelligente de modèles et une sélection d'outils. Détecte et applique automatiquement les modèles de problèmes du répertoire .github/ISSUE_TEMPLATE à l'aide de la correspondance intelligente. Tire parti des outils par ordre de priorité (GitHub MCP Server → GitHub CLI → REST API). À utiliser lorsque vous devez créer un nouveau problème GitHub, éventuellement avec des étiquettes, des destinataires, des jalons ou une affectation de projet. La compétence gère intelligemment la correspondance des modèles : lorsque plusieurs modèles existent, elle analyse la description du problème pour suggérer le meilleur modèle, en demandant des éclaircissements uniquement en cas d'ambiguïté.
