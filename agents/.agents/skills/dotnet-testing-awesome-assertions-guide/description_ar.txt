إتقان تأكيدات الاختبار بطلاقة وقابلة للقراءة باستخدام AwesomeAssertions.
استخدمه عندما تحتاج إلى كتابة تأكيدات واضحة، أو مقارنة الكائنات، أو التحقق من صحة المجموعات، أو التعامل مع المطابقة المعقدة.
يغطي واجهة برمجة التطبيقات (API) الكاملة بما في ذلك Does() وBeEquivalentTo() وContain() و ThrowAsync() والمزيد.
