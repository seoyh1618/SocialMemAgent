Gehackte interaktive PR-Überprüfung – führt Sie Schritt für Schritt durch PRs, zeigt vorhandenes Feedback der Rezensenten an und hilft dabei, durchdachte Kommentare ohne kognitive Überlastung zu veröffentlichen. Verwenden Sie diese Option, wenn der Benutzer eine Pull-Anfrage überprüfen, PR-Änderungen untersuchen oder Überprüfungskommentare auf GitHub veröffentlichen möchte.
