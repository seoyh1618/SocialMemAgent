يستخرج بيانات التصميم من Figma عبر REST API ويحول التصميمات إلى كود React. يُستخدم عند مزامنة رموز Figma المميزة مع متغيرات CSS، أو تصدير الأيقونات كمكونات SVG، أو إنشاء تعليمات برمجية من تصميمات Figma، أو أتمتة سير عمل التصميم إلى تعليمات برمجية. يُستخدم لاستخراج رمز التصميم وتصدير الأيقونات وإنشاء المكونات ومزامنة Figma المستندة إلى CI.
