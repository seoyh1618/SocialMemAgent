Extrahiert Designdaten aus Figma über die REST-API und konvertiert Designs in React-Code. Verwenden Sie es, wenn Sie Figma-Tokens mit CSS-Variablen synchronisieren, Symbole als SVG-Komponenten exportieren, Code aus Figma-Designs generieren oder Design-to-Code-Workflows automatisieren. Verwendung für die Extraktion von Design-Token, den Export von Symbolen, die Generierung von Komponenten und die CI-basierte Figma-Synchronisierung.
