Извлекает данные дизайна из Figma через REST API и преобразует проекты в код React. Используйте при синхронизации токенов Figma с переменными CSS, экспорте значков как компонентов SVG, создании кода на основе проектов Figma или автоматизации рабочих процессов преобразования дизайна в код. Используйте для извлечения токенов дизайна, экспорта значков, генерации компонентов и синхронизации Figma на основе CI.
