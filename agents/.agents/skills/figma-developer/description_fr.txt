Extrait les données de conception de Figma via l'API REST et convertit les conceptions en code React. À utiliser lors de la synchronisation des jetons Figma avec des variables CSS, de l'exportation d'icônes en tant que composants SVG, de la génération de code à partir de conceptions Figma ou de l'automatisation des flux de travail de conception à code. À utiliser pour l'extraction de jetons de conception, l'exportation d'icônes, la génération de composants et la synchronisation Figma basée sur CI.
