Abilità di rilascio del progetto ABC Jenkins. Supporta l'inferenza intelligente dei parametri e l'attivazione interattiva delle build Jenkins, recupera automaticamente il ramo Git e le informazioni sui tag. Questa abilità viene attivata quando l'utente richiede "rilascio Jenkins", "trigger build", "distribuzione progetto", "rilascio Jenkins" o operazioni simili. Richiede le variabili di ambiente JENKINS_USER e JENKINS_TOKEN.
