مهارة إصدار مشروع ABC Jenkins. يدعم الاستدلال الذكي للمعلمات والتشغيل التفاعلي لبنيات Jenkins، ويجلب معلومات فرع Git ومعلومات العلامة تلقائيًا. يتم تشغيل هذه المهارة عندما يطلب المستخدم "إصدار Jenkins" أو "إنشاء المشغل" أو "نشر المشروع" أو "إصدار Jenkins" أو عمليات مماثلة. يتطلب متغيرات البيئة JENKINS_USER وJENKINS_TOKEN.
