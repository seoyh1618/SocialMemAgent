Как правильно получить доступ к базе данных управления Doltgres с помощью подключений на уровне филиала. Используйте при чтении или записи данных управления (агенты, инструменты, проекты, учетные данные, триггеры, оценщики и т. д.). Срабатывает при: управлении базой данных, withRef, c.get('db'), ManageDbPool, проверке ветки, doltgres, ref-scope, BranchScopedDb, AgentsManageDatabaseClient.
