قم بتحويل ملفات وسلاسل Markdown إلى مستندات DOCX باستخدام @mohtasham/md-to-docx. يُستخدم عندما يحتاج المستخدم إلى تحويل Markdown إلى Word، أو تحويل الملفات المستند إلى CLI، أو مجموعة الخطوط/المحاذاة/التصميم المستندة إلى الخيارات، أو معالجة فاصل جدول المحتويات/الصفحات، أو التنسيق الذي يتوسطه خط/يتوسطه خط، أو التحويل البرمجي في كود العقدة/المتصفح.
