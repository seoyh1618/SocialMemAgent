Broker di token OAuth per agenti AI. Ottieni token di accesso per Google, GitHub, Slack, Linear, Notion, Vercel, Sentry, Asana, Dropbox, Figma e oltre 20 provider con due chiamate API. Utilizzalo quando il tuo agente necessita di autenticazione OAuth, token API, autorizzazione utente o accesso a servizi di terze parti. Gestisce l'intero flusso OAuth (URL di reindirizzamento, schermata di consenso, scambio di token, aggiornamento) in modo che non debba farlo il tuo agente. Funziona con Claude Code, Cursor, Codex, OpenClaw e qualsiasi agente di codifica che necessiti di accesso API autenticato.
