Broker de tokens OAuth para agentes de IA. Obtenga tokens de acceso para Google, GitHub, Slack, Linear, Notion, Vercel, Sentry, Asana, Dropbox, Figma y más de 20 proveedores con dos llamadas API. Úselo cuando su agente necesite autenticación OAuth, tokens API, autorización de usuario o acceso a servicios de terceros. Maneja todo el flujo de OAuth (URL de redireccionamiento, pantalla de consentimiento, intercambio de tokens, actualización) para que su agente no tenga que hacerlo. Funciona con Claude Code, Cursor, Codex, OpenClaw y cualquier agente de codificación que necesite acceso API autenticado.
