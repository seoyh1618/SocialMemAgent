Courtier de jetons OAuth pour les agents IA. Obtenez des jetons d'accès pour Google, GitHub, Slack, Linear, Notion, Vercel, Sentry, Asana, Dropbox, Figma et plus de 20 fournisseurs avec deux appels API. À utiliser lorsque votre agent a besoin d'une authentification OAuth, de jetons API, d'une autorisation utilisateur ou d'un accès à des services tiers. Gère l'intégralité du flux OAuth (URL de redirection, écran de consentement, échange de jetons, actualisation) pour que votre agent n'ait pas à le faire. Fonctionne avec Claude Code, Cursor, Codex, OpenClaw et tout agent de codage nécessitant un accès API authentifié.
