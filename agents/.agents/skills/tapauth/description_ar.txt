وسيط رمز OAuth لوكلاء الذكاء الاصطناعي. احصل على رموز الوصول إلى Google، وGitHub، وSlack، وLinear، وNotion، وVercel، وSentry، وAsana، وDropbox، وFigma، وأكثر من 20 موفرًا من خلال مكالمتين لواجهة برمجة التطبيقات (API). يُستخدم عندما يحتاج وكيلك إلى مصادقة OAuth، أو رموز API المميزة، أو ترخيص المستخدم، أو الوصول إلى خدمات الجهات الخارجية. يتعامل مع تدفق OAuth الكامل - إعادة توجيه عنوان URL، وشاشة الموافقة، وتبادل الرمز المميز، والتحديث - حتى لا يضطر وكيلك إلى القيام بذلك. يعمل مع Claude Code، وCursor، وCodex، وOpenClaw، وأي وكيل ترميز يحتاج إلى وصول معتمد إلى واجهة برمجة التطبيقات.
