Fügen Sie asynchrone Telegram-Bot-Konten in OpenClaw hinzu oder aktualisieren Sie sie. Verwenden Sie diese Option, wenn der Benutzer mehrere Telegram-Bots hinzufügen, Bots parallel ausführen oder OpenClaw bitten möchte, die BotFather-basierte Bereitstellung zu leiten. Erzwingt das Benutzernamenmuster Nebutra[drei Ziffern]_bot und englische Bot-Namen im Philosophenstil.
