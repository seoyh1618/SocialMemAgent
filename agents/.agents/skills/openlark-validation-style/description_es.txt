Especificación de validación de parámetros de caja de funciones del SDK de OpenLark Rust (validación requerida). Úselo cuando necesite estandarizar/revisar la implementación de `validate()`, o cuando los usuarios pregunten sobre la diferencia entre `openlark_core::validate_required` (función) y `openlark_core::validate_required!` (macro), si se necesita una macro, manejo de cadenas vacías, etc.
