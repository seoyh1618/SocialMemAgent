Spécification de validation des paramètres de caisse de fonctionnalités du SDK OpenLark Rust (validation de champ obligatoire). À utiliser lorsque vous devez standardiser/réviser l'implémentation de `validate()`, ou lorsque les utilisateurs demandent la différence entre `openlark_core::validate_required` (fonction) et `openlark_core::validate_required!` (macro), si une macro est nécessaire, comment les chaînes vides sont gérées, etc.
