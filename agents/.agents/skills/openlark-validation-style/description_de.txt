OpenLark Rust SDK Feature-Crate-Parametervalidierungsspezifikation (erforderliche Feldvalidierung). Verwenden Sie dies, wenn Sie die „validate()“-Implementierung vereinheitlichen/überprüfen müssen oder wenn Benutzer nach dem Unterschied zwischen „openlark_core::validate_required“ (Funktion) und „openlark_core::validate_required!“ (Makro) fragen, ob ein Makro erforderlich ist, wie mit leeren Zeichenfolgen umgegangen wird usw.
