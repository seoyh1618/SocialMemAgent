Configuration complète du SDK Sentry pour Node.js, Bun et Deno. À utiliser lorsqu'on vous demande d'« ajouter Sentry à Node.js », « ajouter Sentry à Bun », « ajouter Sentry à Deno », « installer @sentry/node », « @sentry/bun » ou « @sentry/deno », ou configurer la surveillance des erreurs, le traçage, la journalisation, le profilage, les métriques, les crons ou la surveillance de l'IA pour les environnements d'exécution JavaScript/TypeScript côté serveur.
