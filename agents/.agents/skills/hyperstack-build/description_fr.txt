Créez des piles Hyperstack personnalisées à partir des IDL du programme Solana à l'aide du Rust DSL. Couvre les définitions d'entités, les mappages de champs, les vues, les champs calculés, la résolution PDA et le déploiement avec la CLI hs. À utiliser lorsque l'utilisateur souhaite créer sa propre pile de streaming de données en temps réel.
