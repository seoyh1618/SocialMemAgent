Создавайте собственные стеки Hyperstack из IDL-файлов программы Solana с помощью Rust DSL. Охватывает определения сущностей, сопоставления полей, представления, вычисляемые поля, разрешение PDA и развертывание с помощью hs CLI. Используйте, когда пользователь хочет создать свой собственный стек потоковой передачи данных в реальном времени.
