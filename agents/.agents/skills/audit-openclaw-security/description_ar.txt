تدقيق وتقوية أمان OpenClaw (البوابة + الوكلاء). يُستخدم عندما يطلب المستخدم تدقيق/تأمين/تقوية OpenClaw؛ عند استكشاف أخطاء التعرض المحفوفة بالمخاطر وإصلاحها (خاصة واجهة المستخدم/مستوى التحكم في بوابة الويب على المنفذ 18789)؛ عند مراجعة التحكم في الوصول إلى الرسائل المباشرة/المجموعة (الإقران/القوائم المسموح بها/بوابة الإشارة)؛ أذونات الأداة (exec/fs/browser/nodes/gateway/cron)؛ مخاطر سلسلة التوريد للمكونات الإضافية/المهارات؛ الأسرار/النصوص/الاحتفاظ بالسجل؛ أو عند نشر OpenClaw على جهاز Mac mini أو كمبيوتر محمول شخصي أو مضيف Docker أو Cloud VM (AWS EC2/VPS).
