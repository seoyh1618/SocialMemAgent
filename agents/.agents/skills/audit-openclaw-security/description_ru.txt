Аудит и усиление безопасности OpenClaw (шлюз + агенты). Используйте, когда пользователь запрашивает аудит/защиту/укрепление OpenClaw; при устранении неполадок, связанных с риском воздействия (особенно веб-интерфейса/плоскости управления шлюза на порту 18789); при проверке управления доступом к DM/группе (сопряжение/белые списки/упоминание); разрешения инструмента (exec/fs/browser/nodes/gateway/cron); плагины/навыки, риск в цепочке поставок; секреты/расшифровки/сохранение журналов; или при развертывании OpenClaw на Mac mini, персональном ноутбуке, хосте Docker или облачной виртуальной машине (AWS EC2/VPS).
