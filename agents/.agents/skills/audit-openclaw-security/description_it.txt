Controlla e rafforza la sicurezza di OpenClaw (Gateway + agenti). Da utilizzare quando l'utente chiede di controllare/proteggere/rafforzare OpenClaw; durante la risoluzione dei problemi di esposizione rischiosa (in particolare l'interfaccia utente web/piano di controllo del gateway sulla porta 18789); durante la revisione del controllo dell'accesso a DM/gruppi (associazione/liste consentite/gating delle menzioni); permessi dello strumento (exec/fs/browser/nodes/gateway/cron); rischi legati alla catena di fornitura di plug-in/competenze; segreti/trascrizioni/conservazione dei registri; o quando si distribuisce OpenClaw su un Mac mini, laptop personale, host Docker o VM cloud (AWS EC2/VPS).
