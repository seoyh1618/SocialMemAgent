Prüfen und verstärken Sie die Sicherheit von OpenClaw (Gateway + Agents). Wird verwendet, wenn der Benutzer OpenClaw prüfen/sichern/härten möchte; bei der Fehlerbehebung bei riskanten Gefährdungen (insbesondere der Gateway-Web-Benutzeroberfläche/Steuerungsebene auf Port 18789); bei der Überprüfung der DM-/Gruppenzugriffskontrolle (Pairing/Zulassungslisten/Erwähnungs-Gating); Tool-Berechtigungen (exec/fs/browser/nodes/gateway/cron); Plugins/Skills-Lieferkettenrisiko; Aufbewahrung von Geheimnissen/Transkripten/Protokollen; oder bei der Bereitstellung von OpenClaw auf einem Mac mini, einem persönlichen Laptop, einem Docker-Host oder einer Cloud-VM (AWS EC2/VPS).
