Рекомендации по использованию набора инструментов Zerone CLI. Охватывает четыре основные функции: генерацию кода интерфейса API (zerone api), управление значками шрифтов (zerone font_grabber), создание внешних проектов (create-zerone) и генерацию рабочего журнала (zerone log). Используйте этот навык в следующих сценариях: создание модулей API, создание кода интерфейса, обновление интерфейсов, pnpm API, добавление новых модулей внутреннего интерфейса, значки шрифтов iconfont, обновление значков, шрифт pnpm, использование значков значков, создание проектов внешнего интерфейса, pnpm create Zerone, инициализация строительных лесов, ежедневные/еженедельные/ежемесячные отчеты, нулевой журнал. Он также должен срабатывать, даже когда пользователи просто упоминают такие ключевые слова, как «интерфейс», «значок», «новый проект», «ежедневный/еженедельный отчет» и т. д.
