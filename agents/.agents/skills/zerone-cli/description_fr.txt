Directives d'utilisation de la boîte à outils Zerone CLI. Couvre quatre fonctions principales : génération de code d'interface API (api zéro), gestion des icônes de police (zerone font_grabber), échafaudage de projet frontal (create-zerone) et génération de journaux de travail (journal zéro). Utilisez cette compétence dans les scénarios suivants : génération de modules API, génération de code d'interface, mise à jour des interfaces, API pnpm, ajout de nouveaux modules d'interface backend, icônes de police iconfont, mise à jour des icônes, police pnpm, utilisation des icônes, création de projets frontend, pnpm create zerone, initialisation de l'échafaudage, rapports de travail quotidiens/hebdomadaires/mensuels, journal zérone. Il doit également être déclenché même lorsque les utilisateurs mentionnent simplement des mots-clés tels que « interface », « icône », « nouveau projet », « rapports quotidiens/hebdomadaires », etc.
