Pautas de uso del kit de herramientas Zerone CLI. Cubre cuatro funciones principales: generación de código de interfaz API (zerone api), administración de íconos de fuentes (zerone font_grabber), andamiaje del proyecto frontend (create-zerone) y generación de registros de trabajo (zerone log). Utilice esta habilidad en los siguientes escenarios: generar módulos API, generar código de interfaz, actualizar interfaces, pnpm api, agregar nuevos módulos de interfaz backend, íconos de fuentes iconfont, actualizar íconos, fuente pnpm, uso de íconos, crear proyectos frontend, pnpm create zerone, inicialización de andamios, informes de trabajo diarios/semanales/mensuales, registro zerone. También debería activarse incluso cuando los usuarios simplemente mencionen palabras clave como "interfaz", "icono", "nuevo proyecto", "informes diarios/semanales", etc.
