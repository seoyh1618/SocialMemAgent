Zerone CLI ツールキットの使用ガイドライン。 API インターフェイス コード生成 (zerone api)、フォント アイコン管理 (zerone font_grabber)、フロントエンド プロジェクト スキャフォールディング (create-zerone)、作業ログ生成 (zerone log) の 4 つの主要な機能をカバーします。このスキルは次のシナリオで使用します: API モジュールの生成、インターフェイス コードの生成、インターフェイスの更新、pnpm API、新しいバックエンド インターフェイス モジュールの追加、iconfont フォント アイコン、アイコンの更新、pnpm フォント、アイコンの使用、フロントエンド プロジェクトの作成、pnpm create zerone、スキャフォールディングの初期化、日次/週次/月次の作業レポート、zerone ログ。また、ユーザーが「インターフェイス」、「アイコン」、「新しいプロジェクト」、「日次/週次レポート」などのキーワードを単に言及した場合でもトリガーされる必要があります。
