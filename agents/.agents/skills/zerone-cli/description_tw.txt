Zerone CLI 工具包使用指南。涵蓋四個主要功能：API介面程式碼產生（zeroone api）、字型圖示管理（zerone font_grabber）、前端專案鷹架（create-zerone）、工作日誌產生（zeroone log）。在以下場景中使用此技能：產生API模組、生成介面程式碼、更新介面、pnpm api、新增新的後端介面模組、iconfont字體圖示、更新圖示、pnpm字體、圖示使用、建立前端專案、pnpm建立zerone、腳手架初始化、每日/每週/每月工作報告、zerone日誌。即使用戶只是提及「介面」、「圖示」、「新項目」、「每日/每週報告」等關鍵字，也應該觸發它。
