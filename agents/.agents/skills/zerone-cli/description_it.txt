Specifiche di utilizzo del toolkit CLI Zerone. Copre quattro funzioni principali: generazione del codice dell'interfaccia API (zero api), gestione delle icone dei caratteri (zero font_grabber), impalcatura del progetto front-end (create-zerone) e generazione del registro di lavoro (zerone log). Utilizza questa competenza nei seguenti scenari: generazione di moduli API, generazione di codice di interfaccia, aggiornamento di interfacce, API pnpm, aggiunta di nuovi moduli di interfaccia back-end, icone dei caratteri iconfont, aggiornamento delle icone, carattere pnpm, utilizzo delle icone, creazione di progetti front-end, pnpm crea zerone, inizializzazione dello scaffolding, report giornalieri/settimanali/mensili, registro zerone. Anche quando gli utenti menzionano semplicemente parole chiave come "interfaccia", "icona", "nuovo progetto" o "rapporti giornalieri/settimanali", dovrebbe essere attivato.
