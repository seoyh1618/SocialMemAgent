Zerone CLI 툴킷 사용 지침. API 인터페이스 코드 생성(zeroone api), 글꼴 아이콘 관리(zeronefont_grabber), 프런트엔드 프로젝트 스캐폴딩(create-zerone) 및 작업 로그 생성(zerone log)의 네 가지 주요 기능을 다룹니다. 다음 시나리오에서 이 기술을 사용하십시오: API 모듈 생성, 인터페이스 코드 생성, 인터페이스 업데이트, pnpm api, 새 백엔드 인터페이스 모듈 추가, iconfont 글꼴 아이콘, 아이콘 업데이트, pnpm 글꼴, 아이콘 사용, 프런트엔드 프로젝트 생성, pnpm create zerone, 스캐폴딩 초기화, 일일/주별/월별 작업 보고서, zeroone 로그. 또한 사용자가 "인터페이스", "아이콘", "새 프로젝트", "일별/주간 보고서" 등과 같은 키워드를 단순히 언급하는 경우에도 트리거되어야 합니다.
