Servicios principales de AWS para desarrolladores de aplicaciones. Cubre S3 (almacenamiento, URL prefirmadas, ciclo de vida), Lambda (funciones, capas, arranques en frío), IAM (roles, políticas, privilegios mínimos), DynamoDB (diseño de tabla única, GSI/LSI, transmisiones), SQS/SNS (colas, temas, distribución), CloudFront (CDN, almacenamiento en caché), RDS/Aurora (Postgres/MySQL, agrupación de conexiones), ECR/ECS/Fargate (contenedores), Route 53 (DNS), Secrets Manager y CDK v2 (TypeScript IaC, construcciones, pilas, pruebas).

Úselo al crear infraestructura de AWS, escribir pilas de CDK, configurar políticas de IAM, diseñar tablas de DynamoDB, configurar funciones de Lambda, crear URL prefirmadas de S3, implementar contenedores en ECS/Fargate o configurar distribuciones de CloudFront.
