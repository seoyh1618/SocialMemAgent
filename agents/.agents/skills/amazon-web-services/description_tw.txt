面向應用程式開發人員的核心 AWS 服務。涵蓋 S3（儲存、預簽名 URL、生命週期）、Lambda（函數、層、冷啟動）、IAM（角色、策略、最小權限）、DynamoDB（單表設計、GSI/LSI、流）、SQS/SNS（佇列、主題、磁力表）、CloudFront（CDN、Tipak）、RDS/AMot（Post）/ (DNS)、Secrets Manager 和 CDK v2（TypeScript IaC、建構、堆疊、測試）。

在建置 AWS 基礎架構、編寫 CDK 堆疊、設定 IAM 原則、設計 DynamoDB 表、設定 Lambda 函數、建立 S3 預簽名 URL、在 ECS/Fargate 上部署容器或設定 CloudFront 發行版時使用。
