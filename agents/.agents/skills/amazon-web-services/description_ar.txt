خدمات AWS الأساسية لمطوري التطبيقات. يغطي S3 (التخزين، عناوين URL المحددة مسبقًا، دورة الحياة)، Lambda (الوظائف، الطبقات، البداية الباردة)، IAM (الأدوار، السياسات، الامتيازات الأقل)، DynamoDB (تصميم جدول واحد، GSI/LSI، التدفقات)، SQS/SNS (قوائم الانتظار، المواضيع، التوزيع)، CloudFront (CDN، التخزين المؤقت)، RDS/Aurora (Postgres/MySQL، تجمع الاتصالات)، ECR/ECS/Fargate (الحاويات)، والطريق 53 (DNS)، ومدير الأسرار، وCDK v2 (TypeScript IaC، والإنشاءات، والمكدسات، والاختبار).

يُستخدم عند إنشاء البنية التحتية لـ AWS، أو كتابة مجموعات CDK، أو تكوين سياسات IAM، أو تصميم جداول DynamoDB، أو إعداد وظائف Lambda، أو إنشاء عناوين URL الموقعة مسبقًا على S3، أو نشر الحاويات على ECS/Fargate، أو تكوين توزيعات CloudFront.
