Основные сервисы AWS для разработчиков приложений. Охватывает S3 (хранилище, заданные URL-адреса, жизненный цикл), Lambda (функции, уровни, холодный запуск), IAM (роли, политики, минимальные привилегии), DynamoDB (однотабличный дизайн, GSI/LSI, потоки), SQS/SNS (очереди, темы, разветвление), CloudFront (CDN, кэширование), RDS/Aurora (Postgres/MySQL, пул соединений), ECR/ECS/Fargate (контейнеры), Route 53 (DNS), Secrets Manager и CDK v2 (TypeScript IaC, конструкции, стеки, тестирование).

Используйте при построении инфраструктуры AWS, написании стеков CDK, настройке политик IAM, разработке таблиц DynamoDB, настройке функций Lambda, создании заранее назначенных URL-адресов S3, развертывании контейнеров в ECS/Fargate или настройке дистрибутивов CloudFront.
