Kerndienste von AWS für Anwendungsentwickler. Deckt S3 (Speicher, vorsignierte URLs, Lebenszyklus), Lambda (Funktionen, Ebenen, Kaltstarts), IAM (Rollen, Richtlinien, geringste Privilegien), DynamoDB (Einzeltabellendesign, GSI/LSI, Streams), SQS/SNS (Warteschlangen, Themen, Fan-Out), CloudFront (CDN, Caching), RDS/Aurora (Postgres/MySQL, Verbindungspooling), ECR/ECS/Fargate (Container) ab. Route 53 (DNS), Secrets Manager und CDK v2 (TypeScript IaC, Konstrukte, Stacks, Tests).

Verwendung beim Aufbau einer AWS-Infrastruktur, beim Schreiben von CDK-Stacks, beim Konfigurieren von IAM-Richtlinien, beim Entwerfen von DynamoDB-Tabellen, beim Einrichten von Lambda-Funktionen, beim Erstellen vorsignierter S3-URLs, beim Bereitstellen von Containern auf ECS/Fargate oder beim Konfigurieren von CloudFront-Verteilungen.
