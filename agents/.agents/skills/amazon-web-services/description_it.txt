Servizi AWS principali per gli sviluppatori di applicazioni. Copre S3 (storage, URL preimpostati, ciclo di vita), Lambda (funzioni, livelli, avviamenti a freddo), IAM (ruoli, policy, privilegi minimi), DynamoDB (progettazione a tabella singola, GSI/LSI, flussi), SQS/SNS (code, argomenti, fan-out), CloudFront (CDN, caching), RDS/Aurora (Postgres/MySQL, pooling di connessioni), ECR/ECS/Fargate (contenitori), Route 53 (DNS), Secrets Manager e CDK v2 (TypeScript IaC, costrutti, stack, test).

Da utilizzare durante la creazione di infrastrutture AWS, la scrittura di stack CDK, la configurazione di policy IAM, la progettazione di tabelle DynamoDB, l'impostazione di funzioni Lambda, la creazione di URL prefirmati S3, la distribuzione di contenitori su ECS/Fargate o la configurazione di distribuzioni CloudFront.
