Hintergrundkontext für das Gesetz zur elektronischen Buchführung (電子帳簿保存法) im Shinkoku-Steuererklärungs-Plugin. Enthält Anforderungen an die elektronische Buchhaltung, die Scannerspeicherung, die obligatorische elektronische Transaktionsdatenspeicherung und den Compliance-Status von Shinkoku. Dieser Skill kann nicht vom Benutzer aufgerufen werden – Claude lädt ihn automatisch, wenn er auf Fragen zur Einhaltung der elektronischen Buchhaltung antwortet.
