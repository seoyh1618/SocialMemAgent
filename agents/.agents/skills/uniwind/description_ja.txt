Uniwind — React Native 用の Tailwind CSS v4 スタイル。 className で Tailwind を使用する React Native プロジェクトでコンポーネントを追加、構築、またはスタイル設定するときに使用します。トリガー: React Native コンポーネントの className、RN コードの Tailwind クラス、@import 'uniwind' を使用した global.css、withUniwindConfig、Uniwind を使用した metro.config.js、サードパーティ コンポーネントの withUniwind、useResolveClassNames、useCSSVariable、useUniwind、dark:/light: テーマ、プラットフォーム セレクター (ios:/android:/native:/web:/tv:)、data-[prop=value]セレクター、レスポンシブ ブレークポイント (sm:/md:/lg:)、tailwind-variants、tv() コンポーネント バリアント、ScopedTheme、Uniwind.setTheme、Uniwind.updateCSSVariables、Uniwind.updateInsets、@layer テーマ、@variant、@theme ディレクティブ、@theme static、@utility、React Native の CSS 変数、アクセント プレフィックス パターン、colorClassName、tintColorClassName、 contentContainerClassName、Uniwind Pro (アニメーション、トランジション、シャドウ ツリー、ネイティブ インセット)、セーフ エリア ユーティリティ (p-safe/pt-safe/-safe-or-/-safe-offset-)、RN のグラデーション、カスタム CSS クラス、hairlineWidth()、fontScale()、pixelRatio()、light-dark()、OKLCH カラー、cn ユーティリティ、tailwind-merge、HeroUI Native、反応ネイティブ再利用可能、Gluestack。また、「スタイルが適用されない」、「className が機能しない」、「Uniwind セットアップを監査する」、「構成を確認する」、「コンポーネントを追加する」、「コンポーネントのスタイルを設定する」場合にもトリガーされます。 NativeWind から Uniwind への移行は処理しません。そのためには、 merge-nativewind-to-uniwind スキルを使用します。
