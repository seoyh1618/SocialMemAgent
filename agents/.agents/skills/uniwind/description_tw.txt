Uniwind - React Native 的 Tailwind CSS v4 樣式。在使用 Tailwind 和 className 的 React Native 專案中新增、建置或設計元件樣式時使用。觸發條件：React Native 元件上的 className、RN 程式碼中的 Tailwind 類別、帶有 @import 'uniwind' 的 global.css、withUniwindConfig、帶有 Uniwind 的 Metro.config.js、用於第三方元件的 withUniwind、useResolveClassNames、UniuseSV 。指令、@theme static、@utility、React Native 中的 CSS 變數、重音前綴模式、colorClassName、tintColorClassName、contentContainerClassName、Uniwind Pro（動畫、過渡、陰影）樹、原生插圖）、安全區域（p-safe/pt-safe/-safe-or-/sa類別、hairlineWidth()、fontScale()、pixelRatio()、light-dark()、OKLCH 顏色、cn 實用程式、tailwind-merge、HeroUI Native、react-native-reusables、Gluestack。也會觸發：「樣式未套用」、「className 不起作用」、「審核 Uniwind 設定」、「檢查我的配置」、「新增元件」、「樣式元件」。不處理 NativeWind 到 Uniwind 的遷移 - 為此使用 migrate-nativewind-to-uniwind 技能。
