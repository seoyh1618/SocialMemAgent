Uniwind: estilo Tailwind CSS v4 para React Native. Úselo al agregar, crear o diseñar componentes en un proyecto de React Native que usa Tailwind con className. Activadores en: className en componentes React Native, clases Tailwind en código RN, global.css con @import 'uniwind', withUniwindConfig, metro.config.js con Uniwind, withUniwind para componentes de terceros, useResolveClassNames, useCSSVariable, useUniwind, dark:/light: tematización, selectores de plataforma (ios:/android:/native:/web:/tv:), selectores de datos-[prop=valor], puntos de interrupción receptivos (sm:/md:/lg:), variantes de viento de cola, variantes de componentes tv(), ScopedTheme, Uniwind.setTheme, Uniwind.updateCSSVariables, Uniwind.updateInsets, @layer theme, @variant, @theme directiva, @theme static, @utility, variables CSS en React Native, patrón de prefijo de acento, colorClassName, tintColorClassName, contentContainerClassName, Uniwind Pro (animaciones, transiciones, árbol de sombras, inserciones nativas), utilidades de área segura (p-safe/pt-safe/-safe-or-/-safe-offset-), gradientes en RN, clases CSS personalizadas, hairlineWidth(), fontScale(), pixelRatio(), light-dark(), colores OKLCH, utilidad cn, tailwind-merge, HeroUI Native, react-native-reusables, Gluestack. También se activa en: "estilos que no se aplican", "nombre de clase no funciona", "auditar la configuración de Uniwind", "verificar mi configuración", "agregar un componente", "estilo de un componente".
