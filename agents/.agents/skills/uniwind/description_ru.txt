Uniwind — стиль Tailwind CSS v4 для React Native. Используйте при добавлении, создании или стилизации компонентов в проекте React Native, который использует Tailwind с именем класса. Запускается по: className для компонентов React Native, классам Tailwind в коде RN, global.css с @import 'uniwind', withUniwindConfig, Metro.config.js с Uniwind, withUniwind для сторонних компонентов, useResolveClassNames, useCSSVariable, useUniwind, dark:/light: тематика, селекторы платформ (ios:/android:/native:/web:/tv:), data-[prop=value] селекторы, адаптивные точки останова (sm:/md:/lg:), варианты Tailwind, варианты компонента tv(), ScopedTheme, Uniwind.setTheme, Uniwind.updateCSSVariables, Uniwind.updateInsets, тема @layer, @variant, директива @theme, @theme static, @utility, переменные CSS в React Native, шаблон префикса акцента, colorClassName,tintColorClassName, contentContainerClassName, Uniwind Pro (анимация, переходы, теневое дерево, собственные вставки), утилиты безопасных зон (p-safe/pt-safe/-safe-or-/-safe-offset-), градиенты в RN, пользовательские классы CSS, hairlineWidth(), FontScale(), PixelRatio(), Light-dark(), цвета OKLCH, утилита cn, Tailwind-Merge, HeroUI Native, React-Native-Reusables, Gluestack. Также срабатывает при: «стили не применяются», «имя класса не работает», «проверить настройку Uniwind», «проверить мою конфигурацию», «добавить компонент», «создать стиль компонента». НЕ обрабатывает миграцию с NativeWind на Uniwind — для этого используйте навык миграции-nativewind-to-uniwind.
