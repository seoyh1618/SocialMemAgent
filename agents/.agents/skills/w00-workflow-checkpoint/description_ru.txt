Основные возможности контрольной точки рабочего процесса (с упором на сохранение и загрузку): записывайте ход выполнения контрольной точки и контекст загрузки в GitHub Issues. Подходит для любого этапа рабочего процесса, поддерживает автоматический запуск и частый вызов вручную. Ключевые слова: сохранение, загрузка, контрольная точка, выпуск.
