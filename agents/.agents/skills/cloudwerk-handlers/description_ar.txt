أنماط المعالج والبرامج الوسيطة والمكونات لتطبيقات Cloudwerk على Cloudflare Workers. يُستخدم عند كتابة معالجات المسار أو البرامج الوسيطة أو مكونات الصفحة/التخطيط أو المكونات التفاعلية من جانب العميل. تشغيل المهام التي تتضمن CloudwerkHandler، أو مساعدي الاستجابة (json، أو إعادة التوجيه)، أو روابط Cloudflare (D1، KV، R2)، أو تكوين البرامج الوسيطة، أو PageProps، أو LayoutProps، أو توجيهات العميل.
