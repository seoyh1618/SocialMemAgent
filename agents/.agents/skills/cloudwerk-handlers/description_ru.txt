Шаблоны обработчиков, промежуточного программного обеспечения и компонентов для приложений Cloudwerk на Cloudflare Workers. Используйте при написании обработчиков маршрутов, промежуточного программного обеспечения, компонентов страницы/макета или интерактивных компонентов на стороне клиента. Запускается по задачам, включающим CloudwerkHandler, помощники ответа (json, перенаправление), привязки Cloudflare (D1, KV, R2), композицию промежуточного программного обеспечения, PageProps, LayoutProps или клиентские директивы.
