Создавайте приложения CLI с помощью React. Используйте при создании пользовательского интерфейса терминала, обработке ввода с клавиатуры или создании интерактивных инструментов командной строки с компонентами React. Поддерживает макеты flexbox, стили текста, управление фокусом и специальные возможности.
