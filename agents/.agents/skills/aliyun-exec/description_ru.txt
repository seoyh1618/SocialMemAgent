Проанализируйте конфигурацию анализа требований пользователя (формат JSON), сгенерируйте и выведите список задач TODO. Используйте этот навык, когда пользователь предоставляет конфигурацию анализа требований к ресурсам Alibaba Cloud и ему необходимо преобразовать конфигурацию JSON в список исполняемых задач. Поддерживает только операции запроса ресурсов Alibaba Cloud; отказывается от любых требований к эксплуатации, связанных с изменениями. Выполняйте запросы с помощью инструмента командной строки Aliyun CLI и получайте справку по параметрам через справку aliyun.
