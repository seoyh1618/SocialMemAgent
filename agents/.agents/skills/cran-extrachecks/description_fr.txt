Préparez les packages R pour la soumission CRAN en vérifiant les exigences ad hoc courantes non prises en compte par devtools::check(). À utiliser lorsque : (1) préparer un package pour la première version du CRAN, (2) préparer une mise à jour du package pour une nouvelle soumission du CRAN, (3) examiner un package pour garantir la conformité du CRAN, (4) répondre aux commentaires des réviseurs du CRAN. Couvre les exigences en matière de documentation, les normes du champ DESCRIPTION, la validation d'URL, les exemples et les exigences administratives.
