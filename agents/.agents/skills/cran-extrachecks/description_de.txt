Bereiten Sie R-Pakete für die CRAN-Übermittlung vor, indem Sie nach allgemeinen Ad-hoc-Anforderungen suchen, die von devtools::check() nicht erfasst werden. Wird verwendet, wenn: (1) ein Paket für die erste CRAN-Veröffentlichung vorbereitet wird, (2) ein Paketupdate für die erneute Übermittlung an CRAN vorbereitet wird, (3) ein Paket überprüft wird, um die CRAN-Konformität sicherzustellen, (4) auf das Feedback von CRAN-Prüfern reagiert wird. Behandelt Dokumentationsanforderungen, Feldstandards für BESCHREIBUNGEN, URL-Validierung, Beispiele und administrative Anforderungen.
