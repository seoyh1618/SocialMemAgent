Prepare paquetes R para el envío CRAN verificando los requisitos ad-hoc comunes que no detecta devtools::check(). Úselo cuando: (1) Preparar un paquete para la primera versión de CRAN, (2) Preparar una actualización de paquete para el reenvío de CRAN, (3) Revisar un paquete para garantizar el cumplimiento de CRAN, (4) Responder a los comentarios de los revisores de CRAN. Cubre requisitos de documentación, estándares de campo DESCRIPCIÓN, validación de URL, ejemplos y requisitos administrativos.
