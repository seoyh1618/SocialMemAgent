Angular 21 واجهة أمامية مستقلة مع بنية نظيفة، وشريحة عمودية (الصفحات حسب الميزة)، والإشارات، وOnPush، وng-zorro. يُستخدم عند إنشاء تطبيقات Angular أو إعادة هيكلتها، أو إضافة صفحات، أو مكونات، أو خدمات، أو توجيهات، أو مسارات، أو عندما يذكر المستخدم الواجهة الأمامية Angular، أو المكونات المستقلة، أو الإشارات.
