استخدم هذه المهارة دائمًا عند إنشاء طلبات السحب أو تحديثها - لا تقم أبدًا بإنشاء أو تعديل العلاقات العامة مباشرة بدونها. يتبع اصطلاحات Sentry لعناوين العلاقات العامة والأوصاف ومراجع المشكلات. قم بتشغيل أي إنشاء علاقات عامة، أو فتح علاقات عامة، أو إرسال علاقات عامة، أو إجراء علاقات عامة، أو تحديث عنوان العلاقات العامة، أو تحديث وصف العلاقات العامة، أو تحرير العلاقات العامة، أو دفع وإنشاء العلاقات العامة، أو إعداد التغييرات لمهمة المراجعة، أو طلب كاتب علاقات عامة.
