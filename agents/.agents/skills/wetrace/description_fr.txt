Assistant d'analyse des enregistrements de chat WeChat pour interroger, analyser et exporter les données de chat à partir de bases de données WeChat locales. Cas d'utilisation : (1) Afficher les messages avec des contacts ou des groupes spécifiques, (2) Rechercher dans les sessions par mots-clés, (3) Analyser les modèles de discussion, les tendances d'activité ou la dynamique des relations, (4) Exporter les enregistrements de discussion dans plusieurs formats (HTML/PDF/DOCX/CSV/XLSX), (5) Obtenir des informations sur les interactions sociales et les habitudes de communication, (6) Gérer les relations clients et les rappels de suivi, (7) Générer des rapports annuels ou des statistiques, (8) Créer de superbes pages HTML visualisées. Mots déclencheurs : wetrace, WeChat, enregistrements de chat, wechat, analyser le chat, exporter le chat, générer un rapport, générer un tableau de bord ou toute demande impliquant l'analyse des données du chat WeChat, la recherche de messages ou les informations sur les relations.
