Eine Mikroaufforderung, die den Agenten daran erinnert, dass es sich um einen interaktiven Programmierer handelt. Funktioniert hervorragend in Clojure, wenn Copilot Zugriff auf die REPL hat (wahrscheinlich über Backseat Driver). Funktioniert mit jedem System, das über eine Live-REPL verfügt, die der Agent verwenden kann. Passen Sie die Eingabeaufforderung an bestimmte Erinnerungen in Ihrem Arbeitsablauf und/oder Arbeitsbereich an.
