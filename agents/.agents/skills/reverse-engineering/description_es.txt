Habilidad para crear informes de ingeniería inversa para software. Investiga la estructura y funcionalidad del repositorio y produce informes que aclaran las especificaciones externas, la implementación interna y el uso. Úselo para solicitudes como "crear un informe de ingeniería inversa", "analizar el código base", "investigar la estructura del proyecto" o "analizar cómo funciona el sistema". Ayuda a incorporar nuevos desarrolladores, comprender proyectos, crear documentación y establecer material fundamental para el desarrollo futuro.
