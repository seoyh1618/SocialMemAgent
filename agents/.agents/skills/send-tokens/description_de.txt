Senden oder übertragen Sie Token an jede Adresse in einer unterstützten Kette (EVM oder Solana). Verwenden Sie diese Option, wenn Sie oder der Benutzer Geld senden, jemanden bezahlen, Token übertragen, Trinkgeld geben, spenden oder Geld an eine andere Wallet-Adresse übertragen möchten. Umfasst Phrasen wie „10 USDC senden an“, „0x bezahlen ...“, „ETH übertragen an“, „Tokens in meine andere Wallet verschieben“.
