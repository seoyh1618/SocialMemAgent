Fügen Sie Temps-Analysen zu React-Anwendungen mit umfassenden Tracking-Funktionen hinzu, darunter Seitenaufrufe, benutzerdefinierte Ereignisse, Scroll-Tracking, Engagement-Überwachung, Sitzungsaufzeichnung und Web Vitals-Leistungsmetriken. Verwenden Sie diese Option, wenn der Benutzer Folgendes möchte: (1) Analysen zu einer React-App hinzufügen (Next.js App Router, Next.js Pages Router, Vite, Create React App oder Remix), (2) Benutzerereignisse oder -interaktionen verfolgen, (3) Scrolltiefe oder Elementsichtbarkeit überwachen, (4) Sitzungsaufzeichnung/Wiedergabe hinzufügen, (5) Web Vitals oder Leistungsmetriken verfolgen, (6) Benutzerinteraktion oder Zeit auf der Seite messen, (7) Benutzer für Analysen identifizieren, (8) Produkt einrichten möchte Analytik oder Telemetrie. Auslöser: „Analyse hinzufügen“, „Ereignisse verfolgen“, „Sitzungsaufzeichnung“, „Web Vitals“, „Benutzerverfolgung“, „Temps-Analyse“, „Reaktionsanalyse“.
