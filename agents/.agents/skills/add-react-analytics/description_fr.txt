Ajoutez l'analyse Temps aux applications React avec des fonctionnalités de suivi complètes, notamment les pages vues, les événements personnalisés, le suivi du défilement, la surveillance de l'engagement, l'enregistrement des sessions et les mesures de performances Web Vitals. À utiliser lorsque l'utilisateur souhaite : (1) ajouter des analyses à une application React (Routeur d'applications Next.js, Routeur de pages Next.js, Vite, Créer une application React ou Remix), (2) Suivre les événements ou les interactions de l'utilisateur, (3) Surveiller la profondeur de défilement ou la visibilité des éléments, (4) Ajouter un enregistrement/relecture de session, (5) Suivre les éléments vitaux du Web ou les mesures de performances, (6) Mesurer l'engagement des utilisateurs ou le temps passé sur la page, (7) Identifier les utilisateurs pour l'analyse, (8) Configurer des analyses de produits ou télémétrie. Déclencheurs : "ajouter des analyses", "suivre les événements", "enregistrement de session", "Web Vitals", "suivi des utilisateurs", "analyses de temps", "analyses de réaction".
