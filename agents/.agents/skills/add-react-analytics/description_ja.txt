ページビュー、カスタムイベント、スクロール追跡、エンゲージメントモニタリング、セッション記録、Web Vitals パフォーマンスメトリクスなどの包括的な追跡機能を備えた React アプリケーションに Temps 分析を追加します。ユーザーが次のことを行う場合に使用します: (1) React アプリに分析を追加する (Next.js App Router、Next.js Pages Router、Vite、Create React App、または Remix)、(2) ユーザー イベントまたはインタラクションを追跡する、(3) スクロールの深さまたは要素の可視性を監視する、(4) セッションの記録/再生を追加する、(5) Web Vitals またはパフォーマンス メトリクスを追跡する、(6) ユーザー エンゲージメントまたはページ滞在時間を測定する、(7) 分析のためにユーザーを識別する、 (8) 製品分析またはテレメトリを設定します。トリガー: 「分析の追加」、「イベントの追跡」、「セッション記録」、「Web Vitals」、「ユーザー追跡」、「一時分析」、「分析の反応」。
