أضف تحليلات Temps إلى تطبيقات React مع إمكانات التتبع الشاملة بما في ذلك مشاهدات الصفحة، والأحداث المخصصة، وتتبع التمرير، ومراقبة التفاعل، وتسجيل الجلسة، ومقاييس أداء Web Vitals. يُستخدم عندما يريد المستخدم: (1) إضافة تحليلات إلى تطبيق React (Next.js App Router، أو Next.js Pages Router، أو Vite، أو إنشاء تطبيق React، أو Remix)، (2) تتبع أحداث المستخدم أو تفاعلاته، (3) مراقبة عمق التمرير أو رؤية العنصر، (4) إضافة تسجيل/إعادة تشغيل الجلسة، (5) تتبع مؤشرات الويب أو مقاييس الأداء، (6) قياس تفاعل المستخدم أو الوقت على الصفحة، (7) تحديد المستخدمين لـ التحليلات، (8) إعداد تحليلات المنتج أو القياس عن بعد. المشغلات: "إضافة تحليلات"، "تتبع الأحداث"، "تسجيل الجلسة"، "مؤشرات الويب الحيوية"، "تتبع المستخدم"، "التحليلات المؤقتة"، "تحليلات التفاعل".
