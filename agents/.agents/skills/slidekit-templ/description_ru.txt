Преобразуйте презентации PDF в шаблоны слайдов HTML, используя подход визуального воспроизведения. Конвейер: PDF → скриншоты слайдов → Клод пишет HTML, соответствующий каждому скриншоту. Используйте, когда пользователь хочет преобразовать PDF-файл в шаблоны слайдов HTML, воспроизвести презентацию в формате HTML или создать многократно используемые шаблоны из существующих колод. Триггеры: «pdf в html», «конвертировать pdf в шаблон», «воспроизвести эту колоду как html», «создать шаблон из pdf».
