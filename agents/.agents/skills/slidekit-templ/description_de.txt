Konvertieren Sie PDF-Präsentationen mithilfe eines visuellen Reproduktionsansatzes in HTML-Folienvorlagen. Pipeline: PDF → Folien-Screenshots → Claude schreibt HTML, das zu jedem Screenshot passt. Verwenden Sie diese Option, wenn der Benutzer eine PDF-Datei in HTML-Folienvorlagen konvertieren, eine Präsentation als HTML reproduzieren oder wiederverwendbare Vorlagen aus vorhandenen Decks erstellen möchte. Auslöser: „PDF in HTML“, „PDF in Vorlage konvertieren“, „Dieses Deck als HTML reproduzieren“, „Vorlage aus PDF erstellen“.
