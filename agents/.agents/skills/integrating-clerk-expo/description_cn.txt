使用 @clerk/clerk-expo 将 Clerk 身份验证集成到 React Native Expo 应用程序中。涵盖 ClerkProvider 设置、安全令牌缓存 (expo-secure-store)、Expo Router/React Navigation 身份验证防护、自定义登录/注册流程（电子邮件/密码 + 电子邮件代码）、SSO/OAuth (useSSO)、使用 Apple 登录 (useSignInWithApple)、生物识别 (useLocalCredentials)、离线支持和生产部署白名单。当用户提及 Clerk + Expo、@clerk/clerk-expo、Expo Router auth、SSO/OAuth 重定向或在移动应用程序中部署 Clerk 时使用。
