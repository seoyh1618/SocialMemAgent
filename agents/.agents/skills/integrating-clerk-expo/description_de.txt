Integriert die Clerk-Authentifizierung in React Native Expo-Apps mithilfe von @clerk/clerk-expo. Behandelt die ClerkProvider-Einrichtung, sicheres Token-Caching (expo-secure-store), Expo Router/React Navigation-Authentifizierungsschutz, benutzerdefinierte Anmelde-/Anmeldeabläufe (E-Mail/Passwort + E-Mail-Codes), SSO/OAuth (useSSO), Anmeldung mit Apple (useSignInWithApple), Biometrie (useLocalCredentials), Offline-Support und Zulassungsliste für die Produktionsbereitstellung. Verwenden Sie diese Option, wenn der Benutzer Clerk + Expo, @clerk/clerk-expo, Expo Router Auth, SSO/OAuth-Weiterleitungen oder die Bereitstellung von Clerk in einer mobilen App erwähnt.
