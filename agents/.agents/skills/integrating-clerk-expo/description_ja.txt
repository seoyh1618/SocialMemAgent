@clark/clark-expo を使用して、Clerk 認証を React Native Expo アプリに統合します。 ClerkProvider のセットアップ、安全なトークン キャッシュ (expo-secure-store)、Expo Router/React Navigation 認証ガード、カスタム サインイン/サインアップ フロー (電子メール/パスワード + 電子メール コード)、SSO/OAuth (useSSO)、Apple でサインイン (useSignInWithApple)、生体認証 (useLocalCredentials)、オフライン サポート、実稼働環境のデプロイメント許可リストをカバーします。ユーザーが Clerk + Expo、@clark/clark-expo、Expo Router 認証、SSO/OAuth リダイレクト、またはモバイル アプリでの Clerk のデプロイについて言及する場合に使用します。
