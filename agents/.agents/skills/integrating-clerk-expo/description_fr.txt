Intègre l'authentification Clerk dans les applications React Native Expo à l'aide de @clerk/clerk-expo. Couvre la configuration de ClerkProvider, la mise en cache sécurisée des jetons (expo-secure-store), les gardes d'authentification Expo Router/React Navigation, les flux de connexion/inscription personnalisés (e-mail/mot de passe + codes de messagerie), SSO/OAuth (useSSO), la connexion avec Apple (useSignInWithApple), la biométrie (useLocalCredentials), la prise en charge hors ligne et la liste d'autorisation de déploiement en production. À utiliser lorsque l'utilisateur mentionne Clerk + Expo, @clerk/clerk-expo, Expo Router auth, les redirections SSO/OAuth ou le déploiement de Clerk dans une application mobile.
