Integra la autenticación de Clerk en las aplicaciones React Native Expo usando @clerk/clerk-expo. Cubre la configuración de ClerkProvider, almacenamiento en caché seguro de tokens (expo-secure-store), protectores de autenticación de Expo Router/React Navigation, flujos de inicio de sesión/registro personalizados (correo electrónico/contraseña + códigos de correo electrónico), SSO/OAuth (useSSO), inicio de sesión con Apple (useSignInWithApple), biometría (useLocalCredentials), soporte fuera de línea y lista de permitidos de implementación de producción. Úselo cuando el usuario mencione Clerk + Expo, @clerk/clerk-expo, autenticación de Expo Router, redirecciones SSO/OAuth o la implementación de Clerk en una aplicación móvil.
