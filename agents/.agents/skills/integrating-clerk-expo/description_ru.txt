Интегрирует аутентификацию Clerk в приложения React Native Expo с помощью @clerk/clerk-expo. Охватывает настройку ClerkProvider, безопасное кэширование токенов (expo-secure-store), средства защиты авторизации Expo Router/React Navigation, настраиваемые процессы входа и регистрации (электронная почта/пароль + коды электронной почты), SSO/OAuth (useSSO), вход в систему с помощью Apple (useSignInWithApple), биометрию (useLocalCredentials), автономную поддержку и список разрешений для производственного развертывания. Используйте, когда пользователь упоминает Clerk + Expo, @clerk/clerk-expo, аутентификацию Expo Router, перенаправления SSO/OAuth или развертывание Clerk в мобильном приложении.
