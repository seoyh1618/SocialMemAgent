Integra l'autenticazione Clerk nelle app React Native Expo utilizzando @clerk/clerk-expo. Copre la configurazione di ClerkProvider, la memorizzazione nella cache sicura dei token (expo-secure-store), le protezioni di autenticazione Expo Router/React Navigation, flussi di accesso/registrazione personalizzati (e-mail/password + codici e-mail), SSO/OAuth (useSSO), Accedi con Apple (useSignInWithApple), dati biometrici (useLocalCredentials), supporto offline e lista consentita della distribuzione di produzione. Da utilizzare quando l'utente menziona Clerk + Expo, @clerk/clerk-expo, Expo Router auth, reindirizzamenti SSO/OAuth o la distribuzione di Clerk in un'app mobile.
