يدمج مصادقة Clerk في تطبيقات React Native Expo باستخدام @clerk/clerk-expo. يغطي إعداد ClerkProvider، والتخزين المؤقت الآمن للرمز المميز (متجر آمن للعرض)، وحراس مصادقة Expo Router/React Navigation، وتدفقات تسجيل الدخول/الاشتراك المخصصة (البريد الإلكتروني/كلمة المرور + رموز البريد الإلكتروني)، وSSO/OAuth (useSSO)، وتسجيل الدخول باستخدام Apple (useSignInWithApple)، والقياسات الحيوية (useLocalCredentials)، والدعم دون اتصال، والقائمة المسموح بها لنشر الإنتاج. يُستخدم عندما يذكر المستخدم Clerk + Expo، أو @clerk/clerk-expo، أو Expo Router auth، أو عمليات إعادة توجيه SSO/OAuth، أو نشر Clerk في تطبيق جوال.
