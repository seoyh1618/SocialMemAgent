Best practice, modi di dire, sicurezza e revisione del codice completi per lo scripting Nushell. Da utilizzare durante la scrittura, la revisione, il controllo o il refactoring degli script Nushell (.nu) per garantire che seguano modelli idiomatici, convenzioni di denominazione, annotazioni di tipo corrette, stile funzionale, best practice di sicurezza e principi di progettazione unici di Nushell. Attiva attività che coinvolgono script Nushell, moduli, comandi personalizzati, pipeline o qualsiasi modifica di file .nu. Aiuta anche a convertire gli script Bash/POSIX in Nushell idiomatico. Copre il sistema di tipi, la manipolazione dei dati, l'ottimizzazione delle prestazioni, il rafforzamento della sicurezza, la revisione degli script e i trucchi comuni.
