Compétence d'optimisation du cache CPU pour C/C++ et Rust. À utiliser pour diagnostiquer les échecs de cache, améliorer la disposition des données pour l'efficacité du cache, utiliser les compteurs de cache de statistiques de performances, comprendre le faux partage, la prélecture ou la structuration des dispositions de données AoS vs SoA. S'active sur les requêtes concernant les échecs de cache, les lignes de cache, le faux partage, les compteurs de cache de performances, l'optimisation de la disposition des données, la prélecture, AoS vs SoA ou les performances du cache L1/L2/L3.
