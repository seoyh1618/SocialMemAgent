مهارة تحسين ذاكرة التخزين المؤقت لوحدة المعالجة المركزية لـ C/C++ وRust. يُستخدم عند تشخيص أخطاء ذاكرة التخزين المؤقت، أو تحسين تخطيط البيانات لكفاءة ذاكرة التخزين المؤقت، أو استخدام عدادات ذاكرة التخزين المؤقت لإحصائيات الأداء، أو فهم المشاركة الزائفة، أو الجلب المسبق، أو هيكلة تخطيطات بيانات AoS مقابل SoA. يتم التنشيط عند الاستعلامات حول أخطاء ذاكرة التخزين المؤقت، أو خطوط ذاكرة التخزين المؤقت، أو المشاركة الخاطئة، أو عدادات ذاكرة التخزين المؤقت للأداء، أو تحسين تخطيط البيانات، أو الجلب المسبق، أو AoS vs SoA، أو أداء ذاكرة التخزين المؤقت L1/L2/L3.
