Capacità di ottimizzazione della cache della CPU per C/C++ e Rust. Da utilizzare per diagnosticare i mancati risultati della cache, migliorare il layout dei dati per l'efficienza della cache, utilizzare i contatori della cache delle statistiche delle prestazioni, comprendere la falsa condivisione, il precaricamento o strutturare i layout dei dati AoS e SoA. Si attiva su query relative a errori di cache, linee di cache, false condivisioni, contatori di cache di prestazioni, ottimizzazione del layout dei dati, precaricamento, AoS vs SoA o prestazioni della cache L1/L2/L3.
