Habilidad de optimización de caché de CPU para C/C++ y Rust. Úselo para diagnosticar errores de caché, mejorar el diseño de datos para la eficiencia de la caché, usar contadores de caché de estadísticas de rendimiento, comprender el intercambio falso, la captación previa o estructurar diseños de datos AoS versus SoA. Se activa en consultas sobre errores de caché, líneas de caché, uso compartido falso, contadores de caché de rendimiento, optimización del diseño de datos, captación previa, AoS vs SoA o rendimiento de caché L1/L2/L3.
