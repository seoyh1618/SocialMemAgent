قم بتسجيل الدخول إلى محفظة وكيل Finance District. يُستخدم عندما تريد أنت أو المستخدم تسجيل الدخول أو تسجيل الدخول أو الاتصال أو الإعداد أو تكوين المحفظة، أو عندما تفشل أي عملية للمحفظة مع ظهور أخطاء المصادقة أو "لم تتم المصادقة". تعد هذه المهارة شرطًا أساسيًا قبل إرسال الرموز المميزة أو المبادلة أو التحقق من الأرصدة أو أي عملية أخرى للمحفظة.
