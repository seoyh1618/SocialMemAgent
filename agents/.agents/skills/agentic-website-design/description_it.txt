Progetta e crea siti Web utilizzando agenti di codifica AI con generatori di siti statici. Copre il flusso di lavoro Astro-first, il perfezionamento visivo iterativo tramite feedback del browser, suggerimenti potenziati dalle competenze (progettazione di frontend, copywriting), animazioni e loop di lucidatura a barra alta. Da utilizzare durante la creazione di un sito Web con un agente AI, la progettazione di pagine di destinazione o l'iterazione del web design con l'assistenza LLM.
