프레임을 순차적인 비디오로 추출하고 확인하여 애니메이션 GIF 파일을 분석합니다.

다음과 같은 경우에 사용하세요:
- 사용자가 GIF 파일 경로를 언급합니다(예: "./demo.gif", "~/Downloads/animation.gif").
- GIF 애니메이션을 분석하거나 이해하고 싶은 사용자
- 사용자가 GIF의 모션, 변화 또는 콘텐츠에 대해 질문합니다.
- 사용자가 분석을 위해 .gif 파일을 첨부하거나 참조합니다.
- 사용자가 GIF 형식으로 녹화된 화면을 검사하고 싶어합니다.
- 사용자가 /gif 슬래시 명령을 호출합니다.

키워드: "GIF", ".gif", "애니메이션", "애니메이션", "프레임", "화면 녹화",
"gif 분석", "gif 분석", "gif 보기", "gif 콘텐츠", "gif 모션"

트리거 패턴:
- 자연어: "이 GIF를 분석하세요: ./demo.gif"
- 슬래시 명령: `/gif <경로>` 또는 `/gif <경로> <메시지>`

트리거되면 Python 스크립트를 사용하여 프레임을 추출하고, 프레임을 순서대로 보고,
연속적인 비디오 시퀀스로 해석됩니다.
