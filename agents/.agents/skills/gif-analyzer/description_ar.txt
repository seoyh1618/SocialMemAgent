قم بتحليل ملفات GIF المتحركة عن طريق استخراج الإطارات وعرضها كفيديو متسلسل.

استخدم عندما:
- يذكر المستخدم مسار ملف GIF (على سبيل المثال، "./demo.gif"، "~/Downloads/animation.gif")
- يريد المستخدم تحليل أو فهم الرسوم المتحركة بتنسيق GIF
- يسأل المستخدم عن الحركة أو التغييرات أو المحتوى في صورة GIF
- يقوم المستخدم بإرفاق ملف بتنسيق .gif أو الإشارة إليه لتحليله
- يريد المستخدم فحص تسجيل الشاشة بتنسيق GIF
- يقوم المستخدم باستدعاء أمر شرطة مائلة /gif

الكلمات الرئيسية: "GIF"، ".gif"، "رسوم متحركة"، "رسوم متحركة"، "إطارات"، "تسجيل الشاشة"،
"تحليل GIF"، "تحليل GIF"، "عرض GIF"، "محتوى GIF"، "حركة GIF"

أنماط الزناد:
- اللغة الطبيعية: "تحليل ملف GIF هذا: ./demo.gif"
- أمر الشرطة المائلة: `/gif <path>` أو `/gif <path> <message>`

عند تشغيلها، قم باستخراج الإطارات باستخدام برنامج Python النصي، وعرض الإطارات بالترتيب،
وتفسيرها على أنها تسلسل فيديو مستمر.
