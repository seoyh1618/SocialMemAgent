Анализируйте анимированные файлы GIF, извлекая и просматривая кадры как последовательное видео.

Используйте, когда:
– Пользователь указывает путь к файлу GIF (например, «./demo.gif», «~/Downloads/animation.gif»).
- Пользователь хочет проанализировать или понять GIF-анимацию.
- Пользователь спрашивает о движении, изменениях или содержимом GIF-файла.
- Пользователь прикрепляет или ссылается на файл .gif для анализа.
- Пользователь хочет просмотреть запись экрана в формате GIF.
- Пользователь вызывает команду косой черты /gif.

Ключевые слова: «GIF», «.gif», «анимация», «анимированные», «кадры», «запись экрана»,
«анализировать gif», «анализ GIF», «просмотреть GIF», «содержимое GIF», «движение GIF»

Триггерные шаблоны:
- Естественный язык: «Проанализируйте этот GIF: ./demo.gif»
- Команда слэша: `/gif <путь>` или `/gif <путь> <сообщение>`

При срабатывании извлекайте кадры с помощью скрипта Python, просматривайте кадры по порядку,
и интерпретировать как непрерывный видеоряд.
