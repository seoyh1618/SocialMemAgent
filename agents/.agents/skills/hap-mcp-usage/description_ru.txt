Навыки автоматической настройки Mingdao Cloud HAP MCP. Запускается **немедленно**, когда пользователи упоминают «настроить MCP», «HAP MCP», «конфигурация MCP», «добавить MCP», «подключение MCP» и т. д. Поддерживает автоматическую настройку для 9 инструментов искусственного интеллекта с автоматической проверкой подключения по завершении.
