Вызов API создания видео RawUGC для создания и управления видео AI (Sora 2 и другие модели). Используйте, когда пользователь хочет создавать AI-видео через RawUGC, интегрировать RawUGC API, проверять статус видео, составлять список видео или работать с Sora/генерацией текста в видео/изображения в видео.
