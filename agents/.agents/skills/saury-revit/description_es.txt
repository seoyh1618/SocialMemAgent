Cree un proyecto de complemento de Revit basado en Saury.Revit.Template. Se activa cuando el usuario solicita crear un complemento de Revit, iniciar un nuevo proyecto de Revit, inicializar un entorno de desarrollo de Revit o utilizar la plantilla saury-revit. Cubre escenarios: (1) crear un nuevo proyecto de complemento de Revit, (2) instalar/actualizar la plantilla saury-revit dotnet, (3) agregar nuevas funciones (Comando/Ver/VerModelo/Servicio) a un proyecto existente. Palabras clave: Revit, complemento, complemento, saury-revit, dotnet new.
