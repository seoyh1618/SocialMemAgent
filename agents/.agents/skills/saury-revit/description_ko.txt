Saury.Revit.Template을 기반으로 Revit 애드인 프로젝트를 생성합니다. 사용자가 Revit 애드인 생성, 새 Revit 프로젝트 시작, Revit 개발 환경 초기화 또는 saury-revit 템플릿 사용을 요청할 때 트리거됩니다. 다루는 시나리오: (1) 새 Revit 애드인 프로젝트 생성, (2) saury-revit dotnet 템플릿 설치/업데이트, (3) 기존 프로젝트에 새 기능(Command/View/ViewModel/Service) 추가. 키워드: Revit, 추가 기능, 플러그인, saury-revit, dotnet new.
