Interactúe con el servicio IIB (Infinite Image Browsing) para buscar, explorar, etiquetar y organizar imágenes generadas por IA. Úselo cuando el usuario necesite buscar imágenes por mensaje/palabra clave, administrar etiquetas de imágenes, organizar archivos en carpetas, obtener parámetros de generación de imágenes o trabajar con una biblioteca de imágenes.
