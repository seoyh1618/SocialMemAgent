Взаимодействуйте со службой IIB (бесконечный просмотр изображений) для поиска, просмотра, маркировки и организации изображений, созданных искусственным интеллектом. Используйте, когда пользователю необходимо выполнить поиск изображений по запросу/ключевому слову, управлять тегами изображений, упорядочить файлы по папкам, получить параметры генерации изображений или работать с библиотекой изображений.
