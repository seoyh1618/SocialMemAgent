Interagisci con il servizio IIB (Infinite Image Browsing) per cercare, sfogliare, taggare e organizzare le immagini generate dall'intelligenza artificiale. Da utilizzare quando l'utente ha bisogno di cercare immagini per prompt/parola chiave, gestire tag di immagine, organizzare file in cartelle, ottenere parametri di generazione di immagini o lavorare con una libreria di immagini.
