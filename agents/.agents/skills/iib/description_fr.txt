Interagissez avec le service IIB (Infinite Image Browsing) pour rechercher, parcourir, marquer et organiser les images générées par l'IA. À utiliser lorsque l'utilisateur doit rechercher des images par invite/mot-clé, gérer les balises d'image, organiser les fichiers dans des dossiers, obtenir des paramètres de génération d'images ou travailler avec une bibliothèque d'images.
