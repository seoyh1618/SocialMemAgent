Referencia CLI de FFmpeg para procesamiento de video y audio, conversión de formato, filtrado y automatización de medios. Úselo al convertir formatos de video, cambiar el tamaño o recortar video, recortar por tiempo, reemplazar o extraer audio, mezclar pistas de audio, superponer texto o imágenes, grabar subtítulos, crear GIF, generar miniaturas, crear presentaciones de diapositivas, cambiar la velocidad de reproducción, codificar con H264/H265/VP9, configurar CRF/bitrate, usar aceleración de GPU, crear guiones gráficos o ejecutar ffprobe. Cubre filter_complex, selectores de flujo, -map, -c copy, seek, escala, pad, crop, concat, drawtext, zoompan, xfade.
