مرجع FFmpeg CLI لمعالجة الفيديو والصوت، وتحويل التنسيق، والتصفية، وأتمتة الوسائط. يُستخدم عند تحويل تنسيقات الفيديو، أو تغيير حجم الفيديو أو اقتصاصه، أو القطع حسب الوقت، أو استبدال الصوت أو استخراجه، أو خلط المسارات الصوتية، أو تراكب النص أو الصور، أو نسخ الترجمات، أو إنشاء صور GIF، أو إنشاء صور مصغرة، أو إنشاء عروض شرائح، أو تغيير سرعة التشغيل، أو التشفير باستخدام H264/H265/VP9، أو ضبط CRF/معدل البت، أو استخدام تسريع GPU، أو إنشاء القصص المصورة، أو تشغيل ffprobe. يغطي مرشح_معقد، محددات الدفق، -map، -c نسخة، تسعى، مقياس، لوحة، المحاصيل، concat، drawtext، Zoompan، xfade.
