FFmpeg-CLI-Referenz für Video- und Audioverarbeitung, Formatkonvertierung, Filterung und Medienautomatisierung. Zum Konvertieren von Videoformaten, Ändern der Größe oder Zuschneiden von Videos, Trimmen nach Zeit, Ersetzen oder Extrahieren von Audio, Mischen von Audiospuren, Überlagern von Text oder Bildern, Brennen von Untertiteln, Erstellen von GIFs, Generieren von Miniaturansichten, Erstellen von Diashows, Ändern der Wiedergabegeschwindigkeit, Codieren mit H264/H265/VP9, Festlegen von CRF/Bitrate, Verwenden der GPU-Beschleunigung, Erstellen von Storyboards oder Ausführen von ffprobe. Deckt filter_complex, Stream-Selektoren, -map, -c copy, Seeking, Scale, Pad, Crop, Concat, Drawtext, Zoompan, Xfade ab.
