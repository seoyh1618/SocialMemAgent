Справочник по CLI FFmpeg для обработки видео и аудио, преобразования форматов, фильтрации и автоматизации мультимедиа. Используйте при конвертации видеоформатов, изменении размера или обрезки видео, обрезке по времени, замене или извлечении звука, микшировании звуковых дорожек, наложении текста или изображений, записи субтитров, создании GIF-файлов, создании миниатюр, создании слайд-шоу, изменении скорости воспроизведения, кодировании с помощью H264/H265/VP9, настройке CRF/битрейта, использовании ускорения графического процессора, создании раскадровок или запуске ffprobe. Охватывает filter_complex, селекторы потоков, -map, -c копирование, поиск, масштабирование, заполнение, обрезку, объединение, рисование текста, масштабирование, xfade.
