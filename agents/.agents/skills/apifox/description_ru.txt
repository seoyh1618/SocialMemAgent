Инструмент запроса документации ABC Medical Cloud API. Считывает и запрашивает документ спецификации OpenAPI ABC API (4209 интерфейсов), поддерживая поиск по модулю, пути и методу с автоматическим разрешением $ref. Варианты использования: (1) Запрос определений интерфейса API (2) Поиск конкретных функциональных интерфейсов (3) Экспорт сводной документации интерфейса (4) Просмотр статистики интерфейса
