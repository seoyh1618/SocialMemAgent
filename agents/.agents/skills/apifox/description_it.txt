Strumento di query sulla documentazione API di ABC Medical Cloud. Legge ed esegue query sul documento delle specifiche OpenAPI dell'API ABC (4.209 interfacce), supportando ricerche per modulo, percorso e metodo, con risoluzione automatica $ref. Casi d'uso: (1) Query sulle definizioni dell'interfaccia API (2) Ricerca di interfacce funzionali specifiche (3) Esportazione di riepiloghi della documentazione dell'interfaccia (4) Visualizzazione delle statistiche dell'interfaccia
