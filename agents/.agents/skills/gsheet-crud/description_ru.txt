Руководство пользователя API GSheet-CRUD. Используйте Google Таблицы в качестве базы данных RESTful API. Используйте этот навык, когда пользователям необходимо манипулировать данными Google Таблиц через API, включая: запрос данных (GET), вставку данных (POST), обновление данных (PUT), удаление данных (DELETE).
