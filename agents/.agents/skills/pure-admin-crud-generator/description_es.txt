Genere páginas CRUD y módulos de enrutador para un administrador puro a partir de definiciones de API locales de swagger. DEBE usarse siempre que necesite crear una lista de administración/editar/páginas de detalles, vistas del panel o configuraciones de ruta desde métodos API existentes en src/api/swagger/Api.ts. Esta habilidad reemplaza la creación manual de páginas Vue; úsela para cualquier tarea de desarrollo del panel de administración que involucre páginas basadas en API.
