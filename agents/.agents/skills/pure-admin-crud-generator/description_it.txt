Genera pagine CRUD e moduli router per pure-admin-thin dalle definizioni API spavalderia locali. DEVE essere utilizzato ogni volta che è necessario eseguire l'impalcatura di elenchi di amministrazione/modifica/pagine di dettaglio, visualizzazioni di dashboard o instradare configurazioni da metodi API esistenti in src/api/swagger/Api.ts. Questa competenza sostituisce la creazione manuale della pagina Vue: utilizzala per qualsiasi attività di sviluppo del pannello di amministrazione che coinvolga pagine basate su API.
