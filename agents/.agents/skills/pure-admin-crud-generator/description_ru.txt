Создавайте страницы CRUD и модули маршрутизатора для чистого администрирования на основе локальных определений API Swagger. ДОЛЖЕН использоваться всякий раз, когда вам нужно создать шаблоны страниц списка/редактирования/подробных сведений администратора, представлений информационной панели или конфигураций маршрутизации из существующих методов API в src/api/swagger/Api.ts. Этот навык заменяет создание страниц Vue вручную — используйте его для любой задачи разработки панели администратора, включающей страницы, управляемые API.
