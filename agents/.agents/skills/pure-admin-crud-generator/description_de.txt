Generieren Sie CRUD-Seiten und Router-Module für Pure-Admin-Thin aus lokalen Swagger-API-Definitionen. MUSS immer dann verwendet werden, wenn Sie Admin-Listen-/Bearbeitungs-/Detailseiten, Dashboard-Ansichten erstellen oder Konfigurationen aus vorhandenen API-Methoden in src/api/swagger/Api.ts weiterleiten müssen. Diese Fähigkeit ersetzt die manuelle Vue-Seitenerstellung – verwenden Sie sie für jede Admin-Panel-Entwicklungsaufgabe, die API-gesteuerte Seiten umfasst.
