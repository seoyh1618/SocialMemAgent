Навыки работы с серверной частью PocketBase через REST API и пакетный режим Go. Обеспечивает сбор CRUD, запись CRUD, аутентификацию суперпользователя/пользователя, резервное копирование и восстановление, генерацию файлов миграции (JS и Go), перехватчики Go, настраиваемые маршруты и рекомендации по проектированию правил API, отношений и шаблонов безопасности. Используйте для запросов, связанных с PocketBase, pb_migrations, управлением коллекциями, операциями с записями, внедрением инфраструктуры Go и дизайном серверной части.
