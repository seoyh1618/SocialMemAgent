مهارة تشغيل واجهة PocketBase الخلفية عبر REST API ووضع حزمة Go. يوفر مجموعة CRUD، وتسجيل CRUD، ومصادقة المستخدم المتميز/المستخدم، والنسخ الاحتياطي والاستعادة، وإنشاء ملفات الترحيل (JS وGo)، وخطافات Go، والمسارات المخصصة، وإرشادات التصميم لقواعد API والعلاقات وأنماط الأمان. يُستخدم للطلبات المتعلقة بـ PocketBase وpb_migrations وإدارة المجموعة وعمليات التسجيل وتضمين إطار عمل Go وتصميم الواجهة الخلفية.
