Competenza per il funzionamento del backend PocketBase tramite API REST e modalità pacchetto Go. Fornisce raccolta CRUD, registrazione CRUD, autenticazione superutente/utente, backup e ripristino, generazione di file di migrazione (JS e Go), hook Go, percorsi personalizzati e indicazioni di progettazione per regole API, relazioni e modelli di sicurezza. Utilizzare per richieste relative a PocketBase, pb_migrations, gestione delle raccolte, operazioni sui record, incorporamento del framework Go e progettazione backend.
