Habilidad para operar el backend de PocketBase a través de API REST y modo de paquete Go. Proporciona CRUD de recopilación, CRUD de registros, autenticación de superusuario/usuario, copia de seguridad y restauración, generación de archivos de migración (JS y Go), enlaces Go, rutas personalizadas y orientación de diseño para reglas, relaciones y patrones de seguridad de API. Úselo para solicitudes relacionadas con PocketBase, pb_migrations, gestión de colecciones, operaciones de registros, integración del marco Go y diseño de backend.
