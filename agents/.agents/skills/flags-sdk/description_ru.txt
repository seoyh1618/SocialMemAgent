Подробное руководство по реализации флагов функций и A/B-тестов с использованием Flags SDK (пакет npm `flags`). Используйте, когда: (1) Создание или объявление флагов функций с помощью flag(), (2) Настройка поставщиков/адаптеров флагов функций (Vercel, Statsig, LaunchDarkly, PostHog, GrowthBook, Hypertune, Edge Config, OpenFeature, Flagsmith, Reflag, Split, Optimizely или пользовательские адаптеры), (3) Реализация шаблонов предварительных вычислений для статических страниц с флагами функций, (4) Настройка контекста оценки с помощью `identify` и `dedupe`, (5) Интеграция Flags Explorer / панели инструментов Vercel, (6) Работа с флагами функций в Next.js (App Router, Pages Router, Middleware) или SvelteKit, (7) Написание пользовательских адаптеров, (8) Шифрование/дешифрование значений флагов для панели инструментов, (9) Любая задача, связанная с `flags`, `flags/next`, `flags/sveltekit`, `flags/react` или `@flags-sdk/*`. Срабатывает по: флагам функций, A/B-тестированию, экспериментам, SDK флагов, адаптерам флагов, флагам предварительных вычислений, обозревателю флагов, шлюзам функций, переопределениям флагов.
