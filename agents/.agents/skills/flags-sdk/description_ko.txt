Flags SDK('flags' npm 패키지)를 사용하여 기능 플래그 및 A/B 테스트를 구현하기 위한 종합 가이드입니다. 사용 시기: (1) `flag()`를 사용하여 기능 플래그 생성 또는 선언, (2) 기능 플래그 공급자/어댑터 설정(Vercel, Statsig, LaunchDarkly, PostHog, GrowthBook, Hypertune, Edge Config, OpenFeature, Flagsmith, Reflag, Split, Optimizely 또는 사용자 정의 어댑터), (3) 기능 플래그가 있는 정적 페이지에 대한 사전 계산 패턴 구현, (4) `identify`를 사용하여 평가 컨텍스트 설정 및 `dedupe`, (5) 플래그 탐색기/Vercel 도구 모음 통합, (6) Next.js(App Router, Pages Router, Middleware) 또는 SvelteKit의 기능 플래그 작업, (7) 사용자 정의 어댑터 작성, (8) 도구 모음의 플래그 값 암호화/복호화, (9) `flags`, `flags/next`, `flags/sveltekit`, `flags/react` 또는 `@flags-sdk/*` 패키지. 트리거 대상: 기능 플래그, A/B 테스트, 실험, 플래그 SDK, 플래그 어댑터, 사전 계산 플래그, 플래그 탐색기, 기능 게이트, 플래그 재정의.
