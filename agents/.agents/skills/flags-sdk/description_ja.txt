Flags SDK (「flags」 npm パッケージ) を使用して機能フラグと A/B テストを実装するための包括的なガイド。次の場合に使用します: (1) `flag()` を使用して機能フラグを作成または宣言する、(2) 機能フラグ プロバイダー/アダプター (Vercel、Statsig、LaunchDarkly、PostHog、GrowthBook、Hypertune、Edge Config、OpenFeature、Flagsmith、Reflag、Split、Optimizely、またはカスタム アダプター) を設定する、(3) 機能フラグを使用して静的ページの事前計算パターンを実装する、 (4) `identify` および `dedupe` を使用した評価コンテキストの設定、(5) Flags Explorer / Vercel ツールバーの統合、(6) Next.js (App Router、Pages Router、Middleware) または SvelteKit での機能フラグの操作、(7) カスタム アダプターの作成、(8) ツールバーのフラグ値の暗号化/復号化、(9) `flags`、`flags/next` に関連するタスク、 `flags/sveltekit`、`flags/react`、または `@flags-sdk/*` パッケージ。トリガー: 機能フラグ、A/B テスト、実験、フラグ SDK、フラグ アダプター、事前計算フラグ、フラグ エクスプローラー、機能ゲート、フラグ オーバーライド。
