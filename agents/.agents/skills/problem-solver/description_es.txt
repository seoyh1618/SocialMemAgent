Resolución de problemas paso a paso con explicaciones detalladas. Utiliza un enfoque estructurado (identificar, planificar, ejecutar, verificar) para problemas de matemáticas, física, química y informática. Puede proporcionar sugerencias antes de encontrar soluciones. Úselo al resolver problemas académicos, resolver preguntas de práctica o aprender métodos de resolución de problemas. Desencadenantes: resuelva el problema, paso a paso, muestre el trabajo, explique la solución, cómo resolverlo, ayuda con el problema.
