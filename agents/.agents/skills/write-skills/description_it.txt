Crea competenze agente di alta qualità (file SKILL.md) che seguono la specifica aperta delle competenze agente. Da utilizzare quando viene richiesto di creare una competenza, scrivere un file SKILL.md, creare pacchetti di istruzioni per l'agente, creare una funzionalità riutilizzabile o trasformare un flusso di lavoro in una competenza condivisibile. Produce competenze conformi alle specifiche, concise e ben strutturate, ottimizzate per l'individuazione e l'esecuzione degli agenti.
