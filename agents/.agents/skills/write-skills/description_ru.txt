Создает высококачественные навыки агента (файлы SKILL.md), соответствующие открытой спецификации навыков агента. Используйте, когда вас попросят создать навык, написать SKILL.md, упаковать инструкции для агента, создать возможность многократного использования или превратить рабочий процесс в общий навык. Формирует соответствующие спецификациям, краткие, хорошо структурированные навыки, оптимизированные для обнаружения и выполнения агентов.
