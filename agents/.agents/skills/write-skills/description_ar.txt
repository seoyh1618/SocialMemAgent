ينشئ مهارات الوكيل عالية الجودة (ملفات SKILL.md) التي تتبع مواصفات مهارات الوكيل المفتوحة. استخدمها عندما يُطلب منك إنشاء مهارة، أو كتابة SKILL.md، أو حزم تعليمات الوكيل، أو بناء قدرة قابلة لإعادة الاستخدام، أو تحويل سير العمل إلى مهارة قابلة للمشاركة. ينتج مهارات متوافقة مع المواصفات وموجزة ومنظمة بشكل جيد ومُحسّنة لاكتشاف الوكيل وتنفيذه.
