Erstellt hochwertige Agent Skills (SKILL.md-Dateien), die der offenen Agent Skills-Spezifikation folgen. Verwenden Sie es, wenn Sie aufgefordert werden, einen Skill zu erstellen, ein SKILL.md zu schreiben, Agentenanweisungen zu verpacken, eine wiederverwendbare Funktion zu erstellen oder einen Workflow in einen gemeinsam nutzbaren Skill umzuwandeln. Erstellt spezifikationskonforme, prägnante und gut strukturierte Fertigkeiten, die für die Erkennung und Ausführung von Agenten optimiert sind.
