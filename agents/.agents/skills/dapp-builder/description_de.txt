Erstellen Sie dezentrale Anwendungen auf Freenet mit River als Vorlage. Leitet durch die Gestaltung von Verträgen (gemeinsamer Status), Delegierten (privater Status) und der Benutzeroberfläche. Verwenden Sie diese Option, wenn der Benutzer eine neue Freenet-dApp erstellen, einen Vertragsstatus entwerfen, Delegaten implementieren oder eine mit Freenet verbundene Benutzeroberfläche erstellen möchte.
