Progetta e genera schemi di database Go + Ent per progetti di layout di sfere dai requisiti. Da utilizzare quando gli utenti descrivono modelli di dati, relazioni tra entità, tabelle di database o modifiche allo schema, inclusi prompt, documenti Markdown, file di proto/servizi esistenti o comportamenti dimostrativi. Produce piani di schemi pronti per l'implementazione con definizioni di campi, strategie di ID, progettazioni di relazioni, indici, annotazioni di entproto e indicazioni sull'integrazione di associazione/rendering/servizio.
