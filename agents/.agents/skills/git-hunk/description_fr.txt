Mise en scène de morceaux non interactive pour git. À utiliser lorsque :
- Mise en scène de morceaux individuels par hachage de contenu
- Liste des mecs non mis en scène ou mis en scène
- Scripter ou automatiser le staging partiel de git
- Travailler avec les alternatives git add -p pour les LLM/scripts
Déclencheurs : "git hunk", "stage hunk", "hunk staging", "partial staging", "git-hunk", "stage by hash"
