Неинтерактивная подготовка фрагментов для git. Используйте, когда:
- Создание отдельных фрагментов различий по хэшу контента.
- Листинг неподготовленных или подготовленных кусков
- Создание сценариев или автоматизация частичной подготовки git.
- Работа с альтернативами git add -p для LLM/скриптов.
Триггеры: «git hunk», «этапный кусок», «постановочный кусок», «частичная постановка», «git-hunk», «поэтапный хэш»
