Nicht interaktives Hunk-Staging für Git. Verwenden Sie, wenn:
- Staging einzelner Diff-Hunks nach Inhalts-Hash
- Auflistung nicht inszenierter oder inszenierter Kerle
- Skripterstellung oder Automatisierung von teilweisem Git-Staging
– Arbeiten mit git add -p-Alternativen für LLMs/Skripte
Auslöser: „git hunk“, „stage hunk“, „hunk staging“, „partial staging“, „git-hunk“, „stage by hash“
