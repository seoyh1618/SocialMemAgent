Найдите справочную документацию, руководства и страницы конфигурации CE.SDK Node.js.

Используйте, когда пользователю нужна документация CE.SDK для Node.js — конфигурация, безголовый рендеринг,
параметры экспорта, пакетная обработка или инструкции по началу работы. Также запускается "IMG.LY", "CreativeEditor",
«CE.SDK» или «cesdk», если пользователю нужна существующая страница документа Node.js.

Не для написания кода (используйте сборку) или объяснения концепций (используйте объяснение).

<пример>
Контекст: пользователь спрашивает о конфигурации Node.js.
пользователь: «Как мне настроить движок в Node.js?»
помощник: «Я буду использовать /cesdk:docs-node для поиска параметров конфигурации».
</пример>

<пример>
Контекст: пользователю необходима генерация на стороне сервера.
пользователь: «Как использовать CE.SDK для создания образов на стороне сервера?»
помощник: «Позвольте мне использовать /cesdk:docs-node, чтобы найти соответствующую документацию».
</пример>
