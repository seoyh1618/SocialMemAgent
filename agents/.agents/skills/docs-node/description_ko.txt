CE.SDK Node.js 참조 문서, 가이드 및 구성 페이지를 찾아보세요.

사용자가 Node.js용 CE.SDK 문서가 필요할 때 사용 — 구성, 헤드리스 렌더링,
내보내기 옵션, 일괄 처리 또는 시작 지침. 또한 "IMG.LY", "CreativeEditor"에 의해 트리거됩니다.
사용자에게 기존 Node.js 문서 페이지가 필요한 경우 "CE.SDK" 또는 "cesdk"입니다.

코드 작성(빌드 사용) 또는 개념 설명(설명 사용)용이 아닙니다.

<예>
컨텍스트: 사용자가 Node.js 구성에 대해 질문합니다.
user: "Node.js에서 엔진을 어떻게 구성하나요?"
조수: "구성 옵션을 조회하려면 /cesdk:docs-node를 사용하겠습니다."
</예>

<예>
컨텍스트: 사용자에게 서버측 생성이 필요합니다.
user: "서버 측 이미지 생성을 위해 CE.SDK를 어떻게 사용합니까?"
조수: "관련 문서를 찾으려면 /cesdk:docs-node를 사용하겠습니다."
</예>
