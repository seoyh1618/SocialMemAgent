ابحث عن المستندات والأدلة وصفحات التكوين المرجعية لـ CE.SDK Node.js.

يُستخدم عندما يحتاج المستخدم إلى مستندات CE.SDK لـ Node.js — التكوين، والعرض بدون رأس،
خيارات التصدير، أو معالجة الدفعات، أو تعليمات البدء. تم تشغيله أيضًا بواسطة "IMG.LY"، "CreativeEditor"،
"CE.SDK" أو "cesdk" عندما يحتاج المستخدم إلى صفحة مستند Node.js موجودة.

ليس لكتابة التعليمات البرمجية (استخدم البناء) أو شرح المفاهيم (استخدم الشرح).

<مثال>
السياق: يسأل المستخدم عن تكوين Node.js
user: "كيف يمكنني تكوين المحرك في Node.js؟"
المساعد: "سأستخدم /cesdk:docs-node للبحث عن خيارات التكوين."
</مثال>

<مثال>
السياق: يحتاج المستخدم إلى إنشاء من جانب الخادم
user: "كيف يمكنني استخدام CE.SDK لإنشاء الصور من جانب الخادم؟"
المساعد: "اسمح لي باستخدام /cesdk:docs-node للعثور على الوثائق ذات الصلة."
</مثال>
