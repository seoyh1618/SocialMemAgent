Busque documentos de referencia, guías y páginas de configuración de CE.SDK Node.js.

Úselo cuando el usuario necesite documentos CE.SDK para Node.js: configuración, renderizado sin cabeza,
opciones de exportación, procesamiento por lotes o instrucciones de introducción. También activado por "IMG.LY", "CreativeEditor",
"CE.SDK" o "cesdk" cuando el usuario necesita una página de documento Node.js existente.

No para escribir código (use compilación) o explicaciones de conceptos (use explicación).

<ejemplo>
Contexto: el usuario pregunta sobre la configuración de Node.js
usuario: "¿Cómo configuro el motor en Node.js?"
asistente: "Usaré /cesdk:docs-node para buscar opciones de configuración".
</ejemplo>

<ejemplo>
Contexto: el usuario necesita generación del lado del servidor
usuario: "¿Cómo uso CE.SDK para la generación de imágenes del lado del servidor?"
asistente: "Déjame usar /cesdk:docs-node para encontrar la documentación relevante".
</ejemplo>
