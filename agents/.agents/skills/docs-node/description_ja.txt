CE.SDK Node.js のリファレンス ドキュメント、ガイド、構成ページを検索してください。

ユーザーが Node.js の CE.SDK ドキュメントを必要とする場合に使用します (構成、ヘッドレス レンダリング、
エクスポート オプション、バッチ処理、または開始手順。 「IMG.LY」、「CreativeEditor」によってもトリガーされます。
「CE.SDK」、またはユーザーが既存の Node.js ドキュメント ページを必要とする場合は「cesdk」。

コードの作成 (build を使用) や概念の説明 (explain を使用) のためではありません。

<例>
コンテキスト: ユーザーが Node.js 構成について尋ねる
ユーザー: 「Node.js でエンジンを構成するにはどうすればよいですか?」
アシスタント: 「/cesdk:docs-node を使用して構成オプションを調べます。」
</例>

<例>
コンテキスト: ユーザーはサーバー側で生成する必要がある
ユーザー: 「サーバー側のイメージ生成に CE.SDK を使用するにはどうすればよいですか?」
アシスタント: 「関連ドキュメントを見つけるには /cesdk:docs-node を使用させてください。」
</例>
