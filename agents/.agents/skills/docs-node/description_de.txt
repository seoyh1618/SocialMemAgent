Suchen Sie nach CE.SDK Node.js Referenzdokumenten, Leitfäden und Konfigurationsseiten.

Verwenden Sie diese Option, wenn der Benutzer CE.SDK-Dokumente für Node.js benötigt – Konfiguration, Headless-Rendering,
Exportoptionen, Stapelverarbeitung oder Erste-Schritte-Anweisungen. Wird auch durch „IMG.LY“, „CreativeEditor“ ausgelöst.
„CE.SDK“ oder „cesdk“, wenn der Benutzer eine vorhandene Node.js-Dokumentseite benötigt.

Nicht zum Schreiben von Code (Build verwenden) oder Konzepterklärungen (Explain verwenden).

<Beispiel>
Kontext: Der Benutzer fragt nach der Node.js-Konfiguration
Benutzer: „Wie konfiguriere ich die Engine in Node.js?“
Assistent: „Ich verwende /cesdk:docs-node, um nach Konfigurationsoptionen zu suchen.“
</Beispiel>
