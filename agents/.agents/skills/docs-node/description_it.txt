Cerca documenti di riferimento, guide e pagine di configurazione di CE.SDK Node.js.

Da utilizzare quando l'utente necessita di documenti CE.SDK per Node.js: configurazione, rendering headless,
opzioni di esportazione, elaborazione batch o istruzioni introduttive. Attivato anche da "IMG.LY", "CreativeEditor",
"CE.SDK" o "cesdk" quando l'utente necessita di una pagina di documenti Node.js esistente.

Non per scrivere codice (usa build) o spiegazioni di concetti (usa spiega).
