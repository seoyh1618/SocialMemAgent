Automazione dell'API Web Slack tramite la CLI slack-rs (Rust). Da utilizzare quando è necessario autenticarsi su Slack tramite OAuth (PKCE), gestire più profili di spazio di lavoro, chiamare metodi API Web Slack arbitrari (ad esempio chat.postMessage, conversazioni.list, users.info) ed eseguire operazioni Slack sicure con script dal terminale. Include accesso remoto assistito da tunnel (vedi auth login --help), esportazione/importazione di profili crittografati e una protezione di sicurezza in scrittura tramite SLACKCLI_ALLOW_WRITE. Le credenziali vengono archiviate in un archivio basato su file in ~/.config/slack-rs/.
