Automatisation de l'API Web Slack via la CLI slack-rs (Rust). À utiliser lorsque vous devez vous authentifier auprès de Slack via OAuth (PKCE), gérer plusieurs profils d'espace de travail, appeler des méthodes arbitraires de l'API Web Slack (par exemple, chat.postMessage, conversations.list, users.info) et exécuter des opérations Slack scriptées sécurisées à partir du terminal. Comprend une connexion à distance assistée par tunnel (voir auth login --help), une exportation/importation de profil crypté et une protection de sécurité en écriture via SLACKCLI_ALLOW_WRITE. Les informations d'identification sont stockées dans un stockage basé sur des fichiers sous ~/.config/slack-rs/.
