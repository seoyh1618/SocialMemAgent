Автоматизация Slack Web API через CLI slack-rs (Rust). Используйте, когда вам нужно пройти аутентификацию в Slack через OAuth (PKCE), управлять несколькими профилями рабочей области, вызывать произвольные методы Slack Web API (например,chat.postMessage, разговоры.список, пользователи.информация) и запускать безопасные скриптовые операции Slack с терминала. Включает удаленный вход в систему с помощью туннеля (см. auth login --help), экспорт/импорт зашифрованного профиля и защиту от записи через SLACKCLI_ALLOW_WRITE. Учетные данные хранятся в файловом хранилище ~/.config/slack-rs/.
