أتمتة Slack Web API عبر Slack-rs CLI (Rust). استخدمه عندما تحتاج إلى مصادقة Slack عبر OAuth (PKCE)، وإدارة ملفات تعريف مساحة العمل المتعددة، واستدعاء أساليب Slack Web API التعسفية (مثل chat.postMessage، وconversations.list، وusers.info)، وتشغيل عمليات Slack المكتوبة الآمنة من الوحدة الطرفية. يتضمن تسجيل الدخول عن بعد بمساعدة النفق (راجع تسجيل الدخول إلى المصادقة --مساعدة)، وتصدير/استيراد ملف التعريف المشفر، وحارس أمان الكتابة عبر SLACKCLI_ALLOW_WRITE. يتم تخزين بيانات الاعتماد في وحدة تخزين قائمة على الملفات ضمن ~/.config/slack-rs/.
