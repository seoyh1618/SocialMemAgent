Automatización de Slack Web API a través de la CLI de slack-rs (Rust). Úselo cuando necesite autenticarse en Slack a través de OAuth (PKCE), administrar múltiples perfiles de espacio de trabajo, llamar a métodos arbitrarios de la API web de Slack (por ejemplo, chat.postMessage, conversations.list, users.info) y ejecutar operaciones seguras con scripts de Slack desde la terminal. Incluye inicio de sesión remoto asistido por túnel (consulte inicio de sesión de autenticación --ayuda), exportación/importación de perfiles cifrados y una protección de seguridad contra escritura a través de SLACKCLI_ALLOW_WRITE. Las credenciales se almacenan en un almacenamiento basado en archivos en ~/.config/slack-rs/.
