Создает API ASP.NET Core, доступ к данным EF Core, gRPC, SignalR и серверные службы с промежуточным программным обеспечением, безопасностью (OAuth, JWT, OWASP), устойчивостью, обменом сообщениями, OpenAPI, .NET Aspire, семантическим ядром, HybridCache, обратным прокси-сервером YARP, кэшированием вывода, документами Office (Excel, Word, PowerPoint), PDF и шаблонами архитектуры. Охватывает 32 тематические области. Не используйте для шаблонов отрисовки пользовательского интерфейса или разработки конвейера CI/CD.
