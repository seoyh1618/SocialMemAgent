ينشئ واجهات برمجة تطبيقات ASP.NET Core، والوصول إلى بيانات EF Core، وgRPC، وSignalR، وخدمات الواجهة الخلفية باستخدام البرامج الوسيطة، والأمان (OAuth، JWT، OWASP)، والمرونة، والمراسلة، وOpenAPI، و.NET Aspire، وSemantic Kernel، وHybridCache، وYARP الوكيل العكسي، والتخزين المؤقت للمخرجات، ومستندات Office (Excel، وWord، وPowerPoint)، وPDF، وأنماط الهندسة المعمارية. يمتد إلى 32 مجالًا للموضوع. لا تستخدمه لأنماط عرض واجهة المستخدم أو تأليف خط أنابيب CI/CD.
