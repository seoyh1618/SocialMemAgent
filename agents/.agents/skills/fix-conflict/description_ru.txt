Автоматически разрешать конфликты слияния в текущем PR. Извлекает базовую ветку, пытается выполнить слияние, идентифицирует конфликтующие файлы, анализирует обе стороны, разрешает конфликты, проверяет локально и отправляет. Используйте, если в PR обнаружены конфликты слияния с помощью poll-pr-status.sh.
