Превращает источники знаний в Obsidian StudyVault. Два режима: (1) Режим документа — PDF/текст/веб-источники → учебные заметки с практическими вопросами. (2) Режим базы кода — проект исходного кода → хранилище для новых разработчиков. Режим определяется автоматически на основе маркеров проекта в CWD.
