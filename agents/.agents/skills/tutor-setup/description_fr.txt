Transforme les sources de connaissances en Obsidian StudyVault. Deux modes : (1) Mode Document – Sources PDF/texte/Web → notes d'étude avec questions pratiques. (2) Mode base de code — projet de code source → coffre-fort d'intégration pour les nouveaux développeurs. Le mode est détecté automatiquement en fonction des marqueurs de projet dans CWD.
