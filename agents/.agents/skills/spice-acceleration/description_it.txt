Accelera i dati localmente per prestazioni delle query inferiori al secondo. Da utilizzare quando si abilita l'accelerazione dei dati, si sceglie un motore (Arrow, DuckDB, SQLite, Cayenne), si configurano le modalità di aggiornamento, si impostano policy di conservazione, si creano snapshot, si aggiungono indici o si materializzano set di dati.
