Spezialkenntnisse im Testen von Container-Datenbanken mit Testcontainern.  
Wird verwendet, wenn echtes Datenbankverhalten erforderlich ist, unter Verwendung von SQL Server/PostgreSQL/MySQL-Containern und beim Testen von EF Core/Dapper.  
Deckt den Start von Containern, Datenbankmigrationen, Testisolation, Containerfreigabe usw. ab.
