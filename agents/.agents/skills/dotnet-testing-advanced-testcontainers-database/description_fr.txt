Compétences spécialisées dans les tests de bases de données conteneurisées avec Testcontainers.  
Utilisé lorsqu'un comportement réel de base de données est nécessaire, en utilisant des conteneurs SQL Server/PostgreSQL/MySQL, en testant EF Core/Dapper.  
Couvre le démarrage des conteneurs, les migrations de bases de données, l'isolation des tests, le partage de conteneurs, etc.
