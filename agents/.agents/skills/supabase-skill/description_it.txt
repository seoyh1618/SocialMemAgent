Configura e gestisci progetti Supabase utilizzando MCP (Model Context Protocol). Utilizza questa competenza quando lavori con i database Supabase, imposti server MCP, progetta schemi di database, implementa la sicurezza a livello di riga, gestisci migrazioni o costruisci architetture di dati moderne con PostgreSQL. Essenziale per lo sviluppo di Supabase, la progettazione di database e le operazioni di database basate sull'intelligenza artificiale.
