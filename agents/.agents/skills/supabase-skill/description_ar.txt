تكوين وإدارة مشاريع Supabase باستخدام MCP (بروتوكول السياق النموذجي). استخدم هذه المهارة عند العمل مع قواعد بيانات Supabase، أو إعداد خوادم MCP، أو تصميم مخططات قاعدة البيانات، أو تنفيذ أمان مستوى الصف، أو إدارة عمليات الترحيل، أو إنشاء بنيات بيانات حديثة باستخدام PostgreSQL. ضروري لتطوير Supabase وتصميم قواعد البيانات وعمليات قواعد البيانات التي تعمل بالذكاء الاصطناعي.
