Настраивайте проекты Supabase и управляйте ими с помощью MCP (протокол контекста модели). Используйте этот навык при работе с базами данных Supabase, настройке серверов MCP, разработке схем баз данных, реализации безопасности на уровне строк, управлении миграциями или построении современных архитектур данных с помощью PostgreSQL. Необходим для разработки Supabase, проектирования баз данных и операций с базами данных на базе искусственного интеллекта.
