Verwalten Sie die globale Tool-Berechtigungskonfiguration von Claude Code, indem Sie MCP-Befehle oder andere Tools automatisch zu „allowedTools“ hinzufügen, um eine manuelle Genehmigung bei jeder Verwendung zu vermeiden. Arbeitsablauf: Bestätigen Sie den Befehl, den der Benutzer hinzufügen muss -> Bestätigen Sie die Hinzufügungsebene (Standard global ~/.claude.json) -> führen Sie das Hinzufügen aus -> überprüfen und erinnern Sie an einen Neustart.
