Управляйте глобальной конфигурацией разрешений инструментов Claude Code, автоматически добавляя команды MCP или другие инструменты в список разрешенных инструментов, чтобы избежать ручного утверждения при каждом их использовании. Рабочий процесс: подтвердите команду, которую пользователь должен добавить -> подтвердите уровень добавления (глобальный ~/.claude.json по умолчанию) -> выполните добавление -> проверьте и напомните о перезапуске.
