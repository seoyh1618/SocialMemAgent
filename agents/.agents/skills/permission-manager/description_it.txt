Gestisci la configurazione dei permessi globali degli strumenti di Claude Code, aggiungendo automaticamente comandi MCP o altri strumenti agli strumenti consentiti per evitare l'approvazione manuale ogni volta che vengono utilizzati. Flusso di lavoro: conferma il comando che l'utente deve aggiungere -> conferma il livello di aggiunta (predefinito globale ~/.claude.json) -> esegui l'aggiunta -> verifica e ricorda di riavviare.
