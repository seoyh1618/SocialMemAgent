Administre la configuración de permisos de herramientas globales de Claude Code, agregando automáticamente comandos MCP u otras herramientas a las herramientas permitidas para evitar la aprobación manual cada vez que se usan. Flujo de trabajo: confirme el comando que el usuario necesita agregar -> confirme el nivel de adición (global predeterminado ~/.claude.json) -> ejecute la adición -> verifique y recuerde reiniciar.
