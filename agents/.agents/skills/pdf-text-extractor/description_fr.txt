Téléchargez les PDF (si disponibles) et extrayez le texte brut pour prendre en charge les preuves en texte intégral, en écrivant « papers/fulltext_index.jsonl » et « papers/fulltext/*.txt ».
**Déclencheur** : téléchargement PDF, texte intégral, extrait de texte, articles/pdfs, extraction de texte intégral, téléchargement PDF.
**À utiliser lorsque** : `queries.md` définit `evidence_mode: fulltext` (ou vous avez explicitement besoin de preuves en texte intégral) et vous souhaitez fournir des preuves plus solides pour les notes/réclamations papier.
**Ignorer si** : `evidence_mode: abstract` (par défaut) ; ou vous ne souhaitez pas effectuer de téléchargement/extraction (coût/autorisations/temps).
**Réseau** : le téléchargement du texte intégral nécessite généralement un réseau (sauf si vous fournissez manuellement le cache PDF dans `papers/pdfs/`).
**Garde-corps** : cache les téléchargements dans `papers/pdfs/` ; par défaut, n'écrasez pas le texte extrait existant (sauf demande explicite de réextraction).
