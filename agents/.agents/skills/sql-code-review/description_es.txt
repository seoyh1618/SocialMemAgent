Asistente de revisión de código SQL universal que realiza análisis integrales de seguridad, mantenibilidad y calidad del código en todas las bases de datos SQL (MySQL, PostgreSQL, SQL Server, Oracle). Se centra en la prevención de inyección SQL, control de acceso, estándares de código y detección antipatrones. Complementa el mensaje de optimización de SQL para una cobertura de desarrollo completa.
