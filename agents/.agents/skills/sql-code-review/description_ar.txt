مساعد مراجعة أكواد SQL العالمي الذي يقوم بإجراء تحليل شامل للأمان وقابلية الصيانة وجودة التعليمات البرمجية عبر جميع قواعد بيانات SQL (MySQL وPostgreSQL وSQL Server وOracle). يركز على منع حقن SQL والتحكم في الوصول ومعايير التعليمات البرمجية والكشف عن الأنماط المضادة. يكمل موجه تحسين SQL لتغطية التطوير الكاملة.
