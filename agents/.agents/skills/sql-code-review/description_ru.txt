Универсальный помощник по проверке кода SQL, который выполняет комплексный анализ безопасности, удобства обслуживания и качества кода во всех базах данных SQL (MySQL, PostgreSQL, SQL Server, Oracle). Основное внимание уделяется предотвращению SQL-инъекций, контролю доступа, стандартам кода и обнаружению анти-шаблонов. Дополняет подсказку по оптимизации SQL для полного охвата разработки.
