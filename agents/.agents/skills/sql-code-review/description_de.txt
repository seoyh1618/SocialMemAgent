Universeller SQL-Codeüberprüfungsassistent, der umfassende Sicherheits-, Wartbarkeits- und Codequalitätsanalysen für alle SQL-Datenbanken (MySQL, PostgreSQL, SQL Server, Oracle) durchführt. Der Schwerpunkt liegt auf der Verhinderung von SQL-Injection, Zugriffskontrolle, Codestandards und Anti-Pattern-Erkennung. Ergänzt die SQL-Optimierungsaufforderung für eine vollständige Entwicklungsabdeckung.
