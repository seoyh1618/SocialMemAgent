Assistant de révision de code SQL universel qui effectue une analyse complète de la sécurité, de la maintenabilité et de la qualité du code sur toutes les bases de données SQL (MySQL, PostgreSQL, SQL Server, Oracle). Se concentre sur la prévention des injections SQL, le contrôle d'accès, les normes de code et la détection anti-modèle. Complète l'invite d'optimisation SQL pour une couverture complète du développement.
