Esta habilidad SÓLO debe usarse cuando el usuario solicita explícitamente una ejecución inmediata y sin confirmación usando frases como "intercambio rápido", "ejecutar inmediatamente", "intercambio sin confirmación", "intercambio rápido ahora", "ejecución instantánea" u "omitir confirmación e intercambio". El usuario debe indicar claramente que desea omitir el paso de revisión/confirmación. NO use esta habilidad si el usuario menciona que desea revisar, confirmar, verificar o verificar antes de ejecutar; use swap-build + swap-execute en su lugar. NO utilice esta habilidad para solicitudes de intercambio generales como "intercambiar ETH por USDC" o "intercambiar tokens"; esas deben ir a la construcción de intercambio. Esta habilidad ejecuta un script de shell que genera el intercambio a través de fast-swap.sh y luego transmite inmediatamente la transacción. PELIGROSO: no hay confirmación antes de enviar transacciones reales.
