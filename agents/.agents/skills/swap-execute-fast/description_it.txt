Questa abilità dovrebbe essere utilizzata SOLO quando l'utente richiede esplicitamente l'esecuzione immediata, senza conferma, utilizzando frasi come "scambio rapido", "esegui immediatamente", "scambio senza conferma", "scambio rapido ora", "esecuzione istantanea" o "salta conferma e scambia". L'utente deve indicare chiaramente che desidera ignorare la fase di revisione/conferma. NON utilizzare questa abilità se l'utente menziona di voler rivedere, confermare, controllare o verificare prima dell'esecuzione: utilizzare invece swap-build + swap-execute. NON utilizzare questa abilità per richieste di swap generiche come "scambia ETH in USDC" o "trade token": queste dovrebbero essere inviate a swap-build. Questa abilità esegue uno script di shell che crea lo scambio tramite fast-swap.sh quindi trasmette immediatamente la transazione. PERICOLOSO: nessuna conferma prima dell'invio di transazioni reali.
