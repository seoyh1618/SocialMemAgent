Используйте библиотеку компонентов Kanpeki в проектах React. Активируйте, когда пользователь хочет добавить, использовать, настроить или составить доступные компоненты пользовательского интерфейса из Kanpeki (@kanpeki/*). Охватывает установку, шаблоны компонентов, формы, темы и темный режим с помощью React Aria Components + Tailwind CSS 4.1 + CVA.
