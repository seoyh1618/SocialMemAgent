Utilisez la bibliothèque de composants Kanpeki dans les projets React. Activez lorsque l'utilisateur souhaite ajouter, utiliser, personnaliser ou composer des composants d'interface utilisateur accessibles à partir de Kanpeki (@kanpeki/*). Couvre l'installation, les modèles de composants, les formulaires, les thèmes et le mode sombre avec React Aria Components + Tailwind CSS 4.1 + CVA.
