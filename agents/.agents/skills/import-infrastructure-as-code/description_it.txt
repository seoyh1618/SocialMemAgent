Importa le risorse di Azure esistenti in Terraform utilizzando l'individuazione dell'interfaccia della riga di comando di Azure e i moduli verificati di Azure (AVM). Da utilizzare quando richiesto per eseguire il reverse engineering dell'infrastruttura di Azure attiva, generare Infrastructure as Code da abbonamenti/gruppi di risorse/ID di risorse esistenti, mappare le dipendenze, ricavare indirizzi di importazione esatti dall'origine del modulo scaricato, prevenire deviazioni della configurazione e produrre file Terraform basati su AVM pronti per la convalida e la pianificazione in qualsiasi tipo di risorsa di Azure.
