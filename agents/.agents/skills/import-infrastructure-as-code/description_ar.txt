قم باستيراد موارد Azure الموجودة إلى Terraform باستخدام اكتشاف Azure CLI ووحدات Azure التي تم التحقق منها (AVM). استخدمه عندما يُطلب منك إجراء هندسة عكسية للبنية الأساسية المباشرة لـ Azure، وإنشاء البنية الأساسية كتعليمات برمجية من الاشتراكات/مجموعات الموارد/معرفات الموارد الموجودة، وتبعيات الخرائط، واشتقاق عناوين الاستيراد الدقيقة من مصدر الوحدة النمطية التي تم تنزيلها، ومنع انحراف التكوين، وإنتاج ملفات Terraform المستندة إلى AVM جاهزة للتحقق من الصحة والتخطيط عبر أي نوع من موارد Azure.
