Импортируйте существующие ресурсы Azure в Terraform с помощью обнаружения Azure CLI и проверенных модулей Azure (AVM). Используйте, когда вас попросят выполнить реверс-инжиниринг действующей инфраструктуры Azure, сгенерировать инфраструктуру как код на основе существующих подписок/групп ресурсов/идентификаторов ресурсов, сопоставить зависимости, получить точные адреса импорта из загруженного источника модуля, предотвратить дрейф конфигурации и создать файлы Terraform на основе AVM, готовые для проверки и планирования для любого типа ресурсов Azure.
