Importe recursos de Azure existentes a Terraform mediante el descubrimiento de la CLI de Azure y los módulos verificados de Azure (AVM). Úselo cuando se le solicite realizar ingeniería inversa en la infraestructura de Azure en vivo, generar infraestructura como código a partir de suscripciones/grupos de recursos/ID de recursos existentes, asignar dependencias, derivar direcciones de importación exactas del origen del módulo descargado, evitar cambios de configuración y producir archivos Terraform basados ​​en AVM listos para la validación y planificación en cualquier tipo de recurso de Azure.
