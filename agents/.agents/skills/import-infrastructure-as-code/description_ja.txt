Azure CLI 検出と Azure Verified Modules (AVM) を使用して、既存の Azure リソースを Terraform にインポートします。ライブ Azure インフラストラクチャのリバース エンジニアリング、既存のサブスクリプション/リソース グループ/リソース ID からのコードとしてのインフラストラクチャの生成、依存関係のマッピング、ダウンロードしたモジュール ソースからの正確なインポート アドレスの導出、構成のドリフトの防止、あらゆる Azure リソース タイプにわたる検証と計画に備えた AVM ベースの Terraform ファイルの生成を求められた場合に使用します。
