Importez des ressources Azure existantes dans Terraform à l’aide de la découverte Azure CLI et des modules Azure Verified (AVM). À utiliser lorsqu'on vous le demande pour effectuer une rétro-ingénierie de l'infrastructure Azure en direct, générer une infrastructure en tant que code à partir d'abonnements/groupes de ressources/ID de ressources existants, mapper les dépendances, dériver des adresses d'importation exactes à partir de la source de module téléchargée, empêcher la dérive de configuration et produire des fichiers Terraform basés sur AVM prêts à être validés et planifiés sur n'importe quel type de ressource Azure.
