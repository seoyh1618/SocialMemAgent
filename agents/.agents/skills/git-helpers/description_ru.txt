Помощник по рабочему процессу Git для обычных коммитов, проверки кода с оценкой достоверности и управления запросами на включение. Используйте при внесении изменений, проверке кода, создании PR, создании описаний PR. Также используйте, когда пользователь говорит, что он закончил кодирование и готов к фиксации, хочет получить отзыв о своих изменениях, ему нужно отправить и открыть PR или просит проверить свой код перед слиянием. Триггеры по «фиксации», «обзору», «нажатию», «созданию PR», «описанию PR», «обобщению изменений», «готово», «готово к отправке», «проверьте мой код».
