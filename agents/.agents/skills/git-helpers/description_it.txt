Assistente al flusso di lavoro Git per commit convenzionali, revisione del codice con punteggio di affidabilità e gestione delle richieste pull. Da utilizzare quando si apportano modifiche, si rivede il codice, si creano PR, si generano descrizioni PR. Da utilizzare anche quando l'utente afferma di aver terminato la codifica e di essere pronto per il commit, desidera feedback sulle modifiche apportate, deve inviare e aprire un PR o chiede di controllare il proprio codice prima della fusione.
