À utiliser lorsqu'on vous demande de corriger, résoudre ou traiter un PR GitHub, y compris les commentaires d'examen, les suggestions Copilot et les échecs des tests CI. Se déclenche sur des expressions telles que « corriger PR », « corriger les commentaires PR », « résoudre l'examen », « adresser les commentaires du Copilot », « corriger les commentaires d'examen », « corriger CI », « corriger les tests ayant échoué sur PR » ou lorsqu'on lui donne un numéro PR à corriger.
