Verwenden Sie diese Option, wenn Sie aufgefordert werden, einen GitHub-PR zu beheben, zu lösen oder zu beheben – einschließlich Überprüfungskommentaren, Copilot-Vorschlägen und CI-Testfehlern. Wird bei Ausdrücken wie „PR beheben“, „PR-Kommentare beheben“, „Überprüfung lösen“, „Copilot-Feedback adressieren“, „Überprüfungskommentare beheben“, „CI beheben“, „fehlgeschlagene PR-Tests beheben“ ausgelöst oder wenn eine PR-Nummer zur Korrektur angegeben wird.
