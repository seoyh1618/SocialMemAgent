Используйте, когда вас просят исправить или решить проблему с запросом на отправку запроса на GitHub, включая комментарии к обзору, предложения второго пилота и неудачные тесты CI. Запускается по таким фразам, как «исправить PR», «исправить комментарии к PR», «разрешить проверку», «обратиться к отзыву второго пилота», «исправить комментарии к обзору», «исправить CI», «исправить неудачные тесты по PR» или при присвоении номера PR для исправления.
