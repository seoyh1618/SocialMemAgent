Формируйте и управляйте рабочими пространствами с несколькими репозиториями для агентов кодирования ИИ.
Делегаты для CLI с несколькими репозиториями (в комплекте). Используйте, когда пользователь говорит
«создать рабочую область», «настроить несколько репозиториев», «рабочую область с несколькими репозиториями»,
«добавить репозиторий в рабочую область», «удалить репозиторий из рабочей области»,
«статус рабочей области», «проверить работоспособность рабочей области», «инициализировать несколько репозиториев»,
«оркестровать репозитории», «связывать репозитории».
Также используйте, когда у пользователя есть несколько репозиториев и ему нужен центральный
точка координации для агентов ИИ (Claude Code, Codex, Gemini CLI).
