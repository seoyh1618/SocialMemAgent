Erstellen Sie ein Gerüst und verwalten Sie Multi-Repo-Arbeitsbereiche für KI-Coding-Agents.
Delegiert an die Multirepo-Space-CLI (gebündelt). Verwenden Sie es, wenn der Benutzer dies sagt
„Arbeitsbereich erstellen“, „Multirepo einrichten“, „Multi-Repo-Arbeitsbereich“,
„Repository zum Arbeitsbereich hinzufügen“, „Repository aus Arbeitsbereich entfernen“,
„Workspace-Status“, „Workspace-Zustand prüfen“, „init multi-repo“,
„Repos orchestrieren“, „Repositories verknüpfen“.
Auch verwenden, wenn der Benutzer mehrere Repos hat und ein zentrales benötigt
Koordinationsstelle für KI-Agenten (Claude Code, Codex, Gemini CLI).
