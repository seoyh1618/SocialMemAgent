Registro de métodos abreviados de teclado en todo el sistema en macOS mediante monitoreo NSEvent (simple, a nivel de aplicación) y API Carbon EventHotKey (confiable, en todo el sistema). Cubre NSEvent.addGlobalMonitorForEvents y addLocalMonitorForEvents, CGEvent tap para simulación de pulsaciones de teclas, Carbon RegisterEventHotKey para teclas de acceso rápido en todo el sistema, manejo de indicadores modificadores (.deviceIndependentFlagsMask), asignaciones de códigos de teclas comunes, eliminación de rebotes, requisitos de permisos de accesibilidad (AXIsProcessTrusted) y SwiftUI .onKeyPress para accesos directos en la aplicación. Úselo al implementar atajos de teclado globales, paneles activados por teclas de acceso rápido o monitoreo de eventos clave en todo el sistema.
