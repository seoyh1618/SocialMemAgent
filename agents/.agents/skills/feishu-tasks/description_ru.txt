Создавайте, читайте и управляйте задачами Feishu с помощью автоматической авторизации пользователей. Используйте, когда вам нужно создавать задачи, которые ваш пользователь может напрямую редактировать, читать списки задач, управлять деталями задач или проверять события календаря. Поддерживает автоматическое обновление и сохранение токена между сеансами. Все операции выполняются с использованием удостоверения пользователя, обеспечивая надлежащие разрешения.
