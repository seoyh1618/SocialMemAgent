Implementación + bucle de auditoría utilizando equipos de agentes paralelos con pases estructurados de simplificación, refuerzo y documentación. Genera agentes de implementación para hacer el trabajo, luego audita a los agentes para encontrar complejidad, brechas de seguridad y desviaciones de especificaciones, luego realiza un bucle hasta que el código se compila limpiamente, todas las pruebas pasan y los auditores no encuentran ningún problema o se alcanza el límite del bucle. Úselo cuando: implemente funciones de una especificación o plan, refuerce el código existente, solucione un lote de problemas o cualquier tarea de varios archivos que se beneficie de un ciclo de compilación, verificación y corrección.
