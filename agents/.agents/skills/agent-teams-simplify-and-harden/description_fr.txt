Boucle de mise en œuvre + d'audit utilisant des équipes d'agents parallèles avec des passes structurées de simplification, de renforcement et de documentation. Génère des agents d'implémentation pour effectuer le travail, puis audite les agents pour détecter la complexité, les failles de sécurité et les écarts de spécifications, puis effectue une boucle jusqu'à ce que le code se compile proprement, que tous les tests réussissent et que les auditeurs ne trouvent aucun problème ou que le plafond de boucle soit atteint. À utiliser dans les cas suivants : implémentation de fonctionnalités à partir d'une spécification ou d'un plan, renforcement du code existant, résolution d'un lot de problèmes ou toute tâche multi-fichiers bénéficiant d'un cycle de construction-vérification-réparation.
