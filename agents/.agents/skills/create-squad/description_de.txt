Erstellen Sie in Vapi Multi-Assistenten-Trupps mit Übergaben zwischen spezialisierten Sprachagenten. Verwenden Sie diese Funktion, wenn Sie komplexe Sprach-Workflows erstellen, die mehrere Assistenten mit unterschiedlichen Rollen erfordern, z. B. von der Triage zur Buchung oder zur Übergabe vom Vertrieb an den Support.
