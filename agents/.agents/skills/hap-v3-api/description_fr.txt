Expertise professionnelle dans la création de pages et la manipulation de données à l'aide des API Mingdao Cloud HAP V3. Chaque fois que les utilisateurs mentionnent « API HAP V3 », « API HAP », « appels API », « interfaces de données », « Appkey », « Sign » ou « authentification API », cette compétence doit être utilisée. Il fournit un guide complet d'utilisation de l'API HAP V3 : configuration de l'authentification, appels API, utilisation du filtre, opérations sur les données, etc. Si l'utilisateur a déjà configuré HAP MCP, l'IA doit automatiquement extraire les clés d'authentification de la configuration MCP.
