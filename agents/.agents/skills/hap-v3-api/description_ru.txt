Профессиональный опыт создания страниц и манипулирования данными с использованием API-интерфейсов Mingdao Cloud HAP V3. Каждый раз, когда пользователи упоминают «HAP V3 API», «HAP API», «Вызовы API», «Интерфейсы данных», «Ключ приложения», «Подпись» или «Аутентификация API», необходимо использовать этот навык. Он предоставляет полное руководство по использованию API HAP V3: настройка аутентификации, вызовы API, использование фильтров, операции с данными и т. д. Если пользователь уже настроил HAP MCP, AI должен автоматически извлечь ключи аутентификации из конфигурации MCP.
