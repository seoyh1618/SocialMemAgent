Competenza professionale nella creazione di pagine e nella manipolazione dei dati utilizzando le API Mingdao Cloud HAP V3. Ogni volta che gli utenti menzionano "API HAP V3", "API HAP", "chiamate API", "interfacce dati", "Appkey", "Sign" o "autenticazione API", è necessario utilizzare questa competenza. Fornisce una guida completa all'utilizzo dell'API HAP V3: configurazione dell'autenticazione, chiamate API, utilizzo del filtro, operazioni sui dati, ecc. Se l'utente ha già configurato HAP MCP, l'IA dovrebbe estrarre automaticamente le chiavi di autenticazione dalla configurazione MCP.
