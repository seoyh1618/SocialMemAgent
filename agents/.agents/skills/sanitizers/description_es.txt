Habilidad de desinfección del compilador para la detección de errores en tiempo de ejecución en C/C++. Úselo al habilitar e interpretar AddressSanitizer (ASan), UndefinedBehaviorSanitizer (UBSan), ThreadSanitizer (TSan), MemorySanitizer (MSan) o LeakSanitizer (LSan) con GCC o Clang. Se activa ante consultas sobre indicadores de desinfectante, informes de desinfectante, ASAN_OPTIONS, errores de memoria, carreras de datos, comportamiento indefinido, lecturas no inicializadas o elegir qué desinfectante usar para una clase de error determinada.
