Analysiert eine MLflow-Sitzung – eine Folge von Spuren einer Chat-Konversation oder Interaktion mit mehreren Runden. Verwenden Sie diese Option, wenn der Benutzer eine Chat-Konversation debuggen, eine Sitzung oder den Chat-Verlauf überprüfen, herausfinden möchte, wo in einem Chat mit mehreren Runden ein Fehler aufgetreten ist, oder Muster über Runden hinweg analysieren möchte. Löst aus bei „Diese Sitzung analysieren“, „Was ist in dieser Konversation passiert“, „Sitzung debuggen“, „Chatverlauf überprüfen“, „Wo ist dieser Chat fehlgeschlagen“, „Sitzungsspuren“, „Chat analysieren“, „Diesen Chat debuggen“.
