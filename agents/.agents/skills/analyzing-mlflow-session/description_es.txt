Analiza una sesión de MLflow: una secuencia de seguimientos de una conversación o interacción de chat de varios turnos. Úselo cuando el usuario solicite depurar una conversación de chat, revisar la sesión o el historial de chat, encontrar dónde salió mal un chat de varios turnos o analizar patrones entre turnos. Activadores de "analizar esta sesión", "qué pasó en esta conversación", "depurar sesión", "revisar el historial de chat", "dónde salió mal este chat", "rastreos de sesión", "analizar chat", "depurar este chat".
