قم بإعداد مصادقة Shopify CLI والوصول إلى Admin API لمتجر ما. سير العمل: تثبيت واجهة سطر الأوامر (CLI)، والمصادقة، وإنشاء تطبيق مخصص، ورمز الوصول إلى المتجر، والتحقق. يُستخدم عند الاتصال بمتجر Shopify، أو إعداد الوصول إلى واجهة برمجة التطبيقات (API)، أو استكشاف مشكلات المصادقة وإصلاحها باستخدام الرموز المميزة لـ Shopify CLI أو Admin API.
