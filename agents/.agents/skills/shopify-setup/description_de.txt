Richten Sie die Shopify-CLI-Authentifizierung und den Admin-API-Zugriff für einen Shop ein. Arbeitsablauf: CLI installieren, authentifizieren, benutzerdefinierte App erstellen, Zugriffstoken speichern, überprüfen. Verwenden Sie diese Option, wenn Sie eine Verbindung zu einem Shopify-Shop herstellen, den API-Zugriff einrichten oder Authentifizierungsprobleme mit Shopify-CLI- oder Admin-API-Tokens beheben.
