Настройте аутентификацию Shopify CLI и доступ к API администратора для магазина. Рабочий процесс: установка CLI, аутентификация, создание собственного приложения, сохранение токена доступа, проверка. Используйте при подключении к магазину Shopify, настройке доступа к API или устранении проблем с аутентификацией с помощью токенов Shopify CLI или Admin API.
