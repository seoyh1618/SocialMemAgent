Configura la autenticación CLI de Shopify y el acceso a la API de administración para una tienda. Flujo de trabajo: instalar CLI, autenticar, crear una aplicación personalizada, almacenar el token de acceso y verificar. Úselo al conectarse a una tienda Shopify, configurar el acceso a la API o solucionar problemas de autenticación con Shopify CLI o tokens de API de administración.
