Мета-навык для объединения Three.js, GSAP ScrollTrigger, React Three Fiber, Motion и React Spring для создания сложных 3D-веб-проектов. Используйте при создании приложений, интегрирующих несколько библиотек 3D и анимации, требующих шаблонов архитектуры, управления состоянием и оптимизации производительности во всем стеке. Запускается при выполнении задач, связанных с интеграцией библиотек, мультибиблиотечной архитектурой, 3D-опытом с прокруткой, 3D-анимацией на основе физики или сложными интерактивными 3D-приложениями.
