Этот навык следует использовать, когда пользователь запрашивает «анализ сеанса», «세션 분석», «оценить выполнение навыка», «스킬 실행 검증», «проверить журналы сеанса», «로그 분석», предоставляет идентификатор сеанса с путем к навыку или хочет проверить, правильно ли навык выполнялся в прошлом сеансе. Последующий анализ сеансов Claude Code для проверки поведения навыков/агентов/перехватчиков на соответствие спецификациям SKILL.md.
