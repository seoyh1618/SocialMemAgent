Этот навык следует использовать при переводе файлов документации Docusaurus (Markdown, MDX и i18n JSON) на другие языки с сохранением всего синтаксиса, структуры и форматирования. Разработан для рабочего процесса учебного приложения Agent Factory i18n с отслеживанием состояния через манифест.json, чтобы избежать дублирования работы и возобновить прерванные переводы. Используйте, когда пользователь говорит: «перевести», «локализовать», «перевести на урду/испанский/французский», «статус перевода», «что нужно перевести», «перевести главу» или «продолжить перевод». Поддерживает языки: en, fr, de, es, zh, ar, ur, hi.
