Комплексный набор инструментов для проверки, анализа и тестирования YAML-ресурсов Kubernetes. Используйте этот навык при проверке манифестов Kubernetes, отладке синтаксических ошибок YAML, выполнении пробных тестов кластеров или работе с пользовательскими определениями ресурсов (CRD), требующими поиска документации.
