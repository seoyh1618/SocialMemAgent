Kit di strumenti completo per la convalida, l'linting e il test delle risorse YAML Kubernetes. Utilizza questa competenza durante la convalida dei manifest Kubernetes, il debug degli errori di sintassi YAML, l'esecuzione di test di prova sui cluster o l'utilizzo di definizioni di risorse personalizzate (CRD) che richiedono la ricerca della documentazione.
