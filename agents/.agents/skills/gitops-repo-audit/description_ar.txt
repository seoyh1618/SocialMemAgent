تدقيق مستودعات Flux CD GitOps للهيكل والأمان والامتثال لواجهة برمجة التطبيقات وأفضل الممارسات. استخدم هذه المهارة عندما يطلب المستخدم تدقيق مستودع GitOps أو تحليله أو مراجعته أو التحقق من صحته أو فحصه. استخدمه أيضًا عندما يذكر المستخدمون بنية Flux repo، أو أفضل ممارسات GitOps، أو التحقق من صحة البيان، أو واجهات برمجة التطبيقات المهملة، أو مراجعة الأمان، أو تنظيم المستودع - حتى لو لم يذكروا "التدقيق" صراحةً.
