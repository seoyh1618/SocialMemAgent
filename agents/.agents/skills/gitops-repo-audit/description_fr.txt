Auditez les référentiels Flux CD GitOps pour la structure, la sécurité, la conformité des API et les meilleures pratiques. Utilisez cette compétence chaque fois que l'utilisateur demande à auditer, analyser, examiner, valider ou vérifier un référentiel GitOps. Utilisez-le également lorsque les utilisateurs mentionnent la structure du référentiel Flux, les meilleures pratiques GitOps, la validation du manifeste, les API obsolètes, l'examen de la sécurité ou l'organisation du référentiel, même s'ils ne disent pas explicitement « audit ».
