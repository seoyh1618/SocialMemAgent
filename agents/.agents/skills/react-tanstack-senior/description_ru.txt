Опыт старшего/ведущего разработчика React 20 лет назад в экосистеме TanStack (запрос, маршрутизатор, таблица, форма, запуск). Основные навыки: (1) Создание приложений React с библиотеками TanStack, (2) Проверка/рефакторинг кода React для очистки кода, (3) Отладка проблем React/TanStack, (4) Настройка структуры проекта, которую можно поддерживать, (5) Оптимизация работы приложений React, (6) Использование библиотеки для других вариантов использования, (7) Общие сведения ошибки и утечки памяти, (8) Внедрение лучших практик KISS и меньше, да лучше. Ключевые слова-триггеры: React, TanStack, React Query, TanStack Router, TanStack Table, TanStack Form, TanStack Start, Vinxi, чистый код, рефакторинг, производительность, отладка.
