Trasforma le idee grezze in suggerimenti LLM di livello professionale. Analizza testo, immagini, collegamenti e documenti per creare suggerimenti ottimizzati utilizzando framework comprovati (CoT, Few-Shot, Persona, ecc.).
DA USARE QUANDO: l'utente desidera migliorare un prompt, creare un prompt da zero, ottimizzare un prompt esistente, convertire un'idea vaga in un prompt strutturato, analizzare il motivo per cui un prompt non funziona o chiede "scrivimi un prompt per...", "migliora questo prompt", "progetti questo prompt".
NON UTILIZZARE QUANDO: l'utente vuole eseguire il prompt stesso (basta eseguirlo), vuole un aiuto generale per la scrittura senza contesto, richiede codice/articoli/tweet (usa invece le competenze appropriate) o vuole parlare della teoria del prompt engineering senza produrre un prompt.
CASI BORDO: - "Correggi questo prompt" → questa abilità (ottimizzazione) - "Scrivimi un post sul blog" → NON questa abilità (creazione di contenuto, non creazione di prompt) - "Scrivimi un prompt che genera post sul blog" → questa abilità - "Perché il mio prompt non funziona?" → questa abilità (diagnosi + correzione) - "اكتب لي برومبت" → questa abilità - "حسن هالبرومبت" → questa abilità - "اكتب لي مقال" → NON questa abilità (usa katib-al-maqalat)
INPUT: idea approssimativa, richiesta esistente, immagini, collegamenti, documenti o qualsiasi combinazione. OUTPUT: prompt ottimizzato in un blocco di codice, pronto per la copia. SUCCESSO: il messaggio è chiaro, strutturato, utilizza un framework appropriato e raggiunge l'obiettivo dell'utente.
