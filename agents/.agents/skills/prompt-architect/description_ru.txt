Превратите грубые идеи в подсказки LLM профессионального уровня. Анализирует текст, изображения, ссылки и документы для создания оптимизированных подсказок с использованием проверенных инфраструктур (CoT, Few-Shot, Persona и т. д.).
ИСПОЛЬЗУЙТЕ, КОГДА: пользователь хочет улучшить подсказку, создать подсказку с нуля, оптимизировать существующую подсказку, преобразовать смутную идею в структурированную подсказку, проанализировать, почему подсказка не работает, или просит «напишите мне подсказку для...», «улучшите эту подсказку», «подскажите спроектировать это».
НЕ ИСПОЛЬЗУЙТЕ КОГДА: пользователь хочет выполнить приглашение самостоятельно (просто запустите его), хочет получить общую помощь в написании без контекста приглашения, запрашивает код/статьи/твиты (вместо этого используйте соответствующие навыки) или хочет поговорить о теории разработки подсказок, не создавая подсказку.
КРАЙНИЕ СЛУЧАИ: - «Исправить это приглашение» → этот навык (оптимизация) — «Напишите мне сообщение в блоге» → НЕ этот навык (создание контента, а не создание приглашения) — «Напишите мне приглашение, которое генерирует сообщения в блоге» → этот навык — «Почему мое приглашение не работает?» → этот навык (диагностика + исправление) — «اكتب لي برومبت» → этот навык — «حسن هالبرومبت» → этот навык — «اكتب لي مقال» → НЕ этот навык (используйте катиб-аль-макалат)
ВХОДЫ: приблизительная идея, существующая подсказка, изображения, ссылки, документы или любая комбинация. РЕЗУЛЬТАТЫ: Оптимизированное приглашение в блоке кода, готовое к копированию. УСПЕХ: Подсказка понятна, структурирована, использует соответствующую структуру и достигает цели пользователя.
