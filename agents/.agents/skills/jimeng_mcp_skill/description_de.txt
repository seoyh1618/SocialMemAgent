Verwenden Sie jimeng-mcp-server für die KI-Bild- und Videogenerierung. Nutzen Sie diese Fähigkeit, wenn der Benutzer Bilder aus Text generieren, mehrere Bilder zusammenstellen, Videos aus Textbeschreibungen erstellen oder statische Bilder animieren möchte. Es unterstützt vier Kernfunktionen: Text-zu-Bild, Bild-Compositing, Text-zu-Video und Bild-zu-Video. Erfordert, dass der Jimeng-mcp-Server lokal ausgeführt wird oder über SSE/HTTP zugänglich ist.
