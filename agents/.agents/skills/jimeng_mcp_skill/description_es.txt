Utilice jimeng-mcp-server para la generación de imágenes y videos de IA. Emplee esta habilidad cuando el usuario solicite generar imágenes a partir de texto, componer varias imágenes, crear videos a partir de descripciones de texto o animar imágenes estáticas. Admite cuatro capacidades principales: texto a imagen, composición de imágenes, texto a video e imagen a video. Requiere que jimeng-mcp-server se ejecute localmente o sea accesible a través de SSE/HTTP.
