Используйте jimeng-mcp-server для создания изображений и видео AI. Используйте этот навык, когда пользователь запрашивает создание изображений из текста, композицию нескольких изображений, создание видео из текстовых описаний или анимацию статических изображений. Он поддерживает четыре основные возможности: преобразование текста в изображение, компоновку изображений, преобразование текста в видео и изображение в видео. Требуется, чтобы сервер jimeng-mcp работал локально или был доступен через SSE/HTTP.
