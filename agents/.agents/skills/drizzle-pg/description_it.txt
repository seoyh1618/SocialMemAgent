Riferimento Drizzle ORM per PostgreSQL: definizione dello schema, query typesafe, relazioni e migrazioni con drizzle-kit. Da utilizzare quando: (1) si definiscono schemi pgTable con tipi di colonna, indici, vincoli o enumerazioni, (2) si scrivono query o join di selezione/inserimento/aggiornamento/eliminazione, (3) si definiscono relazioni e si utilizza l'API di query relazionale (db.query.*), (4) si esegue drizzle-kit generate/migrate/push/pull, (5) si configura drizzle.config.ts, (6) si utilizza l'operatore template sql`` o (7) lavorare con le estensioni PostGIS/pg_vettoriale.
