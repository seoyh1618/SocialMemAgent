مرجع Drizzle ORM لـ PostgreSQL - تعريف المخطط واستعلامات الكتابة الآمنة والعلاقات والترحيلات باستخدام drizzle-kit. يُستخدم عندما: (1) تعريف مخططات pgTable بأنواع الأعمدة أو الفهارس أو القيود أو التعدادات، (2) كتابة استعلامات أو روابط تحديد/إدراج/تحديث/حذف، (3) تحديد العلاقات واستخدام واجهة برمجة تطبيقات الاستعلام العلائقي (db.query.*)، (4) تشغيل drizzle-kit generator/migrate/push/pull، (5) تكوين drizzle.config.ts، (6) استخدام عامل تشغيل القالب sql``، أو (7) العمل مع امتدادات PostGIS/pg_vector.
