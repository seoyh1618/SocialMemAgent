Справочник Drizzle ORM для PostgreSQL — определение схемы, типобезопасные запросы, отношения и миграции с помощью drizzle-kit. Используйте, когда: (1) определяете схемы pgTable с типами столбцов, индексами, ограничениями или перечислениями, (2) пишете запросы выбора/вставки/обновления/удаления или объединения, (3) определяете отношения и используете API реляционных запросов (db.query.*), (4) запускаете Drizzle-kitgenerate/migrate/push/pull, (5) настраиваете drizzle.config.ts, (6) используете оператор шаблона sql`` или (7) работа с расширениями PostGIS/pg_vector.
