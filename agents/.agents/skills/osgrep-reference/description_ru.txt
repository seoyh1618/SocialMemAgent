Комплексный справочник по интерфейсу командной строки и стратегии поиска для поиска семантического кода osgrep. Используйте для получения подробных параметров CLI, команд управления индексами, рекомендаций по стратегии поиска (архитектурных и целевых запросов) и устранения неполадок. Дополняет плагин osgrep, который управляет жизненным циклом демона.
