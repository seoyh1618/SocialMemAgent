Шаблоны интеграции для сервера Mapbox MCP в приложениях искусственного интеллекта и средах агентов. Охватывает интеграцию среды выполнения с pydantic-ai, mastra, LangChain и пользовательскими агентами. Используйте при создании приложений на базе искусственного интеллекта, которым необходимы геопространственные возможности.
