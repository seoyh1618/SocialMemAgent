Envoltorio completo Pure JS para servicios web Amap: proporciona planificación de rutas V5 (conducción/EV/caminar/ciclismo/tránsito interurbano), codificación geográfica (avance/retroceso), búsqueda de puntos de interés (palabra clave/cercano/polígono) y sugerencias de entrada; caché LRU incorporada, reintento automático y cancelación de solicitudes; definiciones completas de TypeScript; adecuado para la capa lógica multiplataforma, requiere un paquete central o de navegación para configurar Key.
