Amap Web Service Pure JS Full Wrapper : fournit une planification d'itinéraire V5 (conduite/VE/marche/vélo/transport interurbain), un géocodage (avant/arrière), une recherche de POI (mot-clé/à proximité/polygone) et des suggestions de saisie ; cache LRU intégré, nouvelle tentative automatique et annulation de demande ; définitions complètes de TypeScript ; adapté à la couche logique multiplateforme, nécessite la configuration de Key avec le noyau ou le package de navigation.
