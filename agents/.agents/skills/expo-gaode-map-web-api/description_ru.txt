Полная JS-оболочка веб-сервиса Amap: обеспечивает планирование маршрута V5 (автомобиль/электромобиль/ходьба/велосипед/междугородний транспорт), геокодирование (вперед/назад), поиск POI (ключевое слово/поблизости/полигон) и предложения по вводу; встроенный кэш LRU, автоповтор и отмена запроса; полные определения TypeScript; уровень кросс-платформенной логики, требуется ядро ​​или навигационный пакет с настроенным ключом.
