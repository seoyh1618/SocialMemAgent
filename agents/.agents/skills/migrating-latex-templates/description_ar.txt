يتم استخدامه عندما يطلب المستخدم صراحةً "ترحيل الاقتراح" أو "ترقية القالب" أو "ترحيل الإصدارات المشتركة" أو "تحويل الاقتراح القديم إلى قالب جديد". ترحيل مقترحات NSFC LaTeX بذكاء إلى القالب الجديد، استنادًا إلى سير عمل من خمس مراحل (التحليل ← رسم الخرائط ← التخطيط ← التنفيذ ← التحقق من الصحة)، والتعامل تلقائيًا مع التغييرات الهيكلية، وإعادة تنظيم المحتوى، والتحديثات المرجعية؛ يدعم المطابقة الدلالية المستندة إلى الذكاء الاصطناعي والرجوع التجريبي، ويوفر مخرجات تشغيل معزولة باستخدام برنامج نصي للترحيل بنقرة واحدة.
