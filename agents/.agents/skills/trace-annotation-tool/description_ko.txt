LLM 오류 분석 중 개방형 코딩을 위한 사용자 정의 추적 주석 웹 앱을 생성합니다. 사용자가 LLM 추적을 검토하고, 자유 형식 설명으로 실패에 주석을 달고, 1차 정성적 라벨링(오픈 코딩)을 수행하려는 경우에 사용합니다. 또한 사용자가 "추적 주석 달기", "추적 검토 도구", "코딩 도구 열기", "추적 레이블 지정", "주석 인터페이스 구축", "LLM 출력 검토"를 언급하거나 실패 분류를 구축하기 전에 파이프라인 추적을 수동으로 검사하려는 경우에도 사용합니다. 이 기술은 FastHTML, TailwindCSS 및 HTMX를 사용하여 맞춤형 Python 웹 애플리케이션을 생성합니다.
