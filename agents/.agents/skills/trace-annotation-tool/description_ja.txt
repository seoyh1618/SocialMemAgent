LLM エラー分析中にオープン コーディング用のカスタム トレース アノテーション Web アプリを生成します。ユーザーが LLM トレースをレビューし、自由形式のコメントで障害に注釈を付け、ファーストパスの定性的ラベル付け (オープン コーディング) を行う場合に使用します。また、ユーザーが「トレースに注釈を付ける」、「トレース レビュー ツール」、「オープン コーディング ツール」、「トレースにラベルを付ける」、「注釈インターフェイスを構築する」、「LLM 出力を確認する」と言及した場合、または障害分類を構築する前にパイプライン トレースを手動で検査したい場合にも使用します。このスキルは、FastHTML、TailwindCSS、および HTMX を使用して、カスタマイズされた Python Web アプリケーションを作成します。
