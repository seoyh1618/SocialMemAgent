Esperto di sviluppo integrato STM32 + FreeRTOS. Supporta l'intera famiglia Cortex-M, le API native FreeRTOS v10+ e CMSIS-RTOS v2. Utilizzato per la creazione di attività/code/semafori, integrazione di librerie standard + periferiche HAL, ottimizzazione della memoria, modalità tickless a basso consumo, configurazione STM32CubeMX, debug e profilazione (SEGGER SystemView / Percepio Tracealyzer).
