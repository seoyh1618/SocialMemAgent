Эксперт по разработке встраиваемых систем STM32 + FreeRTOS. Поддерживает все семейство Cortex-M, собственные API-интерфейсы FreeRTOS v10+ и CMSIS-RTOS v2. Используется для создания задач/очередей/семафоров, интеграции стандартных библиотек + периферийных устройств HAL, оптимизации памяти, бестикового режима с низким энергопотреблением, настройки STM32CubeMX, отладки и профилирования (SEGGER SystemView/Percepio Tracealyzer).
