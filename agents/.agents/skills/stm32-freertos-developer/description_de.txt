STM32 + FreeRTOS Embedded-Entwicklungsexperte. Unterstützt die gesamte Cortex-M-Familie, native FreeRTOS v10+ und CMSIS-RTOS v2 APIs. Wird zum Erstellen von Aufgaben/Warteschlangen/Semaphoren, zur Integration von Standardbibliotheken + HAL-Peripheriegeräten, zur Speicheroptimierung, zum Tickless-Modus mit geringem Stromverbrauch, zur STM32CubeMX-Konfiguration, zum Debuggen und zur Profilerstellung (SEGGER SystemView / Percepio Tracealyzer) verwendet.
