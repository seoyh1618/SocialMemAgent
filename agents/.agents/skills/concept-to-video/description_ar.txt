تحويل أي مفهوم إلى فيديو توضيحي متحرك باستخدام Manim (Python). استخدمه عندما يريد المستخدم تصورات متحركة أو رسومات متحركة أو إخراج فيديو (MP4/GIF) للمفاهيم الفنية. يغطي: الرسوم المتحركة للهندسة المعمارية، وتصورات تدفق البيانات، وخطوات الخوارزمية، وشرح خطوط الأنابيب، والأدلة الرياضية، والمقارنات، وتفاعلات الوكيل، وحلقات التدريب، أو أي طلب "تحريك X" / "إنشاء فيديو لـ X". قم أيضًا بتشغيل عمليات تحرير مشهد Manim الحالية أو إعادة العرض. قم بتشغيل: "إنشاء فيديو"، أو "تحريك هذا"، أو "إنشاء شرح"، أو "مفهوم للفيديو"، أو "رسوم متحركة"، أو "إظهار هذا كفيديو"، أو "رسم متحرك"، أو "تصور هذه العملية"، أو أي مفهوم + طلب إخراج فيديو/رسوم متحركة. استخدم هذه المهارة حتى للطلبات "الحيوية" غير الرسمية المتعلقة بالأفكار التقنية.
