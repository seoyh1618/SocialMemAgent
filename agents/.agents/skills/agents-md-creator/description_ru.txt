Навык создания «AGENTS.md», руководства для агентов кодирования ИИ. Используйте его, если хотите создать файл, содержащий контекст и инструкции для агентов ИИ по работе над проектом. Запускается такими запросами, как «Создать AGENTS.md», «Создать инструкцию для агентов ИИ» или «Создать агентно-ориентированный README». Это открытый стандартный формат, который может использоваться несколькими агентами искусственного интеллекта, такими как OpenAI Codex, Claude Code, GitHub Copilot и Cursor.
