Une compétence pour créer "AGENTS.md", un manuel d'instructions pour les agents de codage IA. Utilisez-le lorsque vous souhaitez créer un fichier qui rassemble le contexte et les instructions permettant aux agents IA de travailler sur un projet. Déclenché par des requêtes telles que « Créer AGENTS.md », « Créer un manuel d'instructions pour les agents IA » ou « Créer un README orienté agent ». Il s'agit d'un format standard ouvert qui peut être partagé entre plusieurs agents d'IA tels que OpenAI Codex, Claude Code, GitHub Copilot et Cursor.
