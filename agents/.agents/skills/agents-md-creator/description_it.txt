Un'abilità per creare "AGENTS.md", un manuale di istruzioni per gli agenti di codifica AI. Usalo quando desideri creare un file che raccolga il contesto e le istruzioni per consentire agli agenti AI di lavorare su un progetto. Attivato da richieste come "Crea AGENTS.md", "Crea un manuale di istruzioni per agenti AI" o "Crea un README orientato agli agenti". È un formato standard aperto che può essere condiviso tra più agenti AI come OpenAI Codex, Claude Code, GitHub Copilot e Cursor.
