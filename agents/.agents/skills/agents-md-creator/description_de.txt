Eine Fertigkeit zum Erstellen von „AGENTS.md“, einer Bedienungsanleitung für KI-Codierungsagenten. Verwenden Sie es, wenn Sie eine Datei erstellen möchten, die Kontext und Anweisungen für KI-Agenten zur Arbeit an einem Projekt sammelt. Ausgelöst durch Anfragen wie „AGENTS.md erstellen“, „Anleitung für KI-Agenten erstellen“ oder „Agentenorientierte README-Datei erstellen“. Es handelt sich um ein offenes Standardformat, das von mehreren KI-Agenten wie OpenAI Codex, Claude Code, GitHub Copilot und Cursor gemeinsam genutzt werden kann.
