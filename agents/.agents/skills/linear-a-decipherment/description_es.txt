Análisis computacional de inscripciones lineales A utilizando la hipótesis semítica de Cyrus H. Gordon. Esta habilidad se debe utilizar cuando se trabaja con transliteraciones lineales A, descifrado de lengua minoica, análisis de inscripciones, frecuencia de signos o estadísticas de coocurrencia, extracción de esqueletos consonánticos, comparación de raíces protosemíticas, búsquedas de léxico de Gordon, patrones de fórmulas de libación o preparación de datos de entrenamiento para modelos de descifrado de ML. Desencadenantes: proyección lineal A, proyección lineal B, escritura minoica, corpus GORILA, tabletas HT/ZA/PK, cognados semíticos, análisis de transliteración, valores de signos, silabario, escrituras no descifradas, epigrafía computacional, Festos, Hagia Triada, lashon ha-kretan.
