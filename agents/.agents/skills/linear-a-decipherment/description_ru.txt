Компьютерный анализ надписей линейного письма А с использованием семитской гипотезы Сайруса Х. Гордона. Этот навык следует использовать при работе с транслитерацией линейного письма А, расшифровкой минойского языка, анализом надписей, статистикой частоты или совместного появления знаков, извлечением скелета согласных, сравнением прото-семитских корней, поиском лексикона Гордона, шаблонами формул возлияния или подготовкой обучающих данных для моделей дешифрования ML. Триггеры: линейное письмо A, проекция линейного письма B, минойское письмо, корпус GORILA, таблички HT/ZA/PK, семитские родственные слова, анализ транслитерации, значения знаков, слоговое письмо, нерасшифрованные сценарии, компьютерная эпиграфика, Фест, Айя Триада, лашон ха-критан.
