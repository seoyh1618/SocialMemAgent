التحليل الحسابي للنقوش الخطية A باستخدام فرضية Cyrus H. Gordon السامية. يجب استخدام هذه المهارة عند العمل مع النسخ الصوتية الخطية A، أو فك رموز اللغة المينوية، أو تحليل النقوش، أو إحصائيات تكرار الإشارة أو التواجد المشترك، أو استخراج الهيكل العظمي الساكن، أو مقارنة الجذر السامي البدائي، أو عمليات البحث عن معجم جوردون، أو أنماط صيغة الإراقة، أو إعداد بيانات التدريب لنماذج فك رموز تعلم الآلة. المحفزات: الخطي A، الإسقاط الخطي B، النص المينوي، مجموعة GORILA، أقراص HT/ZA/PK، الكلمات المشتركة السامية، تحليل التحويل الصوتي، قيم الإشارة، المقاطع، النصوص غير المشفرة، الكتابات الحسابية، Phaistos، Hagia Triada، lashon ha-kretan.
