Esta guía cubre la filosofía de diseño, los conceptos básicos y el uso práctico del marco AgentScope. Utilice esta habilidad siempre que el usuario quiera hacer algo con la biblioteca AgentScope (Python). Esto incluye crear aplicaciones de agentes usando AgentScope, responder preguntas sobre AgentScope, buscar orientación sobre cómo usar AgentScope, buscar ejemplos o información específica (funciones/clases/módulos).
