Публикация Claude Code Skill на GitHub одним щелчком мыши, автоматическая проверка, завершение файлов, создание репозитория, отправка и проверка возможности установки. Срабатывает, когда пользователь говорит «опубликовать этот навык», «опубликовать навык», «отправить навык на GitHub», «поделиться этим навыком», «/publish-skill». Поддерживает навыки публикации из текущего каталога или указанного пути.
