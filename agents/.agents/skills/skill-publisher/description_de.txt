Veröffentlichen Sie Claude Code Skill mit einem Klick auf GitHub, mit automatischer Validierung, Dateivervollständigung, Repository-Erstellung, Push und Installationsüberprüfung. Wird ausgelöst, wenn der Benutzer „Diesen Skill veröffentlichen“, „Skill veröffentlichen“, „Skill auf GitHub übertragen“, „Diesen Skill teilen“ oder „/publish-skill“ sagt. Unterstützt die Veröffentlichung des Skills im aktuellen Verzeichnis oder unter einem angegebenen Pfad.
