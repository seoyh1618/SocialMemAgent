Глубокий дизайн персонажей для агентов Agentforce с 50-балльной оценкой. СИГНАЛИЗИРУЕТСЯ, когда: пользователь создает персональные данные агента, определяет личность/личность агента, создает персональные документы, кодирует персональные данные в поля Agent Builder или задает вопрос о тоне/голосе/регистрации агента. НЕ ЗАПУСКАЕТСЯ при: создании метаданных агента (используйте sf-ai-agentforce), тестировании агентов (используйте sf-ai-agentforce-testing) или DSL сценария агента (используйте sf-ai-agentscript).
