Conception de personnalité approfondie pour les agents Agentforce avec une notation de 50 points. DÉCLENCHEMENT lorsque : l'utilisateur conçoit des personnages d'agent, définit la personnalité/l'identité de l'agent, crée des documents de personnage, encode le personnage dans les champs Agent Builder ou pose des questions sur le ton/la voix/l'enregistrement de l'agent. NE PAS DÉCLENCHER lorsque : la création de métadonnées d'agent (utilisez sf-ai-agentforce), le test d'agents (utilisez sf-ai-agentforce-testing) ou l'agent Script DSL (utilisez sf-ai-agentscript).
