Diseño de personalidad profundo para agentes de Agentforce con puntuación de 50 puntos. SE DISPARA cuando: el usuario diseña personas de agentes, define la personalidad/identidad del agente, crea documentos de personas, codifica personas en campos de Agent Builder o pregunta sobre el tono/voz/registro del agente. NO DISPARAR cuando: cree metadatos del agente (use sf-ai-agentforce), pruebe agentes (use sf-ai-agentforce-testing) o Agent Script DSL (use sf-ai-agentscript).
