تصميم شخصي عميق لعملاء Agentforce مع تسجيل 50 نقطة. يتم تشغيله عندما: يقوم المستخدم بتصميم شخصيات الوكيل، أو تعريف شخصية/هوية الوكيل، أو إنشاء مستندات شخصية، أو تشفير الشخصية في حقول Agent Builder، أو السؤال عن نغمة/صوت/تسجيل الوكيل. لا تقم بالتشغيل عند: إنشاء بيانات تعريف وكيل (استخدم sf-ai-agentforce)، أو وكلاء الاختبار (استخدم sf-ai-agentforce-testing)، أو Agent Script DSL (استخدم sf-ai-agentscript).
