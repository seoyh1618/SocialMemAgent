Estrategia de abastecimiento de imágenes para proyectos de UI. Utilice primero recursos GRATUITOS: DiceBear/Boring Avatars para avatares, Unsplash/Picsum para fotos, unDraw/Storyset para ilustraciones, Haikei para fondos. Generación de IA (DALL-E) solo cuando se necesitan activos de marca personalizados y no existe una alternativa gratuita.
