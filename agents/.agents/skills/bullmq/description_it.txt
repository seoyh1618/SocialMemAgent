Riferimento al sistema di code BullMQ per code di processi, lavoratori, flussi e pianificatori supportati da Redis. Utilizzare quando: (1) creare code e lavoratori con BullMQ, (2) aggiungere lavori (ritardati, con priorità, ripetibili, deduplicati), (3) impostare gerarchie di lavori padre-figlio FlowProducer, (4) configurare strategie di ripetizione, limitazione della velocità o concorrenza, (5) implementare pianificatori di lavori con modelli cron/intervallo, (6) preparare BullMQ per la produzione (arresto graduale, configurazione Redis, monitoraggio) o (7) debug di lavori bloccati o problemi di connessione
