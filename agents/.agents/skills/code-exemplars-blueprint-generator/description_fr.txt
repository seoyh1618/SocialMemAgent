Générateur d'invites indépendant de la technologie qui crée des invites d'IA personnalisables pour analyser les bases de code et identifier des exemples de code de haute qualité. Prend en charge plusieurs langages de programmation (.NET, Java, JavaScript, TypeScript, React, Angular, Python) avec une profondeur d'analyse, des méthodes de catégorisation et des formats de documentation configurables pour établir des normes de codage et maintenir la cohérence entre les équipes de développement.
