Настройте и используйте модуль nuxt-users для Nuxt 3 и Nuxt 4. Используйте при добавлении аутентификации, управления пользователями, ролей, сброса пароля, настройки базы данных (SQLite/MySQL/PostgreSQL) или команд CLI (миграция, создание пользователя). Охватывает nuxt.config (nuxtUsers), составные элементы (useAuthentication, useUsers, usePublicPaths, usePasswordValidation, useNuxtUsersLocale), компоненты (NUsersLoginForm, NUsersLogoutLink и т. д.) и авторизацию (белый список, разрешения).
