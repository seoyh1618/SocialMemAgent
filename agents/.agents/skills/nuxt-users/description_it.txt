Configura e utilizza il modulo nuxt-users per Nuxt 3 e Nuxt 4. Utilizzalo quando aggiungi autenticazione, gestione utenti, ruoli, reimpostazione password, configurazione database (SQLite/MySQL/PostgreSQL) o comandi CLI (migrate, create-user). Copre nuxt.config (nuxtUsers), componenti componibili (useAuthentication, useUsers, usePublicPaths, usePasswordValidation, useNuxtUsersLocale), componenti (NUsersLoginForm, NUsersLogoutLink e così via) e autorizzazione (whitelist, autorizzazioni).
