Configure y use el módulo nuxt-users para Nuxt 3 y Nuxt 4. Úselo al agregar autenticación, administración de usuarios, roles, restablecimiento de contraseña, configuración de bases de datos (SQLite/MySQL/PostgreSQL) o comandos CLI (migrar, crear usuario). Cubre nuxt.config (nuxtUsers), elementos componibles (useAuthentication, useUsers, usePublicPaths, usePasswordValidation, useNuxtUsersLocale), componentes (NUsersLoginForm, NUsersLogoutLink, etc.) y autorización (lista blanca, permisos).
