Konfigurieren und verwenden Sie das Modul „nuxt-users“ für Nuxt 3 und Nuxt 4. Verwenden Sie es beim Hinzufügen von Authentifizierung, Benutzerverwaltung, Rollen, Kennwortzurücksetzung, Datenbankeinrichtung (SQLite/MySQL/PostgreSQL) oder CLI-Befehlen (migrieren, Benutzer erstellen). Deckt nuxt.config (nuxtUsers), Composables (useAuthentication, useUsers, usePublicPaths, usePasswordValidation, useNuxtUsersLocale), Komponenten (NUsersLoginForm, NUsersLogoutLink usw.) und Autorisierung (Whitelist, Berechtigungen) ab.
