قم بتكوين وحدة nuxt-users واستخدامها لـ Nuxt 3 وNuxt 4. استخدمها عند إضافة المصادقة، أو إدارة المستخدم، أو الأدوار، أو إعادة تعيين كلمة المرور، أو إعداد قاعدة البيانات (SQLite/MySQL/PostgreSQL)، أو أوامر CLI (الترحيل، إنشاء المستخدم). يغطي nuxt.config (nuxtUsers)، والمكونات القابلة للتركيب (useAuthentication، وuseUsers، وusePublicPaths، وusePasswordValidation، وuseNuxtUsersLocale)، والمكونات (NUsersLoginForm، وNUsersLogoutLink، وما إلى ذلك)، والترخيص (القائمة البيضاء، والأذونات).
