Configurez et utilisez le module nuxt-users pour Nuxt 3 et Nuxt 4. À utiliser lors de l'ajout d'authentification, de gestion des utilisateurs, de rôles, de réinitialisation de mot de passe, de configuration de base de données (SQLite/MySQL/PostgreSQL) ou de commandes CLI (migrer, créer un utilisateur). Couvre nuxt.config (nuxtUsers), les composables (useAuthentication, useUsers, usePublicPaths, usePasswordValidation, useNuxtUsersLocale), les composants (NUsersLoginForm, NUsersLogoutLink, etc.) et l'autorisation (liste blanche, autorisations).
