عندما يريد المستخدم تحسين البيانات الوصفية لمتجر التطبيقات - العنوان أو العنوان الفرعي أو حقل الكلمة الرئيسية أو الوصف. يُستخدم أيضًا عندما يذكر المستخدم "تحسين العنوان الخاص بي" أو "بيانات تعريف ASO" أو "حقل الكلمة الرئيسية" أو "حدود الأحرف" أو "وصف التطبيق" أو "كتابة الترجمة الخاصة بي". لاكتشاف الكلمات الرئيسية، راجع البحث عن الكلمات الرئيسية. للحصول على عمليات تدقيق ASO الكاملة، راجع aso-audit.
