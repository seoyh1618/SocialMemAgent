Lorsque l'utilisateur souhaite optimiser les métadonnées de l'App Store : titre, sous-titre, champ de mot-clé ou description. À utiliser également lorsque l'utilisateur mentionne « optimiser mon titre », « métadonnées ASO », « champ de mot-clé », « limites de caractères », « description de l'application » ou « écrire mon sous-titre ». Pour la découverte de mots-clés, voir recherche de mots-clés. Pour les audits ASO complets, voir aso-audit.
