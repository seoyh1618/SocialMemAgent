Habilidad de depuración de GDB para programas C/C++. Úselo al iniciar una sesión de GDB, establecer puntos de interrupción, recorrer el código, inspeccionar variables, depurar fallas, usar la depuración inversa (grabar/reproducir), depurar remotamente con gdbserver o cargar volcados de núcleo. Se activa ante consultas sobre comandos GDB, fallos de segmentación, bloqueos, puntos de vigilancia, puntos de interrupción condicionales, impresoras bonitas, secuencias de comandos Python GDB o depuración de subprocesos múltiples.
