GDB-Debugger-Fähigkeit für C/C++-Programme. Verwenden Sie diese Option, wenn Sie eine GDB-Sitzung starten, Haltepunkte festlegen, Code schrittweise durchlaufen, Variablen überprüfen, Abstürze debuggen, Reverse-Debugging (Aufzeichnen/Wiedergeben), Remote-Debugging mit gdbserver verwenden oder Core-Dumps laden. Wird bei Abfragen zu GDB-Befehlen, Segfaults, Hängen, Watchpoints, bedingten Haltepunkten, Pretty-Printern, Python-GDB-Skripting oder Multithread-Debugging aktiviert.
