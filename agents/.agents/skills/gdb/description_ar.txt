مهارة مصحح أخطاء GDB لبرامج C/C++. يُستخدم عند بدء جلسة GDB، أو تعيين نقاط التوقف، أو التنقل خلال التعليمات البرمجية، أو فحص المتغيرات، أو تصحيح الأخطاء، أو استخدام التصحيح العكسي (التسجيل/إعادة التشغيل)، أو تصحيح الأخطاء عن بُعد باستخدام gdbserver، أو تحميل عمليات تفريغ النواة. يتم التنشيط عند الاستعلامات حول أوامر GDB، أو أخطاء segfaults، أو حالات التعليق، أو نقاط المراقبة، أو نقاط التوقف الشرطية، أو الطابعات الجميلة، أو برمجة Python GDB النصية، أو تصحيح الأخطاء متعدد الخيوط.
