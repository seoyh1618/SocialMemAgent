Навыки отладчика GDB для программ на C/C++. Используйте при запуске сеанса GDB, установке точек останова, пошаговом выполнении кода, проверке переменных, отладке сбоев, использовании обратной отладки (запись/воспроизведение), удаленной отладке с помощью gdbserver или загрузке дампов ядра. Активируется при запросах о командах GDB, ошибках сегментов, зависаниях, точках наблюдения, условных точках останова, красивых принтерах, сценариях Python GDB или многопоточной отладке.
