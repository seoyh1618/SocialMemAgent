Compétence de débogueur GDB pour les programmes C/C++. À utiliser lors du démarrage d'une session GDB, de la définition de points d'arrêt, de la navigation pas à pas dans le code, de l'inspection des variables, du débogage des plantages, de l'utilisation du débogage inversé (enregistrement/relecture), du débogage à distance avec gdbserver ou du chargement des core dumps. S'active sur les requêtes concernant les commandes GDB, les erreurs de segmentation, les blocages, les points de surveillance, les points d'arrêt conditionnels, les jolies imprimantes, les scripts Python GDB ou le débogage multithread.
