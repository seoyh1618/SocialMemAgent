Conseils pour gérer le cycle de vie du package R selon les principes de Tidyverse à l’aide du package de cycle de vie. À utiliser dans les cas suivants : (1) configuration d'une infrastructure de cycle de vie dans un package, (2) dépréciation de fonctions ou d'arguments, (3) renommage de fonctions ou d'arguments, (4) remplacement de fonctions, (5) marquage de fonctions comme expérimentales, (6) compréhension des étapes du cycle de vie (stables, expérimentales, obsolètes, remplacées) ou (7) écriture d'assistants de dépréciation pour des scénarios complexes.
