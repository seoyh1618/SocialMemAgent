إدارة بيئات Python الافتراضية (.venv) تلقائيًا في الأوامر الطرفية. قم دائمًا بتنشيط .venv قبل تشغيل أوامر Python/pip. يدعم أنظمة التشغيل macOS وLinux وWindows من خلال التنشيط المتوافق مع shell. يُستخدم عند تنفيذ برامج Python النصية أو تثبيت الحزم أو تشغيل خوادم التطوير. أمر بالغ الأهمية لإدارة البيئة المتسقة.
