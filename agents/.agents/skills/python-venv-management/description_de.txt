Verwalten Sie virtuelle Python-Umgebungen (.venv) automatisch in Terminalbefehlen. Aktivieren Sie immer .venv, bevor Sie Python/pip-Befehle ausführen. Unterstützt macOS, Linux und Windows mit Shell-fähiger Aktivierung. Wird zum Ausführen von Python-Skripten, zum Installieren von Paketen oder zum Ausführen von Entwicklungsservern verwendet. Entscheidend für ein konsistentes Umgebungsmanagement.
