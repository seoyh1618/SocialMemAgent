Gérez automatiquement les environnements virtuels Python (.venv) dans les commandes du terminal. Activez toujours .venv avant d'exécuter les commandes Python/pip. Prend en charge macOS, Linux et Windows avec activation compatible Shell. À utiliser lors de l'exécution de scripts Python, de l'installation de packages ou de l'exécution de serveurs de développement. Critique pour une gestion cohérente de l’environnement.
