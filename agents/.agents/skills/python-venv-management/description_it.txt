Gestisci automaticamente gli ambienti virtuali Python (.venv) nei comandi del terminale. Attiva sempre .venv prima di eseguire i comandi Python/pip. Supporta macOS, Linux e Windows con attivazione compatibile con la shell. Da utilizzare durante l'esecuzione di script Python, l'installazione di pacchetti o l'esecuzione di server di sviluppo. Fondamentale per una gestione coerente dell'ambiente.
