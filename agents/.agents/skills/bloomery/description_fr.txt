Tutoriel interactif qui guide les ingénieurs dans la création de leur propre agent de codage (boucle agent) à partir de zéro à l'aide d'appels HTTP bruts à une API LLM. Prend en charge Gemini, OpenAI (et les points de terminaison compatibles) et Anthropic. Prend en charge TypeScript, Python, Go et Ruby. Détecte automatiquement les progrès. À utiliser lorsque quelqu'un dit « créer un agent », « apprenez-moi des agents » ou « /build-agent ».
