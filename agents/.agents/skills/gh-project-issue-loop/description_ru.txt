Непрерывно обрабатывайте доступные элементы проекта GitHub, один раз подготавливая контекст проекта, а затем циклично реализуя связанные проблемы с помощью навыка gh-issue-to-pr. Используйте, когда вас просят очистить отставание на доске проекта GitHub, выполнить пакетную доставку проблем проекта или неоднократно выбирать и реализовывать открытые проблемы проекта с помощью gh CLI.
