قم بمعالجة عناصر مشروع GitHub المتاحة باستمرار من خلال إعداد سياق المشروع مرة واحدة ثم تنفيذ المشكلات المرتبطة في حلقة عبر مهارة gh-issue-to-pr. استخدمه عندما يُطلب منك مسح الأعمال المتراكمة من لوحة مشروع GitHub، أو مشكلات المشروع التي يتم تسليمها على دفعات، أو اختيار مشكلات المشروع المفتوحة وتنفيذها بشكل متكرر باستخدام gh CLI.
