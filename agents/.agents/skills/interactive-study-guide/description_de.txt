Verwandeln Sie einen Codebase-Studienführer in ein ausgefeiltes interaktives Web-Erlebnis. Diese Fertigkeit sollte verwendet werden, wenn der Benutzer über eine fertige Markdown-Datei für den Studienführer (von codebase-study-guide oder ähnlichem) verfügt und diese in eine interaktive pädagogische App umwandeln möchte. Wird bei Anfragen wie „Diesen Studienführer interaktiv gestalten“, „In ein interaktives Erlebnis verwandeln“, „Diesen Studienführer visualisieren“, „Eine interaktive Version erstellen“ ausgelöst oder wenn ein Benutzer eine .md-Datei für den Studienführer hat und eine umfassendere Präsentation wünscht. Erstellt eine von Vite bereitgestellte einseitige App mit scrollgesteuertem Storytelling, interaktiven Architekturdiagrammen, animierten Code-Komplettlösungen und progressiver Offenlegung.
