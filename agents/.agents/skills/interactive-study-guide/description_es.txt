Transforme una guía de estudio de código base en una experiencia web interactiva pulida. Esta habilidad debe usarse cuando el usuario tiene un archivo de rebajas de guía de estudio completo (de codebase-study-guide o similar) y desea convertirlo en una aplicación pedagógica interactiva. Se activan solicitudes como "hacer que esta guía de estudio sea interactiva", "convertir esto en una experiencia interactiva", "visualizar esta guía de estudio", "crear una versión interactiva" o cuando un usuario tiene un archivo .md de guía de estudio y desea una presentación más completa. Produce una aplicación de una sola página servida por Vite con narración basada en desplazamiento, diagramas de arquitectura interactivos, tutoriales de código animados y divulgación progresiva.
