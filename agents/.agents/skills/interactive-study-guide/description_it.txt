Trasforma una guida allo studio della base di codice in un'esperienza web interattiva raffinata. Questa competenza dovrebbe essere utilizzata quando l'utente ha un file di markdown della guida allo studio completato (da codebase-study-guide o simile) e desidera trasformarlo in un'app pedagogica interattiva. Si attiva su richieste come "rendi interattiva questa guida allo studio", "trasformala in un'esperienza interattiva", "visualizza questa guida allo studio", "crea una versione interattiva" o quando un utente ha un file .md della guida allo studio e desidera una presentazione più ricca. Produce un'app a pagina singola servita da Vite con narrazione basata su scorrimento, diagrammi di architettura interattivi, procedure dettagliate animate sul codice e divulgazione progressiva.
