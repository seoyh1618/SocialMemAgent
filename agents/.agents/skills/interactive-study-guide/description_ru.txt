Превратите руководство по изучению кодовой базы в совершенный интерактивный веб-интерфейс. Этот навык следует использовать, когда у пользователя есть готовый файл уценки учебного пособия (из codebase-study-guide или аналогичного) и он хочет превратить его в интерактивное педагогическое приложение. Запускается по таким запросам, как «сделать это учебное пособие интерактивным», «превратить это в интерактивный опыт», «визуализировать это учебное пособие», «создать интерактивную версию» или когда у пользователя есть файл учебного пособия .md и он хочет более насыщенную презентацию. Создает одностраничное приложение, обслуживаемое Vite, с рассказыванием историй с помощью прокрутки, интерактивными диаграммами архитектуры, анимированными пошаговыми руководствами по коду и прогрессивным раскрытием информации.
