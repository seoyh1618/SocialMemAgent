ابحث عن قصص عملاء Microsoft واسترجعها من موقع قصص عملاء Microsoft الرسمي (https://www.microsoft.com/en-us/customers/search). يُستخدم عندما يطلب المستخدم العثور على دراسات حالة العملاء أو قصص النجاح أو الأمثلة المرجعية لاعتماد تقنية Microsoft. يدعم التصفية حسب المنتج (Azure، وM365، وDynamics 365، وما إلى ذلك)، والمنطقة/البلد، والصناعة، واحتياجات العمل، وحجم المؤسسة، والبحث عن الكلمات الرئيسية. يمكن أيضًا جلب تفاصيل القصة الفردية. تتضمن المشغلات النموذجية أسئلة مثل "البحث عن قصص العملاء حول Azure OpenAI في اليابان" أو "إظهار شركات الرعاية الصحية التي تستخدم Microsoft 365 Copilot" أو "日本の製造業でAIを活用した事例を探して".
