تقرير تحليلي أسبوعي يجمع بين GA4 وGoogle Search Console وMicrosoft Clarity في بريد إلكتروني بتنسيق HTML مميز. يُستخدم لمهام Sunday cron الآلية التي تنشئ تقارير أسبوعية لتحليلات المنتج. يتتبع المستخدمين، والجلسات، ومشاهدات الصفحة، وأداء تحسين محركات البحث (الاستعلامات، ومرات الظهور، ونسبة النقر إلى الظهور، والموضع)، واحتكاك UX (النقرات الميتة، والنقرات الغاضبة، والرجوع السريع)، والاتجاهات الأسبوعية. يتضمن تتبع المساءلة (هل فعلنا ما قلناه؟)، والتوصيات ذات الأولوية مع تقييمات التأثير/الجهد، وقوائم مرجعية قابلة للتنفيذ. الشخصية: مستشار تحسين محركات البحث (SEO) بقيمة 15 ألف دولار شهريًا. المخرجات: بريد إلكتروني بتنسيق HTML + ملخص للدردشة.
