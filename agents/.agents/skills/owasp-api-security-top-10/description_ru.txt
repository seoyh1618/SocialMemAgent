Топ-10 безопасности OWASP API — предотвращение, обнаружение и устранение проблем безопасности REST/GraphQL/API. Используйте при разработке или проверке API — авторизация на уровне объектов и функций, аутентификация, ограничение скорости и потребления ресурсов, конфиденциальные бизнес-потоки, SSRF, инвентаризация и управление версиями API или использование сторонних API.
