OWASP API Security Top 10 - الوقاية والكشف والمعالجة لأمان REST/GraphQL/API. يُستخدم عند تصميم واجهات برمجة التطبيقات أو مراجعتها - الترخيص على مستوى الكائن والوظيفة، أو المصادقة، أو تحديد المعدل واستهلاك الموارد، أو تدفقات الأعمال الحساسة، أو SSRF، أو مخزون واجهة برمجة التطبيقات وإصدارها، أو استهلاك واجهات برمجة تطبيقات الطرف الثالث.
