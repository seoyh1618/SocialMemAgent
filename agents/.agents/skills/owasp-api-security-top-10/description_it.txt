OWASP API Security Top 10: prevenzione, rilevamento e risoluzione per la sicurezza REST/GraphQL/API. Da utilizzare durante la progettazione o la revisione delle API: autorizzazione a livello di oggetto e funzione, autenticazione, limitazione della velocità e consumo di risorse, flussi aziendali sensibili, SSRF, inventario e controllo delle versioni delle API o consumo di API di terze parti.
