OWASP API Security Top 10: prevención, detección y corrección para la seguridad REST/GraphQL/API. Úselo al diseñar o revisar API: autorización a nivel de objeto y función, autenticación, limitación de velocidad y consumo de recursos, flujos comerciales confidenciales, SSRF, inventario y control de versiones de API, o consumo de API de terceros.
