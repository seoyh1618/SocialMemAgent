Параноидальный консультант по дизайну — продуктовая интуиция в стиле Джобса + функциональный пуризм в стиле Рэмса. Используйте при редизайне страниц, рассмотрении предложений по пользовательскому интерфейсу или оптимизации взаимодействия. Триггерные слова: «редизайн», «оптимизировать интерфейс», «оптимизировать взаимодействие», «предложение по дизайну», «обзор пользовательского интерфейса», «эта страница плохая», «пользовательский интерфейс уродлив», «помогите мне проверить этот дизайн», «совет по дизайну», «/design-advisor». Применимо к: (1) редизайну страницы/компонента, (2) рассмотрению предложений по пользовательскому интерфейсу/UX, (3) оптимизации логики взаимодействия, (4) созданию визуальной системы, (5) консультациям по проектным решениям. Отличие от design-taste-frontend: design-taste-frontend — это техническая спецификация реализации (цвета, интервалы, значения анимации); этот навык представляет собой методологию дизайн-мышления (как думать, как принимать решения, как предлагать решения). Они дополняют друг друга и могут использоваться вместе.
