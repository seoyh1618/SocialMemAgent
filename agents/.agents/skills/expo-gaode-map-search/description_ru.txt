Высокопроизводительное решение Gaode Native Search: реализовано с помощью собственного SDK, без ограничений веб-квот; обеспечивает поиск POI по всему сценарию (ключевое слово/поблизости/вдоль маршрута/полигона/идентификаторы); поддерживает подсказки по вводу (автозаполнение) и обратное геокодирование (от координаты до адреса); включены полные определения TypeScript; требует инициализации с помощью ядра или навигационного пакета.
