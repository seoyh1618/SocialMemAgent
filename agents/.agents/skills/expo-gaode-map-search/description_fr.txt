Solution haute performance Gaode Native Search : mise en œuvre avec le SDK natif, aucune limite de quota Web ; fournit une recherche de POI dans un scénario complet (mot-clé/à proximité/le long de l'itinéraire/polygone/détails d'identification) ; prend en charge les conseils de saisie (complétion automatique) et le géocodage inversé (coordonnées à adresser) ; comprend des définitions TypeScript complètes ; nécessite l'initialisation du noyau ou du package de navigation.
