[Lecture obligatoire · Obligatoire] Normes obligatoires de développement front-end de l'entreprise. Vous devez lire cette compétence avant d'écrire ou de modifier un code frontend. Couvre toutes les contraintes de développement, y compris les normes obligatoires de l'API de composition, les conventions de dénomination, les règles d'appel de l'API, la gestion de l'état Pinia et le fractionnement des composants.
