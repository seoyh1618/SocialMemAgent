Оцените качество навыков Claude Code по 6 взвешенным параметрам: качество вступительной части, покрытие триггеров, структурная полнота, глубина контента, последовательность и целостность, а также соответствие CONTRIBUTING.md. Составляет отчеты о результатах аудита с выводами по степени серьезности и практическими рекомендациями. Два режима: (1) Быстрый аудит — полный отчет по одному навыку с взвешенной оценкой по каждому параметру; (2) Полный аудит — все навыки в репо, сравнительная таблица рейтингов, а также сводные данные по каждому навыку. Срабатывает при: «оценить навык», «проверить качество навыка», «оценить навык», «проверить навык», «проверить полноту навыка», «оценить этот навык», «проверить качество навыка», «оценить навык», «оценить навык», «проверить навык», «насколько хорош этот навык», или когда пользователь запрашивает отзыв о файле SKILL.md. Используйте этот навык при проверке навыков перед развертыванием, сравнении качества навыков в репозитории или для диагностики того, почему навык не активируется по соответствующим запросам.
