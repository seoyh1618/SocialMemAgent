قم بتقييم جودة مهارة Claude Code عبر 6 أبعاد مرجحة: جودة المادة الأمامية، وتغطية الزناد، والاكتمال الهيكلي، وعمق المحتوى، والاتساق والنزاهة، والامتثال لـ CONTRIBUTING.md. يُنتج تقارير تدقيق مُسجلة مع نتائج مصنفة حسب الخطورة وتوصيات قابلة للتنفيذ. وضعان: (1) التدقيق السريع — مهارة واحدة، تقرير كامل لكل بُعد مع تسجيل النقاط المرجحة؛ (2) التدقيق الكامل - جميع المهارات في الريبو، وجدول التصنيف المقارن بالإضافة إلى ملخصات لكل مهارة. مشغلات على: "تقييم المهارة"، "تدقيق جودة المهارة"، "تسجيل المهارة"، "مراجعة المهارة"، "التحقق من اكتمال المهارة"، "تقييم هذه المهارة"، "التحقق من جودة المهارة"، "تقدير المهارة"، "تقييم المهارة"، "تدقيق المهارة"، "مدى جودة هذه المهارة"، أو عندما يطلب المستخدم تعليقات على ملف SKILL.md. استخدم هذه المهارة عند مراجعة المهارات قبل النشر، أو مقارنة جودة المهارة عبر الريبو، أو تشخيص سبب فشل تنشيط المهارة في الاستعلامات ذات الصلة.
