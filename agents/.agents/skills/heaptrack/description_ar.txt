مهارة ملف تعريف الذاكرة heaptrack لنظام التشغيل Linux. يُستخدم عند تعقب عمليات تخصيص الكومة، أو البحث عن تسرب الذاكرة، أو قياس ذروة استخدام الكومة، أو تحديد نقاط التخصيص الفعالة، أو مقارنة سلوك التخصيص بين عمليات التشغيل. يتم التنشيط عند الاستعلامات حول heaptrack، أو ملفات تعريف الكومة، أو تحليل تخصيص الذاكرة، أو heaptrack_print، أو نقاط تخصيص التخصيص، أو اكتشاف تسرب الذاكرة باستخدام heaptrack.
