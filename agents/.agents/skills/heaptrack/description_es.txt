Habilidad de perfilador de memoria heaptrack para Linux. Úselo para realizar un seguimiento de las asignaciones del montón, encontrar pérdidas de memoria, medir el uso máximo del montón, identificar puntos de acceso de asignación o comparar el comportamiento de la asignación entre ejecuciones. Se activa ante consultas sobre heaptrack, perfiles de montón, análisis de asignación de memoria, heaptrack_print, puntos de acceso de asignación o detección de fugas de memoria con heaptrack.
