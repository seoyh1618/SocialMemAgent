Generar y evaluar nombres para empresas, productos, marcas y características utilizando un proceso estructurado de tres fases (Identificar, Inventar, Implementar) basado en la metodología Lexicon Branding de David Placek. Úselo cuando un usuario solicite nombrar una empresa, nombrar un producto, hacer una lluvia de ideas sobre nombres de marcas, evaluar una lista corta de nombres, cambiar el nombre de algo, crear un resumen creativo para nombrar o encontrar un nombre para una startup, aplicación o servicio. Acepta una descripción del negocio, producto o marca a nombrar.
