Загружается при рисовании любой диаграммы Excalidraw. Предоставляет цветовую палитру (шестнадцатеричные коды), формулы определения размера для предотвращения усечения текста, правила интервалов для предотвращения перекрытия, стили стрелок, шаблоны макетов и шаблоны диаграмм для архитектуры, блок-схем и диаграмм ER. Используйте, когда вас просят нарисовать, визуализировать, построить диаграмму или создать любую диаграмму.
