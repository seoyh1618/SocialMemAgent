Implementiert abruferweiterte Generierungspipelines. Verwenden Sie es beim Aufbau von Systemen zum Abrufen von Dokumenten, beim Auswählen von Chunking-Strategien, beim Auswählen von Einbettungsmodellen, beim Konfigurieren von Vektorspeichern, beim Implementieren einer Hybridsuche oder beim Bewerten der RAG-Qualität. Verwendung für Einbettungsstrategie, Vektorspeicher, Abrufpipelines, Chunking, Hybridsuche, Re-Ranking, Abruf mehrerer Abfragen, Abruf übergeordneter Dokumente, kontextbezogene Komprimierung, MMR-Diversitätsauswahl, reziproke Rangfolgefusion und Auswertung. Verwenden Sie für die Auswahl und Governance der KB-Architektur die Fähigkeit „Knowledge-Base-Manager“. Verwenden Sie für Wissensgraphen die Fähigkeit „Knowledge-Graph-Builder“.
