# Feature Value vs Effort Scorer

## Purpose
Prioritize features using structured, transparent scoring.

## Criteria
- User impact
- Revenue potential
- Strategic alignment
- Confidence level
- Engineering effort

## Steps
1. Score each criterion (1–5).
2. Calculate value score vs effort score.
3. Rank relative priority.

## Output

### Feature Name
### Value Score
### Effort Score
### Priority
High / Medium / Low

### Recommendation
Build Now / Later / Do Not Build
