Руководство по качеству кода на основе Toss Frontend Fundamentals. Предоставляет обратную связь по четырем критериям — читаемость, предсказуемость, связность и связанность — во время проверки кода, написания нового кода и рефакторинга. Активируется для обеспечения качества внешнего кода, чистоты кода и запросов на улучшение кода.
