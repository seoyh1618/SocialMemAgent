Für die Client-Entwicklung mit Token auf Solana ist Light Token 200-mal günstiger als SPL und weist nur minimale Änderungen auf. Der Skill umfasst Anleitungen zum Erstellen von Mints, zugehörigen Token-Konten, zum Übertragen, Genehmigen, Brennen, Verpacken und mehr. @lightprotocol/compressed-token (TypeScript) und light_token_client (Rust).
