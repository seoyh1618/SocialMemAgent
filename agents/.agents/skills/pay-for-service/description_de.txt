Stellen Sie mit USDC kostenpflichtige Anfragen an x402-fähige APIs. Verwenden Sie diese Option, wenn Sie oder der Benutzer eine kostenpflichtige API aufrufen, eine x402-Zahlung durchführen, einen Basardienst nutzen oder für eine API-Anfrage bezahlen möchten. Umfasst Ausdrücke wie „diese API aufrufen“, „diesen Dienst nutzen“, „für die Anfrage bezahlen“, „einen kostenpflichtigen Anruf tätigen“ und „vom x402-Endpunkt abrufen“.
