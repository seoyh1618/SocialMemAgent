Envoyez des requêtes payantes aux API compatibles x402 à l'aide de l'USDC. À utiliser lorsque vous ou l'utilisateur souhaitez appeler une API payante, effectuer un paiement x402, utiliser un service de bazar ou payer pour une demande d'API. Couvre des expressions telles que « appeler cette API », « utiliser ce service », « payer pour la demande », « passer un appel payant », « récupérer à partir du point de terminaison x402 ».
