Отправляйте платные запросы к API с поддержкой x402, используя USDC. Используйте, когда вы или пользователь хотите вызвать платный API, совершить платеж x402, использовать базарный сервис или оплатить запрос API. Охватывает такие фразы, как «вызовите этот API», «используйте эту услугу», «оплатите запрос», «сделайте платный вызов», «получите из конечной точки x402».
