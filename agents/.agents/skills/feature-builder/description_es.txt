Cree funciones completas de React con una arquitectura en capas adecuada que incluya componentes de interfaz de usuario, lógica empresarial, integración de API y gestión de estado. Utilice esta habilidad cuando los usuarios soliciten implementar funciones como "autenticación de usuario", "carrito de compras", "lista de productos", "carga de archivos" o cualquier funcionalidad completa que requiera interfaz de usuario + lógica empresarial + recuperación de datos. Genera todas las capas: presentación (componentes), lógica empresarial (ganchos/almacenamiento/validación) y acceso a datos (llamadas API/React Query). Se integra con el generador de componentes de reacción para la interfaz de usuario y proporciona código mantenible y listo para producción siguiendo las mejores prácticas.
