Erstellen Sie vollständige React-Funktionen mit einer geeigneten Schichtarchitektur, einschließlich UI-Komponenten, Geschäftslogik, API-Integration und Statusverwaltung. Verwenden Sie diese Fähigkeit, wenn Benutzer die Implementierung von Funktionen wie „Benutzerauthentifizierung“, „Warenkorb“, „Produktliste“, „Datei-Upload“ oder einer vollständigen Funktionalität anfordern, die eine Benutzeroberfläche + Geschäftslogik + Datenabruf erfordert. Generiert alle Ebenen – Präsentation (Komponenten), Geschäftslogik (Hooks/Stores/Validierung) und Datenzugriff (API-Aufrufe/React Query). Integriert sich in den React-Component-Generator für die Benutzeroberfläche und stellt produktionsbereiten, wartbaren Code gemäß Best Practices bereit.
