UI コンポーネント、ビジネス ロジック、API 統合、状態管理などの適切な階層化アーキテクチャを使用して、完全な React 機能を構築します。このスキルは、ユーザーが「ユーザー認証」、「ショッピング カート」、「商品リスト」、「ファイル アップロード」などの機能、または UI + ビジネス ロジック + データの取得を必要とする完全な機能の実装を要求する場合に使用します。すべてのレイヤー - プレゼンテーション (コンポーネント)、ビジネス ロジック (フック/ストア/検証)、データ アクセス (API 呼び出し/React Query) を生成します。 UI 用の React-component-generator と統合し、ベスト プラクティスに従って本番環境に対応した保守可能なコードを提供します。
