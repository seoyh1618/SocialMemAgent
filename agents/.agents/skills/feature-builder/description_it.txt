Crea funzionalità React complete con un'adeguata architettura a più livelli che include componenti dell'interfaccia utente, logica aziendale, integrazione API e gestione dello stato. Utilizza questa competenza quando gli utenti richiedono l'implementazione di funzionalità come "autenticazione utente", "carrello degli acquisti", "elenco dei prodotti", "caricamento di file" o qualsiasi funzionalità completa che richieda interfaccia utente + logica aziendale + recupero dei dati. Genera tutti i livelli: presentazione (componenti), logica aziendale (hook/negozi/convalida) e accesso ai dati (chiamate API/React Query). Si integra con react-component-generator per l'interfaccia utente e fornisce codice gestibile e pronto per la produzione seguendo le migliori pratiche.
