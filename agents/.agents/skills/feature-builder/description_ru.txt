Создавайте полные функции React с правильной многоуровневой архитектурой, включая компоненты пользовательского интерфейса, бизнес-логику, интеграцию API и управление состоянием. Используйте этот навык, когда пользователи запрашивают реализацию таких функций, как «аутентификация пользователя», «корзина покупок», «список продуктов», «загрузка файлов» или любую полную функциональность, требующую пользовательский интерфейс + бизнес-логику + выборку данных. Генерирует все уровни — представление (компоненты), бизнес-логику (перехватчики/сохранения/проверка) и доступ к данным (вызовы API/запросы React). Интегрируется с генератором компонентов реагирования для пользовательского интерфейса и предоставляет готовый к использованию и поддерживаемый код в соответствии с лучшими практиками.
