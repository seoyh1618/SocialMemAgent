Créez des fonctionnalités React complètes avec une architecture en couches appropriée comprenant des composants d'interface utilisateur, une logique métier, une intégration d'API et une gestion d'état. Utilisez cette compétence lorsque les utilisateurs demandent la mise en œuvre de fonctionnalités telles que « authentification utilisateur », « panier », « liste de produits », « téléchargement de fichiers » ou toute fonctionnalité complète nécessitant une interface utilisateur + une logique métier + une récupération de données. Génère toutes les couches : présentation (composants), logique métier (hooks/magasins/validation) et accès aux données (appels API/React Query). S'intègre à React-Component-Generator pour l'interface utilisateur et fournit un code maintenable et prêt pour la production conformément aux meilleures pratiques.
