Implémente une inférence d'apprentissage automatique local hautes performances dans le navigateur à l'aide d'ONNX Runtime Web. Utilisez cette compétence lorsque l'utilisateur a besoin de fonctionnalités d'IA axées sur la confidentialité, à faible latence ou hors ligne (par exemple, classification d'images, détection d'objets ou NLP) sans traitement côté serveur.
