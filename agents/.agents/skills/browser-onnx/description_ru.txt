Реализует высокопроизводительный вывод локального машинного обучения в браузере с помощью ONNX Runtime Web. Используйте этот навык, когда пользователю нужны возможности искусственного интеллекта с приоритетом конфиденциальности, малой задержкой или автономные возможности (например, классификация изображений, обнаружение объектов или НЛП) без обработки на стороне сервера.
