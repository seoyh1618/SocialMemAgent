Tidyverse または Shiny ブログの規則に従って、プロフェッショナルなパッケージ リリースのブログ投稿を作成します。ユーザーが以下を行う必要がある場合に使用します: (1) Tidyverse.org または Shiny.posit.co の R または Python パッケージのリリース発表ブログ投稿を作成する、(2) NEWS/変更ログのコンテンツをブログ形式に変換する、(3) 寄稿者リストを含む謝辞セクションを生成する、(4) 特定のブログ プラットフォーム要件に従って投稿をフォーマットする。自動投稿者取得と包括的なスタイル ガイダンスにより、Tidyverse (hugodown) と Shiny (Quarto) の両方のブログ形式をサポートします。
