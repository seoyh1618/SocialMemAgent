قم بإنشاء منشورات مدونة لإصدار الحزمة الاحترافية وفقًا لاتفاقيات مدونة Tidyverse أو Shiny. يُستخدم عندما يحتاج المستخدم إلى: (1) كتابة منشور مدونة لإعلان الإصدار لحزمة R أو Python لـ tidyverse.org أو Shiny.posit.co، (2) تحويل محتوى NEWS/changelog إلى تنسيق مدونة، (3) إنشاء أقسام الشكر مع قوائم المساهمين، (4) تنسيق المنشورات وفقًا لمتطلبات النظام الأساسي للمدونة المحددة. يدعم كلاً من تنسيقات مدونات Tidyverse (hugodown) وShiny (Quarto) مع جلب تلقائي للمساهم وتوجيهات شاملة للأسلوب.
