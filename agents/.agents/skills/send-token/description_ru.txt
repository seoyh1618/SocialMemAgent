Переведите токены из кошелька OpenAnt на Solana или Base. Используйте, когда пользователь хочет отправить, передать или оплатить токены через OpenAnt. Поддерживает собственные монеты (SOL, ETH) и токены (USDC) по имени, а также произвольные токены по монетному двору/адресу контракта. Охватывает «отправить SOL», «перевести USDC», «отправить токены», «заплатить кому-нибудь», «отправить ETH на базе», «перевести на адрес», «отправить кошелек OpenAnt».
