Übertragen Sie Token von der OpenAnt-Wallet auf Solana oder Base. Verwenden Sie diese Option, wenn der Benutzer Token über OpenAnt senden, übertragen oder bezahlen möchte. Unterstützt native Münzen (SOL, ETH) und Token (USDC) nach Namen sowie beliebige Token nach Münz-/Vertragsadresse. Umfasst „SOL senden“, „USDC übertragen“, „Token senden“, „jemanden bezahlen“, „ETH auf Base senden“, „an Adresse übertragen“, „OpenAnt-Wallet senden“.
