دليل لتحديث مكونات WordPress الإضافية إلى أحدث ميزات المكونات الإضافية.
يُستخدم عندما: (1) تحديث أدوات البناء أو اعتماد @wordpress/scripts، (2) الاعتماد
ميزات جديدة مثل Abilities API، أو Loader المحسّن، أو سير عمل i18n، (3) المزامنة
خطوط أنابيب CI/CD، أو تكوينات ضمان الجودة، أو التبعيات، (4) يطلب المستخدم "التحديث من
النموذج المعياري أو "المزامنة مع النموذج المعياري" أو "الترحيل إلى النموذج المعياري الأحدث".
