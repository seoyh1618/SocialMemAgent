Guía para actualizar los complementos de WordPress a las últimas funciones estándar de complementos.
Úselo cuando: (1) actualice herramientas de compilación o adopte @wordpress/scripts, (2) adopte
nuevas características como Abilities API, Loader mejorado o flujos de trabajo i18n, (3) Sincronización
Canalizaciones de CI/CD, configuraciones de control de calidad o dependencias. (4) El usuario solicita "actualizar desde
texto repetitivo", "sincronizar con texto repetitivo" o "migrar al texto repetitivo más reciente".
