Guida per l'aggiornamento dei plugin WordPress alle ultime funzionalità plugin-boilerplate.
Da utilizzare quando: (1) si aggiornano gli strumenti di creazione o si adotta @wordpress/script, (2) si adotta
nuove funzionalità come API Abilities, caricatore migliorato o flussi di lavoro i18n, (3) sincronizzazione
Pipeline CI/CD, configurazioni QA o dipendenze, (4) L'utente chiede di "aggiornare da
boilerplate", "sincronizza con boilerplate" o "migra all'ultimo boilerplate".
