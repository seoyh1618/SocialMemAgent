Implemente el patrón de comando en Angular usando la biblioteca @ssv/ngx.command. Úselo para encapsular acciones con seguimiento del estado de ejecución, habilitar/deshabilitar controles y administrar operaciones asíncronas. Se activan los estados de los botones, la ejecución de acciones, el envío de formularios o la implementación de un patrón de comando.
