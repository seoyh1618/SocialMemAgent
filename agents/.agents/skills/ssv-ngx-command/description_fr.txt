Implémentez le modèle de commande dans Angular à l'aide de la bibliothèque @ssv/ngx.command. À utiliser pour encapsuler des actions avec le suivi de l'état d'exécution, activer/désactiver les contrôles et gérer les opérations asynchrones. Déclencheurs sur les états des boutons, l'exécution d'actions, les soumissions de formulaires ou la mise en œuvre d'un modèle de commande.
