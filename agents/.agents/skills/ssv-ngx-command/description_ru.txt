Реализуйте шаблон команды в Angular, используя библиотеку @ssv/ngx.command. Используйте для инкапсуляции действий с отслеживанием состояния выполнения, включением/отключением элементов управления и управлением асинхронными операциями. Запускается по состояниям кнопок, выполнению действий, отправке формы или реализации шаблона команды.
