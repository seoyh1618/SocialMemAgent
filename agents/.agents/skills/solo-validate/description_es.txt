Puntuación de idea de inicio a través de S.E.E.D. verificación de nicho + análisis STREAM de 6 capas + inversión del Abogado del Diablo, pila de selección automática y generación de PRD con criterios de aceptación. Úselo cuando el usuario diga "validar idea", "calificar esta idea", "debería construir esto", "ir o matar", "generar PRD" o "evaluar oportunidad". NO lo use para investigación profunda (use /investigación primero) o marco de solo toma de decisiones (use /stream).
