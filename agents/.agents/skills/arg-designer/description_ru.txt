Комплексный набор инструментов для проектирования ARG (Игра в альтернативной реальности) для создания захватывающих головоломок. Этот навык следует использовать, когда: (1) проектирует ARG или след головоломки с нуля, (2) создает скрытый контент с помощью шифров, стеганографии или кодирования, (3) создает пути (точки входа) к скрытому контенту, (4) объединяет элементы головоломки в связное повествование, (5) реализует определенные шифры или методы кодирования, (6) выбирает платформы для развертывания ARG (Discord, Twilio, YouTube), (7) рассматривает дизайн ARG для Принципы ТИНАГ. ОБЯЗАТЕЛЬНЫЕ ТРИГГЕРЫ: ARG, игра в альтернативной реальности, след головоломки, шифр, стеганография, скрытое сообщение, начало пути, TINAG, поиск головоломок, трансмедиа, художественная литература, кодирование сообщения, скрытие данных.
