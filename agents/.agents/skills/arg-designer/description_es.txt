Completo kit de herramientas de diseño ARG (juego de realidad alternativa) para crear experiencias de rompecabezas inmersivas. Esta habilidad debe usarse cuando: (1) Diseñar un ARG o una ruta de rompecabezas desde cero, (2) Crear contenido oculto con cifrados, esteganografía o codificación, (3) Construir senderos (puntos de entrada) a contenido oculto, (4) Conectar elementos de rompecabezas en una narrativa coherente, (5) Implementar cifrados o métodos de codificación específicos, (6) Elegir plataformas para la implementación de ARG (Discord, Twilio, YouTube), (7) Revisar el diseño de ARG para Principios TINAG. DISPARADORES OBLIGATORIOS: ARG, juego de realidad alternativa, ruta de rompecabezas, cifrado, esteganografía, mensaje oculto, inicio de ruta, TINAG, búsqueda de rompecabezas, transmedia, no ficción, codificar mensaje, ocultar datos
