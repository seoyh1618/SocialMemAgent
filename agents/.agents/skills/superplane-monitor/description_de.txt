Überwachen und debuggen Sie die Ausführung von SuperPlane-Workflows. Untersuchen Sie Ausführungen, diagnostizieren Sie Fehler und verwalten Sie Ausführungswarteschlangen. Verwenden Sie diese Option, wenn ein Workflow fehlschlägt, eine Ausführung hängen bleibt oder der Benutzer den Ausführungsstatus überprüfen möchte. Löst auf „Debug“, „Fehlgeschlagen“, „Ausführung“, „Ausführungsstatus“, „Stuck“, „Warteschlange“, „Fehlerbehebung“ aus.
