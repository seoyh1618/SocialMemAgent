Обеспечивает целостность базы данных для систем PostgreSQL и Prisma. Используйте при разработке схем, написании миграций или настройке безопасности на уровне строк. Используйте для типобезопасного SQL, выравнивания имен, проверки ограничений, политик RLS с нулевым доверием, первичных ключей UUIDv7 и развертываний с нулевым временем простоя.
