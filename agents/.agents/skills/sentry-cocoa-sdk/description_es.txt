Configuración completa de Sentry SDK para plataformas Apple (iOS, macOS, tvOS, watchOS, visionOS). Úselo cuando se le solicite "agregar Sentry a iOS", "agregar Sentry a Swift", "instalar sentry-cocoa" o configurar el monitoreo de errores, el seguimiento, la creación de perfiles, la reproducción de sesiones o el registro para aplicaciones de Apple. Soporta SwiftUI y UIKit.
