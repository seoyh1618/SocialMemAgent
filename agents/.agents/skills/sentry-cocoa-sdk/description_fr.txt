Configuration complète du SDK Sentry pour les plateformes Apple (iOS, macOS, tvOS, watchOS, visionOS). À utiliser lorsqu'on vous demande « d'ajouter Sentry à iOS », « d'ajouter Sentry à Swift », « d'installer Sentry-cocoa » ou de configurer la surveillance des erreurs, le traçage, le profilage, la relecture de session ou la journalisation pour les applications Apple. Prend en charge SwiftUI et UIKit.
