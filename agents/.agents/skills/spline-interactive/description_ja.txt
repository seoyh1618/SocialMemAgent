ビジュアル エディター、アニメーション、Web エクスポートを備えたブラウザベースの 3D デザイン ツール。このスキルは、コードなしで 3D シーンを作成する場合、インタラクティブな Web エクスペリエンスをデザインする場合、3D UI のプロトタイプを作成する場合、React/Web にエクスポートする場合、またはデザイナーに適した 3D コンテンツを構築する場合に使用します。スプライン、ノーコード 3D、ビジュアル 3D エディター、3D アニメーション、状態ベースのインタラクション、React Spline 統合、またはシーン エクスポートに関連するタスクをトリガーします。コードよりもビジュアル ツールを好むデザイナー向けの Three.js の代替手段。
