أداة تصميم ثلاثية الأبعاد تعتمد على المتصفح مع محرر مرئي ورسوم متحركة وتصدير الويب. استخدم هذه المهارة عند إنشاء مشاهد ثلاثية الأبعاد بدون تعليمات برمجية، أو تصميم تجارب ويب تفاعلية، أو إنشاء نماذج أولية لواجهة المستخدم ثلاثية الأبعاد، أو التصدير إلى React/web، أو إنشاء محتوى ثلاثي الأبعاد صديق للمصمم. تشغيل المهام التي تتضمن Spline، أو 3D بدون كود، أو محرر مرئي ثلاثي الأبعاد، أو رسوم متحركة ثلاثية الأبعاد، أو تفاعلات قائمة على الحالة، أو تكامل React Spline، أو تصدير المشهد. بديل لـ Three.js للمصممين الذين يفضلون الأدوات المرئية على التعليمات البرمجية.
