Браузерный инструмент 3D-дизайна с визуальным редактором, анимацией и веб-экспортом. Используйте этот навык при создании 3D-сцен без кода, проектировании интерактивных веб-интерфейсов, прототипировании 3D-интерфейса, экспорте в React/web или создании удобного для дизайнеров 3D-контента. Запускается при выполнении задач, включающих Spline, 3D без кода, визуальный 3D-редактор, 3D-анимацию, взаимодействия на основе состояний, интеграцию React Spline или экспорт сцен. Альтернатива Three.js для дизайнеров, которые предпочитают визуальные инструменты коду.
