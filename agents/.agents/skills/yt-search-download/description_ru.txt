Поиск видео на YouTube, загрузка видео, инструмент для загрузки субтитров. Интегрирует API данных YouTube v3 для расширенного поиска и yt-dlp для загрузки видео/аудио/субтитров.
Основные возможности: поиск по ключевым словам по всему сайту, просмотр каналов, сортировка по времени/количеству просмотров/релевантности, загрузка видео, извлечение аудио (MP3), загрузка субтитров (китайский и английский), просмотр сведений о видео.
Триггерные сценарии (используйте этот навык для любой операции на YouTube):
- Поиск: «поиск на YouTube», «поиск на YouTube», «найти видео на YouTube», «поиск последних видео на [канале]», «найти на YouTube [имя]», «последние обновления на YouTube [имя]», «видео на YouTube о XXX».
- Просмотр каналов: «просмотреть канал», «проверить последние видео [канала]», «какие видео недавно опубликовал [имя]», «обновления [имя] YouTube», «просмотреть список видео [канал]»
- Загрузка видео: «загрузить YouTube», «загрузить это видео», «загрузить видео с YouTube», «сохранить это видео», «сохранить видео».
- Загрузка аудио: «извлечь аудио», «загрузить аудио», «YouTube в MP3», «только аудио».
- Загрузка субтитров: «загрузить субтитры», «извлечь субтитры», «получить субтитры», «загрузить субтитры YouTube», «китайские субтитры», «английские субтитры», «извлечь текст», «видео в текст», «получить видеотекст».
- Информация о видео: «детали видео», «информация о видео», «сколько длится это видео».
Соответствие ключевому слову: активируйте этот навык всякий раз, когда сообщение пользователя содержит «YouTube», «油管», «YT», «yt» плюс слова действия, такие как поиск/загрузка/субтитры/канал/видео.
