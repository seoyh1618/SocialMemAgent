YouTube影片搜尋、影片下載、字幕下載工具。整合了用於高級搜尋的 YouTube Data API v3 和用於下載視訊/音訊/字幕的 yt-dlp。
核心能力：全站關鍵字搜尋、頻道瀏覽、依時間/觀看次數/相關性排序、影片下載、音訊擷取（MP3）、字幕下載（中英文）、影片詳情檢視。
觸發場景（將此技能用於任何 YouTube 操作）：
- 搜尋：「搜尋 YouTube」、「YouTube 搜尋」、「尋找 YouTube 影片」、「搜尋 [頻道] 最新影片」、「尋找 [姓名] 的 YouTube」、「[姓名] 最近的 YouTube 更新」、「關於 XXX 的 YouTube 影片”
- 頻道瀏覽：“瀏覽頻道”、“查看[頻道]最新視頻”、“[名稱]最近發布了哪些視頻”、“[名稱] YouTube 更新”、“查看[頻道]視頻列表”
- 影片下載：“下載YouTube”、“下載此影片”、“下載YouTube影片”、“儲存此影片”、“儲存影片”
- 音訊下載：“提取音訊”、“下載音訊”、“YouTube 到 MP3”、“僅音訊”
- 字幕下載：「下載字幕」、「擷取字幕」、「取得字幕」、「下載 YouTube 字幕」、「中文字幕」、「英文字幕」、「擷取文字」、「影片轉文字」、「取得視訊文字」
- 視訊訊息：“視訊詳細資料”、“視訊訊息”、“該視訊長度”
關鍵字匹配：當用戶訊息中包含「YouTube」、「油管」、「YT」、「yt」以及搜尋/下載/字幕/頻道/影片等動作詞時，觸發此技能。
