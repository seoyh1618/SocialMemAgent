YouTube视频搜索、视频下载、字幕下载工具。集成了用于高级搜索的 YouTube Data API v3 和用于下载视频/音频/字幕的 yt-dlp。
核心能力：全站关键词搜索、频道浏览、按时间/观看次数/相关性排序、视频下载、音频提取（MP3）、字幕下载（中英文）、视频详情查看。
触发场景（将此技能用于任何 YouTube 操作）：
- 搜索：“搜索 YouTube”、“YouTube 搜索”、“查找 YouTube 视频”、“搜索 [频道] 最新视频”、“查找 [姓名] 的 YouTube”、“[姓名] 最近的 YouTube 更新”、“关于 XXX 的 YouTube 视频”
- 频道浏览：“浏览频道”、“查看[频道]最新视频”、“[名称]最近发布了哪些视频”、“[名称] YouTube 更新”、“查看[频道]视频列表”
- 视频下载：“下载YouTube”、“下载此视频”、“下载YouTube视频”、“保存此视频”、“保存视频”
- 音频下载：“提取音频”、“下载音频”、“YouTube 到 MP3”、“仅音频”
- 字幕下载：“下载字幕”、“提取字幕”、“获取字幕”、“下载 YouTube 字幕”、“中文字幕”、“英文字幕”、“提取文字”、“视频转文字”、“获取视频文字”
- 视频信息：“视频详细信息”、“视频信息”、“该视频时长”
关键词匹配：当用户消息中包含“YouTube”、“油管”、“YT”、“yt”以及搜索/下载/字幕/频道/视频等动作词时，触发此技能。
