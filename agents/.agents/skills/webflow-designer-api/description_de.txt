Arbeiten Sie mit der Webflow Designer-API – entweder durch die Erstellung von Designer-Erweiterungen (Iframes innerhalb des Webflow Designers) oder durch die Generierung von Code-Snippets für den Designer API Playground. Behandelt Elementmanipulation, Stile, Komponenten, Seiten, Variablen, Assets, Fehlerbehandlung, CLI-Nutzung und UI-Designmuster. Verwenden Sie diese Option beim Erstellen, Debuggen oder Ändern von Designer-Erweiterungen ODER wenn der Benutzer Designer-API-Code in der Playground-App ausführen möchte.
