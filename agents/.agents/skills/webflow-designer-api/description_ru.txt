Работайте с API-интерфейсом конструктора Webflow — либо путем создания расширений конструктора (iframe внутри конструктора Webflow), либо путем создания фрагментов кода для игровой площадки Designer API. Охватывает манипуляции с элементами, стили, компоненты, страницы, переменные, ресурсы, обработку ошибок, использование CLI и шаблоны проектирования пользовательского интерфейса. Используйте при создании, отладке или изменении расширений Designer, ИЛИ когда пользователь хочет запустить код Designer API в приложении Playground.
