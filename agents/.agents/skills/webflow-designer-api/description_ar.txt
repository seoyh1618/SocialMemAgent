العمل مع Webflow Designer API - إما عن طريق إنشاء ملحقات المصمم (إطارات iframe داخل Webflow Designer) أو عن طريق إنشاء مقتطفات من التعليمات البرمجية لـ Designer API Playground. يغطي معالجة العناصر والأنماط والمكونات والصفحات والمتغيرات والأصول ومعالجة الأخطاء واستخدام واجهة سطر الأوامر (CLI) وأنماط تصميم واجهة المستخدم. يُستخدم عند إنشاء ملحقات المصمم أو تصحيح الأخطاء أو تعديلها، أو عندما يريد المستخدم تشغيل كود Designer API في تطبيق Playground.
