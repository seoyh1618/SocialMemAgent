Trabaje con la API de Webflow Designer, ya sea creando extensiones de Designer (iframes dentro de Webflow Designer) o generando fragmentos de código para Designer API Playground. Cubre la manipulación de elementos, estilos, componentes, páginas, variables, activos, manejo de errores, uso de CLI y patrones de diseño de UI. Úselo al crear, depurar o modificar extensiones de Designer, O cuando el usuario desee ejecutar el código API de Designer en la aplicación Playground.
