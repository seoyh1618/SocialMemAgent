Генератор инструкций по миграции и развитию кода для GitHub Copilot. Анализирует различия между двумя версиями проекта (ветвями, фиксациями или выпусками) для создания точных инструкций, позволяющих Copilot поддерживать согласованность во время миграции технологий, масштабного рефакторинга или обновления версии платформы.
