Wenn der Benutzer ein Produkt an Verzeichnisse, Startplattformen, kuratierte Listen oder App Stores senden möchte. Liest „product-marketing-context.md“, sofern vorhanden, und generiert fertige Übermittlungsinhalte für die jeweilige Plattform, die eingefügt werden können. Wird auch verwendet, wenn der Benutzer „Verzeichniseinreichung“, „gelistet werden“, „App-Store-Eintrag“, „An Verzeichnisse senden“, „kuratierte Liste“, „Liste der besten Tools“, „Taaft“, „Produktsuche“, „Verzeichnisanzeigen“, „Newsletter-Funktion“, „Verzeichniskampagne“, „Produktinformationen für Verzeichnis“, „Beschreibung pro Plattform anpassen“, „Shopify App Store“, „Chrome Web Store“, „An Verzeichnis senden“, „Auf Produkt starten“ erwähnt „Jagd“, „Navigationsseite“ oder „Produktverzeichnis“.
