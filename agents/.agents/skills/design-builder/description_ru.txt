Конвейер проектирования в код (извлечение копии из URL-адресов, извлечение токенов дизайна из изображений, затем создание компонентов React или вариантов предварительного просмотра HTML). Используйте при извлечении контента с веб-сайтов, извлечении систем дизайна, генерации кода внешнего интерфейса, предварительном просмотре вариантов дизайна, отправке в Figma через MCP. Также используйте, когда пользователь хочет создать страницу на основе URL-адреса или снимка экрана, изменить дизайн существующего сайта, создать визуальные прототипы или сгенерировать код на основе дизайна. Триггеры: «извлечь копию», «извлечь дизайн», «создать внешний интерфейс», «создать варианты», «экспортировать дизайн», «отправить в Figma», «создать на основе образца», «перепроектировать это», «создать прототип».
