Полная настройка Sentry SDK для браузерного JavaScript. Используйте, когда вас просят «добавить Sentry на веб-сайт», «установить @sentry/browser» или настроить мониторинг ошибок, трассировку, воспроизведение сеанса или ведение журнала для стандартного JavaScript, jQuery, статических сайтов или WordPress.
