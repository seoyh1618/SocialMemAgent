Полное руководство по разработке плагинов для конвейеров, включающее создание плагинов, спецификации конфигурации Task.json, многоязычные примеры (Python/Java/NodeJS/Golang), спецификации ввода/вывода, стандарты кодов ошибок, процесс выпуска и методы отладки. Используйте при разработке плагинов конвейера BlueShield, настройке Task.json, обработке ввода-вывода плагина или устранении ошибок плагина.
