Guía completa para el desarrollo de complementos de Pipeline, que cubre la creación de complementos, especificaciones de configuración de task.json, ejemplos en varios idiomas (Python/Java/NodeJS/Golang), especificaciones de entrada/salida, estándares de códigos de error, proceso de lanzamiento y métodos de depuración. Úselo al desarrollar complementos de canalización de BlueShield, configurar task.json, manejar E/S de complementos o solucionar errores de complementos.
