Guide complet de développement de plugins de pipeline, couvrant la création de plugins, les spécifications de configuration task.json, les exemples multilingues (Python/Java/NodeJS/Golang), les spécifications d'entrée/sortie, les normes de code d'erreur, le processus de publication et les méthodes de débogage. À utiliser lors du développement de plugins de pipeline BlueShield, de la configuration de task.json, de la gestion des E/S de plugin ou du dépannage des erreurs de plugin.
