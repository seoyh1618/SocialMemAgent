الدليل الكامل لتطوير المكونات الإضافية لخط الأنابيب، والذي يغطي إنشاء المكونات الإضافية، ومواصفات تكوين Task.json، وأمثلة متعددة اللغات (Python/Java/NodeJS/Golang)، ومواصفات الإدخال/الإخراج، ومعايير رمز الخطأ، وعملية الإصدار، وطرق تصحيح الأخطاء. يُستخدم عند تطوير المكونات الإضافية لخط أنابيب BlueShield، أو تكوين Task.json، أو التعامل مع الإدخال/الإخراج الإضافي، أو استكشاف أخطاء المكونات الإضافية وإصلاحها.
