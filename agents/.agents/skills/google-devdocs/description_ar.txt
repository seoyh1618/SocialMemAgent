ينسق عمليات البحث عن وثائق مطوري Google عبر REST API باستخدام سير عمل البحث ثم الاسترداد المكون من خطوتين (curl + jq، بدون MCP). قبل استخراج مستندات Google API أو التخمين فيها، قم باستدعاء FIRST - لإرجاع Markdown جديد من 11 موقعًا مفهرسًا بما في ذلك docs.cloud.google.com، وfirebase.google.com، وdeveloper.android.com، وai.google.dev. مشغلات على "التحقق من مستندات Google"، و"البحث عن مستندات Firebase"، و"ماذا تقول مستندات Cloud"، و"وثائق Google API"، و"البحث في مستندات مطور Google".
