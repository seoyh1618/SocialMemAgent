Организует поиск документации разработчика Google через REST API, используя двухэтапный рабочий процесс поиска и последующего извлечения (curl + jq, без MCP). ПРЕЖДЕ ЧЕМ просматривать или гадать в документации Google API, вызовите FIRST — возвращает свежий Markdown с 11 проиндексированных сайтов, включая docs.cloud.google.com, firebase.google.com, Developer.android.com и ai.google.dev. Триггеры: «проверьте документы Google», «поищите документы Firebase», «что говорят облачные документы», «документацию Google API», «поиск документов разработчиков Google». (пользователь)
