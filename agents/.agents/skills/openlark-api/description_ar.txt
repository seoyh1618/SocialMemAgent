مواصفات تنفيذ واجهة برمجة تطبيقات مشروع OpenLark (مرجع سريع). يُستخدم لإضافة/إعادة بناء واجهات برمجة التطبيقات Feishu Open Platform API: تحديد مسار الملف، وتنفيذ Body/Response + Builder(Request)، ومحاذاة ثوابت/تعدادات نقطة النهاية، وإكمال عمليات تصدير mod.rs، وتوضيح توقيع الأسلوب من أجل "استدعاء واجهات برمجة التطبيقات من جانب الخادم" واصطلاحات تمرير RequestOption. تشغيل الكلمات الرئيسية: واجهة API، ملف API، Feishu API، إضافة API، استدعاء API من جانب الخادم
