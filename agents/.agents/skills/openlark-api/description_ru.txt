Спецификация реализации API проекта OpenLark (Краткий справочник). Используется для добавления/рефакторинга API-интерфейсов открытой платформы Feishu: определение пути к файлу, реализация Body/Response + Builder(Request), выравнивание констант/перечислений конечной точки, завершение экспорта mod.rs и уточнение сигнатуры метода для «вызова серверных API» и соглашений о передаче RequestOption. Ключевые слова-триггеры: интерфейс API, файл API, API Feishu, добавление API, вызов API на стороне сервера.
