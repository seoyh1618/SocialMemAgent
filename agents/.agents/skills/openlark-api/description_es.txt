Especificación de implementación de API del proyecto OpenLark (referencia rápida). Se utiliza para agregar/refactorizar las API de la plataforma abierta Feishu: determinar la ruta del archivo, implementar Body/Response + Builder (Solicitud), alinear constantes/enumeraciones de puntos finales, completar exportaciones mod.rs y aclarar la firma del método para "llamar a las API del lado del servidor" y las convenciones de paso de RequestOption. Palabras clave de activación: interfaz API, archivo API, API Feishu, agregar API, llamar a API del lado del servidor
