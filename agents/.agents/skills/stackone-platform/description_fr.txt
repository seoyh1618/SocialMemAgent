Gérez les ressources StackOne, notamment les clés API, les comptes liés, les journaux et les webhooks. À utiliser lorsque l'utilisateur demande de « configurer StackOne », « répertorier mes comptes », « déboguer les erreurs d'API », « vérifier l'état de l'intégration » ou « configurer les webhooks ». Couvre l'authentification, la gestion des comptes et le dépannage. Ne PAS utiliser pour créer des agents IA (utilisez des agents stackone) ou pour découvrir les capacités des connecteurs (utilisez des connecteurs stackone).
