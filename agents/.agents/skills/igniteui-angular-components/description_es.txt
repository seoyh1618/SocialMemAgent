Todos los componentes de Ignite UI Angular UI: configuración y arquitectura de la aplicación, controles de formulario (grupo de entrada, combo, selección, selectores de fecha/hora, calendario, casilla de verificación, radio, interruptor, control deslizante, formularios reactivos), diseño (pestañas, navegación inferior, paso a paso, acordeón, divisor, cajón de navegación, administrador de diseño), visualización de datos (lista, árbol, tarjeta, chips, avatar, insignia, icono, carrusel, paginador, barra de progreso, barra de progreso lineal, barra de progreso circular, chat), comentarios/superposiciones (diálogo, barra de refrigerios, brindis, banner), directivas (botón, ondulación, información sobre herramientas, arrastrar y soltar), administrador de muelle y gráficos (gráfico de áreas, gráfico de barras, gráfico de columnas, gráfico de acciones/financiero, gráfico circular, IgxCategoryChart, IgxFinancialChart, IgxDataChart, IgxPieChart). Úselo para cualquier pregunta sobre la interfaz de usuario angular de Ignite UI que no sea de cuadrícula.
