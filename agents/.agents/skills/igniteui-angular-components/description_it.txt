Tutti i componenti dell'interfaccia utente angolare di Ignite UI: configurazione e architettura dell'applicazione, controlli dei moduli (gruppo di input, combo, selezione, selettori data/ora, calendario, casella di controllo, radio, interruttore, dispositivo di scorrimento, moduli reattivi), layout (schede, navigazione inferiore, stepper, fisarmonica, divisore, cassetto di navigazione, gestione layout), visualizzazione dati (elenco, albero, scheda, chip, avatar, badge, icona, carosello, impaginatore, barra di avanzamento, barra di avanzamento lineare, avanzamento circolare barra, chat), feedback/sovrapposizioni (dialogo, snackbar, toast, banner), direttive (pulsante, ondulazione, descrizione comando, trascina e rilascia), Dock Manager e grafici (grafico ad area, grafico a barre, grafico a colonne, grafico azionario/finanziario, grafico a torta, IgxCategoryChart, IgxFinancialChart, IgxDataChart, IgxPieChart). Utilizzare per qualsiasi domanda sull'interfaccia utente angolare di Ignite UI non basata sulla griglia.
