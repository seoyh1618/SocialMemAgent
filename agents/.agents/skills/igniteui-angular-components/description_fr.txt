Tous les composants de l'interface utilisateur angulaire d'Ignite UI : configuration et architecture de l'application, contrôles de formulaire (groupe d'entrée, combo, sélection, sélecteurs de date/heure, calendrier, case à cocher, radio, commutateur, curseur, formulaires réactifs), disposition (onglets, navigation inférieure, pas à pas, accordéon, séparateur, tiroir de navigation, gestionnaire de mise en page), affichage des données (liste, arbre, carte, puces, avatar, badge, icône, carrousel, paginateur, barre de progression, barre de progression linéaire, barre de progression circulaire, chat), commentaires/superpositions (dialogue, Snackbar, Toast, bannière), directives (bouton, ondulation, info-bulle, glisser-déposer), gestionnaire de Dock et graphiques (graphique en aires, graphique à barres, graphique à colonnes, graphique boursier/financier, graphique à secteurs, IgxCategoryChart, IgxFinancialChart, IgxDataChart, IgxPieChart). À utiliser pour toute question d'interface utilisateur angulaire d'Ignite UI sans grille.
