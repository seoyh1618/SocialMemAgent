Utilice esta habilidad para revisar el código de IA de múltiples modelos. Se activa cada vez que el usuario solicita revisar cambios de código, auditar una diferencia, verificar la calidad del código, revisar un PR, revisar confirmaciones o revisar cambios no confirmados antes de enviar o fusionar. También se activa cuando dicen "revisión de código", "revisar mis cambios", "verificar esto antes de fusionar" o quieren múltiples perspectivas sobre el código. Ejecuta revisiones de Codex, Gemini y Claude en paralelo y luego sintetiza un informe unificado. NO lo use para revisar documentación, rebajas o archivos sin código, ni para cambios triviales de una sola línea.
