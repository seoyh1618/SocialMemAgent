Verwenden Sie diese Fähigkeit für die Überprüfung von KI-Code mit mehreren Modellen. Wird immer dann ausgelöst, wenn der Benutzer darum bittet, Codeänderungen zu überprüfen, einen Unterschied zu prüfen, die Codequalität zu überprüfen, eine PR zu überprüfen, Commits zu überprüfen oder nicht festgeschriebene Änderungen zu überprüfen, bevor er übertragen oder zusammengeführt wird. Wird auch ausgelöst, wenn sie „Codeüberprüfung“, „Meine Änderungen überprüfen“, „Überprüfen Sie dies vor dem Zusammenführen“ sagen oder mehrere Perspektiven auf den Code wünschen. Führt Codex-, Gemini- und Claude-Reviews parallel durch und erstellt dann einen einheitlichen Bericht. NICHT zum Überprüfen von Dokumentationen, Markdown- oder Nicht-Code-Dateien oder für triviale einzeilige Änderungen verwenden.
