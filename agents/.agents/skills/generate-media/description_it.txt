Abilità di generazione di media drammatici in forma abbreviata. Sulla base di un catalogo di opere singole già generato, chiama l'API di Google Gemini (un singolo modello di immagine configurabile per generare immagini di personaggi/storyboard) e salva le immagini generate nelle cartelle degli episodi corrispondenti. Supporta la configurazione preimpostata dello stile visivo. Parole chiave: generazione di immagini, Gemini, API di Google, immagini dello storyboard, immagini di riferimento dei personaggi, generazione di media, stile visivo.
