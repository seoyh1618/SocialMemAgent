Habilidad de generación de medios de formato corto. Basado en un catálogo de trabajo único ya generado, llame a la API de Google Gemini (un modelo de imagen único configurable para imágenes de personajes/guiones gráficos), almacene las imágenes generadas en los directorios de episodios correspondientes. Admite configuración de estilo visual preestablecida. Palabras clave: generación de imágenes, Gemini, API de Google, imágenes de guiones gráficos, imágenes de referencia de personajes, generación de medios, estilo visual.
