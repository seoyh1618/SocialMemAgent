مهارة توليد الوسائط القصيرة. استنادًا إلى كتالوج العمل الفردي الذي تم إنشاؤه بالفعل، اتصل بـ Google Gemini API (نموذج صورة واحد قابل للتكوين لإنشاء صور الشخصيات/القصة المصورة)، واحفظ الصور التي تم إنشاؤها في مجلدات الحلقات المعنية. يدعم التكوين المسبق للنمط المرئي. الكلمات المفتاحية: توليد الصور، الجوزاء، Google API، صور القصة المصورة، الصور المرجعية للشخصية، توليد الوسائط، النمط البصري.
