Инструмент анализа стоимости инвестиций в акции, обеспечивающий проверку акций, углубленный анализ отдельных акций, сравнение отраслей и функции расчета оценки. Основываясь на теории стоимостного инвестирования, он использует акшаре для получения общедоступных финансовых данных, подходящих для обычных инвесторов с низкочастотной торговлей.
