Carica in modo PROATTIVO quando l'attività prevede l'ottimizzazione della velocità, la riduzione delle dimensioni del pacchetto o il miglioramento della reattività. Da utilizzare quando l'utente dice "rendilo più veloce", "riduci le dimensioni del bundle", "correggi query lente", "ottimizza il rendering" o "controlla Core Web Vitals". Copre l'analisi dei bundle e lo scuotimento degli alberi, l'ottimizzazione delle query del database (N+1, indicizzazione), le prestazioni di rendering di React (re-render, memoizzazione), l'ottimizzazione della cascata di rete, il rilevamento delle perdite di memoria, le prestazioni lato server e il miglioramento dei Core Web Vitals (LCP, FID, CLS).
