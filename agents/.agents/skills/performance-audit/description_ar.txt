قم بالتحميل بشكل استباقي عندما تتضمن المهمة تحسين السرعة أو تقليل حجم الحزمة أو تحسين الاستجابة. يُستخدم عندما يقول المستخدم "اجعله أسرع"، أو "تقليل حجم الحزمة"، أو "إصلاح الاستعلامات البطيئة"، أو "تحسين العرض"، أو "التحقق من مؤشرات أداء الويب الأساسية". يغطي تحليل الحزم واهتزاز الأشجار، وتحسين استعلام قاعدة البيانات (N+1، الفهرسة)، وأداء عرض React (إعادة العرض، والحفظ)، وتحسين شلال الشبكة، واكتشاف تسرب الذاكرة، والأداء من جانب الخادم، وتحسين Core Web Vitals (LCP، FID، CLS).
