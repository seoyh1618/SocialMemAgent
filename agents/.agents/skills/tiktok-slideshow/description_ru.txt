Создает карусели изображений TikTok (слайд-шоу с наложением текста на фотографии) через API ViralBaby. Используйте, когда пользователь хочет: создавать слайд-шоу или карусели TikTok, находить/искать фоновые изображения для контента в социальных сетях, публиковать или загружать контент слайд-шоу в TikTok, редактировать текст слайдов с помощью искусственного интеллекта или управлять коллекциями изображений для создания контента. НЕ используйте для: общего управления учетными записями TikTok, аналитики или показателей TikTok, редактирования видео или создания видео (только для слайд-шоу фотографий), платформ социальных сетей, не относящихся к TikTok, или любых задач, не связанных с созданием визуального контента слайд-шоу для TikTok.
