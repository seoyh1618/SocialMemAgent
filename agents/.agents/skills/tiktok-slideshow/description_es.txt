Crea carruseles de imágenes de TikTok (presentaciones de diapositivas con superposiciones de texto en fotos) a través de la API ViralBaby. Úselo cuando el usuario desee: crear presentaciones de diapositivas o carruseles de TikTok, buscar/buscar imágenes de fondo para contenido de redes sociales, publicar o cargar contenido de presentaciones de diapositivas en TikTok, editar texto de diapositivas con IA o administrar colecciones de imágenes para la creación de contenido. NO lo use para: administración general de cuentas de TikTok, análisis o métricas de TikTok, edición o creación de videos (esto es solo para presentaciones de fotos), plataformas de redes sociales que no sean de TikTok o cualquier tarea no relacionada con la creación de contenido visual de presentaciones de diapositivas para TikTok.
