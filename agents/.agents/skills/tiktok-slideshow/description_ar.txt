ينشئ مجموعات صور TikTok (عروض شرائح تحتوي على تراكبات نصية على الصور) عبر واجهة برمجة تطبيقات ViralBaby. استخدمه عندما يريد المستخدم: إنشاء عروض شرائح أو دوارات TikTok، أو العثور على/البحث عن صور الخلفية لمحتوى الوسائط الاجتماعية، أو نشر محتوى عرض الشرائح أو تحميله إلى TikTok، أو تحرير نص الشرائح باستخدام الذكاء الاصطناعي، أو إدارة مجموعات الصور لإنشاء المحتوى. لا تستخدمه من أجل: الإدارة العامة لحساب TikTok، أو تحليلات أو مقاييس TikTok، أو تحرير الفيديو أو إنشاء الفيديو (هذا لعرض شرائح الصور فقط)، أو منصات الوسائط الاجتماعية غير التابعة لـ TikTok، أو أي مهمة لا علاقة لها بإنشاء محتوى عرض شرائح مرئي لـ TikTok.
