Vérifiez le solde USDC sur les réseaux. À utiliser lorsque vous ou l'utilisateur souhaitez vérifier le solde, voir combien d'argent se trouve dans le portefeuille, consulter les fonds ou vérifier l'USDC disponible. Couvre des expressions telles que « vérifier mon solde », « combien d'USDC ai-je », « quel est mon solde », « afficher les fonds », « le solde du portefeuille ».
