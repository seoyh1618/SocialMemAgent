يكتب ويحافظ على قصص القصص المصورة واختبارات التفاعل باستخدام تنسيق CSF3. يغطي القصص المكونة، ووظائف اللعب، واختبار الانحدار البصري باستخدام اللوني، واختبار إمكانية الوصول. يُستخدم عند إنشاء وثائق المكونات، أو كتابة اختبارات التفاعل، أو تصحيح أخطاء الاختبار، أو تكوين اللقطات المرئية، أو تحديث بنية القصة. يستخدم لـ CSF3، ووظائف التشغيل، وuserEvent، واستعلامات مكتبة الاختبار، والتكوين اللوني، وautodocs، وMDX.
