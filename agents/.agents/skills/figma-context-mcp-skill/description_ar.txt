قم بترجمة عقد Figma إلى كود جاهز للإنتاج بدقة مرئية 1:1 باستخدام Framelink Figma-Context-MCP (get_figma_data وdownload_figma_images). يتم تشغيله عندما يقدم المستخدم عناوين URL لـ Figma أو معرفات العقدة، أو يطلب تنفيذ المكونات/الصفحات التي يجب أن تتطابق مع Figma بدقة، ويجب أن يستخدم سير العمل Figma-Context-MCP بدلاً من خادم Figma MCP الرسمي.
