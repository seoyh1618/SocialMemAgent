Переведите узлы Figma в готовый к использованию код с визуальной точностью 1:1 с помощью Framelink Figma-Context-MCP (get_figma_data и download_figma_images). Запускается, когда пользователь предоставляет URL-адреса Figma или идентификаторы узлов или просит реализовать компоненты/страницы, которые должны точно соответствовать Figma, и рабочий процесс должен использовать Figma-Context-MCP вместо официального сервера Figma MCP.
