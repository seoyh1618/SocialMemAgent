Создавайте и развертывайте автономные агенты — сценарии, которые запускаются по расписанию, вызывают LLM, очищают Интернет, хранят данные и действуют без вмешательства пользователя. Используйте, когда пользователь говорит «создай мне агента», «создай бота», «автоматизируй X по расписанию», «отслеживай эту страницу», «подводи итоги X каждый день», «предупреждай меня, когда Y изменится» или хочет развернуть код, который работает автономно через Sapiom. НЕ используйте для разовых вызовов API или развертывания существующих скриптов — вместо этого используйте sapiom-deploy.
