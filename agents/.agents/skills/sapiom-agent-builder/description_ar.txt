قم ببناء ونشر وكلاء مستقلين - نصوص برمجية تعمل وفقًا لجدول زمني، واستدعاء LLMs، واستخراج الويب، وتخزين البيانات، والتصرف دون تدخل المستخدم. يُستخدم عندما يقول المستخدم "قم ببناء وكيل لي"، أو "إنشاء روبوت"، أو "أتمتة X وفقًا لجدول زمني"، أو "مراقبة هذه الصفحة"، أو "تلخيص X كل يوم"، أو "تنبيهي عندما يتغير Y"، أو يريد نشر تعليمات برمجية تعمل بشكل مستقل عبر Sapiom. لا تستخدمه لاستدعاءات واجهة برمجة التطبيقات (API) لمرة واحدة أو نشر البرامج النصية الموجودة - استخدم sapiom-deploy بدلاً من ذلك.
