自律エージェントを構築して展開します。これは、スケジュールに従って実行し、LLM を呼び出し、Web をスクレイピングし、データを保存し、ユーザーの介入なしに動作するスクリプトです。ユーザーが「エージェントを構築して」、「ボットを作成」、「スケジュールに従って X を自動化」、「このページを監視」、「X を毎日要約」、「Y が変更されたときに通知して」といった場合、または Sapiom 経由で自律的に実行されるコードをデプロイしたい場合に使用します。 1 回限りの API 呼び出しや既存のスクリプトのデプロイには使用しないでください。代わりに sapiom-deploy を使用してください。
