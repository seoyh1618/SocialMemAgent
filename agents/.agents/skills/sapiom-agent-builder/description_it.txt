Costruisci e distribuisci agenti autonomi: script che vengono eseguiti in base a una pianificazione, chiamano LLM, esplorano il Web, archiviano dati e agiscono senza l'intervento dell'utente. Da utilizzare quando l'utente dice "creami un agente", "crea un bot", "automatizza X in base a una pianificazione", "monitora questa pagina", "riepiloga X ogni giorno", "avvisa quando Y cambia" o desidera distribuire codice che viene eseguito in modo autonomo tramite Sapiom. NON utilizzare per chiamate API una tantum o per la distribuzione di script esistenti: utilizzare invece sapiom-deploy.
