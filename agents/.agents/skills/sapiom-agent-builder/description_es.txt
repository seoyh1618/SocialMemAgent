Cree e implemente agentes autónomos: scripts que se ejecutan según una programación, llaman a LLM, rastrean la web, almacenan datos y actúan sin la intervención del usuario. Úselo cuando el usuario diga "constrúyame un agente", "crea un bot", "automatice X según un cronograma", "supervise esta página", "resuma X todos los días", "aviseme cuando Y cambie" o quiera implementar código que se ejecute de forma autónoma a través de Sapiom. NO lo use para llamadas API únicas ni para implementar scripts existentes; use sapiom-deploy en su lugar.
