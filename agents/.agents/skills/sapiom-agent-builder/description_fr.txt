Créez et déployez des agents autonomes : des scripts qui s'exécutent selon un calendrier, appellent des LLM, explorent le Web, stockent des données et agissent sans intervention de l'utilisateur. À utiliser lorsque l'utilisateur dit « créer un agent », « créer un bot », « automatiser X selon un planning », « surveiller cette page », « résumer X tous les jours », « m'alerter lorsque Y change » ou souhaite déployer du code qui s'exécute de manière autonome via Sapiom. Ne PAS utiliser pour des appels d'API ponctuels ou pour déployer des scripts existants - utilisez plutôt sapiom-deploy.
