Создавайте передовые приложения TypeScript на Cloudflare Workers. Охватывает Workers API, платформу Hono, хранилище KV/D1/R2, устойчивые объекты, очереди и шаблоны тестирования. Используйте при создании бессерверных рабочих процессов, пограничных функций или сервисов, развернутых с помощью Cloudflare.
