Дает агенту возможность отправлять, получать, искать электронную почту и управлять ею непосредственно с терминала или через локальный HTTP API. Используйте этот навык, когда агенту необходимо выполнять задачи электронной почты: отправлять сообщения, читать входящие, отвечать, пересылать, управлять контактами, организовывать с помощью тегов/папок/фильтров, планировать фоновую синхронизацию, настраивать веб-перехватчики для новых событий электронной почты или автоматизировать рабочие процессы электронной почты. Поддерживает структурированный вывод (--format json/markdown/html), выбор полей (--fields), стандартизированные коды завершения и локальный REST API с документами OpenAPI. Работает с поставщиками IMAP/SMTP, включая Gmail, Outlook, QQ Mail и другими. Активируется по ключевым словам: отправка электронной почты, проверка почтового ящика, ответ, пересылка, автоматизация электронной почты, контакты, шаблон электронной почты, уведомления, веб-перехватчик, http API, формат json, выбор поля, подача, openapi.
