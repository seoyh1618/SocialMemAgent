يمنح الوكيل القدرة على إرسال رسائل البريد الإلكتروني واستلامها والبحث فيها وإدارتها مباشرةً من الجهاز الطرفي أو عبر واجهة برمجة تطبيقات HTTP المحلية. استخدم هذه المهارة عندما يحتاج الوكيل إلى التعامل مع مهام البريد الإلكتروني: إرسال الرسائل، أو قراءة البريد الوارد، أو الرد، أو إعادة التوجيه، أو إدارة جهات الاتصال، أو التنظيم باستخدام العلامات/المجلدات/المرشحات، أو جدولة مزامنة الخلفية، أو إعداد خطافات الويب لأحداث البريد الإلكتروني الجديدة، أو أتمتة سير عمل البريد الإلكتروني. يدعم الإخراج المنظم (--format json/markdown/html)، واختيار الحقل (--fields)، ورموز الخروج الموحدة، وواجهة برمجة تطبيقات REST المحلية مع مستندات OpenAPI. يعمل مع موفري IMAP/SMTP بما في ذلك Gmail وOutlook وQQ Mail وغيرهم. يتم التنشيط عند الكلمات الرئيسية: إرسال بريد إلكتروني، التحقق من البريد الوارد، الرد، إعادة التوجيه، أتمتة البريد الإلكتروني، جهات الاتصال، قالب البريد الإلكتروني، الإشعارات، خطاف الويب، http api، تنسيق json، اختيار الحقل، الخدمة، openapi.
