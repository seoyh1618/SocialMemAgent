Le brinda al agente la capacidad de enviar, recibir, buscar y administrar correos electrónicos directamente desde la terminal o mediante una API HTTP local. Utilice esta habilidad cuando el agente necesite manejar tareas de correo electrónico: enviar mensajes, leer la bandeja de entrada, responder, reenviar, administrar contactos, organizar con etiquetas/carpetas/filtros, programar sincronización en segundo plano, configurar webhooks para nuevos eventos de correo electrónico o automatizar flujos de trabajo de correo electrónico. Admite salida estructurada (--format json/markdown/html), selección de campos (--fields), códigos de salida estandarizados y una API REST local con documentos OpenAPI. Funciona con proveedores IMAP/SMTP, incluidos Gmail, Outlook, QQ Mail y otros. Se activa con palabras clave: enviar correo electrónico, revisar la bandeja de entrada, responder, reenviar, automatización de correo electrónico, contactos, plantilla de correo electrónico, notificaciones, webhook, http api, formato json, selección de campo, servir, openapi.
