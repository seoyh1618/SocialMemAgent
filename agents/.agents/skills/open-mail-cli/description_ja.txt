エージェントは、端末から直接、またはローカル HTTP API 経由で電子メールを送信、受信、検索、管理できるようになります。このスキルは、エージェントがメッセージの送信、受信トレイの読み取り、返信、転送、連絡先の管理、タグ/フォルダー/フィルターによる整理、バックグラウンド同期のスケジュール、新しい電子メール イベントの Webhook の設定、電子メール ワークフローの自動化などの電子メール タスクを処理する必要がある場合に使用します。構造化出力 (--format json/markdown/html)、フィールド選択 (--fields)、標準化された終了コード、および OpenAPI ドキュメントを使用したローカル REST API をサポートします。 Gmail、Outlook、QQ Mail などの IMAP/SMTP プロバイダーと連携します。キーワードでアクティブ化します: 電子メールの送信、受信箱のチェック、返信、転送、電子メールの自動化、連絡先、電子メール テンプレート、通知、Webhook、http API、フォーマット JSON、フィールド選択、配信、openapi。
