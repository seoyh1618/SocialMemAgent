使代理程式能夠直接從終端或透過本機 HTTP API 發送、接收、搜尋和管理電子郵件。當客服人員需要處理電子郵件任務時使用此技能：發送訊息、閱讀收件匣、回覆、轉發、管理聯絡人、使用標籤/資料夾/過濾器進行組織、安排後台同步、為新電子郵件事件設定 Webhooks 或自動化電子郵件工作流程。支援結構化輸出（--format json/markdown/html）、欄位選擇（--fields）、標準化退出程式碼以及帶有 OpenAPI 文件的本機 REST API。可與 IMAP/SMTP 供應商搭配使用，包括 Gmail、Outlook、QQ 郵件等。啟動關鍵字：發送電子郵件、檢查收件匣、回覆、轉寄、電子郵件自動化、聯絡人、電子郵件範本、通知、webhook、http api、json 格式、欄位選擇、服務、openapi。
