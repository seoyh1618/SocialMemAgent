Donne à l'agent la possibilité d'envoyer, de recevoir, de rechercher et de gérer des e-mails directement depuis le terminal ou via une API HTTP locale. Utilisez cette compétence lorsque l'agent doit gérer des tâches de courrier électronique : envoyer des messages, lire la boîte de réception, répondre, transférer, gérer les contacts, organiser avec des balises/dossiers/filtres, planifier la synchronisation en arrière-plan, configurer des webhooks pour de nouveaux événements de courrier électronique ou automatiser les flux de travail de courrier électronique. Prend en charge la sortie structurée (--format json/markdown/html), la sélection de champs (--fields), les codes de sortie standardisés et une API REST locale avec la documentation OpenAPI. Fonctionne avec les fournisseurs IMAP/SMTP, notamment Gmail, Outlook, QQ Mail et autres. S'active sur les mots-clés : envoyer un e-mail, vérifier la boîte de réception, répondre, transférer, automatisation des e-mails, contacts, modèle d'e-mail, notifications, webhook, API http, format json, sélection de champ, servir, openapi.
