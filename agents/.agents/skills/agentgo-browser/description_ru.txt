Автоматизирует взаимодействие с браузером с помощью распределенного облачного кластера браузеров AgentGo через playwright@1.51.0. Используйте, когда пользователю необходимо перемещаться по веб-сайтам, взаимодействовать с веб-страницами, заполнять формы, делать снимки экрана, тестировать веб-приложения или извлекать информацию — работает в удаленных облачных браузерах AgentGo вместо локального браузера.
