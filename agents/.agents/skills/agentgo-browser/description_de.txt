Automatisiert Browser-Interaktionen mithilfe des verteilten Cloud-Browser-Clusters von AgentGo über playwright@1.51.0. Wird verwendet, wenn der Benutzer durch Websites navigieren, mit Webseiten interagieren, Formulare ausfüllen, Screenshots machen, Webanwendungen testen oder Informationen extrahieren muss – läuft auf den Remote-Cloud-Browsern von AgentGo statt auf einem lokalen Browser.
