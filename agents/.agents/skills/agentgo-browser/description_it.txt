Automatizza le interazioni del browser utilizzando il cluster di browser cloud distribuito di AgentGo tramite playwright@1.51.0. Da utilizzare quando l'utente ha bisogno di navigare nei siti web, interagire con le pagine web, compilare moduli, acquisire screenshot, testare applicazioni web o estrarre informazioni, in esecuzione sui browser cloud remoti di AgentGo invece che su un browser locale.
