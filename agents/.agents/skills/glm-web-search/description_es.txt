Utilice el MCP de búsqueda en línea de GLM para realizar búsquedas web. Condiciones de activación: (1) El usuario solicita una búsqueda en línea, una búsqueda web o una búsqueda de información; (2) Es necesario consultar las últimas noticias, información o datos; (3) Se utiliza la función web_search de GLM.
