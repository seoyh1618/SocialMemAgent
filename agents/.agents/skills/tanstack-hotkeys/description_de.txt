Anleitung zum Implementieren von Tastaturkürzeln in React mit @tanstack/react-hotkeys. Verwenden Sie diese Funktion, wenn Sie Hotkey-/Shortcut-Funktionen erstellen, Tastaturkürzel registrieren, Tastenfolgen verwalten, benutzerdefinierte Shortcuts aufzeichnen, gehaltene Tasten verfolgen oder Hotkeys für die Anzeige in React-Anwendungen formatieren.
