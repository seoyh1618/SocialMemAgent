Guía para implementar atajos de teclado en React usando @tanstack/react-hotkeys. Úselo al crear funciones de teclas de acceso rápido/atajos, registrar atajos de teclado, manejar secuencias de teclas, grabar atajos personalizados, rastrear teclas retenidas o formatear teclas de acceso rápido para mostrarlas en aplicaciones React.
