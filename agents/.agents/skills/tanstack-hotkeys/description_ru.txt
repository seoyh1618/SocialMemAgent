Руководство по реализации сочетаний клавиш в React с помощью @tanstack/react-hotkeys. Используйте при создании функций горячих клавиш/сочетаний, регистрации сочетаний клавиш, обработке последовательностей клавиш, записи пользовательских сочетаний клавиш, отслеживании удерживаемых клавиш или форматировании горячих клавиш для отображения в приложениях React.
