دليل لتنفيذ اختصارات لوحة المفاتيح في React باستخدام @tanstack/react-hotkeys. يُستخدم عند إنشاء ميزات مفتاح التشغيل السريع/الاختصار، أو تسجيل اختصارات لوحة المفاتيح، أو التعامل مع تسلسلات المفاتيح، أو تسجيل الاختصارات المخصصة، أو تتبع المفاتيح المحفوظة، أو تنسيق مفاتيح التشغيل السريع للعرض في تطبيقات React.
