Genera file YAML di flashcard supportati dalla scienza dell'apprendimento dal contenuto della lezione. Da utilizzare quando: (1) un file .md di una lezione necessita della generazione di flashcard, (2) una directory di capitolo necessita di flashcard per tutte le lezioni, (3) l'utente dice "genera flashcard", "crea flashcard", "crea carte per" o fa riferimento a /genera-flashcard. Produce file .flashcards.yaml adiacenti ai file .md delle lezioni, utilizzati dal plugin comment-flashcards e renderizzati dai componenti Flashcards.
