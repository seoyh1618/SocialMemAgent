Generieren Sie lernwissenschaftlich fundierte Lernkarten-YAML-Dateien aus Unterrichtsinhalten. Verwenden Sie dies, wenn: (1) eine .md-Lektionsdatei Lernkarten generieren muss, (2) ein Kapitelverzeichnis Lernkarten für alle Lektionen benötigt, (3) der Benutzer „Lernkarten generieren“, „Lernkarten erstellen“, „Karten erstellen für“ sagt oder auf /generate-flashcards verweist. Erstellt .flashcards.yaml-Dateien neben den .md-Lektionsdateien, die vom Remark-Flashcards-Plugin verarbeitet und von Flashcards-Komponenten gerendert werden.
