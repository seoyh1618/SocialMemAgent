Создавайте YAML-файлы обучающих и научных карточек на основе содержания урока. Используйте, когда: (1) файл урока .md требует создания карточек, (2) для каталога глав нужны карточки для всех уроков, (3) пользователь говорит «создать карточки», «создать карточки», «создать карточки для» или ссылается на /generate-flashcards. Создает файлы .flashcards.yaml рядом с файлами уроков .md, используемые плагином примечаний-flashcards и визуализируемые компонентами Flashcards.
