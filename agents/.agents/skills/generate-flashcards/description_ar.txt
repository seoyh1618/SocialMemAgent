قم بإنشاء ملفات YAML للبطاقات التعليمية المدعومة بالعلم من محتوى الدرس. يُستخدم عندما: (1) يحتاج ملف الدرس .md إلى إنشاء بطاقات تعليمية، (2) يحتاج دليل الفصل إلى بطاقات تعليمية لجميع الدروس، (3) يقول المستخدم "إنشاء بطاقات تعليمية"، أو "إنشاء بطاقات تعليمية"، أو "إنشاء بطاقات لـ"، أو مراجع / إنشاء بطاقات تعليمية. يُنتج ملفات .flashcards.yaml مجاورة لملفات الدرس .md، والتي يستهلكها البرنامج الإضافي لـ comments-flashcards ويتم تقديمها بواسطة مكونات Flashcards.
