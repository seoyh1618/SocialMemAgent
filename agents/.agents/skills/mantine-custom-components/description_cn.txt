构建与 Mantine 主题、样式 API 和核心功能集成的自定义组件。在以下情况下使用此技能：(1) 使用factory()、polymorphicFactory() 或 genericFactory() 创建新组件，(2) 添加样式 API 支持（classNames、styles、vars、unstyled），(3) 通过 createVarsResolver 实现 CSS 变量，(4) 使用子组件和共享上下文构建复合组件，(5) 通过 Component.extend() 向 MantineProvider 注册组件，或 (6) 任何涉及 Factory 的任务， @mantine/core 中的 useProps、useStyles、BoxProps、StylesApiProps 或 ElementProps。
