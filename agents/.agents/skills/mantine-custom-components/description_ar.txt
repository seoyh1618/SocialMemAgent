أنشئ مكونات مخصصة تتكامل مع سمات Mantine وStyles API والميزات الأساسية. استخدم هذه المهارة عندما: (1) إنشاء مكون جديد باستخدام Factory()، أو polymorphicFactory()، أو genericFactory()، (2) إضافة دعم Styles API (classNames، Styles، vars، unstyled)، (3) تنفيذ متغيرات CSS عبر createVarsResolver، (4) إنشاء مكونات مركبة مع مكونات فرعية وسياق مشترك، (5) تسجيل مكون في MantineProvider عبر Component.extend()، أو (6) أي مهمة تتضمن Factory أو useProps أو useStyles أو BoxProps أو StylesApiProps أو ElementProps في @mantine/core.
