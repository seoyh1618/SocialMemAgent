Crea componenti personalizzati che si integrano con i temi, l'API Styles e le funzionalità principali di Mantine. Utilizzare questa abilità quando: (1) si crea un nuovo componente utilizzando factory(), polymorphicFactory() o genericFactory(), (2) si aggiunge il supporto API Styles (classNames, stili, vars, unstyled), (3) si implementano variabili CSS tramite createVarsResolver, (4) si creano componenti composti con sottocomponenti e contesto condiviso, (5) si registra un componente con MantineProvider tramite Component.extend() o (6) qualsiasi attività che coinvolge Factory, useProps, useStyles, BoxProps, StylesApiProps o ElementProps in @mantine/core.
