建立與 Mantine 主題、樣式 API 和核心功能整合的自訂元件。在下列情況下使用此技能：(1) 使用factory()、polymorphicFactory() 或 genericFactory() 建立新元件，(2) 新增樣式 API 支援（classNames、styles、vars、unstyled），(3) 透過 createVarsResolver 實作 CSS 變數，(4) 使用子元件和共享上下文來建構複合組件，500000m.註冊元件，或 (6) 任何涉及 Factory 的任務， @mantine/core 中的 useProps、useStyles、BoxProps、StylesApiProps 或 ElementProps。
