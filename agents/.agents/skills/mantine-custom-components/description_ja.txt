Mantine のテーマ、スタイル API、およびコア機能と統合するカスタム コンポーネントを構築します。このスキルは、(1) Factory()、polymorphicFactory()、または genericFactory() を使用して新しいコンポーネントを作成する場合、(2) スタイル API サポート (classNames、styles、vars、unstyled) を追加する場合、(3) createVarsResolver を介して CSS 変数を実装する場合、(4) サブコンポーネントと共有コンテキストを含む複合コンポーネントを構築する場合、(5) Component.extend() を介して MantineProvider にコンポーネントを登録する場合、または(6) @mantine/core 内の Factory、useProps、useStyles、BoxProps、StylesApiProps、または ElementProps に関連するタスク。
