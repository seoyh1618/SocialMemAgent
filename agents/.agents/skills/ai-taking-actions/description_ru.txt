Создайте искусственный интеллект, который выполняет действия, вызывает API и делает все автономно. Используйте, когда вам нужен искусственный интеллект для вызова API, использования инструментов, выполнения вычислений, поиска в Интернете и действий на основе результатов, взаимодействия с базами данных или выполнения многоэтапных задач. Работает на базе агентов DSPy (ReAct, CodeAct).
