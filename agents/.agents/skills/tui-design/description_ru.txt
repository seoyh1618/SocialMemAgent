Этот навык следует использовать при проектировании пользовательских интерфейсов терминала, создании макетов TUI, выборе цветовых схем TUI, реализации навигации с помощью клавиатуры, создании панелей мониторинга терминала или работе с любой платформой TUI. Активируется при упоминании дизайна TUI, пользовательского интерфейса терминала, макета Ratatui, компонентов Ink, текстовых виджетов, представлений Bubbletea, цветовой палитры терминала, дизайна привязки клавиш, макета панели, разделенных панелей, информационной панели терминала, символов рисования прямоугольников, спарклайнов, индикаторов выполнения, модальных диалогов, управления фокусом или доступности терминала.
