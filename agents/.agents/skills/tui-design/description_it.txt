Questa competenza dovrebbe essere utilizzata durante la progettazione delle interfacce utente del terminale, la creazione di layout TUI, la scelta degli schemi di colori TUI, l'implementazione della navigazione tramite tastiera, la creazione di dashboard del terminale o l'utilizzo di qualsiasi framework TUI. Si attiva su menzioni di progettazione TUI, interfaccia utente del terminale, layout Ratatui, componenti input penna, widget testuali, visualizzazioni Bubbletea, tavolozza dei colori del terminale, progettazione delle combinazioni di tasti, layout del pannello, riquadri divisi, dashboard del terminale, caratteri di disegno di riquadri, sparkline, barre di avanzamento, finestre di dialogo modali, gestione del focus o accessibilità del terminale.
