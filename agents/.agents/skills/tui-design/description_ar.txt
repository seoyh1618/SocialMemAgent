يجب استخدام هذه المهارة عند تصميم واجهات المستخدم الطرفية، أو إنشاء تخطيطات TUI، أو اختيار أنظمة ألوان TUI، أو تنفيذ التنقل عبر لوحة المفاتيح، أو إنشاء لوحات معلومات طرفية، أو العمل مع أي إطار عمل TUI. يتم التنشيط عند الإشارة إلى تصميم TUI، وواجهة المستخدم الطرفية، وتخطيط Ratatui، ومكونات الحبر، والأدوات النصية، وطرق عرض Bubbletea، ولوحة الألوان الطرفية، وتصميم ربط المفاتيح، وتخطيط اللوحة، والأجزاء المقسمة، ولوحة المعلومات الطرفية، وأحرف رسم المربع، وخطوط المؤشرات، وأشرطة التقدم، ومربعات الحوار المشروطة، وإدارة التركيز، أو إمكانية الوصول إلى الوحدة الطرفية.
