Создавайте конечные точки OpenAPI/REST из маршрутизаторов tRPC. Используйте, если у вас есть API tRPC, но вам необходимо предоставить конечные точки REST для сторонних интеграций, мобильных приложений или общедоступной документации API. Обеспечивает автоматическое создание пользовательского интерфейса Swagger и спецификаций OpenAPI. Выберите этот вариант, если вам нужны типобезопасные внутренние API (tRPC) с резервным вариантом REST для внешних потребителей.
