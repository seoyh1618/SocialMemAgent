قم بإنشاء نقاط نهاية OpenAPI/REST من أجهزة توجيه tRPC. يُستخدم عندما يكون لديك واجهة برمجة تطبيقات tRPC ولكنك تحتاج إلى الكشف عن نقاط نهاية REST لعمليات تكامل الجهات الخارجية أو تطبيقات الهاتف المحمول أو وثائق واجهة برمجة التطبيقات العامة. يوفر إنشاءًا تلقائيًا لمواصفات Swagger UI وOpenAPI. اختر هذا عندما تريد واجهات برمجة تطبيقات داخلية آمنة للنوع (tRPC) مع احتياطي REST للمستهلكين الخارجيين.
