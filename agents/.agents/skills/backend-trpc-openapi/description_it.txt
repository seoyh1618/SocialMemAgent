Genera endpoint OpenAPI/REST dai router tRPC. Utilizzalo quando disponi di un'API tRPC ma devi esporre endpoint REST per integrazioni di terze parti, app mobili o documentazione API pubblica. Fornisce l'interfaccia utente Swagger automatica e la generazione di specifiche OpenAPI. Scegli questa opzione se desideri API interne indipendenti dai tipi (tRPC) con fallback REST per consumatori esterni.
