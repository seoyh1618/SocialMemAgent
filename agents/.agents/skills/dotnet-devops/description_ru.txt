Настраивает конвейеры .NET CI/CD (действия GitHub с setup-dotnet, кэш NuGet, многократно используемые рабочие процессы; Azure DevOps с DotNetCoreCLI, шаблоны, многоэтапные), контейнеризацию (многоэтапные файлы Dockerfiles, Compose, без root), упаковку (авторская разработка NuGet, генераторы исходного кода, подпись MSIX), управление выпусками (NBGV, SemVer, журналы изменений, выпуски GitHub), и наблюдаемость (OpenTelemetry, проверки работоспособности, структурированное ведение журнала, PII). Охватывает 18 тематических областей. Не используйте для шаблонов реализации API или пользовательского интерфейса уровня приложения.
