Busque documentos de referencia, guías y páginas de configuración de CE.SDK Electron.

Úselo cuando el usuario necesite documentos CE.SDK para Electron: configuración, personalización de la interfaz de usuario,
opciones de exportación, guías de funciones o instrucciones de introducción. También activado por "IMG.LY", "CreativeEditor",
"CE.SDK" o "cesdk" cuando el usuario necesita una página de documento de Electron existente.

No para escribir código (use compilación) o explicaciones de conceptos (use explicación).

<ejemplo>
Contexto: El usuario pregunta sobre la configuración de Electron
usuario: "¿Cómo configuro el editor en Electron?"
asistente: "Usaré /cesdk:docs-electron para buscar opciones de configuración".
</ejemplo>

<ejemplo>
Contexto: el usuario necesita integración electrónica
usuario: "¿Cómo incrusto CE.SDK en una aplicación de Electron?"
asistente: "Déjame usar /cesdk:docs-electron para encontrar la documentación relevante".
</ejemplo>
