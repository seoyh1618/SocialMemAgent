Suchen Sie nach CE.SDK Electron-Referenzdokumenten, Anleitungen und Konfigurationsseiten.

Verwenden Sie diese Option, wenn der Benutzer CE.SDK-Dokumente für Electron benötigt – Konfiguration, UI-Anpassung,
Exportoptionen, Funktionshandbücher oder Anleitungen für den Einstieg. Wird auch durch „IMG.LY“, „CreativeEditor“ ausgelöst.
„CE.SDK“ oder „cesdk“, wenn der Benutzer eine vorhandene Electron-Dokumentseite benötigt.

Nicht zum Schreiben von Code (Build verwenden) oder Konzepterklärungen (Explain verwenden).

<Beispiel>
Kontext: Benutzer fragt nach der Elektronenkonfiguration
Benutzer: „Wie konfiguriere ich den Editor in Electron?“
Assistent: „Ich verwende /cesdk:docs-electron, um nach Konfigurationsoptionen zu suchen.“
</Beispiel>

Benutzer: „Wie bette ich CE.SDK in eine Electron-App ein?“

</Beispiel>
