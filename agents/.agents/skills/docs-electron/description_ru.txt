Найдите справочную документацию, руководства и страницы конфигурации CE.SDK Electron.

Используйте, когда пользователю нужна документация CE.SDK для Electron — конфигурация, настройка пользовательского интерфейса,
параметры экспорта, руководства по функциям или инструкции по началу работы. Также запускается "IMG.LY", "CreativeEditor",
«CE.SDK» или «cesdk», если пользователю нужна существующая страница документа Electron.

Не для написания кода (используйте сборку) или объяснения концепций (используйте объяснение).

<пример>
Контекст: Пользователь спрашивает о конфигурации Electron.
пользователь: «Как мне настроить редактор в Electron?»
помощник: «Я буду использовать /cesdk:docs-electron для поиска параметров конфигурации».
</пример>

<пример>
Контекст: пользователю нужна интеграция с Electron
пользователь: «Как мне встроить CE.SDK в приложение Electron?»
помощник: «Позвольте мне использовать /cesdk:docs-electron, чтобы найти соответствующую документацию».
</пример>
