Aggiorna e perfeziona in modo completo il design dell'interfaccia utente di un sito Web esistente, migliorando layout, tipografia, spaziatura, profondità, gerarchia visiva e perfezionamento dei componenti, preservando la tavolozza dei colori esistente del sito. Aggiunge anche animazioni fluide e sfalsate di sfocatura+rivelazione di scorrimento verso l'alto utilizzando Framer Motion (o CSS se HTML semplice) attivate quando gli elementi entrano nel viewport. Utilizza questa abilità ogni volta che l'utente desidera migliorare, modernizzare, perfezionare o animare un sito Web esistente o una pagina React/HTML. Si attiva anche quando l'utente menziona solo "rendilo migliore", "animalo", "migliora il design", "aggiorna l'interfaccia utente", "rendilo più moderno" o "aggiungi animazioni di scorrimento": questa abilità gestisce tutti questi, non solo l'animazione.
