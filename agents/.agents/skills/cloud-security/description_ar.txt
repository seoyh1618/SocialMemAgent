مهارة تقييم أمان السحابة المتعددة لـ AWS وAzure وGCP. يجب استخدام هذه المهارة عند إجراء عمليات تدقيق أمان السحابة، والمسح بحثًا عن التكوينات الخاطئة، واختبار سياسات IAM، وتدقيق أذونات التخزين، وتحديد مسارات تصعيد الامتيازات. يتم تشغيلها عند طلبات تدقيق أمان السحابة، أو فحص AWS/Azure/GCP، أو التحقق من التكوينات الخاطئة للسحابة، أو إجراء اختبار اختراق السحابة.
