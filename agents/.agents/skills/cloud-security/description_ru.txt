Навыки оценки мультиоблачной безопасности для AWS, Azure и GCP. Этот навык следует использовать при проведении аудита облачной безопасности, сканировании на предмет неправильных конфигураций, тестировании политик IAM, аудите разрешений на хранение и определении путей повышения привилегий. Запускается по запросам на аудит безопасности облака, сканирование AWS/Azure/GCP, проверку неправильных конфигураций облака или выполнение тестирования на проникновение в облако.
