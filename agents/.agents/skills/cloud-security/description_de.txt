Fähigkeit zur Multi-Cloud-Sicherheitsbewertung für AWS, Azure und GCP. Diese Fähigkeit sollte bei der Durchführung von Cloud-Sicherheitsüberprüfungen, der Suche nach Fehlkonfigurationen, dem Testen von IAM-Richtlinien, der Prüfung von Speicherberechtigungen und der Identifizierung von Eskalationspfaden für Berechtigungen eingesetzt werden. Wird bei Anfragen zur Prüfung der Cloud-Sicherheit, zum Scannen von AWS/Azure/GCP, zur Überprüfung von Cloud-Fehlkonfigurationen oder zur Durchführung von Cloud-Penetrationstests ausgelöst.
