Habilidad de evaluación de seguridad de múltiples nubes para AWS, Azure y GCP. Esta habilidad debe usarse al realizar auditorías de seguridad en la nube, buscar configuraciones incorrectas, probar políticas de IAM, auditar permisos de almacenamiento e identificar rutas de escalada de privilegios. Activa solicitudes para auditar la seguridad de la nube, escanear AWS/Azure/GCP, verificar configuraciones incorrectas de la nube o realizar pruebas de penetración en la nube.
