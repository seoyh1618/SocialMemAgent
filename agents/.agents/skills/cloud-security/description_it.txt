Competenza di valutazione della sicurezza multi-cloud per AWS, Azure e GCP. Questa competenza dovrebbe essere utilizzata durante l'esecuzione di controlli di sicurezza del cloud, la scansione di configurazioni errate, il test delle policy IAM, il controllo delle autorizzazioni di archiviazione e l'identificazione dei percorsi di escalation dei privilegi. Si attiva su richieste di controllo della sicurezza del cloud, scansione di AWS/Azure/GCP, controllo di configurazioni errate del cloud o esecuzione di test di penetrazione del cloud.
