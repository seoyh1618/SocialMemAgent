Используйте Rabbit API для создания видео с помощью ИИ. Поддерживает такие модели, как Veo, Sora, Kling, Seedance и т. д., с режимами одиночного видео и длинного видео (многосегментная композиция). Используйте, когда пользователь запрашивает создание видео, создание видео или ему требуется серверная часть для создания видео.
