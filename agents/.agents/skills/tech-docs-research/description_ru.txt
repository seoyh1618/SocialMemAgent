Автоматизированный процесс исследования технической документации с помощью Firecrawl. Используйте, когда пользователь просит изучить или узнать технические темы из источников документации. Запускается по таким запросам, как «Изучите документацию по [технологии]», «Найдите документы по [теме]», «Узнайте, как [технология] работает из документации», «Изучите [техническую тему] в официальной документации» или «Что документация говорит о [функции]». Этот навык организует многоэтапный исследовательский процесс, позволяющий обнаружить соответствующие URL-адреса документации посредством сопоставления, параллельного сбора контента и создания структурированных сводок.
