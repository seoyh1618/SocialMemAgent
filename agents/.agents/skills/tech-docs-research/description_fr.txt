Flux de travail automatisé de recherche de documentation technique à l'aide de Firecrawl. À utiliser lorsque l'utilisateur demande à rechercher, enquêter ou en savoir plus sur des sujets techniques à partir de sources de documentation. Se déclenche sur des requêtes telles que "Rechercher la documentation sur [la technologie]", "Rechercher des documents sur [le sujet]", "Découvrir le fonctionnement de la [technologie] à partir de la documentation", "Enquêter sur [le sujet technique] dans la documentation officielle" ou "Que dit la documentation sur [la fonctionnalité]". Cette compétence orchestre un processus de recherche en plusieurs étapes pour découvrir les URL de documentation pertinentes via le mappage, le scraping du contenu en parallèle et la génération de résumés structurés.
