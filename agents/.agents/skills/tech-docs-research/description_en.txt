Automated technical documentation research workflow using Firecrawl. Use when the user asks to research, investigate, or learn about technical topics from documentation sources. Triggers on requests like "Research [technology] documentation", "Find docs about [topic]", "Learn how [technology] works from docs", "Investigate [technical topic] in official docs", or "What does documentation say about [feature]". This skill orchestrates a multi-step research process discovering relevant documentation URLs via mapping, scraping content in parallel, and generating structured summaries.
