Ghostty: emulador de terminal multiplataforma, rápido y rico en funciones, con aceleración de GPU y interfaz de usuario nativa.
Úselo al configurar el terminal Ghostty, personalizar combinaciones de teclas, temas, integración de shell o trabajar con secuencias de escape y códigos de control VT/ANSI.
