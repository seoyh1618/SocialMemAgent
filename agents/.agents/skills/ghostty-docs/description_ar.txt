Ghostty - محاكي طرفي سريع وغني بالميزات ومتعدد المنصات مع تسريع وحدة معالجة الرسومات وواجهة مستخدم أصلية.
يُستخدم عند تكوين محطة Ghostty، أو تخصيص روابط المفاتيح، أو السمات، أو تكامل الصدفة، أو العمل مع تسلسلات الهروب وأكواد التحكم VT/ANSI.
الكلمات الرئيسية: Ghostty، المحاكي الطرفي، تسريع وحدة معالجة الرسومات، التكوين، روابط المفاتيح، السمات، تكامل الصدفة، تسلسلات الهروب، تسلسلات vt، CSI، OSC، تسلسلات التحكم، المؤشر، الألوان، النظام الأساسي المشترك.
