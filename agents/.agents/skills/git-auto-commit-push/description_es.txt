Git prepara, confirma y empuja automáticamente. Utilice esta habilidad cuando el usuario solicite confirmar código, enviar código, confirmar, enviar, confirmar y enviar, auto-commit, stage-commit-push o cualquier operación de git commit/push. También se aplica cuando el usuario dice "comprometerlo", "enviar el código", "ayúdame a confirmar", "enviarlo hacia arriba", "comprometer solo", "solo confirmar" o expresiones coloquiales similares.
