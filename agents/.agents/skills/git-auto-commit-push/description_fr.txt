Git prépare, valide et pousse automatiquement. Utilisez cette compétence lorsque l'utilisateur demande de valider du code, de pousser du code, de valider, de pousser, de valider et de pousser, de valider automatiquement, de stage-commit-push ou toute opération git commit/push. S'applique également lorsque l'utilisateur dit "commit it", "push the code", "help me commit", "push it up", "commit only", "just commit" ou des expressions familières similaires.
