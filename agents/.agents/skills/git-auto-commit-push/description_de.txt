Git führt automatisch Staging, Commits und Pushs durch. Verwenden Sie diese Fähigkeit, wenn der Benutzer einen Commit-Code, einen Push-Code, einen Commit, einen Push, einen Commit und Push, einen automatischen Commit, einen Stage-Commit-Push oder einen beliebigen Git-Commit-/Push-Vorgang anfordert. Gilt auch, wenn der Benutzer „commit it“, „push the code“, „help me commit“, „push it up“, „commit only“, „just commit“ oder ähnliche umgangssprachliche Ausdrücke sagt.
