Git автоматически размещает, фиксирует и отправляет данные. Используйте этот навык, когда пользователь запрашивает фиксацию кода, отправку кода, фиксацию, отправку, фиксацию и отправку, автоматическую фиксацию, этапную фиксацию-нажатие или любую операцию git commit/push. Также применяется, когда пользователь говорит «зафиксируйте это», «отправьте код», «помогите мне зафиксировать», «отправьте его вверх», «только зафиксируйте», «просто зафиксируйте» или подобные разговорные выражения.
