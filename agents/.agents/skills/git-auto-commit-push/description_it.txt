Git esegue stage, commit e push automaticamente. Utilizza questa abilità quando l'utente chiede di eseguire il commit del codice, il push del codice, il commit, il push, il commit e il push, il commit automatico, lo stage-commit-push o qualsiasi operazione git commit/push. Si applica anche quando l'utente dice "commit it", "push the code", "help me commit", "push it up", "commit only", "just commit" o espressioni colloquiali simili.
