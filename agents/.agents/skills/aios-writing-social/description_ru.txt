Организуйте создание контента для социальных сетей на разных платформах. Используйте, когда вас просят написать, создать или набросать контент для социальных сетей без указания платформы, или при запросе пакетов для нескольких платформ, календарей контента или планирования социальных сетей. Делегирует навыки, специфичные для платформы, для LinkedIn, Twitter/X, Instagram, Facebook и Threads. НЕ используйте для сообщений в блогах, электронных писем, рекламы или целевых страниц.
