Una habilidad integral para hacer cumplir las reglas de arquitectura, estilo, seguridad y calidad específicas del proyecto. 
Utilice esta habilidad cuando: (1) revise el código para verificar su cumplimiento, (2) genere nuevos componentes o módulos, 
(3) Refactorizar el código existente, o (4) Configurar los estándares de codificación de un nuevo proyecto.
