Une compétence complète pour appliquer les règles d’architecture, de style, de sécurité et de qualité spécifiques au projet. 
Utilisez cette compétence lorsque : (1) Vérifiez la conformité du code, (2) Générez de nouveaux composants ou modules, 
(3) Refactoriser le code existant, ou (4) Mettre en place les normes de codage d'un nouveau projet.
