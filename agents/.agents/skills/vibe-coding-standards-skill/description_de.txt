Eine umfassende Fähigkeit zur Durchsetzung projektspezifischer Architektur-, Stil-, Sicherheits- und Qualitätsregeln. 
Verwenden Sie diese Fähigkeit, wenn: (1) Code auf Konformität überprüft, (2) neue Komponenten oder Module generiert, 
(3) Refactoring des vorhandenen Codes oder (4) Einrichten der Codierungsstandards eines neuen Projekts.
