Управляйте инфраструктурой Yandex Cloud через yc CLI.
Используйте, когда пользователь запрашивает создание, настройку, управление или устранение неполадок любого
Ресурс Yandex Cloud: ВМ, диски, сети, группы безопасности, базы данных
(PostgreSQL, MySQL, ClickHouse, Redis/Valkey, MongoDB, OpenSearch, Greenplum, Kafka),
Kubernetes, бессерверные функции/контейнеры, хранилище S3, CDN, балансировщики нагрузки,
Секреты защищенного хранилища, KMS, сертификаты, DNS, реестр контейнеров, DataProc,
Передача данных, ведение журналов, журналы аудита, организации, WAF или любые другие службы YC.
Триггеры: Yandex Cloud, yc CLI, YC, Managed-postgresql, Managed-Kubernetes,
вычислительный экземпляр, бессерверная функция, сеть VPC, альб, сейф, облако Яндекса.
