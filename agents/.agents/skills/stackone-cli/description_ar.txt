قم ببناء ونشر موصلات StackOne المخصصة باستخدام واجهة سطر الأوامر (CLI) ومحرك الموصل. يُستخدم عندما يطلب المستخدم "إنشاء موصل مخصص"، أو "نشر الموصل الخاص بي"، أو "استخدام StackOne AI builder"، أو "إعداد CI/CD للموصلات"، أو "اختبار الموصل محليًا"، أو "تثبيت StackOne CLI". يغطي سير عمل تطوير الموصل بالكامل بدءًا من الحرف الأول وحتى النشر. لا تستخدمه لاستخدام الموصلات الموجودة (استخدم موصلات Stackone) أو إنشاء وكلاء الذكاء الاصطناعي (استخدم وكلاء Stackone).
