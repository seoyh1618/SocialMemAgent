Получите страницу Notion по заголовку, экспортируйте в Markdown, преобразуйте Markdown в HTML с помощью предоставленного пользователем файла CSS и создайте черновик Weixin с помощью wxcli. Используйте, когда вас просят опубликовать содержимое Notion в черновик Weixin или при перемещении страниц Notion в черновик Weixin в формате HTML.
