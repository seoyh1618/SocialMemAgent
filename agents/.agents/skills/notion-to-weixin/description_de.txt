Rufen Sie eine Notion-Seite nach Titel ab, exportieren Sie sie nach Markdown, konvertieren Sie Markdown mit einer vom Benutzer bereitgestellten CSS-Datei in HTML und erstellen Sie über wxcli einen Weixin-Entwurf. Verwenden Sie diese Option, wenn Sie aufgefordert werden, Notion-Inhalte in der Weixin-Draftbox zu veröffentlichen, oder wenn Sie Notion-Seiten als HTML in den Weixin-Entwurf verschieben.
