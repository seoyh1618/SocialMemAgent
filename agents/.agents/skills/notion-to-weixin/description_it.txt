Recupera una pagina Notion per titolo, esporta in Markdown, converti Markdown in HTML con un file CSS fornito dall'utente e crea una bozza Weixin tramite wxcli. Da utilizzare quando viene richiesto di pubblicare il contenuto di Notion nella bozza di Weixin o quando si spostano le pagine di Notion nella bozza di Weixin come HTML.
