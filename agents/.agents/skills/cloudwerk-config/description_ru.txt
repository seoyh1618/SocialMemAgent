Справочник по настройке приложений Cloudwerk. Используйте при настройке cloudwerk.config.ts, выборе средства рендеринга, настройке плагинов Vite или настройке псевдонимов путей. Запускается при выполнении задач, связанных с настройкой проекта, выбором средства рендеринга (hono-jsx или реагированием), настройкой Vite или настройкой CSS Tailwind.
