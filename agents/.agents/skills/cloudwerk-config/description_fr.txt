Référence de configuration pour les applications Cloudwerk. À utiliser lors de la configuration de cloudwerk.config.ts, du choix d'un moteur de rendu, de la configuration des plugins Vite ou de la configuration des alias de chemin. Déclencheurs sur des tâches impliquant la configuration du projet, la sélection du moteur de rendu (hono-jsx vs réagir), la configuration de Vite ou la configuration du CSS Tailwind.
