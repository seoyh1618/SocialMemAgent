Referencia de configuración para aplicaciones Cloudwerk. Úselo al configurar cloudwerk.config.ts, elegir un renderizador, configurar complementos de Vite o configurar alias de ruta. Activadores de tareas que involucran la configuración del proyecto, la selección del renderizador (hono-jsx vs react), la configuración de Vite o la configuración de Tailwind CSS.
