Initiiert eine neue Arbeitseinheit (Track) in einem vom Dirigenten verwalteten Projekt. Verwenden Sie diese Option, wenn der Benutzer eine neue Funktion starten, einen Fehler beheben oder Wartungsarbeiten durchführen möchte. Diese Fähigkeit führt den Benutzer durch die interaktive Erstellung von Spezifikationen und die detaillierte Implementierungsplanung.
