Créez un ensemble complet de compétences Sentry SDK pour n’importe quelle plateforme. À utiliser lorsqu'on vous demande de « créer une compétence SDK », « d'ajouter une nouvelle compétence de plate-forme », « d'écrire une compétence Sentry pour X » ou de créer un nouvel ensemble de compétences sentry-<platform>-sdk avec le flux de l'assistant et les fichiers de référence des fonctionnalités.
