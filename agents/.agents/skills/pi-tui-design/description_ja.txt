@mariozechner/pi-tui と @mariozechner/pi-coding-agent を使用して、pi 用に特別に細工された TUI コンポーネントを作成します。インタラクティブなターミナル UI を構築するときに使用します。カスタム コンポーネント、オーバーレイ、ダイアログ、ダッシュボード、ウィジェット、データ視覚化、アニメーション要素、ゲームのようなインターフェイス、または pi 拡張機能やカスタム ツール内のビジュアル TUI 作業などです。トリガー: 「TUI コンポーネントの構築」、「ダッシュボードの作成」、「オーバーレイの作成」、「インタラクティブ ウィジェット」、「ターミナル UI」、「カスタム コンポーネント」、「pi-tui」、または視覚的でインタラクティブなターミナル インターフェイスを作成するためのリクエスト。既存の TUI コンポーネントを美しくしたり再設計したりするときにも使用します。
