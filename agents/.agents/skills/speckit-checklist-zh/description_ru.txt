Профессиональный инструмент, который генерирует индивидуальные контрольные списки для текущих функций в зависимости от потребностей пользователя. Специально разработанный для проверки качества требований, он производит «английские модульные тесты» для проверки полноты, ясности и последовательности требований. Триггерные слова: контрольный список спецификаций, контрольный список, проверка требований, проверка качества, контрольный список, проверка требований, проверка качества, проверка спецификации.
