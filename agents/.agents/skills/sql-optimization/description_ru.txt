Универсальный помощник по оптимизации производительности SQL для комплексной настройки запросов, стратегий индексации и анализа производительности всех баз данных SQL (MySQL, PostgreSQL, SQL Server, Oracle). Обеспечивает анализ плана выполнения, оптимизацию нумерации страниц, пакетные операции и рекомендации по мониторингу производительности.
