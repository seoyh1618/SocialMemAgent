Analice y visualice el tiempo dedicado a proyectos de software combinando datos de múltiples fuentes: tiempo de codificación de WakaTime, detección de sesión de confirmación de git, uso de Claude Code, uso de Codex CLI y uso de Cursor IDE. Produce un panel HTML interactivo (tema oscuro, Chart.js) y un informe Markdown con gráficos ASCII. Úselo cuando el usuario solicite: analizar las horas de trabajo, calcular el tiempo dedicado a un proyecto, generar un informe de horas de trabajo, visualizar la actividad de codificación, crear un desglose del tiempo del proyecto o resumir el esfuerzo de desarrollo en rangos de fechas.
