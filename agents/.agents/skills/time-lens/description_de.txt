Analysieren und visualisieren Sie die für Softwareprojekte aufgewendete Zeit, indem Sie Daten aus mehreren Quellen kombinieren: WakaTime-Codierungszeit, Git-Commit-Sitzungserkennung, Claude-Code-Nutzung, Codex-CLI-Nutzung und Cursor-IDE-Nutzung. Erstellt sowohl ein interaktives HTML-Dashboard (dunkles Design, Chart.js) als auch einen Markdown-Bericht mit ASCII-Diagrammen. Verwenden Sie es, wenn der Benutzer Folgendes wünscht: Arbeitsstunden analysieren, für ein Projekt aufgewendete Zeit berechnen, einen Arbeitsstundenbericht erstellen, Codierungsaktivitäten visualisieren, eine Projektzeitaufschlüsselung erstellen oder den Entwicklungsaufwand über Datumsbereiche hinweg zusammenfassen.
