Analizza e visualizza il tempo impiegato su progetti software combinando dati provenienti da più fonti: tempo di codifica WakaTime, rilevamento sessioni git commit, utilizzo del codice Claude, utilizzo della CLI Codex e utilizzo dell'IDE Cursor. Produce sia una dashboard HTML interattiva (a tema scuro, Chart.js) sia un report Markdown con grafici ASCII. Da utilizzare quando l'utente chiede di: analizzare l'orario di lavoro, calcolare il tempo impiegato su un progetto, generare un report sull'orario di lavoro, visualizzare l'attività di codifica, creare una suddivisione temporale del progetto o riepilogare l'impegno di sviluppo in intervalli di date.
