複数のソースからのデータ (WakaTime コーディング時間、git コミット セッション検出、クロード コードの使用状況、Codex CLI の使用状況、カーソル IDE の使用状況) からのデータを組み合わせて、ソフトウェア プロジェクトに費やした時間を分析および視覚化します。インタラクティブな HTML ダッシュボード (ダークテーマ、Chart.js) と ASCII グラフを含む Markdown レポートの両方を生成します。ユーザーが次のことを行う場合に使用します: 作業時間の分析、プロジェクトに費やした時間の計算、作業時間レポートの生成、コーディング アクティビティの視覚化、プロジェクト時間の内訳の作成、または日付範囲にわたる開発作業の要約。
