قم بتحليل وتصور الوقت المستغرق في مشاريع البرامج من خلال الجمع بين البيانات من مصادر متعددة: وقت تشفير WakaTime، واكتشاف جلسة التزام git، واستخدام Claude Code، واستخدام Codex CLI، واستخدام Cursor IDE. يُنتج لوحة تحكم HTML تفاعلية (ذات سمة داكنة، Chart.js) وتقرير Markdown مع مخططات ASCII. يُستخدم عندما يطلب المستخدم: تحليل ساعات العمل، أو حساب الوقت المنقضي في مشروع، أو إنشاء تقرير ساعات العمل، أو تصور نشاط الترميز، أو إنشاء تقسيم زمني للمشروع، أو تلخيص جهود التطوير عبر النطاقات الزمنية.
