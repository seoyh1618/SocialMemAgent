Génère des fichiers de collection Bruno (.bru) à partir d'Express, Next.js, Fastify ou d'autres routes API. Crée des collections organisées avec des environnements, une authentification et une structure de dossiers pour le client API Bruno open source. À utiliser lorsque les utilisateurs demandent « générer une collection bruno », « tester l'API bruno », « créer des fichiers bru » ou « importer bruno ».
