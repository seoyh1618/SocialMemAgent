CLI für statische Swift/iOS-Analyse. Verwenden Sie „depgraph“, um herauszufinden, wer eine Funktion aufruft, was kaputt geht, wenn Sie eine Datei ändern, Aufruforte und Explosionsradius vor dem Refactoring zu verfolgen und Symbolabhängigkeiten zwischen Dateien zuzuordnen. Verwenden Sie „ask“, um die Swift/iOS/tvOS/watchOS/macOS-Dokumentation und Best Practices einzusehen.
