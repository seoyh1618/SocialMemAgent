CLI d'analyse statique Swift/iOS. Utilisez `depgraph` pour trouver qui appelle une fonction, ce qui se casse si vous modifiez un fichier, suivez les sites d'appel et le rayon d'explosion avant la refactorisation et mappez les dépendances des symboles entre les fichiers. Utilisez « ask » pour consulter la documentation et les meilleures pratiques Swift/iOS/tvOS/watchOS/macOS.
