CLI de análisis estático de Swift/iOS. Utilice `depgraph` para encontrar quién llama a una función, qué se interrumpe si cambia un archivo, rastrea los sitios de llamadas y el radio de explosión antes de refactorizar y asigna dependencias de símbolos entre archivos. Utilice "preguntar" para consultar la documentación y las mejores prácticas de Swift/iOS/tvOS/watchOS/macOS.
