إنشاء مكونات AEM وتوسيعها وتكوينها من البداية إلى النهاية. يغطي بنية عقدة المكونات (cq:Component)، ومربعات حوار Touch UI (cq:dialog with Granite UI)، ونماذج Sling، ومكتبات العملاء، وتكوين التحرير (cq:editConfig)، وقوالب المحتوى. يُستخدم عند إنشاء مكونات AEM جديدة، أو إضافة أو تعديل مربعات حوار المكونات، أو توصيل نماذج Sling، أو إعداد Clientlibs، أو توسيع المكونات الأساسية، أو تكوين سلوك التحرير، أو استكشاف أخطاء عرض المكونات وإصلاحها. يتم التنشيط أيضًا عندما يذكر المستخدم cq:Component، أو cq:dialog، أو ComponentGroup، أو عناصر واجهة مستخدم Granite UI، أو بنية مكون AEM.
