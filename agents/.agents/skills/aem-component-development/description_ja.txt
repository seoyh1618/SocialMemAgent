AEM コンポーネントをエンドツーエンドで構築、拡張、設定します。コンポーネント ノード構造 (cq:Component)、タッチ UI ダイアログ (Cq:dialog with Granite UI)、Sling モデル、クライアント ライブラリ、編集構成 (cq:editConfig)、およびコンテンツ テンプレートについて説明します。新しい AEM コンポーネントの作成、コンポーネント ダイアログの追加または変更、Sling モデルの配線、clientlib のセットアップ、コア コンポーネントの拡張、編集動作の設定、またはコンポーネント レンダリングのトラブルシューティングを行うときに使用します。ユーザーが cq:Component、cq:dialog、componentGroup、Granite UI ウィジェット、または AEM コンポーネント アーキテクチャに言及したときにもアクティブ化します。
