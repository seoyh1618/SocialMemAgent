Cree, amplíe y configure componentes de AEM de un extremo a otro. Cubre la estructura de nodos del componente (cq:Component), cuadros de diálogo de Touch UI (cq:dialog with Granite UI), modelos Sling, bibliotecas cliente, configuración de edición (cq:editConfig) y plantillas de contenido. Utilícelo al crear nuevos componentes de AEM, agregar o modificar cuadros de diálogo de componentes, conectar modelos de Sling, configurar clientlibs, ampliar componentes principales, configurar el comportamiento de edición o solucionar problemas de representación de componentes. Actívelo también cuando el usuario mencione cq:Component, cq:dialog, componenteGroup, widgets de Granite UI o arquitectura de componentes AEM.
