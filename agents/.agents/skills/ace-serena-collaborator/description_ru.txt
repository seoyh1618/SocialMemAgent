Ace-Tool сотрудничает с Serena MCP в двухэтапном рабочем процессе. Запускается, когда: (1) понимание структуры кодовой базы, (2) семантический поиск кода, (3) расположение кода на уровне символа, (4) навигация и анализ кода. Обеспечивает двухэтапное сотрудничество: семантический поиск Ace-Tool → точное местоположение Serena, подчеркивая безопасность только для чтения и взаимодополняемость инструментов.
