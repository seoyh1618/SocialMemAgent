GitHub 知识库管理器，用于在本地存储、索引和查询 GitHub 存储库。
维护一个 BASE.md 索引文件，其中包含所有已保存存储库的摘要。

当用户提出与以下相关的问题或请求时触发此技能：
- 特定的GitHub存储库（下载、查询、分析、总结、比较）
- GitHub 问题、PR、版本或存储库搜索
- 仓库、GitHub项目、开源项目的下载、查询、分析等操作

不要触发不相关的 git 操作（例如，“init a local git repo”、“git commit”）。

核心行动：
1.显式下载请求→需要URL→克隆或拉取→总结→更新BASE.md
2.查询+repo存在本地→拉取→也许更新摘要→回答问题
3. 查询+存储库非本地 → 通过 gh/curl 搜索 GitHub → 回答问题（不要克隆）
