GitHub 知識庫管理器，用於在本機上儲存、索引和查詢 GitHub 儲存庫。
維護一個 BASE.md 索引文件，其中包含所有已儲存儲存庫的摘要。

當使用者提出與以下相關的問題或請求時觸發此技能：
- 特定的GitHub儲存庫（下載、查詢、分析、總結、比較）
- GitHub 問題、PR、版本或儲存庫搜索
- 倉庫、GitHub專案、開源專案的下載、查詢、分析等操作

不要觸發不相關的 git 操作（例如，「init a local git repo」、「git commit」）。

核心行動：
1.明確下載請求→需要URL→複製或拉取→總結→更新BASE.md
2.查詢+repo存在本地→拉取→也許更新摘要→回答問題
3. 查詢+儲存庫非本地 → 透過 gh/curl 搜尋 GitHub → 回答問題（不要複製）
