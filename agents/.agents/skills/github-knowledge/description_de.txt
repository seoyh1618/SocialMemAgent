GitHub-Wissensdatenbankmanager zum lokalen Speichern, Indizieren und Abfragen von GitHub-Repositorys.
Verwaltet eine BASE.md-Indexdatei mit Zusammenfassungen aller gespeicherten Repos.

Lösen Sie diesen Skill aus, wenn der Benutzer eine Frage stellt oder eine Anfrage im Zusammenhang mit Folgendem stellt:
- Ein bestimmtes GitHub-Repository (herunterladen, abfragen, analysieren, zusammenfassen, vergleichen)
- GitHub-Probleme, PRs, Releases oder Repository-Suche
- 仓库、GitHub项目、开源项目 的下载、查询、分析等操作

NICHT für nicht verwandte Git-Vorgänge auslösen (z. B. „ein lokales Git-Repo initiieren“, „Git-Commit“).

Kernaktionen:
1. Explizite Download-Anfrage → URL erforderlich → Klonen oder Abrufen → Zusammenfassen → BASE.md aktualisieren
2. Abfrage + Repo existiert lokal → Pull → vielleicht Zusammenfassung aktualisieren → Frage beantworten
3. Abfrage + Repo NICHT lokal → GitHub über gh/curl durchsuchen → Frage beantworten (NICHT klonen)
