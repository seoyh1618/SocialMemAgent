GitHub 리포지토리를 로컬로 저장, 색인화 및 쿼리하기 위한 GitHub 지식 기반 관리자입니다.
저장된 모든 저장소의 요약이 포함된 BASE.md 인덱스 파일을 유지 관리합니다.

사용자가 다음과 관련된 질문을 하거나 요청할 때 이 스킬을 트리거합니다.
- 특정 GitHub 저장소(다운로드, 쿼리, 분석, 요약, 비교)
- GitHub 문제, PR, 릴리스 또는 저장소 검색
- 仓库、GitHub项目、开源项目、下载、查询、分析等操작성

관련 없는 git 작업(예: "로컬 git repo 초기화", "git commit")에 대해 트리거하지 마세요.

핵심 활동:
1. 명시적 다운로드 요청 → URL 필요 → 복제 또는 가져오기 → 요약 → BASE.md 업데이트
2. 쿼리 + 저장소가 로컬에 존재함 → 풀 → 요약 업데이트 → 질문에 답변
3. 로컬이 아닌 쿼리 + 저장소 → gh/curl을 통해 GitHub 검색 → 질문에 답변(복제하지 않음)
