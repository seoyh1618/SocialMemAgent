مدير قاعدة معارف GitHub لتخزين مستودعات GitHub وفهرستها والاستعلام عنها محليًا.
يحتفظ بملف فهرس BASE.md مع ملخصات لجميع عمليات إعادة الشراء المحفوظة.

قم بتشغيل هذه المهارة عندما يطرح المستخدم سؤالاً أو يقدم طلبًا يتعلق بما يلي:
- مستودع GitHub محدد (التنزيل والاستعلام والتحليل والتلخيص والمقارنة)
- مشكلات GitHub أو العلاقات العامة أو الإصدارات أو البحث في المستودع
- موقع الويب، موقع GitHub، موقع الويب، موقع الويب، موقع الويب، موقع الويب

لا تقم بتشغيل عمليات git غير ذات صلة (على سبيل المثال، "init a local git repo"، "git Commit").

الإجراءات الأساسية:
1. طلب تنزيل صريح → يتطلب عنوان URL → استنساخ أو سحب → تلخيص → تحديث BASE.md
2. الاستعلام + الريبو موجود محليًا → سحب → ربما تحديث الملخص → الإجابة على السؤال
3. الاستعلام + الريبو ليس محليًا → ابحث في GitHub عبر gh/curl → أجب عن السؤال (لا تستنسخ)
