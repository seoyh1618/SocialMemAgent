Менеджер базы знаний GitHub для хранения, индексирования и локального запроса репозиториев GitHub.
Поддерживает индексный файл BASE.md со сводками всех сохраненных репозиториев.

Активируйте этот навык, когда пользователь задает вопрос или делает запрос, связанный с:
- Конкретный репозиторий GitHub (загрузка, запрос, анализ, обобщение, сравнение)
- Проблемы GitHub, PR, релизы или поиск в репозитории.
- 仓库、GitHub项目、开源项目 的下载、查询、分析等操作

НЕ запускайте несвязанные операции git (например, «инициализировать локальный репозиторий git», «git commit»).

Основные действия:
1. Явный запрос на загрузку → запросить URL → клонировать или извлечь → суммировать → обновить BASE.md.
2. Запрос + репозиторий существует локально → получение → возможно, обновление сводки → ответ на вопрос.
3. Запрос + репо НЕ локально → поиск на GitHub через gh/curl → ответ на вопрос (НЕ клонировать)
