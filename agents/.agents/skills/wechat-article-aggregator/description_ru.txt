Агрегатор статей официального аккаунта WeChat. Пакетно получайте список последних статей указанных блоггеров через API mptext.top, загружайте содержимое статьи и анализируйте его в Markdown/HTML/обычный текст. Поддерживает поиск по официальному имени учетной записи или ложному идентификатору с 8 предустановленными популярными учетными записями AI-tech. Варианты использования: получение последних статей из определенной учетной записи, пакетное сканирование контента от нескольких блоггеров, совокупное чтение статей открытого доступа, сбор технических новостей. Ключевые слова: извлечение статей OA, очистка статей WeChat, агрегация OA, сбор статей, API mptext.
