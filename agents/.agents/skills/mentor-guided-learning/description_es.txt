El mentor a nivel de proyecto inicia y ejecuta la habilidad. Úselo cuando el usuario invoca `$mentor-guided-learning`: detecta el uso por primera vez frente al aprendizaje continuo. La primera vez que lo use, configure todos los archivos del sistema de aprendizaje (`.LEARNING`, perfil de estudiante, raíz del proyecto `AGENTS.md`/`GEMINI.md`/`CLAUDE.md`, `.gitignore`). En el aprendizaje continuo, lea los archivos de reglas de tiempo de ejecución sincronizados y `.LEARNING` y continúe. Utilice también esta habilidad cuando el usuario solicite actualizar la configuración del perfil.
