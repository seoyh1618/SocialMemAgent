Avviamento del mentore a livello di progetto e abilità di gestione. Da utilizzare quando l'utente invoca `$mentor-guided-learning`: rileva il primo utilizzo rispetto all'apprendimento continuo. Al primo utilizzo, configura tutti i file del sistema di apprendimento (`.LEARNING`, profilo dello studente, root del progetto `AGENTS.md`/`GEMINI.md`/`CLAUDE.md`, `.gitignore`). Durante l'apprendimento continuo, leggere i file delle regole di runtime sincronizzate e ".LEARNING" e continuare. Utilizza questa abilità anche quando l'utente chiede di aggiornare le impostazioni del profilo.
