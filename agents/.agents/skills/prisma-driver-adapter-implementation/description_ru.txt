Необходима ссылка для работы адаптера драйвера Prisma v7. Используйте при реализации или изменении адаптеров, добавлении драйверов базы данных или работе с интерфейсами SqlDriverAdapter/Transaction. Содержит важные детали контракта, которые невозможно вывести из примеров кода, включая протокол жизненного цикла транзакции, требования к сопоставлению ошибок и контрольный список проверки. Существующие реализации не заменяют этот навык.
