Referencia necesaria para el funcionamiento del adaptador del controlador Prisma v7. Úselo al implementar o modificar adaptadores, agregar controladores de bases de datos o tocar interfaces SqlDriverAdapter/Transaction. Contiene detalles críticos del contrato que no se pueden deducir de ejemplos de código, incluido el protocolo del ciclo de vida de la transacción, los requisitos de mapeo de errores y la lista de verificación de verificación. Las implementaciones existentes no reemplazan esta habilidad.
