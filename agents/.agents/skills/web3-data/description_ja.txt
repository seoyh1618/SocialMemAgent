Chainbase API を使用して Web3 オンチェーン データを探索します。このスキルは、ユーザーがブロックチェーン データ、トークン所有者、ウォレット アドレス、トークン価格、ENS ドメイン、トランザクション、DeFi ポートフォリオ、またはオンチェーン分析について尋ねたときに使用します。トリガーには、「上位保有者」、「保有者」、「ウォレット アドレス」、「トークン価格」、「トークン転送」、「ENS ドメイン」、「オンチェーン データ」、「ブロックチェーン クエリ」、「オンチェーン SQL クエリ」、または Ethereum、BSC、Polygon、Arbitrum、Optimism、Base、Avalanche、zkSync、およびその他の EVM にわたる Web3/ブロックチェーン データを検索、分析、または探索するためのリクエストが含まれます。鎖。
