قم بإنشاء الصور والنصوص من خلال واجهة برمجة تطبيقات Gemini Web API ذات الهندسة العكسية. يدعم إنشاء النص، والمطالبة بالصورة، وإدخال رؤية الصورة المرجعية، والحوار متعدد المنعطفات. يُستخدم عندما تحتاج مهارات أخرى إلى واجهة خلفية لإنشاء الصور أو عندما يطلب المستخدم "إنشاء صور باستخدام Gemini" أو "إنشاء نص Gemini".
