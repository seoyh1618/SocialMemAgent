Создавайте изображения и текст с помощью обратного проектирования Gemini Web API. Поддерживает генерацию текста, подсказку к изображению, ввод эталонного изображения и многоходовой диалог. Используйте, когда другим навыкам требуется серверная часть для создания изображений или когда пользователь просит «создать изображения с помощью Gemini» или «создать текст Gemini».
