GitHub CLI (gh)-Befehlsreferenz für Repository-Suche, Codeerkennung, CI/CD-Überwachung, Workflow-Erstellung und Automatisierung. Auslöser für „gh“-Befehle, „github cli“, Durchsuchen von Repos nach Dateien/Verzeichnissen (.skilz, .cursor, .codex, Dockerfile), Überwachen von GitHub Actions-Workflows, Überprüfen des PR-CI-Status, Herunterladen von Artefakten, Erstellen von PRs/Issues/Repos über die Befehlszeile, Auslösen von Workflows, Forken von Repos, Batch-Vorgängen, „Schreiben Sie einen Workflow“, „Github-Aktionen“, „CI/CD-Pipeline“, „Workflow Yaml“ und „Matrix“. baut". Verwendung für GH Search, GH Run, GH PR, GH Issue, GH Repo, GH Workflow, GH API, Workflow-Erstellung und Automatisierung von GitHub-Vorgängen.
