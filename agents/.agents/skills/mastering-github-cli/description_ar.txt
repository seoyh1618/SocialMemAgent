مرجع أمر GitHub CLI (gh) للبحث في المستودع، واكتشاف التعليمات البرمجية، ومراقبة CI/CD، وتأليف سير العمل، والأتمتة. مشغلات لأوامر "gh"، "github cli"، البحث عن مستودعات للملفات/الدلائل (.skilz، .cursor، .codex، Dockerfile)، مراقبة سير عمل GitHub Actions، التحقق من حالة PR CI، تنزيل العناصر، إنشاء PRs/قضايا/repos من سطر الأوامر، تشغيل سير العمل، تفرع عمليات إعادة الشراء، العمليات المجمعة، "كتابة سير عمل"، "إجراءات github"، "خط أنابيب CI/CD"، و"yaml سير العمل"، و"إنشاءات المصفوفة". يُستخدم للبحث عن gh، وتشغيل gh، وgh pr، وgh Issue، وgh repo، وgh Workflow، وgh api، وتأليف سير العمل، وأتمتة عمليات GitHub.
