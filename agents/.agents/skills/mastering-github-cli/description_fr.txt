Référence des commandes GitHub CLI (gh) pour la recherche dans le référentiel, la découverte de code, la surveillance CI/CD, la création de flux de travail et l'automatisation. Déclencheurs sur les commandes « gh », « github cli », recherche de fichiers/répertoires dans les dépôts (.skilz, .cursor, .codex, Dockerfile), surveillance des workflows GitHub Actions, vérification de l'état PR CI, téléchargement d'artefacts, création de PR/problèmes/dépôts à partir de la ligne de commande, déclenchement de workflows, fork de dépôts, opérations par lots, « écriture d'un workflow », « actions github », « pipeline CI/CD », « workflow yaml » et « builds matriciels ». À utiliser pour gh search, gh run, gh pr, gh issue, gh repo, gh workflow, gh api, la création de workflow et l'automatisation des opérations GitHub.
