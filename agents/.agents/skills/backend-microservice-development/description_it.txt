Standard di sviluppo di microservizi backend che coprono la struttura delle directory, l'architettura a più livelli (API/servizio/DAO), l'inserimento delle dipendenze, la gestione della configurazione e le best practice di Spring Boot. Utilizzato quando gli utenti eseguono lo sviluppo back-end, creano nuovi microservizi, scrivono codice Kotlin/Java o progettano l'architettura dei servizi.
