Normes de développement de microservices backend couvrant la structure des répertoires, l'architecture en couches (API/Service/DAO), l'injection de dépendances, la gestion de la configuration et les meilleures pratiques Spring Boot. Utilisé lorsque les utilisateurs effectuent du développement backend, créent de nouveaux microservices, écrivent du code Kotlin/Java ou conçoivent une architecture de service.
