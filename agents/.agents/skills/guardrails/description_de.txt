In den Agentenlebenszyklus integrierte Tore zur Überprüfung der Codequalität. Nutzen Sie diese Fertigkeit beim Schreiben, Modifizieren, Überprüfen oder Debuggen von Code – einschließlich neuer Funktionen, Fehlerbehebungen, Refaktorierungen, Fehlerbehebung, CI/CD-Einrichtung oder Projekt-Bootstrapping. Wird auch verwendet, wenn der Benutzer „Qualität“, „Teststrategie“, „CI-Pipeline“, „Leitlinien“, „Debugging“ erwähnt oder fragt, wie die Codezuverlässigkeit verbessert werden kann. Wenn Sie Code schreiben oder versuchen zu verstehen, warum Code nicht funktioniert, ist diese Fähigkeit anwendbar.
