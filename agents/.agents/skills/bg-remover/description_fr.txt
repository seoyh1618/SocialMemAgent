Outil intelligent de suppression d’arrière-plan d’image. Utilisé lorsque les utilisateurs demandent « supprimer l'arrière-plan de l'image », « découper l'image », « supprimer l'arrière-plan », « générer une image d'arrière-plan transparente » ou des besoins similaires. Après avoir téléchargé une image, il identifie automatiquement le sujet et supprime l'arrière-plan, générant une image PNG avec transparence.
