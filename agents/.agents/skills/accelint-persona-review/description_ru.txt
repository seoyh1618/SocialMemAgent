Оценивайте дизайн Figma с точки зрения оператора посредством критики дизайна и оценки пользовательского опыта. Используйте при проверке UX для конкретных ролей пользователей (например, техник воздушного наблюдения, директор по вооружению), проведении обзоров дизайна или оценке операторских интерфейсов. Анализирует когнитивную нагрузку, модели общения, болевые точки и видимость системы. Работает с документами Figma MCP (рабочий стол/URL) и Outline.
