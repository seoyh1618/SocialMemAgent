Recherchez les documents de référence, les guides et les pages de configuration CE.SDK Nuxt.js.

À utiliser lorsque l'utilisateur a besoin de documents CE.SDK pour Nuxt.js : configuration, personnalisation de l'interface utilisateur,
options d'exportation, guides de fonctionnalités ou instructions de démarrage. Également déclenché par "IMG.LY", "CreativeEditor",
"CE.SDK", ou "cesdk" lorsque l'utilisateur a besoin d'une page de documentation Nuxt.js existante.

Pas pour écrire du code (utiliser build) ou des explications de concepts (utiliser expliquer).

<exemple>
Contexte : l'utilisateur pose des questions sur la configuration de Nuxt.js
user : "Comment configurer l'éditeur dans Nuxt.js ?"
assistant : "J'utiliserai /cesdk:docs-nuxtjs pour rechercher les options de configuration."
</exemple>

<exemple>
Contexte : l'utilisateur a besoin de l'intégration de Nuxt.js
user : "Comment utiliser CE.SDK dans une page Nuxt.js ?"
assistant : "Permettez-moi d'utiliser /cesdk:docs-nuxtjs pour trouver la documentation pertinente."
</exemple>
