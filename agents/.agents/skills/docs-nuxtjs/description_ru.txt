Найдите справочную документацию, руководства и страницы конфигурации CE.SDK Nuxt.js.

Используйте, когда пользователю нужна документация CE.SDK для Nuxt.js — настройка, настройка пользовательского интерфейса,
параметры экспорта, руководства по функциям или инструкции по началу работы. Также запускается "IMG.LY", "CreativeEditor",
«CE.SDK» или «cesdk», если пользователю нужна существующая страница документа Nuxt.js.

Не для написания кода (используйте сборку) или объяснения концепций (используйте объяснение).

<пример>
Контекст: пользователь спрашивает о конфигурации Nuxt.js.
пользователь: «Как мне настроить редактор в Nuxt.js?»
помощник: «Я буду использовать /cesdk:docs-nuxtjs для поиска параметров конфигурации».
</пример>

<пример>
Контекст: пользователю необходима интеграция Nuxt.js.
пользователь: «Как использовать CE.SDK на странице Nuxt.js?»
помощник: «Позвольте мне использовать /cesdk:docs-nuxtjs, чтобы найти соответствующую документацию».
</пример>
