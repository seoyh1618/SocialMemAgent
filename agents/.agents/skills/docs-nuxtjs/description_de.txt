Suchen Sie nach CE.SDK Nuxt.js Referenzdokumenten, Anleitungen und Konfigurationsseiten.

Verwenden Sie diese Option, wenn der Benutzer CE.SDK-Dokumente für Nuxt.js benötigt – Konfiguration, UI-Anpassung,
Exportoptionen, Funktionshandbücher oder Anleitungen für den Einstieg. Wird auch durch „IMG.LY“, „CreativeEditor“ ausgelöst.
„CE.SDK“ oder „cesdk“, wenn der Benutzer eine vorhandene Nuxt.js-Dokumentseite benötigt.

Nicht zum Schreiben von Code (Build verwenden) oder Konzepterklärungen (Explain verwenden).

<Beispiel>
Kontext: Der Benutzer fragt nach der Konfiguration von Nuxt.js
Benutzer: „Wie konfiguriere ich den Editor in Nuxt.js?“
Assistent: „Ich verwende /cesdk:docs-nuxtjs, um nach Konfigurationsoptionen zu suchen.“
</Beispiel>
