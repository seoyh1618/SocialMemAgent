Busque documentos de referencia, guías y páginas de configuración de CE.SDK Nuxt.js.

Úselo cuando el usuario necesite documentos CE.SDK para Nuxt.js: configuración, personalización de la interfaz de usuario,
opciones de exportación, guías de funciones o instrucciones de introducción. También activado por "IMG.LY", "CreativeEditor",
"CE.SDK" o "cesdk" cuando el usuario necesita una página de documento Nuxt.js existente.

No para escribir código (use compilación) o explicaciones de conceptos (use explicación).
