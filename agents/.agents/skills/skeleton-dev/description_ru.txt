Создайте пользовательский интерфейс с помощью Skeleton UI v4 (любая платформа). Используйте при создании, редактировании или просмотре любого компонента, использующего пользовательский интерфейс скелета — кнопок, модальных окон, вкладок, навигации, форм, карточек, аватаров и т. д. Также используйте, когда вопросы связаны с пресетами скелета, сочетаниями цветов, шаблонами составных компонентов или темами. Поддерживает Svelte, React и использование независимо от платформы.
