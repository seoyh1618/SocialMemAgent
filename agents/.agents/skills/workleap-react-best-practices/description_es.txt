Pautas de optimización del rendimiento de React para aplicaciones de una sola página (SPA) en Workleap. Úselo al escribir, revisar o refactorizar el código de React SPA para garantizar patrones de rendimiento óptimos. Activadores de tareas que involucran componentes de React, administración de estado, optimización de paquetes, prevención de repetición de renderizado, rendimiento de renderizado o mejoras de rendimiento de JavaScript. Cubre la eliminación de cascadas asíncronas, la reducción del tamaño del paquete, la optimización de la renderización, la eficiencia de la renderización, las microoptimizaciones de JS y los patrones avanzados de React. NO cubre la representación del lado del servidor (SSR), Next.js ni los componentes del servidor.
