رد فعل إرشادات تحسين الأداء لتطبيقات الصفحة الواحدة (SPA) في Workleap. يُستخدم عند كتابة كود React SPA أو مراجعته أو إعادة هيكلته لضمان أنماط الأداء المثالية. يتم تشغيل المهام التي تتضمن مكونات React، أو إدارة الحالة، أو تحسين الحزمة، أو منع إعادة العرض، أو أداء العرض، أو تحسينات أداء JavaScript. يغطي إزالة الشلال غير المتزامن، وتقليل حجم الحزمة، وتحسين إعادة العرض، وكفاءة العرض، والتحسينات الدقيقة لـ JS، وأنماط React المتقدمة. لا يغطي العرض من جانب الخادم (SSR) أو Next.js أو مكونات الخادم.
