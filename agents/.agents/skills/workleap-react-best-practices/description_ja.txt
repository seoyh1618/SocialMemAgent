Workleap のシングル ページ アプリケーション (SPA) の React パフォーマンス最適化ガイドライン。 React SPA コードを作成、レビュー、またはリファクタリングするときに使用して、最適なパフォーマンス パターンを確保します。 React コンポーネント、状態管理、バンドルの最適化、再レンダリングの防止、レンダリングのパフォーマンス、または JavaScript のパフォーマンスの向上に関連するタスクをトリガーします。非同期ウォーターフォールの除去、バンドル サイズの削減、再レンダリングの最適化、レンダリング効率、JS のマイクロ最適化、高度な React パターンをカバーします。サーバーサイド レンダリング (SSR)、Next.js、サーバー コンポーネントは対象外です。
