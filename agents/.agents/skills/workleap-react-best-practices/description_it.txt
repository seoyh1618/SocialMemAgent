Linee guida per l'ottimizzazione delle prestazioni di React per applicazioni a pagina singola (SPA) su Workleap. Utilizzare durante la scrittura, la revisione o il refactoring del codice React SPA per garantire modelli di prestazioni ottimali. Trigger su attività che coinvolgono componenti React, gestione dello stato, ottimizzazione dei bundle, prevenzione del re-rendering, prestazioni di rendering o miglioramenti delle prestazioni JavaScript. Copre l'eliminazione delle cascate asincrone, la riduzione delle dimensioni dei bundle, l'ottimizzazione del re-render, l'efficienza del rendering, le micro-ottimizzazioni JS e i pattern React avanzati. NON copre il rendering lato server (SSR), Next.js o i componenti server.
