React-Richtlinien zur Leistungsoptimierung für Single Page Applications (SPA) bei Workleap. Wird beim Schreiben, Überprüfen oder Refactoring von React SPA-Code verwendet, um optimale Leistungsmuster sicherzustellen. Wird bei Aufgaben ausgelöst, die React-Komponenten, Zustandsverwaltung, Bundle-Optimierung, Re-Rendering-Verhinderung, Rendering-Leistung oder JavaScript-Leistungsverbesserungen betreffen. Behandelt die Beseitigung asynchroner Wasserfälle, die Reduzierung der Bundle-Größe, die Optimierung des erneuten Renderns, die Rendering-Effizienz, JS-Mikrooptimierungen und erweiterte Reaktionsmuster. Deckt NICHT serverseitiges Rendering (SSR), Next.js oder Serverkomponenten ab.
