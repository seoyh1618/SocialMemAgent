Рекомендации по оптимизации производительности React для одностраничных приложений (SPA) на Workleap. Используйте при написании, проверке или рефакторинге кода React SPA, чтобы обеспечить оптимальные шаблоны производительности. Запускается при выполнении задач, включающих компоненты React, управление состоянием, оптимизацию пакетов, предотвращение повторного рендеринга, производительность рендеринга или улучшение производительности JavaScript. Охватывает устранение асинхронного водопада, уменьшение размера пакета, оптимизацию повторного рендеринга, эффективность рендеринга, микрооптимизацию JS и расширенные шаблоны React. НЕ распространяется на серверный рендеринг (SSR), Next.js или серверные компоненты.
