Trasforma una fase di modifica di OpenSpec in richieste di build pronte per l'interfaccia utente attraverso una pipeline in 3 passaggi. Generazione PRD, quindi specifica UX, quindi richieste di ordine di creazione. Da utilizzare quando si avvia l'implementazione di una fase di funzionalità OpenSpec o quando l'utente menziona "openspec-to-prompts", "pipeline di funzionalità" o chiede di convertire una fase di specifica in richieste di implementazione. Richiede un ID modifica openspec e un numero di fase come input.
