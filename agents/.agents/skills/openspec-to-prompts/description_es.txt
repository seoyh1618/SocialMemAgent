Transforme una fase de cambio de OpenSpec en mensajes de compilación listos para la interfaz de usuario a través de un proceso de 3 pasos. Generación de PRD, luego especificación de UX y luego indicaciones de orden de compilación. Úselo al iniciar la implementación de una fase de característica de OpenSpec, o cuando el usuario menciona "openspec-to-prompts", "feature pipeline" o solicita convertir una fase de especificación en indicaciones de implementación. Requiere una identificación de cambio de especificación abierta y un número de fase como entrada.
