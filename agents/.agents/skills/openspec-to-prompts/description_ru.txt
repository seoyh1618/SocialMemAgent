Превратите этап изменения OpenSpec в готовые к пользовательскому интерфейсу запросы сборки с помощью трехэтапного конвейера. Генерация PRD, затем спецификация UX, затем подсказки по порядку сборки. Используйте при запуске реализации фазы функции OpenSpec или когда пользователь упоминает «открытие спецификации до подсказок», «конвейер функций» или просит преобразовать фазу спецификации в подсказки реализации. В качестве входных данных требуется идентификатор изменения openspec и номер фазы.
