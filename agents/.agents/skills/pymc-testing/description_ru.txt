Тестирование моделей PyMC с помощью pytest. Используйте при написании модульных тестов для байесовских моделей, настройке тестовых приспособлений, имитации выборки MCMC или тестировании структуры модели. Охватывает pymc.testing.mock_sample, фикстуры pytest и различие между быстрыми структурными тестами (издевательством) и медленными тестами апостериорного вывода. Запускается по: тестированию PyMC, pytest, модульным тестам для моделей, макетной выборке, тестовым приспособлениям, CI/CD для байесовских моделей.
