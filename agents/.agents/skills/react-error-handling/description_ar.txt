تفاعل مع حدود الأخطاء وواجهات المستخدم الاحتياطية لاكتشاف أخطاء العرض. يُستخدم عند التعامل مع أعطال المكونات، أو عرض حالات الخطأ، أو تنفيذ استرداد الأخطاء، أو منع أعطال الصفحة الكاملة. يُستخدم لـ ErrorBoundary، وcomponentDidCatch، وgetDerivedStateFromError، والخطأ الاحتياطي، واسترداد الأخطاء، ومعالجة الأعطال، ومكتبة حدود خطأ التفاعل.
