Límites de error de React y UI de respaldo para detectar errores de renderizado. Úselo cuando maneje fallas de componentes, muestre estados de error, implemente recuperación de errores o evite fallas de página completa. Úselo para ErrorBoundary, componenteDidCatch, getDerivedStateFromError, respaldo de errores, recuperación de errores, manejo de fallas, biblioteca de límites de errores de reacción.
