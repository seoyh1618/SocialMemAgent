Границы ошибок React и резервные пользовательские интерфейсы для обнаружения ошибок рендеринга. Используйте при обработке сбоев компонентов, отображении состояний ошибок, реализации восстановления после ошибок или предотвращении полностраничных сбоев. Используйте для ErrorBoundary, компонентDidCatch, getDerivedStateFromError, резервного копирования ошибок, восстановления ошибок, обработки сбоев, библиотеки границ реагирования на ошибки.
