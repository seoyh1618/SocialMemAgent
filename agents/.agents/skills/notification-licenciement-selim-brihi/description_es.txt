Guía para la redacción de notificaciones de licencia conforme al derecho del trabajo francés. Utiliser ce skills quand l'utilisateur demande de rédiger, préparer, créer ou editer una carta de notificación de licencia, notamment pour faute grave, faute lourde ou motive personal. Le Skill Guide la recopilación de información precisa y la redacción de una notificación jurídica sólida con todos los elementos obligatorios del derecho francés.
