44 のプラットフォームにわたる 66 のソースからトレンド ニュースや注目のトピックを取得する CLI ツール。タイトル、URL、メタデータを含む構造化ニュース項目を返します。

用途:
- 中国および国際的なプラットフォームからトレンド/ホットニュースを取得
- ソーシャル メディア、テクノロジー、金融、ニュース サイトにわたるホットなトピックの監視
- さらなる処理のために構造化ニュース データを JSON として取得する
- 利用可能なニュースソースのリスト

npm インストールが必要です。一部のソースには環境変数 (PRODUCTHUNT_API_TOKEN) が必要です。一部のソースは Cloudflare (linuxdo) によってブロックされる可能性があります。
