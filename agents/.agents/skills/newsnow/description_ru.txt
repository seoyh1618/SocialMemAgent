Инструмент CLI для получения актуальных новостей и горячих тем из 66 источников на 44 платформах. Возвращает структурированные новости с заголовками, URL-адресами и метаданными.

ИСПОЛЬЗОВАТЬ ДЛЯ:
- Получение трендовых/горячих новостей с китайских и международных платформ.
- Мониторинг горячих тем в социальных сетях, на сайтах, посвященных технологиям, финансам и новостям.
- Получение структурированных новостных данных в формате JSON для дальнейшей обработки.
- Список доступных источников новостей

Требуется установка npm. Некоторым источникам требуются переменные окружения (PRODUCTHUNT_API_TOKEN). Некоторые источники могут быть заблокированы Cloudflare (linuxdo).
