Herramienta CLI para obtener noticias de actualidad y temas de actualidad de 66 fuentes en 44 plataformas. Devuelve noticias estructuradas con títulos, URL y metadatos.

UTILIZAR PARA:
- Obtención de tendencias y noticias de actualidad de plataformas chinas e internacionales.
- Seguimiento de temas candentes en redes sociales, tecnología, finanzas y sitios de noticias.
- Obtención de datos de noticias estructurados como JSON para su posterior procesamiento.
- Listado de fuentes de noticias disponibles

Requiere instalación de npm. Algunas fuentes necesitan variables de entorno (PRODUCTHUNT_API_TOKEN). Algunas fuentes pueden estar bloqueadas por Cloudflare (linuxdo).
