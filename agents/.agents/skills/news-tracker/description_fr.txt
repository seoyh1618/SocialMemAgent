Outil de suivi des actualités et d’acquisition dynamique d’informations. Récupère les dernières nouvelles sur des sujets spécifiques et prend en charge des questions et réponses intelligentes basées sur des informations à jour. Déclenché lorsque les utilisateurs doivent : (1) interroger les dernières nouvelles dans un domaine (par exemple, IA, technologie, finance) ; (2) se renseigner sur les développements d'une entreprise (par exemple, OpenAI, Google, Tesla) ; (3) suivre le statut récent d'une personne (par exemple, Musk, Sam Altman) ; (4) obtenir des actualités régionales (par exemple, les actualités technologiques chinoises) ; (5) connaître le dernier statut d'un produit (par exemple, ChatGPT, Claude) ; (6) suivre la progression d'un événement (par exemple, CES) ; (7) poser des questions nécessitant des informations actuelles (par exemple, « Quel est le modèle d'IA le plus puissant actuellement »). Les mots-clés déclencheurs incluent : actualités, développements, derniers, récents, statut, progrès, informations, etc.
