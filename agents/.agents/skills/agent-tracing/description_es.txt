CLI de seguimiento del agente para inspeccionar instantáneas de ejecución del agente. Úselo cuando el usuario mencione "seguimiento de agentes", "rastreo", "instantánea", quiera depurar la ejecución del agente, inspeccionar llamadas de LLM, ver datos del motor de contexto o analizar los pasos del agente. Activadores de tareas de depuración del agente, inspección de seguimiento o análisis de ejecución.
