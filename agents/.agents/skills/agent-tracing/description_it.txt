CLI di tracciamento dell'agente per l'ispezione degli snapshot di esecuzione dell'agente. Da utilizzare quando l'utente menziona "tracciamento agente", "traccia", "istantanea", desidera eseguire il debug dell'esecuzione dell'agente, ispezionare le chiamate LLM, visualizzare i dati del motore di contesto o analizzare i passaggi dell'agente. Attiva attività di debug dell'agente, ispezione della traccia o analisi dell'esecuzione.
