CLI de traçage d’agent pour inspecter les instantanés d’exécution d’agent. À utiliser lorsque l'utilisateur mentionne « traçage d'agent », « trace », « instantané », souhaite déboguer l'exécution de l'agent, inspecter les appels LLM, afficher les données du moteur contextuel ou analyser les étapes de l'agent. Déclencheurs sur les tâches de débogage d’agent, d’inspection de trace ou d’analyse d’exécution.
