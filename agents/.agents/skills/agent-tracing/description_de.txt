Agent-Tracing-CLI zum Überprüfen von Agent-Ausführungs-Snapshots. Verwenden Sie diese Option, wenn der Benutzer „Agent-Tracing“, „Trace“, „Snapshot“ erwähnt, die Agent-Ausführung debuggen, LLM-Aufrufe überprüfen, Kontext-Engine-Daten anzeigen oder Agent-Schritte analysieren möchte. Löst bei Agent-Debugging-, Trace-Inspektion- oder Ausführungsanalyseaufgaben aus.
