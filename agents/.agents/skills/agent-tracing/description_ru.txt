CLI трассировки агента для проверки снимков выполнения агента. Используйте, когда пользователь упоминает «трассировку агента», «трассировку», «снимок», хочет отладить выполнение агента, проверить вызовы LLM, просмотреть данные механизма контекста или проанализировать шаги агента. Запускается при отладке агента, проверке трассировки или выполнении задач анализа.
