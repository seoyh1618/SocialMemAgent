Специализированные навыки создания и настройки тестовых проектов xUnit.  
Используется при создании тестового проекта, настройке структуры проекта, настройке пакетов NuGet и организации тестовых папок.  
Охватывает конфигурацию csproj, управление пакетами, структуру проекта и настройки xunit.runner.json.
