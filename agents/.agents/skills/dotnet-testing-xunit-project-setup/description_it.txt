Competenze specializzate nella creazione e configurazione di progetti di test xUnit.  
Utilizzato quando si stabilisce un progetto di test, si imposta la struttura del progetto, si configurano i pacchetti NuGet e si organizzano le cartelle di test.  
Copre la configurazione di csproj, la gestione dei pacchetti, la struttura del progetto e le impostazioni di xunit.runner.json.
