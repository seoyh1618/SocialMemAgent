أنشئ روبوتات دردشة متعددة المنصات باستخدام Chat SDK (حزمة `chat` npm). يُستخدم عندما يريد المطورون (1) إنشاء Slack أو Teams أو Google Chat أو Discord أو GitHub أو Linear bot، (2) استخدام Chat SDK للتعامل مع الإشارات أو الرسائل أو التفاعلات أو أوامر الشرطة المائلة أو البطاقات أو الوسائط أو البث، (3) إعداد معالجات webhook لمنصات الدردشة، (4) إرسال بطاقات تفاعلية أو بث استجابات الذكاء الاصطناعي إلى منصات الدردشة. مشغلات على "chat sdk"، و"chat bot"، و"slack bot"، و"teams bot"، و"discord bot"، و"@chat-adapter"، لبناء روبوتات تعمل عبر منصات دردشة متعددة.
