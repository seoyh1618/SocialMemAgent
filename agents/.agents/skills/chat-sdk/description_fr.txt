Créez des chatbots multiplateformes avec Chat SDK (package npm `chat`). À utiliser lorsque les développeurs souhaitent (1) créer un robot Slack, Teams, Google Chat, Discord, GitHub ou Linear, (2) utiliser le SDK Chat pour gérer les mentions, les messages, les réactions, les commandes slash, les cartes, les modaux ou le streaming, (3) configurer des gestionnaires de webhook pour les plateformes de chat, (4) envoyer des cartes interactives ou diffuser des réponses d'IA aux plateformes de chat. Déclencheurs sur "chat sdk", "chat bot", "slack bot", "teams bot", "discord bot", "@chat-adapter", créant des robots qui fonctionnent sur plusieurs plateformes de chat.
