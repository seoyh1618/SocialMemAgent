Erstellen Sie plattformübergreifende Chat-Bots mit dem Chat SDK (npm-Paket „chat“). Verwenden Sie diese Option, wenn Entwickler (1) einen Slack-, Teams-, Google Chat-, Discord-, GitHub- oder Linear-Bot erstellen möchten, (2) das Chat SDK verwenden möchten, um Erwähnungen, Nachrichten, Reaktionen, Slash-Befehle, Karten, Modalitäten oder Streaming zu verarbeiten, (3) Webhook-Handler für Chat-Plattformen einrichten möchten, (4) interaktive Karten senden oder KI-Antworten an Chat-Plattformen streamen möchten. Löst auf „Chat SDK“, „Chat Bot“, „Slack Bot“, „Teams Bot“, „Discord Bot“, „@Chat-Adapter“ aus und erstellt Bots, die auf mehreren Chat-Plattformen funktionieren.
