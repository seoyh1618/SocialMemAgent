Chat SDK (`chat` npm パッケージ) を使用してマルチプラットフォームのチャット ボットを構築します。開発者が次のことを行う場合に使用します。(1) Slack、Teams、Google Chat、Discord、GitHub、または Linear ボットを構築する、(2) Chat SDK を使用してメンション、メッセージ、リアクション、スラッシュ コマンド、カード、モーダル、またはストリーミングを処理する、(3) チャット プラットフォームの Webhook ハンドラーを設定する、(4) インタラクティブ カードを送信するか、チャット プラットフォームに AI 応答をストリーミングする。 「chat sdk」、「chat bot」、「slack bot」、「teams bot」、「discord bot」、「@chat-adapter」でトリガーされ、複数のチャット プラットフォームで動作するボットを構築します。
