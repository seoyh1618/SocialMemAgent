Создавайте многоплатформенные чат-боты с помощью Chat SDK (пакет npm `chat`). Используйте, когда разработчики хотят (1) создать бота Slack, Teams, Google Chat, Discord, GitHub или Linear, (2) использовать Chat SDK для обработки упоминаний, сообщений, реакций, косых команд, карточек, модальных окон или потоковой передачи, (3) настроить обработчики веб-перехватчиков для чат-платформ, (4) отправлять интерактивные карточки или передавать потоковые ответы AI на чат-платформы. Запускает «chat sdk», «чат-бот», «slack bot», «командный бот», «discord bot», «@chat-adapter», создавая ботов, которые работают на нескольких чат-платформах.
