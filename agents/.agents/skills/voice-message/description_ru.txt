Отправляйте голосовые сообщения через каналы чата (Telegram, Discord, Feishu/Lark, Signal, WhatsApp и другие), используя Edge-tts для преобразования текста в речь и ffmpeg для преобразования звука. ВАЖНО. Feishu/Lark НЕ поддерживает asVoice=true через инструмент сообщений; вы ДОЛЖНЫ использовать этот навык для отправки голосовых сообщений на Feishu, иначе он отправит вложенный файл вместо голосового пузыря. | 通过 Edge-tts 文字转语音和 ffmpeg 音频转换，在各聊天渠道（Telegram、Discord、飞书、Signal、WhatsApp等）发送语音消息。飞书不支持 message 工具的 asVoice=true, 必须使用本 навык才能发送语音气泡而非文件附件。
