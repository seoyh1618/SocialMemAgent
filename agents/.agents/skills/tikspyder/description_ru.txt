Запустите инструмент TikSpyder для сбора данных TikTok — выполните поиск по ключевому слову, имени пользователя или хэштегу, загрузите видео, извлеките ключевые кадры и экспортируйте структурированные данные. Используйте этот навык всякий раз, когда пользователь хочет собрать данные TikTok, выполнить поиск по профилям или хэштегам TikTok, найти видео TikTok, загрузить контент TikTok, запустить Tikspyder или запустить интерфейс TikSpyder Streamlit. Также срабатывает, когда пользователь упоминает сбор данных из TikTok, даже если он не произносит «тикспайдер» по имени.
