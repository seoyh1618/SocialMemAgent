Напишите идеальные модульные тесты и просмотрите существующие. Ориентирован исключительно на модульное тестирование — тестирует один модуль изолированно. Не для интеграции или тестов E2E. Два режима: создание новых тестов на основе анализа кода или проверка существующих тестов на наличие антишаблонов и проблем с качеством. Языконезависимый, основанный на передовом опыте Бека, Фаулера, Хорикова и Google.
