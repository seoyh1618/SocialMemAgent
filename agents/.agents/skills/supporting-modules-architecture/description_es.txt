Guía de arquitectura del módulo de soporte BK-CI, que cubre la gestión de credenciales (Ticket), el entorno de compilación (Environment), el servicio de notificación (Notify), los registros de compilación (Log), las puertas de calidad (Quality), las interfaces abiertas (OpenAPI) y otros módulos de servicio de soporte. Úselo cuando desarrolle las características de estos módulos o cuando necesite comprender la arquitectura del servicio de soporte.
