Проанализируйте проект и создайте или улучшите файл автоматизации сборки (Makefile, Taskfile.yml, Justfile, Magefile.go). Если файл сборки уже существует, улучшите его, добавив недостающие цели и рекомендации. Используйте, когда пользователь говорит «создать make-файл», «создать файл задачи», «добавить только файл», «мастер установки» или «автоматизация сборки».
