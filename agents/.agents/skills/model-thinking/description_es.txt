Conjunto de herramientas de modelos mentales para pensar con mayor claridad, tomar mejores decisiones y resolver problemas. Úselo cuando los usuarios enfrentan problemas complejos, necesitan apoyo para tomar decisiones, desean analizar situaciones desde múltiples ángulos, organizar información, comprender sistemas, predecir resultados o aprender sobre modelos mentales específicos. Los desencadenantes incluyen frases como "ayúdame a pensar", "analizar este problema", "qué modelos se aplican aquí", "cómo debo decidir", "evaluar opciones" o referencias directas a modelos (por ejemplo, "usar pensamiento de segundo orden", "aplicar inversión"). 中文觸發：「思維模型」、「幫我分析」、「決策分析」、「多角度思考」、「怎麼判斷」、「幫我想清楚」、「系統思考」、「風險評估」.
