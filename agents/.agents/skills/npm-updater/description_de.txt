Schritt-für-Schritt-Abhängigkeitsaktualisierungs-Workflow für Node.js-Projekte. Verwenden Sie diese Option, wenn der Benutzer npm/yarn/pnpm/bun-Abhängigkeiten in package.json aktualisieren, aktualisieren oder erweitern möchte. Verarbeitet Minor-/Patch-Batching, Hauptversionsanalyse mit Überprüfung des Änderungsprotokolls, Erkennung von Migrationsanleitungen für Frameworks (Next.js, Prisma, React usw.) und Überprüfung nach dem Update. Unterstützt alle Paketmanager (npm, Yarn, Pnpm, Bun) mit automatischer Erkennung.
