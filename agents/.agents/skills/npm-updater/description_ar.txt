سير عمل تحديث التبعية خطوة بخطوة لمشاريع Node.js. يُستخدم عندما يطلب المستخدم تحديث تبعيات npm/yarn/pnpm/bun أو ترقيتها أو رفعها في package.json. يتعامل مع عمليات التجميع الصغيرة/التصحيحات، وتحليل الإصدارات الرئيسية من خلال مراجعة سجل التغيير، واكتشاف دليل الترحيل للأطر (Next.js، وPrisma، وReact، وما إلى ذلك)، والتحقق بعد التحديث. يدعم جميع مديري الحزم (npm، الغزل، pnpm، كعكة) مع الكشف التلقائي.
