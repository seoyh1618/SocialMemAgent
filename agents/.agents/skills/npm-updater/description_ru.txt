Пошаговый процесс обновления зависимостей для проектов Node.js. Используйте, когда пользователь запрашивает обновление, обновление или изменение зависимостей npm/yarn/pnpm/bun в package.json. Осуществляет пакетную обработку второстепенных/исправлений, анализ основных версий с просмотром журнала изменений, поиск руководств по миграции для платформ (Next.js, Prisma, React и т. д.) и проверку после обновления. Поддерживает все менеджеры пакетов (npm, Yarn, pnpm, Bun) с автоматическим обнаружением.
