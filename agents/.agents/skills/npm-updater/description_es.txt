Flujo de trabajo de actualización de dependencias paso a paso para proyectos de Node.js. Úselo cuando el usuario solicite actualizar o aumentar las dependencias de npm/yarn/pnpm/bun en package.json. Admite todos los administradores de paquetes (npm, Yarn, pnpm, bun) con detección automática.
