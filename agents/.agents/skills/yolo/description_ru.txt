Навыки автоматизации браузера для развертываний Lovable. Активируется, когда:
- yolo_mode: включено в CLAUDE.md
- Запуск команд /deploy-edge или /apply-migration.
— После git push, когда auto_deploy: включено (автоматическое обнаружение)
- Любое упоминание о «режиме йоло», «автоматизации Lovable», «автоматизации браузера».

Автоматически переходит на Lovable.dev и отправляет запросы на развертывание.
Запускает проверочные тесты на основе конфигурации.
Автоматическое развертывание после нажатия git, если оно включено.
