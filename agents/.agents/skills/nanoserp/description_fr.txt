Recherche Web gratuite et grattage de pages via DuckDuckGo. À utiliser lorsque l'agent doit rechercher des informations sur le Web, récupérer et lire le contenu d'une page Web ou rassembler des liens à partir d'une page. Se déclenche sur des tâches nécessitant une recherche sur le Web, du web scraping, une récupération d'URL ou une recherche sur Internet. Aucune clé ou compte API requis.
