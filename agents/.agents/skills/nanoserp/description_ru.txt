Бесплатный поиск в Интернете и парсинг страниц через DuckDuckGo. Используйте, когда агенту необходимо выполнить поиск информации в Интернете, получить и прочитать содержимое веб-страницы или собрать ссылки со страницы. Запускается при выполнении задач, требующих веб-поиска, очистки веб-страниц, получения URL-адресов или интернет-исследований. Никаких ключей API или учетных записей не требуется.
