Kostenlose Websuche und Page Scraping über DuckDuckGo. Verwenden Sie diese Option, wenn der Agent das Internet nach Informationen durchsuchen, Webseiteninhalte abrufen und lesen oder Links von einer Seite sammeln muss. Wird bei Aufgaben ausgelöst, die eine Websuche, Web Scraping, URL-Abruf oder Internetrecherche erfordern. Keine API-Schlüssel oder Konten erforderlich.
