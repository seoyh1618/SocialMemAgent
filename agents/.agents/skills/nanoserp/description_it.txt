Ricerca web gratuita e scraping di pagine tramite DuckDuckGo. Da utilizzare quando l'agente deve cercare informazioni sul Web, recuperare e leggere i contenuti di una pagina Web o raccogliere collegamenti da una pagina. Si attiva su attività che richiedono ricerca sul Web, web scraping, recupero di URL o ricerca su Internet. Non sono necessarie chiavi API o account.
