Compétence du navigateur AWS AgentCore pour les tests Web automatisés et la capture d'écran. Utilisez cette compétence lorsque l'utilisateur demande : - Tester si un site Web fonctionne / 测试网站是否正常 - Prendre une capture d'écran d'une page Web / 截取网页截图 - Vérifier les erreurs JavaScript ou les erreurs de console - Vérifier les points de terminaison de l'état de l'API - Ouvrir une URL dans un navigateur cloud - Déboguer les problèmes de chargement du site Web Cette compétence utilise le navigateur AWS Bedrock AgentCore (basé sur le cloud, non navigateur local requis).
