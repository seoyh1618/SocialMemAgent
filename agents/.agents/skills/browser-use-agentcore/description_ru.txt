Навыки работы с браузером AWS AgentCore для автоматического веб-тестирования и создания снимков экрана. Используйте этот навык, когда пользователь просит: - Проверить, работает ли веб-сайт / 测试网站是否正常 - Сделать снимок экрана веб-страницы / 截取网页截图 - Проверить наличие ошибок JavaScript или ошибок консоли - Проверить конечные точки работоспособности API - Открыть URL-адрес в облачном браузере - Отладить проблемы с загрузкой веб-сайта Для этого навыка используется браузер AWS Bedrock AgentCore (облако, локальный браузер не требуется).
