Competenza del browser AWS AgentCore per test Web automatizzati e acquisizione di screenshot. Utilizzare questa competenza quando l'utente chiede di: - Verificare se un sito Web funziona / 测试网站是否正常 - Acquisire uno screenshot di una pagina Web / 截取网页截图 - Verificare la presenza di errori JavaScript o errori della console - Verificare gli endpoint di integrità dell'API - Aprire un URL in un browser cloud - Eseguire il debug dei problemi di caricamento del sito Web Questa competenza utilizza il browser AWS Bedrock AgentCore (basato su cloud, senza browser locale necessario).
