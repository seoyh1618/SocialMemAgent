Habilidad de AWS AgentCore Browser para pruebas web automatizadas y captura de pantalla. Utilice esta habilidad cuando el usuario solicite: - Probar si un sitio web funciona / 测试网站是否正常 - Tomar una captura de pantalla de una página web / 截取网页截图 - Verificar errores de JavaScript o errores de consola - Verificar puntos finales de estado de API - Abrir una URL en un navegador en la nube - Depurar problemas de carga del sitio web Esta habilidad utiliza AWS Bedrock AgentCore Browser (basado en la nube, no Se necesita un navegador local).
