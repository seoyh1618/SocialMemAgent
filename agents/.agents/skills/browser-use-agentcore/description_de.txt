AWS AgentCore Browser-Skill für automatisierte Webtests und Screenshot-Erfassung. Verwenden Sie diesen Skill, wenn der Benutzer Folgendes verlangt: – Testen, ob eine Website funktioniert / 测试网站是否正常 – Erstellen eines Screenshots einer Webseite / 截取网页截图 – Überprüfen auf JavaScript-Fehler oder Konsolenfehler – Überprüfen von API-Zustandsendpunkten – Öffnen einer URL in einem Cloud-Browser – Debuggen von Website-Ladeproblemen Dieser Skill verwendet AWS Bedrock AgentCore Browser (cloudbasiert, kein lokaler Browser). benötigt).
