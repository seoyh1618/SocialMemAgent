مهارة متصفح AWS AgentCore لاختبار الويب الآلي والتقاط لقطات الشاشة. استخدم هذه المهارة عندما يطلب المستخدم القيام بما يلي: - اختبار ما إذا كان موقع الويب يعمل / التحقق - التقاط لقطة شاشة لصفحة الويب / التحقق من أخطاء JavaScript أو أخطاء وحدة التحكم - التحقق من نقاط نهاية سلامة واجهة برمجة التطبيقات - فتح عنوان URL في متصفح سحابي - تصحيح مشكلات تحميل موقع الويب تستخدم هذه المهارة متصفح AWS Bedrock AgentCore (قائم على السحابة، لا حاجة لمتصفح محلي).
