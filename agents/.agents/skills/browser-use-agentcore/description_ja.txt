自動化された Web テストとスクリーンショット キャプチャのための AWS AgentCore ブラウザ スキル。ユーザーが次のことを要求した場合に、このスキルを使用します。 - ウェブサイトが機能するかどうかをテストする / 测试网站が正常かどうか - ウェブページのスクリーンショットを撮る / 截取网页截图 - JavaScript エラーまたはコンソール エラーをチェックする - API 健全性エンドポイントを確認する - クラウドブラウザで URL を開く - ウェブサイトの読み込みの問題をデバッグする このスキルは、AWS Bedrock AgentCore ブラウザ (クラウドベース、ローカルブラウザは必要ありません) を使用します。
