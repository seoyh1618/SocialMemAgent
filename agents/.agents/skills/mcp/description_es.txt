Guía para crear y administrar servidores Model Context Protocol (MCP). Cómo agregar herramientas personalizadas. Casos de uso: (1) Cuando desea agregar herramientas personalizadas a Claude Code (2) Cuando se integra con servicios externos (3) Cuando desea crear herramientas de automatización específicas del proyecto (4) Cuando desea saber cómo configurar un servidor MCP. Ejemplos de activadores: "Crear un servidor MCP" "Agregar herramientas personalizadas" "Configuración de MCP" "API externa como herramienta" "MCP del ejecutor de pruebas"
