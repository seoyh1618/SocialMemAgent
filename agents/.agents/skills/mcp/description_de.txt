Leitfaden zum Erstellen und Verwalten von Model Context Protocol (MCP)-Servern. So fügen Sie benutzerdefinierte Tools hinzu. Anwendungsfälle: (1) Wenn Sie benutzerdefinierte Tools zu Claude Code hinzufügen möchten (2) Bei der Integration mit externen Diensten (3) Wenn Sie projektspezifische Automatisierungstools erstellen möchten (4) Wenn Sie wissen möchten, wie Sie einen MCP-Server einrichten. Trigger-Beispiele: „Einen MCP-Server erstellen“ „Benutzerdefinierte Tools hinzufügen“ „MCP-Konfiguration“ „Externe API als Tool“ „Test Runner MCP“
