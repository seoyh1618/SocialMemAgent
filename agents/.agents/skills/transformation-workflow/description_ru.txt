Практическое руководство по применению шести преобразований HUMMBL (перспектива, инверсия, композиция, декомпозиция, рекурсия, метасистемы). Включает, когда использовать каждое преобразование, шаблоны комбинаций, шаблоны анализа, форматы вывода, примеры из реальной жизни и распространенные ошибки. Необходим для эффективного применения ментальных моделей при решении проблем и анализе.
