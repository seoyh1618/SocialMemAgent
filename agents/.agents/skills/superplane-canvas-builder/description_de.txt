Entwerfen und erstellen Sie SuperPlane-Workflow-Canvas basierend auf den Anforderungen. Übersetzt Workflow-Beschreibungen in Canvas-YAML mit Triggern, Komponenten, Kanten und Ausdrücken. Verwenden Sie diese Option, wenn der Benutzer einen neuen Workflow erstellen, eine Leinwand erstellen, eine Pipeline entwerfen oder Komponenten vernetzen möchte. Löst auf „Canvas erstellen“, „Workflow erstellen“, „Pipeline entwerfen“ und „Automatisieren“ aus.
