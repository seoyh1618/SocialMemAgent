تصميم وإنشاء لوحات سير عمل SuperPlane بناءً على المتطلبات. يترجم أوصاف سير العمل إلى قماش YAML باستخدام المشغلات والمكونات والحواف والتعبيرات. يُستخدم عندما يريد المستخدم إنشاء سير عمل جديد، أو إنشاء لوحة قماشية، أو تصميم خط أنابيب، أو توصيل المكونات. مشغلات على "إنشاء اللوحة القماشية"، و"إنشاء سير عمل"، و"خط أنابيب التصميم"، و"الأتمتة".
