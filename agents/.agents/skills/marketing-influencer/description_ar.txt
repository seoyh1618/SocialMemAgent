متخصص في شراكة المؤثرين والمبدعين يغطي تحديد المؤثرين والتواصل وإدارة الحملات وبرامج المحتوى الذي ينشئه المستخدمون وبرامج المؤثرين التابعين واستراتيجية اقتصاد المبدعين. يُستخدم عندما يريد المستخدم العثور على مؤثرين، أو التخطيط لحملات مؤثرين، أو كتابة رسائل توعية، أو التفاوض على الشراكات، أو إنشاء برامج سفراء، أو تطوير إستراتيجيات UGC، أو إعداد برامج تابعة لمنشئي المحتوى، أو إدارة علاقات المؤثرين. مشغلات أيضًا لأصحاب النفوذ الجزئي، أو أصحاب النفوذ النانوي، أو سفراء العلامات التجارية، أو المحتوى المدعوم، أو تعاون المبدعين، أو عائد استثمار المؤثرين.
