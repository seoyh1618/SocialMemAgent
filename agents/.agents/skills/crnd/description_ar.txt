قم بجدولة وإدارة وظائف cron والمهام المجدولة لمرة واحدة والوظائف المتكررة باستخدام crnd CLI. يُستخدم عندما يريد المستخدم إعداد وظائف cronjobs، أو المهام المجدولة، أو المهام لمرة واحدة، أو المهام المتكررة/المتكررة، أو أتمتة المهام في أوقات محددة، أو إدارة عمليات الخلفية. تتضمن المشغلات طلبات مثل "تشغيل هذا كل يوم في الساعة 2 صباحًا"، أو "جدولة نسخة احتياطية"، أو "إعداد مهمة متكررة"، أو "تشغيل هذا مرة واحدة غدًا"، أو "إنشاء مهمة كرون".
