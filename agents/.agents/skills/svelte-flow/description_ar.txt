أنشئ محررات قائمة على العقد ومخططات تفاعلية وتصورات التدفق باستخدام Svelte Flow. يُستخدم عند إنشاء برامج تحرير سير العمل، أو مخططات تدفق البيانات، أو المخططات التنظيمية، أو الخرائط الذهنية، أو تصورات العمليات، أو محررات DAG، أو أي واجهة مستخدم لرسم بياني تفاعلي. يدعم العقد/الحواف المخصصة والتخطيطات (dagre والتسلسل الهرمي) والرسوم المتحركة والميزات المتقدمة مثل الاتصال القرب والحواف العائمة والتكبير/التصغير السياقي.
