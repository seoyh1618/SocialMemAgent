قم بإنشاء نقاط نهاية REST API آمنة لـ Frappe Framework v15 من خلال المصادقة والأذونات والتحقق المناسب. المشغلات: "إنشاء واجهة برمجة التطبيقات"، و"نقطة النهاية الجديدة"، و"واجهة برمجة تطبيقات frappe"، و"واجهة برمجة تطبيقات الراحة"، و"طريقة القائمة البيضاء"، و"/frappe-api". يُنشئ نقاط نهاية متوافقة مع واجهة برمجة التطبيقات v2 مع أفضل ممارسات التحقق من صحة النوع والأمان.
