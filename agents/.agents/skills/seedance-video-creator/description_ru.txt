Seedance 2.0: универсальный инструмент для профессиональной генерации раскадровки + создания видео. Используйте его, когда пользователи хотят создавать видеоролики для раскадровки, создавать клипы с помощью Seedance/Jimeng, им нужны профессиональные подсказки для раскадровки и прямой вывод видео. Поддерживает ссылки на несколько изображений, руководство по раскадровке, создание клипов на основе API и автоматическую загрузку.
