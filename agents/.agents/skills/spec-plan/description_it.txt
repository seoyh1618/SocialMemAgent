Quando l'utente desidera eseguire uno sviluppo basato sulle specifiche, DEVE utilizzare questa abilità: prima far convergere i requisiti e i vincoli in una specifica implementabile e rivedibile, quindi procedere all'implementazione. Questa abilità produce solo un file Spec scritto in /docs/specs/ ed effettua un commit git (commettendo solo quella Spec). Questa abilità non scrive codice di implementazione.
