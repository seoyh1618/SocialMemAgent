قم بتشغيل محفظة nova CLI بأمان للمصادقة وفحص الرصيد وإرسال الأموال والسحوبات وإدارة المفاتيح عبر الشبكة الرئيسية وشبكة الاختبار. يفرض تحليل JSON/TOON الحتمي، والتحقق من صحة رمز الخروج، والتأكيدات المالية، والتحقق من الشبكة، وقواعد صارمة للتعامل مع السرية. يُستخدم عند تنفيذ أتمتة محفظة nova، أو عمليات نقل العملات المستقرة، أو روابط المطالبة، أو الصادرات، أو أي معاملة blockchain عبر nova CLI.
