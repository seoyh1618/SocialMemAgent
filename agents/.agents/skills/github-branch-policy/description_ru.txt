Аудит управления ветвями репозитория GitHub и гигиены рабочих процессов. Используйте, когда вас просят просмотреть наборы правил, необходимые проверки статуса, ограничения обновления, поведение обхода/обновления, настройки удаления при слиянии, надежность рабочего процесса автоматического слияния, устаревшие ветки, регистрации фиктивных рабочих процессов или смещение политики ветвей.
