دليل لإنشاء وتحديث وإهمال خدمات RPC السحابية المختلطة في Sentry. يُستخدم عندما يُطلب منك "إضافة طريقة RPC"، أو "إنشاء خدمة RPC"، أو "خدمة سحابية مختلطة"، أو "نموذج RPC جديد"، أو "إيقاف أسلوب RPC"، أو "إزالة نقطة نهاية RPC"، أو "خدمة عبر الصومعة"، أو "RPC الإقليمي"، أو "خدمة صومعة التحكم". يغطي سقالات الخدمة، وتوقيعات الطريقة، ونماذج RPC، ومحللات المنطقة، والاختبار، وسير عمل الإهمال الآمن.
