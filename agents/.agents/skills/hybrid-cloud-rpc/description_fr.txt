Guide pour la création, la mise à jour et la dépréciation des services RPC de cloud hybride dans Sentry. À utiliser lorsqu'on vous demande "ajouter une méthode RPC", "créer un service RPC", "un service cloud hybride", "un nouveau modèle RPC", "déprécier la méthode RPC", "supprimer le point de terminaison RPC", "un service inter-silo", "un RPC régional" ou un "service de silo de contrôle". Couvre l'échafaudage de services, les signatures de méthodes, les modèles RPC, les résolveurs de régions, les tests et les workflows de dépréciation sécurisés.
