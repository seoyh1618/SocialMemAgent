Schreiben Sie ideale E2E-Tests und überprüfen Sie vorhandene. Konzentriert sich ausschließlich auf End-to-End-Tests – vollständige Benutzerreisen durch einen echten Browser. Nicht für Unit- oder Integrationstests. Two modes: generate new E2E tests from feature analysis, or review existing E2E tests for antipatterns, flakiness, and quality issues. Framework-unabhängig, basierend auf Best Practices von Beck, Fowler, Cohn, Rauch und Google.
