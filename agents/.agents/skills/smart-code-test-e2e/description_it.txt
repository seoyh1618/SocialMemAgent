Scrivi test E2E ideali e rivedi quelli esistenti. Incentrato esclusivamente sui test end-to-end: percorsi completi dell'utente attraverso un browser reale. Non per test unitari o di integrazione. Due modalità: generazione di nuovi test E2E dall'analisi delle funzionalità o revisione dei test E2E esistenti per antipattern, instabilità e problemi di qualità. Indipendente dal framework, basato sulle best practice di Beck, Fowler, Cohn, Rauch e Google.
