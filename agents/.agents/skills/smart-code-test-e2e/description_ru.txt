Напишите идеальные тесты E2E и просмотрите существующие. Ориентирован исключительно на сквозное тестирование — полное взаимодействие пользователя с реальным браузером. Не для модульных или интеграционных тестов. Два режима: создание новых тестов E2E на основе анализа функций или проверка существующих тестов E2E на наличие антишаблонов, нестабильности и проблем с качеством. Не зависит от платформы, основан на передовом опыте Бека, Фаулера, Кона, Рауха и Google.
