Úselo solo cuando el usuario solicite explícitamente "verificar/optimizar las citas en el cuerpo de la revisión `{topic}_review.tex`" o "ejecutar check-review-alignment".

A través de la comprensión semántica de la IA del anfitrión, verifique cada cita con el contenido fuente; **solo cuando se encuentre un error fatal en la cita**, realice la reescritura mínima de la "oración que contiene la cita" y reutilice el script de representación `sistematic-literature-review` para generar PDF/Word (el script en sí no llama a una API LLM localmente).

Principio básico: **no cambiar por cambiar**. Si no está seguro de si se trata de un error fatal, déjelo como está y marque una advertencia en el informe.

⚠️ No aplica para:
- Usuarios que simplemente desean generar texto de revisión sistemática (use revisión de literatura sistemática)
- Usuarios que solo desean agregar/verificar entradas de BibTeX (use el flujo de trabajo de administración de dorsales dedicado)
