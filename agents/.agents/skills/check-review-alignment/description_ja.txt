ユーザーが「レビュー本文 `{topic}_review.tex` の引用をチェック/最適化する」または「check-review-alignment を実行する」ことを明示的に要求した場合にのみ使用します。

ホスト AI の意味理解を通じて、各引用をソース コンテンツと照合して検証します。 **致命的な引用エラーが見つかった場合のみ**、「引用を含む文」の最小限の書き換えを実行し、`systematic-literature-review` レンダリング スクリプトを再利用して PDF/Word を出力します (スクリプト自体はローカルで LLM API を呼び出しません)。

基本原則: **変更のために変更しない**。致命的なエラーかどうか不明な場合は、そのままにしてレポートに警告を付けます。

⚠️以下には適用されません:
- 単に系統的レビューのテキストを生成したいユーザー (systematic-literature-review を使用)
- BibTeX エントリの追加/検証のみを希望するユーザー (専用の Bib 管理ワークフローを使用)
