قم بالتحميل بشكل استباقي عندما تتضمن المهمة إنشاء ميزة كاملة عبر طبقات متعددة. يُستخدم عندما يقول المستخدم "إنشاء ميزة"، أو "إضافة ملفات تعريف المستخدمين"، أو "إنشاء لوحة معلومات"، أو أي طلب يشمل قاعدة البيانات وواجهة برمجة التطبيقات وواجهة المستخدم والاختبارات. ينسق العمل متعدد الوكلاء بشكل تسلسلي: المخطط وعمليات الترحيل ونقاط نهاية واجهة برمجة التطبيقات ومكونات واجهة المستخدم والاختبارات والمراجعة. يتعامل محرك وقت التشغيل مع سلاسل WRFC تلقائيًا عبر توجيهات <gv>. يعالج ترتيب التبعية ومشاركة النوع عبر الطبقات.
