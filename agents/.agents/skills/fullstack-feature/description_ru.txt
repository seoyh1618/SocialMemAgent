Загружайте ПРОАКТИВНО, если задача предполагает создание полноценного объекта на нескольких уровнях. Используйте, когда пользователь говорит «создать функцию», «добавить профили пользователей», «создать панель мониторинга» или любой запрос, охватывающий базу данных, API, пользовательский интерфейс и тесты. Последовательно организует работу нескольких агентов: схему и миграцию, конечные точки API, компоненты пользовательского интерфейса, тесты и проверку. Механизм выполнения автоматически обрабатывает цепочки WRFC с помощью директив <gv>. Управляет упорядочиванием зависимостей и обменом типами между уровнями.
