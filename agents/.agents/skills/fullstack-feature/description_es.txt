Cargue PROACTIVAMENTE cuando la tarea implique crear una función completa en varias capas. Úselo cuando el usuario diga "crear una función", "agregar perfiles de usuario", "crear un panel" o cualquier solicitud que abarque base de datos, API, UI y pruebas. Orquesta el trabajo de múltiples agentes de forma secuencial: esquema y migraciones, puntos finales de API, componentes de UI, pruebas y revisión. El motor de ejecución maneja las cadenas WRFC automáticamente mediante directivas <gv>. Maneja el orden de dependencia y el intercambio de tipos entre capas.
