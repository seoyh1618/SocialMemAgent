Laden Sie PROAKTIV, wenn die Aufgabe den Aufbau eines vollständigen Features über mehrere Ebenen hinweg umfasst. Wird verwendet, wenn der Benutzer sagt: „Funktion erstellen“, „Benutzerprofile hinzufügen“, „Dashboard erstellen“ oder eine Anfrage, die Datenbank, API, Benutzeroberfläche und Tests umfasst. Orchestriert die Arbeit mehrerer Agenten nacheinander: Schema und Migrationen, API-Endpunkte, UI-Komponenten, Tests und Überprüfung. Die Laufzeit-Engine verarbeitet WRFC-Ketten automatisch über <gv>-Direktiven. Verwaltet die Abhängigkeitsreihenfolge und die schichtübergreifende Typfreigabe.
