Chargez de manière PROACTIVE lorsque la tâche implique la création d’une fonctionnalité complète sur plusieurs couches. À utiliser lorsque l'utilisateur dit « créer une fonctionnalité », « ajouter des profils utilisateur », « créer un tableau de bord » ou toute demande couvrant une base de données, une API, une interface utilisateur et des tests. Orchestre le travail multi-agents de manière séquentielle : schéma et migrations, points de terminaison d'API, composants d'interface utilisateur, tests et révision. Le moteur d'exécution gère automatiquement les chaînes WRFC via les directives <gv>. Gère l’ordre des dépendances et le partage de types entre couches.
