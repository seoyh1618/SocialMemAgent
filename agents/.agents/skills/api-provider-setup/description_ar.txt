قم بإضافة وتكوين موفري ترحيل واجهة برمجة التطبيقات (API) التابعين لجهات خارجية إلى OpenClaw. استخدم هذه المهارة عندما يحتاج المستخدمون إلى إضافة موفري واجهة برمجة التطبيقات (API) الجدد، أو تكوين المرحلات، أو إعداد نقاط نهاية النموذج المخصص. يدعم تنسيقات API المتوافقة مع Anthropic والمتوافقة مع OpenAI.
