Service Worker API 実装ガイド — 登録、ライフサイクル管理、キャッシュ戦略、プッシュ通知、バックグラウンド同期。次の場合に使用します: (1) Service Worker ファイル (sw.js) の作成または変更、(2) オフライン ファースト キャッシュの実装 (キャッシュ ファースト、ネットワーク ファースト、stale-while-revalidate)、(3) プッシュ通知またはバックグラウンド同期の設定、(4) Service Worker の登録、スコープ、または更新の問題のデバッグ、(5) ナビゲーション プリロードの実装、(6) ユーザーによる「Service Worker」、「sw.js」、「オフライン サポート」の言及、 「キャッシュ戦略」、「プッシュ通知」、「バックグラウンド同期」、「ワークボックス代替」、または「PWA キャッシュ」。
