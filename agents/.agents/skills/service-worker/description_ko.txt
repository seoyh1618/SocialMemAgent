Service Worker API 구현 가이드 - 등록, 수명주기 관리, 캐싱 전략, 푸시 알림 및 백그라운드 동기화. 다음 경우에 사용하십시오: (1) 서비스 워커 파일(sw.js) 생성 또는 수정, (2) 오프라인 우선 캐싱 구현(캐시 우선, 네트워크 우선, 오래된 재검증 중), (3) 푸시 알림 또는 백그라운드 동기화 설정, (4) 서비스 워커 등록, 범위 또는 업데이트 문제 디버깅, (5) 탐색 사전 로드 구현, (6) 사용자가 '서비스 워커', 'sw.js', '오프라인 지원', '캐시 전략', '푸시 알림'을 언급하는 경우, '백그라운드 동기화', '워크박스 대체' 또는 'PWA 캐싱'.
