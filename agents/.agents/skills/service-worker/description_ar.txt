دليل تنفيذ Service Worker API - التسجيل وإدارة دورة الحياة واستراتيجيات التخزين المؤقت والإشعارات المباشرة والمزامنة في الخلفية. يُستخدم عندما: (1) إنشاء أو تعديل ملفات عامل الخدمة (sw.js)، (2) تنفيذ التخزين المؤقت دون اتصال أولاً (ذاكرة التخزين المؤقت أولاً، الشبكة أولاً، التي لا معنى لها أثناء إعادة التحقق)، (3) إعداد إشعارات الدفع أو مزامنة الخلفية، (4) تصحيح أخطاء تسجيل عامل الخدمة أو النطاق أو مشكلات التحديث، (5) تنفيذ التحميل المسبق للتنقل، (6) يذكر المستخدم "عامل الخدمة"، "sw.js"، "الدعم دون اتصال"، "استراتيجية التخزين المؤقت"، "إشعار الدفع" أو "مزامنة الخلفية" أو "بديل مربع العمل" أو "التخزين المؤقت لـ PWA".
