Загружайте ПРОАКТИВНО при декомпозиции запроса пользователя на параллельную работу агента. Используйте, когда пользователь говорит «создайте это», «реализуйте эту функцию» или любой запрос, требующий одновременной работы нескольких агентов. Управляет декомпозицией задачи на распараллеливаемые блоки, назначением агентов с сопоставлением навыков, построением графа зависимостей и агрегированием результатов. Механизм выполнения автоматически осуществляет координацию цепочки WRFC с помощью директив <gv>.
