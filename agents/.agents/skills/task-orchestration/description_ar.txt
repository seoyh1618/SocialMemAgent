قم بالتحميل بشكل استباقي عند تحليل طلب المستخدم إلى عمل وكيل موازي. يُستخدم عندما يقول المستخدم "قم ببناء هذا"، أو "نفذ هذه الميزة"، أو أي طلب يتطلب وكلاء متعددين يعملون بشكل متزامن. يوجه تحليل المهام إلى وحدات قابلة للتوازي، وتعيين الوكيل مع مطابقة المهارات، وبناء الرسم البياني للتبعية، وتجميع النتائج. يتعامل محرك وقت التشغيل مع تنسيق سلسلة WRFC تلقائيًا عبر توجيهات <gv>.
