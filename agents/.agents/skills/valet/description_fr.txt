À utiliser lorsque l'utilisateur souhaite gérer les agents Valet, les canaux, les connecteurs, les organisations ou les secrets via la CLI Valet. Gère la création, le déploiement, la liaison, le démontage et tous les flux de travail en plusieurs étapes. À utiliser également lorsqu'on vous demande de "créer un agent", "déployer un agent", "concevoir un agent", "me construire un agent qui...", "créer un connecteur", "configurer un webhook", ou tout ce qui implique la plateforme Valet ou toute demande de création et de déploiement d'agents IA. À utiliser également lorsqu'on vous demande « apprendre de cette session », « capturer ce flux de travail », « enregistrer ceci en tant qu'agent », « rendre cela répétable » ou lors de l'écriture de fichiers SOUL.md.
