Next.js を使用してプログレッシブ Web アプリを構築します: サービス ワーカー、オフライン サポート、キャッシュ戦略、プッシュ通知、インストール プロンプト、Web アプリ マニフェスト。 PWA の作成、オフライン機能の追加、Service Worker の構成、プッシュ通知の実装、インストール プロンプトの処理、または PWA パフォーマンスの最適化を行うときに使用します。トリガー: PWA、プログレッシブ Web アプリ、Service Worker、オフライン、キャッシュ戦略、Web マニフェスト、プッシュ通知、インストール可能なアプリ、Serwist、next-pwa、ワークボックス、バックグラウンド同期。
