Помощник по обслуживанию клиентов в сфере электронной коммерции с искусственным интеллектом, поддерживающий запросы заказов, рекомендации по продуктам и обработку билетов. Он может использовать Playwright MCP для автоматического управления серверной частью электронной коммерции и запроса реального статуса заказа.
