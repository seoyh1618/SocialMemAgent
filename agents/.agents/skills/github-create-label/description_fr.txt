Créez des étiquettes de problème GitHub à l’aide de la sélection intelligente d’outils. Exploite automatiquement les outils par ordre de priorité (GitHub MCP Server → GitHub CLI → REST API). À utiliser lorsque vous devez créer de nouvelles étiquettes dans un référentiel avec une personnalisation facultative (nom, description, couleur). Inclut la prise en charge de la détection des méthodes d'authentification disponibles et un repli gracieux entre les outils.
