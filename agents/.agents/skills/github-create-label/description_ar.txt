أنشئ تسميات مشكلات GitHub باستخدام التحديد الذكي للأدوات. الاستفادة تلقائيًا من الأدوات بترتيب الأسبقية (GitHub MCP Server → GitHub CLI → REST API). استخدمه عندما تحتاج إلى إنشاء تسميات جديدة في مستودع مع تخصيص اختياري (الاسم والوصف واللون). يتضمن دعمًا لاكتشاف طرق المصادقة المتاحة والرجوع السهل بين الأدوات.
