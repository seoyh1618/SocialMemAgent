Создавайте метки проблем GitHub, используя интеллектуальный выбор инструментов. Автоматически использует инструменты в порядке приоритета (GitHub MCP Server → GitHub CLI → REST API). Используйте, когда вам нужно создать новые метки в репозитории с дополнительной настройкой (имя, описание, цвет). Включает поддержку обнаружения доступных методов аутентификации и плавный переход между инструментами.
