قم بقراءة واستكشاف أدلة/ملفات التعليمات البرمجية التي يقدمها المستخدم بعناية ودقة، وتحليل مقتطفات التعليمات البرمجية المقابلة وتقسيمها للرجوع إليها، مع الدقة وصولاً إلى المستوى الوظيفي. قم باستخراج المقتطفات فقط دون تغيير محتوى التعليمات البرمجية أو بنية تخطيط الصفحة، ووضعها في الدليل ./fragment-extractor. قم بإنشاء ملفات قالب مقتطف التعليمات البرمجية وفقًا لتنسيق القالب المحدد، مما يضمن التكامل الوظيفي والتطبيق العملي لمقتطفات التعليمات البرمجية.
