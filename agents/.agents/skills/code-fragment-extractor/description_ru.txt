Внимательно и тщательно прочитайте и изучите каталоги/файлы кода, предоставленные пользователем, проанализируйте и разложите соответствующие фрагменты кода для справки с детализацией до функционального уровня. Извлекайте только фрагменты, не изменяя содержимое кода или структуру макета страницы, и помещайте их в каталог ./fragment-extractor. Создавайте файлы шаблонов фрагментов кода в соответствии с указанным форматом шаблона, обеспечивая функциональную целостность и практичность фрагментов кода.
