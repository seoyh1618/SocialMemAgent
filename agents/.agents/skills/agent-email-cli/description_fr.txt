Utilisez la CLI de messagerie agent pour créer des boîtes de réception jetables, rechercher de nouveaux messages, récupérer les détails complets des messages et gérer les profils de boîtes aux lettres locales. À utiliser lorsque l'utilisateur a besoin d'un accès à la boîte de réception de courrier électronique depuis un terminal pour les flux de travail LLM ou d'automatisation des agents.
