قم بتشغيل واجهة سطر الأوامر للبريد الإلكتروني للوكيل لإنشاء صناديق وارد يمكن التخلص منها، واستقصاء البريد الجديد، واسترداد تفاصيل الرسالة الكاملة، وإدارة ملفات تعريف صندوق البريد المحلي. يُستخدم عندما يحتاج المستخدم إلى الوصول إلى صندوق البريد الإلكتروني المستند إلى المحطة الطرفية لسير عمل LLM أو أتمتة الوكيل.
