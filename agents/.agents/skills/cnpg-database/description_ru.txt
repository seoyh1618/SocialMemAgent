CloudNative-PG (CNPG) Управление базами данных PostgreSQL для домашней лаборатории Kubernetes.
Охватывает кластер общей платформы, выделенные кластеры приложений, цепочки учетных данных и мониторинг.

Используйте, когда: (1) добавляется новая база данных для приложения, (2) создается выделенный кластер CNPG,
(3) Настройка учетных данных базы данных и репликации между пространствами имен, (4) Настройка PgBouncer
пулеры, (5) Отладка подключения к базе данных или работоспособности кластера CNPG, (6) Добавление PostgreSQL
расширения для специализированных рабочих нагрузок.
