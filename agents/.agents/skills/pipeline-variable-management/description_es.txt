Una guía completa para la gestión de variables de canalización, que cubre el ciclo de vida de las variables (creación, inicialización, actualización, almacenamiento, transferencia, consulta) y extensiones de campos variables (definición de campo, extensión de tipo, alcance, herencia). Úselo al desarrollar características variables, manejar el paso de variables, extender campos variables o depurar problemas de variables.
