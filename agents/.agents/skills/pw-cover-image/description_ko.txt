기사 콘텐츠에 대한 멋진 표지 이미지를 생성하기 위한 전문 도구입니다.

핵심 기능:
- 기사 내용을 분석하고 가장 적합한 시각적 스타일을 자동으로 선택합니다.
- 손으로 그린 스타일로 고품질 표지 이미지 생성
- 19가지 사전 설정 스타일 지원(우아함, 청사진, 대담한 편집 등)
- 다양한 화면비 지원(2.35:1 시네마틱, 16:9 와이드스크린, 1:1 정사각형)
- 핵심 테마 자동 추출 및 타이틀 텍스트 생성

사용 시기:
- 사용자가 "표지 이미지 생성", "기사 표지 만들기" 또는 "표지 만들기"를 명시적으로 요청합니다.
- 사용자가 기사 내용을 제공하고 첨부된 이미지가 필요합니다.
- 사용자는 블로그, WeChat 기사 또는 소셜 미디어에 대한 표지가 필요합니다.

해당 사항 없음:
- 사용자는 단순히 커버 만드는 방법을 묻습니다. (단지 조언만 제공)
- 사용자가 기존 이미지를 편집해야 합니다(이미지 편집 도구 사용).
- 표지가 아닌 이미지는 사용자가 생성해야 합니다. (일반 이미지 생성 도구 사용)
