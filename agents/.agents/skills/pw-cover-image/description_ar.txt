أداة متخصصة لإنشاء صور غلاف مذهلة لمحتوى المقالة.

الميزات الأساسية:
- يحلل محتوى المقالة ويختار تلقائيًا النمط البصري الأكثر ملاءمة
- إنشاء صور غلاف عالية الجودة بأسلوب مرسوم باليد
- يدعم 19 نمطًا محددًا مسبقًا (أنيق، مخطط، تحريري جريء، إلخ.)
- يدعم نسب العرض إلى الارتفاع المتعددة (2.35:1 سينمائي، 16:9 شاشة عريضة، 1:1 مربع)
- يستخرج السمات الأساسية تلقائيًا وينشئ نص العنوان

متى تستخدم:
- يطلب المستخدم صراحةً "إنشاء صورة غلاف" أو "إنشاء غلاف للمقالة" أو "إنشاء غلاف"
- يقدم المستخدم محتوى المقالة ويحتاج إلى صور مصاحبة
- يغطي احتياجات المستخدم للمدونات أو مقالات WeChat أو وسائل التواصل الاجتماعي

لا ينطبق:
- يسأل المستخدم ببساطة عن كيفية عمل غلاف (فقط قدم النصيحة)
- يحتاج المستخدم إلى تحرير الصور الموجودة (استخدم أدوات تحرير الصور)
- يحتاج المستخدم إلى إنشاء صور غير الغلاف (استخدم أدوات إنشاء الصور العامة)
