جلب إشعارات GitHub غير المقروءة للعلاقات العامة المفتوحة حيث يتم طلب المراجعة من فريق محدد أو يتم فتحها بواسطة أحد أعضاء الفريق. استخدم عندما يُطلب منك "العثور على الممثلين الدائمين الذين أحتاج إلى مراجعتهم"، أو "إظهار طلبات المراجعة الخاصة بي"، أو "ما يحتاج إلى مراجعتي"، أو "جلب طلبات مراجعة GitHub"، أو "التحقق من قائمة انتظار مراجعة الفريق".
