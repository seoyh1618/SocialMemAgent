Получайте непрочитанные уведомления GitHub для открытых запросов на проверку, если проверка запрошена у указанной команды или открыта членом команды. Используйте, когда вас просят «найти запросы на проверку, которые мне нужно просмотреть», «показать мои запросы на проверку», «что требует моей проверки», «получить запросы на проверку GitHub» или «проверить очередь командной проверки».
