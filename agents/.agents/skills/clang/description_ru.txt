Навыки компиляции Clang/LLVM для проектов C/C++. Используйте при работе с clang или clang++ для диагностики, инструментов очистки, замечаний по оптимизации, статического анализа с clang-tidy, LTO через lld или при переходе с GCC на Clang. Активируется по запросам о флагах clang, clang-tidy, clang-формате, улучшенных сообщениях об ошибках, цепочках инструментов Apple/FreeBSD или оптимизации, специфичной для LLVM. Охватывает выбор флагов, диагностическую настройку и интеграцию с инструментами LLVM.
