مهارة مترجم Clang/LLVM لمشاريع C/C++. يُستخدم عند العمل مع clang أو clang++ للتشخيص، وأدوات التعقيم، وملاحظات التحسين، والتحليل الثابت باستخدام clang-tidy، وLTO عبر lld، أو عند الترحيل من دول مجلس التعاون الخليجي إلى Clang. يتم التنشيط عند الاستعلامات حول إشارات clang، أو clang-tidy، أو تنسيق clang، أو رسائل الخطأ الأفضل، أو سلاسل أدوات Apple/FreeBSD، أو التحسينات الخاصة بـ LLVM. يغطي اختيار العلم وضبط التشخيص والتكامل مع أدوات LLVM.
