Compétence de compilateur Clang/LLVM pour les projets C/C++. À utiliser lorsque vous travaillez avec clang ou clang++ pour les diagnostics, l'instrumentation de désinfection, les remarques d'optimisation, l'analyse statique avec clang-tidy, LTO via lld ou lors de la migration de GCC vers Clang. S'active sur les requêtes concernant les indicateurs clang, clang-tidy, clang-format, de meilleurs messages d'erreur, les chaînes d'outils Apple/FreeBSD ou les optimisations spécifiques à LLVM. Couvre la sélection des indicateurs, le réglage des diagnostics et l'intégration avec les outils LLVM.
