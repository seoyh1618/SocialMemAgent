Eine einzige Quelle der Wahrheit und Bibliothekar für ALLE offiziellen Claude-Dokumentationen. Verwaltet die Speicherung, das Scraping, die Erkennung und die Auflösung der lokalen Dokumentation. Wird zum Suchen, Lokalisieren, Durchsuchen oder Lösen von Claude-Dokumentationen verwendet. Entdecken von Dokumenten anhand von Schlüsselwörtern, Kategorien, Tags oder Abfragen in natürlicher Sprache; Scraping aus Sitemaps oder Dokumentenkarten; Verwalten von Indexmetadaten (Schlüsselwörter, Tags, Aliase); oder Index aus dem Dateisystem neu erstellen. Führen Sie Skripte aus, um Dokumentation zu durchsuchen, zu finden und aufzulösen. Behandelt die doc_id-Auflösung, die Schlüsselwortsuche, Abfragen in natürlicher Sprache, die Filterung von Kategorien/Tags, die Alias-Auflösung, das Parsen von sitemap.xml, die Verarbeitung von Dokumentenkarten, die Extraktion von Markdown-Unterabschnitten für den internen Gebrauch, die Hash-basierte Drifterkennung und die umfassende Indexpflege.
