Используйте, когда пользователь явно запрашивает «контроль качества предложения/контроль качества/предварительную проверку качества/проверку подлинности цитаты/проверку длины и структуры». Выполнять контроль качества предложений NSFC только для чтения: запускать параллельные многопоточные независимые проверки на предмет жесткого стиля, рисков поддельного/неправильного цитирования, распределения длины и разделов, логической ясности и т. д., а также выводить стандартизированный отчет по контролю качества; промежуточные файлы по умолчанию архивируются в «скрытой рабочей области внутри папки доставки (.nsfc-qc/)» и остаются совместимыми с устаревшим `.nsfc-qc/`.
