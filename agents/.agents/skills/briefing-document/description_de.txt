Erstellt umfassende Briefing-Dokumente, die Quellen zu für die Geschäftsleitung geeigneten Berichten zusammenfassen. Erstellt eine Zusammenfassung mit kritischen Erkenntnissen, einer detaillierten thematischen Analyse mit Belegen und objektiven Schlussfolgerungen. Verwenden Sie diese Option, wenn Sie ein Briefing erstellen, Forschungsergebnisse zusammenfassen, Quellen zusammenfassen, eine Zusammenfassung verfassen oder fragen: „Erstellen Sie ein Briefing-Dokument.“
