قم بالتحميل بشكل استباقي عندما تتضمن المهمة الامتثال لإمكانية الوصول أو التصميم الشامل. يُستخدم عندما يقول المستخدم "التحقق من إمكانية الوصول"، أو "التدقيق في WCAG"، أو "إصلاح مشكلات قارئ الشاشة"، أو "إضافة التنقل عبر لوحة المفاتيح"، أو "التحقق من تباين الألوان". يغطي توافق WCAG 2.1 AA عبر HTML الدلالي، وأنماط ARIA، والتنقل عبر لوحة المفاتيح، ودعم قارئ الشاشة، ونسب تباين الألوان، وإمكانية الوصول إلى النماذج، وبدائل الوسائط، وإدارة التركيز. ينتج تقارير تدقيق منظمة مع تصنيفات الخطورة.
