Загружайте ПРОАКТИВНО, если задача связана с обеспечением доступности или инклюзивным дизайном. Используйте, когда пользователь говорит «проверить доступность», «проверить WCAG», «исправить проблемы с программой чтения с экрана», «добавить навигацию с помощью клавиатуры» или «проверить цветовой контраст». Охватывает соответствие WCAG 2.1 AA в отношении семантического HTML, шаблонов ARIA, навигации с помощью клавиатуры, поддержки чтения с экрана, коэффициентов цветовой контрастности, доступности форм, альтернатив мультимедиа и управления фокусом. Создает структурированные аудиторские отчеты с уровнями серьезности.
