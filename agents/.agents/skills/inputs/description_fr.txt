Préparer les entrées pour les méthodes MTHDS. À utiliser lorsque l'utilisateur dit « préparer les entrées », « créer des entrées », « utiliser mes fichiers », « générer des données de test », « modèle », « synthétiser les entrées », « simuler les entrées », « J'ai un PDF/image/document à utiliser », « créer des exemples de données » ou souhaite créer inputs.json pour exécuter un pipeline .mthds. Gère les fichiers fournis par l'utilisateur, la génération de données synthétiques, les modèles d'espace réservé et les approches mixtes. Par défaut, le mode automatique est activé.
