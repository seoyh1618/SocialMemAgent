Подготовьте входные данные для методов MTHDS. Используйте, когда пользователь говорит «подготовить входные данные», «создать входные данные», «использовать мои файлы», «сгенерировать тестовые данные», «шаблон», «синтезировать входные данные», «макетировать входные данные», «у меня есть PDF/изображение/документ для использования», «создать образец данных» или он хочет создать inputs.json для запуска конвейера .mthds. Обрабатывает предоставленные пользователем файлы, генерацию синтетических данных, шаблоны-заполнители и смешанные подходы. По умолчанию используется автоматический режим.
