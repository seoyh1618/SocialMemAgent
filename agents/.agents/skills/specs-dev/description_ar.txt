خطط لسير عمل التطوير أولاً مع بوابات المراجعة. يُستخدم عند تنفيذ الميزات، أو إعادة البناء، أو أي مهمة تتطلب تخطيطًا منظمًا، وتنفيذًا متكررًا مع المراجعات، والالتزامات الواضحة. يتم تشغيلها عند طلبات مثل "تنفيذ الميزة X"، أو "التخطيط والإنشاء"، أو "التطوير القائم على المواصفات"، أو عندما يريد المستخدم إجراء تغييرات منضبطة ومراجعة للتعليمات البرمجية.
