Cree, instale, inicie y depure aplicaciones HarmonyOS/OpenHarmony utilizando la cadena de herramientas DevEco Studio. Úselo cuando Claude necesite: (1) crear aplicaciones OHOS con hvigorw, (2) instalar archivos HAP en dispositivos a través de hdc, (3) iniciar o depurar aplicaciones OHOS, (4) analizar pilas de fallas con hstack, (5) tomar capturas de pantalla del dispositivo o (6) detectar el entorno de DevEco Studio y las herramientas SDK. Detecta automáticamente la instalación de DevEco Studio y configura JAVA_HOME, PATH y cadena de herramientas.
