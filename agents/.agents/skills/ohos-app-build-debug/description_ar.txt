قم ببناء وتثبيت وتشغيل وتصحيح تطبيقات HarmonyOS/OpenHarmony باستخدام سلسلة أدوات DevEco Studio. يُستخدم عندما يحتاج Claude إلى: (1) إنشاء تطبيقات OHOS باستخدام hvigorw، (2) تثبيت ملفات HAP على الأجهزة عبر hdc، (3) تشغيل تطبيقات OHOS أو تصحيح أخطاءها، (4) تحليل مكدسات الأعطال باستخدام hstack، (5) التقاط لقطات شاشة للجهاز، أو (6) اكتشاف بيئة DevEco Studio وأدوات SDK. يكتشف تلقائيًا تثبيت DevEco Studio ويقوم بتكوين JAVA_HOME وPATH وtoolchain.
