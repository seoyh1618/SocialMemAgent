DevEco Studio ツールチェーンを使用して、HarmonyOS/OpenHarmony アプリケーションを構築、インストール、起動、デバッグします。クロードが次のことを行う必要がある場合に使用します: (1) hvigorw を使用して OHOS アプリを構築する、(2) hdc を介して HAP ファイルをデバイスにインストールする、(3) OHOS アプリケーションを起動またはデバッグする、(4) hstack を使用してクラッシュ スタックを解析する、(5) デバイスのスクリーンショットを撮る、または (6) DevEco Studio 環境と SDK ツールを検出する。 DevEco Studio のインストールを自動的に検出し、JAVA_HOME、PATH、およびツールチェーンを構成します。
