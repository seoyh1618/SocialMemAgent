Crea, installa, avvia ed esegui il debug delle applicazioni HarmonyOS/OpenHarmony utilizzando la toolchain DevEco Studio. Da utilizzare quando Claude deve: (1) creare app OHOS con hvigorw, (2) installare file HAP sui dispositivi tramite hdc, (3) avviare o eseguire il debug di applicazioni OHOS, (4) analizzare stack di arresti anomali con hstack, (5) acquisire screenshot del dispositivo o (6) rilevare l'ambiente DevEco Studio e gli strumenti SDK. Rileva automaticamente l'installazione di DevEco Studio e configura JAVA_HOME, PATH e toolchain.
