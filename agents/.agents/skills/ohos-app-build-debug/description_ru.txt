Создавайте, устанавливайте, запускайте и отлаживайте приложения HarmonyOS/OpenHarmony с помощью набора инструментов DevEco Studio. Используйте, когда Клоду необходимо: (1) создать приложения OHOS с помощью hvigorw, (2) установить файлы HAP на устройства через hdc, (3) запустить или отладить приложения OHOS, (4) проанализировать стеки сбоев с помощью hstack, (5) сделать снимки экрана устройства или (6) обнаружить среду DevEco Studio и инструменты SDK. Автоматически обнаруживает установку DevEco Studio и настраивает JAVA_HOME, PATH и набор инструментов.
