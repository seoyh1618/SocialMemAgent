AI 개인 메모리 시스템의 메모리 기록 기능. 대화에서 주요 정보를 자동으로 추출하여 적절한 수준에 기록합니다.  
사용 사례:  
(1) 사용자가 "메모리 시스템에 녹음", "이것을 기억해" 또는 "이 대화를 저장"이라고 말하는 경우  
(2) 중요한 사건, 결정, 선호 표현이 감지된 경우  
(3) 사용자가 중요한 작업을 완료하거나 결정을 내릴 때.  

이 스킬은 L1 Context Layer, L2 Behavior Layer, L3 Cognitive Layer에 기록할지 여부를 자동으로 결정하거나 L4 Core Layer 업데이트를 제안합니다.
