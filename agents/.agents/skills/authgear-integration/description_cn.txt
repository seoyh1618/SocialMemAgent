将 Authgear 身份验证集成到 Web、移动和后端应用程序中。当开发人员请求“添加身份验证”、“集成 Authgear”、“实现登录/注销”、“验证 JWT 令牌”、“向 React/React Native/Vue/Flutter/Android/iOS/后端应用添加身份验证”或提及 Authgear SDK 设置时使用。支持 React SPA、React Native、Android、iOS、Flutter、Vue.js、Next.js、Ionic（前端）和 Python、Node.js、Go、Java、PHP、ASP.NET（后端 JWT 验证），并具有自动依赖项安装、配置、身份验证流程、受保护的路由、用户配置文件页面和 API 集成模式。
