Authgear 認証を Web、モバイル、バックエンド アプリケーションに統合します。開発者が「認証の追加」、「Authgear の統合」、「ログイン/ログアウトの実装」、「JWT トークンの検証」、「React/React Native/Vue/Flutter/Android/iOS/バックエンド アプリへの認証の追加」を要求する場合、または Authgear SDK の設定について言及する場合に使用します。フロントエンドでは React SPA、React Native、Android、iOS、Flutter、Vue.js、Next.js、Ionic、バックエンドでは Python、Node.js、Go、Java、PHP、ASP.NET をサポートし、依存関係の自動インストール、構成、認証フロー、保護されたルート、ユーザー プロファイル ページ、API 統合パターンによる JWT 検証をサポートします。
