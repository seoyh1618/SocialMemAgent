多代理協作 Android 到 HarmonyOS 程式碼遷移工作流程（迭代）。當使用者需要將 Android 專案移轉到 HarmonyOS 時觸發，提供： 1）Analyzer Agent 掃描程式碼結構 2）Planner Agent 起草遷移計畫 3）Translator Agent 執行程式碼轉換 4）Validator Agent 使用 ohos-app-build-debug 技能編譯、設定和設定裝置上的首屏功能 5） Comparator Agent 檢查是否有缺失功能 7) UI Comparator Agent 驗證 UI 一致性 8) 將未完成的部分回饋給 Planner Agent 進行進一步迭代。包括程式碼分析腳本、API 映射表、元件轉換模式和自動驗證工具。觸發關鍵字：Android遷移到鴻溝、Android遷移、落實遷移、Java/Kotlin到ArkTS、Activity到Page、功能對比、UI對比。
