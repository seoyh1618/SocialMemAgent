Многоагентный совместный рабочий процесс миграции кода с Android на HarmonyOS (итеративный). Запускается, когда пользователю необходимо перенести проект Android на HarmonyOS, обеспечивая: 1) Агент анализатора сканирует структуру кода 2) Агент планировщика составляет проект плана миграции 3) Агент переводчика выполняет преобразование кода 4) Агент валидатора компилирует и проверяет с помощью навыка ohos-app-build-debug, упаковывает приложение и проверяет первый экран на устройстве 5) Агент тестировщика проверяет функциональность (запускается после миграции каждого модуля) 6) После миграции всех модулей функция Агент сравнения проверяет наличие отсутствующих функций. 7) Агент сравнения пользовательского интерфейса проверяет согласованность пользовательского интерфейса. 8) Передает незавершенные части обратно агенту планировщика для дальнейшей итерации. Включает сценарии анализа кода, таблицы сопоставления API, шаблоны преобразования компонентов и инструменты автоматической проверки. Ключевые слова-триггеры: миграция Android на HarmonyOS, миграция Android, миграция HarmonyOS, Java/Kotlin в ArkTS, активность на страницу, сравнение функций, сравнение пользовательского интерфейса.
