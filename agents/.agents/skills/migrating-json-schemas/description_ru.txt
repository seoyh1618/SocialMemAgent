Переносит схемы JSON между черновыми версиями для использования с z-схемой. Используйте, когда пользователь хочет обновить схемы с черновика-04 до черновика-2020-12, преобразовать черновые форматы, обновить устаревшие ключевые слова, заменить id на $id, преобразовать определения в $defs, перенести элементы в prefixItems, заменить зависимости наdependentRequired илиdependentSchemas, принять unevaluatedProperties или unevaluatedItems или адаптировать схемы к новым функциям схемы JSON.
