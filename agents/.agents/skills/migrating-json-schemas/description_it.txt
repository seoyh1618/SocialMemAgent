Migra gli schemi JSON tra versioni bozza da utilizzare con z-schema. Da utilizzare quando l'utente desidera aggiornare gli schemi da draft-04 a draft-2020-12, convertire tra formati bozza, aggiornare parole chiave obsolete, sostituire id con $id, convertire definizioni in $defs, migrare elementi in prefixItems, sostituire le dipendenze condependentRequired odependentSchemas, adottare unevaluatedProperties o unevaluatedItems o adattare gli schemi alle funzionalità dello schema JSON più recenti.
