Migre les schémas JSON entre les versions brouillon pour les utiliser avec z-schema. À utiliser lorsque l'utilisateur souhaite mettre à niveau les schémas de draft-04 à draft-2020-12, convertir entre des formats de brouillon, mettre à jour des mots-clés obsolètes, remplacer l'identifiant par $id, convertir les définitions en $defs, migrer les éléments vers prefixItems, remplacer les dépendances pardependentRequired oudependentSchemas, adopter unevaluatedProperties ou unvaluatedItems, ou adapter les schémas aux nouvelles fonctionnalités du schéma JSON.
