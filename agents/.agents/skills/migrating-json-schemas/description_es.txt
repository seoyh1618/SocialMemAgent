Migra esquemas JSON entre versiones borrador para usarlos con esquema z. Úselo cuando el usuario desee actualizar esquemas del borrador-04 al borrador-2020-12, convertir entre formatos de borrador, actualizar palabras clave obsoletas, reemplazar id con $id, convertir definiciones a $defs, migrar elementos a prefixItems, reemplazar dependencias con dependienteRequired o dependienteSchemas, adoptar propiedades no evaluadas o elementos no evaluados, o adaptar esquemas a características de esquema JSON más nuevas.
