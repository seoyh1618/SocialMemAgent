ترحيل مخططات JSON بين إصدارات المسودات لاستخدامها مع مخطط z. يُستخدم عندما يريد المستخدم ترقية المخططات من Draft-04 إلى Draft-2020-12، أو التحويل بين تنسيقات المسودة، أو تحديث الكلمات الأساسية المهملة، أو استبدال id بـ $id، أو تحويل التعريفات إلى $defs، أو ترحيل العناصر إلى prefixItems، أو استبدال التبعيات بالمخططات التابعة المطلوبة أو التابعة، أو اعتماد خصائص غير مقيمة أو عناصر غير مقيمة، أو تكييف المخططات مع ميزات JSON Schema الأحدث.
