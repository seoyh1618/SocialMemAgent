Migriert JSON-Schemas zwischen Entwurfsversionen zur Verwendung mit Z-Schema. Verwenden Sie diese Option, wenn der Benutzer Schemata von Draft-04 auf Draft-2020-12 aktualisieren, zwischen Entwurfsformaten konvertieren, veraltete Schlüsselwörter aktualisieren, ID durch $id ersetzen, Definitionen in $defs konvertieren, Elemente in prefixItems migrieren, Abhängigkeiten durch dependencyRequired oder dependencySchemas ersetzen, unevaluatedProperties oder unevaluatedItems übernehmen oder Schemata an neuere JSON-Schemafunktionen anpassen möchte.
