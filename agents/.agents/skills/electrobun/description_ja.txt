Electrobun を使用してデスクトップ アプリケーションを構築、スキャフォールディング、デバッグ、出荷します。Electrobun は、ランタイムとネイティブ OS Web ビューとして Bun を使用する超高速 TypeScript デスクトップ アプリケーション フレームワークです。このスキルは、プロジェクト スキャフォールディング、ウィンドウ管理、メイン ↔ Web ビュー RPC 通信、ネイティブ UI 統合、自動更新、配布をカバーする、TypeScript を使用してクロスプラットフォーム デスクトップ アプリケーションを構築するための包括的なガイダンスを提供します。このスキルは、ユーザーが Electrobun について言及した場合、TypeScript または Bun を使用してデスクトップ アプリを構築したい場合、BrowserWindow、BrowserView、メイン プロセスと Web ビュー プロセス間の RPC、アプリのバンドル、クロスプラットフォーム デスクトップ開発について質問する場合、またはメニュー、トレイ アイコン、アップデーターのセットアップ、配布用のパッケージ化についてサポートが必要な場合に使用します。また、ユーザーが「デスクトップ アプリ」と発言し、Bun を使用または検討している場合にもトリガーされます。一般的なトリガーには、「Electrobun アプリの作成」、「デスクトップ アプリケーション TypeScript」、「Bun デスクトップ アプリ」、「BrowserWindow」、「webview RPC」、「Electrobun チュートリアル」、「デスクトップ アプリ アップデーター」、「クロスプラットフォーム デスクトップ」、「Bun を使用した macOS アプリ」、「Windows デスクトップ アプリ TypeScript」、または Web テクノロジと Bun ランタイムを使用したネイティブ デスクトップ アプリケーションの構築に関するディスカッションが含まれます。
