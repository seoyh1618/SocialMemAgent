使用 Electrobun 建置、建置、偵錯和發佈桌面應用程式 — Electrobun 是一個超快的 TypeScript 桌面應用程式框架，使用 Bun 作為執行時間和本機作業系統 Web 視圖。該技能為使用 TypeScript 建立跨平台桌面應用程式提供了全面的指導，涵蓋專案腳手架、視窗管理、main↔webview RPC 通訊、本機 UI 整合、自動更新和分發。當使用者提到 Electrobun、想要使用 TypeScript 或 Bun 建立桌面應用程式、詢問 BrowserWindow、BrowserView、主進程和 Webview 進程之間的 RPC、應用程式捆綁、跨平台桌面開發，或需要選單、托盤圖示、更新程式設定或分發打包方面的協助時，請使用此技能。當他們說“桌面應用程式”並且正在使用或考慮使用 Bun 時也會觸發。常見的觸發器包括：「建立 Electrobun 應用程式」、「桌面應用程式 TypeScript」、「Bun 桌面應用程式」、「BrowserWindow」、「webview RPC」、「Electrobun 教學課程」、「桌面應用程式更新程式」、「跨平台更新桌面」、「使用 BunmacOS 教學課程」、「桌面應用程式更新運行時建立本機桌面應用程式的任何討論。
