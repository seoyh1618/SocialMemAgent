Cree, desarrolle, depure y envíe aplicaciones de escritorio con Electrobun, un marco de trabajo de aplicaciones de escritorio TypeScript ultrarrápido que utiliza Bun como tiempo de ejecución y vistas web nativas del sistema operativo. Esta habilidad proporciona una guía integral para crear aplicaciones de escritorio multiplataforma con TypeScript, que cubre el andamiaje del proyecto, la administración de ventanas, la comunicación RPC principal↔webview, la integración de la interfaz de usuario nativa, las actualizaciones automáticas y la distribución. Utilice esta habilidad cuando el usuario mencione Electrobun, quiera crear una aplicación de escritorio con TypeScript o Bun, pregunte sobre BrowserWindow, BrowserView, RPC entre los procesos principal y de vista web, agrupación de aplicaciones, desarrollo de escritorio multiplataforma o necesite ayuda con menús, iconos de bandeja, configuración del actualizador o empaquetado para distribución. También se activa cuando dicen "aplicación de escritorio" y están usando o considerando Bun.
