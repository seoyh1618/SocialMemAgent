使用 Electrobun 构建、搭建、调试和发布桌面应用程序 — Electrobun 是一个超快的 TypeScript 桌面应用程序框架，使用 Bun 作为运行时和本机操作系统 Web 视图。该技能为使用 TypeScript 构建跨平台桌面应用程序提供了全面的指导，涵盖项目脚手架、窗口管理、main↔webview RPC 通信、本机 UI 集成、自动更新和分发。当用户提到 Electrobun、想要使用 TypeScript 或 Bun 构建桌面应用程序、询问 BrowserWindow、BrowserView、主进程和 Webview 进程之间的 RPC、应用程序捆绑、跨平台桌面开发，或者需要菜单、托盘图标、更新程序设置或分发打包方面的帮助时，请使用此技能。当他们说“桌面应用程序”并且正在使用或考虑使用 Bun 时也会触发。常见的触发器包括：“创建 Electrobun 应用程序”、“桌面应用程序 TypeScript”、“Bun 桌面应用程序”、“BrowserWindow”、“webview RPC”、“Electrobun 教程”、“桌面应用程序更新程序”、“跨平台桌面”、“使用 Bun 的 macOS 应用程序”、“Windows 桌面应用程序 TypeScript”，或有关使用 Web 技术和 Bun 运行时构建本机桌面应用程序的任何讨论。
