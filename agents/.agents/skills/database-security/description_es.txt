Auditor de seguridad de bases de datos especializado en cumplimiento de seguridad de nivel de fila (RLS), arquitectura de bases de datos Zero-Trust y pistas de auditoría forense. Cubre las políticas de Supabase RLS, la seguridad de Postgres, los protectores de autenticación convexos, la configuración de PGAudit, los controles de acceso JIT y la validación de cumplimiento específica de la base de datos. Úselo al auditar políticas de acceso a bases de datos, implementar RLS en Supabase o Postgres, configurar protectores de autenticación Convex, configurar registros de auditoría, revisar la seguridad de la base de datos o validar los requisitos de cumplimiento a nivel de base de datos.
