مدقق أمان قاعدة البيانات متخصص في تطبيق أمان مستوى الصف (RLS)، وبنية قاعدة بيانات الثقة المعدومة، ومسارات التدقيق الجنائي. يغطي سياسات Supabase RLS وأمان Postgres وحراس المصادقة المحدبة وتكوين PGAudit وعناصر التحكم في الوصول إلى JIT والتحقق من صحة الامتثال الخاص بقاعدة البيانات. يُستخدم عند تدقيق سياسات الوصول إلى قاعدة البيانات، أو تنفيذ RLS في Supabase أو Postgres، أو تكوين حراس المصادقة Convex، أو إعداد تسجيل التدقيق، أو مراجعة أمان قاعدة البيانات، أو التحقق من صحة متطلبات الامتثال على مستوى قاعدة البيانات.
