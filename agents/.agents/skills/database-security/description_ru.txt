Аудитор безопасности баз данных, специализирующийся на обеспечении безопасности на уровне строк (RLS), архитектуре базы данных с нулевым доверием и следах судебно-медицинской экспертизы. Охватывает политики Supabase RLS, безопасность Postgres, защиту аутентификации Convex, конфигурацию PGAudit, контроль доступа JIT и проверку соответствия конкретной базы данных. Используйте при аудите политик доступа к базе данных, реализации RLS в Supabase или Postgres, настройке защиты аутентификации Convex, настройке ведения журнала аудита, проверке безопасности базы данных или проверке требований соответствия на уровне базы данных.
