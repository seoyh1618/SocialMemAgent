Flusso di lavoro di implementazione incentrato su Git. Applica il checkout pulito, crea un ramo con nome appropriato, tiene traccia dei progressi in un file di markdown WIP ed esegue il commit continuo in modo che i log git fungano da canale di monitoraggio principale. Utilizzare all'avvio delle istruzioni, offrire per qualsiasi attività di implementazione basata sul piano.
