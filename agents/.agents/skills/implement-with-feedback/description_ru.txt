Рабочий процесс реализации, ориентированный на Git. Обеспечивает чистое извлечение, создает ветку с соответствующим названием, отслеживает ход выполнения в файле уценки WIP и осуществляет непрерывную фиксацию, поэтому журналы git служат основным каналом мониторинга. Используйте при запуске инструкции, предложения для любой плановой задачи реализации.
