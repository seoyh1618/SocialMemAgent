Strumento di query sulla documentazione API di ABC Medical Cloud. Legge ed esegue query sulla specifica OpenAPI dell'API ABC (oltre 5.000 endpoint), supporta la ricerca per modulo, percorso e metodo e risolve automaticamente i riferimenti $ref. Utilizza una struttura di memorizzazione nella cache suddivisa in moduli per query veloci. Casi d'uso: (1) Query sulle definizioni degli endpoint API (2) Ricerca di endpoint funzionali specifici (3) Query sulle definizioni dello schema (4) Visualizzazione delle statistiche degli endpoint.
