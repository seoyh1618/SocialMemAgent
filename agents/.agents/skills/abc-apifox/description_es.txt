Herramienta de consulta de documentación API de ABC Medical Cloud. Lee y consulta la especificación OpenAPI de la API ABC (más de 5000 puntos finales), admite búsquedas por módulo, ruta y método, y resuelve automáticamente referencias $ref. Utiliza una estructura de almacenamiento en caché dividida en módulos para consultas rápidas. Casos de uso: (1) Consultar definiciones de puntos finales de API (2) Buscar puntos finales funcionales específicos (3) Definiciones de esquemas de consulta (4) Ver estadísticas de puntos finales.
