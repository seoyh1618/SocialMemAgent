Guía de arquitectura del módulo de autenticación de permisos de autenticación, que cubre la integración de IAM, el modelo de permisos RBAC, la validación de permisos de recursos, la migración de permisos y la autenticación OAuth. Se utiliza cuando los usuarios desarrollan funciones de permisos, configuran recursos de IAM, implementan validación de permisos o manejan flujos de autenticación.
