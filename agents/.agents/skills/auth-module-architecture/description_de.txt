Leitfaden zur Architektur des Auth Permission Authentication Module, der die IAM-Integration, das RBAC-Berechtigungsmodell, die Validierung von Ressourcenberechtigungen, die Migration von Berechtigungen und die OAuth-Authentifizierung abdeckt. Wird verwendet, wenn Benutzer Berechtigungsfunktionen entwickeln, IAM-Ressourcen konfigurieren, Berechtigungsvalidierung implementieren oder Authentifizierungsflüsse verarbeiten.
