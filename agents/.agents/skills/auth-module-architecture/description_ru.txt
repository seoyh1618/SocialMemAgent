Руководство по архитектуре модуля аутентификации разрешений на аутентификацию, охватывающее интеграцию IAM, модель разрешений RBAC, проверку разрешений на ресурсы, миграцию разрешений и аутентификацию OAuth. Используется, когда пользователи разрабатывают функции разрешений, настраивают ресурсы IAM, реализуют проверку разрешений или обрабатывают потоки аутентификации.
