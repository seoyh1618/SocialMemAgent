Échafaudez et créez des SPA statiques prêts pour la production avec Vite, React, Tailwind CSS v4, la prise en charge de PWA, l'intégrité des sous-ressources (SRI) et l'optimisation des images. À utiliser lors de la création d'un nouveau React SPA, de la configuration d'une pile Vite + React + Tailwind, de l'activation du comportement hors ligne/PWA, du renforcement de la sortie de build avec SRI, de l'ajout d'une optimisation d'image ou de la configuration de tests Vitest/Playwright facultatifs.
