Supporta e crea SPA statiche pronte per la produzione con Vite, React, Tailwind CSS v4, supporto PWA, Subresource Integrity (SRI) e ottimizzazione delle immagini. Da utilizzare quando si crea una nuova React SPA, si configura uno stack Vite + React + Tailwind, si abilita il comportamento offline/PWA, si rafforza l'output di build con SRI, si aggiunge l'ottimizzazione delle immagini o si configurano test Vitest/Playwright opzionali.
