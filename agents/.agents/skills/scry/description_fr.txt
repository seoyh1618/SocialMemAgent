Interrogez l'API ExoPriors Scry : recherche SQL sur HTTPS dans plus de 229 millions d'entités couvrant les forums, les articles, les médias sociaux, les documents gouvernementaux et les marchés de prédiction. Comprend la résolution d'identité d'auteur multiplateforme (acteurs, personnes, alias), la navigation dans les graphiques académiques OpenAlex (auteurs, citations, institutions, concepts), les artefacts partageables et les jugements d'agent structurés. À utiliser lorsque la tâche implique : API Scry, ExoPriors, /v1/scry/query, scry.search, scry.entities, vues matérialisées, recherche de corpus, infrastructure épistémique, 229 millions d'entités, recherche lexicale, BM25, jugements d'agent structurés, partages scry, analyse inter-corpus, qui est cette personne, identité multiplateforme, OpenAlex, graphique de citation, graphique de co-auteur, articles académiques, recherche d'auteur. PAS pour : la composition de recherche sémantique/vecteur ou l'algèbre d'intégration (utilisez des vecteurs scry), le reclassement basé sur LLM (utilisez scry-rerank) ou les propres sources de données Postgres / non-ExoPriors locales de l'utilisateur.
