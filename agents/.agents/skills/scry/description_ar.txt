الاستعلام عن واجهة برمجة تطبيقات ExoPriors Scry - بحث SQL-over-HTTPS عبر أكثر من 229 مليون كيان يشمل المنتديات والأبحاث ووسائل التواصل الاجتماعي والسجلات الحكومية وأسواق التنبؤ. يتضمن تحليل هوية المؤلف عبر الأنظمة الأساسية (الممثلين، والأشخاص، والأسماء المستعارة)، والتنقل في الرسم البياني الأكاديمي لـ OpenAlex (المؤلفون، والاستشهادات، والمؤسسات، والمفاهيم)، والعناصر القابلة للمشاركة، وأحكام الوكيل المنظمة. يُستخدم عندما تتضمن المهمة: Scry API، ExoPriors، /v1/scry/query، scry.search، scry.entities، العروض المادية، بحث المجموعة، البنية التحتية المعرفية، 229 مليون كيان، البحث المعجمي، BM25، أحكام الوكيل المنظم، مشاركات scry، التحليل عبر المجموعة، من هو هذا الشخص، الهوية عبر الأنظمة الأساسية، OpenAlex، الرسم البياني للاقتباس، الرسم البياني للمؤلف المشارك، الأوراق الأكاديمية، بحث المؤلف. ليس من أجل: تكوين البحث الدلالي/المتجه أو تضمين الجبر (استخدم ناقلات scry)، أو إعادة الترتيب المستندة إلى LLM (استخدم إعادة ترتيب scry)، أو مصادر بيانات Postgres المحلية / غير التابعة لـ ExoPriors.
