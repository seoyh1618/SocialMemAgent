Logotipos de marcas generados por lotes. Dos puntos de entrada: (1) Después de que el usuario describe la marca/propósito, se generan diez variantes del logotipo en masa; (2) Después de que el usuario carga una imagen de referencia, se analiza y se crean logotipos. Ofrece siete opciones de estilos convencionales. Utiliza la API de Gemini para generar y guardar en la carpeta de imágenes de Obsidian. Frases desencadenantes: “generar logotipo”, “diseñar logotipo”, “hacer un logotipo”, “logotipos por lotes”, “diseñarme un emblema”.
