Генерирует описания запросов на включение путем сравнения текущей ветки с родительской веткой. Создает семантические PR-заголовки и заполняет PR-шаблоны. Используйте, когда пользователь просит создать описание PR, подготовить запрос на включение или создать описание запроса на слияние. Пользователь может включать в запрос идентификаторы билетов (например, билеты: NN-123, TB-456) из системы отслеживания компании; рассматривайте их как идентификаторы связанных проблем для PR.
