Genera descripciones de solicitudes de extracción comparando la rama actual con la rama principal. Crea títulos de relaciones públicas semánticas de estilo compromiso y completa plantillas de relaciones públicas. Úselo cuando el usuario solicite generar una descripción de PR, preparar una solicitud de extracción o crear una descripción de solicitud de fusión. El usuario puede incluir ID de boleto en la solicitud (por ejemplo, boletos: NN-123, TB-456) del sistema de seguimiento de la empresa; trátelos como ID de problemas relacionados para el PR.
