Génère des descriptions de demandes d'extraction en comparant la branche actuelle avec la branche parent. Crée des titres PR de style commit sémantique et remplit les modèles PR. À utiliser lorsque l'utilisateur demande à générer une description PR, à préparer une demande d'extraction ou à créer une description de demande de fusion. L'utilisateur peut inclure des identifiants de ticket dans la demande (par exemple, tickets : NN-123, TB-456) provenant du système de suivi de l'entreprise ; traitez-les comme les identifiants de problème associés pour le PR.
