Genera le descrizioni delle richieste pull confrontando il ramo corrente con il ramo padre. Crea titoli PR in stile commit semantico e compila modelli PR. Da utilizzare quando l'utente richiede di generare una descrizione PR, preparare una richiesta pull o creare una descrizione della richiesta di unione. L'utente può includere nella richiesta gli ID dei ticket (es. ticket: NN-123, TB-456) provenienti dal sistema di tracciamento aziendale; trattarli come ID dei problemi correlati per il PR.
