Crea, elenca, rimuovi ed esegui agenti Claude Code autonomi pianificati. Gli agenti vengono eseguiti con un timer tramite macOS launchd, eseguono qualsiasi richiesta senza testa e forniscono risultati tramite messaggi Beeper e notifiche macOS. Utilizzalo per ricerche ricorrenti, monitoraggio, build notturne o qualsiasi attività che desideri che Claude esegua con il pilota automatico.
