Создавайте, перечисляйте, удаляйте и запускайте запланированные автономные агенты Claude Code. Агенты запускаются по таймеру через macOS launchd, автоматически выполняют любые запросы и доставляют результаты с помощью сообщений Beeper и уведомлений macOS. Используйте для повторяющихся исследований, мониторинга, ночных сборок или любых задач, которые Клод должен выполнять на автопилоте.
