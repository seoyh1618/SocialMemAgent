Créez, répertoriez, supprimez et exécutez des agents Claude Code autonomes planifiés. Les agents s'exécutent selon une minuterie via macOS launchd, exécutent n'importe quelle invite sans tête et fournissent des résultats via des messages Beeper et des notifications macOS. À utiliser pour les recherches récurrentes, la surveillance, les constructions nocturnes ou toute tâche que vous souhaitez que Claude effectue en pilote automatique.
