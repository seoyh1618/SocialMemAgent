Cree reglas de Trae IDE (.trae/rules/*.md) para las restricciones de comportamiento de la IA. Úselo cuando el usuario desee: crear una regla de proyecto, configurar pautas de estilo de código, aplicar convenciones de nomenclatura, hacer que la IA siempre haga X o personalizar el comportamiento de la IA para archivos específicos. Activadores en: '创建 regla', 'regla del proyecto', '.trae/rules/', 'AGENTS.md', 'CLAUDE.md', 'hacer que la IA siempre use PascalCase'. NO lo use para habilidades (use trae-skill-writer) o agentes (use trae-agent-writer).
