Erstellen Sie Trae-IDE-Regeln (.trae/rules/*.md) für KI-Verhaltensbeschränkungen. Verwenden Sie diese Option, wenn der Benutzer: eine Projektregel erstellen, Codestilrichtlinien einrichten, Namenskonventionen erzwingen, die KI immer X ausführen lassen oder das KI-Verhalten für bestimmte Dateien anpassen möchte. Auslöser bei: „创建-Regel“, „Projektregel“, „.trae/rules/“, „AGENTS.md“, „CLAUDE.md“, „KI immer PascalCase verwenden lassen“. NICHT für Fertigkeiten (verwenden Sie trae-skill-writer) oder Agenten (verwenden Sie trae-agent-writer) verwenden.
