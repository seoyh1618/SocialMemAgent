Créez des règles Trae IDE (.trae/rules/*.md) pour les contraintes de comportement de l'IA. À utiliser lorsque l'utilisateur souhaite : créer une règle de projet, définir des directives de style de code, appliquer des conventions de dénomination, faire en sorte que l'IA fasse toujours X ou personnaliser le comportement de l'IA pour des fichiers spécifiques. Se déclenche sur : 'règle de projet', 'règle de projet', '.trae/rules/', 'AGENTS.md', 'CLAUDE.md', 'faire en sorte que l'IA utilise toujours PascalCase'. Ne PAS utiliser pour les compétences (utilisez trae-skill-writer) ou les agents (utilisez trae-agent-writer).
