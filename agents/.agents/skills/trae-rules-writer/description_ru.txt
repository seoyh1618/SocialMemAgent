Создайте правила Trae IDE (.trae/rules/*.md) для ограничений поведения ИИ. Используйте, когда пользователь хочет: создать правило проекта, настроить рекомендации по стилю кода, обеспечить соблюдение соглашений об именах, заставить ИИ всегда выполнять X или настроить поведение ИИ для определенных файлов. Срабатывает по: «правилу 创建», «правилу проекта», «.trae/rules/», «AGENTS.md», «CLAUDE.md», «заставить ИИ всегда использовать PascalCase». НЕ используйте для навыков (используйте trae-skill-writer) или агентов (используйте trae-agent-writer).
