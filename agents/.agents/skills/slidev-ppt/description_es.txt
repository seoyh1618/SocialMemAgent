Utilice Slidev para crear rápidamente presentaciones profesionales. Admite múltiples temas preestablecidos (negocios, tecnología, creativo, minimalista), genera esquemas automáticamente, inicia un servidor de desarrollo para vista previa y exporta a PDF/PPTX. Utilice esta habilidad cuando necesite: 1) crear una presentación PPT; 2) hacer diapositivas con Markdown; 3) dar una charla técnica o un lanzamiento de producto; 4) exportar la presentación como PDF u otros formatos.
