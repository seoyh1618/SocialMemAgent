Используйте Slidedev для быстрого создания профессиональных презентаций. Поддерживает несколько предустановленных тем (бизнес, технологии, креатив, минимализм), автоматически создает контуры, запускает сервер разработки для предварительного просмотра, экспортирует в PDF/PPTX. Используйте этот навык, когда вам нужно: 1) создать презентацию PPT; 2) делать слайды с помощью Markdown; 3) провести технический доклад или представить продукт; 4) экспортируйте презентацию в PDF или другие форматы.
