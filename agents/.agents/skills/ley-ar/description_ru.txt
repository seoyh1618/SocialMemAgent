Ищите в аргентинских юридических базах данных (SAIJ, JUBA, CSJN, JUSCABA) юриспруденцию, законодательство, краткие описания дел и доктрины, используя интерфейс командной строки «ley». Используйте, когда пользователь спрашивает об аргентинском законодательстве, судебных решениях, правовых прецедентах, фаллосах, юриспруденции, законодательстве или упоминает SAIJ, JUBA, CSJN, JUSCABA. Поддерживает параллельный поиск по базам данных, вывод JSON/таблиц/текста и фильтрацию по юрисдикции. Построен на основе реконструированных серверов MCP (hernan-cc) — прямые HTTP-вызовы, уровень MCP не требуется.
