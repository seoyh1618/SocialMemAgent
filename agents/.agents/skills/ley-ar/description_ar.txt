ابحث في قواعد البيانات القانونية الأرجنتينية (SAIJ، JUBA، CSJN، JUSCABA) عن الفقه والتشريعات وملخصات القضايا والعقيدة باستخدام `ley` CLI. يُستخدم عندما يسأل المستخدم عن القانون الأرجنتيني، أو قرارات المحكمة، أو السوابق القانونية، أو الفقه، أو التشريع، أو يذكر SAIJ، أو JUBA، أو CSJN، أو JUSCABA. يدعم البحث الموازي عبر قواعد البيانات، وإخراج JSON/الجدول/النص، والتصفية حسب الاختصاص. تم تصميمه من خوادم MCP ذات هندسة عكسية (hernan-cc) - مكالمات HTTP المباشرة، دون الحاجة إلى طبقة MCP.
