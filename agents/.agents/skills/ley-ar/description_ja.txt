`ley` CLI を使用して、アルゼンチンの法律データベース (SAIJ、JUBA、CSJN、JUSCABA) を検索して、法学、法律、事件の概要、および教義を検索します。ユーザーがアルゼンチンの法律、裁判所の判決、判例、法律、判例、立法について質問する場合、または SAIJ、JUBA、CSJN、JUSCABA について言及する場合に使用します。データベース間の並列検索、JSON/テーブル/テキスト出力、管轄区域によるフィルタリングをサポートします。リバース エンジニアリングされた MCP サーバー (hernan-cc) から構築 — 直接 HTTP 呼び出し、MCP レイヤは必要ありません。
