Busque en bases de datos jurídicas argentinas (SAIJ, JUBA, CSJN, JUSCABA) jurisprudencia, legislación, resúmenes de casos y doctrina utilizando la CLI `ley`. Utilizar cuando el usuario pregunta sobre leyes argentinas, decisiones judiciales, precedentes legales, fallos, jurisprudencia, legislación, o menciona SAIJ, JUBA, CSJN, JUSCABA. Admite búsqueda paralela en bases de datos, salida JSON/tabla/texto y filtrado por jurisdicción. Creado a partir de servidores MCP de ingeniería inversa (hernan-cc): llamadas HTTP directas, no se necesita capa MCP.
