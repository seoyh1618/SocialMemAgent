Cerca nei database legali argentini (SAIJ, JUBA, CSJN, JUSCABA) giurisprudenza, legislazione, sintesi di casi e dottrina utilizzando la CLI "ley". Utilizzare quando l'utente chiede informazioni sulla legge argentina, decisioni giudiziarie, precedenti legali, fallos, jurisprudencia, legislación o menziona SAIJ, JUBA, CSJN, JUSCABA. Supporta la ricerca parallela tra database, output JSON/tabella/testo e filtraggio per giurisdizione. Costruito da server MCP sottoposti a reverse engineering (hernan-cc): chiamate HTTP dirette, non è necessario alcun livello MCP.
