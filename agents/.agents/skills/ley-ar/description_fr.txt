Recherchez dans les bases de données juridiques argentines (SAIJ, JUBA, CSJN, JUSCABA) la jurisprudence, la législation, les résumés de cas et la doctrine à l'aide de la CLI «ley». À utiliser lorsque l'utilisateur pose des questions sur la loi argentine, les décisions de justice, les précédents juridiques, les fallos, la jurisprudencia, la legislación, ou mentionne SAIJ, JUBA, CSJN, JUSCABA. Prend en charge la recherche parallèle dans les bases de données, la sortie JSON/table/texte et le filtrage par juridiction. Construit à partir de serveurs MCP d'ingénierie inverse (hernan-cc) : appels HTTP directs, aucune couche MCP n'est nécessaire.
