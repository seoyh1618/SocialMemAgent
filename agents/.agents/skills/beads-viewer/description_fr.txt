Bead Viewer - Moteur de tri prenant en charge les graphiques pour les projets Bead. Calcule le PageRank, l'intermédiarité, le chemin critique et les cycles. À utiliser pour trier les tâches de perles, analyser les graphiques de dépendances, trouver des goulots d'étranglement, détecter les dépendances circulaires, planifier des pistes d'exécution parallèles ou générer des données d'avancement de sprint.
