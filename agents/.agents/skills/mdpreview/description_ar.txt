عارض Markdown الذي يتم إعادة تحميله مباشرة بموضوع Catppuccin مع دعم علامات التبويب المتعددة والتعليقات التوضيحية الهامشية وجدول المحتويات على الشريط الجانبي ووضع Chrome --app. يجب استخدام هذه المهارة عند معاينة ملفات .md في قارئ تحريري مصقول، أو إضافة الذكاء الاصطناعي أو التعليقات التوضيحية البشرية إلى المستندات، أو إطلاق جلسة مراجعة تعاونية للمستندات.
