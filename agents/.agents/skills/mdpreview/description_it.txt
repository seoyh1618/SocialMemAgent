Visualizzatore Markdown con ricarica live a tema Catppuccin con supporto multi-scheda, annotazioni sui margini, sommario della barra laterale e modalità --app Chrome. Questa capacità dovrebbe essere utilizzata quando si visualizza l'anteprima dei file .md in un lettore editoriale raffinato, si aggiungono annotazioni umane o AI ai documenti o si avvia una sessione di revisione collaborativa dei documenti.
