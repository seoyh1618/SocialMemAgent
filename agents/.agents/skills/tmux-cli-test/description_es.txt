Pruebe las aplicaciones CLI de forma interactiva mediante sesiones tmux. Úselo al probar aplicaciones TUI (panel de control de ratatui, indicaciones interactivas), verificar la salida del comando CLI, probar la navegación con el teclado o validar la representación del terminal. Lanza comandos en tmux, espera condiciones (nunca duerme), captura fotogramas, envía pulsaciones de teclas y afirma la salida. Diseñado específicamente para gpu-cli pero funciona con cualquier CLI. Utilice esta habilidad cuando se le solicite probar, verificar o realizar controles de calidad de cualquier flujo de comandos CLI o UI basado en terminal.
