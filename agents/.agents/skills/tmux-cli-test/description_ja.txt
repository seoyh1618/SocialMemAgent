tmux セッションを使用して、CLI アプリケーションを対話的にテストします。 TUI アプリ (ratatui ダッシュボード、対話型プロンプト) のテスト、CLI コマンド出力の検証、キーボード ナビゲーションのテスト、またはターミナル レンダリングの検証時に使用します。 tmux でコマンドを起動し、条件付きで待機し (スリープしない)、フレームをキャプチャし、キー入力を送信し、出力でアサートします。特に gpu-cli 用に設計されていますが、どの CLI でも動作します。ターミナルベースの UI または CLI コマンド フローのテスト、検証、または QA を求められた場合は、このスキルを使用します。
