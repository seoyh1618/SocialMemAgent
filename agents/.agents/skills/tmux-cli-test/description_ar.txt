اختبار تطبيقات CLI بشكل تفاعلي باستخدام جلسات tmux. يُستخدم عند اختبار تطبيقات TUI (لوحة معلومات راتاتوي، والمطالبات التفاعلية)، أو التحقق من مخرجات أمر CLI، أو اختبار التنقل عبر لوحة المفاتيح، أو التحقق من صحة العرض الطرفي. يطلق الأوامر في tmux، وينتظر الشروط (لا ينام أبدًا)، ويلتقط الإطارات، ويرسل ضغطات المفاتيح، ويؤكد على الإخراج. مصمم خصيصًا لـ gpu-cli ولكنه يعمل مع أي CLI. استخدم هذه المهارة عندما يُطلب منك اختبار أي واجهة مستخدم أو تدفق أوامر واجهة سطر الأوامر (CLI) المستندة إلى المحطة الطرفية أو التحقق منها أو ضمان الجودة.
