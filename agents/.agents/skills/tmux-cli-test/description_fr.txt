Testez les applications CLI de manière interactive à l'aide de sessions tmux. À utiliser pour tester les applications TUI (tableau de bord ratatui, invites interactives), vérifier la sortie des commandes CLI, tester la navigation au clavier ou valider le rendu du terminal. Lance des commandes dans tmux, attend les conditions (ne dort jamais), capture des images, envoie des pressions sur les touches et affirme la sortie. Spécialement conçu pour gpu-cli mais fonctionne avec n'importe quelle CLI. Utilisez cette compétence lorsqu'on vous demande de tester, vérifier ou contrôler la qualité de tout flux de commandes d'interface utilisateur ou CLI basé sur un terminal.
