Habilidad de vinculación dinámica para bibliotecas compartidas de Linux/ELF. Úselo al depurar fallas de carga de bibliotecas, configurar RPATH vs RUNPATH, comprender el control de versiones de soname, usar dlopen/dlsym para sistemas de complementos, interposición LD_PRELOAD o controlar la visibilidad de símbolos. Se activa en consultas sobre bibliotecas compartidas, dlopen, LD_LIBRARY_PATH, RPATH, soname, LD_PRELOAD, visibilidad de símbolos o errores "no se puede abrir el archivo de objeto compartido".
