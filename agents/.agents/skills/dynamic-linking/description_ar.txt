مهارة الارتباط الديناميكي لمكتبات Linux/ELF المشتركة. يُستخدم عند تصحيح أخطاء فشل تحميل المكتبة، أو تكوين RPATH مقابل RUNPATH، أو فهم إصدار soname، أو استخدام dlopen/dlsym لأنظمة المكونات الإضافية، أو مداخلة LD_PRELOAD، أو التحكم في رؤية الرمز. يتم التنشيط عند الاستعلامات حول المكتبات المشتركة، أو dlopen، أو LD_LIBRARY_PATH، أو RPATH، أو soname، أو LD_PRELOAD، أو رؤية الرمز، أو أخطاء "لا يمكن فتح ملف كائن مشترك".
