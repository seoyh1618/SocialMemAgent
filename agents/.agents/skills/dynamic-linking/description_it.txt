Capacità di collegamento dinamico per librerie condivise Linux/ELF. Da utilizzare durante il debug degli errori di caricamento delle librerie, la configurazione di RPATH vs RUNPATH, la comprensione del controllo delle versioni di Soname, l'utilizzo di dlopen/dlsym per sistemi di plugin, l'interposizione LD_PRELOAD o il controllo della visibilità dei simboli. Si attiva su query su librerie condivise, dlopen, LD_LIBRARY_PATH, RPATH, soname, LD_PRELOAD, visibilità dei simboli o errori "impossibile aprire il file oggetto condiviso".
