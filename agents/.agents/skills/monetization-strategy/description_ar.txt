عندما يريد المستخدم تصميم أو تحسين تحقيق الدخل من تطبيقه - التسعير أو نظام حظر الاشتراك غير المدفوع أو الاشتراكات أو عمليات الشراء داخل التطبيق. يُستخدم أيضًا عندما يذكر المستخدم "التسعير" أو "نظام حظر الاشتراك غير المدفوع" أو "الاشتراك" أو "الشراء داخل التطبيق" أو "كيفية تحقيق الدخل" أو "تحسين الإيرادات" أو "التجربة المجانية" أو "التحويل إلى الإصدار المدفوع". لمعرفة تأثير الاحتفاظ، راجع تحسين الاحتفاظ. للحصول على أسعار تنافسية، راجع تحليل المنافسين.
