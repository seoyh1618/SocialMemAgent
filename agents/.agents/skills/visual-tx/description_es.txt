Analice las transacciones de Ethereum obteniendo datos de seguimiento semántico y generando una explicación visual en HTML. Utilice esta habilidad siempre que el usuario proporcione un hash de transacción, solicite analizar una transacción, quiera comprender qué sucedió en una transmisión o pregunte sobre llamadas internas, transferencias de tokens, intercambios o cualquier actividad en cadena dentro de una transacción. Activa frases como "escanear este tx", "qué sucedió en esta transacción", "rastrear este tx", "explicar esta transacción" o cualquier hash de 66 caracteres con prefijo 0x.
