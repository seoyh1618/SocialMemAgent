Analysieren Sie Ethereum-Transaktionen, indem Sie semantische Trace-Daten abrufen und eine visuelle HTML-Erklärung generieren. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer einen Transaktions-Hash bereitstellt, um die Analyse einer Transaktion bittet, verstehen möchte, was in einer Sendung passiert ist, oder nach internen Aufrufen, Token-Transfers, Swaps oder anderen Aktivitäten in der Kette innerhalb einer Transaktion fragt. Triggern Sie auf Phrasen wie „diese Sendung scannen“, „was ist in dieser Transaktion passiert“, „diese Sendung verfolgen“, „diese Transaktion erklären“ oder einen beliebigen 66-Zeichen-Hash mit dem Präfix 0x.
