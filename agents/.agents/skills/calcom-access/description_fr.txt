Accédez à l'API Cal.com via CLI avec la clé API 1Password. À utiliser lorsque l'utilisateur souhaite répertorier les réservations, créer des types d'événements, gérer les horaires ou mentionner l'accès à cal.com. DÉCLENCHEURS - calcom, cal.com, réservations, liste des réservations, types d'événements, horaires, disponibilité, création d'une page de réservation.
