# Learning kirkpatrick evaluation
Apply Kirkpatrick's 4 levels: Reaction, Learning, Behavior, Results/ROI.

## CLI: /learning.kirkpatrick-evaluation --program 'leadership training' --level 3 --measure 'behavior change'
## Exit Codes: 0=success, 1=insufficient data, 2=invalid metrics
