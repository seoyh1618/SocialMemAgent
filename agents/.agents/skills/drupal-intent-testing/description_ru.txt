Исследовательское, целенаправленное тестирование пользовательского интерфейса для сайтов Drupal с использованием CLI агент-браузер (на основе дерева доступности; модель видения не требуется). Используйте, чтобы «копаться как человек»: создавать/редактировать контент, изменять конфигурацию модуля, создавать страницы Canvas и проверять предполагаемый UX/выход. Поддерживает управляемое тестирование, начальное фазз-тестирование/тестирование обезьян, а также парное сравнение до/после с артефактами (снимками, скриншотами, консолью/ошибками).
