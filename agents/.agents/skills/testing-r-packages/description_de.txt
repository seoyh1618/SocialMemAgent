Best Practices zum Schreiben von R-Pakettests mit testthat Version 3+. Wird zum Schreiben, Organisieren oder Verbessern von Tests für R-Pakete verwendet. Behandelt Teststruktur, Erwartungen, Vorrichtungen, Snapshots, Mocking und moderne Testthat-3-Muster, einschließlich autarker Tests, ordnungsgemäßer Bereinigung mit Withr und Snapshot-Tests.
