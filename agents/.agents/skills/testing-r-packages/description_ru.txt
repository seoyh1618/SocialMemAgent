Рекомендации по написанию тестов пакета R с использованием testthat версии 3+. Используйте при написании, организации или улучшении тестов для пакетов R. Охватывает структуру тестирования, ожидания, приспособления, снимки, макетирование и современные шаблоны тестирования, включая самодостаточные тесты, правильную очистку с помощью withr и тестирование снимков.
