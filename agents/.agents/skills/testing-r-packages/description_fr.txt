Meilleures pratiques pour écrire des tests de packages R à l’aide de testthat version 3+. À utiliser lors de l'écriture, de l'organisation ou de l'amélioration des tests pour les packages R. Couvre la structure des tests, les attentes, les appareils, les instantanés, les simulations et les tests modernes selon 3 modèles, y compris les tests autosuffisants, le nettoyage approprié avec withr et les tests d'instantanés.
