Rust-Cross-Compilation-Fähigkeit. Verwenden Sie diese Option, wenn Sie Rust-Binärdateien für eine andere Zielarchitektur oder ein anderes Betriebssystem erstellen, Cross oder Cargo-Zigbuild für eine hermetische Cross-Kompilierung verwenden, .cargo/config.toml für Cross-Ziele konfigurieren oder auf eingebettete und Bare-Metal-Systeme abzielen. Wird bei Abfragen zu Rust-Cross-Compilation, Rustup-Zielen, Cross-Tool, Cargo-Zigbuild, Aarch64-Unknown-Linux-Gnu, Thumbv7m-None-Eabi oder Building für Embedded aktiviert.
