Rustのクロスコンパイルスキル。異なるターゲット アーキテクチャまたは OS 用の Rust バイナリをビルドする場合、ハーメチック クロス コンパイルに Cross または Cargo-zigbuild を使用する場合、クロス ターゲット用に .cargo/config.toml を設定する場合、または組み込みおよびベアメタル システムをターゲットにする場合に使用します。 Rust クロスコンパイル、rustup ターゲット、クロスツール、cargo-zigbuild、aarch64-unknown-linux-gnu、thumbv7m-none-eabi、または組み込み用のビルドに関するクエリでアクティブになります。
