Habilidad de compilación cruzada de Rust. Úselo al compilar archivos binarios de Rust para una arquitectura o sistema operativo de destino diferente, usando cross o cargo-zigbuild para una compilación cruzada hermética, configurando .cargo/config.toml para objetivos cruzados o apuntando a sistemas integrados y sin sistema operativo. Se activa ante consultas sobre compilación cruzada de Rust, objetivos de oxidación, herramienta cruzada, cargo-zigbuild, aarch64-unknown-linux-gnu, thumbv7m-none-eabi o compilación para incrustados.
