Abilità di compilazione incrociata di Rust. Da utilizzare quando si creano binari Rust per un'architettura o un sistema operativo di destinazione diverso, utilizzando cross o cargo-zigbuild per la compilazione incrociata ermetica, configurando .cargo/config.toml per destinazioni incrociate o prendendo di mira sistemi embedded e bare-metal. Si attiva su query relative alla compilazione incrociata di Rust, target ruggine, cross tool, cargo-zigbuild, aarch64-unknown-linux-gnu, Thumbv7m-none-eabi o build per embedded.
