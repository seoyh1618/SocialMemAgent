Навыки кросс-компиляции Rust. Используйте при сборке двоичных файлов Rust для другой целевой архитектуры или ОС, использовании Cross или Cargo-zigbuild для герметичной кросс-компиляции, настройке .cargo/config.toml для перекрестных целей или при настройке встроенных и аппаратных систем. Активируется по запросам о кросс-компиляции Rust, целях Rustup, кросс-инструментах, Cargo-ZigBuild, aarch64-unknown-Linux-GNU, Thumbv7M-none-EABI или сборке для встраиваемых систем.
