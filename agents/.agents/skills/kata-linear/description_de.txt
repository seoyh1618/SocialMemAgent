Linearer Ticketlebenszyklus für jedes Projekt. Verwenden Sie diese Option, wenn Sie mit der Arbeit an einem linearen Problem beginnen, die Arbeit an einem Problem beenden oder fragen, woran als nächstes gearbeitet werden soll. Auslöser: KAT-N starten, aufnehmen, umsetzen, beenden, abschließen, erledigt, schließen, was kommt als nächstes, nächstes Ticket, nächste Ausgabe. Behandelt die Blockervalidierung, Statusübergänge, das Laden von Kontexten, die Erstellung von Zweigen, das Evidence Gating und die Kettenförderung.
