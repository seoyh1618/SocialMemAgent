Guida completa per i framework Colorffy UI e CSS. Colorffy UI è una moderna libreria di componenti Vue 3/Nuxt 3 con oltre 70 componenti senza stile e senza testa (pulsanti, schede, moduli, finestre di dialogo, navigazione, ecc.) con supporto TypeScript. Colorffy CSS è un framework SCSS espressivo con sistema di colori tonali, classi di utilità e sistemi di layout (griglia, flexbox). Da utilizzare quando si lavora con i pacchetti Colorffy, si impostano progetti Vue/Nuxt con Colorffy, si scelgono componenti, si applicano stili, si utilizzano classi di utilità, si implementano layout o si integra l'interfaccia utente Colorffy con Colorffy CSS o stili personalizzati.
