Guía completa para los frameworks Colorffy UI y CSS. Colorffy UI es una biblioteca de componentes moderna de Vue 3/Nuxt 3 con más de 70 componentes sin estilo y sin cabeza (botones, tarjetas, formularios, cuadros de diálogo, navegación, etc.) con soporte para TypeScript. Colorffy CSS es un marco SCSS expresivo con sistema de color tonal, clases de utilidad y sistemas de diseño (cuadrícula, flexbox). Úselo cuando trabaje con paquetes Colorffy, configure proyectos Vue/Nuxt con Colorffy, elija componentes, aplique estilos, use clases de utilidad, implemente diseños o integre la interfaz de usuario de Colorffy con Colorffy CSS o estilos personalizados.
