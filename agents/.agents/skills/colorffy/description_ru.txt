Полное руководство по фреймворкам Colorffy UI и CSS. Colorffy UI — это современная библиотека компонентов Vue 3/Nuxt 3 с более чем 70 неоформленными компонентами без заголовка (кнопки, карточки, формы, диалоговые окна, навигация и т. д.) с поддержкой TypeScript. Colorffy CSS — это выразительная среда SCSS с тональной системой цвета, служебными классами и системами макетов (сетка, флексбокс). Используйте при работе с пакетами Colorffy, настройке проектов Vue/Nuxt с Colorffy, выборе компонентов, применении стилей, использовании служебных классов, реализации макетов или интеграции пользовательского интерфейса Colorffy с Colorffy CSS или пользовательскими стилями.
