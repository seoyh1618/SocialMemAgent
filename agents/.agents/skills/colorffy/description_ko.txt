Colorffy UI 및 CSS 프레임워크에 대한 전체 가이드입니다. Colorffy UI는 TypeScript를 지원하는 70개 이상의 스타일이 지정되지 않은 헤드리스 구성 요소(버튼, 카드, 양식, 대화 상자, 탐색 등)가 포함된 최신 Vue 3/Nuxt 3 구성 요소 라이브러리입니다. Colorffy CSS는 색조 색상 시스템, 유틸리티 클래스 및 레이아웃 시스템(그리드, flexbox)을 갖춘 표현력이 뛰어난 SCSS 프레임워크입니다. Colorffy 패키지 작업, Colorffy로 Vue/Nuxt 프로젝트 설정, 구성 요소 선택, 스타일 적용, 유틸리티 클래스 사용, 레이아웃 구현 또는 Colorffy UI를 Colorffy CSS 또는 사용자 정의 스타일과 통합할 때 사용합니다.
