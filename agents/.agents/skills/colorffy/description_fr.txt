Guide complet pour les frameworks Colorffy UI et CSS. Colorffy UI est une bibliothèque de composants Vue 3 / Nuxt 3 moderne avec plus de 70 composants sans style et sans tête (boutons, cartes, formulaires, boîtes de dialogue, navigation, etc.) avec prise en charge de TypeScript. Colorffy CSS est un framework SCSS expressif avec un système de couleurs tonales, des classes utilitaires et des systèmes de mise en page (grille, flexbox). À utiliser lorsque vous travaillez avec des packages Colorffy, configurez des projets Vue/Nuxt avec Colorffy, choisissez des composants, appliquez des styles, utilisez des classes utilitaires, implémentez des mises en page ou intégrez l'interface utilisateur Colorffy avec Colorffy CSS ou des styles personnalisés.
