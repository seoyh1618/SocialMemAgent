الدليل الكامل لأطر عمل Colorffy UI وCSS. Colorffy UI عبارة عن مكتبة مكونات Vue 3 / Nuxt 3 حديثة تحتوي على أكثر من 70 مكونًا بدون نمط وبدون رأس (الأزرار والبطاقات والنماذج ومربعات الحوار والتنقل وما إلى ذلك) مع دعم TypeScript. Colorffy CSS هو إطار عمل SCSS معبر مع نظام الألوان اللونية، وفئات المرافق، وأنظمة التخطيط (الشبكة، flexbox). يُستخدم عند العمل مع حزم Colorffy، أو إعداد مشاريع Vue/Nuxt باستخدام Colorffy، أو اختيار المكونات، أو تطبيق الأنماط، أو استخدام فئات الأدوات المساعدة، أو تنفيذ التخطيطات، أو دمج Colorffy UI مع Colorffy CSS أو الأنماط المخصصة.
