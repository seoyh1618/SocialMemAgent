API повторной отправки электронной почты для отправки транзакционных и маркетинговых писем. Используйте при интеграции доставки электронной почты, управлении доменами, обработке веб-перехватчиков, построении рабочих процессов электронной почты с помощью React Email, управлении контактами или отправке рассылок. Используйте для повторной отправки, электронной почты, транзакций, широковещательной рассылки, контактов, сегментов, веб-перехватчиков, домена.
