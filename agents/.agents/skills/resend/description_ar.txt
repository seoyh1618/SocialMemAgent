إعادة إرسال واجهة برمجة تطبيقات البريد الإلكتروني لإرسال رسائل البريد الإلكتروني الخاصة بالمعاملات والتسويق. يُستخدم عند دمج تسليم البريد الإلكتروني، أو إدارة المجالات، أو التعامل مع خطافات الويب، أو إنشاء سير عمل البريد الإلكتروني باستخدام React Email، أو إدارة جهات الاتصال، أو إرسال عمليات البث. يُستخدم لإعادة الإرسال والبريد الإلكتروني والمعاملات والبث وجهات الاتصال والشرائح وخطافات الويب والمجال.
