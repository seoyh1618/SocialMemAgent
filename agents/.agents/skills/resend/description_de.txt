E-Mail-API zum erneuten Senden von Transaktions- und Marketing-E-Mails. Zur Verwendung bei der Integration der E-Mail-Zustellung, der Verwaltung von Domänen, der Handhabung von Webhooks, dem Aufbau von E-Mail-Workflows mit React Email, der Verwaltung von Kontakten oder dem Senden von Broadcasts. Verwendung für erneutes Senden, E-Mail, Transaktion, Broadcast, Kontakte, Segmente, Webhooks, Domäne.
