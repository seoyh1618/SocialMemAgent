مجموعة من الأدوات لإنشاء عناصر HTML ثابتة ومفصلة ومكتفية ذاتيًا - الرسوم البيانية والرسوم البيانية التفاعلية والمرئيات المعمارية ولوحات معلومات البيانات والتسليمات المرئية الغنية. استخدم هذه المهارة عندما يُطلب منك إنشاء عنصر HTML تفاعلي، أو إنشاء مكون ويب قائم بذاته، أو إنشاء تصور HTML ثابت، أو إنشاء رسم تخطيطي تفاعلي، أو إنتاج مخطط معلوماتي مرئي، أو عرض أي مرئيات عالية الكثافة كملف HTML واحد. سلسلة أدوات بناء صفرية - لا توجد React، ولا Vite، ولا Parcel. HTML5 خالص + CSS3 (شبكة/Flexbox) + SVG مضمّن. مشغلات على: "إنشاء HTML تفاعلي"، "إنشاء مكون ويب قائم بذاته"، "إنشاء تصور HTML ثابت"، "إنشاء رسم تخطيطي تفاعلي"، "إنتاج مخطط معلومات بياني"، "العرض كقطعة أثرية بتنسيق HTML"، "إنشاء لوحة معلومات مرئية"، "إنشاء رسم تخطيطي يمكنني فتحه في المتصفح".
