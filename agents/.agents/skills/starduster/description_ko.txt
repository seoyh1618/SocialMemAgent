카탈로그 GitHub는 구조화된 흑요석 금고에 저장소를 별표 표시했습니다.
AI 합성 요약, 정규화된 주제 분류, 그래프 최적화
필터링된 보기를 위한 위키링크 및 Obsidian Bases(.base) 색인 파일.
gh CLI를 통해 저장소 메타데이터 및 README를 가져오고 저장소를 다음으로 분류합니다.
카테고리 및 정규화된 주제를 사용하여 개별 저장소 노트를 생성합니다.
머리말을 작성하고 카테고리/주제/작가에 대한 허브 노트를 생성합니다.
그래프 보기 연결 지점 역할을 합니다.

사용자가 다음을 원할 때 이 기술을 사용하십시오.
(1) GitHub 스타를 Obsidian에 카탈로그화하거나 색인화합니다.
(2) 별표 표시된 저장소에서 검색 가능한 지식 기반 생성
(3) GitHub 스타에서 패턴을 구성하고 발견합니다.
(4) GitHub 스타를 구조화된 마크다운 노트로 내보내기
(5) 주제, 언어 또는 작성자별로 별표 표시된 저장소의 그래프를 작성합니다.

특정 URL을 노트에 저장/정리하려면 대신 kcap을 사용하세요.
AI 트윗을 검색하려면 대신 ai-twitter-radar를 사용하세요.
