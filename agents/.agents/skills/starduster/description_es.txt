Catalogue los repositorios destacados de GitHub en una bóveda estructurada de Obsidian con
Resúmenes sintetizados por IA, taxonomía de temas normalizada, optimización de gráficos
wikilinks y archivos de índice de Obsidian Bases (.base) para vistas filtradas.
Obtiene metadatos de repositorio y archivos README a través de gh CLI, clasifica los repositorios en
categorías y temas normalizados, genera notas de repositorio individuales con
frontmatter y crea notas centrales para categorías/temas/autores que
sirven como puntos de conexión de vista de gráfico.

Utilice esta habilidad cuando los usuarios quieran:
(1) Catalogar o indexar sus estrellas de GitHub en Obsidian
(2) Cree una base de conocimientos con capacidad de búsqueda a partir de repositorios destacados
(3) Organizar y descubrir patrones en sus estrellas de GitHub
(4) Exportar estrellas de GitHub como notas de rebajas estructuradas
(5) Cree un gráfico de repositorios destacados por tema, idioma o autor

Para guardar/destilar una URL específica en una nota, use kcap en su lugar.
Para explorar tweets de IA, utilice ai-twitter-radar en su lugar.
