Cataloguez les référentiels favoris de GitHub dans un coffre-fort Obsidian structuré avec
Résumés synthétisés par l'IA, taxonomie thématique normalisée, optimisée pour les graphiques
liens wiki et fichiers d'index Obsidian Bases (.base) pour les vues filtrées.
Récupère les métadonnées du dépôt et les README via gh CLI, classe les dépôts en
catégories et sujets normalisés, génère des notes de dépôt individuelles avec
frontmatter et crée des notes de hub pour les catégories/sujets/auteurs qui
servir de points de connexion de vue graphique.

Utilisez cette compétence lorsque les utilisateurs souhaitent :
(1) Cataloguer ou indexer leurs étoiles GitHub dans Obsidian
(2) Créez une base de connaissances consultable à partir des dépôts favoris
(3) Organisez et découvrez des modèles dans leurs étoiles GitHub
(4) Exportez les étoiles GitHub sous forme de notes de démarque structurées
(5) Créez un graphique des dépôts favoris par sujet, langue ou auteur

Pour enregistrer/distiller une URL spécifique vers une note, utilisez plutôt kcap.
Pour parcourir les tweets de l'IA, utilisez plutôt ai-twitter-radar.
