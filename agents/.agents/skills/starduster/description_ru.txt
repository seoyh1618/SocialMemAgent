Каталог GitHub поместил репозитории в структурированное хранилище Obsidian с
Резюме, синтезированные с помощью искусственного интеллекта, нормализованная таксономия тем, оптимизация для графиков
wikilinks и индексные файлы Obsidian Bases (.base) для отфильтрованных представлений.
Извлекает метаданные репозитория и файлы README через gh CLI, классифицирует репозитории на
категории и нормализованные темы, генерирует отдельные заметки репо с
Frontmatter и создает центральные заметки для категорий/тем/авторов, которые
служат точками подключения графического представления.

Используйте этот навык, когда пользователи хотят:
(1) Каталогизировать или индексировать свои звезды GitHub в Obsidian.
(2) Создайте базу знаний с возможностью поиска из помеченных репозиториев.
(3) Организовывать и обнаруживать закономерности в своих звездах GitHub.
(4) Экспортируйте звезды GitHub в виде структурированных заметок с уценкой.
(5) Создайте график избранных репозиториев по темам, языкам или авторам.

Чтобы сохранить/переместить определенный URL-адрес в заметку, используйте вместо этого kcap.
Для просмотра твитов ИИ используйте вместо этого ai-twitter-radar.
