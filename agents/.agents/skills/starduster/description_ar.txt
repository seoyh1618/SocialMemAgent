مستودعات كتالوج GitHub المميزة بنجمة في قبو Obsidian منظم مع
ملخصات مُركبة بواسطة الذكاء الاصطناعي، وتصنيف موضوعي موحد، ومُحسّن للرسم البياني
روابط wiki، وملفات فهرس Obsidian Bases (.base) للعروض المصفاة.
جلب البيانات التعريفية لمستودعات الريبو وملفات README عبر gh CLI، ويصنف عمليات إعادة الشراء إلى
الفئات والموضوعات الطبيعية، يُنشئ ملاحظات الريبو الفردية باستخدام
Frontmatter، ويقوم بإنشاء ملاحظات محورية للفئات/الموضوعات/المؤلفين
بمثابة نقاط اتصال عرض الرسم البياني.

استخدم هذه المهارة عندما يريد المستخدمون:
(1) قم بفهرسة نجوم GitHub الخاصة بهم أو فهرستها في Obsidian
(2) قم بإنشاء قاعدة معرفية قابلة للبحث من المستودعات المميزة بنجمة
(3) تنظيم واكتشاف الأنماط في نجوم GitHub الخاصة بهم
(4) قم بتصدير نجوم GitHub كملاحظات تخفيض منظمة
(5) أنشئ رسمًا بيانيًا للمستودعات المميزة بنجمة حسب الموضوع أو اللغة أو المؤلف

لحفظ/تقطير عنوان URL محدد لملاحظة، استخدم kcap بدلاً من ذلك.
لتصفح تغريدات الذكاء الاصطناعي، استخدم ai-twitter-radar بدلاً من ذلك.
