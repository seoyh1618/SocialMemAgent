GitHub のスター付きリポジトリを構造化された Obsidian ボールトにカタログ化します。
AI によって合成された概要、正規化されたトピック分類、グラフに最適化
ウィキリンク、およびフィルターされたビュー用の Obsidian Bases (.base) インデックス ファイル。
gh CLI 経由でリポジトリのメタデータと README を取得し、リポジトリを次のように分類します。
カテゴリと正規化されたトピックを使用して、個別のリポジトリ ノートを生成します。
フロントマターを作成し、カテゴリー/トピック/著者のハブ ノートを作成します。
グラフビューの接続ポイントとして機能します。

このスキルは、ユーザーが次のことを行う場合に使用します。
(1) GitHub スターを Obsidian にカタログ化またはインデックス化する
(2) スター付きリポジトリから検索可能なナレッジ ベースを作成する
(3) GitHub スターのパターンを整理して発見する
(4) GitHub スターを構造化されたマークダウン ノートとしてエクスポート
(5) トピック、言語、または著者ごとにスター付きリポジトリのグラフを作成する

特定の URL をメモに保存/抽出するには、代わりに kcap を使用します。
AI ツイートを閲覧するには、代わりに ai-twitter-radar を使用してください。
