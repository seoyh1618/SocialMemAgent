مهارة التشويش للبحث الآلي عن الأخطاء المستندة إلى المدخلات في C/C++. يُستخدم عند إعداد أهداف الضباب libFuzzer أو AFL++، أو تحديد نقاط دخول الضباب حول التحليل أو حدود الإدخال/الإخراج، أو دمج التشويش في CI، أو إدارة المواد المجمعة، أو الجمع بين التشويش والمطهرات. يتم التنشيط عند الاستعلامات حول libFuzzer، أو AFL، أو afl-fuzz، أو أهداف fuzz، أو إدارة المجموعة، أو التشويش الموجه بالتغطية، أو تكامل OSS-Fuzz.
