Compétence de débogage Zig. À utiliser lors du débogage de programmes Zig avec GDB ou LLDB, en interprétant les paniques d'exécution de Zig, en utilisant std.debug.print pour le traçage, la configuration des versions de débogage ou le débogage de programmes Zig dans VS Code. S'active sur les requêtes concernant le débogage de Zig, les paniques Zig, zig gdb, zig lldb, std.debug.print, les traces de pile Zig ou les traces de retour d'erreur Zig.
