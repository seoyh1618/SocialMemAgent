سير عمل تطوير لعبة Stake الشاملة للرياضيات وعقد RGS وتشغيل الواجهة الأمامية وبوابات الامتثال. يُستخدم عند إنشاء ألعاب Stake أو تحديثها، أو تحديد أوضاع اللعبة وأهداف RTP، أو التحقق من صحة بيانات تعريف الكتب/الفهرس التي تم إنشاؤها، أو التحقق من صحة تدفقات الأحداث، أو دمج تشغيل حدث الواجهة الأمامية، أو تنفيذ اتصال RGS ووضع إعادة التشغيل، أو إعداد فحوصات النشر بما في ذلك متطلبات اللغة الاجتماعية والولاية القضائية.
