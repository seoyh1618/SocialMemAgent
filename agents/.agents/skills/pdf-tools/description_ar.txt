هندسة PDF للاستخراج والتوليد والتعديل وملء النماذج. يُستخدم عند استخراج النص أو الجداول من ملفات PDF، أو إنشاء ملفات PDF باستخدام Puppeteer، أو تعديل ملفات PDF باستخدام pdf-lib، أو ملء نماذج PDF، أو تنفيذ أمان PDF. يستخدم للتعرف الضوئي على الحروف بمساعدة الذكاء الاصطناعي، وتحويل HTML إلى PDF، وخطوط معالجة المستندات.
