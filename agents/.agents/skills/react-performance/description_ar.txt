أنماط تحسين أداء React بما في ذلك الحفظ وتقسيم التعليمات البرمجية وتقليل حجم الحزمة وإزالة إعادة العرض والتوصيف. يغطي التحسين التلقائي لـ React Compiler، واستهداف المذكرة اليدوية/useMemo/useCallback، وReact.lazy مع Suspense، وتجنب الملفات البرميلية، ورؤية المحتوى للقوائم الكبيرة، وstartTransition للتحديثات غير العاجلة، وتنميط React DevTools.

يُستخدم عند تحسين أداء تطبيق React، أو تقليل حجم الحزمة، أو التخلص من عمليات إعادة العرض غير الضرورية، أو تصحيح أخطاء المكونات البطيئة، أو تقسيم التعليمات البرمجية، أو تحديد مواصفات اختناقات العرض. يستخدم لتدقيق الأداء، وتحليل الحزمة، وإعادة التشخيص، والتحميل البطيء، والمحاكاة الافتراضية.
