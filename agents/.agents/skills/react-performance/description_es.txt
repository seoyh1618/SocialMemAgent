Patrones de optimización del rendimiento de React que incluyen memorización, división de código, reducción del tamaño del paquete, eliminación de repetición de renderizado y creación de perfiles. Cubre la optimización automática de React Compiler, la orientación manual de memo/useMemo/useCallback, React.lazy con Suspense, prevención de archivos de barril, visibilidad de contenido para listas grandes, startTransition para actualizaciones no urgentes y creación de perfiles de React DevTools.

Úselo al optimizar el rendimiento de la aplicación React, reducir el tamaño del paquete, eliminar renderizaciones innecesarias, depurar componentes lentos, dividir código o crear perfiles de cuellos de botella en la renderización. Úselo para auditoría de rendimiento, análisis de paquetes, diagnóstico de renderizado, carga diferida y virtualización.
