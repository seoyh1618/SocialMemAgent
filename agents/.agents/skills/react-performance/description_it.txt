Modelli di ottimizzazione delle prestazioni di React tra cui memorizzazione, suddivisione del codice, riduzione delle dimensioni del bundle, eliminazione del re-rendering e profilazione. Copre l'ottimizzazione automatica del compilatore React, il targeting manuale di memo/useMemo/useCallback, React.lazy con Suspense, l'evitamento dei file barile, la visibilità dei contenuti per elenchi di grandi dimensioni, startTransition per aggiornamenti non urgenti e la profilazione di React DevTools.

Da utilizzare per l'ottimizzazione delle prestazioni dell'app React, la riduzione delle dimensioni del bundle, l'eliminazione di nuovi rendering non necessari, il debug di componenti lenti, la suddivisione del codice o la profilazione dei colli di bottiglia del rendering. Utilizzare per controllo delle prestazioni, analisi dei bundle, diagnosi di nuovo rendering, caricamento lento, virtualizzazione.
