À utiliser pour optimiser les performances de l'application React, réduire la taille du bundle, éliminer les rendus inutiles, déboguer des composants lents, diviser le code ou profiler les goulots d'étranglement du rendu. À utiliser pour l'audit des performances, l'analyse des bundles, le diagnostic de nouveau rendu, le chargement différé, la virtualisation.
