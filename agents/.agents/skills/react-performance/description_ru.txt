Шаблоны оптимизации производительности React, включая запоминание, разделение кода, уменьшение размера пакета, устранение повторного рендеринга и профилирование. Охватывает автоматическую оптимизацию компилятора React, ручной таргетинг memo/useMemo/useCallback, React.lazy с Suspense, предотвращение бочкообразных файлов, видимость контента для больших списков, startTransition для несрочных обновлений и профилирование React DevTools.

Используйте при оптимизации производительности приложения React, уменьшении размера пакета, устранении ненужных повторных рендерингов, отладке медленных компонентов, разделении кода или профилировании узких мест рендеринга. Используйте для аудита производительности, анализа пакетов, диагностики повторного рендеринга, отложенной загрузки, виртуализации.
