استخدم هذه المهارة عند العمل مع Elasticsearch بأي صفة — تصميم تعيينات الفهرس، أو كتابة الاستعلامات أو تحسينها (Query DSL، ES|QL، KQL)، أو تخطيط بنية المجموعة، أو تكوين مسارات استيعاب، أو ضبط الأداء، أو استكشاف أخطاء سلامة المجموعة وإصلاحها، أو تنفيذ ميزات البحث، أو إنشاء بحث مدعوم بالذكاء الاصطناعي باستخدام المستردات. قم بالتنشيط عندما يذكر المستخدم Elasticsearch، أو OpenSearch، أو Elastic Stack، أو استعلامات Kibana، أو البحث المستند إلى Lucene، أو البحث المتجه/الدلالي/الهجين باستخدام Elasticsearch، أو المستردات، أو LogsDB، أو TSDB، أو Elasticsearch Serverless، أو أي موضوع فهرس/جزء/رسم خرائط/محلل.
