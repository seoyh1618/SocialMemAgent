Создавайте современные интерактивные веб-презентации на базе Reveal.js. Поддерживает как однофайловые HTML-презентации, так и управление репозиториями с несколькими презентациями с помощью развертывания GitHub Pages. Используйте для создания слайдов, презентаций, наборов слайдов, репозиториев презентаций с автоматически генерируемыми индексными страницами. Имеет расширенную анимацию, заметки докладчика, режим обзора и полный доступ к API Reveal.js.
