Erstellen Sie moderne, interaktive Webpräsentationen mit Reveal.js. Unterstützt sowohl Einzeldatei-HTML-Präsentationen als auch die Repository-Verwaltung mehrerer Präsentationen mit GitHub Pages-Bereitstellung. Zum Erstellen von Folien, Präsentationen, Foliendecks und Präsentationsrepositorys mit automatisch generierten Indexseiten. Bietet erweiterte Animationen, Sprechernotizen, Übersichtsmodus und vollständigen Reveal.js-API-Zugriff.
