Interrogez et gérez les problèmes d’analyse statique de BlackDuck Coverity via la CLI Polaris. À utiliser lorsque l'utilisateur pose des questions sur les problèmes de couverture, les résultats de l'analyse de code, les résultats de l'analyse statique, les projets/branches Polaris, l'état de tri, les résultats de rejet ou toute tâche liée à BlackDuck/Polaris. Les expressions déclencheurs incluent « vérifier la couverture », « afficher les problèmes », « lister les projets », « problème de triage », « rejeter les résultats », « polaris », « analyse statique », « résultats de l'analyse du code », « problèmes de couverture », « résultats de sécurité ».
