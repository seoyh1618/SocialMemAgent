Навыки моделирования памяти на C++ и Rust для параллельного программирования. Используйте при понимании порядка памяти, написании структур данных без блокировок, использовании атомов std::atomic или Rust, диагностике гонок данных или выборе правильного порядка памяти для атомарных операций. Активируется при запросах об упорядочении памяти, получении-высвобождении, seq_cst, смягченной атомарности, событиях-раньше, барьерах памяти, std::atomic или атомарном упорядочении Rust.
