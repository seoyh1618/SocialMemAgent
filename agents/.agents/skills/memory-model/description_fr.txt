Compétence en modèle de mémoire C++ et Rust pour la programmation simultanée. À utiliser pour comprendre l'ordre de la mémoire, écrire des structures de données sans verrouillage, utiliser std::atomic ou Rust atomics, diagnostiquer des courses de données ou sélectionner l'ordre de mémoire correct pour les opérations atomiques. S'active sur les requêtes concernant l'ordre de la mémoire, l'acquisition-libération, seq_cst, les atomes détendus, arrive avant, les barrières de mémoire, std::atomic ou l'ordre atomique Rust.
