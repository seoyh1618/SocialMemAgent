Competenze sui modelli di memoria C++ e Rust per la programmazione simultanea. Da utilizzare per comprendere l'ordinamento della memoria, scrivere strutture dati senza blocchi, utilizzare std::atomic o Rust atomics, diagnosticare corse di dati o selezionare l'ordine di memoria corretto per operazioni atomiche. Si attiva su query su ordinamento della memoria, acquire-release, seq_cst, rilassati atomici, succede-prima, barriere di memoria, std::atomic o ordinamento atomico di Rust.
