مهارة نموذج ذاكرة C++ وRust للبرمجة المتزامنة. يُستخدم عند فهم ترتيب الذاكرة، أو كتابة هياكل بيانات خالية من القفل، أو استخدام ذرات std::atomic أو Rust، أو تشخيص سباقات البيانات، أو تحديد ترتيب الذاكرة الصحيح للعمليات الذرية. يتم تنشيطه عند الاستعلامات المتعلقة بترتيب الذاكرة، أو الاكتساب-الإصدار، أو seq_cst، أو الذرات المستقرة، أو ما يحدث قبل ذلك، أو حواجز الذاكرة، أو std::atomic، أو الترتيب الذري الصدأ.
