Analyse une seule trace MLflow pour répondre à une requête utilisateur à ce sujet. À utiliser lorsque l'utilisateur fournit un ID de trace et demande de déboguer, d'enquêter, de rechercher des problèmes, de déterminer la cause première des erreurs, de comprendre le comportement ou d'analyser la qualité. Déclencheurs sur « analyser cette trace », « qu'est-ce qui n'a pas fonctionné avec cette trace », « trace de débogage », « enquêter sur la trace », « pourquoi cette trace a-t-elle échoué », « cause première de cette trace ».
