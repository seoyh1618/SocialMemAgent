Überprüfen Sie Codeänderungen aus mehreren Spezialistenperspektiven gleichzeitig. Verwenden Sie diese Option, wenn Sie eine gründliche Überprüfung einer PR, eines Zweigs oder einer Reihe von Änderungen in Bezug auf Sicherheit, Leistung, Korrektheit, Grenzfälle und Welleneffekte wünschen. Erzeugt parallele Reviewer-Agenten, die sich jeweils auf eine andere Linse konzentrieren, und fügt sie dann zu einer einheitlichen Rezension zusammen.
