Examinez les modifications du code depuis plusieurs perspectives spécialisées en parallèle. À utiliser lorsque vous souhaitez un examen approfondi d'un PR, d'une branche ou d'un ensemble de modifications couvrant la sécurité, les performances, l'exactitude, les cas extrêmes et les effets d'entraînement. Génère des agents de révision parallèles qui se concentrent chacun sur un objectif différent, puis synthétisent en une révision unifiée.
