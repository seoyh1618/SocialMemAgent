Этот навык следует использовать при создании, аудите или обслуживании файлов AGENTS.md и конфигурации Codex CLI для любого проекта, включая инициализацию AGENTS.md для совместимости между агентами (Codex, Cursor, Copilot, Devin, Jules, Amp, Gemini CLI), создание файлов config.toml или .rules, создание шаблонов .agents/skills/, преобразование CLAUDE.md в AGENTS.md или аудит существующих конфигураций агентов на предмет раздувания и устаревания. Дополняет codex-orchestrator (который выполняет субагенты; этот навык создает файлы конфигурации, которые они используют).
