HTML とエンティティ コンポーネント アーキテクチャを使用してブラウザベースの 3D、VR、AR エクスペリエンスを構築するための宣言型 Web フレームワーク。このスキルは、最小限の JavaScript で WebXR アプリケーション、VR エクスペリエンス、AR エクスペリエンス、360 度メディア ビューア、または没入型 Web コンテンツを作成するときに使用します。 A-Frame、WebXR、VR 開発、AR 開発、エンティティ コンポーネント システム、宣言型 3D、または HTML ベースの 3D シーンに関連するタスクをトリガーします。アクセシブルな HTML ファーストのアプローチを採用した Three.js 上に構築されています。
