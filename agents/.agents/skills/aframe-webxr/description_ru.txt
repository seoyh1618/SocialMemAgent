Декларативная веб-инфраструктура для создания браузерных 3D-, VR- и AR-приложений с использованием HTML и архитектуры сущностных компонентов. Используйте этот навык при создании приложений WebXR, приложений виртуальной реальности, дополненной реальности, средств просмотра мультимедиа с обзором на 360 градусов или иммерсивного веб-контента с минимальным использованием JavaScript. Запускается по задачам, включающим A-Frame, WebXR, разработку VR, разработку AR, систему сущностей-компонентов, декларативное 3D или 3D-сцены на основе HTML. Построен на Three.js с доступным подходом, ориентированным на HTML.
