Deklaratives Web-Framework zum Erstellen browserbasierter 3D-, VR- und AR-Erlebnisse mithilfe von HTML und Entity-Component-Architektur. Nutzen Sie diese Fähigkeit, wenn Sie WebXR-Anwendungen, VR-Erlebnisse, AR-Erlebnisse, 360-Grad-Medienbetrachter oder immersive Webinhalte mit minimalem JavaScript erstellen. Wird bei Aufgaben ausgelöst, die A-Frame, WebXR, VR-Entwicklung, AR-Entwicklung, Entity-Component-System, deklaratives 3D oder HTML-basierte 3D-Szenen umfassen. Basierend auf Three.js mit barrierefreiem HTML-First-Ansatz.
