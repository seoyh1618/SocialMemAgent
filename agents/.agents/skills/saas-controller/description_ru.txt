Настраивайте и управляйте службами saas-контроллера в проектах devenv. Используйте при настройке облачных служб (Zuplo, Frontegg), запуске sc up для локальной разработки, развертывании с помощью sc Deploy, управлении секретами с помощью профилей secretspec или устранении неполадок saas-контроллера.
