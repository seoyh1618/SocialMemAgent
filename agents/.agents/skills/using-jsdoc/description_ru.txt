Руководство по документированию кода JavaScript и TypeScript с использованием JSDoc. Используйте при добавлении документации к функциям, классам, методам, переменным, типам, интерфейсам, модулям или любому элементу кода. Включает синтаксис тегов, выражения типов, шаблоны для параметров, возвраты, универсальные шаблоны и документацию, специфичную для TypeScript. Запускайте такие фразы, как «задокументировать эту функцию», «добавить JSDoc», «документировать с помощью JSDoc» или «добавить комментарии к документации».
