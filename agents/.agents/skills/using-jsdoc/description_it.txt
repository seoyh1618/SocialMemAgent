Guida per documentare il codice JavaScript e TypeScript utilizzando JSDoc. Da utilizzare quando si aggiunge documentazione a funzioni, classi, metodi, variabili, tipi, interfacce, moduli o qualsiasi elemento di codice. Include sintassi dei tag, espressioni di tipo, modelli per parametri, resi, generici e documentazione specifica di TypeScript. Attivalo con frasi come "documenta questa funzione", "aggiungi JSDoc", "documenta con JSDoc" o "aggiungi commenti alla documentazione".
