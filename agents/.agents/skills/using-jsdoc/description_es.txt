Guía para documentar código JavaScript y TypeScript usando JSDoc. Úselo al agregar documentación a funciones, clases, métodos, variables, tipos, interfaces, módulos o cualquier elemento de código. Incluye sintaxis de etiquetas, expresiones de tipo, patrones para parámetros, devoluciones, genéricos y documentación específica de TypeScript. Dispare con frases como "documentar esta función", "agregar JSDoc", "documentar con JSDoc" o "agregar comentarios de documentación".
