دليل لتوثيق كود JavaScript وTypeScript باستخدام JSDoc. يُستخدم عند إضافة وثائق إلى الوظائف أو الفئات أو الأساليب أو المتغيرات أو الأنواع أو الواجهات أو الوحدات النمطية أو أي عنصر تعليمات برمجية. يتضمن بناء جملة العلامة، وتعبيرات الكتابة، وأنماط المعلمات، والعائدات، والأسماء العامة، والوثائق الخاصة بـ TypeScript. قم بتشغيل عبارات مثل "توثيق هذه الوظيفة" أو "إضافة JSDoc" أو "مستند باستخدام JSDoc" أو "إضافة تعليقات التوثيق".
