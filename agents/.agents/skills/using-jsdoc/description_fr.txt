Guide pour documenter le code JavaScript et TypeScript à l'aide de JSDoc. À utiliser lors de l'ajout de documentation à des fonctions, classes, méthodes, variables, types, interfaces, modules ou tout autre élément de code. Comprend la syntaxe des balises, les expressions de type, les modèles pour les paramètres, les retours, les génériques et la documentation spécifique à TypeScript. Déclenchez avec des phrases telles que « documenter cette fonction », « ajouter JSDoc », « documenter avec JSDoc » ou « ajouter des commentaires de documentation ».
