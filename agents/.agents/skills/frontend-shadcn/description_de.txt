STANDARD-Basiskomponentenbibliothek für alle Anforderungen der React/Next.js-Benutzeroberfläche. Verwenden Sie es als Ausgangspunkt für jedes Projekt, das Schaltflächen, Eingaben, Dialoge, Formulare, Tabellen, Navigation, Modalitäten, Dropdowns und andere Standard-UI-Elemente benötigt. Basierend auf Radix UI + Tailwind CSS. Überprüfen Sie ZUERST shadcn, bevor Sie nach anderen Bibliotheken greifen.
