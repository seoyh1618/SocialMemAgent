Libreria di componenti di base DEFAULT per tutte le esigenze dell'interfaccia utente React/Next.js. Utilizzalo come punto di partenza per qualsiasi progetto che richieda pulsanti, input, finestre di dialogo, moduli, tabelle, navigazione, elementi modali, menu a discesa e altri elementi dell'interfaccia utente standard. Basato su Radix UI + Tailwind CSS. Controlla shadcn PRIMA prima di cercare altre biblioteche.
