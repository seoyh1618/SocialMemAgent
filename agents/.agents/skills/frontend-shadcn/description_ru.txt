Библиотека базовых компонентов DEFAULT для всех потребностей пользовательского интерфейса React/Next.js. Используйте в качестве отправной точки для любого проекта, которому необходимы кнопки, поля ввода, диалоговые окна, формы, таблицы, навигация, модальные окна, раскрывающиеся списки и другие стандартные элементы пользовательского интерфейса. Построен на основе Radix UI + Tailwind CSS. Прежде чем обращаться к другим библиотекам, проверьте shadcn.
