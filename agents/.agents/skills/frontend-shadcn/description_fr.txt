Bibliothèque de composants de base DEFAULT pour tous les besoins de l'interface utilisateur React/Next.js. Utilisez-le comme point de départ pour tout projet nécessitant des boutons, des entrées, des boîtes de dialogue, des formulaires, des tableaux, une navigation, des modaux, des listes déroulantes et d'autres éléments d'interface utilisateur standard. Construit sur Radix UI + Tailwind CSS. Vérifiez shadcn D’ABORD avant de rechercher d’autres bibliothèques.
