Создает конечные точки HTTP для приложений Wix CLI. Используйте только тогда, когда пользователь специально запрашивает конечную точку серверной части. Используйте при создании конечных точек REST API, серверных обработчиков HTTP или серверной логики. Триггеры включают в себя внутренний API, конечную точку HTTP, методы HTTP, обработку форм, загрузку файлов.
