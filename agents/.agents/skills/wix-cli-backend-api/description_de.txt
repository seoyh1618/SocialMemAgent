Erstellt HTTP-Endpunkte für Wix CLI-Apps. Nur verwenden, wenn der Benutzer ausdrücklich nach einem Backend-Endpunkt fragt. Wird beim Erstellen von REST-API-Endpunkten, Backend-HTTP-Handlern oder serverseitiger Logik verwendet. Zu den Auslösern gehören Backend-API, HTTP-Endpunkt, HTTP-Methoden, Formularverarbeitung und Datei-Uploads.
