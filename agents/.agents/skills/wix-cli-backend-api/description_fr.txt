Crée des points de terminaison HTTP pour les applications Wix CLI. À utiliser uniquement lorsque l'utilisateur demande spécifiquement un point de terminaison backend. À utiliser lors de la création de points de terminaison d'API REST, de gestionnaires HTTP back-end ou d'une logique côté serveur. Les déclencheurs incluent l'API backend, le point de terminaison HTTP, les méthodes HTTP, la gestion des formulaires et les téléchargements de fichiers.
