ينشئ نقاط نهاية HTTP لتطبيقات Wix CLI. يُستخدم فقط عندما يطلب المستخدم على وجه التحديد نقطة نهاية خلفية. يُستخدم عند إنشاء نقاط نهاية REST API أو معالجات HTTP الخلفية أو المنطق من جانب الخادم. تتضمن المشغلات واجهة برمجة تطبيقات الواجهة الخلفية ونقطة نهاية HTTP وأساليب HTTP ومعالجة النماذج وتحميل الملفات.
