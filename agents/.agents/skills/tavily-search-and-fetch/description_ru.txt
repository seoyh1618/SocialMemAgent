Веб-(исследования) поиск с использованием API-интерфейсов Tavily. Выполняйте поиск в Интернете, извлекайте контент из URL-адресов, находите карты сайтов или получайте синтезированные с помощью ИИ отчеты об исследованиях с цитатами. Используйте для фактического поиска, текущих событий, финансовых данных, полного извлечения контента, URL-адресов карт сайта или подробных исследовательских отчетов. Темы: общее, новости, финансы.
