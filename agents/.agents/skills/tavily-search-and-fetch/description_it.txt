(Ri)ricerca sul Web utilizzando le API Tavily. Cerca sul Web, estrai contenuti dagli URL, scopri le mappe dei siti o ottieni rapporti di ricerca sintetizzati dall'intelligenza artificiale con citazioni. Utilizzare per ricerche basate su fatti, eventi attuali, dati finanziari, estrazione completa di contenuti, URL della mappa del sito o rapporti di ricerca completi. Argomenti: generale, attualità, finanza.
