Используйте при создании виджетов, которые расширяют страницы панели управления Wix для сторонних бизнес-приложений. Триггеры включают в себя плагин информационной панели, слот информационной панели, расширение страницы информационной панели, плагин Wix Stores, плагин Wix Bookings, плагин Wix Blog, ObserveState, виджет информационной панели. НЕ используйте для отдельных страниц информационной панели (используйте wix-cli-dashboard-page) или пользовательского интерфейса, ориентированного на сайт (используйте виджеты сайта/плагины сайта).
