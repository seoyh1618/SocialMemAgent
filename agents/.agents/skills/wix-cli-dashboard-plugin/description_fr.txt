À utiliser lors de la création de widgets qui étendent les pages du tableau de bord Wix pour les applications professionnelles propriétaires. Les déclencheurs incluent le plugin de tableau de bord, l'emplacement de tableau de bord, l'extension de la page du tableau de bord, le plugin Wix Stores, le plugin Wix Bookings, le plugin Wix Blog, observeState, le widget de tableau de bord. Ne PAS utiliser pour les pages de tableau de bord autonomes (utilisez wix-cli-dashboard-page) ou l'interface utilisateur du site (utilisez les widgets/plugins de site).
