Da utilizzare quando si creano widget che estendono le pagine della dashboard Wix per le app aziendali di prima parte. I trigger includono plug-in dashboard, slot dashboard, estensione pagina dashboard, plug-in Wix Stores, plug-in Wix Bookings, plug-in Wix Blog,observerState, widget dashboard. NON utilizzare per pagine dashboard autonome (utilizza wix-cli-dashboard-page) o interfaccia utente rivolta al sito (utilizza widget/plug-in del sito).
