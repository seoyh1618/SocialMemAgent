Úselo al crear widgets que extiendan las páginas del panel de Wix para aplicaciones comerciales propias. Los activadores incluyen el complemento del panel, la ranura del panel, ampliar la página del panel, el complemento de Wix Stores, el complemento de Wix Bookings, el complemento de Wix Blog, observeState, el widget del panel. NO lo use para páginas de panel independientes (use wix-cli-dashboard-page) o interfaz de usuario orientada al sitio (use widgets/complementos del sitio).
