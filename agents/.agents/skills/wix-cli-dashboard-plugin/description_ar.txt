استخدمه عند إنشاء عناصر واجهة مستخدم تعمل على توسيع صفحات لوحة تحكم Wix لتطبيقات أعمال الطرف الأول. تتضمن المشغلات المكوّن الإضافي للوحة المعلومات، وفتحة لوحة المعلومات، وصفحة لوحة المعلومات الموسعة، والمكوّن الإضافي لمتاجر Wix، والمكوّن الإضافي لـ Wix Bookings، والمكوّن الإضافي لمدونة Wix، وobservateState، وعنصر واجهة المستخدم للوحة المعلومات. لا تستخدمه لصفحات لوحة المعلومات المستقلة (استخدم صفحة wix-cli-dashboard) أو واجهة المستخدم المواجهة للموقع (استخدم أدوات الموقع/المكونات الإضافية للموقع).
