Verwenden Sie diese Option, wenn Sie Widgets erstellen, die die Dashboard-Seiten von Wix für Erstanbieter-Geschäftsanwendungen erweitern. Zu den Auslösern gehören Dashboard-Plugin, Dashboard-Slot, erweiterte Dashboard-Seite, Wix Stores-Plugin, Wix Bookings-Plugin, Wix Blog-Plugin, ObservState und Dashboard-Widget. NICHT für eigenständige Dashboard-Seiten (verwenden Sie wix-cli-dashboard-page) oder eine Website-bezogene Benutzeroberfläche (verwenden Sie Website-Widgets/Website-Plugins) verwenden.
