مجموعة أدوات ضمان الجودة المتوافقة مع مستوى الأساس ISTQB (CTFL) للاختبار اليدوي والآلي. استخدم عندما يُطلب منك إنشاء خطط اختبار، أو إستراتيجيات اختبار، أو شروط اختبار، أو حالات اختبار، أو تقارير أخطاء، أو سجلات عيوب، أو مجموعات انحدار، أو مصفوفات إمكانية التتبع، أو مواثيق استكشافية. يدعم الاختبار القائم على المخاطر، وتقنيات تصميم الاختبار (تقسيم التكافؤ، وتحليل القيمة الحدودية، وجداول القرار، وانتقالات الحالة)، وتقدير الاختبار، ومراجعات الاختبار الثابت، وإدارة عملية الاختبار. يتضمن إرشادات أتمتة الكاتب المسرحي لتنفيذ الاختبار.
