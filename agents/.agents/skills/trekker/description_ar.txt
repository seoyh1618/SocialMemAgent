تتبع مهام Trekker CLI لوكلاء ترميز الذكاء الاصطناعي. يغطي الملاحم والمهام والمهام الفرعية والتبعيات والتعليقات والبحث عن النص الكامل وسجل تدقيق السجل وسير العمل بنمط كانبان مع تخزين SQLite المحلي.

يُستخدم عند إنشاء أدوات تتبع المهام، أو إدارة جلسات عمل الوكيل، أو تتبع الملحمات والتبعيات، أو تنفيذ تتبع المشكلات المحلية أولاً باستخدام Trekker CLI.
