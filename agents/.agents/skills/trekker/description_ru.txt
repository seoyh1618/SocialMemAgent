Отслеживание задач Trekker CLI для агентов кодирования искусственного интеллекта. Охватывает эпики, задачи, подзадачи, зависимости, комментарии, полнотекстовый поиск, журнал аудита истории и рабочий процесс в стиле канбан с локальным хранилищем SQLite.

Используйте при создании средств отслеживания задач, управлении сеансами работы агентов, отслеживании эпиков и зависимостей или реализации локального отслеживания проблем с помощью Trekker CLI.
