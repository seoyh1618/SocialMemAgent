透過逆向工程 Gemini Web API 產生文字和圖像。

核心能力：
- 文字生成：使用 Gemini 模型產生文字回應
- 圖像生成：根據文字提示建立圖像並將其保存在本地
- 視覺輸入：支援圖像到圖像或圖像到文字的參考圖像
- 多輪對話：透過sessionId保持上下文連續性

何時使用：
- 當您需要產生高品質的AI圖像（封面、插圖等）時
- 當您需要透過 Gemini 模型產生文字時
- 當您想要根據參考影像建立變體或描述時
- 作為其他技能的圖像生成後端（pw-cover-image、pw-redbook-image）

不適合：
- 生產環境需要官方API支援和穩定性保證
- 對 API 更改敏感的關鍵業務流程
- 高頻批量呼叫（可能會觸發速率限制）

重要提醒：
- 首次使用需要用戶同意免責聲明
- 需要Google帳號驗證（瀏覽器登入自動開啟）
- 中國大陸需要代理配置
- 這是一個非官方 API，可能隨時中斷
