リバースエンジニアリングされた Gemini Web API によるテキストと画像の生成。

コア機能:
- テキスト生成: Gemini モデルを使用してテキスト応答を生成します。
- 画像生成: テキスト プロンプトから画像を作成し、ローカルに保存します。
- 視覚入力: 画像から画像または画像からテキストへの参照画像をサポート
- マルチターンダイアログ: sessionId を通じてコンテキストの連続性を維持します。

いつ使用するか:
・高品質なAI画像（表紙、イラストなど）を生成したい場合
- Gemini モデルによるテキスト生成が必要な場合
- 参考画像を元にバリエーションや説明文を作成したい場合
- 他のスキルの画像生成バックエンドとして (pw-cover-image、pw-redbook-image)

以下には適していません:
- 公式 API サポートと安定性保証を必要とする運用環境
- API の変更に敏感な重要なビジネス プロセス
- 高頻度のバッチ呼び出し (レート制限がトリガーされる可能性があります)

重要な注意事項:
- 初めて使用する場合、ユーザーは免責事項に同意する必要があります
- Googleアカウント認証が必要（ブラウザログインが自動で開きます）
- 中国本土ではプロキシ構成が必要です
- これは非公式 API であるため、いつでも壊れる可能性があります
