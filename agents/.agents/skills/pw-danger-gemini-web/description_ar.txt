إنشاء النصوص والصور عبر واجهة برمجة تطبيقات Gemini Web API ذات الهندسة العكسية.

القدرات الأساسية:
- إنشاء النص: إنشاء استجابات نصية باستخدام نموذج الجوزاء
- إنشاء الصور: إنشاء صور من المطالبات النصية وحفظها محليًا
- الإدخال المرئي: دعم الصور المرجعية لتحويل الصورة إلى صورة أو صورة إلى نص
- حوار متعدد المنعطفات: حافظ على استمرارية السياق من خلال معرف الجلسة

متى تستخدم:
- عندما تحتاج إلى إنشاء صور عالية الجودة بتقنية الذكاء الاصطناعي (الأغلفة والرسوم التوضيحية وما إلى ذلك)
- عندما تحتاج إلى إنشاء نص عبر نموذج الجوزاء
- عندما تريد إنشاء متغيرات أو أوصاف بناءً على صورة مرجعية
- كخلفية لتوليد الصور لمهارات أخرى (pw-cover-image، pw-redbook-image)

غير مناسب لـ:
- بيئات الإنتاج التي تتطلب دعم API الرسمي وضمانات الاستقرار
- العمليات التجارية الهامة الحساسة لتغيرات واجهة برمجة التطبيقات
- مكالمات جماعية عالية التردد (قد تؤدي إلى حدود للمعدل)

تذكيرات هامة:
- يتطلب الاستخدام لأول مرة موافقة المستخدم على إخلاء المسؤولية
- مطلوب مصادقة حساب Google (يتم فتح تسجيل الدخول للمتصفح تلقائيًا)
- تكوين الوكيل مطلوب في البر الرئيسي للصين
- هذه واجهة برمجة تطبيقات غير رسمية وقد تنقطع في أي وقت
