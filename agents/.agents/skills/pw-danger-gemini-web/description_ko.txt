리버스 엔지니어링된 Gemini Web API를 통한 텍스트 및 이미지 생성.

핵심 기능:
- 텍스트 생성: Gemini 모델을 사용하여 텍스트 응답 생성
- 이미지 생성: 텍스트 프롬프트에서 이미지를 생성하고 로컬에 저장합니다.
- 시각적 입력: 이미지 대 이미지 또는 이미지 대 텍스트에 대한 참조 이미지 지원
- 다중 회전 대화: sessionId를 통해 컨텍스트 연속성을 유지합니다.

사용 시기:
- 고품질의 AI 이미지(표지, 일러스트 등)를 생성해야 하는 경우
- Gemini 모델을 통한 텍스트 생성이 필요한 경우
- 참조 이미지를 기반으로 변형이나 설명을 만들고 싶은 경우
- 다른 기술을 위한 이미지 생성 백엔드로 사용(pw-cover-image, pw-redbook-image)

적합하지 않은 대상:
- 공식 API 지원 및 안정성 보장이 필요한 프로덕션 환경
- API 변경에 민감한 중요한 비즈니스 프로세스
- 고주파 일괄 호출(속도 제한이 발생할 수 있음)

중요 알림:
- 최초 사용시 면책조항에 대한 사용자 동의가 필요합니다.
- 구글 계정 인증 필요 (브라우저 로그인 자동열림)
- 중국 본토에 프록시 구성이 필요함
- 비공식 API이므로 언제든지 중단될 수 있습니다.
