通过逆向工程 Gemini Web API 生成文本和图像。

核心能力：
- 文本生成：使用 Gemini 模型生成文本响应
- 图像生成：根据文本提示创建图像并将其保存在本地
- 视觉输入：支持图像到图像或图像到文本的参考图像
- 多轮对话：通过sessionId保持上下文连续性

何时使用：
- 当您需要生成高质量的AI图像（封面、插图等）时
- 当您需要通过 Gemini 模型生成文本时
- 当您想要根据参考图像创建变体或描述时
- 作为其他技能的图像生成后端（pw-cover-image、pw-redbook-image）

不适合：
- 生产环境需要官方API支持和稳定性保证
- 对 API 更改敏感的关键业务流程
- 高频批量调用（可能会触发速率限制）

重要提醒：
- 首次使用需要用户同意免责声明
- 需要谷歌帐户身份验证（浏览器登录自动打开）
- 中国大陆需要代理配置
- 这是一个非官方 API，可能随时中断
