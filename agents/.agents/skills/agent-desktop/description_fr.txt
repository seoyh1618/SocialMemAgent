Automatisation du bureau via des arborescences d'accessibilité natives du système d'exploitation à l'aide de la CLI agent-desktop. À utiliser lorsqu'un agent IA doit observer, interagir avec ou automatiser des applications de bureau (cliquer sur des boutons, remplir des formulaires, naviguer dans les menus, lire l'état de l'interface utilisateur, cocher des cases, faire défiler, faire glisser, saisir du texte, prendre des captures d'écran, gérer les fenêtres, utiliser le presse-papiers). Couvre 50 commandes pour l'observation, l'interaction, le clavier/souris, le cycle de vie des applications, le presse-papiers et l'attente. Se déclenche sur : "cliquer sur le bouton", "remplir le formulaire", "ouvrir l'application", "lire l'interface utilisateur", "automatiser le bureau", "l'arborescence d'accessibilité", "l'application instantanée", "saisir dans le champ", "menu de navigation", "activer la case à cocher", "prendre une capture d'écran", "automatisation du bureau", "agent-bureau" ou toute tâche d'interaction avec l'interface graphique du bureau. Prend en charge macOS (Phase 1), avec Windows et Linux prévus.
