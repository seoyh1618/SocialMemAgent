أتمتة سطح المكتب عبر أشجار إمكانية الوصول إلى نظام التشغيل الأصلي باستخدام واجهة سطر الأوامر لسطح المكتب للوكيل. يُستخدم عندما يحتاج وكيل الذكاء الاصطناعي إلى مراقبة تطبيقات سطح المكتب أو التفاعل معها أو تشغيلها تلقائيًا (النقر على الأزرار، وملء النماذج، والتنقل في القوائم، وقراءة حالة واجهة المستخدم، وتبديل مربعات الاختيار، والتمرير، والسحب، وكتابة النص، والتقاط لقطات الشاشة، وإدارة النوافذ، واستخدام الحافظة). يغطي 50 أمرًا عبر المراقبة والتفاعل ولوحة المفاتيح/الماوس ودورة حياة التطبيق والحافظة والانتظار. المشغلات عند: "النقر على الزر"، أو "ملء النموذج"، أو "فتح التطبيق"، أو "قراءة واجهة المستخدم"، أو "أتمتة سطح المكتب"، أو "شجرة إمكانية الوصول"، أو "تطبيق اللقطة"، أو "الكتابة في الحقل"، أو "التنقل في القائمة"، أو "تبديل مربع الاختيار"، أو "التقاط لقطة شاشة"، أو "أتمتة سطح المكتب"، أو "وكيل سطح المكتب"، أو أي مهمة تفاعل لواجهة المستخدم الرسومية لسطح المكتب. يدعم نظام التشغيل macOS (المرحلة 1)، مع التخطيط لنظامي التشغيل Windows وLinux.
