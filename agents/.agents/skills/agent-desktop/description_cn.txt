使用代理桌面 CLI 通过本机操作系统可访问性树进行桌面自动化。当 AI 代理需要观察、交互或自动化桌面应用程序（单击按钮、填写表单、导航菜单、读取 UI 状态、切换复选框、滚动、拖动、键入文本、截取屏幕截图、管理窗口、使用剪贴板）时使用。涵盖 50 个命令，涉及观察、交互、键盘/鼠标、应用程序生命周期、剪贴板和等待。触发：“单击按钮”、“填写表单”、“打开应用程序”、“读取 UI”、“自动化桌面”、“辅助功能树”、“快照应用程序”、“在字段中键入”、“导航菜单”、“切换复选框”、“截屏”、“桌面自动化”、“代理桌面”或任何桌面 GUI 交互任务。支持 macOS（第一阶段），计划支持 Windows 和 Linux。
