에이전트-데스크탑 CLI를 사용하는 기본 OS 접근성 트리를 통한 데스크탑 자동화. AI 에이전트가 데스크톱 애플리케이션을 관찰, 상호 작용 또는 자동화해야 할 때 사용합니다(버튼 클릭, 양식 채우기, 메뉴 탐색, UI 상태 읽기, 확인란 전환, 스크롤, 드래그, 텍스트 입력, 스크린샷 찍기, 창 관리, 클립보드 사용). 관찰, 상호 작용, 키보드/마우스, 앱 수명 주기, 클립보드 및 대기 전반에 걸쳐 50가지 명령을 다룹니다. 트리거: "버튼 클릭", "양식 채우기", "앱 열기", "UI 읽기", "데스크탑 자동화", "접근성 트리", "스냅샷 앱", "필드에 입력", "메뉴 탐색", "체크박스 전환", "스크린샷 찍기", "데스크톱 자동화", "에이전트-데스크톱" 또는 모든 데스크톱 GUI 상호 작용 작업. macOS(1단계)를 지원하며 Windows 및 Linux도 지원될 예정입니다.
