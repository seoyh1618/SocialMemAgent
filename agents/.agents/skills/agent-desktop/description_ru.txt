Автоматизация рабочего стола с помощью собственных деревьев доступности ОС с использованием интерфейса командной строки агента-рабочего стола. Используйте, когда агенту ИИ необходимо наблюдать, взаимодействовать с настольными приложениями или автоматизировать их (нажимать кнопки, заполнять формы, перемещаться по меню, читать состояние пользовательского интерфейса, переключать флажки, прокручивать, перетаскивать, вводить текст, делать снимки экрана, управлять окнами, использовать буфер обмена). Охватывает 50 команд для наблюдения, взаимодействия, клавиатуры/мыши, жизненного цикла приложения, буфера обмена и ожидания. Запускается по: «нажать кнопку», «заполнить форму», «открыть приложение», «прочитать пользовательский интерфейс», «автоматизировать рабочий стол», «дерево доступности», «снимок приложения», «ввести текст в поле», «навигация по меню», «переключить флажок», «сделать снимок экрана», «автоматизация рабочего стола», «агент-рабочий стол» или любую задачу взаимодействия с графическим интерфейсом рабочего стола. Поддерживает macOS (фаза 1), планируется использовать Windows и Linux.
