使用代理桌面 CLI 透過本機作業系統可存取性樹實現桌面自動化。當 AI 代理程式需要觀察、互動或自動化桌面應用程式（點擊按鈕、填寫表單、導航選單、讀取 UI 狀態、切換複選框、捲動、拖曳、鍵入文字、截取螢幕截圖、管理視窗、使用剪貼簿）時使用。涵蓋 50 個命令，涉及觀察、互動、鍵盤/滑鼠、應用程式生命週期、剪貼簿和等待。觸發：「點擊按鈕」、「填寫表單」、「開啟應用程式」、「讀取 UI」、「自動化桌面」、「輔助功能樹」、「快照應用程式」、「在欄位中鍵入」、「導覽功能表」、「切換複選框」、「截圖」、「桌面自動化」、「代理桌面」或任何桌面 GUI 互動式任務。支援 macOS（第一階段），計劃支援 Windows 和 Linux。
