エージェント デスクトップ CLI を使用したネイティブ OS アクセシビリティ ツリーによるデスクトップ オートメーション。 AI エージェントがデスクトップ アプリケーションを観察、操作、または自動化する必要がある場合に使用します (ボタンのクリック、フォームへの入力、メニューの移動、UI 状態の読み取り、チェックボックスの切り替え、スクロール、ドラッグ、テキストの入力、スクリーンショットの撮影、ウィンドウの管理、クリップボードの使用)。観察、対話、キーボード/マウス、アプリのライフサイクル、クリップボード、待機にわたる 50 のコマンドをカバーします。トリガー: 「ボタンをクリック」、「フォームに入力」、「アプリを開く」、「UI の読み取り」、「デスクトップの自動化」、「アクセシビリティ ツリー」、「アプリのスナップショット」、「フィールドへの入力」、「メニューへの移動」、「チェックボックスの切り替え」、「スクリーンショットの取得」、「デスクトップ オートメーション」、「エージェントとデスクトップ」、または任意のデスクトップ GUI インタラクション タスク。 macOS (フェーズ 1) をサポートし、Windows と Linux もサポートする予定です。
