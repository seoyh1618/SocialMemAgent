Automazione desktop tramite alberi di accessibilità del sistema operativo nativo utilizzando la CLI dell'agente-desktop. Da utilizzare quando un agente AI deve osservare, interagire o automatizzare le applicazioni desktop (fare clic su pulsanti, compilare moduli, navigare nei menu, leggere lo stato dell'interfaccia utente, attivare/disattivare caselle di controllo, scorrere, trascinare, digitare testo, acquisire screenshot, gestire finestre, utilizzare gli appunti). Copre 50 comandi tra osservazione, interazione, tastiera/mouse, ciclo di vita dell'app, appunti e attesa. Si attiva su: "pulsante di clic", "compila modulo", "apri app", "leggi interfaccia utente", "automatizzazione del desktop", "albero di accessibilità", "app per istantanee", "digitazione nel campo", "menu di navigazione", "attiva/disattiva casella di controllo", "acquisisci screenshot", "automazione del desktop", "agente-desktop" o qualsiasi attività di interazione con la GUI del desktop. Supporta macOS (Fase 1), con Windows e Linux pianificati.
