Desktop-Automatisierung über native Betriebssystem-Zugänglichkeitsbäume mithilfe der Agent-Desktop-CLI. Wird verwendet, wenn ein KI-Agent Desktop-Anwendungen beobachten, mit ihnen interagieren oder sie automatisieren muss (Schaltflächen anklicken, Formulare ausfüllen, Menüs navigieren, UI-Status lesen, Kontrollkästchen umschalten, scrollen, ziehen, Text eingeben, Screenshots erstellen, Fenster verwalten, Zwischenablage verwenden). Deckt 50 Befehle in den Bereichen Beobachtung, Interaktion, Tastatur/Maus, App-Lebenszyklus, Zwischenablage und Warten ab. Wird ausgelöst bei: „Schaltfläche anklicken“, „Formular ausfüllen“, „App öffnen“, „Benutzeroberfläche lesen“, „Desktop automatisieren“, „Eingabehilfenbaum“, „Snapshot-App“, „In Feld eingeben“, „Menü navigieren“, „Kontrollkästchen umschalten“, „Screenshot erstellen“, „Desktop-Automatisierung“, „Agent-Desktop“ oder jede Desktop-GUI-Interaktionsaufgabe. Unterstützt macOS (Phase 1), Windows und Linux sind geplant.
