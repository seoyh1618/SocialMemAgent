Regole di modifica del modello di autorizzazione IAM RBAC, che coprono le definizioni del tipo di risorsa, la configurazione delle autorizzazioni dell'operazione, gli script di migrazione delle autorizzazioni, l'implementazione del callback IAM. Utilizzato quando gli utenti modificano il modello di autorizzazione, aggiungono nuovi tipi di risorse, configurano le autorizzazioni operative o scrivono script di migrazione delle autorizzazioni.
