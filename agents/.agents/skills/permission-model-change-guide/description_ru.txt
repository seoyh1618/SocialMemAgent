Правила изменения модели разрешений IAM RBAC, охватывающие определения типов ресурсов, конфигурацию разрешений на операции, сценарии миграции разрешений, реализацию обратного вызова IAM. Используется, когда пользователи изменяют модель разрешений, добавляют новые типы ресурсов, настраивают разрешения на операции или пишут сценарии миграции разрешений.
