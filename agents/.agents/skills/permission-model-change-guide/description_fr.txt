Règles de modification du modèle d'autorisation IAM RBAC, couvrant les définitions de types de ressources, la configuration des autorisations d'opération, les scripts de migration d'autorisations et la mise en œuvre des rappels IAM. Utilisé lorsque les utilisateurs modifient le modèle d'autorisation, ajoutent de nouveaux types de ressources, configurent des autorisations d'opération ou écrivent des scripts de migration d'autorisation.
