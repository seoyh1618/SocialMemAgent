Reglas de cambio del modelo de permisos de IAM RBAC, que cubren definiciones de tipos de recursos, configuración de permisos de operación, scripts de migración de permisos e implementación de devolución de llamadas de IAM. Se utiliza cuando los usuarios modifican el modelo de permisos, agregan nuevos tipos de recursos, configuran permisos de operación o escriben scripts de migración de permisos.
