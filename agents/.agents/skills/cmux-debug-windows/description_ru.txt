Управляйте окнами отладки cmux и соответствующей проводкой меню отладки для отладки боковой панели, фоновой отладки и дополнительной отладки строки меню. Используйте это, когда пользователь просит открыть/настроить эти элементы управления отладкой, добавить или изменить записи меню «Отладка» или сделать/копировать объединенный снимок конфигурации отладки.
