Gérez les fenêtres de débogage cmux et le câblage du menu de débogage associé pour le débogage de la barre latérale, le débogage en arrière-plan et le débogage supplémentaire de la barre de menus. Utilisez-le lorsque l'utilisateur demande d'ouvrir/régler ces contrôles de débogage, d'ajouter ou d'ajuster les entrées du menu Debug, ou de capturer/copier un instantané de configuration de débogage combiné.
