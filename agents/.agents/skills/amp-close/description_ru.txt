Ритуал завершения сеанса усилителя. Вызов FIRST при завершении сеанса — 4-фазный цикл GODAR (сбор контекста, ориентация посредством отражения, принятие решения «сейчас-следующий», действие с передачей обслуживания и предварительным извлечением), который предотвращает молчаливое прекращение работы, записывает перекрестные передачи обслуживания, которые CC может обнаружить, и сохраняет доступную для поиска память. ОБЯЗАТЕЛЬНО перед закрытием любой существенной темы Amp. Срабатывает при «подведении итогов», «давайте закончим», «закрытие», «окончание сеанса». (пользователь)
