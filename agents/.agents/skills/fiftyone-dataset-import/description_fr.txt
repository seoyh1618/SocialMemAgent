Importe des ensembles de données dans FiftyOne avec détection automatique du format. Prend en charge tous les types de médias (images, vidéos, nuages ​​de points), les formats d'étiquettes (COCO, YOLO, VOC, KITTI), les ensembles de données groupés multimodaux et les ensembles de données Hugging Face Hub. À utiliser lors de l'importation d'ensembles de données à partir de fichiers locaux ou de Hugging Face, du chargement de données de conduite autonome ou de la création d'ensembles de données groupés.
