Konsolidieren Sie eine Databricks REPL-Sitzung in einer einzigen, sauberen Python-Datei. Verwenden Sie diese Fähigkeit, wenn der Benutzer eine REPL-Sitzung in einem festschreibbaren Skript abschließen, exportieren oder konsolidieren möchte. Löst auf Anfragen aus, um die Sitzungsausgabe zu konsolidieren, ein endgültiges Skript aus REPL-Befehlen zu erstellen, eine Sitzung nach Python zu exportieren, REPL-Artefakte in Produktionscode zu bereinigen oder einen Databricks-Workflow abzuschließen.
