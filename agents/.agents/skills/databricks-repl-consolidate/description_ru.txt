Объедините сеанс REPL Databricks в один чистый файл Python. Используйте этот навык, когда пользователь хочет завершить, экспортировать или объединить сеанс REPL в фиксируемый сценарий. Запускается по запросам на консолидацию выходных данных сеанса, создание окончательного сценария из команд REPL, экспорт сеанса в Python, очистку артефактов REPL в рабочий код или завершение рабочего процесса Databricks.
