アカウントの作成、API キーの生成、Senpi MCP サーバー接続の構成により、AI エージェントを Senpi 取引プラットフォームにオンボードします。ウォレット、テレグラム、またはエージェントが生成したウォレット ID をサポートします。ユーザーが「Senpi をセットアップする」、「Senpi にオンボードする」、「Senpi に接続する」、「Senpi をインストールする」、「Senpi に登録する」と言う場合、または自律取引のためにエージェントが Senpi に自己登録する必要がある場合に使用します。取引操作、戦略管理、または市場クエリには使用しないでください。それらには、Senpi MCP サーバーがすでに接続されている必要があります。
