Подключает агента ИИ к торговой платформе Senpi путем создания учетной записи, генерации ключа API и настройки подключения к серверу Senpi MCP. Поддерживает кошелек, Telegram или идентификатор кошелька, созданный агентом. Используйте, когда пользователь говорит «настроить Senpi», «подключиться к Senpi», «подключиться к Senpi», «установить Senpi», «зарегистрироваться в Senpi» или когда агенту необходимо самостоятельно зарегистрироваться в Senpi для автономной торговли. НЕ используйте для торговых операций, управления стратегиями или рыночных запросов — для этого требуется уже подключенный сервер Senpi MCP.
