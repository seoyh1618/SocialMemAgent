Incorpora un agente de IA en la plataforma comercial Senpi creando una cuenta, generando una clave API y configurando la conexión del servidor Senpi MCP. Admite billetera, Telegram o identidad de billetera generada por agentes. Úselo cuando el usuario diga "configurar Senpi", "integrarse a Senpi", "conectarse a Senpi", "instalar Senpi", "registrarse con Senpi" o cuando el agente necesita registrarse automáticamente con Senpi para realizar transacciones autónomas. NO lo utilice para operaciones comerciales, gestión de estrategias o consultas de mercado; estas requieren que el servidor Senpi MCP ya esté conectado.
