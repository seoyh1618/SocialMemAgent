Integra un agente AI nella piattaforma di trading Senpi creando un account, generando una chiave API e configurando la connessione al server Senpi MCP. Supporta portafoglio, Telegram o identità del portafoglio generata dall'agente. Da utilizzare quando l'utente dice "configura Senpi", "onboarding a Senpi", "connettiti a Senpi", "installa Senpi", "registrati con Senpi" o quando l'agente deve registrarsi autonomamente con Senpi per il trading autonomo. NON utilizzare per operazioni di trading, gestione della strategia o query di mercato: queste richiedono che il server Senpi MCP sia già connesso.
