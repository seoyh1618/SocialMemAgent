Этот навык следует использовать, когда пользователь просит «просмотреть PR и сохранить результаты», «провести обзор PR с документацией», «создать документ обзора PR», «просмотреть и задокументировать PR», «сохранить обзор PR в документах», «задокументировать обзор PR» или упоминает о проверке PR с намерением сохранить результаты проверки. Выполняет комплексную PR-проверку, используя набор инструментов pr-review-toolkit с моделью опуса, и публикует результаты в виде PR-комментариев.
