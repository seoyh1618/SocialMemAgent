Configuration complète du SDK Sentry pour Go. À utiliser lorsqu'on vous demande d'« ajouter Sentry to Go », « d'installer Sentry-go », de « configurer Sentry in Go » ou de configurer la surveillance des erreurs, le traçage, la journalisation, les métriques ou les crons pour les applications Go. Prend en charge net/http, Gin, Echo, Fiber, FastHTTP, Iris et Negroni.
