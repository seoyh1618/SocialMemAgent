Полная настройка Sentry SDK для Go. Используйте, когда вас просят «добавить Sentry в Go», «установить Sentry-go», «настроить Sentry в Go» или настроить мониторинг ошибок, трассировку, ведение журнала, метрики или cron для приложений Go. Поддерживает net/http, Gin, Echo, Fiber, FastHTTP, Iris и Negroni.
