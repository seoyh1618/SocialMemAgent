Especificación de generación de datos simulada de Apifox. Utilice esta habilidad al generar datos simulados, Apifox Mock JSON o scripts Apifox Mock. Admite la generación JSON de sintaxis Mock.js, scripts simulados avanzados (asociaciones de campos, lógica condicional, parámetros de solicitud de lectura) y archivos simulados locales. Cubre reglas generales como la inferencia semántica de campos, el seguimiento de tipos y el reconocimiento de enumeraciones.
