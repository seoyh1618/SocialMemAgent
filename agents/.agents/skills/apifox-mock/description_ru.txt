Спецификация генерации макетных данных Apifox. Используйте этот навык при создании фиктивных данных, Apifox Mock JSON или скриптов Apifox Mock. Поддерживает генерацию JSON синтаксиса Mock.js, расширенные макетные сценарии (ассоциации полей, условную логику, параметры запроса на чтение) и локальные макетные файлы. Охватывает общие правила, такие как семантический вывод полей, трассировка типов и распознавание перечислений.
