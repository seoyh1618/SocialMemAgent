Навыки Google Cloud CLI (gcloud) экспертного уровня для управления ресурсами GCP. Используйте при работе
с «командами gcloud», «развертыванием облачного запуска», «alloydb», «cloud sql», «идентификацией рабочей нагрузки»
федерация», «разрешения iam», «сеть vpc», «секретный менеджер» или «реестр артефактов».
Охватывает установку, аутентификацию, IAM, Cloud Run, Cloud Storage, VPC, AlloyDB, Firebase,
и интеграция CI/CD с GitHub Actions и Cloud Build.
