مهارة Google Cloud CLI (gcloud) على مستوى الخبراء لإدارة موارد Google Cloud Platform. استخدم عند العمل
باستخدام "أوامر gcloud"، و"نشر التشغيل السحابي"، و"alloydb"، و"cloud sql"، و"هوية عبء العمل"
الاتحاد أو "أذونات iam" أو "شبكة vpc" أو "المدير السري" أو "التسجيل الاصطناعي".
يغطي التثبيت، والمصادقة، وIAM، وCloud Run، وCloud Storage، وVPC، وAlloyDB، وFirebase،
وتكامل CI/CD مع GitHub Actions وCloud Build.
