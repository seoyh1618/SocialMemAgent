con "comandi gcloud", "distribuzione esecuzione cloud", "alloydb", "cloud sql", "identità del carico di lavoro
federazione", "autorizzazioni iam", "rete vpc", "gestione segreti" o "registro degli artefatti".
Copre installazione, autenticazione, IAM, Cloud Run, Cloud Storage, VPC, AlloyDB, Firebase,
e integrazione CI/CD con GitHub Actions e Cloud Build.
