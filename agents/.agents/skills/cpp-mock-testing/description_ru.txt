Автоматизирует создание пробных тестов для проектов C++ с использованием платформы Google Mock (GMock) с согласованными шаблонами тестирования программного обеспечения. Используйте при создании тестов с имитируемыми зависимостями, макетированием интерфейса, проверкой поведения или когда пользователь упоминает макеты, заглушки, подделки или GMock.
