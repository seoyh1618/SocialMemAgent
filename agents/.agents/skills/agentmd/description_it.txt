Genera file di contesto CLAUDE.md / AGENTS.md / COPILOT.md minimi e supportati dalla ricerca per le CLI dell'agente di codifica. Basato su "Evaluating AGENTS.md" (ETH Zurigo, febbraio 2026) che ha rilevato che i file di contesto generati automaticamente DIMINUISCONO le prestazioni di circa il 3% e aumentano i costi del 20-23%, mentre i file minimi scritti da persone migliorano le prestazioni di circa il 4%. Da utilizzare quando l'utente dice "genera CLAUDE.md", "crea AGENTS.md", "genera file di contesto", "agentmd", "crea CLAUDE.md consigliato", "genera istruzioni agente", "file di contesto init" o qualsiasi richiesta per creare/migliorare un file di contesto dell'agente di codifica per un repository. Sostituisce il comando predefinito /init che genera file di contesto gonfi e controproducenti.
