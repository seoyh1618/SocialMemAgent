Создавайте минимальные, подтвержденные исследованиями контекстные файлы CLAUDE.md / AGENTS.md / COPILOT.md для кодирования CLI агентов. На основе «Оценки AGENTS.md» (ETH Zurich, февраль 2026 г.), в ходе которой было обнаружено, что автоматически создаваемые контекстные файлы СНИЖАЮТ производительность примерно на 3 % и увеличивают затраты на 20–23 %, в то время как минимальное количество файлов, написанных человеком, повышает производительность примерно на 4 %. Используйте, когда пользователь говорит «создать CLAUDE.md», «создать AGENTS.md», «создать файл контекста», «agentmd», «создать рекомендуемый CLAUDE.md», «создать инструкции агента», «файл контекста инициализации» или любой запрос на создание/улучшение файла контекста агента кодирования для репозитория. Заменяет команду /init по умолчанию, которая генерирует раздутые и непродуктивные контекстные файлы.
