Generieren Sie minimale, forschungsgestützte CLAUDE.md-/AGENTS.md-/COPILOT.md-Kontextdateien für die Codierung von Agent-CLIs. Basierend auf „Evaluating AGENTS.md“ (ETH Zürich, Februar 2026), das ergab, dass automatisch generierte Kontextdateien die Leistung um ca. 3 % VERRINGERN und die Kosten um 20–23 % erhöhen, während minimale, von Menschen geschriebene Dateien die Leistung um ca. 4 % verbessern. Wird verwendet, wenn der Benutzer „CLAUDE.md generieren“, „AGENTS.md erstellen“, „Kontextdatei generieren“, „agentmd“, „empfohlene CLAUDE.md erstellen“, „Agentenanweisungen generieren“, „Kontextdatei initieren“ oder eine Anfrage zum Erstellen/Verbessern einer Codierungsagentenkontextdatei für ein Repository sagt. Ersetzt den Standardbefehl /init, der aufgeblähte, kontraproduktive Kontextdateien generiert.
