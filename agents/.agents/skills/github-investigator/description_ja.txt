GitHub 上の特定のトピックに関する問題、PR、コードに関するディスカッションを調査し、概要レポートを作成するスキル。 ghコマンドを使用します。このスキルは、ユーザーが GitHub リポジトリ内のディスカッションの調査、問題/PR のリポジトリ間分析、特定の機能/バグ/コンポーネントの履歴の理解、または GitHub での意思決定の追跡を要求するときに使用します。また、「このリポジトリでは ○○ についてどのような議論がありますか?」、「○○の歴史を調べてください」、「関連する問題と PR をまとめてください」、または「なぜ ○○ はこのように設計されたのですか?」などのリクエストにも対応します。 GitHub について明示的に言及されていない場合でも、OSS 機能の議論や設計履歴を調査するためにこのスキルを使用してください。
