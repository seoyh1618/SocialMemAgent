gh 명령을 사용하여 GitHub의 특정 주제와 관련된 이슈, PR 및 코드에 대한 토론을 조사하고 요약하는 기술입니다. 사용자가 GitHub 저장소의 토론 조사, 문제/PR 교차 분석, 특정 기능/버그/구성 요소의 기록 이해 또는 GitHub의 의사 결정 추적을 요청할 때 이를 사용합니다. “이 저장소에는 ○○에 대해 어떤 논의가 있나요?”, “○○의 역사를 살펴보세요”, “관련 이슈 및 홍보를 요약해 주세요”, “○○은 왜 이렇게 설계되었나요?”와 같은 요청을 처리합니다. GitHub가 명시적으로 언급되지 않더라도 OSS 기능 토론이나 설계 기록을 조사하는 데 이 기술을 사용하세요.
