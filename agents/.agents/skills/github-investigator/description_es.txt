Una habilidad para investigar y resumir discusiones sobre problemas, relaciones públicas y código relacionado con temas específicos en GitHub, utilizando el comando gh. Se utiliza cuando el usuario solicita la investigación de discusiones en un repositorio de GitHub, el análisis cruzado de problemas/PR, la comprensión del historial de una característica/error/componente en particular o el seguimiento de las decisiones tomadas en GitHub. Maneja solicitudes como "¿Qué discusiones existen sobre ○○ en este repositorio?", "Consulta el historial de ○○", "Compila problemas y relaciones públicas relacionados" o "¿Por qué ○○ se diseñó de esta manera?" Emplee esta habilidad para investigar discusiones sobre características de OSS o el historial de diseño, incluso cuando no se mencione explícitamente "GitHub".
