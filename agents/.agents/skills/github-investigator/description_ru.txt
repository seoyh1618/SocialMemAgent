Навык исследования дискуссий по проблемам, запросам на участие и коду, относящимся к определенной теме на GitHub, и созданию сводного отчета. Использует команду gh. Используется, когда пользователь запрашивает расследование обсуждений в репозитории GitHub, анализ проблем/запросов между репозиториями, отслеживание истории конкретной функции/ошибки/компонента или отслеживание решений, принятых на GitHub. Он также обрабатывает такие запросы, как «Какие обсуждения ○○ ведутся в этом репозитории?», «Посмотрите историю ○○», «Соберите связанные проблемы и PR» или «Почему ○○ был разработан таким образом?». Даже если GitHub не упоминается явно, этот навык используется для изучения обсуждений функций OSS и истории проектирования.
