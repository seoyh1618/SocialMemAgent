La capacità di indagare discussioni su problemi, PR e codice relativi a un argomento specifico su GitHub e creare un rapporto di riepilogo. Utilizza il comando gh. Utilizza questa abilità quando un utente chiede di indagare sulle discussioni in un repository GitHub, analizzare in modo incrociato problemi/PR, comprendere la cronologia di una particolare funzionalità/bug/componente o monitorare il processo decisionale su GitHub. Gestisce anche richieste come "Quali discussioni ci sono su ○○ in questo repository?", "Esamina la cronologia di ○○", "Compila problemi e PR correlati" o "Perché ○○ è stato progettato in questo modo?" Anche se GitHub non è menzionato esplicitamente, utilizzare questa competenza quando si esaminano le discussioni sulle funzionalità OSS o la cronologia della progettazione.
