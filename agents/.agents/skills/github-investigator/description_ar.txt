القدرة على التحقيق في المناقشات حول المشكلات والعلاقات العامة والتعليمات البرمجية المتعلقة بموضوعات محددة على GitHub وإنشاء تقارير موجزة. يستخدم الأمر gh. يتم استخدامه عندما يطلب المستخدم التحقيق في المناقشات في مستودع GitHub، أو التحليل الشامل للقضايا/العلاقات العامة، أو تتبع تاريخ ميزة/خطأ/مكون معين، أو تتبع عملية اتخاذ القرار على GitHub. يتعامل أيضًا مع استعلامات مثل "ما هي المناقشات التي جرت حول ○○ في هذا المستودع؟"، أو "انظر إلى تاريخ ○○"، أو "تلخيص المشكلات ذات الصلة والعلاقات العامة"، أو "لماذا تم تصميم ○○ بهذه الطريقة؟". استخدم هذه المهارة حتى لو لم يتم ذكر GitHub بشكل صريح، طالما أن المهمة تتضمن التحقيق في مناقشات الميزات أو تاريخ التصميم في OSS.
