Compétence pour enquêter et résumer les discussions sur les problèmes, les relations publiques et le code liés à un sujet spécifique sur GitHub à l'aide de la commande gh. À utiliser lorsqu'un utilisateur demande à enquêter sur des discussions dans un référentiel GitHub, à analyser de manière croisée des problèmes/PR, à suivre l'historique d'une fonctionnalité/un bug/un composant spécifique ou à suivre une prise de décision sur GitHub. Il traite des requêtes telles que « Quelles discussions existent sur ○○ dans ce référentiel ? », « Rechercher l'historique de ○○ », « Résumer les problèmes et les relations publiques associés » ou « Pourquoi ○○ a-t-il été conçu de cette façon ? Même si GitHub n'est pas explicitement mentionné, utilisez cette compétence pour étudier les discussions sur les fonctionnalités OSS et l'historique de conception.
