تدقيق فئات Tailwind CSS بحثًا عن التعارضات والتكرار والأدوات المساعدة المكررة وأنماط الإصدار 3 القديمة ومشكلات ترتيب الفئات. يُستخدم عند مراجعة جودة تعليمات Tailwind البرمجية، أو تصحيح أخطاء الأنماط غير المتوقعة، أو تدقيق المكونات، أو عندما يطلب المستخدم التحقق من فئات Tailwind الخاصة به، أو العثور على تعارضات الفئات، أو تنظيف التكرارات، أو الترحيل من الإصدار 3 إلى الإصدار 4. تشغيل المهام التي تتضمن ملفات HTML/JSX/TSX/Vue/Svelte مع فئات الأداة المساعدة Tailwind CSS.
