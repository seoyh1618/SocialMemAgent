Auditez les classes CSS Tailwind pour détecter les conflits, les redondances, les utilitaires en double, les modèles v3 obsolètes et les problèmes d'ordre des classes. À utiliser lors de l'examen de la qualité du code Tailwind, du débogage de styles inattendus, de l'audit de composants ou lorsqu'un utilisateur demande de vérifier ses classes Tailwind, de rechercher des conflits de classes, de nettoyer les doublons ou de migrer de la v3 vers la v4. Déclencheurs sur des tâches impliquant des fichiers HTML/JSX/TSX/Vue/Svelte avec les classes utilitaires CSS Tailwind.
