Аудит CSS-классов Tailwind на наличие конфликтов, избыточности, дублирующихся утилит, устаревших шаблонов версии 3 и проблем с порядком классов. Используйте при проверке качества кода Tailwind, отладке неожиданных стилей, аудите компонентов или когда пользователь просит проверить свои классы Tailwind, найти конфликты классов, удалить дубликаты или перейти с версии 3 на версию 4. Запускается при выполнении задач, связанных с файлами HTML/JSX/TSX/Vue/Svelte с служебными классами CSS Tailwind.
