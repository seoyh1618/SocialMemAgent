Lorsque l'utilisateur souhaite suivre les changements de classement dans les graphiques de l'App Store, trouver les principaux gagnants et perdants, détecter les applications qui entrent dans le top 100 ou identifier les applications qui sortent des graphiques. À utiliser également lorsque l'utilisateur mentionne « qui évolue dans le graphique », « changements de classement », « qui monte », « qui est en baisse », « nouvelles entrées dans le graphique », « les meilleurs gagnants » ou « changements de marché ». Pour un aperçu plus large du marché, voir Market-Pulse. Pour une analyse des mots clés compétitifs, voir analyse des concurrents.
