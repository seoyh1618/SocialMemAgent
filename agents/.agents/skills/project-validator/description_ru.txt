Проверяет конфигурацию и структуру проекта приложений в процессе как для веб-приложений, так и для приложений React Native. Используйте при проверке правильности настройки гранита.config.ts, проверке существования необходимых файлов, проверке наличия зависимостей package.json от платформы, обеспечении соответствия структуры проекта соглашениям о невыбрасываемых приложениях или отладке проблем с настройкой проекта.
