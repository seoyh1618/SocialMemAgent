Validiert die Konfiguration und Struktur von Apps-in-Toss-Projekten sowohl für Web- als auch für React Native-Apps. Verwenden Sie diese Funktion, wenn Sie überprüfen, ob „granite.config.ts“ richtig konfiguriert ist, ob erforderliche Dateien vorhanden sind, ob die Datei „package.json“ Framework-Abhängigkeiten aufweist, ob die Projektstruktur den „Apps-in-toss“-Konventionen entspricht oder ob Probleme bei der Projekteinrichtung behoben werden.
