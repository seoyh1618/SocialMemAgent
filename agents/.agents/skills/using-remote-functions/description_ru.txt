Создавайте компоненты SvelteKit с помощью удаленных функций для типобезопасного взаимодействия клиент-сервер. Используйте при создании компонентов, которым необходимо получать данные, отправлять формы или выполнять команды сервера. Удаленные функции работают на уровне компонента, а не на уровне страницы.
