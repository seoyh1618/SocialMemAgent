Erstellen Sie SvelteKit-Komponenten mithilfe von Remote-Funktionen für eine typsichere Client-Server-Kommunikation. Wird zum Erstellen von Komponenten verwendet, die Daten abrufen, Formulare senden oder Serverbefehle ausführen müssen. Remote-Funktionen funktionieren auf Komponentenebene, nicht auf Seitenebene.
