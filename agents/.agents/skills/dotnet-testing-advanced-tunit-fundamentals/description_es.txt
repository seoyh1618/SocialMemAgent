TUnit: la guía completa para principiantes sobre el marco de pruebas de próxima generación.
Cubre el descubrimiento de pruebas impulsado por el generador de código fuente, el soporte de compilación AOT y las aserciones asíncronas fluidas.
Incluye la creación de proyectos, el atributo [Prueba], la gestión del ciclo de vida, el control de concurrencia y el mapeo de sintaxis de xUnit.
