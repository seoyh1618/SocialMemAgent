قم بإنشاء صور قائمة Etsy من قوالب HTML باستخدام Playwright. يدعم العرض الفردي والدفعي مع عنوان قابل للتخصيص، وعنوان فرعي، ونص الشارة، وصور المنتج. يُستخدم عند إنشاء صور قائمة منتجات Etsy، أو نماذج بالحجم الطبيعي، أو تشغيل إنشاء مجموعة لمنتجات متعددة (على سبيل المثال، المواد القابلة للطباعة الدمية، والتنزيلات الرقمية). يعمل مع أي قالب HTML يستخدم العناصر النائبة PRODUCT_IMAGE_URL وTITLE_TEXT وSUBTITLE_TEXT وBADGE_LEFT_TEXT.
