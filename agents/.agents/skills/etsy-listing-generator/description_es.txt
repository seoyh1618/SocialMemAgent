Genera imágenes de listados de Etsy a partir de plantillas HTML usando Playwright. Admite renderizado único y por lotes con títulos, subtítulos, texto de insignia e imágenes de productos personalizables. Utilízalo al crear imágenes de listados de productos de Etsy, maquetas o ejecutar generación por lotes para varios productos (por ejemplo, imprimibles de marionetas, descargas digitales). Funciona con cualquier plantilla HTML que utilice marcadores de posición PRODUCT_IMAGE_URL, TITLE_TEXT, SUBTITLE_TEXT, BADGE_LEFT_TEXT.
