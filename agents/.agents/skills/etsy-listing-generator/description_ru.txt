Создавайте изображения списков Etsy из шаблонов HTML с помощью Playwright. Поддерживает одиночный и пакетный рендеринг с настраиваемым заголовком, подзаголовком, текстом значка и изображениями продуктов. Используйте при создании изображений списков продуктов Etsy, макетов или пакетной генерации для нескольких продуктов (например, кукольных распечаток, цифровых загрузок). Работает с любым шаблоном HTML, в котором используются заполнители PRODUCT_IMAGE_URL, TITLE_TEXT, SUBTITLE_TEXT, BADGE_LEFT_TEXT.
