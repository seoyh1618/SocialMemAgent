Создавайте тесты Robot Framework с помощью SeleniumLibrary для автоматизации браузера с помощью Selenium WebDriver. Используйте, когда вас просят создать веб-тесты пользовательского интерфейса, автоматизировать браузеры, взаимодействовать с формами, обрабатывать несколько окон/фреймов или выполнять JavaScript.
