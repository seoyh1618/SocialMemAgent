Supermemory عبارة عن بنية تحتية متطورة للذاكرة والسياق لعملاء الذكاء الاصطناعي. استخدم هذه المهارة عند إنشاء التطبيقات التي تحتاج إلى ذاكرة مستمرة، أو تخصيص المستخدم، أو الاحتفاظ بالسياق على المدى الطويل، أو البحث الدلالي عبر قواعد المعرفة. فهو يوفر Memory API لسياق المستخدم الذي تم تعلمه، وملفات تعريف المستخدم للحقائق الثابتة/الديناميكية، وRAG للبحث الدلالي. مثالي لروبوتات الدردشة والمساعدين والتطبيقات كثيفة المعرفة.
