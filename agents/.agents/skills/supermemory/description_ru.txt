Суперпамять — это современная инфраструктура памяти и контекста для агентов ИИ. Используйте этот навык при создании приложений, которым требуется постоянная память, персонализация пользователя, долгосрочное сохранение контекста или семантический поиск по базам знаний. Он предоставляет API памяти для изученного контекста пользователя, профили пользователей для статических/динамических фактов и RAG для семантического поиска. Идеально подходит для чат-ботов, помощников и наукоемких приложений.
