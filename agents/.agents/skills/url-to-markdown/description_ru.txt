Преобразуйте URL-адрес общедоступной веб-страницы в Markdown и сохраните его как многоразовый файл `.md` с помощью прилагаемого скрипта. Сначала отдайте предпочтение «https://r.jina.ai/<url>» и возвращайтесь к «https://markdown.new/», только если «r.jina.ai» недоступен. Используйте это всякий раз, когда пользователь хочет превратить общедоступную веб-страницу, статью, страницу документации, сообщение в блоге, примечание к выпуску или ссылочный URL-адрес в Markdown для чтения, архивирования, обобщения, извлечения, подготовки RAG или повторного использования нижестоящего агента, даже если они явно не упоминают уценку или сохранение файла.
