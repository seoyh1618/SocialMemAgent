Konvertieren Sie eine öffentliche Webseiten-URL in Markdown und speichern Sie sie mit dem gebündelten Skript als wiederverwendbare „.md“-Datei. Bevorzugen Sie zuerst „https://r.jina.ai/<url>“ und greifen Sie nur dann auf „https://markdown.new/“ zurück, wenn „r.jina.ai“ nicht verfügbar ist. Verwenden Sie dies immer dann, wenn der Benutzer eine öffentliche Webseite, einen Artikel, eine Dokumentationsseite, einen Blogbeitrag, einen Versionshinweis oder eine Referenz-URL zum Lesen, Archivieren, Zusammenfassen, Extrahieren, RAG-Vorbereiten oder zur Wiederverwendung nachgelagerter Agenten in Markdown umwandeln möchte, auch wenn er Markdown oder das Speichern einer Datei nicht ausdrücklich erwähnt.
