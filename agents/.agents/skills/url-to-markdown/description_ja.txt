パブリック Web ページの URL を Markdown に変換し、バンドルされたスクリプトとともに再利用可能な `.md` ファイルとして保存します。最初に「https://r.jina.ai/<url>」を優先し、「r.jina.ai」が使用できない場合にのみ「https://markdown.new/」にフォールバックします。ユーザーが公開 Web ページ、記事、ドキュメント ページ、ブログ投稿、リリース ノート、または参照 URL を読み取り、アーカイブ、要約、抽出、RAG 準備、またはダウンストリーム エージェントの再利用のためにマークダウンに変換する場合は、マークダウンやファイルの保存について明示的に言及していない場合でも、これを使用します。
