Convierta la URL de una página web pública en Markdown y guárdela como un archivo `.md` reutilizable con el script incluido. Prefiere `https://r.jina.ai/<url>` primero y solo recurre a `https://markdown.new/` si `r.jina.ai` no está disponible. Utilice esto siempre que el usuario desee convertir una página web pública, un artículo, una página de documentación, una publicación de blog, una nota de versión o una URL de referencia en Markdown para leer, archivar, resumir, extraer, preparar RAG o reutilizar agentes posteriores, incluso si no mencionan explícitamente Markdown o guardar un archivo.
