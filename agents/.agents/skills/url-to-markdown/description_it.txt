Converti l'URL di una pagina web pubblica in Markdown e salvalo come file `.md` riutilizzabile con lo script in bundle. Preferisci prima `https://r.jina.ai/<url>` e utilizza solo `https://markdown.new/` se `r.jina.ai` non è disponibile. Utilizzalo ogni volta che l'utente desidera trasformare una pagina Web pubblica, un articolo, una pagina di documentazione, un post di blog, una nota di rilascio o un URL di riferimento in Markdown per la lettura, l'archiviazione, il riepilogo, l'estrazione, la preparazione RAG o il riutilizzo dell'agente downstream, anche se non menziona esplicitamente markdown o il salvataggio di un file.
