Convertissez une URL de page Web publique en Markdown et enregistrez-la en tant que fichier « .md » réutilisable avec le script fourni. Préférez d'abord `https://r.jina.ai/<url>`, et revenez uniquement à `https://markdown.new/` si `r.jina.ai` n'est pas disponible. Utilisez-le chaque fois que l'utilisateur souhaite transformer une page Web publique, un article, une page de documentation, un article de blog, une note de version ou une URL de référence en Markdown pour la lecture, l'archivage, le résumé, l'extraction, la préparation RAG ou la réutilisation d'un agent en aval, même s'ils ne mentionnent pas explicitement la démarque ou l'enregistrement d'un fichier.
