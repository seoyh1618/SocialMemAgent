قم بتحويل عنوان URL لصفحة ويب عامة إلى Markdown واحفظه كملف `.md` قابل لإعادة الاستخدام باستخدام البرنامج النصي المجمع. فضل `https://r.jina.ai/<url>` أولاً، والرجوع فقط إلى `https://markdown.new/` إذا كان `r.jina.ai` غير متاح. استخدم هذا عندما يريد المستخدم تحويل صفحة ويب عامة أو مقالة أو صفحة وثائق أو منشور مدونة أو ملاحظة إصدار أو عنوان URL مرجعي إلى Markdown للقراءة أو الأرشفة أو التلخيص أو الاستخراج أو إعداد RAG أو إعادة استخدام الوكيل النهائي، حتى لو لم يذكروا صراحة تخفيض السعر أو حفظ ملف.
