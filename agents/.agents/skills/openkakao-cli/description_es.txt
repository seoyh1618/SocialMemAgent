Trabaje con OpenKakao CLI (`openkakao-rs`) para KakaoTalk en macOS. Úselo siempre que el usuario solicite autenticarse, inspeccionar chats, leer mensajes, enviar mensajes, observar el tráfico en tiempo real, automatizar a partir de datos de chat, crear enlaces o webhooks, verificar la firma de webhooks, administrar tokens, inspeccionar el estado de recuperación de autenticación u operar flujos de trabajo de KakaoTalk desatendidos desde la terminal. Esto también debería activarse cuando el usuario menciona `watch`, `hook`, `webhook`, `LOCO`, `chat_id`, `auth-status`, `doctor`, `launchd`, o quiere conectar OpenKakao a scripts locales, agentes, SQLite, cron o launchd.
