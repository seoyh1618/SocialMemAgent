在 macOS 上使用 OpenKakao CLI (`openkakao-rs`) 進行 KakaoTalk。每當使用者要求進行身份驗證、檢查聊天、讀取訊息、發送訊息、觀看即時流量、自動處理聊天資料、建立掛鉤或 Webhook、驗證 Webhook 簽章、管理令牌、檢查身份驗證復原狀態或從終端操作無人值守的 KakaoTalk 工作流程時使用。當使用者提到“watch”、“hook”、“webhook”、“LOCO”、“chat_id”、“auth-status”、“doctor”、“launchd”，或想要將 OpenKakao 連接到本機腳本、代理、SQLite、cron 或 launchd 時，也會觸發此操作。
