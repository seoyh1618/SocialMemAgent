macOS 上のカカオトークでは OpenKakao CLI (`openkakao-rs`) を使用します。ユーザーが認証、チャットの検査、メッセージの読み取り、メッセージの送信、リアルタイム トラフィックの監視、チャット データからの自動化、フックまたは Webhook の構築、Webhook 署名の検証、トークンの管理、認証回復状態の検査、または端末からの無人 KakaoTalk ワークフローの操作を要求するたびに使用します。これは、ユーザーが「watch」、「hook」、「webhook」、「LOCO」、「chat_id」、「auth-status」、「doctor」、「launchd」に言及した場合、または OpenKakao をローカル スクリプト、エージェント、SQLite、cron、または launchd に接続したい場合にもトリガーされます。
