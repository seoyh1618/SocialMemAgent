macOS에서 카카오톡용 OpenKakao CLI(`openkakao-rs`)를 사용하세요. 사용자가 인증, 채팅 검사, 메시지 읽기, 메시지 보내기, 실시간 트래픽 감시, 채팅 데이터 자동화, 후크 또는 웹훅 구축, 웹훅 서명 확인, 토큰 관리, 인증 복구 상태 검사 또는 터미널에서 무인 카카오톡 워크플로 운영을 요청할 때마다 사용합니다. 이는 사용자가 `watch`, `hook`, `webhook`, `LOCO`, `chat_id`, `auth-status`, `doctor`, `launchd`를 언급하거나 OpenKakao를 로컬 스크립트, 에이전트, SQLite, cron 또는 launchd에 연결하려고 할 때도 트리거되어야 합니다.
