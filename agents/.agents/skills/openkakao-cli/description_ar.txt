العمل مع OpenKakao CLI (`openkakao-rs`) لـ KakaoTalk على نظام التشغيل macOS. يُستخدم عندما يطلب المستخدم المصادقة أو فحص الدردشات أو قراءة الرسائل أو إرسال الرسائل أو مشاهدة حركة المرور في الوقت الفعلي أو التشغيل التلقائي لبيانات الدردشة أو إنشاء خطافات أو خطافات ويب أو التحقق من توقيع خطاف الويب أو إدارة الرموز المميزة أو فحص حالة استرداد المصادقة أو تشغيل مسارات عمل KakaoTalk غير المراقبة من الجهاز. يجب أن يتم تشغيل هذا أيضًا عندما يذكر المستخدم "watch"، أو "hook"، أو "webhook"، أو "LOCO"، أو "chat_id"، أو "auth-status"، أو "doctor"، أو "launchd"، أو يريد ربط OpenKakao بالنصوص البرمجية المحلية، أو الوكلاء، أو SQLite، أو cron، أو Launchd.
