在 macOS 上使用 OpenKakao CLI (`openkakao-rs`) 进行 KakaoTalk。每当用户要求进行身份验证、检查聊天、读取消息、发送消息、观看实时流量、自动处理聊天数据、构建挂钩或 Webhook、验证 Webhook 签名、管理令牌、检查身份验证恢复状态或从终端操作无人值守的 KakaoTalk 工作流程时使用。当用户提到“watch”、“hook”、“webhook”、“LOCO”、“chat_id”、“auth-status”、“doctor”、“launchd”，或者想要将 OpenKakao 连接到本地脚本、代理、SQLite、cron 或 launchd 时，也会触发此操作。
