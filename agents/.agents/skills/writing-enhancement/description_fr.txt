Améliorez la qualité d’écriture, corrigez les erreurs et transformez le style. À utiliser lorsqu'on vous le demande pour relire, éditer, améliorer la clarté, corriger la grammaire/les fautes de frappe, améliorer la rédaction académique, analyser la qualité de la prose, humaniser le contenu technique ou rédiger des essais. Les déclencheurs incluent « améliorer cette écriture », « corriger les fautes de frappe », « relire », « rendre cela plus clair », « style académique », « humaniser », « rédiger un essai », « analyser mon écriture ».
