تحسين جودة الكتابة وإصلاح الأخطاء وتغيير الأسلوب. استخدمه عندما يُطلب منك التدقيق اللغوي أو التحرير أو تحسين الوضوح أو إصلاح القواعد النحوية/الأخطاء المطبعية أو تحسين الكتابة الأكاديمية أو تحليل جودة النثر أو إضفاء الطابع الإنساني على المحتوى الفني أو كتابة المقالات. تتضمن المشغلات "تحسين هذه الكتابة"، "إصلاح الأخطاء المطبعية"، "التدقيق اللغوي"، "اجعل هذا أكثر وضوحًا"، "الأسلوب الأكاديمي"، "إضفاء طابع إنساني"، "كتابة مقال"، "تحليل كتاباتي".
