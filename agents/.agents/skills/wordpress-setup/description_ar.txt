اتصل بموقع WordPress عبر WP-CLI عبر SSH أو REST API. سير العمل: التحقق من واجهة سطر الأوامر (CLI)، واختبار اتصال SSH، وإعداد المصادقة، والتحقق من الوصول، وحفظ التكوين. يُستخدم عند الاتصال بموقع WordPress، أو إعداد الوصول إلى WP-CLI، أو إنشاء كلمات مرور للتطبيقات، أو استكشاف مشكلات اتصال WordPress وإصلاحها.
