Stellen Sie über WP-CLI über SSH oder REST API eine Verbindung zu einer WordPress-Site her. Arbeitsablauf: CLI überprüfen, SSH-Verbindung testen, Authentifizierung einrichten, Zugriff überprüfen, Konfiguration speichern. Verwenden Sie diese Option, wenn Sie eine Verbindung zu einer WordPress-Site herstellen, den WP-CLI-Zugriff einrichten, Anwendungskennwörter erstellen oder WordPress-Verbindungsprobleme beheben.
