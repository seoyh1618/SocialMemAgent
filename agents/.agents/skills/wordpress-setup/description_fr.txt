Connectez-vous à un site WordPress via WP-CLI via SSH ou API REST. Workflow : vérifiez la CLI, testez la connexion SSH, configurez l'authentification, vérifiez l'accès, enregistrez la configuration. À utiliser lors de la connexion à un site WordPress, de la configuration de l'accès WP-CLI, de la création de mots de passe d'application ou du dépannage des problèmes de connexion WordPress.
