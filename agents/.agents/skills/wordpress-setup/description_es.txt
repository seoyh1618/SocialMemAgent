Conéctese a un sitio de WordPress a través de WP-CLI a través de SSH o API REST. Flujo de trabajo: verifique la CLI, pruebe la conexión SSH, configure la autenticación, verifique el acceso, guarde la configuración. Úselo al conectarse a un sitio de WordPress, configurar el acceso a WP-CLI, crear contraseñas de aplicaciones o solucionar problemas de conexión de WordPress.
