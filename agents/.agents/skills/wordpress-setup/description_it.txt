Connettiti a un sito WordPress tramite WP-CLI su SSH o API REST. Flusso di lavoro: controlla la CLI, verifica la connessione SSH, imposta l'autenticazione, verifica l'accesso, salva la configurazione. Da utilizzare quando ci si connette a un sito WordPress, si configura l'accesso WP-CLI, si creano password per le applicazioni o si risolvono i problemi di connessione di WordPress.
