Подключитесь к сайту WordPress через WP-CLI через SSH или REST API. Рабочий процесс: проверьте CLI, проверьте SSH-соединение, настройте аутентификацию, проверьте доступ, сохраните конфигурацию. Используйте при подключении к сайту WordPress, настройке доступа WP-CLI, создании паролей приложений или устранении проблем с подключением WordPress.
