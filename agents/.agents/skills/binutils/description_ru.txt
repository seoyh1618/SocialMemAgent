Навыки GNU binutils для манипуляций и анализа двоичных файлов. Используйте при использовании ar для статических библиотек, Strip или Objcopy для двоичной обработки, addr2line для преобразования адресов в исходные местоположения, строк для извлечения текста или C++filt для расшифровки имен C++. Активируется при запросах о ar, Strip, objcopy, addr2line, строках, c++filt, ranlib или двоичных задачах постобработки.
