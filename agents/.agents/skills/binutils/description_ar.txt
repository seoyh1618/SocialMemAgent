مهارة GNU binutils في المعالجة والتحليل الثنائي. يُستخدم عند استخدام ar للمكتبات الثابتة، أو strip أو objcopy للمعالجة الثنائية، أو addr2line لتحويل العناوين إلى مواقع المصدر، أو السلاسل لاستخراج النص، أو c++filt لفك تشابك الأسماء في C++. يتم التنشيط عند الاستعلامات حول ar، أو strip، أو objcopy، أو addr2line، أو strings، أو c++filt، أو ranlib، أو مهام ما بعد المعالجة الثنائية.
