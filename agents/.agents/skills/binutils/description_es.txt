Habilidad binutils de GNU para manipulación y análisis binarios. Úselo cuando use ar para bibliotecas estáticas, strip u objcopy para procesamiento binario, addr2line para convertir direcciones a ubicaciones de origen, cadenas para extracción de texto o c++filt para exigencia de nombres en C++. Se activa ante consultas sobre ar, strip, objcopy, addr2line, strings, c++filt, ranlib o tareas de posprocesamiento binario.
