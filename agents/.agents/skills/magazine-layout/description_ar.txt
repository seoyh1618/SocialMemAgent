قم بتحويل محتوى النص إلى صفحة HTML ذات تصميم جميل على طراز المجلة مع تخطيط احترافي وأنماط مرئية متعددة. سيناريوهات التشغيل: (1) يريد المستخدم تحويل النص/المقالات إلى HTML على شكل مجلة، (2) يذكر المستخدم "تخطيط المجلة"، "تصميم المجلة"، "تخطيط المقالة"، "التخطيط الاحترافي"، "تجميل النص"، "تخطيط المجلة"، (3) يحتاج المستخدم إلى صفحة HTML منسقة بشكل أنيق، (4) يحتاج المستخدم إلى تصدير محتوى التصميم بصيغة PDF. يدعم 12 نمطًا مرئيًا فريدًا باستخدام Tailwind CSS.
