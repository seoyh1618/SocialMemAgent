テキスト コンテンツを、プロフェッショナルなレイアウトと複数のビジュアル スタイルを備えた美しくスタイルされた雑誌スタイルの HTML ページに変換します。トリガーシナリオ: (1) ユーザーがテキスト/記事を雑誌スタイルの HTML に変換したい、(2) ユーザーが「雑誌のレイアウト」、「雑誌のデザイン」、「記事のレイアウト」、「プロフェッショナルなレイアウト」、「テキストの美化」、「雑誌のレイアウト」について言及、(3) ユーザーがエレガントにフォーマットされた HTML ページを必要としている、(4) ユーザーがデザインのコンテンツを PDF としてエクスポートする必要がある。 Tailwind CSS を使用して 12 のユニークな視覚スタイルをサポートします。
