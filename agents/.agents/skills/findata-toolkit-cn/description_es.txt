Conjunto de herramientas de datos financieros de acciones A. Proporciona scripts para obtener cotizaciones de acciones A en tiempo real, indicadores financieros, cambios en las acciones de directores y supervisores, flujos de capital en dirección norte y datos macroeconómicos (LPR, IPC/PPI, PMI, financiación social, M2). Úselo cuando se necesiten datos del mercado de acciones A en tiempo real para respaldar el análisis de inversiones. Todas las fuentes de datos son gratuitas y no requieren clave API.
