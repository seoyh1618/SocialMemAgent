Инструментарий для финансовых данных A-share. Предоставляет сценарии для получения в реальном времени котировок акций A, финансовых показателей, изменений долей директоров и надзорных органов, потоков капитала на север, макроэкономических данных (LPR, CPI/PPI, PMI, социальное финансирование, M2). Используйте, когда для поддержки инвестиционного анализа необходимы рыночные данные по акциям А в реальном времени. Все источники данных бесплатны и не требуют ключа API.
