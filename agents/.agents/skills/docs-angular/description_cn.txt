查找 CE.SDK Angular 参考文档、指南和配置页面。

当用户需要 Angular 的 CE.SDK 文档时使用 — 配置、UI 定制、
导出选项、功能指南或入门说明。也由“IMG.LY”、“CreativeEditor”触发，
当用户需要现有的 Angular 文档页面时，为“CE.SDK”或“cesdk”。

不适用于编写代码（使用构建）或概念解释（使用解释）。

<示例>
上下文：用户询问 Angular 配置
用户：“如何在 Angular 中配置编辑器？”
助理：“我将使用 /cesdk:docs-angular 来查找配置选项。”
</示例>

<示例>
背景：用户需要 Angular 集成
用户：“如何在 Angular 组件中使用 CE.SDK？”
助理：“让我使用 /cesdk:docs-angular 来查找相关文档。”
</示例>
