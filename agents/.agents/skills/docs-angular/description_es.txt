Busque documentos de referencia, guías y páginas de configuración de CE.SDK Angular.

Úselo cuando el usuario necesite documentos CE.SDK para Angular: configuración, personalización de la interfaz de usuario,
opciones de exportación, guías de funciones o instrucciones de introducción. También activado por "IMG.LY", "CreativeEditor",
"CE.SDK" o "cesdk" cuando el usuario necesita una página de documento Angular existente.

No para escribir código (use compilación) o explicaciones de conceptos (use explicación).

<ejemplo>
Contexto: el usuario pregunta sobre la configuración angular
usuario: "¿Cómo configuro el editor en Angular?"
asistente: "Usaré /cesdk:docs-angular para buscar opciones de configuración".
</ejemplo>
