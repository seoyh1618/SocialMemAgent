Recherchez les documents de référence, les guides et les pages de configuration CE.SDK Angular.

À utiliser lorsque l'utilisateur a besoin de documents CE.SDK pour Angular : configuration, personnalisation de l'interface utilisateur,
options d'exportation, guides de fonctionnalités ou instructions de démarrage. Également déclenché par "IMG.LY", "CreativeEditor",
"CE.SDK" ou "cesdk" lorsque l'utilisateur a besoin d'une page de documentation Angular existante.

Pas pour écrire du code (utiliser build) ou des explications de concepts (utiliser expliquer).

<exemple>
Contexte : l'utilisateur pose des questions sur la configuration angulaire
user : "Comment configurer l'éditeur dans Angular ?"
assistant : "J'utiliserai /cesdk:docs-angular pour rechercher les options de configuration."
</exemple>

<exemple>
Contexte : l'utilisateur a besoin d'une intégration angulaire
user : "Comment utiliser CE.SDK dans un composant angulaire ?"
assistant : "Permettez-moi d'utiliser /cesdk:docs-angular pour trouver la documentation pertinente."
</exemple>
