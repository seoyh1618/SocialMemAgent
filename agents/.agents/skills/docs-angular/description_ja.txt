CE.SDK Angular のリファレンス ドキュメント、ガイド、構成ページを検索してください。

ユーザーが Angular の CE.SDK ドキュメントを必要とする場合 (構成、UI カスタマイズ、
エクスポート オプション、機能ガイド、または開始手順。 「IMG.LY」、「CreativeEditor」によってもトリガーされます。
「CE.SDK」、またはユーザーが既存の Angular ドキュメント ページを必要とする場合は「cesdk」。

コードの作成 (build を使用) や概念の説明 (explain を使用) のためではありません。

<例>
コンテキスト: ユーザーが Angular 構成について質問する
ユーザー: 「Angular でエディタを設定するにはどうすればよいですか?」
アシスタント: 「/cesdk:docs-angular を使用して構成オプションを調べます。」
</例>

<例>
コンテキスト: ユーザーは Angular 統合を必要としています
ユーザー: 「Angular コンポーネントで CE.SDK を使用するにはどうすればよいですか?」
アシスタント: 「/cesdk:docs-angular を使用して関連ドキュメントを見つけさせてください。」
</例>
