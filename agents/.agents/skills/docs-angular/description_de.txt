Suchen Sie nach CE.SDK Angular-Referenzdokumenten, Leitfäden und Konfigurationsseiten.

Verwenden Sie diese Option, wenn der Benutzer CE.SDK-Dokumente für Angular benötigt – Konfiguration, UI-Anpassung,
Exportoptionen, Funktionshandbücher oder Anleitungen für den Einstieg. Wird auch durch „IMG.LY“, „CreativeEditor“ ausgelöst.
„CE.SDK“ oder „cesdk“, wenn der Benutzer eine vorhandene Angular-Dokumentseite benötigt.

Nicht zum Schreiben von Code (Build verwenden) oder Konzepterklärungen (Explain verwenden).

<Beispiel>
Kontext: Benutzer fragt nach der Angular-Konfiguration
Benutzer: „Wie konfiguriere ich den Editor in Angular?“
Assistent: „Ich verwende /cesdk:docs-angular, um nach Konfigurationsoptionen zu suchen.“
</Beispiel>

<Beispiel>
Kontext: Benutzer benötigt Angular-Integration
Benutzer: „Wie verwende ich CE.SDK in einer Angular-Komponente?“
Assistent: „Lassen Sie mich /cesdk:docs-angular verwenden, um die relevante Dokumentation zu finden.“
</Beispiel>
