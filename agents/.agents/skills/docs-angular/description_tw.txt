尋找 CE.SDK Angular 參考文件、指南和設定頁面。

當使用者需要 Angular 的 CE.SDK 文件時使用 — 配置、UI 定制、
匯出選項、功能指南或入門說明。也由「IMG.LY」、「CreativeEditor」觸發，
當使用者需要現有的 Angular 文件頁面時，為“CE.SDK”或“cesdk”。

不適用於編寫程式碼（使用建構）或概念解釋（使用解釋）。
