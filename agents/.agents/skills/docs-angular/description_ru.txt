Найдите справочную документацию, руководства и страницы конфигурации CE.SDK Angular.

Используйте, когда пользователю нужна документация CE.SDK для Angular — настройка, настройка пользовательского интерфейса,
параметры экспорта, руководства по функциям или инструкции по началу работы. Также запускается "IMG.LY", "CreativeEditor",
«CE.SDK» или «cesdk», если пользователю нужна существующая страница документа Angular.

Не для написания кода (используйте сборку) или объяснения концепций (используйте объяснение).

<пример>
Контекст: пользователь спрашивает о конфигурации Angular.
пользователь: «Как мне настроить редактор в Angular?»
помощник: «Я буду использовать /cesdk:docs-angular для поиска параметров конфигурации».
</пример>

<пример>
Контекст: пользователю нужна интеграция с Angular.
пользователь: «Как использовать CE.SDK в компоненте Angular?»
помощник: «Позвольте мне использовать /cesdk:docs-angular, чтобы найти соответствующую документацию».
</пример>
