Capacità di pubblicazione di contenuti Xiaohongshu. Supporta due modalità di pubblicazione: (1) modalità immagine-testo: immagini + testo breve; (2) Modalità formato lungo: articoli lunghi + modelli di layout. Supporta due metodi di input: l'utente fornisce contenuto completo e immagini/URL di immagini per la pubblicazione diretta oppure fornisce l'URL di una pagina Web per l'estrazione automatica di contenuti ed immagini. Utilizza la modalità formato lungo quando l'utente dice "pubblica articolo lungo", altrimenti utilizza per impostazione predefinita la modalità testo immagine.
