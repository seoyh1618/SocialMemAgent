Xiaohongshu content posting skills. Supports two publishing modes: (1) Image-text mode – images + short text; (2) Long-form mode – long article + layout template. Supports two input methods: user provides complete content and images/image URLs, then publishes directly; or provides a webpage URL, and content and images are extracted automatically. Use long-form mode when user says “post long article,” otherwise default to image-text mode.
