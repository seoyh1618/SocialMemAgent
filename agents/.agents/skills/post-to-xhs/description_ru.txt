Навыки публикации контента Xiaohongshu. Поддерживает два режима публикации: (1) режим изображения-текста – изображения + короткий текст; (2) Режим длинной формы – длинная статья + шаблон макета. Поддерживает два метода ввода: пользователь предоставляет полный контент и изображения/URL-адреса изображений для прямой публикации; или предоставляет URL-адрес веб-страницы для автоматического извлечения контента и изображений. Используйте режим длинной формы, когда пользователь говорит «发长文», в противном случае по умолчанию используется режим «изображение-текст».
