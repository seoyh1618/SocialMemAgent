Compétences en publication de contenu de Xiaohongshu. Prend en charge deux modes de publication : (1) Mode image-texte – images + texte court ; (2) Mode long – articles longs + modèles de mise en page. Prend en charge deux méthodes de saisie : l'utilisateur fournit un contenu complet et des images/URL d'images pour une publication directe ; ou fournit une URL de page Web pour l’extraction automatique de contenu et d’images. Utilisez le mode long lorsque l'utilisateur dit « publier un article long », sinon le mode image-texte est utilisé par défaut.
