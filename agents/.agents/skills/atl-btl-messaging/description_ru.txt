Платформа обмена сообщениями «над чертой» (ATL) и «под чертой» (BTL) для ориентации руководителей, а не конечных пользователей. Используйте при составлении сообщений для разных уровней старшинства, определении длины и тона сообщения или адаптации охвата для вице-президентов/уровня C по сравнению с менеджерами/IC.
