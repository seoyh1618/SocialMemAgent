Memory-lancedb-pro OpenClaw プラグインのメンテナンス、デバッグ、アップグレードのための包括的なガイド。ハイブリッド検索 (Vector + BM25)、クロスエンコーダー再ランキング、マルチスコープ分離、ノイズ フィルタリング、適応検索、および管理 CLI を備えた、強化された LanceDB ベースの長期メモリ システムです。このスキルは次の場合に使用します:(1) 新機能の開発またはmemory-lancedb-proのバグ修正、(2) 取得パイプラインの変更(ベクトル検索、BM25、RRFフュージョン、再ランキング、スコアリングステージ)、(3) 埋め込みプロバイダーの追加または変更、(4) スコープ/アクセス制御ロジックの更新、(5) エージェントツールまたはCLIコマンドの変更、(6) メモリ品質の問題のトラブルシューティング(ノイズ、重複、低リコール)、(7) JSONL セッション蒸留パイプライン、(8) メモリ バックエンド間でのデータの移行、または (9) 拡張機能を計画するためのプラグインのアーキテクチャの理解。
