Guía completa para mantener, depurar y actualizar el complemento Memory-lancedb-pro OpenClaw: un sistema de memoria a largo plazo mejorado respaldado por LanceDB con recuperación híbrida (Vector + BM25), reclasificación de codificadores cruzados, aislamiento de múltiples alcances, filtrado de ruido, recuperación adaptativa y una CLI de administración. Utilice esta habilidad cuando: (1) desarrolle nuevas funciones o corrija errores en Memory-lancedb-pro, (2) modifique el proceso de recuperación (búsqueda de vectores, BM25, fusión RRF, reclasificación, etapas de puntuación), (3) agregue o cambie proveedores de incrustación, (4) actualice la lógica de control de acceso/alcance, (5) modifique las herramientas del agente o los comandos CLI, (6) solucione problemas de calidad de la memoria (ruido, duplicados, baja recuperación), (7) trabaje en la sesión JSONL canalización de destilación, (8) migrar datos entre backends de memoria o (9) comprender la arquitectura del complemento para planificar mejoras.
