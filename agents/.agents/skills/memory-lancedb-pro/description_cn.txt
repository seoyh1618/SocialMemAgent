维护、调试和升级 memory-lancedb-pro OpenClaw 插件的综合指南 — 一个增强的 LanceDB 支持的长期内存系统，具有混合检索（Vector + BM25）、跨编码器重新排序、多范围隔离、噪声过滤、自适应检索和管理 CLI。在以下情况下使用此技能：(1) 开发新功能或修复内存-lancedb-pro 中的错误，(2) 修改检索管道（矢量搜索、BM25、RRF 融合、重新排名、评分阶段），(3) 添加或更改嵌入提供程序，(4) 更新范围/访问控制逻辑，(5) 修改代理工具或 CLI 命令，(6) 排除内存质量问题（噪声、重复、低召回率），(7) 处理 JSONL 会话蒸馏管道， (8) 在内存后端之间迁移数据，或 (9) 了解插件的架构以计划增强功能。
