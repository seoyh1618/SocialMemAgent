Implementar un proyecto a partir de su documentación y especificación. Úselo cuando se le solicite "implementar proyecto", "continuar implementación", "compilar a partir de documentos", "implementar a partir de especificaciones" o cuando el usuario desee implementar progresivamente un proyecto documentado siguiendo una lista de verificación de tareas pendientes. Lee documentos de proyectos, crea planes de implementación y listas de tareas pendientes, e implementa de forma incremental con pruebas y confirmaciones.
