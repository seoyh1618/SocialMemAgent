جلب أسعار البيع بالتجزئة لـ Azure في الوقت الفعلي باستخدام واجهة برمجة تطبيقات Azure Retail Price (prices.azure.com) وتقدير استهلاك ائتمان وكيل Copilot Studio. يُستخدم عندما يسأل المستخدم عن تكلفة أي خدمة Azure، أو يريد مقارنة أسعار SKU، أو يحتاج إلى بيانات تسعير لتقدير التكلفة، أو يذكر تسعير Azure، أو تكاليف Azure، أو فواتير Azure، أو يسأل عن تسعير Copilot Studio، أو أرصدة Copilot، أو تقدير استخدام الوكيل. يغطي الحوسبة والتخزين والشبكات وقواعد البيانات والذكاء الاصطناعي وCopilot Studio وجميع عائلات خدمات Azure الأخرى.
