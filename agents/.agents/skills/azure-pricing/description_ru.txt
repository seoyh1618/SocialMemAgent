Получает розничные цены Azure в режиме реального времени с помощью API розничных цен Azure (prices.azure.com) и оценивает потребление кредитов агентом Copilot Studio. Используйте, когда пользователь спрашивает о стоимости любой службы Azure, хочет сравнить цены SKU, ему нужны данные о ценах для оценки затрат, упоминает цены Azure, затраты Azure, выставление счетов Azure или спрашивает о ценах Copilot Studio, кредитах Copilot или оценке использования агента. Охватывает вычисления, хранилище, сети, базы данных, искусственный интеллект, Copilot Studio и все другие семейства сервисов Azure.
