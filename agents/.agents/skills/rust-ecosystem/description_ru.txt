Эксперт по экосистеме ржавчины. Обрабатывает выбор крейта, рекомендации библиотек, сравнение платформ, выбор асинхронной среды выполнения, библиотеки сериализации, веб-фреймворки и т. д. Триггеры: крейт, библиотека, фреймворк, экосистема, асинхронная среда выполнения, tokio, async-std, serde, reqwest, axum, 库, 框架, 生态
