Base de données de type sécurisé ORM pour TypeScript/Node.js. À utiliser lorsque vous avez besoin d'un accès à la base de données avec une intégration complète de TypeScript : types générés automatiquement à partir du schéma, des migrations et du générateur de requêtes. Idéal pour PostgreSQL, MySQL, SQLite, MongoDB. Choisissez Prisma plutôt que SQL brut ou Knex lorsque la sécurité des types et l'expérience des développeurs sont des priorités.
