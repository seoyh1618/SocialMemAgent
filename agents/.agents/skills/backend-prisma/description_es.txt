ORM de base de datos con seguridad de tipos para TypeScript/Node.js. Úselo cuando necesite acceso a la base de datos con integración completa de TypeScript: tipos generados automáticamente a partir de esquemas, migraciones y generador de consultas. Lo mejor para PostgreSQL, MySQL, SQLite, MongoDB. Elija Prisma en lugar de SQL sin formato o Knex cuando la seguridad de tipos y la experiencia del desarrollador sean prioridades.
