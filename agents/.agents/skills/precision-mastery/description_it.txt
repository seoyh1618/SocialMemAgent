Caricare SEMPRE prima di iniziare qualsiasi attività. Massimizza l'efficienza dei token per tutte le operazioni sui file, le ricerche e l'esecuzione dei comandi. Copre le modalità di estrazione (contenuto, struttura, simboli, ast, linee), l'ottimizzazione della verbosità, l'invio in batch di più file e l'orchestrazione degli strumenti di rilevamento. Garantisce che gli agenti spendano i token sui risultati, non sulle spese generali.
