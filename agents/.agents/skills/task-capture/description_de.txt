Task Capture Assistant – ein unkompliziertes Tool zur Protokollierung der Aufgabenausführung, das für die Echtzeiterfassung und einen reibungslosen Start entwickelt wurde.  
Folgt der „Erfassen-dann-verarbeiten“-Regel: Beim Start nur aufzeichnen – keine Überprüfung.  
Auslöser: /task-capture, /capture, /record
