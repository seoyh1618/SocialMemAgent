Busque escenas de películas y programas de televisión por diálogo, hora o descripción visual. Descargue videoclips, extraiga fotogramas, busque citas, identifique películas a partir de citas y consulte datos de IMDB. Úselo cuando el usuario quiera encontrar una escena específica, descargar un clip, buscar una cita en una película/programa, extraer un fotograma u obtener información de la película a través de la API de búsqueda de escena.
