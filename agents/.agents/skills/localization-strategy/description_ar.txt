عندما يريد المستخدم تخطيط أو تنفيذ استراتيجية الترجمة للنمو العالمي ومتعدد اللغات. يُستخدم أيضًا عندما يذكر المستخدم "الترجمة المحلية"، أو "متعدد اللغات"، أو "i18n"، أو "التوسع العالمي"، أو "دخول السوق"، أو "استراتيجية الترجمة"، أو "hreflang"، أو "SEO متعدد اللغات"، أو "تحسين محركات البحث الدولية". للاطلاع على سير عمل الترجمة والمسرد ودليل الأسلوب، راجع **الترجمة**.
