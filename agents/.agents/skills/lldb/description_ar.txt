مهارة مصحح أخطاء LLDB لبرامج C/C++/Swift/Objective-C. يُستخدم عند تصحيح الأخطاء باستخدام LLDB على بيئات macOS أو FreeBSD أو Linux-clang، أو تعيين نماذج GDB الذهنية لأوامر LLDB، أو استخدام LLDB في Xcode أو VS Code، أو تصحيح أخطاء Swift/Objective-C. يتم التنشيط عند الاستعلامات حول أوامر LLDB، أو ترحيل GDB إلى LLDB، أو تصحيح أخطاء النظام الأساسي لـ Apple، أو البرمجة النصية لـ LLDB Python، أو تصحيح الأخطاء المتكامل مع IDE مع الثنائيات المبنية على clang.
