Навыки отладчика LLDB для программ C/C++/Swift/Objective-C. Используйте при отладке с помощью LLDB в средах macOS, FreeBSD или Linux-clang, сопоставлении ментальных моделей GDB с командами LLDB, использовании LLDB в Xcode или VS Code или отладке Swift/Objective-C. Активируется при запросах о командах LLDB, миграции GDB в LLDB, отладке платформы Apple, сценариях Python LLDB или отладке, интегрированной в IDE, с помощью двоичных файлов, созданных clang.
