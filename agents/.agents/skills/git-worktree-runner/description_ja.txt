エディターと AI ツールを統合して git worktree を管理するための CLI ツールである gtr (git-worktree-runner) の使用ガイド。このスキルは、ユーザーから次のことについて尋ねられたときに使用します: (1) git ワークツリーの作成、管理、または削除、(2) プロジェクトの gtr / git-gtr のセットアップ、(3) ワークツリーを使用したエディター (Cursor、VS Code、Zed) または AI ツール (Claude Code、Aider、Codex、Gemini) の構成、(4) 複数のブランチ間での並列開発ワークフローの実行、(5) 新しいワークツリーへの config/env ファイルのコピー、(6) 設定ワークツリーの作成/削除用のフック、(7) git ワークツリーの問題のトラブルシューティング、(8) 並列開発のコンテキストでの「gtr」、「git gtr」、「git worktree runner」、または「worktree」に関する言及。キーワードでトリガー: gtr、git-gtr、ワークツリー、並列ブランチ、マルチブランチ ワークフロー。
