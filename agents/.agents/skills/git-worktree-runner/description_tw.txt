gtr (git-worktree-runner) 使用指南，這是一個 CLI 工具，用於透過編輯器和 AI 工具整合來管理 git 工作樹。當使用者詢問以下問題時使用此技能：(1) 建立、管理或刪除 git 工作樹，(2) 為專案設定 gtr / git-gtr，(3) 使用工作樹設定編輯器（Cursor、VS Code、Zed）或 AI 工具（Claude Code、Aider、Codex、Gemini），(4) 跨流程配置設定用於工作樹建立/刪除的鉤子，(7) 解決 git 工作樹問題，(8) 在平行開發上下文中提及「gtr」、「git gtr」、「git worktree runner」或「worktree」。關鍵字觸發：gtr、git-gtr、工作樹、平行分支、多分支工作流程。
