gtr (git-worktree-runner) 使用指南，这是一个 CLI 工具，用于通过编辑器和 AI 工具集成来管理 git 工作树。当用户询问以下问题时使用此技能：(1) 创建、管理或删除 git 工作树，(2) 为项目设置 gtr / git-gtr，(3) 使用工作树配置编辑器（Cursor、VS Code、Zed）或 AI 工具（Claude Code、Aider、Codex、Gemini），(4) 跨多个分支运行并行开发工作流程，(5) 将配置/env 文件复制到新工作树，(6) 设置用于工作树创建/删除的钩子，(7) 解决 git 工作树问题，(8) 在并行开发上下文中提及“gtr”、“git gtr”、“git worktree runner”或“worktree”。关键字触发：gtr、git-gtr、工作树、并行分支、多分支工作流程。
