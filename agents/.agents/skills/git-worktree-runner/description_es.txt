Guía para usar gtr (git-worktree-runner), una herramienta CLI para administrar árboles de trabajo git con editor e integración de herramientas de IA. Utilice esta habilidad cuando el usuario pregunte sobre: (1) crear, administrar o eliminar árboles de trabajo de git, (2) configurar gtr/git-gtr para un proyecto, (3) configurar editores (Cursor, VS Code, Zed) o herramientas de inteligencia artificial (Claude Code, Aider, Codex, Gemini) con árboles de trabajo, (4) ejecutar flujos de trabajo de desarrollo paralelos en múltiples ramas, (5) copiar archivos de configuración/env a nuevos árboles de trabajo, (6) configurar enlaces para la creación/eliminación del árbol de trabajo, (7) solucionar problemas de git worktree, (8) cualquier mención de "gtr", "git gtr", "git worktree runner" o "worktree" en el contexto de desarrollo paralelo. Activadores de palabras clave: gtr, git-gtr, árbol de trabajo, ramas paralelas, flujo de trabajo de múltiples ramas.
