دليل لاستخدام gtr (git-worktree-runner)، وهي أداة CLI لإدارة أشجار عمل git مع تكامل أداة التحرير والذكاء الاصطناعي. استخدم هذه المهارة عندما يسأل المستخدم عن: (1) إنشاء أشجار عمل git أو إدارتها أو إزالتها، (2) إعداد gtr / git-gtr لمشروع، (3) تكوين المحررين (Cursor، VS Code، Zed) أو أدوات الذكاء الاصطناعي (Claude Code، Aider، Codex، Gemini) مع أشجار العمل، (4) تشغيل سير عمل التطوير المتوازي عبر فروع متعددة، (5) نسخ ملفات config/env إلى أشجار عمل جديدة، (6) إعداد الخطافات لإنشاء/إزالة شجرة العمل، (7) استكشاف مشكلات شجرة عمل git وإصلاحها، (8) أي ذكر لـ "gtr" أو "git gtr" أو "git Worktree runner" أو "worktree" في سياق التطوير الموازي. مشغلات الكلمات الرئيسية: gtr، git-gtr، Worktree، الفروع المتوازية، سير العمل متعدد الفروع.
