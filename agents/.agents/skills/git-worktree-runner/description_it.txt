Guida per l'utilizzo di gtr (git-worktree-runner), uno strumento CLI per la gestione degli alberi di lavoro git con editor e integrazione dello strumento AI. Utilizza questa competenza quando l'utente chiede informazioni su: (1) creazione, gestione o rimozione di alberi di lavoro git, (2) configurazione di gtr / git-gtr per un progetto, (3) configurazione di editor (Cursor, VS Code, Zed) o strumenti AI (Claude Code, Aider, Codex, Gemini) con alberi di lavoro, (4) esecuzione di flussi di lavoro di sviluppo paralleli su più rami, (5) copia di file config/env su nuovi alberi di lavoro, (6) impostazione up hook per la creazione/rimozione di worktree, (7) risoluzione dei problemi di git worktree, (8) qualsiasi menzione di "gtr", "git gtr", "git worktree runner" o "worktree" nel contesto di sviluppo parallelo. Trigger su parole chiave: gtr, git-gtr, worktree, rami paralleli, flusso di lavoro multi-ramo.
