Рекомендации по отправке HTTP-запросов с помощью @aptx/api-core. Используется для: создания и настройки RequestClient, реализации промежуточного программного обеспечения/плагинов, замены основных компонентов (транспорт/декодер и т. д.), обработки ошибок, прослушивания событий, использования Context Bag, тестирования расширений. Запускается, когда коду необходимо понять или изменить архитектуру @aptx/api-core, механизмы расширения или рекомендации.
