Оптимизируйте производительность Интернета с помощью сетевых протоколов, загрузки ресурсов и внутренних функций рендеринга браузера. Используйте, когда пользователь упоминает «скорость загрузки страницы», «Основные веб-показатели», «HTTP/2», «подсказки по ресурсам», «задержку сети» или «блокировку рендеринга». Охватывает оптимизацию TCP/TLS, стратегии кэширования, WebSocket/SSE и выбор протокола. Информацию о визуальной производительности пользовательского интерфейса см. в разделе refactoring-ui. Для загрузки шрифта см. веб-типографику.
