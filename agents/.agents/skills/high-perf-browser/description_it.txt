Ottimizza le prestazioni web tramite protocolli di rete, caricamento delle risorse e rendering interno del browser. Da utilizzare quando l'utente menziona "velocità di caricamento della pagina", "Core Web Vitals", "HTTP/2", "suggerimenti sulle risorse", "latenza di rete" o "blocco del rendering". Copre l'ottimizzazione TCP/TLS, le strategie di memorizzazione nella cache, WebSocket/SSE e la selezione del protocollo. Per le prestazioni visive dell'interfaccia utente, vedere refactoring-ui. Per il caricamento dei caratteri, vedere tipografia web.
