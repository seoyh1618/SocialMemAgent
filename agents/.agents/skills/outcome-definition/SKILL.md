# Outcome Definition Framework

## Purpose
Shift thinking from feature delivery to measurable user or business outcomes.

## When to Use
- Before building a feature
- During roadmap planning
- While defining success metrics

## Steps
1. Define the desired user behavior change.
2. Select a measurable success metric.
3. Establish the current baseline.
4. Set a realistic target improvement.
5. Define the evaluation timeframe.

## Output

### Success Metric
### Baseline
### Target
### Timeframe
### How Success Will Be Measured
