Générez automatiquement un cadre de code HarmonyOS à partir de documents de conception, y compris les définitions d'interface IDL, la structure de répertoires, les fichiers d'en-tête, les modèles de fichiers sources et les configurations de build. Avant la génération, il analyse le code OpenHarmony existant pour connaître les normes actuelles de style de code et d'architecture. Applicable aux demandes des utilisateurs : (1) générer un cadre de code, (2) créer des interfaces IDL, (3) générer des configurations BUILD.gn, (4) créer du code de service, (5) initialiser la structure du projet, (6) suivre le style de code existant. Mots clés : génération de code, IDL, BUILD.gn, bundle.json, profil SA, style de code OpenHarmony, génération de code, définition d'interface, configuration de build, style de code OH
