Автоматически создавайте структуру кода HarmonyOS на основе проектных документов, включая определения интерфейса IDL, структуру каталогов, файлы заголовков, шаблоны исходных файлов и конфигурации сборки. Перед генерацией он анализирует существующий код OpenHarmony, чтобы изучить текущий стиль кода и нормы архитектуры. Применимо к запросам пользователей: (1) создать структуру кода, (2) создать интерфейсы IDL, (3) создать конфигурации BUILD.gn, (4) создать сервисный код, (5) инициализировать структуру проекта, (6) следовать существующему стилю кода. Ключевые слова: генерация кода, IDL, BUILD.gn, Bundle.json, профиль SA, стиль кода OpenHarmony, генерация кода, определение интерфейса, конфигурация сборки, стиль кода OH.
