Genere automáticamente un marco de código HarmonyOS a partir de documentos de diseño, incluidas definiciones de interfaz IDL, estructura de directorios, archivos de encabezado, plantillas de archivos fuente y configuraciones de compilación. Antes de la generación, analiza el código OpenHarmony existente para conocer el estilo del código actual y las normas de arquitectura. Aplicable a las solicitudes de los usuarios: (1) generar marco de código, (2) crear interfaces IDL, (3) generar configuraciones BUILD.gn, (4) crear código de servicio, (5) inicializar la estructura del proyecto, (6) seguir el estilo de código existente. Palabras clave: generación de código, IDL, BUILD.gn, paquete.json, perfil SA, estilo de código OpenHarmony, generación de código, definición de interfaz, configuración de compilación, estilo de código OH
