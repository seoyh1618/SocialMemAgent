Guide d'architecture globale BK-CI : une vue panoramique de la collaboration de modules centrée sur les pipelines, couvrant le flux d'exécution complet, les dépendances des modules, le flux de données et les concepts de base. Lisez d'abord lorsque vous avez besoin de comprendre l'architecture du système, d'effectuer un développement inter-modules, d'en savoir plus sur la collaboration inter-modules ou de planifier une conception architecturale.
