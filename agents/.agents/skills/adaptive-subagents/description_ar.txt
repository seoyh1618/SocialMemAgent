قم بتحسين تكلفة الرمز المميز عن طريق توجيه تفويضات أداة الوكيل إلى النموذج الأرخص بما فيه الكفاية.
يتم التشغيل عندما: على وشك استخدام أداة الوكيل، أو أي مهمة متعددة الخطوات، أو التنفيذ، أو تصحيح الأخطاء، أو إعادة البناء، أو البحث، أو الاستكشاف، أو كتابة الاختبار، أو مراجعة التعليمات البرمجية، أو أي مهمة يمكن تقسيمها إلى مهام فرعية.
لا تقم بالتشغيل عندما: إصلاح سطر واحد، أو سؤال تافه، أو عندما يقول المستخدم "لا يوجد توجيه".
