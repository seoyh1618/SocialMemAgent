Responsive HTML-E-Mail-Vorlagengenerierung mit dem MJML 4.x-Framework. Verwenden Sie diese Option, wenn der Benutzer eine E-Mail-Vorlage erstellen, generieren, entwerfen oder erstellen möchte – einschließlich Willkommens-E-Mails, Werbebotschaften, Transaktionsvorlagen, Newslettern oder anderen responsiven E-Mails. Verwenden Sie es auch, wenn der Benutzer MJML in HTML kompilieren, mit vorhandenen MJML-Vorlagen arbeiten oder diese bearbeiten oder Client-übergreifende E-Mail-Rendering-Probleme beheben möchte.
