Konvertiert Claude-Code-Kenntnisse in Gemini-CLI-Erweiterungen und umgekehrt. Verwenden Sie diese Option, wenn der Benutzer einen Skill plattformübergreifend kompatibel machen, einen Skill zwischen Plattformen portieren oder eine universelle Erweiterung erstellen möchte, die sowohl auf Claude Code als auch auf Gemini CLI funktioniert.
