Удаленное управление Linux для Alma через SSH/SFTP с постоянными сеансами SSH с отслеживанием состояния, привязкой сеансов потоков, мостом NL-to-command, терминальным мостом веб-сокета xterm.js, потоком утверждения опасных команд, опцией строгой политики команд, режимами проверки ключей хоста, списком разрешений RBAC и отредактированными для SIEM полями аудита.
