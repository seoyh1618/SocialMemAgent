Interroge le développeur sur le code, l'architecture ou les plans qu'un agent d'IA vient de créer ou de proposer, une question à la fois, avec des corrections de diagramme ASCII. À utiliser après toute implémentation importante, relations publiques, refactorisation ou plan architectural pour garantir que le propriétaire du code humain dispose d'un modèle mental à toute épreuve du système. Les déclencheurs incluent des expressions telles que « interrogez-moi », « est-ce que je comprends cela », « expliquez-moi ce que vous avez construit », « à bord de moi » ou « briefing du propriétaire du code ».
