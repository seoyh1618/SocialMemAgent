يختبر المطور حول التعليمات البرمجية أو البنية أو الخطط التي أنشأها أو اقترحها وكيل الذكاء الاصطناعي للتو - سؤال واحد في كل مرة، مع تصحيحات مخطط ASCII. يُستخدم بعد أي تنفيذ مهم، أو علاقات عامة، أو إعادة بناء، أو خطة معمارية للتأكد من أن مالك الكود البشري لديه نموذج عقلي مضاد للرصاص للنظام. تتضمن المشغلات عبارات مثل "اختبرني"، أو "هل أفهم هذا"، أو "أطلعني على ما قمت بإنشائه"، أو "أنضم إلي"، أو "إحاطة مالك الكود".
