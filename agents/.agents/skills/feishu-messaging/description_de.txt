Feishu-Workflow zum Versenden von Nachrichten und zur Dokumentenerstellung  
Auslöseszenarien: Benutzer erwähnt „Feishu-Nachricht senden“, „Feishu-Dokument“, „jemanden benachrichtigen“, „an Feishu senden“, „Feishu-Benachrichtigung“  
Anwendbar für: Senden von Feishu-Nachrichten, Erstellen von Feishu-Dokumenten, Betreiben mehrdimensionaler Tabellen, Verwalten von Wissensdatenbanken
