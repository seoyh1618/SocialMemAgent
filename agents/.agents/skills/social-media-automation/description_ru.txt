Навыки автоматизированной работы с медиа, поддержка создания контента, создания изображений, управления данными и автоматической публикации для таких платформ, как Xiaohongshu, Douyin, официальные аккаунты WeChat и видеоканалы. Он охватывает весь рабочий процесс, включая поиск по API Tavily, генерацию контента, создание изображений, управление данными HAP и автоматическую публикацию. Добавлена ​​новая функция ежедневных рекомендаций по актуальному контенту. Генерация изображений теперь поддерживает параллельное создание, автоматическую загрузку и пропуск существующих изображений. Настройки профиля учетной записи необходимо считывать из таблицы HAP «Персона учетной записи»; локальные файлы строго запрещены.
