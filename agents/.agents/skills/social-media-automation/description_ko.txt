Xiaohongshu, Douyin, WeChat 공식 계정 및 비디오 채널과 같은 플랫폼을 위한 자동화된 셀프 미디어 운영 기술, 콘텐츠 생성, 이미지 생성, 데이터 관리 및 자동화된 게시를 지원합니다. Tavily API 검색, 콘텐츠 생성, 이미지 생성, HAP 데이터 관리 및 자동 게시의 전체 워크플로를 통합합니다. 일일 트렌드 콘텐츠 추천 기능을 추가합니다. 이미지 생성은 병렬 생성, 자동 다운로드 및 기존 이미지 자동 건너뛰기를 지원합니다. 계정 페르소나 구성은 HAP "계정 페르소나" 테이블에서 읽어야 합니다. 로컬 파일은 엄격히 금지됩니다.
