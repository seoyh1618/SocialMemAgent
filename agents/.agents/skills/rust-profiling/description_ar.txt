مهارة تحديد ملامح الصدأ لتحليل الأداء. يُستخدم عند إنشاء مخططات اللهب من ثنائيات Rust، أو قياس تضخم الشكل الأحادي باستخدام خطوط llvm، أو تحليل الحجم الثنائي باستخدام تضخم الشحن، أو وضع العلامات المعيارية الدقيقة باستخدام Criterion، أو تفسير الإطارات المضمنة في ملفات التعريف. يتم التنشيط عند الاستفسارات حول مخطط Flamegraph للبضائع، أو تضخم البضائع، أو خطوط LLVM، أو معايير المعيار، أو تحديد مواصفات أداء الصدأ، أو تحليل الحجم الثنائي.
