Навыки профилирования ржавчины для анализа производительности. Используйте при создании Flamegraphs из двоичных файлов Rust, измерении раздувания мономорфизации с помощью Cargo-LLVM-линий, анализе размера двоичных файлов с помощью Cargo-Bloat, микробенчмаркинге с помощью Criterion или интерпретации встроенных кадров в профилях. Активируется по запросам о грузо-флейме, грузо-раздувании, грузо-llvm-линиях, тестах критериев, профилировании производительности Rust или анализе двоичного размера.
