Habilidad de creación de perfiles de óxido para análisis de rendimiento. Úselo al generar flamegraphs a partir de binarios de Rust, medir la hinchazón de monomorfización con cargo-llvm-lines, analizar el tamaño binario con cargo-bloat, realizar microbenchmarking con Criterion o interpretar marcos insertados en perfiles. Se activa ante consultas sobre cargo flamegraph, cargo-bloat, cargo-llvm-lines, puntos de referencia de Criterion, perfiles de rendimiento de Rust o análisis de tamaño binario.
