Создайте фирменное изображение в социальной сети, анонсирующее новую функцию или обновление. Анализирует историю git, автоматически определяет бренд по базе кода (Tailwind, переменные CSS, токены дизайна), копирует элементы пользовательского интерфейса и фиксирует их с помощью Playwright. Используйте, когда пользователь хочет создать изображение объявления, например «изображение функции», «изображение объявления», «изображение функции в социальных сетях» или хочет визуально объявить об изменении кода.
