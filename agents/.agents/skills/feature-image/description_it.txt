Genera un'immagine sui social media con il marchio che annuncia una nuova funzionalità o un aggiornamento. Analizza la cronologia git, rileva automaticamente il marchio dalla base di codice (Tailwind, variabili CSS, token di progettazione), replica gli elementi dell'interfaccia utente e acquisisce tramite Playwright. Da utilizzare quando l'utente desidera creare un'immagine di annuncio, dice "immagine di funzionalità", "immagine di annuncio", "immagine social per funzionalità" o desidera annunciare visivamente una modifica del codice.
