Générez une image de marque sur les réseaux sociaux annonçant une nouvelle fonctionnalité ou une mise à jour. Analyse l'historique de Git, détecte automatiquement la marque à partir de la base de code (Tailwind, variables CSS, jetons de conception), réplique les éléments de l'interface utilisateur et capture via Playwright. À utiliser lorsque l'utilisateur souhaite créer une image d'annonce, dit « image de fonctionnalité », « graphique d'annonce », « image sociale pour fonctionnalité » ou souhaite annoncer visuellement un changement de code.
