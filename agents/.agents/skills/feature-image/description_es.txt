Genere una imagen de marca en las redes sociales anunciando una nueva función o actualización. Analiza el historial de git, detecta automáticamente la marca a partir del código base (Tailwind, CSS vars, tokens de diseño), replica elementos de la interfaz de usuario y captura a través de Playwright. Úselo cuando el usuario quiera crear una imagen de anuncio, diga "imagen destacada", "gráfico de anuncio", "imagen social para característica" o quiera anunciar visualmente un cambio de código.
