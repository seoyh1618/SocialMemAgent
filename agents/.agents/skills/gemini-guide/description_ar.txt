ابحث عن وثائق Gemini API وأنماط SDK وأفضل الممارسات الحالية عند الإنشاء باستخدام Google Gemini. يقوم بتعيين المواضيع إلى المستندات المحلية المخزنة مؤقتًا والمصادر المباشرة، ويوفر أنماط @google/genai الصحيحة، ويسلط الضوء على الاستخدام المهمل مقابل استخدام واجهة برمجة التطبيقات الحالية. قم بتشغيل "مستندات الجوزاء"، أو "دليل الجوزاء"، أو "كيفية استخدام الجوزاء"، أو "gemini SDK"، أو "@google/genai"، أو عند إنشاء تعليمات برمجية يتم استيرادها من @google/genai أو google-genai.
