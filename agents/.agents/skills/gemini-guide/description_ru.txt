Ознакомьтесь с документацией Gemini API, шаблонами SDK и текущими рекомендациями по созданию приложений с помощью Google Gemini. Сопоставляет темы с локальными кэшированными документами и живыми источниками, предоставляет правильные шаблоны @google/genai и выделяет устаревшие и текущие версии API. Запускайте с помощью «документов Gemini», «руководства по Gemini», «Как использовать Gemini», «Gemini SDK», «@google/genai» или при создании кода, который импортируется из @google/genai или google-genai.
