Intelligente Datenbankabfragefähigkeiten. Stellen Sie über einen SSH-Tunnel eine Verbindung zu Online-Datenbanken her, unterstützen Sie SQL in natürlicher Sprache, führen Sie Abfragen aus und erkunden Sie Tabellenstrukturen. Verwenden Sie diese Fähigkeit, wenn Benutzer Datenbanken abfragen, nach Daten fragen oder Tabellenstrukturen anzeigen müssen.
