Aprovisione nuevos servidores NixOS en Proxmox para este proyecto nix flake. Guía a través del flujo de trabajo completo: creación de contenedores Proxmox LXC, configuración SSH, configuración de Colmena (patrón inicial/completo) e implementación de aplicaciones con proxy nginx, PostgreSQL e imágenes de contenedores.

Úselo cuando: (1) Crear un nuevo servidor/contenedor en Proxmox, (2) Configurar un nuevo host NixOS con Colmena, (3) Implementar aplicaciones con proxy SSL nginx y/o base de datos PostgreSQL, (4) Agregar nuevas imágenes de contenedor al repositorio.
