Предоставьте новые серверы NixOS на Proxmox для этого проекта Nix Flake. Проводит полный рабочий процесс: создание контейнеров Proxmox LXC, настройку SSH, настройку Colmena (шаблон инициализации/полный) и развертывание приложений с прокси-сервером nginx, PostgreSQL и образами контейнеров.

Используйте, когда: (1) Создание нового сервера/контейнера на Proxmox, (2) Настройка нового хоста NixOS с Colmena, (3) Развертывание приложений с прокси-сервером nginx SSL и/или базой данных PostgreSQL, (4) Добавление новых образов контейнеров в репозиторий.
