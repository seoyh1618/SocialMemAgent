この nix flake プロジェクト用に、Proxmox 上に新しい NixOS サーバーをプロビジョニングします。 Proxmox LXC コンテナーの作成、SSH セットアップ、Colmena 構成 (初期化/フル パターン)、nginx プロキシ、PostgreSQL、コンテナー イメージを使用したアプリケーションのデプロイメントなど、完全なワークフローをガイドします。

次の場合に使用します: (1) Proxmox で新しいサーバー/コンテナを作成する、(2) Colmena を使用して新しい NixOS ホストをセットアップする、(3) nginx SSL プロキシや PostgreSQL データベースを使用してアプリケーションをデプロイする、(4) リポジトリに新しいコンテナ イメージを追加する。
