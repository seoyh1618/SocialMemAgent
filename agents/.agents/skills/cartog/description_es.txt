Navegación de gráficos de código y análisis de impacto. Úselo cuando el usuario pregunta "¿dónde está definido X?", "¿cómo se llama X?", "¿quién importa X?", "¿qué depende de X?", "¿qué se rompe si cambio X?", "ayúdame a refactorizar X", "muéstrame el gráfico de llamadas", "encontrar todos los usos de X", "mostrar estructura de archivos", o necesita navegar por el código, localizar definiciones, rastrear dependencias, evaluar el radio de los cambios, admitir la refactorización (cambiar nombre, extraer, mover) o explorar una base de código desconocida. Admite Python, TypeScript/JavaScript, Rust, Go, Ruby.
