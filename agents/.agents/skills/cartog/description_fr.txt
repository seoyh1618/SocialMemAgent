Navigation dans les graphiques de code et analyse d’impact. À utiliser lorsque l'utilisateur demande « où X est défini ? », « qu'est-ce qui appelle X ? », « qui importe X ? », « qu'est-ce qui dépend de X ? », « qu'est-ce qui se casse si je change X ? Prend en charge Python, TypeScript/JavaScript, Rust, Go, Ruby.
