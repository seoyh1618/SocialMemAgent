التنقل في الرسم البياني للكود وتحليل التأثير. يُستخدم عندما يسأل المستخدم "أين يتم تعريف X؟"، أو "ما الذي يستدعي X؟"، أو "من يستورد X؟"، أو "ما الذي يعتمد على X؟"، أو "ما الذي يكسر إذا قمت بتغيير X؟"، أو "مساعدتي في إعادة بناء X"، أو "إظهار الرسم البياني للاتصال"، أو "العثور على جميع استخدامات X"، أو "إظهار بنية الملف"، أو يحتاج إلى التنقل في التعليمات البرمجية، أو تحديد موقع التعريفات، أو تتبع التبعيات، أو تقييم نطاق الانفجار للتغييرات، أو دعم إعادة البناء (إعادة التسمية، أو الاستخراج، أو النقل)، أو استكشاف مكان غير مألوف. قاعدة التعليمات البرمجية. يدعم بايثون، تايب سكريبت/جافا سكريبت، روست، جو، روبي.
