Создает приложения пользовательского интерфейса .NET в Blazor (шаблоны, компоненты, проверка подлинности, тестирование), MAUI (разработка, AOT, тестирование), Uno Platform (ядро, целевые объекты, MCP, тестирование), WPF (современные версии и миграция), WinUI 3, WinForms, специальные возможности, локализация и выбор платформы пользовательского интерфейса. Включает XAML, MVVM, режимы рендеринга и развертывание для конкретной платформы.
