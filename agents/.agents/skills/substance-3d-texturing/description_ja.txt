Adobe Substance 3D Painter のテクスチャリングとマテリアル作成ワークフローの包括的なスキル。このスキルは、PBR マテリアルの作成、Web/ゲーム エンジン用のテクスチャのエクスポート、リアルタイム レンダリング用の 3D アセットの最適化、またはテクスチャ ワークフローの自動化を行うときに使用します。 Substance 3D Painter、PBR テクスチャリング、マテリアル作成、Three.js、Babylon.js、Unity、Unreal、glTF 最適化、または Python API 自動化のテクスチャ エクスポートに関連するタスクをトリガーします。 threejs-webgl、react-three-fiber、および babylonjs-engine マテリアル用に最適化されたテクスチャを作成します。
