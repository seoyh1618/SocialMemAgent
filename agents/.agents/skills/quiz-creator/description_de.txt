Generieren Sie Quizfragen aus Lernmaterialien mit mehreren Fragetypen (MCQ, Richtig/Falsch, Lückentext, Kurzantwort, Zuordnung). Unterstützt Schwierigkeitsgrade und Lösungsschlüssel. Verwenden Sie es zum Erstellen von Übungsquizzen, Prüfungsvorbereitungsmaterialien, Selbstbewertungstools oder Wissensüberprüfungen anhand von Notizen, Lehrbüchern oder anderen Lerninhalten. Auslöser – Quiz erstellen, Fragen generieren, Übungstest durchführen, Wissensbewertung, Quiz aus Notizen.
