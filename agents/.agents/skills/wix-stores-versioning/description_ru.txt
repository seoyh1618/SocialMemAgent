Обеспечение совместимости каталога Wix Stores V1 и V3 SDK. Используйте при построении любой интеграции, которая взаимодействует с продуктами, инвентарем, заказами или коллекциями Wix Stores. Триггеры включают Wix Stores, API продуктов, API инвентаризации, версию каталога, продукты V3, V1 против V3, продукты магазина, queryProducts, getProduct.
