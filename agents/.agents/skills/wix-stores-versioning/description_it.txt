Gestire la compatibilità dell'SDK del catalogo Wix Stores V1 e V3. Da utilizzare quando si creano integrazioni che interagiscono con prodotti, inventario, ordini o collezioni di Wix Stores. I trigger includono Wix Stores, API dei prodotti, API dell'inventario, versione del catalogo, prodottiV3, V1 vs V3, prodotti del negozio, queryProducts, getProduct.
