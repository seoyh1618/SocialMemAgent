التعامل مع التوافق مع كتالوج Wix Stores V1 وV3 SDK. يُستخدم عند إنشاء أي تكامل يتفاعل مع منتجات Wix Stores أو المخزون أو الطلبات أو المجموعات. تتضمن المشغلات متاجر Wix، وواجهة برمجة تطبيقات المنتجات، وواجهة برمجة تطبيقات المخزون، وإصدار الكتالوج، والمنتجات V3، وV1 مقابل V3، ومنتجات المتجر، والاستعلام عن المنتجات، وgetProduct.
