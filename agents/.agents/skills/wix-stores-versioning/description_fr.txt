Gérer la compatibilité des SDK Wix Stores Catalog V1 et V3. À utiliser lors de la création d'une intégration qui interagit avec les produits, l'inventaire, les commandes ou les collections Wix Stores. Les déclencheurs incluent Wix Stores, l'API des produits, l'API d'inventaire, la version du catalogue, les produits V3, V1 vs V3, les produits du magasin, queryProducts, getProduct.
