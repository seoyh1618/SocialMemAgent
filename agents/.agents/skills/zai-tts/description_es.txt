Conversión de texto a voz utilizando el servicio GLM-TTS mediante el comando `uvx zai-tts` para generar audio a partir de texto. Úselo cuando (1) el usuario solicite salida de audio/voz con el activador o palabra clave "tts". (2) El contenido debe ser hablado en lugar de leído (multitarea, accesibilidad, podcast, conducir, cocinar). (3) Uso de voces preclonadas para el habla.
