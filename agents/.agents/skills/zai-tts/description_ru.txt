Преобразование текста в речь с использованием службы GLM-TTS с помощью команды uvx zai-tts для генерации звука из текста. Используйте, когда (1) Пользователь запрашивает аудио/голосовой вывод с помощью триггера или ключевого слова «tts». (2) Контент нужно произносить, а не читать (многозадачность, доступность, подкасты, вождение, приготовление пищи). (3) Использование предварительно клонированных голосов для речи.
