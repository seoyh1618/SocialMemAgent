Konvertierung von Text in Sprache mithilfe des GLM-TTS-Dienstes über den Befehl „uvx zai-tts“ zum Generieren von Audio aus Text. Verwenden Sie diese Option, wenn (1) der Benutzer eine Audio-/Sprachausgabe mit dem Auslöser oder Schlüsselwort „tts“ anfordert. (2) Inhalte müssen gesprochen statt gelesen werden (Multitasking, Barrierefreiheit, Podcast, Autofahren, Kochen). (3) Verwendung vorgeklonter Stimmen für die Sprache.
