Conversion de texte en parole à l'aide du service GLM-TTS via la commande `uvx zai-tts` pour générer de l'audio à partir de texte. À utiliser lorsque (1) l'utilisateur demande une sortie audio/voix avec le déclencheur ou le mot-clé « tts ». (2) Le contenu doit être parlé plutôt que lu (multitâche, accessibilité, podcast, conduite automobile, cuisine). (3) Utilisation de voix pré-clonées pour la parole.
