Esamina e avvia lo stato dei repository open source (OSS) controllando o creando scaffolding README, CONTRIBUTING, CODE_OF_CONDUCT, LICENSE, SECURITY, modelli GitHub, flussi di lavoro CI, rinnovamento, pubblicazione (pkg-pr-new + npm Trusted Publishing), linting/formattazione, changeset e automazione. Da utilizzare quando viene richiesto di rivedere la configurazione dell'OSS, controllare l'integrità dell'OSS o impostare/aggiungere file e flussi di lavoro dell'OSS.
