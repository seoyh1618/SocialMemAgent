Utilizza MuMuManager.exe per gestire e utilizzare centralmente le istanze dell'emulatore MuMu 12, tra cui configurazione dell'istanza, creazione/clonazione/eliminazione/rinomina, importazione/esportazione, avvio/chiusura/riavvio/controllo finestra, installazione/disinstallazione/avvio/chiusura app, informazioni sul dispositivo, strumenti di sistema, comandi ADB, driver e ordinamento. Ideale per l'automazione batch di scripting della gestione dell'emulatore MuMu.
