Guide d'architecture du module de modèle de pipeline, couvrant la création de modèles, la gestion des versions, l'instanciation, le contrôle des autorisations et l'intégration du magasin R&D. À utiliser lorsque les utilisateurs doivent développer des fonctionnalités de modèle, implémenter la réutilisation de modèles, gérer les versions de modèles ou effectuer une instanciation par lots.
