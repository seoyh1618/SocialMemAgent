Kit de développement logiciel (SDK) des agents Microsoft 365 pour TypeScript/Node.js. Créez des agents multicanaux pour Teams/M365/Copilot Studio avec le routage AgentApplication, l'hébergement Express, les réponses en streaming et l'intégration du client Copilot Studio. Déclencheurs : « SDK Microsoft 365 Agents », « @microsoft/agents-hosting », « AgentApplication », « startServer », « streamingResponse », « Copilot Studio client », « @microsoft/agents-copilotstudio-client ».
