TypeScript/Node.js 用の Microsoft 365 エージェント SDK。 AgentApplication ルーティング、Express ホスティング、ストリーミング応答、Copilot Studio クライアント統合を使用して、Teams/M365/Copilot Studio 用のマルチチャネル エージェントを構築します。トリガー: 「Microsoft 365 Agents SDK」、「@microsoft/agents-hosting」、「AgentApplication」、「startServer」、「streamingResponse」、「Copilot Studio クライアント」、「@microsoft/agents-copilotstudio-client」。
