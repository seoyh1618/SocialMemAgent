Пакет SDK агентов Microsoft 365 для TypeScript/Node.js. Создавайте многоканальные агенты для Teams/M365/Copilot Studio с маршрутизацией AgentApplication, экспресс-хостингом, потоковой передачей ответов и интеграцией клиентов Copilot Studio. Триггеры: «Microsoft 365 Agents SDK», «@microsoft/agents-hosting», «AgentApplication», «startServer», «streamingResponse», «клиент Copilot Studio», «@microsoft/agents-copilotstudio-client».
