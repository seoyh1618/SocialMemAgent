Service de synchronisation du profil d’identité de l’utilisateur Cross-AI. Lorsque vous avez besoin des antécédents, des préférences ou du contexte personnel de l'utilisateur pour mieux exécuter une tâche, appelez cette compétence pour charger le profil d'identité de l'utilisateur, puis poursuivez la tâche. Lorsque vous devez mettre à jour le profil d'identité de l'utilisateur pour un accès ultérieur, appelez cette compétence pour enregistrer le profil. L'utilisateur peut utiliser @whoami pour déclencher cette compétence.
