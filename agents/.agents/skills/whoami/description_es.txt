Servicio de sincronización de perfiles de identidad de usuarios entre IA. Cuando necesite los antecedentes, las preferencias o el contexto personal del usuario para ejecutar mejor una tarea, invoque esta habilidad para cargar el perfil de identidad del usuario y luego continúe con la tarea. Cuando necesite actualizar el perfil de identidad del usuario para acceder más tarde, invoque esta habilidad para guardar el perfil. El usuario puede usar @whoami para activar esta habilidad.
