Служба синхронизации профилей удостоверений пользователей Cross-AI. Если вам нужны сведения о пользователе, его предпочтениях или личном контексте для лучшего выполнения задачи, вызовите этот навык, чтобы загрузить профиль личности пользователя, а затем продолжите выполнение задачи. Если вам необходимо обновить профиль удостоверения пользователя для последующего доступа, вызовите этот навык, чтобы сохранить профиль. Пользователь может использовать @whoami, чтобы активировать этот навык.
