Extrahiert Text, Tabellen und Bilder aus PDFs (einschließlich gescannter PDFs) mithilfe der Mistral OCR API. Verwenden Sie diese Option, wenn der Benutzer eine OCR-Erkennung (OCR) für ein PDF/Bild, das Extrahieren von Text aus einem PDF, das Parsen eines gescannten Dokuments, das Konvertieren eines PDFs in Markdown oder das Extrahieren strukturierter Felder aus einem Dokument anfordert.
