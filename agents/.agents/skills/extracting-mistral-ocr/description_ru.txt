Извлекает текст, таблицы и изображения из PDF-файлов (включая отсканированные PDF-файлы) с помощью API Mistral OCR. Используйте, когда пользователь просит распознать PDF-файл или изображение, извлечь текст из PDF-файла, проанализировать отсканированный документ, преобразовать PDF-файл в Markdown или извлечь структурированные поля из документа.
