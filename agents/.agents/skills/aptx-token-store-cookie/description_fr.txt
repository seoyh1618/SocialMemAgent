Implémentez le stockage de jetons basé sur les cookies à l'aide de @aptx/token-store-cookie. À utiliser lorsque le code doit stocker des jetons d'authentification dans les cookies du navigateur avec : (1) noms de jetons/méta-cookies configurables, (2) synchronisation automatique de l'expiration avec l'expiration du cookie, (3) options de cookies (chemin/sameSite/secure), (4) intégration de l'API createCookieTokenStore
