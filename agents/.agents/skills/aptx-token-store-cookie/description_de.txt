Implementieren Sie einen Cookie-basierten Token-Speicher mit @aptx/token-store-cookie. Wird verwendet, wenn Code Authentifizierungstokens in Browser-Cookies speichern muss mit: (1) konfigurierbaren Token-/Meta-Cookie-Namen, (2) automatischer „expiresAt“-Synchronisierung zum Cookie-Ablauf, (3) Cookie-Optionen (Pfad/sameSite/sicher), (4) API-Integration „createCookieTokenStore“.
