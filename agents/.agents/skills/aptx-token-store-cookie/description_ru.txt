Внедрите хранилище токенов на основе файлов cookie, используя @aptx/token-store-cookie. Используйте, когда коду необходимо хранить токены аутентификации в файлах cookie браузера с помощью: (1) настраиваемых имен токенов/мета файлов cookie, (2) автоматического срока действия при синхронизации срока действия файлов cookie, (3) параметров файлов cookie (path/sameSite/secure), (4) интеграции API createCookieTokenStore.
