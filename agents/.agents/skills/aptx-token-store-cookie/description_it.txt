Implementa l'archiviazione di token basata su cookie utilizzando @aptx/token-store-cookie. Da utilizzare quando il codice deve memorizzare token di autenticazione nei cookie del browser con: (1) nomi di token/meta cookie configurabili, (2) sincronizzazione con scadenza automatica alla scadenza del cookie, (3) opzioni cookie (percorso/stessoSito/sicuro), (4) integrazione API createCookieTokenStore
