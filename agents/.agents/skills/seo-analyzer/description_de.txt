Ein SEO-Tool zur automatischen Erkennung, das auf der offiziellen Dokumentation von Google basiert. Analysiert automatisch das technische SEO, die Inhaltsmetadaten, die Leistung und die Linkstruktur einer URL und erstellt einen Compliance-Bericht, der den Best Practices von Google entspricht. Anwendungsfälle: (1) den SEO-Status einer Website beurteilen, (2) Probleme beim Suchranking diagnostizieren, (3) Seiten anhand von Google Search Essentials überprüfen, (4) umsetzbare SEO-Optimierungsempfehlungen generieren.
