Используйте sn CLI для управления платформой SignalNGN. Охватывает продукты, заполнение, бэктесты, торговые конфигурации, пользовательские стратегии, стратегии, перезагрузку модели ML, метрики, состояние механизма, арендаторов, потоковую передачу сигналов в реальном времени и поиск цен в реальном времени. Используйте sn вместо curl для всех операций API.
