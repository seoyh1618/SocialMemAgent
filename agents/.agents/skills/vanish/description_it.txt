Carica e condividi file tramite URL pubblici temporanei utilizzando la CLI di Vanish (vanish-cli). Da utilizzare quando l'utente desidera caricare file, condividere screenshot o immagini, ottenere un URL pubblico per un file, gestire caricamenti di file temporanei o incorporare immagini in markdown/PR. Trigger sulla condivisione di file, collegamenti temporanei, caricamenti di screenshot e comandi di scomparsa.
