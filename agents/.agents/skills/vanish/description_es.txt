Cargue y comparta archivos a través de URL públicas temporales utilizando Vanish CLI (vanish-cli). Úselo cuando el usuario quiera cargar archivos, compartir capturas de pantalla o imágenes, obtener una URL pública para un archivo, administrar cargas de archivos temporales o incrustar imágenes en rebajas/PR. Activadores al compartir archivos, enlaces temporales, carga de capturas de pantalla y comandos de desaparición.
