Диагностика и устранение проблем конфигурации Vite + React 19 для предварительных сборок TypeScript SPA и WASM. Специализируется на среде выполнения JSX React 19, @vitejs/plugin-react, псевдонимах путей, маршрутизации SPA и поведении сервера разработки, поэтому приложение и предварительный просмотр в браузере четко объединяются без ручных проб и ошибок.
