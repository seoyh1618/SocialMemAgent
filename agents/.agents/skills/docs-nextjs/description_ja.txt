CE.SDK Next.js のリファレンス ドキュメント、ガイド、構成ページを検索してください。

ユーザーが Next.js の CE.SDK ドキュメントを必要とする場合 (構成、UI カスタマイズ、
エクスポート オプション、機能ガイド、または開始手順。 「IMG.LY」、「CreativeEditor」によってもトリガーされます。
「CE.SDK」、またはユーザーが既存の Next.js ドキュメント ページを必要とする場合は「cesdk」。

コードの作成 (build を使用) や概念の説明 (explain を使用) のためではありません。

<例>
コンテキスト: ユーザーが Next.js の構成について尋ねる
ユーザー: 「Next.js でエディターを構成するにはどうすればよいですか?」
アシスタント: 「/cesdk:docs-nextjs を使用して構成オプションを調べます。」
</例>

コンテキスト: SSR の互換性についてユーザーが尋ねる
