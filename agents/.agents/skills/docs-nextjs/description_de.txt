Suchen Sie nach CE.SDK Next.js-Referenzdokumenten, Leitfäden und Konfigurationsseiten.

Verwenden Sie diese Option, wenn der Benutzer CE.SDK-Dokumente für Next.js benötigt – Konfiguration, UI-Anpassung,
Exportoptionen, Funktionshandbücher oder Anleitungen für den Einstieg. Wird auch durch „IMG.LY“, „CreativeEditor“ ausgelöst.
„CE.SDK“ oder „cesdk“, wenn der Benutzer eine vorhandene Next.js-Dokumentseite benötigt.

Nicht zum Schreiben von Code (Build verwenden) oder Konzepterklärungen (Explain verwenden).

<Beispiel>
Kontext: Der Benutzer fragt nach der Konfiguration von Next.js
Benutzer: „Wie konfiguriere ich den Editor in Next.js?“
Assistent: „Ich verwende /cesdk:docs-nextjs, um nach Konfigurationsoptionen zu suchen.“
</Beispiel>
