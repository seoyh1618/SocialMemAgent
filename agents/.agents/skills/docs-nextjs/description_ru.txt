Найдите справочную документацию, руководства и страницы конфигурации CE.SDK Next.js.

Используйте, когда пользователю нужна документация CE.SDK для Next.js — настройка, настройка пользовательского интерфейса,
параметры экспорта, руководства по функциям или инструкции по началу работы. Также запускается "IMG.LY", "CreativeEditor",
«CE.SDK» или «cesdk», если пользователю нужна существующая страница документа Next.js.

Не для написания кода (используйте сборку) или объяснения концепций (используйте объяснение).

<пример>
Контекст: пользователь спрашивает о конфигурации Next.js.
пользователь: «Как мне настроить редактор в Next.js?»
помощник: «Я буду использовать /cesdk:docs-nextjs для поиска параметров конфигурации».
</пример>

<пример>
Контекст: пользователь спрашивает о совместимости SSR.
пользователь: «Работает ли CE.SDK с серверным рендерингом Next.js?»
помощник: «Позвольте мне использовать /cesdk:docs-nextjs, чтобы найти соответствующую документацию».
</пример>
