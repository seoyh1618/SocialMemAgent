Recherchez les documents de référence, les guides et les pages de configuration CE.SDK Next.js.

À utiliser lorsque l'utilisateur a besoin de documents CE.SDK pour Next.js : configuration, personnalisation de l'interface utilisateur,
options d'exportation, guides de fonctionnalités ou instructions de démarrage. Également déclenché par "IMG.LY", "CreativeEditor",
"CE.SDK" ou "cesdk" lorsque l'utilisateur a besoin d'une page de documentation Next.js existante.

Pas pour écrire du code (utiliser build) ou des explications de concepts (utiliser expliquer).

<exemple>
Contexte : l'utilisateur pose des questions sur la configuration de Next.js
user : "Comment configurer l'éditeur dans Next.js ?"
assistant : "J'utiliserai /cesdk:docs-nextjs pour rechercher les options de configuration."
</exemple>

<exemple>
Contexte : l'utilisateur pose des questions sur la compatibilité SSR
user : "CE.SDK fonctionne-t-il avec le rendu côté serveur Next.js ?"
assistant : "Permettez-moi d'utiliser /cesdk:docs-nextjs pour trouver la documentation pertinente."
</exemple>
