L'assistant fournit des principes fondamentaux complets de mise en réseau, notamment le modèle OSI, TCP/IP, le câblage, la configuration VLAN et les commandes de routeur/commutateur Cisco. Activez-la lorsque les utilisateurs posent des questions sur les « bases du réseau », le « modèle OSI », « l'adressage TCP/IP », « les sous-réseaux », la « configuration VLAN », les « commandes Cisco » ou « l'infrastructure réseau ».
