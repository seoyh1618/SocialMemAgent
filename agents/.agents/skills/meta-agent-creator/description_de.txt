Erstellt benutzerdefinierte Subagenten für Claude Code mit YAML-Frontmatter-Konfiguration in Markdown-Dateien. Umfasst Agenten-Scoping (Projekt, Benutzer, CLI, Plugin), Tool-Zugriffskontrolle, Modellauswahl, Berechtigungsmodi, Vorladen von Fertigkeiten und Lebenszyklus-Hooks.

Verwenden Sie diese Option, wenn Sie spezialisierte Subagenten erstellen, den Toolzugriff konfigurieren, Modelle auswählen, Berechtigungsmodi festlegen oder Delegierungsmuster entwerfen. Zur Agentenerstellung, Subagentenkonfiguration und benutzerdefinierten Agenteneinrichtung verwenden.
