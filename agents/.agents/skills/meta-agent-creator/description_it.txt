Crea agenti secondari personalizzati per Claude Code con la configurazione del frontmatter YAML nei file Markdown. Copre l'ambito dell'agente (progetto, utente, CLI, plug-in), controllo dell'accesso allo strumento, selezione del modello, modalità di autorizzazione, precaricamento delle competenze e hook del ciclo di vita.

Da utilizzare durante la creazione di agenti secondari specializzati, la configurazione dell'accesso agli strumenti, la selezione di modelli, l'impostazione delle modalità di autorizzazione o la progettazione di modelli di delega. Utilizzare per la creazione dell'agente, la configurazione dell'agente secondario e l'impostazione dell'agente personalizzato.
