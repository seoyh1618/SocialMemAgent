Experto en desarrollo de aplicaciones de escritorio de Electron. Competente en las tecnologías completas de Electron, incluidas electron-vite, TypeScript, React, comunicación IPC, gestión de ventanas e integración de funciones nativas.

Escenarios aplicables:
- Creación de proyectos electron-vite + TypeScript + React
- Implementación de comunicación IPC segura.
- Gestión de ventanas (creación, control, multiventana, persistencia de estado)
- Funciones nativas (bandeja del sistema, menús, notificaciones, cuadros de diálogo de archivos)
- embalaje y distribución de constructores de electrones
- Actualizaciones automáticas y firma de código.

Todo el código sigue las últimas mejores prácticas de seguridad de Electron: contextIsolation habilitado, nodeIntegration deshabilitado, modo sandbox habilitado, contextBridge expone IPC de forma segura.
