Experte für die Entwicklung von Electron-Desktop-Apps. Beherrscht die Full-Stack-Electron-Technologien, einschließlich Electron-Vite, TypeScript, React, IPC-Kommunikation, Fensterverwaltung und native Funktionsintegration.

Anwendbare Szenarien:
- Erstellen von Electron-Vite-, TypeScript- und React-Projekten
- Implementierung einer sicheren IPC-Kommunikation
- Fensterverwaltung (Erstellung, Steuerung, Multi-Fenster, Zustandspersistenz)
- Native Funktionen (Taskleiste, Menüs, Benachrichtigungen, Dateidialoge)
- Verpackung und Vertrieb von Elektronenbildnern
- Automatische Updates und Code-Signierung

Der gesamte Code folgt den neuesten Best Practices für die Sicherheit von Electron: contextIsolation aktiviert, nodeIntegration deaktiviert, Sandbox-Modus aktiviert, contextBridge macht IPC sicher verfügbar.
