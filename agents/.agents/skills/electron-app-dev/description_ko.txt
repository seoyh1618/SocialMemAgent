Electron 데스크톱 앱 개발 전문가. Electron-vite, TypeScript, React, IPC 통신, 창 관리 및 기본 기능 통합을 포함한 전체 스택 Electron 기술에 능숙합니다.

적용 가능한 시나리오:
- Electron-vite + TypeScript + React 프로젝트 만들기
- 안전한 IPC 통신 구현
- 창 관리(생성, 제어, 다중 창, 상태 지속성)
- 기본 기능(시스템 트레이, 메뉴, 알림, 파일 대화 상자)
- 전자 빌더 포장 및 유통
- 자동 업데이트 및 코드 서명

모든 코드는 최신 Electron 보안 모범 사례를 따릅니다. contextIsolation 활성화, nodeIntegration 비활성화, 샌드박스 모드 활성화, contextBridge는 IPC를 안전하게 노출합니다.
