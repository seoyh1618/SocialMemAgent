Эксперт по разработке настольных приложений Electron. Владение полным набором технологий Electron, включая Electron-Vite, TypeScript, React, IPC-коммуникацию, управление окнами и интеграцию встроенных функций.

Применимые сценарии:
- Создание проектов Electro-Vite + TypeScript + React.
- Внедрение безопасной связи IPC.
- Управление окнами (создание, контроль, многооконность, сохранение состояния)
- Встроенные функции (панель задач, меню, уведомления, диалоговые окна файлов)
- упаковка и распространение электронов
- Автоматические обновления и подписание кода

Весь код соответствует последним лучшим практикам безопасности Electron: contextIsolation включен, nodeIntegration отключен, режим песочницы включен, contextBridge надежно предоставляет IPC.
