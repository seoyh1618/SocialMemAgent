Genere un proyecto completo a partir de la plantilla de pila PRD +: estructura de directorios, configuraciones, CLAUDE.md, git repo y GitHub push. Úselo cuando el usuario diga "proyecto scaffold", "crear nuevo proyecto", "iniciar nueva aplicación", "proyecto de arranque" o "configurar desde PRD". Utiliza SoloGraph para patrones y Context7 para las últimas versiones. NO lo use para funciones de planificación (usar /plan) o generación de PRD (usar /validar).
