Создайте полный проект из PRD + шаблон стека — структура каталогов, конфигурации, CLAUDE.md, git-репо и GitHub push. Используйте, когда пользователь говорит «проект scaffold», «создать новый проект», «запустить новое приложение», «проект начальной загрузки» или «настроить из PRD». Использует SoloGraph для шаблонов и Context7 для последних версий. НЕ используйте функции планирования (используйте /plan) или создания PRD (используйте /validate).
