Genera un progetto completo da PRD + modello di stack: struttura di directory, configurazioni, CLAUDE.md, repository git e push GitHub. Da utilizzare quando l'utente dice "progetto impalcatura", "crea nuovo progetto", "avvia nuova app", "progetto bootstrap" o "configura da PRD". Utilizza SoloGraph per i pattern e Context7 per le versioni più recenti. NON utilizzare per funzionalità di pianificazione (usa /plan) o generazione PRD (usa /convalida).
