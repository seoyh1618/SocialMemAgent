Générez un projet complet à partir du modèle de pile PRD + – structure de répertoires, configurations, CLAUDE.md, dépôt git et push GitHub. À utiliser lorsque l'utilisateur dit « projet d'échafaudage », « créer un nouveau projet », « démarrer une nouvelle application », « projet d'amorçage » ou « configurer à partir du PRD ». Utilise SoloGraph pour les modèles et Context7 pour les dernières versions. Ne PAS utiliser pour les fonctionnalités de planification (utiliser /plan) ou la génération de PRD (utiliser /valider).
