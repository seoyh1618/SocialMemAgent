openfootball (football.json) es una colección gratuita, abierta y de dominio público de datos de partidos de fútbol en formato JSON. Cubre las principales ligas de todo el mundo, incluidas la Premier League inglesa, la Bundesliga, La Liga, la Serie A, la Ligue 1, la Copa del Mundo, la Eurocopa y la Liga de Campeones. Utilice esta habilidad para obtener partidos, resultados y puntajes históricos y actuales de la temporada. No se requiere clave API ni autenticación.
