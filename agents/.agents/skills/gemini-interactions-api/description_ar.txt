استخدم هذه المهارة عند كتابة التعليمات البرمجية التي تستدعي Gemini API لإنشاء النص، أو الدردشة متعددة الأدوار، أو فهم الوسائط المتعددة، أو إنشاء الصور، أو الاستجابات المتدفقة، أو مهام البحث في الخلفية، أو استدعاء الوظائف، أو الإخراج المنظم، أو الترحيل من واجهة برمجة التطبيقات الخاصة بـ generatorContent القديمة. تغطي هذه المهارة Interactions API، وهي الطريقة الموصى بها لاستخدام نماذج Gemini والوكلاء في Python وTypeScript.
