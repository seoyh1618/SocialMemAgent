Utilice esta habilidad al escribir código que llame a la API de Gemini para generación de texto, chat de múltiples turnos, comprensión multimodal, generación de imágenes, transmisión de respuestas, tareas de investigación de antecedentes, llamada de funciones, salida estructurada o migración desde la antigua API generateContent. Esta habilidad cubre la API de Interacciones, la forma recomendada de utilizar modelos y agentes de Gemini en Python y TypeScript.
