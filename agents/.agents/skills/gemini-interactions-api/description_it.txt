Utilizza questa competenza quando scrivi codice che chiama l'API Gemini per la generazione di testo, chat a più turni, comprensione multimodale, generazione di immagini, risposte in streaming, attività di ricerca in background, chiamata di funzioni, output strutturato o migrazione dalla vecchia API generateContent. Questa competenza copre l'API Interactions, il modo consigliato per utilizzare modelli e agenti Gemini in Python e TypeScript.
