Supporta i comandi: [commit|commit_a commit_b] [--verify] [--apply] [--commit] Abilità di revisione interattiva di git diff. Analizza l'output di git diff in singoli blocchi, presenta ogni blocco all'utente con l'analisi per le decisioni di accettazione/rifiuto, verifica la copertura completa e genera un rapporto di revisione Markdown.
