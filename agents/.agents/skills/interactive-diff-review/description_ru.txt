Поддерживает команды: [commit|commit_a commit_b] [--verify] [--apply] [--commit] Интерактивный навык просмотра различий в git. Разбирает выходные данные git diff на отдельные фрагменты, представляет каждый фрагмент пользователю с анализом для принятия/отклонения решений, проверяет полное покрытие и генерирует отчет о проверке Markdown.
