Operazioni, flussi di lavoro e automazione della CLI Git per lo sviluppo moderno (2025). Da utilizzare quando si lavora con repository, commit, rami, fusioni, rebasing, alberi di lavoro, sottomoduli o architetture multi-repo. Include modelli di flusso di lavoro con agenti paralleli, strategie di unione, risoluzione dei conflitti e ottimizzazione di repository di grandi dimensioni. Trigger su comandi git, controllo della versione, conflitti di unione, configurazione dell'albero di lavoro, gestione dei sottomoduli, risoluzione dei problemi del repository, strategia del ramo, operazioni di rebase, decisioni di selezione e integrazione git CI/CD.
