Operaciones, flujos de trabajo y automatización de Git CLI para el desarrollo moderno (2025). Úselo cuando trabaje con repositorios, confirmaciones, ramas, fusiones, cambios de base, árboles de trabajo, submódulos o arquitecturas de múltiples repositorios. Incluye patrones de flujo de trabajo de agentes paralelos, estrategias de fusión, resolución de conflictos y optimización de repositorios grandes. Activadores de comandos git, control de versiones, conflictos de fusión, configuración del árbol de trabajo, administración de submódulos, resolución de problemas del repositorio, estrategia de rama, operaciones de rebase, decisiones selectivas e integración de git CI/CD.
