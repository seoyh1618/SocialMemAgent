Разработайте и создайте пакеты упражнений для глав учебника. Поддерживает упражнения по кодированию (сборка/отладка, применение/измерение) И упражнения по решению бизнес-проблем (практическое занятие/проектирование с использованием исходных данных). Учитывая номер главы или путь, определяет аудиторию (новичок/практик/разработчик), анализирует уроки, группирует их в 5–8 модулей, разрабатывает упражнения (по 2 на модуль + 2–3 завершающих элемента), создает хранилище упражнений, файл урока, файл сводки и обрабатывает перенумерацию викторин. Используйте, когда вас просят создать упражнения для главы, например, «/exercision-pack ch 4» или «/exercision-pack Chapter 6». Вызывает параллельные агенты для создания репозитория и написания уроков, а затем создает перекрестные ссылки на выходные данные.
