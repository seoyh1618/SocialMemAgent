Progetta e genera pacchetti di esercizi per i capitoli dei libri di testo. Supporta esercizi di codifica (creazione/debug, applicazione/misurazione) ED esercizi di risoluzione dei problemi aziendali (pratica/progettazione con dati iniziali). Dato un numero o un percorso di capitolo, rileva il pubblico (principiante/praticante/sviluppatore), analizza le lezioni, raggruppa in 5-8 moduli, progetta esercizi (2 per modulo + 2-3 capisaldi), genera il repository degli esercizi, il file delle lezioni, il file di riepilogo e gestisce la rinumerazione dei quiz. Da utilizzare quando viene richiesto di creare esercizi per un capitolo, ad esempio "/exercise-pack capitolo 4" o "/exercise-pack capitolo 6". Richiama agenti paralleli per la generazione di repository e la scrittura delle lezioni, quindi esegue riferimenti incrociati agli output.
