Очистка миграции внешнего интерфейса для перехода Recollect API v2. Этот навык следует использовать при переключении вызывающих интерфейсов с устаревших URL-адресов маршрутизатора Pages на URL-адреса маршрутизатора приложений версии 2, обновлении обработчиков мутаций, удалении устаревших помощников из supabaseCrudHelpers, удалении устаревших типов из apiTypes.ts и удалении старых файлов Pages Router. Также используйте, когда пользователь упоминает этап 13, обновления вызывающих абонентов, миграцию интерфейса, очистку устаревших версий или просит подключить маршруты v2 к пользовательскому интерфейсу.
