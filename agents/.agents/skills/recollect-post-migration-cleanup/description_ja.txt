Recollect API v2 移行のためのフロントエンド移行のクリーンアップ。このスキルは、フロントエンド呼び出し元をレガシー Pages Router URL から v2 App Router URL に切り替える場合、ミューテーション フックを更新する場合、supabaseCrudHelpers からレガシー ヘルパーを削除する場合、apiTypes.ts からレガシー タイプを削除する場合、および古い Pages Router ファイルを削除する場合に使用する必要があります。ユーザーがフェーズ 13、呼び出し元の更新、フロントエンドの移行、レガシー クリーンアップについて言及したり、v2 ルートを UI に接続するように要求したりする場合にも使用します。
