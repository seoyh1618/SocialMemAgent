Bereinigung der Frontend-Migration für den Übergang zur Recollect API v2. Dieser Skill sollte verwendet werden, wenn Frontend-Aufrufer von alten Pages Router-URLs auf v2 App Router-URLs umgestellt werden, Mutations-Hooks aktualisiert werden, ältere Hilfsprogramme aus supabaseCrudHelpers entfernt werden, ältere Typen aus apiTypes.ts entfernt werden und alte Pages Router-Dateien gelöscht werden. Verwenden Sie es auch, wenn der Benutzer Phase 13, Anruferaktualisierungen, Frontend-Migration, Legacy-Bereinigung erwähnt oder darum bittet, v2-Routen mit der Benutzeroberfläche zu verbinden.
