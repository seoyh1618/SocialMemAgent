تنظيف ترحيل الواجهة الأمامية لانتقال Recollect API v2. يجب استخدام هذه المهارة عند تحويل المتصلين بالواجهة الأمامية من عناوين URL القديمة لجهاز توجيه الصفحات إلى عناوين URL لجهاز توجيه التطبيقات v2، وتحديث خطافات التحويل، وإزالة المساعدين القديمين من SupabaseCrudHelpers، وإزالة الأنواع القديمة من apiTypes.ts، وحذف ملفات Pages Router القديمة. استخدمه أيضًا عندما يذكر المستخدم المرحلة 13، أو تحديثات المتصل، أو ترحيل الواجهة الأمامية، أو التنظيف القديم، أو يطلب توصيل مسارات الإصدار الثاني إلى واجهة المستخدم.
