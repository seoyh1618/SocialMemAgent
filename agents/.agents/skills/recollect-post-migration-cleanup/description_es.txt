Limpieza de migración de frontend para la transición de Recollect API v2. Esta habilidad debe usarse al cambiar las personas que llaman desde la interfaz de URL de Pages Router heredada a la versión 2 de URL de App Router, actualizar enlaces de mutación, eliminar ayudantes heredados de supabaseCrudHelpers, eliminar tipos heredados de apiTypes.ts y eliminar archivos antiguos de Pages Router. Úselo también cuando el usuario mencione la Fase 13, actualizaciones de las personas que llaman, migración de frontend, limpieza heredada o solicite conectar rutas v2 a la interfaz de usuario.
