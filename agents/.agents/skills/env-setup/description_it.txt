Strumento di sincronizzazione con un clic dell'ambiente Claude Code. Sincronizza tutte le configurazioni dal repository GitHub in locale: stili di output, prompt globali CLAUDE.md, configurazione del server MCP. Adatto per ambienti unificati su più dispositivi, ripristino quando si cambia computer, condivisione della configurazione del team, ecc. Utilizza questa competenza quando gli utenti devono sincronizzare la configurazione dell'ambiente Claude Code dal repository GitHub.
