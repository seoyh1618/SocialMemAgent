Ein-Klick-Synchronisierungstool für die Claude Code-Umgebung. Synchronisieren Sie alle Konfigurationen vom GitHub-Repository mit dem lokalen: Ausgabestile, globale CLAUDE.md-Eingabeaufforderungen, MCP-Serverkonfiguration. Geeignet für eine einheitliche Umgebung auf mehreren Geräten, Wiederherstellung beim Computerwechsel, gemeinsame Nutzung von Teamkonfigurationen usw. Verwenden Sie diese Fähigkeit, wenn Benutzer die Claude Code-Umgebungskonfiguration aus dem GitHub-Repository synchronisieren müssen.
