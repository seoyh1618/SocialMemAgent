Herramienta de sincronización con un solo clic del entorno Claude Code. Sincronice todas las configuraciones del repositorio de GitHub al local: estilos de salida, mensajes globales CLAUDE.md, configuración del servidor MCP. Adecuado para un entorno unificado en múltiples dispositivos, recuperación al cambiar de computadora, compartir configuración de equipo, etc. Utilice esta habilidad cuando los usuarios necesiten sincronizar la configuración del entorno de Claude Code desde el repositorio de GitHub.
