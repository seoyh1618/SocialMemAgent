Outil de synchronisation en un clic de l'environnement Claude Code. Synchronisez toutes les configurations du référentiel GitHub vers le local : styles de sortie, invites globales CLAUDE.md, configuration du serveur MCP. Convient pour un environnement unifié sur plusieurs appareils, la récupération lors du changement d'ordinateur, le partage de configuration d'équipe, etc. Utilisez cette compétence lorsque les utilisateurs doivent synchroniser la configuration de l'environnement Claude Code à partir du référentiel GitHub.
