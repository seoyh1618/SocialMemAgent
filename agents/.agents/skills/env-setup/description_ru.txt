Инструмент синхронизации среды Claude Code одним щелчком мыши. Синхронизируйте все конфигурации из репозитория GitHub с локальным: стили вывода, глобальные запросы CLAUDE.md, конфигурацию сервера MCP. Подходит для единой среды на нескольких устройствах, восстановления при переключении компьютеров, совместного использования конфигурации группы и т. д. Используйте этот навык, когда пользователям необходимо синхронизировать конфигурацию среды Claude Code из репозитория GitHub.
