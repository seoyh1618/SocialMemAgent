قم ببناء وكلاء الذكاء الاصطناعي الفعالين وسير العمل الوكيل باستخدام LangGraph في Python. يغطي وكلاء استخدام الأدوات مع حلقات أداة LLM، وسير العمل المتفرع، وذاكرة المحادثة، ومراقبة الإنسان في الحلقة، ومراقبة الإنتاج. استخدم عندما - (1) بناء وكلاء يستخدمون الأدوات والتكرار حتى اكتمال المهمة، (2) إنشاء سير عمل متعدد الخطوات باستخدام الفروع الشرطية، (3) إضافة الثبات/الذاكرة عبر المنعطفات باستخدام نقاط التفتيش، (4) تنفيذ موافقة الإنسان باستخدام المقاطعة ()، (5) تصحيح الأخطاء عبر السفر عبر الزمن أو LangSmith. يغطي StateGraph، والعقد، والحواف، وadd_conditional_edges، وMessagesState، وthread_id، وكائنات الأوامر، ومعالجة ToolMessage. تشمل الأمثلة روبوتات الدردشة، ووكلاء الآلة الحاسبة، وسير العمل المنظم.
