Создавайте агенты ИИ с сохранением состояния и агентские рабочие процессы с помощью LangGraph на Python. Охватывает операторов, использующих инструменты, с циклами инструментов LLM, ветвящимися рабочими процессами, памятью разговоров, человеческим контролем и мониторингом производства. Используйте, когда: (1) создают агенты, использующие инструменты и циклические действия до завершения задачи, (2) создают многоэтапные рабочие процессы с условными ветвями, (3) добавляют постоянство/память между ходами с помощью контрольных точек, (4) реализуют одобрение человека с помощью прерывания(), (5) отладку с помощью путешествий во времени или LangSmith. Охватывает StateGraph, узлы, ребра, add_conditional_edges, MessagesState, thread_id, объекты Command и обработку ToolMessage. Примеры включают чат-ботов, агентов-калькуляторов и структурированные рабочие процессы.
