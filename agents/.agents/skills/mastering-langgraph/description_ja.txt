Python の LangGraph を使用してステートフル AI エージェントとエージェント ワークフローを構築します。 LLM ツール ループ、分岐ワークフロー、会話記憶、人間による監視、および運用監視を備えたツール使用エージェントをカバーします。使用する場合 - (1) ツールを使用し、タスクが完了するまでループするエージェントを構築する、(2) 条件付き分岐を使用したマルチステップのワークフローを作成する、(3) チェックポイント作成ツールを使用してターン全体に永続性/メモリを追加する、(4) Interrupt() を使用して人間による承認を実装する、(5) タイムトラベルまたは LangSmith を使用してデバッグする。 StateGraph、ノード、エッジ、add_conditional_edges、MessagesState、thread_id、Command オブジェクト、および ToolMessage の処理について説明します。例には、チャットボット、電卓エージェント、構造化ワークフローなどがあります。
