Créez des flux de travail durables avec Vercel Workflow DevKit en utilisant les directives « use workflow » et « use step ». À utiliser pour les tâches de longue durée, les tâches en arrière-plan, les agents IA, les webhooks, les tâches planifiées, les tentatives et l'orchestration des flux de travail. Prend en charge Next.js, Vite, Astro, Express, Fastify, Hono, Nitro, Nuxt, SvelteKit.
