Создавайте надежные рабочие процессы с помощью Vercel Workflow DevKit, используя директивы «use workflow» и «use Step». Используйте для долго выполняющихся задач, фоновых заданий, агентов искусственного интеллекта, веб-перехватчиков, запланированных задач, повторных попыток и оркестрации рабочих процессов. Поддерживает Next.js, Vite, Astro, Express, Fastify, Hono, Nitro, Nuxt, SvelteKit.
