قم بإنشاء مخططات معمارية مفصلة ذات طبقات كعناصر HTML قائمة بذاتها مع أيقونات SVG المضمنة، وتخطيط الحاوية المتداخلة لشبكة CSS، وتراكبات اتصال مسار SVG، وأساطير الاتصال المرمزة بالألوان. يُستخدم عندما يطلب المستخدم إنشاء مخططات معمارية، أو مخططات بنية أساسية، أو مخططات طبولوجيا النظام، أو مخططات شبكة، أو مرئيات بنية سحابية، أو مخططات نشر، أو مخططات تدفق التكامل، أو أي طلب يتضمن تمثيلًا مرئيًا لمكونات النظام، والتسلسل الهرمي للاحتواء، والاتصالات البينية. يتم تشغيل مصطلحات مثل "مخطط البنية" أو "مخطط الأشعة تحت الحمراء" أو "مخطط النظام" أو "الطوبولوجيا" أو "مخطط النشر" أو "مخطط الشبكة" أو "هندسة التكامل" أو عندما يقدم المستخدم وصفًا للنظام ويطلب رسمًا مرئيًا/رسميًا.
