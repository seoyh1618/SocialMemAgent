Создавайте подробные диаграммы многоуровневой архитектуры в виде автономных HTML-артефактов со встроенными значками SVG, вложенным макетом контейнера CSS Grid, наложениями соединений путей SVG и легендами соединений с цветовой кодировкой. Используйте, когда пользователь запрашивает создание диаграмм архитектуры, диаграмм инфраструктуры, диаграмм топологии системы, сетевых диаграмм, визуальных элементов облачной архитектуры, диаграмм развертывания, блок-схем интеграции или любого запроса, включающего визуальное представление компонентов системы, их иерархии включения и взаимосвязей. Запускается по таким терминам, как «схема архитектуры», «инфраструктурная диаграмма», «системная диаграмма», «топология», «диаграмма развертывания», «сетевая диаграмма», «архитектура интеграции», или когда пользователь предоставляет описание системы и запрашивает визуальное представление/схему.
