Configuration complète du SDK Sentry pour React Native et Expo. À utiliser lorsqu'on vous demande d'« ajouter Sentry à React Native », « d'installer @sentry/react-native », de « configurer Sentry dans Expo » ou de configurer la surveillance des erreurs, le traçage, le profilage, la relecture de session ou la journalisation pour les applications React Native. Prend en charge Expo géré, Expo nu et Vanilla React Native.
