Configurazione completa dell'SDK Sentry per React Native ed Expo. Utilizzare quando viene richiesto di "aggiungere Sentry a React Native", "installare @sentry/react-native", "configurare Sentry in Expo" o configurare il monitoraggio degli errori, la traccia, la profilazione, la riproduzione della sessione o la registrazione per le applicazioni React Native. Supporta Expo gestito, Expo bare e Vanilla React Native.
