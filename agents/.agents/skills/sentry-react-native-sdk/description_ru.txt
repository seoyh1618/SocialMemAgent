Полная настройка Sentry SDK для React Native и Expo. Используйте, когда вас просят «добавить Sentry в React Native», «установить @sentry/react-native», «настроить Sentry в Expo» или настроить мониторинг ошибок, отслеживание, профилирование, воспроизведение сеанса или ведение журнала для приложений React Native. Поддерживает управляемый Expo, пустой Expo и ванильный React Native.
