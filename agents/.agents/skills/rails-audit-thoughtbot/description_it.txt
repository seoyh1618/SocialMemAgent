Esegui controlli completi del codice delle applicazioni Ruby on Rails basati sulle best practice di thinkbot. Utilizza questa competenza quando l'utente richiede un controllo del codice, una revisione del codice, una valutazione della qualità o un'analisi di un'applicazione Rails. La competenza analizza l'intera base di codice concentrandosi su pratiche di test (RSpec), vulnerabilità di sicurezza, progettazione del codice (controller skinny, modelli di dominio, PORO con ActiveModel), convenzioni Rails, ottimizzazione del database e best practice Ruby. Genera un rapporto dettagliato di controllo del ribasso raggruppato per categoria (test, sicurezza, modelli, controller, progettazione del codice, visualizzazioni) con livelli di gravità (critico, alto, medio, basso) all'interno di ciascuna categoria.
