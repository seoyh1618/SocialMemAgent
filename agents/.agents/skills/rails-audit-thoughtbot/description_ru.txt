Выполняйте комплексный аудит кода приложений Ruby on Rails на основе лучших практик thinkbot. Используйте этот навык, когда пользователь запрашивает аудит кода, проверку кода, оценку качества или анализ приложения Rails. Этот навык анализирует всю кодовую базу с упором на методы тестирования (RSpec), уязвимости безопасности, дизайн кода (тонкие контроллеры, модели предметной области, PORO с ActiveModel), соглашения Rails, оптимизацию базы данных и лучшие практики Ruby. Выводит подробный отчет об аудите уценки, сгруппированный по категориям (Тестирование, Безопасность, Модели, Контроллеры, Разработка кода, Представления) с уровнями серьезности (Критический, Высокий, Средний, Низкий) в каждой категории.
