إجراء عمليات تدقيق شاملة للتعليمات البرمجية لتطبيقات Ruby on Rails استنادًا إلى أفضل ممارسات thinkbot. استخدم هذه المهارة عندما يطلب المستخدم تدقيق التعليمات البرمجية، أو مراجعة التعليمات البرمجية، أو تقييم الجودة، أو تحليل تطبيق Rails. تحلل المهارة قاعدة التعليمات البرمجية بأكملها مع التركيز على ممارسات الاختبار (RSpec)، والثغرات الأمنية، وتصميم التعليمات البرمجية (وحدات التحكم النحيفة، ونماذج المجال، وPORO مع ActiveModel)، واتفاقيات Rails، وتحسين قاعدة البيانات، وأفضل ممارسات Ruby. يُخرج تقريرًا تفصيليًا لتدقيق تخفيض السعر مُجمَّعًا حسب الفئة (الاختبار، والأمان، والنماذج، ووحدات التحكم، وتصميم التعليمات البرمجية، وطرق العرض) مع مستويات الخطورة (حرجة، وعالية، ومتوسطة، ومنخفضة) داخل كل فئة.
