Effectuez des audits complets du code des applications Ruby on Rails basés sur les meilleures pratiques de Thoughtbot. Utilisez cette compétence lorsque l'utilisateur demande un audit de code, une révision de code, une évaluation de la qualité ou une analyse d'une application Rails. La compétence analyse l'ensemble de la base de code en se concentrant sur les pratiques de test (RSpec), les vulnérabilités de sécurité, la conception du code (contrôleurs maigres, modèles de domaine, PORO avec ActiveModel), les conventions Rails, l'optimisation des bases de données et les meilleures pratiques Ruby. Génère un rapport d'audit de démarque détaillé regroupé par catégorie (Tests, Sécurité, Modèles, Contrôleurs, Conception de code, Vues) avec des niveaux de gravité (Critique, Élevé, Moyen, Faible) dans chaque catégorie.
