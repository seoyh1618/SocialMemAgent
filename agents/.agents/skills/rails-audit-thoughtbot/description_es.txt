Realice auditorías integrales de código de aplicaciones Ruby on Rails basadas en las mejores prácticas de thinkbot. Utilice esta habilidad cuando el usuario solicite una auditoría de código, revisión de código, evaluación de calidad o análisis de una aplicación Rails. La habilidad analiza toda la base de código enfocándose en prácticas de prueba (RSpec), vulnerabilidades de seguridad, diseño de código (controladores delgados, modelos de dominio, PORO con ActiveModel), convenciones de Rails, optimización de bases de datos y mejores prácticas de Ruby. Genera un informe de auditoría de rebajas detallado agrupado por categoría (Pruebas, Seguridad, Modelos, Controladores, Diseño de código, Vistas) con niveles de gravedad (Crítico, Alto, Medio, Bajo) dentro de cada categoría.
