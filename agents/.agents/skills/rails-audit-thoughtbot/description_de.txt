Führen Sie umfassende Code-Audits von Ruby on Rails-Anwendungen auf der Grundlage der Best Practices von Thoughtbot durch. Verwenden Sie diese Fähigkeit, wenn der Benutzer eine Codeprüfung, Codeüberprüfung, Qualitätsbewertung oder Analyse einer Rails-Anwendung anfordert. Der Skill analysiert die gesamte Codebasis mit Schwerpunkt auf Testpraktiken (RSpec), Sicherheitslücken, Codedesign (Skinny Controller, Domänenmodelle, PORO mit ActiveModel), Rails-Konventionen, Datenbankoptimierung und Ruby Best Practices. Gibt einen detaillierten Markdown-Auditbericht aus, gruppiert nach Kategorien (Tests, Sicherheit, Modelle, Controller, Codedesign, Ansichten) mit Schweregraden (Kritisch, Hoch, Mittel, Niedrig) innerhalb jeder Kategorie.
