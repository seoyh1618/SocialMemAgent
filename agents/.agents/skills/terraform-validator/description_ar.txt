مجموعة أدوات شاملة للتحقق من صحة تكوينات Terraform وملفات HCL وفحصها واختبارها وأتمتتها. استخدم هذه المهارة عند العمل مع ملفات Terraform (.tf، .tfvars)، أو التحقق من صحة البنية التحتية كرمز، أو تصحيح أخطاء تكوينات Terraform، أو إجراء اختبار التشغيل الجاف باستخدام خطة Terraform، أو العمل مع موفري الخدمات والوحدات المخصصة.
