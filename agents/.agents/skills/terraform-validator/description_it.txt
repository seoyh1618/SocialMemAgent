Kit di strumenti completo per la convalida, l'linting, il test e l'automazione delle configurazioni Terraform e dei file HCL. Utilizza questa competenza quando lavori con file Terraform (.tf, .tfvars), convalidi l'infrastruttura come codice, esegui il debug delle configurazioni Terraform, esegui test di prova con il piano Terraform o lavori con provider e moduli personalizzati.
