Configuración completa de Sentry SDK para Ruby. Úselo cuando se le solicite agregar Sentry a Ruby, instalar sentry-ruby, configurar Sentry en Rails/Sinatra/Rack o configurar monitoreo de errores, seguimiento, registro, métricas, creación de perfiles o crons para aplicaciones Ruby. También maneja la migración desde AppSignal, Honeybadger, Bugsnag, Rollbar o Airbrake. Soporta Rails, Sinatra, Rack, Sidekiq y Resque.
