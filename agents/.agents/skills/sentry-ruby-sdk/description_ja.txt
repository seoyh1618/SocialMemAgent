Ruby 用の完全な Sentry SDK セットアップ。 Sentry を Ruby に追加する、sentry-ruby をインストールする、Rails/Sinatra/Rack で Sentry をセットアップする、または Ruby アプリケーションのエラー監視、トレース、ロギング、メトリクス、プロファイリング、または crons を構成するように求められた場合に使用します。 AppSignal、Honeybadger、Bugsnag、Rollbar、または Airbrake からの移行も処理します。 Rails、Sinatra、Rack、Sidekiq、および Resque をサポートします。
