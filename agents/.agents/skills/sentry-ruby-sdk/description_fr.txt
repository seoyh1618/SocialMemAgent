Configuration complète du SDK Sentry pour Ruby. À utiliser lorsqu'on vous demande d'ajouter Sentry à Ruby, d'installer Sentry-Ruby, de configurer Sentry dans Rails/Sinatra/Rack ou de configurer la surveillance des erreurs, le traçage, la journalisation, les métriques, le profilage ou les crons pour les applications Ruby. Gère également la migration depuis AppSignal, Honeybadger, Bugsnag, Rollbar ou Airbrake. Prend en charge Rails, Sinatra, Rack, Sidekiq et Resque.
