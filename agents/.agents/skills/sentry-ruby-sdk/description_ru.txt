Полная настройка Sentry SDK для Ruby. Используйте, когда вас попросят добавить Sentry в Ruby, установить Sentry-Ruby, настроить Sentry в Rails/Sinatra/Rack или настроить мониторинг ошибок, трассировку, ведение журнала, метрику, профилирование или cron для приложений Ruby. Также обеспечивает миграцию с AppSignal, Honeybadger, Bugsnag, Rollbar или Airbrake. Поддерживает Rails, Sinatra, Rack, Sidekiq и Resque.
