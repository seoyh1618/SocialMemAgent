Migra i progetti da Intercom a Gleap. Rileva l'integrazione dell'SDK Intercom (JavaScript, iOS, Android, React Native, Flutter), la rimuove, installa l'SDK Gleap e sostituisce tutte le chiamate API Intercom con equivalenti Gleap. Copre anche la migrazione dell'API REST lato server. Da utilizzare quando si passa da Intercom a Gleap, si sostituisce Intercom o si esegue la migrazione da Intercom.
