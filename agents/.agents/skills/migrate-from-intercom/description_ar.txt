ترحيل المشاريع من الاتصال الداخلي إلى Gleap. يكتشف تكامل Intercom SDK (JavaScript وiOS وAndroid وReact Native وFlutter)، ويزيله، ويثبت Gleap SDK، ويستبدل جميع مكالمات Intercom API بمكافئات Gleap. يغطي أيضًا ترحيل REST API من جانب الخادم. يُستخدم عند التبديل من Intercom إلى Gleap، أو استبدال Intercom، أو الانتقال بعيدًا عن Intercom.
