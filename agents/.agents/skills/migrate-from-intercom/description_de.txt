Migriert Projekte von Intercom nach Gleap. Erkennt die Intercom SDK-Integration (JavaScript, iOS, Android, React Native, Flutter), entfernt sie, installiert das Gleap SDK und ersetzt alle Intercom API-Aufrufe durch Gleap-Äquivalente. Deckt auch die serverseitige REST-API-Migration ab. Verwenden Sie diese Option, wenn Sie von Intercom zu Gleap wechseln, Intercom ersetzen oder von Intercom weg migrieren.
