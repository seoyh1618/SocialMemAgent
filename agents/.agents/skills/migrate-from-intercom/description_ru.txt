Переносит проекты из Intercom в Gleap. Обнаруживает интеграцию Intercom SDK (JavaScript, iOS, Android, React Native, Flutter), удаляет ее, устанавливает Gleap SDK и заменяет все вызовы Intercom API эквивалентами Gleap. Также рассматривается миграция REST API на стороне сервера. Используйте при переключении с Intercom на Gleap, замене Intercom или переходе с Intercom.
