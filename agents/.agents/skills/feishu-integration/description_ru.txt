Руководство по интеграции API Feishu (Feishu/Lark). Используйте, когда пользователь запрашивает «создать приложение Feishu», «управлять многомерной таблицей», «добавить соавтора», «создать отчет Feishu», «установить разрешения Feishu» или «автоматизировать операции Feishu». Предпочитайте использовать инструменты MCP для интерактивных операций в реальном времени.
