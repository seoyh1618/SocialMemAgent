Guide d'intégration de l'API Feishu (Feishu/Lark). À utiliser lorsque l'utilisateur demande « créer une application Feishu », « gérer une table multidimensionnelle », « ajouter des collaborateurs », « générer des rapports Feishu », « définir des autorisations Feishu » ou « automatiser les opérations Feishu ». Privilégiez l’utilisation des outils MCP pour les opérations interactives en temps réel.
