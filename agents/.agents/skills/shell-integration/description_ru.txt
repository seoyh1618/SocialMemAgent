Шаблоны сценариев оболочки и интеграции терминалов для создания инструментов, интегрируемых с Zsh, Bash и Fish. Охватывает системы завершения (compdef/compadd, Complete/compgen, fish Complete), виджеты ZLE, перехватчики (precmd/preexec/chpwd, PROMPT_COMMAND), строку чтения, ключ связывания, расширение параметров, порядок загрузки ZDOTDIR, системы событий, сокращения, сценарии оболочки POSIX, коды управления терминалом ( escape-последовательности ANSI/CSI), tput, stty, обработку сигналов, управление процессами (управление заданиями, ловушки) и шаблоны распространения подключаемых модулей оболочки.

Используйте при создании плагинов оболочки, написании сценариев завершения, реализации пользовательского интерфейса терминала с escape-последовательностями, управлении точечными файлами, создании сценариев установки, обработке сигналов и управлении процессами или интеграции собственных двоичных файлов с оболочками оболочки.
