Patrones de integración de terminales y secuencias de comandos de Shell para crear herramientas que se integran con Zsh, Bash y Fish. Cubre sistemas de finalización (compdef/compadd, complete/compgen, fish complete), widgets ZLE, ganchos (precmd/preexec/chpwd, PROMPT_COMMAND), línea de lectura, clave de enlace, expansión de parámetros, orden de carga ZDOTDIR, sistemas de eventos, abreviaturas, secuencias de comandos de shell POSIX, códigos de control de terminal (secuencias de escape ANSI/CSI), tput, stty, manejo de señales, gestión de procesos (control de trabajos, trampas) y complementos de shell. patrones de distribución.

Úselo al crear complementos de shell, escribir scripts de finalización, implementar la interfaz de usuario del terminal con secuencias de escape, administrar archivos de puntos, crear scripts de instalación, manejar señales y administración de procesos, o integrar archivos binarios nativos con contenedores de shell.
