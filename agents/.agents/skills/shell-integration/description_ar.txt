البرمجة النصية لـ Shell وأنماط التكامل الطرفية لبناء الأدوات التي تتكامل مع Zsh وBash وFish. يغطي أنظمة الإكمال (compdef/compadd، Complete/compgen، Fish Complete)، أدوات ZLE، الخطافات (precmd/preexec/chpwd، PROMPT_COMMAND)، readline، bindkey، توسيع المعلمة، ترتيب تحميل ZDOTDIR، أنظمة الأحداث، الاختصارات، البرمجة النصية لـ POSIX Shell، رموز التحكم الطرفية (تسلسلات هروب ANSI/CSI)، tput، stty، معالجة الإشارات، إدارة العمليات (التحكم في الوظائف، الفخاخ)، وshell أنماط توزيع البرنامج المساعد.

يُستخدم عند إنشاء المكونات الإضافية لـ Shell، أو كتابة البرامج النصية للإكمال، أو تنفيذ واجهة المستخدم الطرفية بتسلسلات الهروب، أو إدارة ملفات النقاط، أو إنشاء البرامج النصية للتثبيت، أو التعامل مع الإشارات وإدارة العمليات، أو دمج الثنائيات الأصلية مع أغلفة الصدفة.
