Cette compétence doit être utilisée lorsque l'utilisateur doit calculer son impôt sur le revenu, calculer ses déductions ou importer des bordereaux de retenue. Les expressions déclencheurs incluent : « calculer l'impôt sur le revenu », « préparer la déclaration de revenus », « calculer les déductions », « les bordereaux de retenue à l'importation », « le montant de l'impôt sur le revenu », « calculer le montant du paiement », « calculer le montant du remboursement », « Annexe 1 », « Annexe 2 », « Formulaire B », « déductions sur le revenu », « crédits d'impôt ».
