【متى】يتم التمكين عندما يحتوي أمر المستخدم بشكل صريح على "تعاون المخطوطة" أو "تآزر المخطوطة" أو "التعاون مع المخطوطة" أو توجيهات مماثلة؛ 【متى】يتم التمكين عندما يطلب المستخدم استخدام أداة Codex MCP لتحليل التعليمات البرمجية أو إنشاء النموذج الأولي أو المراجعة أو سيناريوهات التطوير التعاوني الأخرى. 【متى لا】لا تستخدمه في مهام تنفيذ التعليمات البرمجية المستقلة، أو كتابة الوثائق النقية، أو سير عمل التطوير المنتظم الذي لا يتضمن المخطوطة.
