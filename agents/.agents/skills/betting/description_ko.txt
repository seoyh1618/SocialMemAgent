베팅 분석 — 배당률 변환, de-vigging, 에지 감지, Kelly 기준, 차익거래 감지, 팔레이 분석 및 라인 이동. 순수 계산, API 호출 없음. ESPN(미국 확률), Polymarket(소수 확률), Kalshi(정수 확률) 등 모든 소스의 확률로 작동합니다.

사용 시기: 사용자가 베팅 크기, 기대값, 에지 분석, 켈리 기준, 차익거래, 팔레이, 라인 이동, 배당률 변환 또는 소스 간 배당률 비교에 대해 질문합니다. 또한 ESPN 및 예측 시장 가격의 배당률이 있고 베팅의 예상 가치가 긍정적인지 평가하려는 경우에도 사용합니다.
다음의 경우에는 사용하지 마십시오: 사용자가 실시간 확률 또는 시장 데이터를 요청합니다. 폴리마켓, kalshi 또는 스포츠 관련 기술을 사용하여 먼저 확률을 가져온 다음 이 기술을 사용하여 분석합니다.
