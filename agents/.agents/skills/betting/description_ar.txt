تحليل الرهان - تحويل الاحتمالات، وإلغاء التلاعب، واكتشاف الحواف، ومعيار كيلي، واكتشاف المراجحة، وتحليل المراهنة، وحركة الخط. حساب خالص، لا توجد مكالمات API. يعمل مع الاحتمالات من أي مصدر: ESPN (الاحتمالات الأمريكية)، Polymarket (الاحتمالات العشرية)، Kalshi (احتمالات الأعداد الصحيحة).

يُستخدم عندما: يسأل المستخدم عن حجم الرهان، أو القيمة المتوقعة، أو تحليل الحافة، أو معيار كيلي، أو المراجحة، أو المكافآت، أو حركة الخط، أو تحويل الاحتمالات، أو مقارنة الاحتمالات عبر المصادر. استخدمه أيضًا عندما يكون لديك احتمالات من ESPN وسعر سوق متوقع وترغب في تقييم ما إذا كان الرهان له قيمة متوقعة إيجابية.
لا تستخدم عندما: يطلب المستخدم الحصول على احتمالات حية أو بيانات السوق — استخدم polymarket أو kalshi أو المهارة الخاصة بالرياضة لجلب الاحتمالات أولاً، ثم استخدم هذه المهارة لتحليلها.
