Node.js および PHP アプリケーション向けの Platformatic Watt を統合、構成、デプロイします。
ユーザーが次のことを要求する場合に使用します。
- 「ワットを追加」、「ワットをセットアップ」、「ワットを統合」、「ワットを構成」
- 「ワットでデプロイ」、「アプリをコンテナ化」、「Kubernetes にデプロイ」
- 「ワットに移行」、「アプリをワットに移植」
- 「watt.json の作成」、「プラットフォームの構成」
- 「wattpm」、「wattpm 作成」、「wattpm インジェクト」、「wattpm ログ」
- wattpm CLI コマンドを使用し、実行中のアプリケーションを管理します
- Node.js アプリケーション サーバーと連携する
- Node.js で PHP、WordPress、または Laravel を実行する
Next.js、Express、Fastify、Koa、Remix、Astro、NestJS、PHP、WordPress、Laravel をサポートします。
