Интегрируйте, настройте и разверните Platformatic Watt для приложений Node.js и PHP.
Используйте, когда пользователи просят:
- «добавить ватт», «настроить ватт», «интегрировать ватт», «настроить ватт»
- «развернуть с помощью ватта», «контейнеризировать мое приложение», «развернуть в Kubernetes»
- «перейти на ватт», «перенести мое приложение на ватт»
- «создать What.json», «настроить платформу»
- «ватт/мин», «ватт/м создать», «ватт/м инъекция», «журналы ватт/мин»
- использовать команды CLI Whattpm, управлять запущенными приложениями
- работа с серверами приложений Node.js
— запускайте PHP, WordPress или Laravel в Node.js
Поддерживает Next.js, Express, Fastify, Koa, Remix, Astro, NestJS, PHP, WordPress и Laravel.
