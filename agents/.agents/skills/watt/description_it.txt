Integra, configura e distribuisci Platformatic Watt per applicazioni Node.js e PHP.

- "aggiungi watt", "imposta watt", "integra watt", "configura watt"
- "distribuisci con watt", "containerizza la mia app", "distribuisci su Kubernetes"
- "migra a watt", "porta la mia app a watt"
- "crea watt.json", "configura piattaforma"
- "wattpm", "wattpm create", "wattpm inject", "wattpm logs"
- utilizzare i comandi CLI wattpm, gestire le applicazioni in esecuzione
- lavorare con i server delle applicazioni Node.js
- esegui PHP, WordPress o Laravel in Node.js
Supporta Next.js, Express, Fastify, Koa, Remix, Astro, NestJS, PHP, WordPress e Laravel.
