为 Node.js 和 PHP 应用程序集成、配置和部署 Platformatic Watt。
当用户要求执行以下操作时使用：
- “添加瓦特”、“设置瓦特”、“集成瓦特”、“配置瓦特”
- “使用 watt 部署”、“容器化我的应用程序”、“部署到 kubernetes”
-“迁移到瓦特”，“将我的应用程序移植到瓦特”
- “创建 watt.json”、“配置平台”
-“wattpm”、“wattpm 创建”、“wattpm 注入”、“wattpm 日志”
- 使用 wattpm CLI 命令，管理正在运行的应用程序
- 使用 Node.js 应用程序服务器
- 在 Node.js 中运行 PHP、WordPress 或 Laravel
支持 Next.js、Express、Fastify、Koa、Remix、Astro、NestJS、PHP、WordPress 和 Laravel。
