為 Node.js 和 PHP 應用程式整合、配置和部署 Platformatic Watt。
當使用者要求執行以下操作時使用：
- “新增瓦特”、“設定瓦特”、“整合瓦特”、“配置瓦特”
- “使用 watt 部署”、“容器化我的應用程式”、“部署到 kubernetes”
-“遷移到瓦特”，“將我的應用程式移植到瓦特”
- “創建 watt.json”、“配置平台”
-“wattpm”、“wattpm 創建”、“wattpm 注入”、“wattpm 日誌”
- 使用 wattpm CLI 命令，管理正在運行的應用程式
- 使用 Node.js 應用程式伺服器
- 在 Node.js 中執行 PHP、WordPress 或 Laravel
支援 Next.js、Express、Fastify、Koa、Remix、Astro、NestJS、PHP、WordPress 和 Laravel。
