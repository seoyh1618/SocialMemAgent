Boîte à outils complète de débogage et de dépannage Kubernetes. Utilisez cette compétence pour diagnostiquer les problèmes de cluster Kubernetes, déboguer les pods défaillants, enquêter sur les problèmes de connectivité réseau, analyser l'utilisation des ressources, dépanner les déploiements ou effectuer des vérifications de l'état du cluster.
