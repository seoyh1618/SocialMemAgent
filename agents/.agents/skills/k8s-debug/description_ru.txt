Комплексный набор инструментов для отладки и устранения неполадок Kubernetes. Используйте этот навык при диагностике проблем кластера Kubernetes, отладке неисправных модулей, исследовании проблем с сетевым подключением, анализе использования ресурсов, устранении неполадок при развертывании или выполнении проверок работоспособности кластера.
