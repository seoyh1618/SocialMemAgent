Habilidad de seguimiento de la opinión pública, resultado final: un informe de opinión pública. Cuando el usuario pregunta "¿Cómo es la opinión pública sobre el evento/tema X?", "¿Cuál es el sentimiento?", "Realizar análisis de sentimiento", "Ejecutar análisis de opinión" o rastrea palabras clave/plataformas y solicita un informe, se utiliza esta habilidad. Acuerdo: cualquier intención relacionada con la opinión pública desencadena el flujo de trabajo completo (rastreo → análisis de sentimiento → generar informe). Esta habilidad permite gatear; el Agente principal realiza el análisis de sentimientos de acuerdo con las indicaciones; el informe es generado por generate_report.
