Навыки мониторинга общественного мнения, конечный результат: отчет об общественном мнении. Когда пользователь спрашивает «Каково общественное мнение по событию/теме X», «Каково мнение», «Провести анализ настроений», «Провести анализ мнения» или просматривает ключевые слова/платформы и запрашивает отчет, используется этот навык. Соглашение: любое намерение, связанное с общественным мнением, запускает весь рабочий процесс (сканирование → анализ настроений → создание отчета). С помощью этого навыка можно ползать; анализ настроений осуществляется основным Агентом по подсказкам; отчет генерируется функциейgenerate_report.
