Compétence de surveillance de l'opinion publique, résultat final : un rapport sur l'opinion publique. Lorsque l'utilisateur demande « Quelle est l'opinion publique sur l'événement/sujet X », « Quel est le sentiment », « Effectuer une analyse des sentiments », « Exécuter une analyse d'opinion » ou explore par mots-clés/plateformes et demande un rapport, cette compétence est utilisée. Accord : toute intention liée à l'opinion publique déclenche le flux de travail complet (exploration → analyse des sentiments → génération d'un rapport). L'exploration se fait grâce à cette compétence ; l'analyse des sentiments est effectuée par l'agent principal selon les invites ; le rapport est généré par generate_report.
