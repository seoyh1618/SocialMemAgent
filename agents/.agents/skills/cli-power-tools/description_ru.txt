Выполняет расширенные операции CLI, преобразование структурированных данных и криминалификацию Unix с помощью современных инструментов на базе Rust. Используйте при поиске в базах кода с помощью ripgrep, преобразовании JSON или YAML с помощью jq, рефакторинге файлов с помощью sed и fd, отладке API с помощью httpie или навигации по каталогам с помощью zoxy.
