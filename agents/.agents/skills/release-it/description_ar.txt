قم ببناء أنظمة جاهزة للإنتاج بأنماط الاستقرار: قواطع الدائرة، والحواجز، والمهلات، ومنطق إعادة المحاولة. يُستخدم عندما يذكر المستخدم "انقطاع الإنتاج"، أو "قاطع الدائرة الكهربائية"، أو "استراتيجية المهلة"، أو "خط أنابيب النشر"، أو "هندسة الفوضى". يغطي تخطيط القدرات، والفحوصات الصحية، وأنماط مكافحة الهشاشة. إذا كنت تريد أنظمة البيانات، انظر أنظمة ddia. إذا كنت تريد بنية النظام، انظر تصميم النظام.
