تحليل المشروع وإنشاء تكوين Docker: Dockerfile (متعدد المراحل dev/prod)، وcompose.yml، وcompose.override.yml (dev)، وcompose.production.yml (تصلب)، و.dockerignore. يشمل تدقيق أمن الإنتاج. يُستخدم عندما يقول المستخدم "dockerize" أو "add docker" أو "docker compose" أو "containerize" أو "setup docker".
