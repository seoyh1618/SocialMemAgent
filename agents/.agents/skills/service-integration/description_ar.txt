قم بالتحميل بشكل استباقي عندما تتضمن المهمة توصيل خدمات خارجية أو واجهات برمجة التطبيقات التابعة لجهات خارجية. يُستخدم عندما يقول المستخدم "أضف إرسال بريد إلكتروني" أو "دمج نظام إدارة المحتوى" أو "إعداد تحميلات الملفات" أو "أضف التحليلات" أو "الاتصال بـ S3". يغطي خدمات البريد الإلكتروني (Resend، SendGrid)، ومنصات CMS (Sanity، Contentful، Payload)، وحلول تحميل الملفات (UploadThing، Cloudinary، S3)، وتكامل التحليلات، ومعالجة خطاف الويب، واسترداد الأخطاء، وإدارة بيانات الاعتماد.
