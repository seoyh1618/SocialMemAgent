Cargue PROACTIVAMENTE cuando la tarea implique conectar servicios externos o API de terceros. Úselo cuando el usuario diga "agregar envío de correo electrónico", "integrar un CMS", "configurar carga de archivos", "agregar análisis" o "conectarse a S3". Cubre servicios de correo electrónico (Resend, SendGrid), plataformas CMS (Sanity, Contentful, Payload), soluciones de carga de archivos (UploadThing, Cloudinary, S3), integración de análisis, manejo de webhooks, recuperación de errores y administración de credenciales.
