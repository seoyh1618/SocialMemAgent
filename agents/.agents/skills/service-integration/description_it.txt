Carica in modo PROATTIVO quando l'attività prevede la connessione di servizi esterni o API di terze parti. Da utilizzare quando l'utente dice "aggiungi invio email", "integra un CMS", "imposta caricamenti di file", "aggiungi analisi" o "connetti a S3". Copre servizi di posta elettronica (Resend, SendGrid), piattaforme CMS (Sanity, Contentful, Payload), soluzioni di caricamento di file (UploadThing, Cloudinary, S3), integrazione di analisi, gestione dei webhook, ripristino degli errori e gestione delle credenziali.
