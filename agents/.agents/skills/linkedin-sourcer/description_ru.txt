Находите и оценивайте кандидатов из LinkedIn, используя библиотеку Python linkedin_scraper. Используйте, когда пользователь хочет (1) очистить профили LinkedIn на предмет данных о кандидатах, (2) оценить кандидатов по описанию должности, (3) сгенерировать логические строки поиска для поиска, (4) создать таблицы показателей, сводки или сравнительные таблицы кандидатов или (5) выполнить любую задачу по подбору/поиску талантов, включающую данные LinkedIn.
