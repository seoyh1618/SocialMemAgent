Busque y evalúe candidatos de LinkedIn utilizando la biblioteca Python linkedin_scraper. Úselo cuando el usuario desee (1) extraer perfiles de LinkedIn en busca de datos de candidatos, (2) evaluar candidatos según una descripción de trabajo, (3) generar cadenas de búsqueda booleanas para abastecimiento, (4) producir cuadros de mando, resúmenes o tablas de comparación de candidatos, o (5) cualquier tarea de reclutamiento/búsqueda de talentos que involucre datos de LinkedIn.
