Кодирует цикл непрерывного совершенствования для агентов, стремящихся к цели: ОЦЕНКА, АНАЛИЗ,
ИССЛЕДОВАНИЕ (гипотеза + доказательства + контраргументы), УЛУЧШЕНИЕ, ПЕРЕОЦЕНКА, РЕШЕНИЕ.
Автоматически фиксирует улучшения (+2 % в чистом виде, отсутствие регресса >5 %) и отменяет ошибки.
Работает со всеми 4 реализациями SDK.
Автоматически активируется при «улучшении агента», «цикле самоусовершенствования», «цикле оценки агента»,
«агенты эталонного тестирования», «запуск цикла улучшений».
