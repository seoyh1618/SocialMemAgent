يشفر حلقة التحسين المستمر للوكلاء الباحثين عن الأهداف: EVAL، ANALYZE،
البحث (الفرضية + الأدلة + الحجج المضادة)، التحسين، إعادة التقييم، اتخاذ القرار.
تنفيذ التحسينات تلقائيًا (+2% صافي، بدون انحدار > 5%) وإرجاع حالات الفشل.
يعمل مع جميع تطبيقات SDK الأربعة.
يتم التنشيط تلقائيًا عند "تحسين الوكيل"، "حلقة التحسين الذاتي"، "حلقة تقييم الوكيل"،
"الوكلاء المعياريون"، "تشغيل دورة التحسين".
