Руководство по разработке проекта FastAPI Best Architecture (fba). Предоставляет полные архитектурные спецификации, стиль кодирования и руководство по разработке плагинов.

Сценарии использования:
- Разработка новых функций или модулей в проекте fba.
- Создание или изменение плагинов (уровень приложения/уровень расширения)
- Написание API, сервисов, CRUD, моделей, схем.
- Миграция базы данных и проектирование моделей.
- Аутентификация JWT и настройка разрешений RBAC.
- Обзор кода и проверка стиля.
