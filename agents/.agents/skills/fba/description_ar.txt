دليل تطوير مشروع FastAPI لأفضل هندسة معمارية (fba). يوفر المواصفات المعمارية الكاملة وأسلوب الترميز وإرشادات تطوير المكونات الإضافية.

سيناريوهات الاستخدام:
- تطوير ميزات أو وحدات جديدة في مشروع fba
- إنشاء أو تعديل المكونات الإضافية (مستوى التطبيق/مستوى الامتداد)
- كتابة واجهات برمجة التطبيقات والخدمات وCRUDs والنماذج والمخططات
- ترحيل قاعدة البيانات وتصميم النماذج
- مصادقة JWT وتكوين إذن RBAC
- مراجعة الكود والتحقق من الأسلوب
