Вычислительный аудит происхождения, проверяющий, что каждое число, таблица, рисунок, порядок и терминология в рукописи получены из кода и сценариев, а не введены вручную. Перекрестные ссылки на исходный код LaTeX с базой кода для обнаружения жестко закодированных значений, устаревших выходных данных, поломок конвейеров и ручного ввода данных. Сопутствующий обзор рукописи: этот навык проверяет документ как прозу; этот навык проверяет, правильно ли сгенерирован документ из кода. Используйте, когда пользователь говорит «проверить происхождение», «проверить воспроизводимость», «проверить мой конвейер», «мои числа взяты из кода», «проверить рукопись по сценариям», «аудит происхождения» или любой запрос на проверку того, что содержимое рукописи восходит к результатам вычислений.
