Audit informatique de provenance vérifiant que chaque numéro, tableau, figure, ordre et terminologie d'un manuscrit est dérivé du code et des scripts - et non saisi manuellement. Croise la source LaTeX avec la base de code pour détecter les valeurs codées en dur, les sorties obsolètes, les pipelines cassés et la saisie manuelle des données. Compagnon de la révision du manuscrit : cette compétence audite le document en prose ; cette compétence vérifie si le document est fidèlement généré à partir du code. À utiliser lorsque l'utilisateur dit « vérifier la provenance », « vérifier la reproductibilité », « auditer mon pipeline », « mes numéros proviennent-ils du code », « vérifier le manuscrit par rapport aux scripts », « audit de provenance » ou toute demande visant à vérifier que le contenu du manuscrit remonte aux sorties informatiques.
