Computergestützte Provenienzprüfung, die überprüft, ob alle Zahlen, Tabellen, Abbildungen, Reihenfolgen und Terminologien in einem Manuskript aus Code und Skripten stammen und nicht manuell eingegeben wurden. Vergleicht die LaTeX-Quelle mit der Codebasis, um hartcodierte Werte, veraltete Ausgaben, defekte Pipelines und manuelle Dateneingabe zu erkennen. Ergänzung zur Manuskriptprüfung: Diese Fähigkeit prüft das Dokument als Prosa; Dieser Skill prüft, ob das Dokument originalgetreu aus Code generiert wurde. Wird verwendet, wenn der Benutzer „Herkunft prüfen“, „Reproduzierbarkeit überprüfen“, „meine Pipeline prüfen“, „sind meine Zahlen aus dem Code“, „Manuskript anhand von Skripten prüfen“, „Herkunftsprüfung“ sagt oder eine Anfrage stellt, um zu überprüfen, ob der Manuskriptinhalt auf Rechenausgaben zurückgeht.
