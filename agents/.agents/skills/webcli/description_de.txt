Surfen Sie im Internet, lesen Sie Seiteninhalte, klicken Sie auf Schaltflächen, füllen Sie Formulare aus, machen Sie Screenshots und erhalten Sie Schnappschüsse zur Barrierefreiheit mit dem webcli Headless-Browser. Wird verwendet, wenn der Benutzer eine Website besuchen, Informationen von einer Webseite sammeln oder mit einer Web-App interagieren möchte.
