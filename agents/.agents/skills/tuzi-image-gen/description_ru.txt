Генерация изображений ИИ с помощью Rabbit API (модель нано-банана), Google, OpenAI, DashScope и Replication. Поддерживает преобразование текста в изображение, эталонные изображения, соотношения сторон и выбор модели. Используйте, когда пользователь просит сгенерировать, создать или нарисовать изображения.
