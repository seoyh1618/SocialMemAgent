Úselo cuando el usuario necesite validar JSON, comparar datos con un esquema, manejar errores de validación, usar validadores de formato personalizados, trabajar con borradores de esquema JSON 04 a 2020-12, configurar esquema z en un proyecto, compilar esquemas con referencias cruzadas, resolver $ref remoto, configurar opciones de validación o inspeccionar detalles de error. Cubre modos de sincronización/asíncrono, manejo seguro de errores, precompilación de esquemas, referencias remotas, tipos de TypeScript y uso de navegador/UMD.
