Valide les données JSON par rapport au schéma JSON à l'aide de la bibliothèque z-schema. À utiliser lorsque l'utilisateur doit valider JSON, vérifier les données par rapport à un schéma, gérer les erreurs de validation, utiliser des validateurs de format personnalisés, travailler avec les versions de schéma JSON 04 à 2020-12, configurer un schéma z dans un projet, compiler des schémas avec des références croisées, résoudre des $ref distants, configurer des options de validation ou inspecter les détails des erreurs. Couvre les modes sync/async, la gestion sécurisée des erreurs, la précompilation de schéma, les références distantes, les types TypeScript et l'utilisation du navigateur/UMD.
