التحقق من صحة بيانات JSON مقابل مخطط JSON باستخدام مكتبة z-schema. يُستخدم عندما يحتاج المستخدم إلى التحقق من صحة JSON، أو التحقق من البيانات مقابل المخطط، أو التعامل مع أخطاء التحقق من الصحة، أو استخدام أدوات التحقق من التنسيق المخصصة، أو العمل مع مسودات مخطط JSON من 04 إلى 2020-12، أو إعداد مخطط z في مشروع، أو تجميع المخططات بإسنادات ترافقية، أو حل $ref البعيد، أو تكوين خيارات التحقق من الصحة، أو فحص تفاصيل الخطأ. يغطي أوضاع المزامنة/غير المتزامنة، والمعالجة الآمنة للأخطاء، والتجميع المسبق للمخطط، والمراجع البعيدة، وأنواع TypeScript، واستخدام المتصفح/UMD.
