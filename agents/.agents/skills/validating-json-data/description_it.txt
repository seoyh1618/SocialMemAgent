Convalida i dati JSON rispetto allo schema JSON utilizzando la libreria z-schema. Da utilizzare quando l'utente deve convalidare JSON, confrontare i dati con uno schema, gestire errori di convalida, utilizzare validatori di formati personalizzati, lavorare con le bozze dello schema JSON da 04 a 2020-12, impostare schema z in un progetto, compilare schemi con riferimenti incrociati, risolvere $ref remoto, configurare opzioni di convalida o controllare i dettagli dell'errore. Copre le modalità di sincronizzazione/asincrona, la gestione sicura degli errori, la precompilazione dello schema, i riferimenti remoti, i tipi TypeScript e l'utilizzo di browser/UMD.
