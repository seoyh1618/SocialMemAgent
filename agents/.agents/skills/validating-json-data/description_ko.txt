z-schema 라이브러리를 사용하여 JSON 스키마에 대해 JSON 데이터의 유효성을 검사합니다. 사용자가 JSON 유효성 검사, 스키마에 대한 데이터 확인, 유효성 검사 오류 처리, 사용자 정의 형식 유효성 검사기 사용, JSON 스키마 초안 04~2020-12 작업, 프로젝트에서 z-스키마 설정, 상호 참조로 스키마 컴파일, 원격 $ref 해결, 유효성 검사 옵션 구성 또는 오류 세부 정보 검사가 필요한 경우에 사용합니다. 동기화/비동기 모드, 안전한 오류 처리, 스키마 사전 컴파일, 원격 참조, TypeScript 유형 및 브라우저/UMD 사용을 다룹니다.
