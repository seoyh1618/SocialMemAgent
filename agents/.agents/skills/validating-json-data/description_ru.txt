Проверяет данные JSON на соответствие схеме JSON с помощью библиотеки z-схемы. Используйте, когда пользователю необходимо проверить JSON, сверить данные со схемой, обработать ошибки проверки, использовать средства проверки пользовательского формата, работать с черновиками схемы JSON с 04 по 2020–12, настроить z-схему в проекте, скомпилировать схемы с перекрестными ссылками, разрешить удаленную ссылку $ref, настроить параметры проверки или просмотреть сведения об ошибке. Охватывает режимы синхронизации/асинхронности, безопасную обработку ошибок, предварительную компиляцию схемы, удаленные ссылки, типы TypeScript и использование браузера/UMD.
