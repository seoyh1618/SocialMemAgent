قم بإنشاء تعريفات الوكيل الفرعي (ملفات Agent.md) للعاملين المستقلين في مجال الذكاء الاصطناعي. يُستخدم عندما يريد المستخدم: إنشاء وكيل، أو إنشاء مصنف تقديرات/مقيم، أو إنشاء مقارنة A/B، أو إنتاج عمال مستقلين، أو إنشاء شيء يعمل بشكل منفصل. يتم تشغيله على: "الوكيل"، و"الوكيل الفرعي"، و"تصنيف المخرجات بشكل مستقل"، و"المقارنة العمياء"، و"تشغيل هذا بالتوازي". لا تستخدم للمهارات (استخدم كاتب المهارات Trae) أو القواعد (استخدم كاتب القواعد Trae).
