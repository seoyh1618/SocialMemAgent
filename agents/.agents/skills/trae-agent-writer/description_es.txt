Cree definiciones de subagente (archivos agente.md) para trabajadores de IA independientes. Úselo cuando el usuario desee: crear un agente, crear un calificador/evaluador, crear un comparador A/B, generar trabajadores independientes o crear algo que se ejecute de forma aislada. Activadores en: '创建 agente', 'subagente', 'calificar resultados de forma independiente', 'comparación ciega', 'ejecutar esto en paralelo'. NO lo use para habilidades (use trae-skill-writer) o reglas (use trae-rules-writer).
