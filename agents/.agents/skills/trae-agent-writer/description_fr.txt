À utiliser lorsque l'utilisateur souhaite : créer un agent, créer un évaluateur/évaluateur, créer un comparateur A/B, générer des travailleurs indépendants ou créer quelque chose qui s'exécute de manière isolée. Se déclenche sur : « 创建 agent », « sous-agent », « noter les sorties indépendamment », « comparaison aveugle », « exécuter ceci en parallèle ». Ne PAS utiliser pour les compétences (utilisez trae-skill-writer) ou les règles (utilisez trae-rules-writer).
