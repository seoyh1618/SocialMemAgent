Создайте определения субагентов (файлы Agent.md) для независимых работников ИИ. Используйте, когда пользователь хочет: создать агент, создать оценщик/оценщик, создать компаратор A/B, создать независимых рабочих процессов или создать что-то, что работает изолированно. Срабатывает по: «агент агента», «субагент», «оценка результатов независимо», «слепое сравнение», «запустить это параллельно». НЕ используйте для навыков (используйте trae-skill-writer) или правил (используйте trae-rules-writer).
