Erstellen Sie Subagentendefinitionen (agent.md-Dateien) für unabhängige KI-Mitarbeiter. Verwenden Sie diese Option, wenn der Benutzer einen Agenten erstellen, einen Bewerter/Bewerter erstellen, einen A/B-Vergleicher erstellen, unabhängige Arbeiter erzeugen oder etwas erstellen möchte, das isoliert ausgeführt wird. Löst aus bei: „Agent-Agent“, „Subagent“, „Ausgaben unabhängig bewerten“, „Blindvergleich“, „Parallel ausführen“. NICHT für Fertigkeiten (verwenden Sie trae-skill-writer) oder Regeln (verwenden Sie trae-rules-writer) verwenden.
