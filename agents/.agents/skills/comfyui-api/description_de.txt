Stellen Sie eine Verbindung zu einer laufenden ComfyUI-Instanz her, stellen Sie Workflows in die Warteschlange, überwachen Sie die Ausführung und rufen Sie Ergebnisse ab. Unterstützt sowohl Online- (REST API) als auch Offline-Modi (JSON-Export). Verwenden Sie diese Option, wenn Sie ComfyUI-Workflows ausführen oder den Serverstatus überprüfen.
