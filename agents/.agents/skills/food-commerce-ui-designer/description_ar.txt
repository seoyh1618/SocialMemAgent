قم بإنشاء واجهات أمامية مميزة على مستوى الإنتاج بجودة تصميم عالية لعلامة تجارية للتجارة الإلكترونية تبيع الصلصات ومرق العظام. استخدم هذه المهارة عند إنشاء أي مكونات ويب أو صفحات أو قوائم منتجات أو أقسام مقصودة أو عناصر واجهة مستخدم للمتجر. يُنشئ تصميمات دافئة وجذابة ومشوقة للطعام مع رسوم متحركة سلسة تعمل بالحركة وجماليات ملونة وودية. استخدم دائمًا هذه المهارة لأي واجهة مستخدم أو صفحة أو مكون أو عمل تخطيط - حتى لو قال المستخدم فقط "إنشاء بطاقة" أو "إنشاء قسم" أو "إضافة رسوم متحركة".
