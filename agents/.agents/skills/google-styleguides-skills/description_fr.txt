Collection complète des guides de style officiels de Google pour 17 langues. Inclut TypeScript, JavaScript, Python, Java, Go, C++, C#, Swift, Objective-C, HTML/CSS, AngularJS, Shell, R, Common Lisp, Vim Script, JSON et Markdown. Normes de codage prêtes pour la production utilisées dans l'ensemble de l'organisation d'ingénierie de Google, formatées pour la consommation des agents d'IA.
