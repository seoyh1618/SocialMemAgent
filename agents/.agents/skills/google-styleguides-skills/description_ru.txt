Полная коллекция официальных руководств по стилю Google для 17 языков. Включает TypeScript, JavaScript, Python, Java, Go, C++, C#, Swift, Objective-C, HTML/CSS, AngularJS, Shell, R, Common Lisp, Vim Script, JSON и Markdown. Готовые к использованию стандарты кодирования, используемые в инженерной организации Google и отформатированные для использования агентами ИИ.
