Этот навык следует использовать, когда пользователь запрашивает «콘텐츠 정리», «아티클 요약», «PDF 학습», «영상 정리», «트윗 정리», «дайджест», «обобщить», «정리해줘» или предоставляет YouTube URL-адрес, URL-адрес X/Twitter (x.com, twitter.com), URL-адрес веб-страницы или файл PDF для анализа. Поддерживает YouTube (расшифровка), X/Twitter (с помощью функции извлечения твитов), веб-страницы (полный контент через браузер) и PDF (текст + изображение на странице). Создает обучение викторине с 9 вопросами на 3 уровнях сложности.
