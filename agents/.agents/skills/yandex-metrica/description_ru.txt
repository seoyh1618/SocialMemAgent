Работа с API Яндекс Метрики: аналитика, отчеты, цели, логи.  
Используйте, когда пользователю нужны данные веб-аналитики из Яндекс Метрики — статистика трафика, отчеты о конверсиях, управление целями, экспорт необработанных журналов, управление счетчиками, сегменты, фильтры или гранты.  
Триггеры: метрика, метрика, аналитика, счётчик, визиты, конверсии, отчёт по трафику, источник трафика, Logs API, экспорт журналов.
