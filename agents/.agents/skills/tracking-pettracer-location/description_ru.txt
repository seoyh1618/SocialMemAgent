Отслеживает GPS-ошейники PetTracer (pettracer.com) через API портала PetTracer: извлекает последнее местоположение домашнего животного, историю местоположений и, при необходимости, передает потоковые обновления через PetTracer WebSocket (SockJS/STOMP). Используйте, когда пользователь спрашивает «где мой питомец/кошка», «отслеживайте мой PetTracer», хочет получить координаты/историю GPS или ему нужна помощь с токенами аутентификации PetTracer или отслеживанием в реальном времени.
