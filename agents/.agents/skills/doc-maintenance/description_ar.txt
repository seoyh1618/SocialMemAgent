التدقيق والصيانة المنهجية للوثائق. يجب استخدام هذه المهارة عندما تكون الوثائق قديمة أو مفقودة أو غير منظمة - بعد العمل على الميزات، أو إعادة البناء، أو ترقيات التبعية، أو كفحص صحي دوري. فهو يصف بنية المجلد للمستندات/ والدليل/، ويرسل وكلاء هايكو الفرعيين لمسح قاعدة التعليمات البرمجية/المستندات، ويوجه إنشاء المستندات إلى وكلاء متخصصين (منشئ المراجع، والكاتب الفني، ودليل التعلم) مع مهندس المستندات كبوابة للجودة.
