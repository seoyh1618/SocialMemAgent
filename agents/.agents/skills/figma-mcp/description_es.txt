Convierta diseños de Figma en código listo para producción utilizando herramientas del servidor MCP. Utilice esta habilidad cuando los usuarios proporcionen URL de Figma, soliciten la conversión de diseño a código, soliciten implementar maquetas de Figma o necesiten extraer tokens de diseño y valores del sistema de los archivos de Figma. Funciona con marcos, componentes y archivos de diseño completos para generar HTML, CSS, React u otro código de interfaz.
