Преобразуйте проекты Figma в готовый к использованию код с помощью серверных инструментов MCP. Используйте этот навык, когда пользователи предоставляют URL-адреса Figma, запрашивают преобразование дизайна в код, просят реализовать макеты Figma или нуждаются в извлечении токенов дизайна и системных значений из файлов Figma. Работает с фреймами, компонентами и целыми файлами дизайна для создания HTML, CSS, React или другого кода внешнего интерфейса.
