قم بتحويل تصميمات Figma إلى تعليمات برمجية جاهزة للإنتاج باستخدام أدوات خادم MCP. استخدم هذه المهارة عندما يقدم المستخدمون عناوين URL لـ Figma، أو يطلبون تحويل التصميم إلى رمز، أو يطلبون تنفيذ نماذج Figma بالحجم الطبيعي، أو يحتاجون إلى استخراج رموز التصميم وقيم النظام من ملفات Figma. يعمل مع الإطارات والمكونات وملفات التصميم بأكملها لإنشاء HTML أو CSS أو React أو أي تعليمات برمجية أخرى للواجهة الأمامية.
