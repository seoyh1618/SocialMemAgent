Converti i progetti Figma in codice pronto per la produzione utilizzando gli strumenti server MCP. Utilizza questa competenza quando gli utenti forniscono URL Figma, richiedono la conversione da progettazione a codice, chiedono di implementare modelli Figma o devono estrarre token di progettazione e valori di sistema dai file Figma. Funziona con frame, componenti e interi file di progettazione per generare codice HTML, CSS, React o altro codice frontend.
