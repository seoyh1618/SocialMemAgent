Convertissez les conceptions Figma en code prêt pour la production à l'aide des outils serveur MCP. Utilisez cette compétence lorsque les utilisateurs fournissent des URL Figma, demandent une conversion de conception en code, demandent à implémenter des maquettes Figma ou doivent extraire des jetons de conception et des valeurs système à partir de fichiers Figma. Fonctionne avec des cadres, des composants et des fichiers de conception entiers pour générer du code HTML, CSS, React ou autre code frontal.
