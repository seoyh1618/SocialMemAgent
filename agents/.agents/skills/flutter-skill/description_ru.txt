Этот навык предоставляет обязательные рекомендации по проектированию пользовательского интерфейса/UX, управлению состоянием, работе в сети и интеграции API, которые необходимо соблюдать при разработке приложений с использованием платформы Flutter. Этот навык не предоставляет дополнительную информацию; он предоставляет важные инструкции по разработке приложений Flutter, которые необходимо строго соблюдать. Всякий раз, когда разработчик делает запросы, связанные с дизайном, пользовательским интерфейсом, UX, эффектами дизайна, управлением состоянием, маршрутизацией, сетью, интеграцией API, дизайном комиксов, запросами, связанными с комиксами, чатом, FCM, push-уведомлениями, сообщениями или уведомлениями, этот навык должен использоваться, и работа должна выполняться точно так, как указано в навыке. В верхней части каждого документа о навыках указан рабочий процесс, которому необходимо всегда следовать. Этот рабочий процесс должен строго соблюдаться.
