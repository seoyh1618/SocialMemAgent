توفر هذه المهارة إرشادات إلزامية لتصميم واجهة المستخدم/تجربة المستخدم وإدارة الحالة والشبكات وتكامل واجهة برمجة التطبيقات التي يجب اتباعها عند تطوير التطبيقات باستخدام إطار عمل Flutter. لا تقدم هذه المهارة معلومات اختيارية؛ فهو يقدم التعليمات الأساسية لتطوير تطبيق Flutter والتي يجب الالتزام بها بدقة. عندما يقوم المطور بتقديم طلبات تتعلق بالتصميم، أو واجهة المستخدم، أو تجربة المستخدم، أو تأثيرات التصميم، أو إدارة الحالة، أو التوجيه، أو الشبكات، أو تكامل واجهة برمجة التطبيقات، أو التصميم الهزلي، أو الطلبات ذات الصلة بالقصص المصورة، أو الدردشة، أو FCM، أو الإشعارات الفورية، أو الرسائل، أو الإشعارات، يجب استخدام هذه المهارة ويجب تنفيذ العمل تمامًا كما تحدد المهارة. يوجد في أعلى كل مستند مهارات سير عمل يجب اتباعه دائمًا. يجب اتباع سير العمل بدقة.
