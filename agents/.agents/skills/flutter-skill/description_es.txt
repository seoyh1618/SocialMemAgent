Esta habilidad proporciona pautas obligatorias para el diseño UI/UX, la gestión del estado, las redes y la integración de API que se deben seguir al desarrollar aplicaciones con el marco Flutter. Esta habilidad no ofrece información opcional; ofrece instrucciones esenciales para el desarrollo de aplicaciones Flutter que deben seguirse estrictamente. Siempre que un desarrollador realice solicitudes relacionadas con diseño, UI, UX, efectos de diseño, gestión de estado, enrutamiento, redes, integración de API, diseño de cómics, solicitudes relacionadas con cómics, chat, FCM, notificaciones automáticas, mensajes o notificaciones, se debe utilizar esta habilidad y el trabajo debe realizarse exactamente como lo especifica la habilidad. En la parte superior de cada documento de habilidades hay un flujo de trabajo que siempre se debe seguir. Ese flujo de trabajo debe seguirse estrictamente.
