Revise las habilidades en cualquier proyecto utilizando un método de doble eje: (1) verificaciones deterministas basadas en código (estructura, scripts, pruebas, seguridad de ejecución) y (2) hallazgos de revisión profunda de LLM. Úselo cuando necesite una puntuación de calidad reproducible para `skills/*/SKILL.md`, desee bloquear fusiones con un umbral de puntuación (por ejemplo, 90+) o necesite elementos de mejora concretos para habilidades de baja puntuación. Funciona en todos los proyectos a través de --project-root.
