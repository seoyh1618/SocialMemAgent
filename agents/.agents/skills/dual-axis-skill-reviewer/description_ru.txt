Проверьте навыки в любом проекте, используя двухосный метод: (1) детерминированные проверки на основе кода (структура, сценарии, тесты, безопасность выполнения) и (2) результаты глубокого анализа LLM. Используйте его, когда вам нужна воспроизводимая оценка качества для `skills/*/SKILL.md`, вы хотите разрешить слияния с пороговым значением оценки (например, 90+) или вам нужны конкретные элементы улучшения для навыков с низкой оценкой. Работает во всех проектах через --project-root.
