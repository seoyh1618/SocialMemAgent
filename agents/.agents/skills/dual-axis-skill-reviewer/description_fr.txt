Examinez les compétences dans n'importe quel projet en utilisant une méthode à deux axes : (1) vérifications déterministes basées sur le code (structure, scripts, tests, sécurité d'exécution) et (2) résultats de l'examen approfondi LLM. À utiliser lorsque vous avez besoin d'un score de qualité reproductible pour `skills/*/SKILL.md`, que vous souhaitez lancer des fusions avec un seuil de score (par exemple 90+) ou que vous avez besoin d'éléments d'amélioration concrets pour des compétences à faible score. Fonctionne sur plusieurs projets via --project-root.
