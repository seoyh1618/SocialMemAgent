Bloc-notes TODO persistant pour le suivi des tâches au fil des sessions. À utiliser lorsque l'utilisateur dit « ajouter à TODO », « ce qu'il y a sur le TODO », « marquer X comme terminé », « afficher la liste TODO », « supprimer de TODO » ou pose des questions sur les tâches en attente. Se déclenche également par battement de cœur pour rappeler les éléments périmés.
