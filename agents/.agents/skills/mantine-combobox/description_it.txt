Crea componenti personalizzati a discesa/selezione/completamento automatico/selezione multipla utilizzando le primitive Combobox di Mantine. Utilizzare questa competenza quando: (1) si crea un nuovo componente di tipo selezione personalizzato con primitive Combobox, (2) si crea un menu a discesa ricercabile, (3) si implementa una variante di input con selezione multipla o tag, (4) si personalizza il rendering delle opzioni, (5) si aggiunge logica di filtro personalizzata o (6) qualsiasi attività che coinvolge useCombobox, Combobox.Target, Combobox.Option o Combobox.Dropdown.
