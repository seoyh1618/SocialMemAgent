Erstellen Sie benutzerdefinierte Dropdown-/Auswahl-/Autovervollständigungs-/Mehrfachauswahl-Komponenten mit den Combobox-Primitiven von Mantine. Verwenden Sie diese Fähigkeit, wenn: (1) eine neue benutzerdefinierte auswahlähnliche Komponente mit Combobox-Primitiven erstellt wird, (2) ein durchsuchbares Dropdown-Menü erstellt wird, (3) eine Mehrfachauswahl- oder Tag-Eingabevariante implementiert wird, (4) das Rendern von Optionen angepasst wird, (5) benutzerdefinierte Filterlogik hinzugefügt wird oder (6) jede Aufgabe, die useCombobox, Combobox.Target, Combobox.Option oder Combobox.Dropdown umfasst.
