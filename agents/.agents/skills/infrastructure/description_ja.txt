この nix flake プロジェクトの NixOS インフラストラクチャを管理します。 Colmena を使用して構成を展開し、Proxmox LXC コンテナーを管理し、サービスのトラブルシューティングを行い、サーバーを保守します。

次の場合に使用します: (1) Colmena を使用した NixOS 構成のデプロイ、(2) Proxmox LXC コンテナの管理 (開始、停止、再起動、ステータス)、(3) SSH または pct exec を介したサーバーの問題のトラブルシューティング、(4) ホスト全体のサービス ステータスの確認、(5) インフラストラクチャのメンテナンス タスク。

アーキテクチャに関する重要な注意事項:
- すべてのサーバーは Proxmox LXC コンテナーです。
