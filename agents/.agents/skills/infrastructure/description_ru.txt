Управляйте инфраструктурой NixOS для этого проекта Nix Flake. Развертывайте конфигурации с помощью Colmena, управляйте контейнерами Proxmox LXC, устраняйте неполадки в службах и обслуживайте серверы.

Используйте, когда: (1) Развертывание конфигураций NixOS с помощью colmena, (2) Управление контейнерами Proxmox LXC (запуск, остановка, перезагрузка, состояние), (3) Устранение неполадок сервера через SSH или pct exec, (4) Проверка состояния службы на хостах, (5) Любая задача обслуживания инфраструктуры.

ВАЖНЫЕ замечания по архитектуре:
- Все серверы представляют собой контейнеры Proxmox LXC.
