Generatore di ponteggi per componenti front-end. Genera automaticamente il codice del componente conforme agli standard di progetto in base alla descrizione del componente, identifica automaticamente lo stack tecnologico (React+MUI o Vue3+AntDesign) e genera il modello corrispondente. Applicabile per: (1) creare nuovi componenti aziendali, (2) creare nuovi componenti comuni, (3) creare nuovi componenti a livello di pagina. Condizione di attivazione: quando l'utente deve creare un nuovo componente.
