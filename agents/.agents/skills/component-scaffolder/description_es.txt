Generador de andamios de componentes frontales. Genera automáticamente código de componente que se ajusta a los estándares del proyecto según las descripciones de los componentes, reconoce automáticamente la pila tecnológica (React+MUI o Vue3+AntDesign) y genera las plantillas correspondientes. Aplicable para: (1) crear nuevos componentes comerciales, (2) crear nuevos componentes comunes, (3) crear nuevos componentes a nivel de página. Condición de activación: cuando el usuario necesita crear un nuevo componente.
