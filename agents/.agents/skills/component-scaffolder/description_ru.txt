Генератор каркасов внешних компонентов. Автоматически генерирует код компонента, соответствующий стандартам проекта, на основе описания компонента, автоматически определяет технологический стек (React+MUI или Vue3+AntDesign) и генерирует соответствующий шаблон. Применимо для: (1) создания новых бизнес-компонентов, (2) создания новых общих компонентов, (3) создания новых компонентов уровня страницы. Условие срабатывания: когда пользователю необходимо создать новый компонент.
