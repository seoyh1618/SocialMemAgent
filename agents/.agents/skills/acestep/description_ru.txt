Используйте ACE-Step API для создания музыки, редактирования песен и создания ремиксов на музыку. Поддерживает преобразование текста в музыку, генерацию текстов, продолжение звука и перерисовку звука. Используйте этот навык, когда пользователи упоминают создание музыки, создание песен, создание музыки, ремикс или продолжение аудио.
