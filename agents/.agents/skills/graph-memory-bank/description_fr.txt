Créez et gérez une banque de mémoire graphique de style Obsidian dans un référentiel de code : de petits nœuds Markdown atomiques avec une interface YAML, des liens croisés, des backlinks explicites et une couverture basée sur la version/l'entité pour une récupération rapide du contexte de l'agent IA. À utiliser lorsqu'on vous demande de créer/mettre à niveau une « banque de mémoire », une « mémoire graphique », des « documents d'obsidienne », un « суперсвязанную графовую документацию », ou lorsque vous avez besoin de documents structurés sous docs/ qui permettent à un agent d'IA d'extraire un contexte minimal mais précis.
