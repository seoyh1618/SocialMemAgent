قم بإنشاء وصيانة بنك ذاكرة الرسم البياني على نمط Obsidian في مستودع التعليمات البرمجية: عقد Markdown الذرية الصغيرة مع المادة الأمامية YAML، والروابط المتقاطعة، والروابط الخلفية الصريحة، والتغطية المستندة إلى الإصدار/الكيان لاسترجاع سياق وكيل الذكاء الاصطناعي بسرعة. استخدمه عندما يُطلب منك إنشاء/ترقية "بنك الذاكرة"، أو "ذاكرة الرسم البياني"، أو "مستندات obsidian"، أو "المستندات المجمعة للجرافيك"، أو عندما تحتاج إلى مستندات منظمة ضمن المستندات/التي تسمح لعامل الذكاء الاصطناعي بسحب سياق بسيط ولكن دقيق.
