Auditer et optimiser l'utilisation des fenêtres contextuelles pour les outils de codage d'IA (Claude Code, OpenCode, etc.). Évalue la répartition des jetons, identifie le gaspillage (compétences en double, règles qui se chevauchent, fichiers d'instructions volumineux, statut git sale, surcharge du serveur MCP) et fournit des recommandations exploitables avec les économies projetées. À utiliser lorsque l'utilisateur dit « vérification du contexte », « réduire le contexte », « vérifier le contexte », « audit du contexte », « quelle est la taille de mon contexte » ou lorsque les sessions semblent lentes.
