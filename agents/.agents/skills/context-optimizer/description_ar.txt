تدقيق وتحسين استخدام نافذة السياق لأدوات ترميز الذكاء الاصطناعي (Claude Code، وOpenCode، وما إلى ذلك). تقدير انهيار الرمز المميز، وتحديد الهدر (المهارات المكررة، والقواعد المتداخلة، وملفات التعليمات المتضخمة، وحالة البوابة القذرة، والحمل الزائد لخادم MCP)، وتقديم توصيات قابلة للتنفيذ مع المدخرات المتوقعة. يُستخدم عندما يقول المستخدم "فحص السياق"، أو "تقليل السياق"، أو "التحقق من السياق"، أو "تدقيق السياق"، أو "ما حجم السياق الخاص بي"، أو عندما تبدو الجلسات بطيئة.
