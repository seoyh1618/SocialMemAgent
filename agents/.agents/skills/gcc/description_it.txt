Git Context Controller (GCC): gestisce la memoria dell'agente come un file system con versione in .GCC/. Questa abilità dovrebbe essere utilizzata quando si lavora su progetti in più fasi che traggono vantaggio dalla persistenza della memoria strutturata, dal monitoraggio delle tappe fondamentali, dalla ramificazione per approcci alternativi e dal ripristino del contesto tra sessioni incrociate. Trigger su comandi /gcc o linguaggio naturale come "impegna questo progresso", "dirama per provare un'alternativa", "unisci risultati", "recupera contesto".
