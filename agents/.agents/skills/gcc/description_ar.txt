Git context Controller (GCC) - يدير ذاكرة الوكيل كنظام ملفات تم إصداره ضمن .GCC/. يجب استخدام هذه المهارة عند العمل على مشاريع متعددة الخطوات تستفيد من ثبات الذاكرة المنظمة، وتتبع المعالم، والتفرع للأساليب البديلة، واستعادة السياق عبر الجلسات. يتم تشغيل أوامر /gcc أو اللغة الطبيعية مثل "تنفيذ هذا التقدم"، أو "الفرع لتجربة بديل"، أو "دمج النتائج"، أو "استعادة السياق".
