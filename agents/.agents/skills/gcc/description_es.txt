Controlador de contexto Git (GCC): administra la memoria del agente como un sistema de archivos versionado en .GCC/. Esta habilidad debe usarse cuando se trabaja en proyectos de varios pasos que se benefician de la persistencia de la memoria estructurada, el seguimiento de hitos, la ramificación para enfoques alternativos y la recuperación del contexto entre sesiones. Activadores en comandos /gcc o lenguaje natural como "confirmar este progreso", "ramificarse para probar una alternativa", "fusionar resultados", "recuperar contexto".
