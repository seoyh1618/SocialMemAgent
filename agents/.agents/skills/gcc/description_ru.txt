Контроллер контекста Git (GCC) — управляет памятью агента как файловой системой с поддержкой версий в .GCC/. Этот навык следует использовать при работе над многоэтапными проектами, которые выигрывают от сохранения структурированной памяти, отслеживания этапов, ветвления для альтернативных подходов и восстановления контекста между сеансами. Запускает команды /gcc или естественный язык, например «зафиксировать этот прогресс», «перейти, чтобы попробовать альтернативу», «объединить результаты», «восстановить контекст».
