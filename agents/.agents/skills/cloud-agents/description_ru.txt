Управляйте агентами Cursor Cloud через API. Запускайте агенты, составляйте список работающих агентов, проверяйте статус, получайте историю разговоров, отправляйте последующие действия, останавливайте или удаляйте агенты и извлекайте изменения ветки агента в локальный репозиторий. Используйте, когда пользователь упоминает облачные агенты, фоновые агенты, запуск задачи в репозитории, проверку статуса агента или получение изменений агента.
