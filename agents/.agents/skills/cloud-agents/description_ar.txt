إدارة وكلاء Cursor Cloud عبر واجهة برمجة التطبيقات. قم بتشغيل الوكلاء، وقائمة الوكلاء قيد التشغيل، والتحقق من الحالة، والحصول على سجل المحادثات، وإرسال المتابعات، وإيقاف الوكلاء أو حذفهم، وسحب تغييرات فرع الوكيل إلى الريبو المحلي. يُستخدم عندما يذكر المستخدم وكلاء السحابة، أو وكلاء الخلفية، أو تشغيل مهمة في الريبو، أو التحقق من حالة الوكيل، أو سحب تغييرات الوكيل.
