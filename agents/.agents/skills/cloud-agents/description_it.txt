Gestisci gli agenti Cursor Cloud tramite l'API. Avvia agenti, elenca gli agenti in esecuzione, controlla lo stato, ottieni la cronologia delle conversazioni, invia follow-up, interrompi o elimina agenti e inserisci le modifiche del ramo dell'agente nel repository locale. Da utilizzare quando l'utente menziona agenti cloud, agenti in background, avvia un'attività su un repository, controlla lo stato dell'agente o apporta modifiche all'agente.
