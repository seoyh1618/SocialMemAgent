Verwalten Sie Cursor Cloud Agents über die API. Starten Sie Agenten, listen Sie laufende Agenten auf, überprüfen Sie den Status, rufen Sie den Konversationsverlauf ab, senden Sie Follow-ups, stoppen oder löschen Sie Agenten und ziehen Sie Agentenzweigänderungen in das lokale Repo. Verwenden Sie diese Option, wenn der Benutzer Cloud-Agenten, Hintergrundagenten, das Starten einer Aufgabe in einem Repository, das Überprüfen des Agentenstatus oder das Abrufen von Agentenänderungen erwähnt.
