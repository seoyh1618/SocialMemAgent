Gérez les agents Cursor Cloud via l'API. Lancez des agents, répertoriez les agents en cours d'exécution, vérifiez l'état, obtenez l'historique des conversations, envoyez des suivis, arrêtez ou supprimez des agents et extrayez les modifications de branche d'agent dans le dépôt local. À utiliser lorsque l'utilisateur mentionne des agents cloud, des agents en arrière-plan, le lancement d'une tâche sur un dépôt, la vérification de l'état de l'agent ou l'extraction de modifications d'agent.
