Ingeniería de firmware ESP32 para proyectos ESP-IDF. Escriba, revise y depure código C/C++ integrado que involucre tareas/colas/temporizadores FreeRTOS, periféricos GPIO/I2C/SPI/UART/ADC/PWM, TWAI/CAN, redes Wi-Fi/BLE, actualizaciones OTA, arranque seguro y cifrado flash, integración de pantalla LVGL, flujos de trabajo de compilación/flash/monitoreo, registro, análisis de fallas, optimización del tamaño de memoria/código, diseño de suspensión/activación de bajo consumo de energía, en el dispositivo Terminales de servicio USB/serie y activación de placa. Úselo cuando se le pide a un agente que implemente funciones de firmware ESP-IDF, revise los cambios integrados para verificar su corrección o condiciones de carrera, investigue fallas de arranque/tiempo de ejecución o pánicos de meditación Guru, interprete registros seriales, solucione problemas de compilación/enlace/flash, optimice el uso de RAM/flash, ajuste el comportamiento de sueño profundo/sueño ligero, refuerce el firmware para producción, agregue una consola de servicio/CLI, integre una pantalla con LVGL o diagnostique problemas de integración de hardware y software en dispositivos de clase ESP32.
