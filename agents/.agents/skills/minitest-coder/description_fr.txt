Écrivez des tests Minitest pour les applications Ruby et Rails. À utiliser lors de la création de fichiers de test, de l'écriture de cas de test ou du test de nouvelles fonctionnalités. Pas pour RSpec — utilisez plutôt rspec-coder. Couvre à la fois les styles traditionnels et spécifiques, les appareils, les simulations et les modèles de test d'intégration Rails.
