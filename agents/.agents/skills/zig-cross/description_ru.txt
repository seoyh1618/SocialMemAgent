Навыки зиг-кросс-компиляции. Используйте при кросс-компиляции программ Zig для различных целей, используя встроенную кросс-компиляцию Zig для встроенных систем, WASM, Windows, ARM или используя zig cc для кросс-компиляции кода C без использования системной кросс-инструментологии. Активируется по запросам о кросс-компиляции Zig, целевых тройках zig, кросс-компиляции zig cc, встроенных целевых объектах Zig или Zig WASM.
