Compétence de compilation croisée Zig. À utiliser lors de la compilation croisée de programmes Zig sur différentes cibles, en utilisant la compilation croisée intégrée de Zig pour les applications embarquées, WASM, Windows, ARM, ou en utilisant zig cc pour compiler de manière croisée du code C sans chaîne d'outils croisée du système. S'active sur les requêtes concernant la compilation croisée Zig, les triples de cibles zig, la compilation croisée zig cc, les cibles intégrées Zig ou Zig WASM.
