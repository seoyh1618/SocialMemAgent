Abilità di compilazione incrociata Zig. Da utilizzare durante la compilazione incrociata di programmi Zig su destinazioni diverse, utilizzando la compilazione incrociata integrata di Zig per embedded, WASM, Windows, ARM o utilizzando zig cc per eseguire la compilazione incrociata di codice C senza una cross-toolchain di sistema. Si attiva su query sulla compilazione incrociata Zig, triple target zig, compilazione incrociata zig cc, target incorporati Zig o Zig WASM.
