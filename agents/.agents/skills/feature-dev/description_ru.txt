Комплексный структурированный рабочий процесс разработки функций, включающий отдельные этапы исследования кодовой базы, проектирования архитектуры, проверки качества и реализации. Используйте при создании новых функций, которые требуют: (1) понимания существующей кодовой базы перед внесением изменений, (2) уточнения неоднозначных требований посредством систематического опроса, (3) проектирования продуманной архитектуры перед реализацией, (4) обеспечения качества посредством многомерной проверки кода. Идеально подходит для функций, охватывающих несколько файлов, требующих архитектурных решений, сложных интеграций или с несколько неясными требованиями.
