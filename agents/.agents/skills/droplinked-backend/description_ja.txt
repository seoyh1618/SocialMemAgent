Droplinked e-commerce バックエンドのコード スタイル、アーキテクチャ パターン、開発ガイド。
NestJS + Prisma (MongoDB) による厳密な階層化アーキテクチャ: コントローラーからサービス ファサード、ユースケース、リポジトリ。

このスキルは次の場合に使用します。
(1) ドロップリンクされたバックエンドでの新しい機能、モジュール、またはエンドポイントの作成
(2) プロジェクト パターンに従うように既存のコードをリファクタリングする
(3) ユースケースとサービスのテストの作成
(4) アーキテクチャ準拠のためのコードのレビュー
(5) データモデルの理解（店舗、商品、注文、カート、販売者、顧客）
(6) モジュール間通信の実装
(7) イベント駆動型の副作用の追加
(8) このコードベースでの Prisma と MongoDB の操作

トリガー: ドロップリンク、ネスト モジュール、ユースケース、サービス ファサード、リポジトリ パターン、カート、注文、ショップ、製品、販売者
