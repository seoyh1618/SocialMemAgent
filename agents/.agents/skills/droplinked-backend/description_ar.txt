نمط التعليمات البرمجية وأنماط الهندسة المعمارية ودليل التطوير للواجهة الخلفية للتجارة الإلكترونية Droplinked.
NestJS + Prisma (MongoDB) مع بنية طبقات صارمة: وحدة تحكم لخدمة الواجهة إلى UseCase إلى المستودع.

استخدم هذه المهارة عندما:
(1) إنشاء ميزات أو وحدات أو نقاط نهاية جديدة في الواجهة الخلفية لـ Droplinked
(2) إعادة هيكلة التعليمات البرمجية الموجودة لتتبع أنماط المشروع
(3) كتابة اختبارات لحالات الاستخدام والخدمات
(4) مراجعة الكود للامتثال المعماري
(5) فهم نموذج البيانات (المحلات التجارية، المنتجات، الطلبات، عربات التسوق، التجار، العملاء)
(6) تنفيذ الاتصالات عبر الوحدات
(7) إضافة آثار جانبية يحركها الحدث
(8) العمل مع Prisma وMongoDB في قاعدة البيانات هذه

المشغلات: Droplinked، Nest Module، Usecase، واجهة الخدمة، نمط المستودع، سلة التسوق، الطلب، المتجر، المنتج، التاجر
