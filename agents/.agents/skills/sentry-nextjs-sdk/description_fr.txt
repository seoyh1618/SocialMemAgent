Configuration complète du SDK Sentry pour Next.js. À utiliser lorsqu'on vous demande d'« ajouter Sentry à Next.js », « d'installer @sentry/nextjs » ou de configurer la surveillance des erreurs, le traçage, la relecture de session, la journalisation, le profilage, la surveillance de l'IA ou les crons pour les applications Next.js. Prend en charge Next.js 13+ avec App Router et Pages Router.
