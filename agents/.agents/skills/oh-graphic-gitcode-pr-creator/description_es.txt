Automatice el flujo de trabajo de creación de solicitudes de extracción de Gitcode para el subsistema gráfico OpenHarmony, incluida la confirmación de código, la creación de problemas y el envío de relaciones públicas con el formato de plantilla adecuado. Úselo cuando el usuario necesite crear un PR para la plataforma Gitcode, especialmente para proyectos gráficos OpenHarmony que requieren plantillas de PR específicas con tablas CodeCheck y etiquetas firmadas por.
