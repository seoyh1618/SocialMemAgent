Automatizza il flusso di lavoro di creazione di richieste pull Gitcode per il sottosistema grafico OpenHarmony, incluso il commit del codice, la creazione di problemi e l'invio di PR con la corretta formattazione del modello. Da utilizzare quando l'utente deve creare un PR per la piattaforma Gitcode, in particolare per progetti grafici OpenHarmony che richiedono modelli PR specifici con tabelle CodeCheck e tag Signed-off-by.
