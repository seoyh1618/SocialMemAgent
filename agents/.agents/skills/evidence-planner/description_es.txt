Genere una "Lista de captura de evidencia" para tareas técnicas, resolución de problemas o misiones de desarrollo para guiar la captura de evidencia precisa.
Se activa cuando el usuario está a punto de iniciar una tarea compleja (por ejemplo, depuración, configuración del entorno) y necesita planificar capturas de pantalla, o cuando el usuario menciona /shotlist, /plan, shot list o lista de verificación de capturas de pantalla.
