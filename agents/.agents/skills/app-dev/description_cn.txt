用于构建、调试、审查和迁移 Freshworks Platform 3.0 市场应用程序的专家级开发技能。使用 Freshworks 应用程序时可用于 (1) 创建新的 Platform 3.0 应用程序（前端、无服务器、混合、OAuth），(2) 调试或修复 Platform 3.0 验证错误，(3) 将 Platform 2.x 应用程序迁移到 3.0，(4) 审核 manifest.json、requests.json 或 oauth_config.json 文件，(5) 实现 Crayons UI 组件，(6) 集成外部 API 或 OAuth 提供程序， (7) 任何涉及 Freshworks Platform 3.0 应用程序开发、FDK CLI 或市场提交的任务。
