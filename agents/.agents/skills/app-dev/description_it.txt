Competenze di sviluppo a livello di esperto per la creazione, il debug, la revisione e la migrazione delle applicazioni del marketplace Freshworks Platform 3.0. Utilizzare quando si lavora con le app Freshworks per (1) creare nuove app della piattaforma 3.0 (frontend, serverless, ibride, OAuth), (2) eseguire il debug o correggere errori di convalida della piattaforma 3.0, (3) migrare le app della piattaforma 2.x alla 3.0, (4) rivedere i file manifest.json, request.json o oauth_config.json, (5) implementare i componenti dell'interfaccia utente Crayons, (6) integrare API esterne o OAuth fornitori, (7) Qualsiasi attività che coinvolga lo sviluppo di app Freshworks Platform 3.0, CLI FDK o invio al marketplace.
