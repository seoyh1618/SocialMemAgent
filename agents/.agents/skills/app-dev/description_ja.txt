Freshworks Platform 3.0 マーケットプレイス アプリケーションを構築、デバッグ、レビュー、移行するためのエキスパート レベルの開発スキル。 (1) 新しいプラットフォーム 3.0 アプリ (フロントエンド、サーバーレス、ハイブリッド、OAuth) の作成、(2) プラットフォーム 3.0 検証エラーのデバッグまたは修正、(3) プラットフォーム 2.x アプリの 3.0 への移行、(4) manifest.json、requests.json、または oauth_config.json ファイルの確認、(5) Crayons UI コンポーネントの実装、(6) の Freshworks アプリを操作する場合に使用します。外部 API または OAuth プロバイダーの統合、(7) Freshworks Platform 3.0 アプリ開発、FDK CLI、またはマーケットプレイスへの提出に関連するタスク。
