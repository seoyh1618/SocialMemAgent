Compétences en développement de niveau expert pour créer, déboguer, réviser et migrer des applications de marché Freshworks Platform 3.0. À utiliser lorsque vous travaillez avec des applications Freshworks pour (1) créer de nouvelles applications Platform 3.0 (frontend, sans serveur, hybride, OAuth), (2) déboguer ou corriger les erreurs de validation de Platform 3.0, (3) migrer des applications Platform 2.x vers 3.0, (4) examiner les fichiers manifest.json, requêtes.json ou oauth_config.json, (5) implémenter des composants de l'interface utilisateur Crayons, (6) intégrer des API externes ou OAuth fournisseurs, (7) Toute tâche impliquant le développement d'applications Freshworks Platform 3.0, FDK CLI ou la soumission sur le marché.
