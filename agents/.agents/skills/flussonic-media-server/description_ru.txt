Эксперт Flussonic Media Server — настройка, API, прямая трансляция, транскодирование, DVR, рестриминг, CDN, протоколы и устранение неполадок. Используйте этот навык всякий раз, когда пользователь упоминает Flussonic, настройку потоковой передачи медиа-сервера, настройку прямой трансляции (источники RTMP/SRT/WebRTC/многоадресную рассылку), перекодирование с аппаратным ускорением (NVENC/QSV), запись и воспроизведение DVR, рестриминг в CDN или социальные сети, доставку HLS/DASH/LL-HLS/WebRTC, защиту DRM, настройку IPTV/OTT, конечные точки Flussonic API, балансировку кластера/нагрузки, мониторинг ввода Retroview и оповещения, информационные панели состояния потоков, интеграция Prometheus/Grafana или любые задачи администрирования сервера потокового видео, связанные с Flussonic. Также срабатывает, когда пользователь спрашивает о сравнении протоколов потоковой передачи, настройках потоковой передачи с низкой задержкой, мониторинге качества потока или настройке производительности медиа-сервера — даже если он не упоминает Flussonic по имени, этот навык имеет глубокие знания, которые могут помочь.
