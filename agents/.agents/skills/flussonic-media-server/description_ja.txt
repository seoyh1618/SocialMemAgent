Flussonic Media Server のエキスパート — 構成、API、ライブ ストリーミング、トランスコーディング、DVR、リストリーミング、CDN、プロトコル、トラブルシューティング。ユーザーが Flussonic、メディア サーバー ストリーミング構成、ライブ インジェスト セットアップ (RTMP/SRT/WebRTC/マルチキャスト ソース)、ハードウェア アクセラレーションによるトランスコーディング (NVENC/QSV)、DVR 録画と再生、CDN またはソーシャル メディアへの再ストリーミング、HLS/DASH/LL-HLS/WebRTC 配信、DRM 保護、IPTV/OTT セットアップ、Flussonic API エンドポイント、クラスター/ロード バランシング、Retroview 入力モニタリングとアラート、ストリームについて言及するたびに、このスキルを使用します。ヘルス ダッシュボード、Prometheus/Grafana 統合、または Flussonic に関連するビデオ ストリーミング サーバー管理タスク。また、ユーザーがストリーミング プロトコルの比較、低レイテンシーのストリーミング設定、ストリーム品質の監視、またはメディア サーバーのパフォーマンスの調整について尋ねたときにもトリガーします。たとえユーザーが Flussonic の名前を挙げていなくても、このスキルには役立つ深い知識があります。
