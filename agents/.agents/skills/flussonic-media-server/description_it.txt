Esperto di Flussonic Media Server: configurazione, API, streaming live, transcodifica, DVR, ristreaming, CDN, protocolli e risoluzione dei problemi. Utilizza questa competenza ogni volta che l'utente menziona Flussonic, configurazione dello streaming del server multimediale, configurazione dell'acquisizione live (sorgenti RTMP/SRT/WebRTC/multicast), transcodifica con accelerazione hardware (NVENC/QSV), registrazione e riproduzione DVR, ristreaming su CDN o social media, consegna HLS/DASH/LL-HLS/WebRTC, protezione DRM, configurazione IPTV/OTT, endpoint API Flussonic, cluster/bilanciamento del carico, monitoraggio e avvisi di input Retroview, integrità del flusso dashboard, integrazione Prometheus/Grafana o qualsiasi attività di amministrazione del server di streaming video che coinvolga Flussonic. Si attiva anche quando l'utente chiede informazioni sul confronto dei protocolli di streaming, sulle configurazioni di streaming a bassa latenza, sul monitoraggio della qualità dello streaming o sull'ottimizzazione delle prestazioni del server multimediale: anche se non menziona Flussonic per nome, questa abilità ha una conoscenza approfondita che può aiutare.
