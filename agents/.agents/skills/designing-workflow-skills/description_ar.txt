يوجه تصميم وهيكلة مهارات Claude Code القائمة على سير العمل من خلال مراحل متعددة الخطوات، وأشجار القرار، وتفويض الوكيل الفرعي، والإفصاح التدريجي. استخدمها عند إنشاء مهارات تتضمن خطوط أنابيب متسلسلة، أو أنماط توجيه، أو بوابات أمان، أو تتبع المهام، أو التنفيذ المرحلي، أو أي سير عمل متعدد الخطوات. ينطبق أيضًا عند مراجعة مهارات سير العمل الحالية أو إعادة هيكلتها من أجل الجودة.
