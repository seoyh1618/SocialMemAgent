Stellen Sie Vercel-Vorschauumgebungen bereit, überprüfen Sie den Status, sehen Sie sich Build-Protokolle an und führen Sie Lighthouse-Audits durch – alles über den Chat. 

Verwenden Sie, wenn:
– Der Benutzer möchte eine Vorschau oder einen Produktions-Build bereitstellen
– Der Benutzer bittet darum, den Status oder die Protokolle einer Bereitstellung zu überprüfen
– Der Benutzer möchte eine Leistungs-/Zugänglichkeitsprüfung für eine Vorschau-URL
– Der Benutzer möchte die letzten Bereitstellungen auflisten
