Проверьте отзывчивость веб-сайта на любой ширине области просмотра с помощью автоматизации браузера. Изменяет размер одного сеанса с помощью точек останова, делает снимки экрана для каждой ширины и обнаруживает переходы макета (изменения столбцов, переключатели навигации, переполнение). Создает сравнительные отчеты, показывающие, где именно нарушены макеты. Запускайте с помощью «проверки реагирования», «проверки реагирования», «теста точки останова», «теста области просмотра», «отзывчивой развертки», «проверки точек останова» или «тестирования на мобильном устройстве».
