Testez la réactivité du site Web sur toutes les largeurs de fenêtre à l’aide de l’automatisation du navigateur. Redimensionne une seule session via des points d'arrêt, capture d'écran chaque largeur et détecte les transitions de mise en page (changements de colonnes, commutateurs de navigation, débordement). Produit des rapports de comparaison montrant exactement où les mises en page se cassent. Déclenchez avec « vérification de réactivité », « vérification réactive », « test de point d'arrêt », « test de fenêtre d'affichage », « balayage réactif », « vérification des points d'arrêt » ou « test sur mobile ».
