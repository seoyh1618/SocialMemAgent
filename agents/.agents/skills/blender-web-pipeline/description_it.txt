Blender esporta flussi di lavoro sul web per modelli 3D e animazioni. Utilizza questa competenza quando esporti modelli Blender in glTF per il web, ottimizzi risorse 3D per Three.js o Babylon.js, modelli di elaborazione batch con script Python, automatizzi flussi di lavoro Blender o crei pipeline 3D pronte per il web. Si attiva su attività che coinvolgono l'esportazione glTF di Blender, lo scripting bpy, l'ottimizzazione delle risorse 3D, la compressione del modello, la cottura delle texture o l'automazione di Blender. Esporta modelli per le competenze threejs-webgl, react-tre-fiber e babylonjs-engine.
