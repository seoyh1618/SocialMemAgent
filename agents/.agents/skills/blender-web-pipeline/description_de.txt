Wird bei Aufgaben ausgelöst, die Blender glTF-Export, BPY-Skripting, 3D-Asset-Optimierung, Modellkomprimierung, Texturbacken oder Blender-Automatisierung umfassen. Exportiert Modelle für die Fähigkeiten Threejs-webgl, React-Three-Fiber und Babylonjs-Engine.
