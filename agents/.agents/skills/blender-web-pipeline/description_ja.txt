Blender から 3D モデルとアニメーションを Web にエクスポートするワークフロー。このスキルは、Blender モデルを Web 用の glTF にエクスポートする場合、Three.js または Babylon.js の 3D アセットを最適化する場合、Python スクリプトを使用してモデルをバッチ処理する場合、Blender ワークフローを自動化する場合、または Web 対応 3D パイプラインを作成する場合に使用します。 Blender glTF エクスポート、bpy スクリプト、3D アセットの最適化、モデル圧縮、テクスチャ ベイク、または Blender オートメーションに関連するタスクをトリガーします。 threejs-webgl、react-three-fiber、および babylonjs-engine スキルのモデルをエクスポートします。
