Herramienta CLI de recuperación de datos de Twitter. Utilícelo cuando el usuario solicite datos de Twitter: perfiles de usuario, recuentos de seguidores, búsquedas de tweets, cronogramas de usuarios, listas de seguidores/seguidos o métricas de usuarios de Twitter. Admite filtrado de campos para resultados estructurados. Para obtener documentación detallada sobre el punto final de API, consulte la habilidad de API original en https://docs.twitterapi.io/skill.md
