CLI-Tool zum Abrufen von Twitter-Daten. Wird verwendet, wenn der Benutzer Twitter-Daten anfordert – Benutzerprofile, Follower-Zählungen, Tweet-Suchen, Benutzer-Timelines, Follower-/Follower-Listen oder Twitter-Benutzermetriken. Unterstützt Feldfilterung für strukturierte Ausgabe. Eine ausführliche API-Endpunktdokumentation finden Sie im Original-API-Skill unter https://docs.twitterapi.io/skill.md
