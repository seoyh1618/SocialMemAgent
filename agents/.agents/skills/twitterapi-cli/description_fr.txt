Outil CLI de récupération de données Twitter. À utiliser lorsque l'utilisateur demande des données Twitter : profils utilisateur, nombre d'abonnés, recherches de tweets, chronologies d'utilisateurs, listes d'abonnés/suivi ou métriques utilisateur Twitter. Prend en charge le filtrage de champs pour une sortie structurée. Pour une documentation détaillée sur les points de terminaison de l'API, consultez la compétence API d'origine sur https://docs.twitterapi.io/skill.md
