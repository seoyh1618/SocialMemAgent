Инструмент CLI для получения данных Twitter. Используйте, когда пользователь запрашивает данные Твиттера — профили пользователей, количество подписчиков, поиск твитов, хронику пользователей, списки подписчиков/подписчиков или показатели пользователей Твиттера. Поддерживает фильтрацию полей для структурированного вывода. Подробную документацию по конечным точкам API см. в исходном описании навыков API по адресу https://docs.twitterapi.io/skill.md.
