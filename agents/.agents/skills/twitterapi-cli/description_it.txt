Strumento CLI per il recupero dei dati di Twitter. Da utilizzare quando l'utente richiede dati di Twitter: profili utente, conteggi di follower, ricerche di tweet, sequenze temporali dell'utente, elenchi di follower/following o metriche utente di Twitter. Supporta il filtraggio dei campi per l'output strutturato. Per la documentazione dettagliata dell'endpoint API, consulta la competenza API originale all'indirizzo https://docs.twitterapi.io/skill.md
