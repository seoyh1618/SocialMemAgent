أداة CLI لاسترجاع بيانات تويتر. يُستخدم عندما يطلب المستخدم بيانات تويتر - الملفات الشخصية للمستخدمين، أو عدد المتابعين، أو عمليات البحث عن التغريدات، أو المخططات الزمنية للمستخدم، أو قوائم المتابعين/المتابعة، أو مقاييس مستخدم تويتر. يدعم الترشيح الميداني للإخراج المنظم. للحصول على وثائق مفصلة لنقطة نهاية واجهة برمجة التطبيقات، راجع مهارة واجهة برمجة التطبيقات الأصلية على https://docs.twitterapi.io/skill.md
