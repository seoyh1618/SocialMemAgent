Interagieren Sie mit der GitHub-API als joelclawgithub[bot]-App. Verwenden Sie es beim Erstellen von PRs, beim Pushen von Commits, beim Verwalten von Problemen, beim Überprüfen des CI-Status, beim Auflisten von Repos oder bei anderen GitHub-API-Vorgängen. Löst aus bei „PR erstellen“, „auf GitHub übertragen“, „Problem öffnen“, „CI prüfen“, „Repos auflisten“, „PR zusammenführen“, „GitHub-API“ oder bei jeder Aufgabe, die programmgesteuerten GitHub-Zugriff erfordert. Alle Aktionen werden als joelclawgithub[bot] angezeigt, nicht als @joelhooks.
