Взаимодействуйте с API GitHub как приложение joelclawgithub[bot]. Используйте при создании PR, отправке коммитов, управлении проблемами, проверке статуса CI, листинге репозиториев или любых операциях API GitHub. Триггеры: «создать PR», «отправить на GitHub», «открыть проблему», «проверить CI», «составить список репозиториев», «объединить PR», «GitHub API» или любую задачу, требующую программного доступа к GitHub. Все действия отображаются как joelclawgithub[bot], а не как @joelhooks.
