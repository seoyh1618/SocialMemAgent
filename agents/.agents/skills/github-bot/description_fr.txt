Interagissez avec l'API GitHub en tant qu'application joelclawgithub[bot]. À utiliser lors de la création de PR, de l'envoi de commits, de la gestion des problèmes, de la vérification de l'état du CI, de la liste des dépôts ou de toute opération de l'API GitHub. Déclencheurs sur « créer un PR », « pousser vers GitHub », « ouvrir un problème », « vérifier CI », « lister les dépôts », « fusionner PR », « API GitHub » ou toute tâche nécessitant un accès programmatique à GitHub. Toutes les actions apparaissent sous la forme joelclawgithub[bot], et non sous la forme @joelhooks.
