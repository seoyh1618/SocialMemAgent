Tendenze, principi e modelli di implementazione del web design moderno per il periodo 2024-2025. Utilizza questa competenza durante la progettazione di siti Web, la creazione di esperienze interattive, l'implementazione di sistemi di progettazione, la garanzia dell'accessibilità o la creazione di interfacce incentrate sulle prestazioni. Attiva attività che coinvolgono tendenze del design moderno, microinterazioni, scrollytelling, minimalismo audace, UX del cursore, glassmorfismo, conformità all'accessibilità, ottimizzazione delle prestazioni o architettura del sistema di progettazione. Fa riferimento a competenze di animazione (GSAP, Framer Motion, React Spring), competenze 3D (Three.js, R3F, Babylon.js) e librerie di componenti per indicazioni sull'implementazione.
