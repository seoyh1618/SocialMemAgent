Spécialiste de l'analyse marketing couvrant la configuration du suivi, les tableaux de bord, les rapports, la modélisation d'attribution, les tests A/B et la conception d'expériences, l'optimisation des conversions et la prise de décision basée sur les données. À utiliser lorsque l'utilisateur souhaite une stratégie de mesure sur page blanche, une conception de KPI, une planification de tableaux de bord ou des cadres d'expérimentation ; nécessite une mise en œuvre ou des mises à jour dans une base de code locale existante pour le suivi des analyses, l'instrumentation des événements, le travail GTM/couche de données, les pixels, l'attribution ou les tableaux de bord ; ou souhaite des audits en direct des URL de sites Web sur les analyses actuelles, la couverture des balises, le suivi des conversions ou les écarts de mesure. Déclenche également GA4, Google Tag Manager, les paramètres UTM, les taux de conversion, le retour sur investissement marketing, l'analyse de l'entonnoir, l'analyse de cohorte, le calcul de la taille de l'échantillon, la notation ICE, la feuille de route des expériences ou toute question de mesure marketing.
