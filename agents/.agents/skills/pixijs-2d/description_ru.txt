Быстрый и легкий механизм 2D-рендеринга для создания интерактивной графики, эффектов частиц и приложений на основе холста с использованием WebGL/WebGPU. Используйте этот навык при создании 2D-игр, систем частиц, интерактивных холстов, спрайт-анимации или наложений пользовательского интерфейса на 3D-сцены. Запускается при выполнении задач, связанных с PixiJS, 2D-рендерингом, листами спрайтов, эффектами частиц, фильтрами или высокопроизводительной графикой на холсте. Альтернатива Canvas2D с ускорением WebGL для рендеринга тысяч спрайтов со скоростью 60 кадров в секунду.
