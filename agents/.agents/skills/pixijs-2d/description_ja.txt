WebGL/WebGPU を使用してインタラクティブなグラフィックス、パーティクル エフェクト、キャンバス ベースのアプリケーションを作成するための、高速かつ軽量の 2D レンダリング エンジン。このスキルは、2D ゲーム、パーティクル システム、インタラクティブ キャンバス、スプライト アニメーション、または 3D シーンの UI オーバーレイを構築するときに使用します。 PixiJS、2D レンダリング、スプライト シート、パーティクル エフェクト、フィルター、または高性能キャンバス グラフィックスに関連するタスクをトリガーします。 WebGL アクセラレーションを備えた Canvas2D の代替として、60 FPS で数千のスプライトをレンダリングします。
