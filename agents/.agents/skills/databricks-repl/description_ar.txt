قم بتنفيذ كود Python على مجموعة Databricks عبر جلسة REPL ذات الحالة. استخدم هذه المهارة عندما يريد المستخدم تشغيل Python على Databricks، أو إجراء تحليل البيانات باستخدام Spark، أو تدريب النماذج على مجموعة، أو الاستعلام عن جداول Unity Catalog، أو استخدام sub_llm() لاستدعاءات LM المتكررة، أو أي مهمة تتطلب سياق تنفيذ Databricks المستمر. استخدم دائمًا --project-dir (على سبيل المثال، example/my-task/) لعزل session.json وrepl_outputs لكل مهمة. يغطي دورة حياة الجلسة (الإنشاء، التنفيذ، الانتظار، الإلغاء، التدمير)، وإدارة ملفات الإخراج، واسترداد الإخلاء.
