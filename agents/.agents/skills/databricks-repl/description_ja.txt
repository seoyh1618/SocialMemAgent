ステートフル REPL セッションを介して Databricks クラスターで Python コードを実行します。このスキルは、ユーザーが Databricks で Python を実行する場合、Spark を使用してデータ分析を実行する場合、クラスター上でモデルをトレーニングする場合、Unity カタログ テーブルにクエリを実行する場合、再帰 LM 呼び出しに sub_llm() を使用する場合、または永続的な Databricks 実行コンテキストを必要とするタスクを実行する場合に使用します。タスクごとに session.json と repl_outputs を分離するには、常に専用の --project-dir (例:examples/my-task/) を使用してください。セッションのライフサイクル (作成、実行、待機、キャンセル、破棄)、出力ファイルの管理、およびエビクションの回復について説明します。
