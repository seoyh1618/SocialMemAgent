AI 개인 메모리 시스템의 파일 스캔 검토 기능. 이번 주에 Obsidian Vault(AI_MEMORY 폴더 제외)에서 수정된 파일을 검사하여 잠재적으로 중요한 이벤트와 결정을 식별합니다. 사용 사례: (1) 사용자가 "파일 검색", "이번 주의 파일 확인" 또는 "파일 변경 사항 검색"이라고 말하는 경우; (2) L1 기록을 보완하기 위한 주간 검토 전; (3) 사용자가 이번 주 Obsidian 활동을 검토하고 싶을 때. 수정된 파일 목록을 보여주고, L1에 어떤 파일을 녹음해야 하는지 사용자에게 물어보고 녹음을 도와주는 스킬입니다.
