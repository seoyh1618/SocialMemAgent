Функция просмотра файлов в системе персональной памяти искусственного интеллекта. Сканирует файлы, измененные на этой неделе, в хранилище Obsidian (за исключением папки AI_MEMORY), чтобы выявить потенциально важные события и решения. Случаи использования: (1) когда пользователь говорит «сканировать файлы», «проверить файлы на этой неделе» или «сканировать изменения в файлах»; (2) перед еженедельным просмотром для дополнения записей L1; (3) когда пользователь хочет просмотреть активность в Obsidian за эту неделю. Этот навык перечисляет измененные файлы, спрашивает пользователя, какие из них следует записать в L1, и помогает их записать.
