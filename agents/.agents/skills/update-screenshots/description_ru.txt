Загрузите базовые снимки экрана из последнего запуска CI и зафиксируйте их. Используйте, когда вас просят обновить, принять или обновить базовые показатели снимков экрана компонента из CI или после того, как действие GitHub для проверки снимков экрана сообщает о различиях. Этот навык следует запускать в качестве субагента.
