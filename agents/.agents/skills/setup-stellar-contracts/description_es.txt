Configure un proyecto de contrato inteligente Stellar/Soroban con OpenZeppelin Contracts para Stellar. Úselo cuando los usuarios necesiten: (1) instalar Stellar CLI y la cadena de herramientas Rust para Soroban, (2) crear un nuevo proyecto de Soroban, (3) agregar dependencias de OpenZeppelin Stellar a Cargo.toml o (4) comprender las convenciones de importación de Soroban y los patrones de contrato para OpenZeppelin.
