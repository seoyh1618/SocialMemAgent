Configura un progetto di contratto intelligente Stellar/Soroban con OpenZeppelin Contracts for Stellar. Da utilizzare quando gli utenti devono: (1) installare Stellar CLI e la toolchain Rust per Soroban, (2) creare un nuovo progetto Soroban, (3) aggiungere dipendenze OpenZeppelin Stellar a Cargo.toml o (4) comprendere le convenzioni di importazione e i modelli contrattuali di Soroban per OpenZeppelin.
