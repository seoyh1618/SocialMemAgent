Mettre en place un projet de contrat intelligent Stellar/Soroban avec OpenZeppelin Contracts pour Stellar. À utiliser lorsque les utilisateurs doivent : (1) installer Stellar CLI et la chaîne d'outils Rust pour Soroban, (2) créer un nouveau projet Soroban, (3) ajouter des dépendances OpenZeppelin Stellar à Cargo.toml, ou (4) comprendre les conventions d'importation Soroban et les modèles de contrat pour OpenZeppelin.
