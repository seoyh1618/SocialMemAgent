قم بإعداد مشروع عقد ذكي Stellar/Soroban باستخدام OpenZeppelin Contracts for Stellar. يُستخدم عندما يحتاج المستخدمون إلى: (1) تثبيت سلسلة أدوات Stellar CLI وRust لـ Soroban، (2) إنشاء مشروع Soroban جديد، (3) إضافة تبعيات OpenZeppelin Stellar إلى Cargo.toml، أو (4) فهم اتفاقيات استيراد Soroban وأنماط العقود لـ OpenZeppelin.
