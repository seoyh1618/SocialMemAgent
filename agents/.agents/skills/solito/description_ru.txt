Этот навык следует использовать при работе над проектами Solito (Expo + Next.js с общей навигацией). Он предоставляет специализированные знания по кросс-платформенной навигации, структуре монорепозитория, шаблонам общих компонентов, обработке кода для конкретной платформы и типичным ошибкам. Используйте этот навык при создании универсальных приложений, реализации навигации, создании общих компонентов пользовательского интерфейса или устранении кроссплатформенных проблем.
