قم بإنشاء شجرة عمل فرعية مميزة كفرع لدليل المشروع الحالي. يتم التشغيل عندما: يطلب المستخدم إنشاء فرع، أو بدء فرع ميزات، أو إعداد شجرة عمل. أمثلة: "إنشاء فرع يسمى إضافة مستخدم للتصدير"، "إعداد شجرة عمل للبحث عن الإصلاح". لا تقم بالتشغيل عندما: يريد المستخدم فقط تبديل الفروع أو عرض المشكلات أو تنفيذ التعليمات البرمجية.
