Compilation croisée avec la compétence GCC pour les cibles embarquées et multi-architectures. À utiliser lors de la configuration de chaînes d'outils cross-gcc, de la configuration de sysroots, de la création pour ARM/AArch64/RISC-V/MIPS à partir d'un hôte x86-64, du dépannage d'erreurs d'architecture incorrecte ou de l'exécution de binaires compilés de manière croisée sous QEMU. S'active sur les requêtes concernant les triplets de compilation croisée, sysroot, pkg-config pour les builds croisés, les chaînes d'outils embarquées ou l'intégration Yocto/Buildroot.
