Кросс-компиляция с использованием GCC для встроенных и мультиархитектурных целей. Используйте при настройке цепочек инструментов между GCC, настройке системных корней, сборке ARM/AArch64/RISC-V/MIPS с хоста x86-64, устранении ошибок неправильной архитектуры или запуске кросс-скомпилированных двоичных файлов под QEMU. Активируется по запросам о триплетах кросс-компиляции, sysroot, pkg-config для кросс-сборок, встроенных цепочках инструментов или интеграции Yocto/Buildroot.
