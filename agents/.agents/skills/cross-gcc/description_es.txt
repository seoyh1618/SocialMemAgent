Compilación cruzada con habilidad GCC para objetivos integrados y de múltiples arquitecturas. Úselo al configurar cadenas de herramientas entre gcc, configurar sysroots, compilar para ARM/AArch64/RISC-V/MIPS desde un host x86-64, solucionar errores de arquitectura incorrecta o ejecutar archivos binarios compilados de forma cruzada en QEMU. Se activa ante consultas sobre tripletes de compilación cruzada, sysroot, pkg-config para compilaciones cruzadas, cadenas de herramientas integradas o integración Yocto/Buildroot.
