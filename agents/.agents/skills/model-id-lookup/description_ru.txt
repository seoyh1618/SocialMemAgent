Запросите у навыка идентификаторы моделей ИИ. Используйте этот навык, когда пользователю необходимо найти или проверить идентификатор модели ИИ. Сначала он поддерживает запросы из локального кэша; если он не найден, он обновляет локальные данные с онлайн-сайта models.dev, а затем выполняет повторный запрос.
