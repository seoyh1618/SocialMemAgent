Проверка сертификата SSL — это навык агента, используемый для проверки срока действия сертификата SSL. Он может проверить, истекает ли срок действия SSL-сертификата для указанного домена, вернуть результаты проверки и использовать mcp-email для отправки подписчикам электронного письма с предупреждением об истечении срока действия.
