Интерфейс командной строки веб-поиска, который использует LLM для поиска, чтения страниц и синтеза ответов с цитатами. Используйте, когда вам нужна актуальная информация из Интернета — новости, документация, недавние события или любая тема, требующая актуальных источников. Вызовите с помощью `nexi --plain "query"` для программного использования.
