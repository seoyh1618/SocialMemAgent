Integración de la plataforma de gestión de desarrollo ágil TAPD. Utilice scripts para llamar a la API TAPD y administrar entidades como requisitos, errores, tareas, iteraciones, casos de prueba y Wiki. Los casos de uso incluyen: (1) consultar/crear/actualizar requisitos, errores, tareas e iteraciones; (2) gestionar casos de prueba y Wiki; (3) gestionar comentarios y registros de trabajo; (4) vincular requisitos a errores; (5) recuperar palabras clave de confirmación del código fuente.
