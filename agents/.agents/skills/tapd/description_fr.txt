Intégration de la plateforme de gestion du développement agile TAPD. Utilisez des scripts pour appeler l'API TAPD et gérer des entités telles que les exigences, les bogues, les tâches, les itérations, les cas de test et le wiki. Les cas d'utilisation incluent : (1) l'interrogation/création/mise à jour des exigences, des bogues, des tâches et des itérations ; (2) gérer les cas de test et le wiki ; (3) gérer les commentaires et les journaux de travail ; (4) relier les exigences aux bogues ; (5) récupérer les mots-clés de validation du code source.
