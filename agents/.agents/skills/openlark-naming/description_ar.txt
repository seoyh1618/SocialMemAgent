إرشادات تسمية OpenLark Rust SDK وتعبير واجهة برمجة التطبيقات العامة (العميل/الخدمة/المورد/الطلب/المنشئ). يُستخدم لإضافة/إعادة هيكلة الأنواع العامة، أو تصميم سلاسل استدعاء تعريفية، أو ضبط صادرات الوحدة النمطية والمقدمة، أو حل مشكلات مثل تسمية DocsService/الارتباك الدلالي وأنماط الاتصال غير المتسقة في مستندات openlark. الكلمات الرئيسية المحفزة: اصطلاحات التسمية، العميل مقابل الخدمة، المورد، إعادة التسمية، سلسلة الاتصال الوصفية، واجهة برمجة التطبيقات العامة
