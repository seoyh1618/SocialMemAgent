Соглашения об именах OpenLark Rust SDK и общедоступных выражениях API (Клиент/Сервис/Ресурс/Запрос/Строитель). Используется для добавления/рефакторинга общедоступных типов, проектирования цепочек метавызовов, настройки экспорта модулей и прелюдии, а также для решения таких проблем, как имя/семантическая путаница DocsService и противоречивые стили вызовов в openlark-docs. Ключевые слова-триггеры: соглашения об именах, клиент против службы, ресурс, переименование, цепочка метавызовов, общедоступный API.
