워크로드 요구 사항, 성능 요구 사항 및 예산 제약 조건을 기반으로 Azure VM 크기, VMSS(VM 확장 집합) 및 구성을 권장합니다. Azure 계정이 필요하지 않습니다. 공개 설명서와 Azure Retail Price API를 사용합니다. 사용 대상: VM 크기 권장, 어떤 VM을 사용해야 하는지, Azure VM 선택, 웹/데이터베이스/ML/배치/HPC용 VM, GPU VM, VM 크기 비교, 가장 저렴한 VM, 워크로드에 가장 적합한 VM, VM 가격 책정, 비용 추정, 버스트 가능/컴퓨팅/메모리/스토리지 최적화 VM, 기밀 컴퓨팅, VM 절충, VM 제품군, VMSS, 확장 집합 권장 사항, VM 자동 크기 조정, 부하 분산 VM, VMSS 대 VM, 확장, 수평 확장, 유연한 오케스트레이션. 사용하지 마십시오: VM 또는 VMSS 배포, 앱 배포(azure-deploy 사용), 기존 VM 조회(azure-resource-lookup 사용), 실행 중인 VM의 비용 최적화(azure-cost-optimization 사용), App Service 또는 AKS와 같은 VM이 아닌 서비스.
