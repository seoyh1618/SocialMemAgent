Consigliare le dimensioni delle VM di Azure, i set di scalabilità di VM (VMSS) e le configurazioni in base ai requisiti del carico di lavoro, alle esigenze di prestazioni e ai vincoli di budget. Nessun account Azure richiesto: utilizza la documentazione pubblica e l'API dei prezzi al dettaglio di Azure. UTILIZZO PER: consigliare le dimensioni della VM, quale VM dovrei utilizzare, scegliere VM di Azure, VM per Web/database/ML/batch/HPC, VM GPU, confrontare le dimensioni della VM, VM più economica, migliore VM per carico di lavoro, prezzi delle VM, stima dei costi, VM ottimizzata con possibilità di burst/calcolo/memoria/archiviazione, elaborazione riservata, compromessi VM, famiglie di VM, VMSS, raccomandazione set di scalabilità, VM con scalabilità automatica, VM con bilanciamento del carico, VMSS vs VM, scalabilità orizzontale, scalabilità orizzontale, orchestrazione flessibile. NON UTILIZZARE PER: distribuzione di macchine virtuali o VMSS, distribuzione di app (usare azure-deploy), ricerca di VM esistenti (usare azure-resource-lookup), ottimizzazione dei costi delle VM in esecuzione (usare azure-cost-optimization), servizi non VM come servizio app o AKS.
