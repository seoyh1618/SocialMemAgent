Recomiende tamaños de máquinas virtuales de Azure, conjuntos de escala de máquinas virtuales (VMSS) y configuraciones en función de los requisitos de carga de trabajo, las necesidades de rendimiento y las restricciones presupuestarias. No se requiere una cuenta de Azure: utiliza documentación pública y la API de precios minoristas de Azure. USO PARA: recomendar tamaño de VM, qué VM debo usar, elegir Azure VM, VM para web/base de datos/ML/batch/HPC, GPU VM, comparar tamaños de VM, VM más barata, mejor VM para carga de trabajo, precios de VM, estimación de costos, VM optimizada para ampliación/cómputo/memoria/almacenamiento, informática confidencial, compensaciones de VM, familias de VM, VMSS, recomendación de conjunto de escalado, VM de escalado automático, VM con equilibrio de carga, VMSS vs VM, escalamiento horizontal, escalamiento horizontal, orquestación flexible. NO LO UTILICE PARA: implementar máquinas virtuales o VMSS, implementar aplicaciones (use azure-deploy), buscar máquinas virtuales existentes (use azure-resource-lookup), optimización de costos de máquinas virtuales en ejecución (use azure-cost-optimization), servicios que no son de VM como App Service o AKS.
