Рекомендуйте размеры виртуальных машин Azure, масштабируемые наборы виртуальных машин (VMSS) и конфигурации на основе требований рабочей нагрузки, требований к производительности и бюджетных ограничений. Учетная запись Azure не требуется — используется общедоступная документация и API розничных цен Azure. ИСПОЛЬЗОВАНИЕ ДЛЯ: рекомендовать размер виртуальной машины, какую виртуальную машину мне следует использовать, выбрать виртуальную машину Azure, виртуальную машину для Интернета/базы данных/ML/пакетной обработки/HPC, виртуальную машину с графическим процессором, сравнить размеры виртуальных машин, самую дешевую виртуальную машину, лучшую виртуальную машину для рабочей нагрузки, цену виртуальной машины, оценку затрат, оптимизированную виртуальную машину с возможностью пакетной загрузки/вычислений/памяти/хранилища, конфиденциальные вычисления, компромиссы между виртуальными машинами, семейства виртуальных машин, VMSS, рекомендации по масштабируемому набору, виртуальные машины с автоматическим масштабированием, виртуальные машины с балансировкой нагрузки, VMSS против VM, масштабирование, горизонтальное масштабирование, гибкая оркестровка. НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: развертывания виртуальных машин или VMSS, развертывания приложений (используйте azure-deploy), поиска существующих виртуальных машин (используйте azure-resource-lookup), оптимизации затрат на работу виртуальных машин (используйте azure-cost-optimization), служб, не связанных с виртуальными машинами, таких как Служба приложений или AKS.
