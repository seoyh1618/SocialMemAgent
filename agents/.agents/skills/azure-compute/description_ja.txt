ワークロード要件、パフォーマンスのニーズ、予算の制約に基づいて、Azure VM サイズ、VM スケール セット (VMSS)、および構成を推奨します。 Azure アカウントは必要ありません。公開ドキュメントと Azure Retail Price API を使用します。用途: 推奨 VM サイズ、使用する VM、Azure VM の選択、Web/データベース/ML/バッチ/HPC 用 VM、GPU VM、VM サイズの比較、最も安い VM、ワークロードに最適な VM、VM 価格、コスト見積もり、バースト可能/コンピューティング/メモリ/ストレージに最適化された VM、機密コンピューティング、VM トレードオフ、VM ファミリー、VMSS、スケール セットの推奨事項、自動スケール VM、ロード バランス VM、VMSS 対 VM、スケール アウト、水平スケーリング、フレキシブルオーケストレーション。次の用途には使用しないでください: VM または VMSS のデプロイ、アプリのデプロイ (azure-deploy を使用)、既存の VM の検索 (azure-resource-lookup を使用)、実行中の VM のコストの最適化 (azure-cost-optimization を使用)、App Service や AKS などの非 VM サービス。
