يوصى بأحجام Azure VM ومجموعات مقياس VM (VMSS) والتكوينات بناءً على متطلبات عبء العمل واحتياجات الأداء وقيود الميزانية. لا يلزم وجود حساب Azure — يستخدم الوثائق العامة وواجهة برمجة التطبيقات لأسعار البيع بالتجزئة في Azure. الاستخدام من أجل: التوصية بحجم VM، أي VM يجب أن أستخدمه، اختر Azure VM، VM للويب/قاعدة البيانات/ML/batch/HPC، GPU VM، قارن أحجام VM، أرخص VM، أفضل VM لأعباء العمل، تسعير VM، تقدير التكلفة، VM محسن قابل للاندفاع/الحساب/الذاكرة/التخزين، الحوسبة السرية، مقايضات VM، عائلات VM، VMSS، توصية مجموعة المقياس، مقياس تلقائي للأجهزة الافتراضية، أجهزة افتراضية متوازنة التحميل، VMSS مقابل VM، والقياس، والقياس الأفقي، والتزامن المرن. لا تستخدم من أجل: نشر VMs أو VMSS، ونشر التطبيقات (استخدم azure-deploy)، والبحث عن الأجهزة الافتراضية الحالية (استخدم azure-resource-lookup)، وتحسين تكلفة تشغيل الأجهزة الافتراضية (استخدم azure-cost-optimization)، والخدمات غير المتعلقة بالأجهزة الافتراضية مثل App Service أو AKS.
