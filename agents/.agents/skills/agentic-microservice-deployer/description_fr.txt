Téléchargez un flux agent comme le microservice FastAPI interne à Coolify, accessible uniquement depuis n8n via Docker interne rouge (sans URL publique). Créez automatiquement le dépôt privé sur GitHub, sous le code, créez l'application sur Coolify et vérifiez que le contrôle de santé est actif. Utilisez toujours cette compétence pour que l'utilisateur souhaite "décharger", "subir la production", "publier l'agent", "faire en sorte que n8n appelle le microservicio", "créer le repo", ou toute variante de conversion de code local en un service corriendo. Également appliqué lorsque l'utilisateur est parce que "vous avez terminé l'agent, maintenant que" ou "comme je l'intègre avec n8n".
