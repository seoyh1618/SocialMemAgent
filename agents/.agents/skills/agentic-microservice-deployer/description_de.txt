Betreiben Sie einen Flujo-Agenten wie den FastAPI-Mikrodienst in Coolify, der nur über n8n über Red Docker intern zugänglich ist (unveröffentlichte URL). Erstellen Sie automatisch das private Repo auf GitHub, geben Sie den Code ein, erstellen Sie die Anwendung auf Coolify und überprüfen Sie, ob der Healthcheck aktiv ist. Es handelt sich hierbei um eine Fähigkeit, die den Benutzer mit „Deplegar“, „Subir an Production“, „Publicator the Agent“, „Hacer que n8n lame al microservice“, „Crear the repo“ oder jede Variation der Konvertierung lokaler Codes in einen Service-Service umfasst. Sie können es auch anwenden, wenn der Benutzer wie folgt vorgeht: „Sie haben den Agenten beendet, jetzt ist es soweit“ oder „wie es mit n8n zusammenhängt“.
