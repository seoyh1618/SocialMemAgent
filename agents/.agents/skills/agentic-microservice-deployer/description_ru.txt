Доступен агентский доступ в виде внутреннего микросервиса FastAPI и Coolify, доступный только для n8n через красный внутренний Docker (без публичного URL). Автоматически создайте личное хранилище на GitHub, введите код, создайте приложение в Coolify и проверьте, что проверка работоспособности активна. У нас есть навык, который позволяет пользователю «отключить», «подать производство», «опубликовать агента», «сделать так, чтобы он не позвонил в микросервис», «создать репозиторий», или выбрать вариант конвертации локального кода в сервисную службу. Также приложение может быть использовано как «я закончил агента, сейчас что» или «как интегрировано с n8n».
