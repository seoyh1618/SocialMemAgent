Automatisiert die Erstellung von Komponententests für Go-Projekte mithilfe des Standardtestpakets mit konsistenten Softwaretestmustern, einschließlich In-Got-Want-, tabellengesteuerten Tests und AAA-Mustern. Verwenden Sie diese Option, wenn Sie Komponententests erstellen, ändern oder überprüfen oder wenn der Benutzer Komponententests, Testabdeckung oder Go-Tests erwähnt.
