التصحيح الآلي لخادم Linux وتحديثات حاوية Docker. يُستخدم عندما يطلب المستخدم تحديث خوادم Linux أو تصحيحها أو ترقيتها، أو تطبيق تحديثات الأمان، أو تحديث حاويات Docker، أو التحقق من تحديثات النظام، أو إدارة صيانة الخادم عبر مضيفين متعددين. يدعم Ubuntu، وDebian، وRHEL، وAlmaLinux، وRocky Linux، وCentOS، وAmazon Linux، وSUSE. يتضمن تكامل PatchMon للكشف التلقائي عن المضيف والتعامل الذكي مع Docker.
