Parches automatizados para servidores Linux y actualizaciones de contenedores Docker. Úselo cuando el usuario solicite actualizar, parchear o actualizar servidores Linux, aplicar actualizaciones de seguridad, actualizar contenedores Docker, buscar actualizaciones del sistema o administrar el mantenimiento del servidor en múltiples hosts. Admite Ubuntu, Debian, RHEL, AlmaLinux, Rocky Linux, CentOS, Amazon Linux y SUSE. Incluye integración de PatchMon para la detección automática de host y manejo inteligente de Docker.
