Patch automatizzate del server Linux e aggiornamenti del contenitore Docker. Da utilizzare quando l'utente richiede di aggiornare, applicare patch o potenziare i server Linux, applicare aggiornamenti di sicurezza, aggiornare i contenitori Docker, verificare la disponibilità di aggiornamenti di sistema o gestire la manutenzione del server su più host. Supporta Ubuntu, Debian, RHEL, AlmaLinux, Rocky Linux, CentOS, Amazon Linux e SUSE. Include l'integrazione PatchMon per il rilevamento automatico dell'host e la gestione intelligente di Docker.
