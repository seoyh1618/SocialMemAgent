Genere una tarjeta de resumen de información médica para un acceso rápido en emergencias. Utilice esta habilidad cuando el usuario esté viajando, preparándose para una visita médica, enfrentando una emergencia o solicita "información de emergencia", "tarjeta médica" o "información de primeros auxilios". Extrae datos clave (alergias, medicamentos, afecciones agudas, implantes) y admite salida multiformato (JSON, texto, código QR) para primeros auxilios o atención médica rápida.
