Genera una scheda riepilogativa di informazioni mediche per un rapido accesso in caso di emergenza. Utilizza questa abilità quando l'utente è in viaggio, si prepara per una visita medica, affronta un'emergenza o chiede "informazioni sull'emergenza", "tessera medica" o "informazioni sul pronto soccorso". Estrae dati chiave (allergie, farmaci, condizioni acute, impianti) e supporta output multiformato (JSON, testo, codice QR) per il pronto soccorso o l'assistenza medica rapida.
