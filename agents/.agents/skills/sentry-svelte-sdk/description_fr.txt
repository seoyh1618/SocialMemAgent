Configuration complète du SDK Sentry pour Svelte et SvelteKit. À utiliser lorsqu'on vous demande « d'ajouter Sentry à Svelte », « d'ajouter Sentry à SvelteKit », « d'installer @sentry/sveltekit » ou de configurer la surveillance des erreurs, le traçage, la relecture de session ou la journalisation pour les applications Svelte ou SvelteKit.
