استعلم عن OpenAlex API من سطر الأوامر باستخدام curl وjq لاكتشاف المنشورات، وتصفيتها، وفرزها، وترقيم الصفحات، والتحقق من توفر PDF. يُستخدم عند البحث عن الأعمال العلمية/المؤلفين/المصادر، أو إنشاء استعلامات OpenAlex أو تصحيح أخطاءها، أو استخراج النتائج، أو تنزيل ملفات PDF المتوفرة باستخدام OPENALEX_API_KEY.
