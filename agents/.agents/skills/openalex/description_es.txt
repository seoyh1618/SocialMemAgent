Consulte la API de OpenAlex desde la línea de comandos con curl y jq para descubrir publicaciones, filtrar, ordenar, paginar y verificar la disponibilidad de PDF. Úselo cuando busque trabajos/autores/fuentes académicos, cree o depure consultas de OpenAlex, extraiga resultados o descargue archivos PDF disponibles usando OPENALEX_API_KEY.
