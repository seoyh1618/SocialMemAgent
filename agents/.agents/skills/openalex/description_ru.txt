Запросите OpenAlex API из командной строки с помощью Curl и JQ для обнаружения, фильтрации, сортировки, нумерации страниц и проверки доступности PDF-файлов. Используйте при поиске научных работ/авторов/источников, построении или отладке запросов OpenAlex, извлечении результатов или загрузке доступных PDF-файлов с помощью OPENALEX_API_KEY.
