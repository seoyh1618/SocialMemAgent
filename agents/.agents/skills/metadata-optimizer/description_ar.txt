تحليل وتحسين البيانات الوصفية لمشاريع Next.js، بما في ذلك العنوان والوصف والرسم البياني المفتوح وبطاقات تويتر. اكتشاف جهاز توجيه التطبيقات أو جهاز توجيه الصفحات تلقائيًا، وتقديم اقتراحات الطول، وتحسين الكلمات الرئيسية، وإرشادات أفضل الممارسات. يدعم تحليل تحسين محركات البحث (SEO) باللغتين الصينية والإنجليزية.
