Genera o sincronizza una "Specifica vivente" (docs/spec.md) dal codice sorgente per eliminare la deriva del codice doc. Da utilizzare durante la creazione, l'aggiornamento o la revisione della documentazione dell'architettura per una directory o un modulo. Si attiva su richieste come "generate spec", "create spec.md", "sincronizza documentazione", "aggiorna documenti di architettura", "/spec.doc" o quando viene richiesto di documentare una directory di codebase.
