Générez ou synchronisez une « spécification vivante » (docs/spec.md) à partir du code source pour éliminer la dérive du code doc. À utiliser lors de la création, de la mise à jour ou de la révision de la documentation d'architecture pour un répertoire ou un module. Se déclenche sur des requêtes telles que « générer une spécification », « créer une spécification.md », « synchroniser la documentation », « mettre à jour les documents d'architecture », « /spec.doc » ou lorsqu'il est demandé de documenter un répertoire de base de code.
