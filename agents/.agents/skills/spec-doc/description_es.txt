Genere o sincronice una "Especificación viva" (docs/spec.md) a partir del código fuente para eliminar la deriva del código de documento. Úselo al crear, actualizar o revisar la documentación de arquitectura de un directorio o módulo. Se activa en solicitudes como "generar especificación", "crear spec.md", "sincronizar documentación", "actualizar documentos de arquitectura", "/spec.doc" o cuando se le solicita documentar un directorio de base de código.
