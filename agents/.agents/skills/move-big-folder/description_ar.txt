انقل المجلدات الكبيرة من محرك الأقراص C إلى محرك أقراص آخر (D/E/F، وما إلى ذلك)، واحتفظ بالمسار الأصلي للوصول إليه عبر روابط رمزية، وقم بتحرير مساحة على محرك الأقراص C. استخدم هذه المهارة عندما يذكر المستخدم أن محرك الأقراص C ممتلئ، أو أن شريط محرك الأقراص C أحمر، أو يقوم بترحيل المجلدات الكبيرة، أو تحرير مساحة محرك الأقراص C، أو ترحيل الارتباط الرمزي.
