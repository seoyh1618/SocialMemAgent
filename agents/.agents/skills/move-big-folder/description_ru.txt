Переместите большие папки с диска C на другой диск (D/E/F и т. д.), сохраните доступ к исходному пути через символические ссылки и освободите место на диске C. Используйте этот навык, когда пользователь упоминает, что диск C заполнен, полоса диска C красная, перенос больших папок, освобождение места на диске C или миграция символических ссылок.
