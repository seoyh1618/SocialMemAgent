Verschieben Sie große Ordner vom Laufwerk C auf ein anderes Laufwerk (D/E/F usw.), behalten Sie den ursprünglichen Pfad über symbolische Links bei und geben Sie Speicherplatz auf dem Laufwerk C frei. Verwenden Sie diesen Skill, wenn der Benutzer erwähnt, dass das Laufwerk C voll ist, die C-Laufwerksleiste rot ist, große Ordner migriert werden, Speicherplatz auf dem Laufwerk C freigegeben wird oder eine Symlink-Migration erfolgt.
