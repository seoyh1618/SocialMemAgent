Sposta cartelle di grandi dimensioni dall'unità C a un'altra unità (D/E/F, ecc.), mantieni il percorso originale accessibile tramite collegamenti simbolici e libera spazio sull'unità C. Utilizza questa abilità quando l'utente menziona che l'unità C è piena, la barra dell'unità C è rossa, si esegue la migrazione di cartelle di grandi dimensioni, si libera spazio sull'unità C o si esegue la migrazione di un collegamento simbolico.
