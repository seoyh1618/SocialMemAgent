Zeichnen Sie Domänenereignisse auf und senden Sie sie an Posteingangsverarbeiter, um Nebenwirkungen, Prüfprotokolle und Aktivitäts-Feeds zu erhalten. Verwenden Sie es, wenn Sie Aktivitätsprotokolle erstellen, externe Dienste synchronisieren oder die Ereigniserstellung von der Verarbeitung entkoppeln. Löst auf Ereignisaufzeichnungen, Audit-Trails, Aktivitäts-Feeds oder Posteingangsmustern aus.
