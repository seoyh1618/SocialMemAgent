Optimieren und strukturieren Sie Benutzeraufforderungen für bessere KI-Reaktionen. Verwenden Sie diese Option, wenn der Benutzer nicht auf Englisch (Chinesisch, Japanisch, Koreanisch usw.) schreibt, die Anfrage vage/unklar ist oder der Benutzer um eine Verbesserung seiner Eingabeaufforderung bittet. Wird ausgelöst bei: „帮我“, „请帮忙“, „お願い“, jeder nicht-englischen komplexen Anfrage. Übersetzt, restrukturiert und zeigt optimierte Eingabeaufforderungen an, bevor Sie fortfahren.
