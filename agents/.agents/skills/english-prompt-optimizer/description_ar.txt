تحسين وإعادة هيكلة مطالبات المستخدم للحصول على استجابات أفضل للذكاء الاصطناعي. يُستخدم عندما يكتب المستخدم بلغة غير الإنجليزية (الصينية واليابانية والكورية وما إلى ذلك)، أو عندما يكون الطلب غامضًا/غير واضح، أو عندما يطلب المستخدم تحسين مطالبته. يتم تشغيله عند: '帮我'، '请帮忙'، 'お願い'، أي طلب معقد بلغة غير الإنجليزية. يترجم ويعيد هيكلة ويظهر المطالبة الأمثل قبل المتابعة.
