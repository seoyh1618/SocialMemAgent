Создайте расширения Sindri V3 для платформы Rust CLI. Используйте при создании расширений V3, понимании структуры Extension.yaml V3, проверке соответствия схеме V3, использовании функций обработки коллизий и контекста проекта, добавлении расширений в матрицу совместимости или обновлении версий программного обеспечения расширений. Охватывает методы установки mise, apt,binary, npm, npm-global, script и гибридную установку.
