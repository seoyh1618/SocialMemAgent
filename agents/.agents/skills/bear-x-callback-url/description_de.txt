Anleitung zum X-Callback-URL-Schema der Bear-App zum Erstellen von Callback-URLs, zum Verstehen von Parametern und zum Verwenden von Bear-Aktionen (Erstellen, Öffnen, Suchen, Tags, Auswahl und verwandte Vorgänge). Verwenden Sie diese Option, wenn ein Benutzer fragt, wie das X-Callback-URL-Schema von Bear verwendet werden soll, beim Erstellen einer Bear-Callback-URL oder beim Erstellen einer Integration, die Bear über die X-Callback-URL steuern muss.
