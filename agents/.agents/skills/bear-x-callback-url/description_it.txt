Guida allo schema x-callback-url dell'app Bear per creare URL di richiamata, comprendere i parametri e utilizzare le azioni Bear (creazione, apertura, ricerca, tag, selezione e operazioni correlate). Da utilizzare quando un utente chiede come utilizzare lo schema x-callback-url di Bear, quando compone un URL di callback Bear o quando crea un'integrazione che deve guidare Bear tramite x-callback-url.
