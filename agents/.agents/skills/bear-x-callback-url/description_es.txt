Guía del esquema x-callback-url de la aplicación Bear para crear URL de devolución de llamada, comprender parámetros y utilizar acciones de Bear (crear, abrir, buscar, etiquetas, selección y operaciones relacionadas). Úselo cuando un usuario pregunta cómo usar el esquema x-callback-url de Bear, al redactar una URL de devolución de llamada de Bear o al crear una integración que necesita impulsar Bear a través de x-callback-url.
