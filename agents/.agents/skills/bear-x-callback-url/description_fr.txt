Conseils sur le schéma x-callback-url de l'application Bear pour créer des URL de rappel, comprendre les paramètres et utiliser les actions Bear (créer, ouvrir, rechercher, balises, sélection et opérations associées). À utiliser lorsqu'un utilisateur demande comment utiliser le schéma x-callback-url de Bear, lors de la composition d'une URL de rappel Bear ou lors de la création d'une intégration qui doit piloter Bear via x-callback-url.
