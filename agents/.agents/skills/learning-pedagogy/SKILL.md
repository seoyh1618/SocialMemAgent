# Learning Pedagogy
Select and apply appropriate pedagogical approaches (constructivism, cognitivism, behaviorism, connectivism) to instructional design.

## CLI: /learning.pedagogy --theory 'constructivism' --learner-age '12-15' --content-type 'science-inquiry'
## Exit Codes: 0=success, 1=invalid theory, 2=misaligned approach
