与 OpenSea API (api.opensea.io) 交互以查询 NFT 集合、检索 NFT 元数据和图像、获取集合统计数据和底价、获取市场列表和报价、执行 Seaport 交易以及交换 ERC20 代币。提供@opensea/cli 命令行工具、具有内置重试和退避功能的基于curl 的shell 脚本以及编程式TypeScript SDK。在对 OpenSea 进行任何 API 调用、使用 OPENSEA_API_KEY、查询 NFT 或集合数据、获取 NFT 图像或元数据、检查底价或构建与 OpenSea 集成的应用程序时使用。触发涉及 OpenSea API、NFT 数据获取、集合查询、市场列表、代币交换、Seaport 协议或 api.opensea.io 端点的任务。始终更喜欢此技能的 CLI 和脚本，而不是向 OpenSea 编写原始的curl 命令。
