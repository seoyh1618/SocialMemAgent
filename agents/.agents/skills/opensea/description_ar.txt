تفاعل مع OpenSea API (api.opensea.io) للاستعلام عن مجموعات NFT، واسترداد البيانات التعريفية والصور لـ NFT، والحصول على إحصائيات المجموعة وأسعار الحد الأدنى، وجلب قوائم وعروض السوق، وتنفيذ عمليات تداول Seaport، ومبادلة رموز ERC20. يوفر أداة سطر الأوامر @opensea/cli، ونصوص shell المستندة إلى حليقة مع إمكانية إعادة المحاولة والتراجع المضمنة، وTypeScript SDK برمجي. يُستخدم عند إجراء أي استدعاءات لواجهة برمجة التطبيقات (API) إلى OpenSea، باستخدام OPENSEA_API_KEY، أو الاستعلام عن بيانات NFT أو جمعها، أو جلب صور NFT أو بيانات التعريف، أو التحقق من أسعار الطوابق، أو إنشاء تطبيقات تتكامل مع OpenSea. يتم تشغيل المهام التي تشير إلى OpenSea API، أو جلب بيانات NFT، أو استعلامات المجموعة، أو قوائم السوق، أو مقايضات الرموز المميزة، أو بروتوكول Seaport، أو نقاط نهاية api.opensea.io. تفضل دائمًا واجهة سطر الأوامر (CLI) والبرامج النصية لهذه المهارة على كتابة أوامر الضفيرة الأولية إلى OpenSea.
