與 OpenSea API (api.opensea.io) 互動以查詢 NFT 集合、檢索 NFT 元數據和圖像、獲取集合統計數據和底價、獲取市場列表和報價、執行 Seaport 交易以及交換 ERC20 代幣。提供@opensea/cli 命令列工具、具有內建重試和退避功能的基於curl 的shell 腳本以及編程式TypeScript SDK。在對 OpenSea 進行任何 API 呼叫、使用 OPENSEA_API_KEY、查詢 NFT 或集合資料、獲取 NFT 映像或元資料、檢查底價或建立與 OpenSea 整合的應用程式時使用。觸發涉及 OpenSea API、NFT 資料獲取、集合查詢、市場列表、代幣交換、Seaport 協定或 api.opensea.io 端點的任務。始終更喜歡此技能的 CLI 和腳本，而不是向 OpenSea 編寫原始的curl 命令。
