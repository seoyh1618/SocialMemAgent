Interagisci con l'API OpenSea (api.opensea.io) per interrogare le raccolte NFT, recuperare metadati e immagini NFT, ottenere statistiche sulle raccolte e prezzi minimi, recuperare elenchi e offerte del mercato, eseguire operazioni Seaport e scambiare token ERC20. Fornisce lo strumento da riga di comando @opensea/cli, script di shell basati su curl con tentativi e backoff integrati e un SDK TypeScript programmatico. Da utilizzare quando si effettuano chiamate API a OpenSea, utilizzando OPENSEA_API_KEY, interrogando NFT o dati di raccolta, recuperando immagini o metadati NFT, controllando i prezzi minimi o creando applicazioni che si integrano con OpenSea. Si attiva su attività che menzionano API OpenSea, recupero di dati NFT, query di raccolta, elenchi di marketplace, scambi di token, protocollo Seaport o endpoint api.opensea.io. Preferisci sempre la CLI e gli script di questa abilità rispetto alla scrittura di comandi curl grezzi su OpenSea.
