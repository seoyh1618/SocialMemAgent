Взаимодействуйте с API OpenSea (api.opensea.io), чтобы запрашивать коллекции NFT, получать метаданные и изображения NFT, получать статистику коллекций и минимальные цены, получать списки и предложения на торговых площадках, совершать сделки в морском порту и обменивать токены ERC20. Предоставляет инструмент командной строки @opensea/cli, сценарии оболочки на основе Curl со встроенными повторами и отсрочками, а также программный SDK TypeScript. Используйте при выполнении любых вызовов API к OpenSea, использовании OPENSEA_API_KEY, запросе NFT или данных сбора, получении изображений или метаданных NFT, проверке минимальных цен или создании приложений, интегрируемых с OpenSea. Запускается по задачам, в которых упоминается API OpenSea, получение данных NFT, запросы сбора, списки торговых площадок, обмен токенами, протокол Seaport или конечные точки api.opensea.io. Всегда отдавайте предпочтение CLI и сценариям этого навыка, а не написанию необработанных команд Curl для OpenSea.
