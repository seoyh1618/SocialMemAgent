OpenSea API (api.opensea.io) と対話して、NFT コレクションのクエリ、NFT メタデータと画像の取得、コレクション統計と最低価格の取得、マーケットプレイスのリストとオファーの取得、シーポート取引の実行、ERC20 トークンの交換を行います。 @opensea/cli コマンドライン ツール、再試行とバックオフが組み込まれたカールベースのシェル スクリプト、およびプログラムによる TypeScript SDK を提供します。 OpenSea への API 呼び出しを行う場合、OPENSEA_API_KEY を使用する場合、NFT またはコレクション データのクエリを実行する場合、NFT 画像またはメタデータを取得する場合、最低価格を確認する場合、または OpenSea と統合するアプリケーションを構築する場合に使用します。 OpenSea API、NFT データの取得、コレクション クエリ、マーケットプレイスのリスト、トークン スワップ、Seaport プロトコル、または api.opensea.io エンドポイントに言及するタスクでトリガーされます。 OpenSea に生のcurlコマンドを書き込むよりも、このスキルのCLIとスクリプトを常に優先してください。
