Interactúe con la API de OpenSea (api.opensea.io) para consultar colecciones de NFT, recuperar metadatos e imágenes de NFT, obtener estadísticas de colecciones y precios mínimos, obtener listados y ofertas del mercado, ejecutar transacciones en puertos marítimos e intercambiar tokens ERC20. Proporciona la herramienta de línea de comandos @opensea/cli, scripts de shell basados ​​en curl con reintento y retroceso integrados y un SDK de TypeScript programático. Úselo al realizar llamadas API a OpenSea, usar OPENSEA_API_KEY, consultar NFT o datos de colección, obtener imágenes o metadatos de NFT, verificar precios mínimos o crear aplicaciones que se integren con OpenSea. Activadores de tareas que mencionan OpenSea API, obtención de datos NFT, consultas de recopilación, listados de mercados, intercambios de tokens, protocolo Seaport o puntos finales api.opensea.io. Prefiera siempre la CLI y los scripts de esta habilidad a escribir comandos curl sin formato en OpenSea.
