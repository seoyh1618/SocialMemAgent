Esta habilidad debe usarse cuando el usuario necesita interactuar directamente con los contratos del protocolo AAVE V3, leer datos en cadena, obtener configuraciones de reserva, obtener tasas APY actuales, simular cambios de posición o ejecutar operaciones de protocolo mediante programación. Proporciona acceso de bajo nivel a contratos de AAVE Pool, proveedor de datos de UI Pool y generación de cotizaciones para operaciones de suministro, préstamo, reembolso y retiro en Ethereum y Arbitrum.
