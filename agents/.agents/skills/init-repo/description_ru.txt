Создает новый репозиторий проекта для разработки агентов на основе описания проблемы. Помогает пользователю пройти через именование, среду выполнения, диспетчер пакетов, разработку, лицензирование и структуру проекта, а затем инициализирует git и файл CLAUDE.md. Используйте, когда пользователь говорит «инициализировать проект», «новый проект», «начать проект», «репозиторий scaffold», «создать репо» или описывает проблему, для которой он хочет создать решение.
