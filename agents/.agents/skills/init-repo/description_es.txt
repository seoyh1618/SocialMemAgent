Crea un nuevo repositorio de proyectos para el desarrollo agente a partir de una descripción del problema. Guía al usuario a través de nombres, tiempo de ejecución, administrador de paquetes, devops, licencias y estructura del proyecto, luego inicializa git y un archivo CLAUDE.md. Úselo cuando el usuario diga "iniciar proyecto", "nuevo proyecto", "iniciar un proyecto", "repositorio andamiado", "crear repositorio" o describe un problema para el que desea crear una solución.
