Élabore un nouveau référentiel de projet pour le développement agent à partir d'une description de problème. Guide l'utilisateur à travers la dénomination, l'exécution, le gestionnaire de packages, le développement, les licences et la structure du projet, puis initialise git et un fichier CLAUDE.md. À utiliser lorsque l'utilisateur dit « initialiser le projet », « nouveau projet », « démarrer un projet », « dépôt d'échafaudage », « créer un dépôt » ou décrit un problème pour lequel il souhaite créer une solution.
