Supporta un nuovo repository di progetti per lo sviluppo di agenti a partire dalla descrizione del problema. Guida l'utente attraverso la denominazione, il runtime, il gestore pacchetti, devops, la licenza e la struttura del progetto, quindi inizializza git e un file CLAUDE.md. Da utilizzare quando l'utente dice "progetto init", "nuovo progetto", "avvia un progetto", "repo per impalcature", "crea repository" o descrive un problema per il quale desidera creare una soluzione.
