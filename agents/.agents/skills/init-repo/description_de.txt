Erstellt aus einer Problembeschreibung ein Gerüst für ein neues Projekt-Repository für die Agentenentwicklung. Führt den Benutzer durch Benennung, Laufzeit, Paketmanager, Entwickler, Lizenzierung und Projektstruktur und initialisiert dann Git und eine CLAUDE.md-Datei. Wird verwendet, wenn der Benutzer „Projekt initieren“, „Neues Projekt“, „Projekt starten“, „Gerüst-Repo“, „Repo erstellen“ sagt oder ein Problem beschreibt, für das er eine Lösung erstellen möchte.
