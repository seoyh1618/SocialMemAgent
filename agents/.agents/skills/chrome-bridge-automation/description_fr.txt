Automatisation du navigateur basée sur la vision en utilisant le mode Midscene Bridge. Fonctionne entièrement à partir de captures d’écran – aucune étiquette DOM ou d’accessibilité n’est requise. Peut interagir avec tous les éléments visibles à l’écran, quelle que soit la pile technologique.

Ce mode se connecte au navigateur Chrome de bureau de l'utilisateur via l'extension Chrome Midscene, préservant les cookies, les sessions et l'état de connexion.

Utilisez cette compétence lorsque l'utilisateur souhaite :
- Parcourez, naviguez ou ouvrez des pages Web dans le navigateur Chrome de l'utilisateur
- Interagissez avec les pages qui nécessitent des sessions de connexion, des cookies ou l'état du navigateur existant
- Grattez, extrayez ou collectez des données de sites Web à l'aide du véritable navigateur de l'utilisateur
- Remplissez des formulaires, cliquez sur des boutons ou interagissez avec des éléments Web
- Vérifier, valider ou tester le comportement de l'interface utilisateur frontend
- Prendre des captures d'écran de pages Web
- Automatisez les flux de travail Web en plusieurs étapes
- Vérifier le contenu ou l'apparence du site Web

Propulsé par Midscene.js (https://midscenejs.com)
