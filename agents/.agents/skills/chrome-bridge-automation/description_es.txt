Automatización del navegador basada en visión utilizando el modo Midscene Bridge. Funciona completamente a partir de capturas de pantalla: no se requieren DOM ni etiquetas de accesibilidad. Puede interactuar con todos los elementos visibles en la pantalla independientemente de la pila de tecnología.

Este modo se conecta al navegador Chrome de escritorio del usuario a través de la extensión Midscene Chrome, preservando las cookies, las sesiones y el estado de inicio de sesión.

Utilice esta habilidad cuando el usuario quiera:
- Explorar, navegar o abrir páginas web en el propio navegador Chrome del usuario
- Interactuar con páginas que requieren sesiones de inicio de sesión, cookies o estado del navegador existente
- Extraer, extraer o recopilar datos de sitios web utilizando el navegador real del usuario
- Complete formularios, haga clic en botones o interactúe con elementos web
- Verificar, validar o probar el comportamiento de la interfaz de usuario
- Tomar capturas de pantalla de páginas web.
- Automatizar flujos de trabajo web de varios pasos
- Verificar el contenido o la apariencia del sitio web
