使用 Midscene Bridge 模式的视觉驱动浏览器自动化。完全通过屏幕截图进行操作——不需要 DOM 或辅助功能标签。无论技术堆栈如何，都可以与屏幕上的所有可见元素进行交互。

此模式通过 Midscene Chrome 扩展连接到用户的桌面 Chrome 浏览器，保留 cookie、会话和登录状态。

当用户想要执行以下操作时使用此技能：
- 在用户自己的 Chrome 浏览器中浏览、导航或打开网页
- 与需要登录会话、cookie 或现有浏览器状态的页面进行交互
- 使用用户的真实浏览器从网站抓取、提取或收集数据
- 填写表格、单击按钮或与网络元素交互
- 验证、验证或测试前端 UI 行为
- 截取网页截图
- 自动化多步骤网络工作流程
- 检查网站内容或外观

由 Midscene.js (https://midscenejs.com) 提供支持
