Midscene Bridge モードを使用したビジョン主導のブラウザ自動化。完全にスクリーンショットから操作できます。DOM やアクセシビリティ ラベルは必要ありません。テクノロジースタックに関係なく、画面上に表示されるすべての要素を操作できます。

このモードは、Midscene Chrome 拡張機能を介してユーザーのデスクトップ Chrome ブラウザに接続し、Cookie、セッション、ログイン状態を保持します。

ユーザーが次のことを行う場合にこのスキルを使用します。
- ユーザー自身の Chrome ブラウザで Web ページを閲覧、移動、または開く
- ログイン セッション、Cookie、または既存のブラウザの状態を必要とするページと対話する
- ユーザーの実際のブラウザを使用して Web サイトからデータをスクレイピング、抽出、または収集する
- フォームに記入する、ボタンをクリックする、または Web 要素を操作する
- フロントエンド UI の動作を検証、検証、またはテストします。
- Webページのスクリーンショットを撮る
- 複数ステップの Web ワークフローを自動化する
- ウェブサイトのコンテンツや外観をチェックする
