使用 Midscene Bridge 模式的視覺驅動瀏覽器自動化。完全透過螢幕截圖進行操作—不需要 DOM 或輔助功能標籤。無論技術堆疊如何，都可以與螢幕上的所有可見元素進行互動。

此模式透過 Midscene Chrome 擴充功能連接到使用者的桌面 Chrome 瀏覽器，保留 cookie、會話和登入狀態。

當使用者想要執行以下操作時使用此技能：
- 在使用者自己的 Chrome 瀏覽器中瀏覽、導航或開啟網頁
- 與需要登入會話、cookie 或現有瀏覽器狀態的頁面進行交互
- 使用使用者的真實瀏覽器從網站抓取、提取或收集數據
- 填寫表格、點擊按鈕或與網路元素交互
- 驗證、驗證或測試前端 UI 行為
- 截取網頁截圖
- 自動化多步驟網路工作流程
- 檢查網站內容或外觀

由 Midscene.js (https://midscenejs.com) 提供支持
