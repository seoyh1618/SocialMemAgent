SolidJS 框架開發技能，用於建立具有細粒度反應性的反應式 Web 應用程式。
在處理 SolidJS 專案時使用，包括：(1) 建立具有訊號、儲存和效果的元件，
(2) 實作反應式狀態管理，(3) 使用控制流程元件（Show、For、Switch/Match），
(4) 使用 Solid Router 設定路由，(5) 使用 SolidStart 建立全端應用程序，
(6) 使用 createResource 取得數據，(7) 用於共享狀態的上下文 API，(8) SSR/SSG 設定。
觸發器：solid、solidjs、solid-js、solid start、solidstart、createSignal、createStore、createEffect。
