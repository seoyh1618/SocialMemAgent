Выполняйте фактические изменения кода через Cursor CLI. После получения результатов анализа/планирования передайте их курсор-агенту для делегирования редактирования кода. Используйте это, когда необходимы фактические изменения файлов, например модификация кода, рефакторинг, добавление функций или исправление ошибок.
