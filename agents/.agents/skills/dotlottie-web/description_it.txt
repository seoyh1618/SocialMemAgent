Implementa le animazioni Lottie utilizzando i runtime dotLottie (@lottiefiles/dotlottie-web e @lottiefiles/dotlottie-react). Da utilizzare durante la creazione, il debug o l'ottimizzazione delle animazioni dotLottie o Lottie in progetti web, inclusi Vanilla JS, React e Next.js. Riguarda la selezione dei pacchetti, i Web Worker, le macchine a stati, i temi, l'override degli slot dinamici, le migliori pratiche relative alle prestazioni e i modelli comuni.
