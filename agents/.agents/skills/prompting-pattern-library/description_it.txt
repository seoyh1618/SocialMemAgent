Libreria completa di modelli di prompt, framework ed esempi comprovati per diversi casi d'uso. Questa abilità dovrebbe essere utilizzata durante la creazione di guide ai prompt, l'analisi dell'efficacia dei prompt, l'insegnamento delle tecniche di prompt o la risoluzione dei problemi relativi ai prompt. Da utilizzare quando si scrive sui prompt, si spiegano i concetti dei prompt ad altri o si migliorano i prompt esistenti.
