Обширная библиотека проверенных шаблонов подсказок, схем и примеров для различных вариантов использования. Этот навык следует использовать при создании руководств по подсказкам, анализе эффективности подсказок, обучении методам подсказок или устранении неполадок, связанных с подсказками. Используйте при написании подсказок, объяснении концепций подсказок другим или улучшении существующих подсказок.
