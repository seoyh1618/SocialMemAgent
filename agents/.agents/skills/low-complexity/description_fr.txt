Appliquez une faible complexité cognitive (SonarSource) et une faible complexité cyclomatique dans TOUS les codes écrits ou modifiés, dans n'importe quel langage, framework ou plate-forme de programmation. Cette compétence DOIT s'activer automatiquement chaque fois que du code est écrit, généré, modifié ou refactorisé — aucun déclencheur explicite n'est nécessaire. Les déclencheurs incluent l'écriture de n'importe quelle fonction, méthode, classe, module, script, gestionnaire, point de terminaison, test ou bloc de code. Se déclenche également sur « faible complexité », « complexité cognitive », « complexité cyclomatique », « réduire la complexité », « simplifier le code », « trop complexe », « refactoriser pour la lisibilité », « nettoyer le code », « implémenter », « corriger un bug », « ajouter une fonctionnalité », « générer un test », « optimiser », « réécrire », « échafauder ».
