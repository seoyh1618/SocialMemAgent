فرض التعقيد المعرفي المنخفض (SonarSource) والتعقيد الدوري المنخفض في جميع التعليمات البرمجية المكتوبة أو المعدلة، في أي لغة برمجة أو إطار عمل أو نظام أساسي. يجب أن يتم تنشيط هذه المهارة تلقائيًا عند كتابة التعليمات البرمجية أو إنشاؤها أو تعديلها أو إعادة هيكلتها - لا حاجة إلى مشغل واضح. تتضمن المشغلات كتابة أي وظيفة أو طريقة أو فئة أو وحدة نمطية أو برنامج نصي أو معالج أو نقطة نهاية أو اختبار أو كتلة تعليمات برمجية. يتم تشغيله أيضًا على "التعقيد المنخفض"، "التعقيد المعرفي"، "التعقيد الدوري"، "تقليل التعقيد"، "تبسيط التعليمات البرمجية"، "معقد للغاية"، "إعادة البناء لسهولة القراءة"، "التعليمات البرمجية النظيفة"، "التنفيذ"، "إصلاح الخلل"، "إضافة ميزة"، "إنشاء اختبار"، "التحسين"، "إعادة الكتابة"، "السقالة".
