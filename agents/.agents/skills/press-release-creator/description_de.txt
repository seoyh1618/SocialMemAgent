Eine Fähigkeit zum Erstellen von Pressemitteilungen und FAQs basierend auf der Working Backwards-Methode von Amazon. Wird während der Planungsphase neuer Dienste oder Funktionen verwendet, um den Wert aus der Sicht des Kunden zu artikulieren und die Richtung für die Entwicklung festzulegen. Ausgelöst durch Anfragen wie „Erstellen Sie eine Pressemitteilung“, „Schreiben Sie eine PR/FAQ“, „Planen Sie mit Working Backwards“ oder „Entwerfen Sie Dienstleistungen kundenorientiert“. Ideal für die Planung und Definition von Anforderungen an IT-Services, Produkte und neue Funktionen.
