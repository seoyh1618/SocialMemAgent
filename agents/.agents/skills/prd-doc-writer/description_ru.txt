Напишите и итеративно уточняйте документы PRD/требований с сюжетно-ориентированной структурой и строгими поэтапными подтверждениями (выравнивание карты пути, одноточечное подтверждение для каждой истории, ворота окончательного поколения). Используйте, когда пользователь просит разобраться/написать/улучшить PRD, документы с требованиями, пользовательские истории, критерии приемки и хочет использовать каркасы ASCII и Mermaid (блок-схема/диаграмма состояний/диаграмма последовательности) для уменьшения двусмысленности и совместного создания документа.
