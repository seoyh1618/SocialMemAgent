Automatisez les boucles de correction de révision de code avec la CLI du codex. Demande des examens au codex, classe les résultats par gravité (P0-P4), corrige les problèmes critiques (P0/P1) à travers des cycles itératifs, reporte les améliorations de la qualité au retard et escalade après 3 cycles d'examen. À utiliser lorsque vous travaillez avec du code nécessitant une correction structurée : la « révision du codex » dans une demande déclenche ce flux de travail.
