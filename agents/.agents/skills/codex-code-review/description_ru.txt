Автоматизируйте циклы исправлений проверки кода с помощью интерфейса командной строки кодекса. Запрашивает проверки из кодекса, классифицирует результаты по серьезности (P0–P4), исправляет критические проблемы (P0/P1) посредством итеративных циклов, откладывает улучшения качества в очередь и эскалирует после трех циклов проверки. Используйте при работе с кодом, требующим структурированного исправления: «проверка кодекса» в запросе запускает этот рабочий процесс.
