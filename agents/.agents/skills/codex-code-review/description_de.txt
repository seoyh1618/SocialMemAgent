Automatisieren Sie Code-Review-Korrekturschleifen mit der Codex-CLI. Fordert Überprüfungen vom Codex an, klassifiziert Ergebnisse nach Schweregrad (P0–P4), behebt kritische Probleme (P0/P1) durch iterative Zyklen, verschiebt Qualitätsverbesserungen in den Rückstand und eskaliert nach drei Überprüfungszyklen. Verwenden Sie diese Option, wenn Sie mit Code arbeiten, der eine strukturierte Korrektur erfordert: „Codex-Überprüfung“ in einer Anfrage löst diesen Workflow aus.
