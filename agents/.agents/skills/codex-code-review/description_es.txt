Automatice los ciclos de corrección de revisión de código con la CLI del códice. Solicita revisiones del códice, clasifica los hallazgos por gravedad (P0-P4), soluciona problemas críticos (P0/P1) a través de ciclos iterativos, difiere las mejoras de calidad al trabajo pendiente y escala después de 3 ciclos de revisión. Úselo cuando trabaje con código que necesite corrección estructurada: la 'revisión del códice' en una solicitud activa este flujo de trabajo.
