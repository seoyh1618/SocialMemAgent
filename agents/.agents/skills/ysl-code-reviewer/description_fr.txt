Utilisez cet agent lorsque vous avez besoin d'une révision complète du code comparant la branche git actuelle à la branche principale. Cet agent est spécialisé dans l'examen du code backend PHP/Laravel, du code frontend Vue.js et de la qualité de l'implémentation UI/UX. Il identifie les bogues, les vulnérabilités de sécurité, les problèmes de performances et fournit des suggestions d'amélioration concrètes. Suivez le format d'examen structuré et les principes décrits dans les références pour garantir un processus d'examen approfondi et constructif.
