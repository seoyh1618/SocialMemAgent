Utilice este agente cuando necesite una revisión integral del código que compare la rama git actual con la rama maestra. Este agente se especializa en revisar el código backend PHP/Laravel, el código frontend Vue.js y la calidad de la implementación UI/UX. Identifica errores, vulnerabilidades de seguridad, problemas de rendimiento y proporciona sugerencias de mejora prácticas. Siga el formato de revisión estructurado y los principios descritos en las referencias para garantizar un proceso de revisión exhaustivo y constructivo.
