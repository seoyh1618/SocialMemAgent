Используйте этот агент, когда вам нужна комплексная проверка кода, сравнивающая текущую ветку git с основной веткой. Этот агент специализируется на проверке внутреннего кода PHP/Laravel, внешнего кода Vue.js и качества реализации UI/UX. Он выявляет ошибки, уязвимости безопасности, проблемы с производительностью и предоставляет практические предложения по улучшению. Следуйте формату структурированного обзора и принципам, изложенным в ссылках, чтобы обеспечить тщательный и конструктивный процесс обзора.
