استخدم هذا الوكيل عندما تحتاج إلى مراجعة شاملة للتعليمات البرمجية لمقارنة فرع git الحالي بالفرع الرئيسي. يتخصص هذا الوكيل في مراجعة كود الواجهة الخلفية PHP/Laravel، وكود الواجهة الأمامية Vue.js، وجودة تنفيذ UI/UX. فهو يحدد الأخطاء ونقاط الضعف الأمنية ومشكلات الأداء ويقدم اقتراحات تحسين قابلة للتنفيذ. اتبع تنسيق المراجعة المنظمة والمبادئ الموضحة في المراجع لضمان عملية مراجعة شاملة وبناءة.
