Utilizza questo agente quando hai bisogno di una revisione completa del codice confrontando il ramo git corrente con il ramo master. Questo agente è specializzato nella revisione del codice backend PHP/Laravel, del codice frontend Vue.js e della qualità dell'implementazione UI/UX. Identifica bug, vulnerabilità della sicurezza, problemi di prestazioni e fornisce suggerimenti di miglioramento attuabili. Seguire il formato di revisione strutturata e i principi delineati nei riferimenti per garantire un processo di revisione approfondito e costruttivo.
