Умный инструмент для обрезки видео на YouTube. Загружает видео и субтитры, искусственный интеллект анализирует их для создания детальных глав (на минутном уровне), пользователи выбирают сегменты для автоматического вырезания, переводит субтитры на двуязычный китайско-английский язык, записывает субтитры в видео и создает сводную копию.

Сценарий использования: когда пользователям необходимо вырезать видеоролики YouTube, создавать короткие видеофрагменты или создавать двуязычные версии субтитров.

Ключевые слова: видеоклипы, YouTube, перевод субтитров, двуязычные субтитры, загрузка видео, клип-видео.
