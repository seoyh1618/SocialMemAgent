Évaluation automatique des risques avant chaque action critique dans les flux de travail agent. Détecte les opérations irréversibles (suppression de fichiers, écritures de base de données, déploiements, paiements), classe le niveau de risque et exige une confirmation avant de continuer. Se déclenche sur des mots-clés destructeurs tels que déployer, supprimer, envoyer, publier, mettre à jour la base de données, traiter le paiement.
