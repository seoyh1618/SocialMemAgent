Memberstack Admin API を使用して、メンバー、プラン、データ テーブルを管理します。ユーザーが Memberstack、API によるメンバー管理、Memberstack データ テーブルについて言及する場合、または Memberstack 内のメンバーまたはデータ レコードを作成/読み取り/更新/削除する統合を構築したい場合は、このスキルを使用します。また、ユーザーが Memberstack API エンドポイント、認証、ページネーション、レコードのクエリ、またはメンバーへのプランの接続について尋ねたときにもトリガーされます。これは、ユーザーが明示的に「Memberstack」とは言わなくても、プランの接続、メンバーのメタデータ、または Memberstack のドメインに合わせた管理メンバー API などの概念を参照している場合でも同様です。
