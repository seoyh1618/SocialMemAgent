Utilizza l'API Memberstack Admin per gestire membri, piani e tabelle dati. Utilizza questa competenza ogni volta che l'utente menziona Memberstack, la gestione dei membri tramite API, le tabelle di dati di Memberstack o desidera creare integrazioni che creano/leggono/aggiornano/eliminano membri o record di dati in Memberstack. Si attiva anche quando l'utente chiede informazioni su endpoint API Memberstack, autenticazione, impaginazione, query di record o connessione di piani ai membri, anche se non dice esplicitamente "Memberstack" ma fa riferimento a concetti come connessioni di piani, metadati dei membri o API dei membri di amministrazione che si allineano al dominio di Memberstack.
