Trabaje con la API de administración de Memberstack para administrar miembros, planes y tablas de datos. Utilice esta habilidad siempre que el usuario mencione Memberstack, administración de miembros a través de API, tablas de datos de Memberstack o quiera crear integraciones que creen/lean/actualicen/eliminen miembros o registros de datos en Memberstack. También se activa cuando el usuario pregunta sobre los puntos finales de la API de Memberstack, la autenticación, la paginación, la consulta de registros o la conexión de planes a los miembros, incluso si no dicen "Memberstack" explícitamente, sino que hacen referencia a conceptos como conexiones de planes, metadatos de miembros o API de miembros administradores que se alinean con el dominio de Memberstack.
