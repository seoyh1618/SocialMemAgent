Работайте с API администратора Memberstack для управления участниками, планами и таблицами данных. Используйте этот навык всякий раз, когда пользователь упоминает Memberstack, управление участниками через API, таблицы данных Memberstack или хочет создать интеграцию, которая создает/читает/обновляет/удаляет участников или записи данных в Memberstack. Также срабатывает, когда пользователь спрашивает о конечных точках API Memberstack, аутентификации, нумерации страниц, запросе записей или подключении планов к участникам — даже если они не говорят «Memberstack» явно, но ссылаются на такие концепции, как соединения планов, метаданные участников или API-интерфейсы участников администратора, которые соответствуют домену Memberstack.
