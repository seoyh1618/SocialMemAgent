Arbeiten Sie mit der Memberstack Admin API, um Mitglieder, Pläne und Datentabellen zu verwalten. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer Memberstack, Mitgliederverwaltung über API oder Memberstack-Datentabellen erwähnt oder Integrationen erstellen möchte, die Mitglieder oder Datensätze in Memberstack erstellen/lesen/aktualisieren/löschen. Wird auch ausgelöst, wenn der Benutzer nach Memberstack-API-Endpunkten, Authentifizierung, Paginierung, Abfrage von Datensätzen oder Verbindungsplänen mit Mitgliedern fragt – auch wenn er nicht explizit „Memberstack“ sagt, sondern auf Konzepte wie Planverbindungen, Mitgliedermetadaten oder Admin-Mitglieder-APIs verweist, die mit der Domäne von Memberstack übereinstimmen.
