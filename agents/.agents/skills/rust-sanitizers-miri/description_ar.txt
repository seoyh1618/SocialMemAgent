مطهرات الصدأ ومهارة ميري للتحقق من سلامة الذاكرة. يُستخدم عند تشغيل AddressSanitizer أو ThreadSanitizer على كود Rust، أو تفسير تقارير المطهر، أو استخدام Miri لاكتشاف السلوك غير المحدد في Rust غير الآمن، أو التحقق من صحة التعليمات البرمجية غير الآمنة. يتم التنشيط عند الاستفسارات حول Rust ASan، أو Rust TSan، أو Miri، أو RUSTFLAGS Sanitize، أو cargo miri، أو Rust UB غير الآمن، أو تفسير مخرجات مطهر Rust.
