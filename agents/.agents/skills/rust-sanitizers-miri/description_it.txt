Disinfettanti antiruggine e competenza Miri per la convalida della sicurezza della memoria. Da utilizzare quando si esegue AddressSanitizer o ThreadSanitizer su codice Rust, si interpretano report di sanitizzazione, si utilizza Miri per rilevare comportamenti indefiniti in Rust non sicuro o si convalida la correttezza del codice non sicuro. Si attiva su domande su Rust ASan, Rust TSan, Miri, RUSTFLAGS sanitize, cargo miri, Rust UB non sicuro o sull'interpretazione dell'output del disinfettante Rust.
