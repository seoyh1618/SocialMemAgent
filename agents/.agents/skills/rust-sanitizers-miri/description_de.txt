Rostdesinfektionsmittel und Miri-Skill zur Validierung der Speichersicherheit. Wird verwendet, wenn Sie AddressSanitizer oder ThreadSanitizer auf Rust-Code ausführen, Sanitizer-Berichte interpretieren, Miri verwenden, um undefiniertes Verhalten in unsicherem Rust zu erkennen, oder die Richtigkeit unsicheren Codes überprüfen. Wird bei Abfragen zu Rust ASan, Rust TSan, Miri, RUSTFLAGS Sanitize, Cargo Miri, unsicherem Rust UB oder der Interpretation der Rust Sanitizer-Ausgabe aktiviert.
