Désinfectants antirouille et compétence Miri pour la validation de la sécurité de la mémoire. À utiliser lors de l'exécution d'AddressSanitizer ou de ThreadSanitizer sur du code Rust, de l'interprétation des rapports de désinfectant, de l'utilisation de Miri pour détecter un comportement non défini dans Rust non sécurisé ou de la validation de l'exactitude du code non sécurisé. S'active sur les requêtes concernant Rust ASan, Rust TSan, Miri, RUSTFLAGS sanitize, cargo miri, Rust UB non sécurisé ou l'interprétation de la sortie du désinfectant Rust.
