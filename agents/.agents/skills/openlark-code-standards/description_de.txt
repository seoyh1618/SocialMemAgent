Fähigkeiten zur Inspektion von OpenLark-Projektcodespezifikationen. Wird verwendet, um die Architekturkonsistenz, API-Implementierungsmuster, Parametervalidierung, Benennungs- und Exportkonventionen innerhalb des Repositorys schnell zu überprüfen und eine umsetzbare Checkliste mit Beweispfaden auszugeben.
