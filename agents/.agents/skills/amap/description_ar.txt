اتصل مباشرة بـ AMap Web Service API عبر البرنامج النصي لإجراء الترميز الجغرافي والترميز الجغرافي العكسي وتحديد الموقع الجغرافي لـ IP والطقس وتخطيط المسار وقياس المسافة وعمليات البحث عن النقاط المهمة. استخدمه عندما يطلب المستخدمون "استعلام AMap"، أو "تخطيط الطريق"، أو "الترميز الجغرافي"، أو "بحث النقاط المهمة"، أو عندما يحتاجون إلى الاتصال بـ AMap API من برنامج نصي لسطر الأوامر.
