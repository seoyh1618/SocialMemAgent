Прямое подключение к API веб-службы AMap через скрипт для выполнения геокодирования, обратного геокодирования, геолокации по IP, прогноза погоды, планирования маршрута, измерения расстояния и поиска POI. Используйте его, когда пользователи запрашивают «запрос AMap», «планирование маршрута», «геокодирование», «поиск POI» или когда им необходимо вызвать API AMap из сценария командной строки.
