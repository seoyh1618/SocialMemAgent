Connectez-vous directement à l'API du service Web AMap via un script pour effectuer le géocodage, le géocodage inversé, la géolocalisation IP, la météo, la planification d'itinéraire, la mesure de distance et les recherches de POI. Utilisez-le lorsque les utilisateurs demandent une « requête AMap », une « planification d'itinéraire », un « géocodage », une « recherche de POI » ou doivent appeler l'API AMap à partir d'un script de ligne de commande.
