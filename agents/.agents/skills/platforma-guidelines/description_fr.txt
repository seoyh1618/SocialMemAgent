Conseils pour créer des applications Go backend avec github.com/platforma-dev/platforma. À utiliser lorsqu'on vous demande de concevoir la structure du projet Platforma, de choisir ou de combiner des packages Platforma (application, base de données, serveur http, authentification, file d'attente, planificateur), d'expliquer ou d'exécuter des commandes CLI natives du framework, ou de fournir des exemples d'intégration pratiques et des flux de travail de migration/exécution.
