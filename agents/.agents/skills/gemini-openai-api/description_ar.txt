دليل تكامل نموذج الجوزاء عبر واجهة برمجة التطبيقات المتوافقة مع OpenAI. يتضمن: (1) تكوين واجهة برمجة التطبيقات الإضافية (ملخص/تصحيح/عاطفة/رؤية)، (2) تنسيق extra_body للتحكم في التفكير، (3) التعامل مع تنسيق الاستجابة (كتل كود تخفيض السعر). يُستخدم عند دمج Gemini كنموذج مساعد، أو تكوين معلمات التفكير، أو التعامل مع تنسيقات إرجاع Gemini API.
