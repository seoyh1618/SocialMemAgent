Guía de integración del modelo Gemini a través de API compatible con OpenAI. Incluye: (1) configuración de API auxiliar (resumen/corrección/emoción/visión), (2) formato extra_body para controlar el pensamiento, (3) manejo del formato de respuesta (bloques de código de rebajas). Úselo al integrar Gemini como modelo auxiliar, configurar parámetros de pensamiento o manejar formatos de retorno de API de Gemini.
