Guide d'intégration du modèle Gemini via l'API compatible OpenAI. Comprend : (1) la configuration de l'API auxiliaire (résumé/correction/émotion/vision), (2) le format extra_body pour contrôler la pensée, (3) la gestion du format de réponse (blocs de code de démarque). À utiliser lors de l'intégration de Gemini en tant que modèle auxiliaire, de la configuration des paramètres de réflexion ou de la gestion des formats de retour de l'API Gemini.
