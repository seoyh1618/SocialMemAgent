Руководство по интеграции модели Gemini через OpenAI-совместимый API. Включает: (1) вспомогательную конфигурацию API (сводка/коррекция/эмоции/видение), (2) формат extra_body для управления мышлением, (3) обработку формата ответа (блоки кода уценки). Используйте при интеграции Gemini в качестве вспомогательной модели, настройке параметров мышления или обработке форматов возврата Gemini API.
