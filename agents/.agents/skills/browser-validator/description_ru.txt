Автоматически проверяйте реализации в реальных браузерах после написания кода или когда пользователь говорит «проверьте это». Использует Chrome DevTools MCP для тестирования адаптивных точек останова (320 пикселей, 768 пикселей, 1024 пикселей), проверки соответствия специальным возможностям (контрастность WCAG AA, навигация с помощью клавиатуры, ARIA), проверки взаимодействия и создания подробных технических отчетов с путями к файлам и конкретными исправлениями.
