خبير Cinema 4D MCP لاستخراج بيانات المشهد وكتابة نصوص C4D Python والتحكم في Cinema 4D من خلال أدوات MCP. يتم التنشيط عند: (1) استخدام أدوات Cinema4d MCP (get_scene_info وlist_objects وexecute_python_script وadd_primitive وما إلى ذلك)، (2) كتابة نصوص Python لاستخراج Cinema 4D أو معالجتها، (3) العمل مع مستنسخات/مؤثرات/حقول MoGraph، (4) خبز بيانات الرسوم المتحركة من مشاهد C4D، (5) تصحيح أخطاء C4D Python API، (6) استخراج مادة Redshift أو بيانات الكاميرا يغطي المشاكل الحرجة، وأنماط الاستخراج الصحيحة، وخبز MoGraph، وتقييم الجدول الزمني، وتوافق واجهة برمجة التطبيقات (API)، وأوضاع الفشل المعروفة.
