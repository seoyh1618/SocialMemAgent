シーン データの抽出、C4D Python スクリプトの作成、MCP ツールによる Cinema 4D の制御を行う Cinema 4D MCP エキスパート。次の場合にアクティブ化します: (1) cinema4d MCP ツール (get_scene_info、list_objects、execute_python_script、add_primitive など) を使用する場合、(2) Cinema 4D 抽出または操作用の Python スクリプトを作成する場合、(3) MoGraph クローン/エフェクター/フィールドを使用する場合、(4) C4D シーンからアニメーション データをベイクする場合、(5) C4D Python API エラーをデバッグする場合、(6) Redshift マテリアルまたはカメラを抽出する場合データ。重要な注意点、正しい抽出パターン、MoGraph ベイク、タイムライン評価、API 互換性、既知の障害モードについて説明します。
