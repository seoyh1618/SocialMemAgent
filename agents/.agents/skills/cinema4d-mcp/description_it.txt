Esperto di Cinema 4D MCP per l'estrazione di dati di scena, la scrittura di script C4D Python e il controllo di Cinema 4D tramite strumenti MCP. Da attivare quando: (1) si utilizzano gli strumenti MCP cinema4d (get_scene_info, list_objects,execute_python_script, add_primitive, ecc.), (2) si scrivono script Python per l'estrazione o la manipolazione di Cinema 4D, (3) si lavora con clonatori/effettori/campi MoGraph, (4) si creano dati di animazione da scene C4D, (5) si esegue il debug di errori API C4D Python, (6) si estrae materiale Redshift o dati della telecamera. Copre i trucchi critici, i modelli di estrazione corretti, la cottura MoGraph, la valutazione della sequenza temporale, la compatibilità API e le modalità di errore note.
