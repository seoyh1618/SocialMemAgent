Cinema 4D MCP-Experte für das Extrahieren von Szenendaten, das Schreiben von C4D-Python-Skripten und die Steuerung von Cinema 4D über MCP-Tools. Aktivieren, wenn: (1) Cinema4d-MCP-Tools (get_scene_info, list_objects,execute_python_script, add_primitive usw.) verwendet werden, (2) Python-Skripte für die Extraktion oder Bearbeitung von Cinema 4D geschrieben werden, (3) mit MoGraph-Klonern/Effektoren/Feldern gearbeitet werden, (4) Animationsdaten aus C4D-Szenen gebacken werden, (5) C4D-Python-API-Fehler behoben werden, (6) Redshift-Material oder Kameradaten extrahiert werden. Behandelt kritische Fallstricke, korrekte Extraktionsmuster, MoGraph-Backen, Zeitachsenauswertung, API-Kompatibilität und bekannte Fehlermodi.
