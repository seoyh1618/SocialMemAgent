Expert Cinema 4D MCP pour l'extraction de données de scène, l'écriture de scripts Python C4D et le contrôle de Cinema 4D via les outils MCP. Activer lorsque : (1) vous utilisez les outils Cinema4d MCP (get_scene_info, list_objects,execute_python_script, add_primitive, etc.), (2) vous écrivez des scripts Python pour l'extraction ou la manipulation de Cinema 4D, (3) vous travaillez avec des cloneurs/effecteurs/champs MoGraph, (4) vous préparez des données d'animation à partir de scènes C4D, (5) vous déboguez des erreurs de l'API Python C4D, (6) vous extrayez du matériel Redshift ou des données de caméra. Couvre les pièges critiques, les modèles d'extraction corrects, la cuisson MoGraph, l'évaluation de la chronologie, la compatibilité API et les modes de défaillance connus.
