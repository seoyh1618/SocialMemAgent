Эксперт Cinema 4D MCP для извлечения данных сцены, написания сценариев C4D Python и управления Cinema 4D с помощью инструментов MCP. Активируйте, когда: (1) используете инструменты Cinema4d MCP (get_scene_info, list_objects, Execute_python_script, add_primitive и т. д.), (2) пишете сценарии Python для извлечения или манипулирования Cinema 4D, (3) работаете с клонерами/эффекторами/полями MoGraph, (4) запекаете данные анимации из сцен C4D, (5) отлаживаете ошибки C4D Python API, (6) извлекаете материал или камеру Redshift данные. Охватывает критические ошибки, правильные шаблоны извлечения, запекание MoGraph, оценку сроков, совместимость API и известные режимы сбоев.
