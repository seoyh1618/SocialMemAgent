クラスターのログとデバッグのために Loki API をクエリします。アプリケーションエラーを調査するときに使用します。
ログの検索、ポッドの問題のデバッグ、またはログ パターンのクエリ。

次の場合に使用します: (1) エラーまたはパターンのログを検索する、(2) ログを介してポッドまたはサービスの問題を調査する、
(3) Kubernetes イベントのクエリ、(4) Flux 調整またはプラットフォーム サービスの失敗のデバッグ、
(5) LogQL クエリの実行、(6) ログのレートまたはボリュームの確認。
