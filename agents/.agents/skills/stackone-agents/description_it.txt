Crea agenti AI che chiamano account collegati a StackOne utilizzando TypeScript SDK, Python SDK, server MCP o protocollo A2A. Da utilizzare quando l'utente chiede di "aggiungere gli strumenti StackOne al mio agente", "configurare MCP con StackOne", "elencare i dipendenti di BambooHR nel mio agente", "integrare StackOne con OpenAI", "creare un agente multi-tenant" o "usa StackOne con LangChain". Supporta OpenAI, Vercel AI SDK, Claude, LangChain, CrewAI, PydanticAI. NON utilizzare per la configurazione del collegamento dell'account (usa stackone-connect) o la gestione della piattaforma (usa stackone-platform).
