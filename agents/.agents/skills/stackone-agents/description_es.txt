Cree agentes de IA que llamen a cuentas vinculadas a StackOne mediante el SDK de TypeScript, el SDK de Python, el servidor MCP o el protocolo A2A. Úselo cuando el usuario solicite "agregar herramientas StackOne a mi agente", "configurar MCP con StackOne", "listar empleados de BambooHR en mi agente", "integrar StackOne con OpenAI", "crear un agente multiinquilino" o "usar StackOne con LangChain". Admite OpenAI, Vercel AI SDK, Claude, LangChain, CrewAI, PydanticAI. NO lo use para la configuración de vinculación de cuentas (use stackone-connect) o para la administración de plataformas (use stackone-platform).
