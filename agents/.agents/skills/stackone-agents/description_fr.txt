Créez des agents IA qui appellent des comptes liés à StackOne à l'aide du SDK TypeScript, du SDK Python, du serveur MCP ou du protocole A2A. À utiliser lorsque l'utilisateur demande « d'ajouter les outils StackOne à mon agent », « configurer MCP avec StackOne », « lister les employés de BambooHR dans mon agent », « intégrer StackOne avec OpenAI », « créer un agent multi-tenant » ou « utiliser StackOne avec LangChain ». Prend en charge OpenAI, Vercel AI SDK, Claude, LangChain, CrewAI, PydanticAI. Ne PAS utiliser pour la configuration de la liaison de compte (utilisez stackone-connect) ou la gestion de la plateforme (utilisez stackone-platform).
