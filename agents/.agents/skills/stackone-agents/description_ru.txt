Создавайте агенты искусственного интеллекта, которые вызывают учетные записи, связанные со StackOne, с помощью TypeScript SDK, Python SDK, сервера MCP или протокола A2A. Используйте, когда пользователь просит «добавить инструменты StackOne в мой агент», «настроить MCP с помощью StackOne», «перечислить сотрудников из BambooHR в моем агенте», «интегрировать StackOne с OpenAI», «создать многопользовательский агент» или «использовать StackOne с LangChain». Поддерживает OpenAI, Vercel AI SDK, Claude, LangChain, CrewAI, PydanticAI. НЕ используйте для настройки привязки учетных записей (используйте stackone-connect) или управления платформой (используйте stackone-platform).
