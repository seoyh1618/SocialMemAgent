Маршрутизация API шлюза, сертификаты TLS и конфигурация WAF для платформы Homelab Kubernetes.

Используйте, когда: (1) предоставление сервиса через HTTPRoute, (2) выбор между внутренним и внешним шлюзами,
(3) Настройка сертификатов TLS с помощью диспетчера сертификатов, (4) Отладка проблем TLS или маршрутизации,
(5) Понимание поведения и настройки WAF (Coraza), (6) Тестирование конечных точек, защищенных WAF.

Триггеры: «httproute», «шлюз», «предоставить услугу», «добавить маршрут», «сертификат», «tls»,
«coraza», «waf», «внутренний шлюз», «внешний шлюз», «dns», «вход»,
«маршрутизация», «менеджер сертификатов», «letencrypt», «homelab-ca»
