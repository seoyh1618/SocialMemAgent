توجيه واجهة API للبوابة وشهادات TLS وتكوين WAF لمنصة homelab Kubernetes.

يُستخدم عندما: (1) عرض خدمة عبر HTTPRoute، (2) الاختيار بين البوابات الداخلية والخارجية،
(3) إعداد شهادات TLS مع مدير الشهادات، (4) تصحيح أخطاء TLS أو مشكلات التوجيه،
(5) فهم سلوك WAF (Coraza) وضبطه، (6) اختبار نقاط النهاية المحمية بـ WAF.
