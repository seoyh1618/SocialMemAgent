Homelab Kubernetes 平台的閘道 API 路由、TLS 憑證和 WAF 設定。

在下列情況下使用：(1) 透過 HTTPRoute 公開服務，(2) 在內部和外部網關之間進行選擇，
(3) 使用 cert-manager 設定 TLS 證書，(4) 調試 TLS 或路由問題，
(5) 了解 WAF (Coraza) 行為和調整，(6) 測試受 WAF 保護的端點。
