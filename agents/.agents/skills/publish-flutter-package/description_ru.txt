Автоматизирует процесс выпуска пакета Flutter с помощью тегов git и действий GitHub. Обрабатывает рабочие области с несколькими пакетами, предложения по управлению версиями SemVer на основе истории git, обновление pubspec.yaml и CHANGELOG.md, а также проверку пробного прогона. Используйте, когда пользователь хочет «выпустить», «опубликовать» или «версию» пакета Flutter.
