Automatiza el proceso de lanzamiento del paquete Flutter mediante etiquetas git y acciones de GitHub. Maneja espacios de trabajo de múltiples paquetes, sugerencias de versiones de SemVer basadas en el historial de git, actualización de pubspec.yaml y CHANGELOG.md, y validación de ensayo. Úselo cuando el usuario quiera "liberar", "publicar" o "versionar" un paquete de Flutter.
