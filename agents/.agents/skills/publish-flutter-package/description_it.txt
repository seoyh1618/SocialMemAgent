Automatizza il processo di rilascio del pacchetto Flutter tramite tag git e azioni GitHub. Gestisce aree di lavoro multipacchetto, suggerimenti sul controllo delle versioni di SemVer basati sulla cronologia git, l'aggiornamento di pubspec.yaml e CHANGELOG.md e la convalida di prova. Da utilizzare quando l'utente desidera "rilasciare", "pubblicare" o "versionare" un pacchetto Flutter.
