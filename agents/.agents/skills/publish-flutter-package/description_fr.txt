Automatise le processus de publication du package Flutter via les balises git et les actions GitHub. Gère les espaces de travail multi-packages, les suggestions de versioning SemVer basées sur l'historique git, la mise à jour de pubspec.yaml et CHANGELOG.md et la validation à sec. À utiliser lorsque l'utilisateur souhaite « publier », « publier » ou « versionner » un package Flutter.
