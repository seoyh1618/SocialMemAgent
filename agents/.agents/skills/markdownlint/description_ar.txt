يضمن تناسق النمط وصحته في ملفات Markdown باستخدام markdownlint-cli2. يُستخدم عند إنشاء أو تحديث أو تحرير أو مراجعة أي ملف Markdown (.md). تشغيل أي مهمة تقوم بإنشاء أو تعديل ملفات .md، بما في ذلك README.md أو CHANGELOG.md أو الوثائق أو أي ملف بامتداد .md. يتم تشغيله أيضًا عندما يذكر المستخدم تخفيض السعر، أو الفحص، أو تنسيق تخفيض السعر، أو نمط تخفيض السعر، أو تخفيض السعر. قم دائمًا بتشغيل جهاز linter بعد كتابة محتوى تخفيض السعر أو تحريره — لا تتجاوز هذه الخطوة.
