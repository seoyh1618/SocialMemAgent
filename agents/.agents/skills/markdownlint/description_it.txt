Garantisce la coerenza e la correttezza dello stile nei file Markdown utilizzando markdownlint-cli2. Da utilizzare durante la creazione, l'aggiornamento, la modifica o la revisione di qualsiasi file Markdown (.md). Si attiva su qualsiasi attività che crea o modifica file .md, inclusi README.md, CHANGELOG.md, documentazione o qualsiasi file con estensione .md. Si attiva anche quando l'utente menziona markdown, linting, formattazione markdown, stile markdown o markdownlint. Esegui sempre il linter dopo aver scritto o modificato il contenuto del markdown: non saltare questo passaggio.
