Руководство по созданию приложений для iOS с использованием Swift 6, iOS 18+, SwiftUI, SwiftData и современных шаблонов параллелизма. Используйте при написании кода Swift/iOS, разработке архитектуры приложений или модернизации устаревших шаблонов. Предотвращает использование устаревших шаблонов, таких как Core Data, ObservableObject, DispatchQueue и NavigationView.
