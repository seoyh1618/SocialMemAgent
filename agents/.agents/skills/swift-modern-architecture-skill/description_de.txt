Leitfaden zum Erstellen von iOS-Apps mit Swift 6, iOS 18+, SwiftUI, SwiftData und modernen Parallelitätsmustern. Verwenden Sie es beim Schreiben von Swift-/iOS-Code, beim Entwerfen einer App-Architektur oder beim Modernisieren älterer Muster. Verhindert veraltete Muster wie Core Data, ObservableObject, DispatchQueue und NavigationView.
