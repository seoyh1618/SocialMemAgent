Competenze di interprete bytecode e compilatore JIT per l'implementazione di runtime di linguaggio in C/C++. Da utilizzare durante la progettazione di cicli di invio di bytecode (switch, goto calcolato, codice thread), l'implementazione di VM basate su stack o registri, l'aggiunta di un semplice JIT utilizzando mmap/mprotect o la comprensione dei compromessi prestazionali nella progettazione dell'interprete. Si attiva su query su VM bytecode, loop di invio, goto computerizzato, nozioni di base sulla compilazione JIT, tracciamento JIT o implementazione di un runtime del linguaggio di scripting.
