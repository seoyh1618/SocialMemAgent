Создавайте и тестируйте смарт-контракты Solidity с помощью инструментария Foundry. Используйте при разработке контрактов Ethereum, написании тестов Forge, развертывании с помощью сценариев или отладке с помощью Cast/Anvil. Запускается по командам Foundry (forge, cast, anvil), тестированию Solidity, разработке смарт-контрактов или таким файлам, как Foundry.toml, *.t.sol, *.s.sol.
