قم ببناء واختبار عقود Solidity الذكية باستخدام مجموعة أدوات Foundry. يُستخدم عند تطوير عقود Ethereum، أو كتابة اختبارات Forge، أو النشر باستخدام البرامج النصية، أو تصحيح الأخطاء باستخدام Cast/Anvil. مشغلات لأوامر Foundry (Forge، Cast، Anvil)، أو اختبار الصلابة، أو تطوير العقود الذكية، أو ملفات مثل Foundry.toml، أو *.t.sol، أو *.s.sol.
