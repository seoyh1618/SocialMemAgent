تطوير Web3 باستخدام MetaMask Smart Accounts Kit. يُستخدم عندما يريد المستخدم إنشاء dApps باستخدام حسابات ذكية ERC-4337، أو إرسال عمليات المستخدم، أو المعاملات المجمعة، أو تكوين الموقعين (EOA، مفتاح المرور، multisig)، أو تنفيذ استخراج الغاز مع جهات الدفع، أو إنشاء تفويضات، أو طلب أذونات متقدمة (ERC-7715). يدعم تكامل Viem وأنواع التوقيعات المتعددة (Dynamic وWeb3Auth وWagmi) والمعاملات غير الغازية وإطار عمل التفويض.
