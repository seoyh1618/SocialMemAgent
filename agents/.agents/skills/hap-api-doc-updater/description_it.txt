Una competenza dedicata al mantenimento e all'aggiornamento della documentazione dell'interfaccia MingDao Cloud HAP. Utilizza questa competenza quando gli utenti devono aggiungere, modificare o eliminare interfacce nella documentazione dell'API HAP o durante la sincronizzazione delle versioni cinese e inglese della documentazione. Supporta sia i tipi di documentazione delle API autorizzate dall'organizzazione (en/zh-Hans) che delle API dell'applicazione (applicazione).
