Une compétence dédiée à la maintenance et à la mise à jour de la documentation de l'interface MingDao Cloud HAP. Utilisez-le lorsque les utilisateurs doivent ajouter, modifier ou supprimer des API dans la documentation de l'API HAP ou synchroniser les versions chinoise et anglaise. Prend en charge à la fois les API autorisées par l'organisation (en/zh-Hans) et les API d'application (application).
