Навык, посвященный поддержке и обновлению документации интерфейса HAP MingDao Cloud. Используйте этот навык, когда пользователям нужно добавить, изменить или удалить интерфейсы в документации HAP API или при синхронизации версий документации на китайском и английском языках. Поддерживает типы документации API, авторизованные организацией (en/zh-Hans), и API приложений (приложение).
