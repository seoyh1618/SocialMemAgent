Eine Fähigkeit speziell für die Pflege und Aktualisierung der HAP-Schnittstellendokumentation von Mingdao Cloud. Verwenden Sie diese Fähigkeit, wenn Benutzer Schnittstellen in der HAP-API-Dokumentation hinzufügen, ändern oder löschen müssen oder wenn sie chinesische und englische Versionen der Dokumentation synchronisieren. Unterstützt sowohl die Dokumentationstypen Organization Authorized API (en/zh-Hans) als auch Application API (application).
