Пользовательский интерфейс и создание HUD для Hytale. Охватывает структуру NoesisGUI, настраиваемые наложения HUD, меню, экраны инвентаря и интерактивные элементы пользовательского интерфейса. Используйте при создании пользовательских интерфейсов, отображений статуса, панелей администратора или любых компонентов визуального пользовательского интерфейса.
