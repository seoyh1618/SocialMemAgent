Interface utilisateur personnalisée et création de HUD pour Hytale. Couvre le framework NoesisGUI, les superpositions HUD personnalisées, les menus, les écrans d'inventaire et les éléments interactifs de l'interface utilisateur. À utiliser lors de la création d'interfaces personnalisées, d'affichages d'état, de panneaux d'administration ou de tout composant visuel de l'interface utilisateur.
