Progetta e stilizza i componenti dell'interfaccia utente per ryOS seguendo i 4 temi del sistema operativo (System 7, macOS Aqua, Windows XP, Windows 98). Da utilizzare quando si creano componenti dell'interfaccia utente, elementi di styling, si lavora con temi, si aggiungono effetti visivi o si implementa l'estetica del sistema operativo retrò.
