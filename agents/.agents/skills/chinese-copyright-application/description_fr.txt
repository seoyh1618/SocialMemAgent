Une boîte à outils complète pour générer du matériel pour les applications de droits d'auteur sur les logiciels en Chine. Extrait automatiquement les informations du code et des documents du projet pour générer le formulaire d'enregistrement des droits d'auteur du logiciel, les documents du code source (30 premières et dernières pages), le manuel d'utilisation et les spécifications de conception, puis les convertit en fichiers PDF. Convient aux mini-programmes, applications Web, applications mobiles, applications de bureau et autres projets logiciels. Utilisez cet outil lorsque vous déposez une demande de droits d'auteur sur un logiciel en Chine.
