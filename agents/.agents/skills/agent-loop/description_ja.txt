Ingest を介して、耐久性のあるマルチエージェント コーディング ループを開始、監視、キャンセルします。ユーザーが自律コーディング ワークロードを実行する場合、複数のストーリーを含む PRD を実行する場合、AFK コーディング セッションを開始する場合、エージェントにプランの機能を実装させる場合、または実行ループを管理する場合に使用します。 「コーディング ループを開始する」、「この PRD を実行する」、「これらのストーリーを実装する」、「AFK を実行してこれをコード化する」、「ループのステータスを確認する」、「ループをキャンセルする」、「joelclaw ループ」、または自律的なマルチストーリー コード実行のリクエストをトリガーします。
