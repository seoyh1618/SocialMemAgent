Configurer et gérer les serveurs MCP (Model Context Protocol) dans Claude Code CLI. Utilisez cette compétence lors de l'ajout, de la suppression, de la liste ou du dépannage des serveurs MCP, ou lorsque des questions se posent concernant la configuration MCP, les types de transport, les étendues ou l'authentification. Indispensable pour connecter Claude Code à des outils externes, des bases de données et des API.
