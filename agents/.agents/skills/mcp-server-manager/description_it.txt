Configura e gestisci i server MCP (Model Context Protocol) nella CLI di Claude Code. Utilizza questa competenza quando aggiungi, rimuovi, elenchi o risolvi i problemi dei server MCP o quando sorgono domande sulla configurazione di MCP, sui tipi di trasporto, sugli ambiti o sull'autenticazione. Essenziale per connettere Claude Code a strumenti, database e API esterni.
