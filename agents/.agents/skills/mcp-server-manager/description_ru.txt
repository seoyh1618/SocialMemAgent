Настраивайте серверы MCP (протокол контекста модели) и управляйте ими в интерфейсе командной строки Claude Code. Используйте этот навык при добавлении, удалении, перечислении или устранении неполадок серверов MCP, а также при возникновении вопросов о конфигурации MCP, типах транспорта, областях или аутентификации. Необходим для подключения Claude Code к внешним инструментам, базам данных и API.
