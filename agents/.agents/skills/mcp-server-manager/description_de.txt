Konfigurieren und verwalten Sie MCP-Server (Model Context Protocol) in der Claude Code CLI. Verwenden Sie diese Fähigkeit, wenn Sie MCP-Server hinzufügen, entfernen, auflisten oder Fehler beheben oder wenn Fragen zur MCP-Konfiguration, Transporttypen, Bereichen oder Authentifizierung auftreten. Unverzichtbar für die Verbindung von Claude Code mit externen Tools, Datenbanken und APIs.
