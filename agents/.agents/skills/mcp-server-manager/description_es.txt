Configure y administre servidores MCP (Protocolo de contexto modelo) en Claude Code CLI. Utilice esta habilidad al agregar, eliminar, enumerar o solucionar problemas de servidores MCP, o cuando surjan preguntas sobre la configuración, los tipos de transporte, los alcances o la autenticación de MCP. Esencial para conectar Claude Code a herramientas, bases de datos y API externas.
