Créez des API Web ASP.NET Core avec .NET 10 (C# 14.0). Prend en charge l'échafaudage de projet, les opérations CRUD, l'intégration d'Entity Framework, l'injection de dépendances, les tests avec xUnit, la conteneurisation Docker et le respect des meilleures pratiques 2025. À utiliser lors de la création d'API REST, de microservices, de services backend, de la mise en œuvre d'opérations CRUD, de la configuration d'Entity Framework, de l'ajout d'authentification/autorisation ou de la conteneurisation d'applications .NET. Déclencheurs sur .NET, ASP.NET Core, C#, Web API, REST API, microservices, dotnet, tâches de développement csharp.
