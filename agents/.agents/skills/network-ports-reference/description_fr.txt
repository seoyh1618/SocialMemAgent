L'assistant fournit des informations complètes de référence sur les ports réseau et les protocoles pour les évaluations de sécurité et les tests d'intrusion. Activez-la lorsque les utilisateurs posent des questions sur les « numéros de port », les « ports communs », les « ports de service », les « ports TCP/UDP », « ce qui s'exécute sur le port X » ou « l'identification du protocole ».
