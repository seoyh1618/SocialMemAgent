Diseñe y redacte agentes e indicaciones para agentes de Claude Code de alta calidad. Úselo al crear o actualizar .claude/agents/*.md para (1) subagentes de propósito único (revisores, implementadores, investigadores) y (2) orquestadores de flujo de trabajo (coordinadores de múltiples fases como revisión previa, desarrollo de funciones, corrección de errores). Cubre activadores de delegación, opciones de herramienta/permiso/modelo, orquestación de herramientas de tarea, transferencias de fase, agregación, puertas de iteración y contratos de salida. Úselo también al decidir entre subagentes, habilidades o orientación de repositorio siempre disponible.
