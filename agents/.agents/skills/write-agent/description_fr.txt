Concevoir et rédiger des agents Claude Code et des invites d'agent de haute qualité. À utiliser lors de la création ou de la mise à jour de .claude/agents/*.md pour (1) les sous-agents à usage unique (réviseurs, implémenteurs, chercheurs) et (2) les orchestrateurs de flux de travail (coordonnateurs multi-phases comme la révision préalable, le développement de fonctionnalités, la correction de bugs). Couvre les déclencheurs de délégation, les choix d'outils/autorisations/modèles, l'orchestration des outils de tâches, les transferts de phases, l'agrégation, les portes d'itération et les contrats de sortie. À utiliser également pour décider entre les sous-agents, les compétences et les conseils de dépôt permanents.
