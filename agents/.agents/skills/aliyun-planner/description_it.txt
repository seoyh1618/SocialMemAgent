Comprensione del linguaggio naturale degli intenti di query delle risorse Alibaba Cloud, pianificazione e generazione di piani di esecuzione CLI Aliyun standardizzati (formato JSON). Utilizzato quando gli utenti interrogano le risorse Alibaba Cloud (ECS, RDS, VPC, SLB, OSS, ecc.), necessitano di ottenere lo stato delle risorse, visualizzare i dettagli delle risorse o analizzare le relazioni tra le risorse. Supporta query semplici, query associate, query composite, query diagnostiche e altri scenari.
