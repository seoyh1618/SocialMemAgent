Lluvia de ideas ligera para corregir errores, pequeñas funciones o mejoras de código que necesitan aclaración antes de la implementación. Activa preguntas profundas para garantizar resultados precisos. Úselo cuando el usuario solicite corregir un error, agregar una pequeña característica, refactorizar el código o realizar mejoras específicas, y el alcance es lo suficientemente pequeño como para que no sea necesaria una sesión completa de lluvia de ideas. NO lo utilice para decisiones arquitectónicas a gran escala o proyectos totalmente nuevos.
