Configuration complète du SDK Sentry pour NestJS. À utiliser lorsqu'on vous demande d'« ajouter Sentry à NestJS », « d'installer @sentry/nestjs », de « configurer Sentry dans NestJS » ou de configurer la surveillance des erreurs, le traçage, le profilage, la journalisation, les métriques, les crons ou la surveillance de l'IA pour les applications NestJS. Prend en charge les adaptateurs Express et Fastify, GraphQL, les microservices, les WebSockets et les tâches en arrière-plan.
