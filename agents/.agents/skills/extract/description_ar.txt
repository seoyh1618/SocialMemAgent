استخرج البيانات المنظمة من مواقع الويب وأنتج نصًا مسرحيًا قابلاً للتنفيذ بالإضافة إلى البيانات المستخرجة. يُستخدم عندما يريد المستخدم استخراج البيانات أو استخراجها أو سحبها أو جمعها أو حصادها من أي موقع ويب - قوائم المنتجات أو الجداول أو نتائج البحث أو الخلاصات أو الملفات الشخصية أو أي محتوى متكرر.
