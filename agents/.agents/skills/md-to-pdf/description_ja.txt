人魚図、LaTeX/KaTeX 数式、表、構文強調表示されたコード ブロック、およびすべての標準的な Markdown 機能を完全にサポートして、Markdown ファイルをプロフェッショナルなスタイルの PDF ドキュメントに変換します。ユーザーが .md ファイルを .pdf に変換する場合、Markdown から PDF レポートを生成する場合、または図、数式、または複雑な書式設定を含む Markdown ソースから印刷用ドキュメントを作成する場合には、このスキルを使用します。トリガー フレーズには、「マークダウンを PDF に変換する」、「この MD から PDF を作成する」、「このマークダウンをレンダリングする」、「マークダウンを PDF としてエクスポート」、「図を含むマークダウンを PDF に」、「数式を含むマークダウンから PDF を作成する」、「PDF レポートを生成する」、「マークダウンを PDF としてエクスポートするにはどうすればよいですか」、「メモを PDF に変換します」、「このマークダウンから PDF を作成できますか」が含まれます。
