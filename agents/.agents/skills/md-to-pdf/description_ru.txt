Преобразуйте файлы Markdown в профессионально оформленные PDF-документы с полной поддержкой диаграмм Mermaid, математических уравнений LaTeX/KaTeX, таблиц, блоков кода с подсветкой синтаксиса и всех стандартных функций Markdown. Используйте этот навык всякий раз, когда пользователь хочет преобразовать файл .md в .pdf, создать отчет в формате PDF из Markdown или создать готовые к печати документы из источников Markdown, содержащие диаграммы, математические вычисления или сложное форматирование. Триггерные фразы включают в себя: «конвертировать уценку в PDF», «создать PDF-файл из этого md», «отобразить эту уценку», «экспортировать уценку в PDF», «уценку в PDF с диаграммами», «pdf из уценки с помощью уравнений», «создать отчет в формате PDF», «как мне экспортировать уценку в формате PDF», «конвертировать мои заметки в PDF», «вы можете сделать PDF-файл из этой уценки».
