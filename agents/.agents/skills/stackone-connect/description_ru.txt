Реализуйте связывание учетных записей с помощью сеансов StackOne Connect и компонента Hub React. Используйте, когда пользователь просит «подключить провайдера», «встроить средство выбора интеграции», «добавить BambooHR в мое приложение», «создать сеанс подключения», «настроить ссылки аутентификации» или «обработать веб-перехватчики учетной записи». Охватывает весь процесс от создания сеанса до обработки веб-перехватчика. НЕ используйте для вызовов API после связывания (используйте stackone-platform) или создания AI-агентов (используйте stackone-агенты).
