StackOne Connect セッションと Hub React コンポーネントを使用してアカウント リンクを実装します。ユーザーが「プロバイダーの接続」、「統合ピッカーの埋め込み」、「アプリへの BambooHR の追加」、「接続セッションの作成」、「認証リンクのセットアップ」、または「アカウント Webhook の処理」を要求する場合に使用します。セッションの作成から Webhook の処理までの完全なフローをカバーします。リンク (stackone-platform を使用) または AI エージェントの構築 (stackone-agents を使用) 後に API 呼び出しを行うためには使用しないでください。
