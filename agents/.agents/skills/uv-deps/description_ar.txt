يمكنك الحفاظ على حزم Python من خلال عمليات التدقيق الأمني ​​أو تحديثات التبعية باستخدام شجرة عمل git المعزولة والأشعة فوق البنفسجية. يُستخدم لعمليات تدقيق الأمان، وإصلاحات CVE، وعمليات فحص الثغرات الأمنية، وتحديثات التبعية، وترقيات الحزم، والحزم القديمة، والإصدارات الكبيرة، وإصلاح ثغرات Python، والتحقق من وجود Python CVEs، وتدقيق حزم Python، وتحديث تبعيات pyproject.toml، وتحديث Python deps، أو عندما يقوم المستخدم بكتابة /uv-deps مع أو بدون أسماء حزم محددة أو أنماط الكرة الأرضية.
