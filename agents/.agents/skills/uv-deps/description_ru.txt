Поддерживайте пакеты Python посредством аудита безопасности или обновлений зависимостей, используя изолированное рабочее дерево git и uv. Используйте для аудита безопасности, исправлений CVE, проверок уязвимостей, обновлений зависимостей, обновлений пакетов, устаревших пакетов, исправления версий, исправления уязвимостей Python, проверки наличия CVE Python, аудита пакетов Python, обновления зависимостей pyproject.toml, модернизации версий Python или когда пользователь вводит /uv-deps с конкретными именами пакетов или шаблонами glob или без них.
