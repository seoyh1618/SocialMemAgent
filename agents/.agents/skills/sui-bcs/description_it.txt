Aiuta Claude Code a comprendere la codifica BCS (Binary Canonical Serialization) della blockchain di Sui, fornendo linee guida sull'utilizzo ed esempi per tipi primitivi, tipi compositi, tipi specifici di Sui, modelli di serializzazione e supporto per la trasformazione. Da utilizzare quando si lavora con la serializzazione BCS nello sviluppo Sui o quando l'utente menziona BCS, serializzazione binaria o codifica dati Move.
