Aide Claude Code à comprendre l'encodage BCS (Binary Canonical Serialization) de la blockchain Sui, en fournissant des directives d'utilisation et des exemples pour les types primitifs, les types composites, les types spécifiques à Sui, les modèles de sérialisation et la prise en charge de la transformation. À utiliser lorsque vous travaillez avec la sérialisation BCS dans le développement Sui ou lorsque l'utilisateur mentionne BCS, la sérialisation binaire ou le codage de données Move.
