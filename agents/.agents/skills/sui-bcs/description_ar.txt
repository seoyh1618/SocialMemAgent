يساعد Claude Code على فهم تشفير BCS (التسلسل الثنائي الكنسي) الخاص بـ Sui، وتوفير إرشادات الاستخدام وأمثلة للأنواع البدائية، والأنواع المركبة، والأنواع الخاصة بـ Sui، وأنماط التسلسل، ودعم التحويل. يُستخدم عند العمل مع تسلسل BCS في تطوير Sui أو عندما يذكر المستخدم BCS أو التسلسل الثنائي أو نقل ترميز البيانات.
