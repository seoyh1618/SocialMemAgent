Configuration complète du SDK Sentry pour PHP. À utiliser lorsqu'on vous demande « d'ajouter Sentry à PHP », « d'installer Sentry/sentry », de « configurer Sentry en PHP » ou de configurer la surveillance des erreurs, le traçage, le profilage, la journalisation, les métriques ou les crons pour les applications PHP. Prend en charge PHP, Laravel et Symfony.
