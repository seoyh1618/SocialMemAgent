Полная настройка Sentry SDK для PHP. Используйте, когда вас просят «добавить Sentry в PHP», «установить Sentry/sentry», «настроить Sentry в PHP» или настроить мониторинг ошибок, трассировку, профилирование, ведение журнала, метрики или cron для приложений PHP. Поддерживает простой PHP, Laravel и Symfony.
