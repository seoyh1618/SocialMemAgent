Навыки создания и публикации на Xiaohongshu. Используется для: (1) создания контента публикации в стиле Xiaohongshu (заголовок + текст + хэштеги) (2) создания связанных изображений для публикации (3) автоматической загрузки на платформу Xiaohongshu Creator (автоматическая загрузка по умолчанию, с ручным руководством в случае неудачи) (4) создания HTML-файла локального предварительного просмотра. Триггерные слова: 小红书创作, создать красную книгу, 小红书, 红书, 笔记创作, 帖子创作
