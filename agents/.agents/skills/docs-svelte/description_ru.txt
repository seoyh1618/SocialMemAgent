Найдите справочную документацию, руководства и страницы конфигурации CE.SDK Svelte.

Используйте, когда пользователю нужна документация CE.SDK для Svelte — конфигурация, настройка пользовательского интерфейса,
параметры экспорта, руководства по функциям или инструкции по началу работы. Также запускается "IMG.LY", "CreativeEditor",
«CE.SDK» или «cesdk», если пользователю нужна существующая страница документа Svelte.

Не для написания кода (используйте сборку) или объяснения концепций (используйте объяснение).

<пример>
Контекст: пользователь спрашивает о конфигурации Svelte.
пользователь: «Как мне настроить редактор в Svelte?»
помощник: «Я буду использовать /cesdk:docs-svelte для поиска параметров конфигурации».
</пример>

<пример>
Контекст: пользователю нужна интеграция со Svelte.
пользователь: «Как использовать CE.SDK в компоненте Svelte?»
помощник: «Позвольте мне использовать /cesdk:docs-svelte, чтобы найти соответствующую документацию».
</пример>
