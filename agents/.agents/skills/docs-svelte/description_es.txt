Busque documentos de referencia, guías y páginas de configuración de CE.SDK Svelte.

Úselo cuando el usuario necesite documentos CE.SDK para Svelte: configuración, personalización de la interfaz de usuario,
opciones de exportación, guías de funciones o instrucciones de introducción. También activado por "IMG.LY", "CreativeEditor",
"CE.SDK" o "cesdk" cuando el usuario necesita una página de documento Svelte existente.

No para escribir código (use compilación) o explicaciones de conceptos (use explicación).

<ejemplo>
Contexto: El usuario pregunta sobre la configuración de Svelte
usuario: "¿Cómo configuro el editor en Svelte?"
asistente: "Usaré /cesdk:docs-svelte para buscar opciones de configuración".
</ejemplo>
