연구 중심 워크플로 및 자동화된 검증을 통한 전문 기술 및 하위 에이전트 생성.
사용 시기: 새 스킬 또는 하위 에이전트 생성, 기존 스킬 검증, 스킬과 하위 에이전트 간 결정, 문서를 스킬/하위 에이전트로 마이그레이션 또는 개별 검증 도구 실행.
기본 트리거: "기술 생성" = 전체 기술 생성(연구 + 실행 계획 포함 12단계) "하위 에이전트 생성" = 하위 에이전트 생성(템플릿 기반 워크플로 포함 8단계) "기술 검증" = 검증 워크플로(3-8단계) "기술 대 하위 에이전트" = 결정 워크플로(0단계) - "문서를 기술로 변환" 생성 권장 = 마이그레이션 워크플로 "토큰 추정" = 토큰 최적화 "보안 스캔" = 보안 감사
워크플로 규정 준수: 검증 체크포인트가 포함된 구조화된 워크플로입니다. 연구 단계(1c-1d단계)에서는 입증된 접근 방식을 기반으로 기술을 보장합니다.
차별화 요소: 연구 중심 창작. 구축 전 웹 검색(3~5개 쿼리) 다중 제안 생성. 10개의 자동화 스크립트(9개 스킬 + 1개 하위 에이전트). 품질 9.0+/10. 이제 하위 에이전트 생성 지원이 제공됩니다.
재사용됨: Anthropic의 init_skill.py 및 package_skill.py(프로덕션 테스트 완료).
