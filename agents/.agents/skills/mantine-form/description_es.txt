Cree formularios usando @mantine/form en el repositorio mantine-9. Utilice esta habilidad cuando: (1) configure un formulario con useForm, (2) agregue reglas de validación o validación asíncrona, (3) trabaje con objetos anidados o campos de matriz, (4) comparta el estado del formulario entre componentes con createFormContext, (5) use el modo no controlado para el rendimiento, (6) use el enlace useField independiente o (7) cualquier tarea que involucre useForm, getInputProps, onSubmit, insertListItem o validación de formulario.
