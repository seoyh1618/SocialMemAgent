Erstellen Sie eine neue KI-Methode von Grund auf mit dem MTHDS-Standard (.mthds-Bundle-Dateien). Wird verwendet, wenn der Benutzer „Pipeline erstellen“, „Workflow erstellen“, „Neue .mthds-Datei“, „Methode erstellen“, „Pipe entwerfen“ sagt oder eine neue Methode von Grund auf erstellen möchte. Führt den Benutzer durch einen 10-Phasen-Bauprozess.
