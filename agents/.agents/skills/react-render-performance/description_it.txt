Riduci al minimo i re-render di React non necessari quando utilizzi lo stato esterno (XState, @xstate/store, Zustand, Redux, Nanostores, contesto). Preferire abbonamenti basati su selettore rispetto a useState(wholeObject). Da utilizzare quando si ha a che fare con lo stato esterno in React, si ottimizzano i nuovi rendering, si scelgono modelli di stato o si integra con queste librerie.
