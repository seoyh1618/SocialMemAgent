قلل من عمليات إعادة عرض React غير الضرورية عند استهلاك الحالة الخارجية (XState، @xstate/store، Zustand، Redux، Nanostores، context). تفضل الاشتراكات المستندة إلى المحدد على useState(wholeObject). يُستخدم عند التعامل مع الحالة الخارجية في React، أو تحسين عمليات إعادة العرض، أو اختيار أنماط الحالة، أو التكامل مع هذه المكتبات.
