Centre de présentation et d'orientation des compétences de test avancées .NET. Déclenché lorsque les utilisateurs demandent des sujets de tests avancés tels que « tests d'intégration », « tests d'API », « tests en conteneurs », « tests de microservices », « migration de framework de tests », « Conteneurs de test », « tests Aspire », etc. Il recommande des combinaisons de sous-compétences appropriées en fonction de besoins spécifiques, couvrant 8 compétences avancées, notamment les tests d'intégration, les conteneurs de test, les tests Aspire et les mises à niveau du framework.
