Этот навык следует использовать, когда пользователь задает вопросы о китовых позициях Hyperliquid, ликвидациях Hyperliquid, открытом интересе Hyperliquid, аналитике трейдеров Hyperliquid, данных о покупателях Hyperliquid, умных деньгах на Hyperliquid или любом запросе, специфичном для Hyperliquid. Используйте, когда пользователь говорит: «Гиперликвидные киты», «Китовые позиции HL», «Ликвидации HL», «Открытый интерес HL», «Трейдер HL», «умные деньги», «Гиперликвидные киты», «HL鲸鱼», «HL持仓», «HL清算», «HL持仓量», «HL交易员». Для получения общих цен/новостей криптовалют используйте aicoin-market. Для биржевой торговли используйте aicoin-trading. Для Freqtrade используйте aicoin-freqtrade.
