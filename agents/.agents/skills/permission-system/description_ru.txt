Спецификация разработки системы разрешений системы управления чрезвычайными ситуациями. Этот навык необходимо использовать при создании новых функциональных модулей, настройке разрешений меню или настройке разрешений данных. Создание SQL для миграции разрешений меню является обязательным при выполнении команд CX.
