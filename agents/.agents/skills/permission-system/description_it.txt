Specifiche di sviluppo per il sistema di autorizzazione del sistema di gestione delle emergenze. Questa abilità deve essere utilizzata durante la creazione di nuovi moduli funzionali, la configurazione delle autorizzazioni dei menu o l'impostazione delle autorizzazioni dei dati. La generazione dell'SQL di migrazione delle autorizzazioni del menu è obbligatoria durante l'esecuzione dei comandi CX.
