Настройте проект смарт-контракта Solidity с помощью OpenZeppelin Contracts. Используйте, когда пользователям необходимо: (1) создать новый проект Hardhat или Foundry, (2) установить зависимости контрактов OpenZeppelin для Solidity, (3) настроить переназначения для Foundry или (4) понять соглашения об импорте Solidity для OpenZeppelin.
