Configure un proyecto de contrato inteligente de Solidity con OpenZeppelin Contracts. Úselo cuando los usuarios necesiten: (1) crear un nuevo proyecto Hardhat o Foundry, (2) instalar dependencias de OpenZeppelin Contracts para Solidity, (3) configurar reasignaciones para Foundry o (4) comprender las convenciones de importación de Solidity para OpenZeppelin.
