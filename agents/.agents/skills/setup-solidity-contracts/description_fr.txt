Mettre en place un projet de contrat intelligent Solidity avec OpenZeppelin Contracts. À utiliser lorsque les utilisateurs doivent : (1) créer un nouveau projet Hardhat ou Foundry, (2) installer les dépendances OpenZeppelin Contracts pour Solidity, (3) configurer les remappages pour Foundry ou (4) comprendre les conventions d'importation Solidity pour OpenZeppelin.
