Configura un progetto di contratto intelligente Solidity con OpenZeppelin Contracts. Da utilizzare quando gli utenti devono: (1) creare un nuovo progetto Hardhat o Foundry, (2) installare le dipendenze di OpenZeppelin Contracts per Solidity, (3) configurare rimappature per Foundry o (4) comprendere le convenzioni di importazione di Solidity per OpenZeppelin.
