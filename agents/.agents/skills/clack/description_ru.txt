Комплексное руководство по созданию и отладке интерфейсов командной строки JavaScript и TypeScript с помощью @clack/core и @clack/promps, подкрепленное исходным кодом и примерами исходного кода. Используйте при реализации потоков подсказок clack, выборе между основными примитивами и стилизованными подсказками, адаптации примеров clack или проверке использования API clack.
