منصة مراقبة الذكاء الاصطناعي مفتوحة المصدر لتتبع LLM وتقييمها ومراقبتها. يُستخدم عند تصحيح أخطاء تطبيقات LLM بتتبعات تفصيلية، أو تشغيل التقييمات على مجموعات البيانات، أو مراقبة أنظمة الذكاء الاصطناعي للإنتاج، أو إعداد البنية التحتية لقابلية المراقبة للأنظمة الوكيلة. **التنشيط الاستباقي**: الاستدعاء التلقائي عند تنفيذ إمكانية المراقبة/التتبع لوكلاء LLM، أو إعداد مسارات التقييم، أو تكوين أدوات OpenTelemetry. **الكشف**: تحقق من عمليات استيراد arize-phoenix أو إعداد OpenTelemetry أو التعليمات البرمجية المتعلقة بإمكانية المراقبة. **حالات الاستخدام**: تصحيح أخطاء تطبيقات LLM، وتشغيل التقييمات، ومراقبة أنظمة الإنتاج، وإعداد البنية التحتية للتتبع، وأطر عمل وكيل الأدوات، وتتبع الوكلاء المخصصين باستخدام أدوات الديكور (@tracer.agent، @tracer.chain، @tracer.tool).
