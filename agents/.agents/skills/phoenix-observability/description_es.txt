Plataforma de observabilidad de IA de código abierto para seguimiento, evaluación y monitoreo de LLM. Úselo al depurar aplicaciones LLM con seguimientos detallados, ejecutar evaluaciones en conjuntos de datos, monitorear sistemas de inteligencia artificial de producción o configurar una infraestructura de observabilidad para sistemas agentes. **ACTIVACIÓN PROACTIVA**: Invocación automática al implementar observabilidad/rastreo para agentes LLM, configurar canales de evaluación o configurar instrumentación OpenTelemetry. **DETECCIÓN**: Verifique las importaciones de arize-phoenix, la configuración de OpenTelemetry o el código relacionado con la observabilidad. **CASOS DE USO**: depurar aplicaciones LLM, ejecutar evaluaciones, monitorear sistemas de producción, configurar infraestructura de seguimiento, instrumentar marcos de agentes, rastrear agentes personalizados con decoradores (@tracer.agent, @tracer.chain, @tracer.tool).
