Платформа наблюдения за искусственным интеллектом с открытым исходным кодом для отслеживания, оценки и мониторинга LLM. Используйте при отладке приложений LLM с подробными трассировками, проведении оценок наборов данных, мониторинге производственных систем искусственного интеллекта или настройке инфраструктуры наблюдения для агентных систем. **ПРАКТИВНАЯ АКТИВАЦИЯ**: автоматический вызов при реализации наблюдения/отслеживания для агентов LLM, настройке конвейеров оценки или настройке инструментов OpenTelemetry. **ОБНАРУЖЕНИЕ**: проверьте импорт arize-phoenix, настройку OpenTelemetry или код, связанный с наблюдаемостью. **СЛУЧАИ ИСПОЛЬЗОВАНИЯ**: отладка приложений LLM, проведение оценок, мониторинг производственных систем, настройка инфраструктуры отслеживания, инструментирование инфраструктур агентов, отслеживание пользовательских агентов с помощью декораторов (@tracer.agent, @tracer.chain, @tracer.tool).
