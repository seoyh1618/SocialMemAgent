LLM 추적, 평가, 모니터링을 위한 오픈 소스 AI 관찰 플랫폼입니다. 자세한 추적을 통해 LLM 애플리케이션을 디버깅하고, 데이터 세트에 대한 평가를 실행하고, 프로덕션 AI 시스템을 모니터링하거나, 에이전트 시스템에 대한 관찰 가능성 인프라를 설정할 때 사용합니다. **사전 활성화**: LLM 에이전트에 대한 관찰/추적을 구현하거나, 평가 파이프라인을 설정하거나, OpenTelemetry 계측을 구성할 때 자동 호출됩니다. **감지**: arize-phoenix 가져오기, OpenTelemetry 설정 또는 관찰 가능성 관련 코드를 확인하세요. **사용 사례**: LLM 앱 디버깅, 평가 실행, 프로덕션 시스템 모니터링, 추적 인프라 설정, 에이전트 프레임워크 계측, 데코레이터(@tracer.agent, @tracer.chain, @tracer.tool)를 사용하여 사용자 지정 에이전트 추적.
