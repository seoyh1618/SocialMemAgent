Piattaforma di osservabilità AI open source per tracciamento, valutazione e monitoraggio LLM. Da utilizzare durante il debug di applicazioni LLM con tracce dettagliate, l'esecuzione di valutazioni su set di dati, il monitoraggio di sistemi IA di produzione o la configurazione di un'infrastruttura di osservabilità per sistemi ad agenti. **ATTIVAZIONE PROATTIVA**: richiamo automatico durante l'implementazione dell'osservabilità/tracciatura per gli agenti LLM, l'impostazione di pipeline di valutazione o la configurazione della strumentazione OpenTelemetry. **RILEVAMENTO**: verifica la presenza di importazioni di arize-phoenix, configurazione di OpenTelemetry o codice relativo all'osservabilità. **CASI D'USO**: debug di app LLM, esecuzione di valutazioni, monitoraggio dei sistemi di produzione, configurazione dell'infrastruttura di tracciamento, strumentazione dei framework degli agenti, tracciamento degli agenti personalizzati con decoratori (@tracer.agent, @tracer.chain, @tracer.tool).
