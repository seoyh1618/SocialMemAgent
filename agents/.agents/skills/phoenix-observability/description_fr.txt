Plateforme d'observabilité d'IA open source pour le traçage, l'évaluation et le suivi LLM. À utiliser lors du débogage d'applications LLM avec des traces détaillées, de l'exécution d'évaluations sur des ensembles de données, de la surveillance des systèmes d'IA de production ou de la configuration d'une infrastructure d'observabilité pour les systèmes agentiques. **ACTIVATION PROACTIVE** : appel automatique lors de la mise en œuvre de l'observabilité/du traçage pour les agents LLM, de la configuration des pipelines d'évaluation ou de la configuration de l'instrumentation OpenTelemetry. **DÉTECTION** : vérifiez les importations arize-phoenix, la configuration d'OpenTelemetry ou le code lié à l'observabilité. **CAS D'UTILISATION** : Débogage d'applications LLM, exécution d'évaluations, surveillance des systèmes de production, configuration de l'infrastructure de traçage, instrumentation des cadres d'agents, traçage d'agents personnalisés avec des décorateurs (@tracer.agent, @tracer.chain, @tracer.tool).
