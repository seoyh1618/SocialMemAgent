Habilidad de optimización guiada por perfiles para C/C++ con GCC y Clang. Úselo cuando exprima el máximo rendimiento en tiempo de ejecución después de niveles de optimización estándar, implemente compilaciones de PGO de dos etapas, recopile datos de perfil o aplique BOLT para la optimización posterior al enlace. Se activa ante consultas sobre PGO, optimización guiada por perfiles, generación de perfiles, uso de perfiles, compilaciones instrumentadas o BOLT.
