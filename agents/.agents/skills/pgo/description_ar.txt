مهارة التحسين الموجهة بالملف الشخصي لـ C/C++ مع دول مجلس التعاون الخليجي وClang. يُستخدم عند الضغط على الحد الأقصى لأداء وقت التشغيل بعد ثبات التحسين القياسي، أو تنفيذ إنشاءات PGO على مرحلتين، أو جمع بيانات الملف الشخصي، أو تطبيق BOLT لتحسين ما بعد الارتباط. يتم التنشيط عند الاستعلامات حول PGO، أو التحسين الموجه بالملف الشخصي، أو fprofile-generate، أو fprofile-use، أو البنيات الآلية، أو BOLT.
