Capacità di ottimizzazione guidata dal profilo per C/C++ con GCC e Clang. Da utilizzare quando si ottengono le massime prestazioni di runtime dopo livelli di ottimizzazione standard, si implementano build PGO a due fasi, si raccolgono dati di profilo o si applica BOLT per l'ottimizzazione post-collegamento. Si attiva su query su PGO, ottimizzazione guidata dal profilo, fprofile-generate, fprofile-use, build strumentate o BOLT.
