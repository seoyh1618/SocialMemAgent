Modifiez les bundles MTHDS existants (fichiers .mthds). À utiliser lorsque l'utilisateur dit « modifier ce canal », « mettre à jour l'invite », « renommer ce concept », « ajouter une étape », « supprimer ce canal », « modifier le flux de travail », « modifier la méthode », « refactoriser ce pipeline » ou souhaite modifier un fichier .mthds existant. Prend en charge le mode automatique pour des modifications claires et le mode interactif pour des modifications complexes.
