Отредактируйте существующие пакеты MTHDS (файлы .mthds). Используйте, когда пользователь говорит «изменить этот канал», «обновить приглашение», «переименовать эту концепцию», «добавить шаг», «удалить этот канал», «изменить рабочий процесс», «изменить метод», «реорганизовать этот конвейер» или хочет внести какие-либо изменения в существующий файл .mthds. Поддерживает автоматический режим для четких изменений и интерактивный режим для сложных модификаций.
