Bearbeiten Sie vorhandene MTHDS-Bundles (.mthds-Dateien). Wird verwendet, wenn der Benutzer sagt „diese Pipe ändern“, „die Eingabeaufforderung aktualisieren“, „dieses Konzept umbenennen“, „einen Schritt hinzufügen“, „diese Pipe entfernen“, „den Workflow ändern“, „die Methode ändern“, „diese Pipeline umgestalten“ oder eine Änderung an einer vorhandenen .mthds-Datei wünscht. Unterstützt den automatischen Modus für klare Änderungen und den interaktiven Modus für komplexe Änderungen.
