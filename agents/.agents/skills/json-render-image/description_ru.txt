Средство рендеринга изображений для json-рендеринга, которое превращает спецификации JSON в изображения SVG и PNG с помощью Satori. Используйте при работе с @json-render/image, создании изображений OG из JSON, создании социальных карточек или рендеринге спецификаций изображений, созданных искусственным интеллектом.
