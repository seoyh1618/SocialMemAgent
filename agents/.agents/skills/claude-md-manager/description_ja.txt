プロジェクトの Claude Code を構成する CLAUDE.md ドキュメント ファイルを作成、監査、および保守します。このスキルは、(1) Claude Code 構成を使用して新しいプロジェクトを初期化する場合、(2) 既存の CLAUDE.md ファイルをレビューまたは改善する場合、(3) 段階的な開示パターンを使用してプロジェクトの指示を整理する場合、(4) 繰り返される指示を永続的なドキュメントに変換する場合、または (5) より大きなコードベース用に Agent_docs/ 構造をセットアップする場合に使用します。 WHAT/WHY/HOW フレームワーク、簡潔性の最適化、ファイル インポート パターンを処理します。
