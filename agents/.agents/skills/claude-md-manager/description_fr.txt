Créez, auditez et maintenez les fichiers de documentation CLAUDE.md qui configurent Claude Code pour les projets. Utilisez cette compétence lors de (1) l'initialisation d'un nouveau projet avec la configuration de Claude Code, (2) la révision ou l'amélioration des fichiers CLAUDE.md existants, (3) l'organisation des instructions de projet à l'aide de modèles de divulgation progressive, (4) la conversion d'instructions répétées en documentation permanente ou (5) la configuration de agents_docs/structures pour des bases de code plus volumineuses. Gère le cadre QUOI/POURQUOI/COMMENT, l’optimisation de la concision et les modèles d’importation de fichiers.
