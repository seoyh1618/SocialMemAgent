Создавайте, проверяйте и обслуживайте файлы документации CLAUDE.md, которые настраивают Claude Code для проектов. Используйте этот навык при (1) инициализации нового проекта с конфигурацией Claude Code, (2) просмотре или улучшении существующих файлов CLAUDE.md, (3) организации инструкций проекта с использованием шаблонов постепенного раскрытия, (4) преобразовании повторяющихся инструкций в постоянную документацию или (5) настройке структур Agent_docs/ для более крупных баз кода. Управляет структурой ЧТО/ПОЧЕМУ/КАК, оптимизацией краткости и шаблонами импорта файлов.
