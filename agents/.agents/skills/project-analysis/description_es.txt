Herramienta de análisis de proyectos para analizar la arquitectura del sistema y el flujo de datos del módulo de una base de código. Utilice esta habilidad cuando los usuarios necesiten comprender la estructura del proyecto, generar diagramas de arquitectura, analizar el flujo de datos entre módulos o producir diagramas de secuencia. Genera gráficos visuales en sintaxis de sirena. Escenarios aplicables: (1) revisión de la arquitectura del proyecto, (2) análisis de dependencia del módulo, (3) seguimiento del flujo de datos, (4) incorporación de nuevos miembros al equipo, (5) generación de documentación técnica
