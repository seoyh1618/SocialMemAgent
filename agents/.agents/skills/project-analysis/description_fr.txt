Outil d'analyse de projet pour analyser l'architecture système et le flux de données des modules d'une base de code. Utilisez cette compétence lorsque les utilisateurs ont besoin de comprendre la structure du projet, de générer des diagrammes d'architecture, d'analyser le flux de données entre les modules ou de produire des diagrammes de séquence. Produit des graphiques visuels dans la syntaxe Mermaid. Scénarios applicables : (1) examen de l'architecture du projet, (2) analyse des dépendances des modules, (3) suivi du flux de données, (4) intégration de nouveaux membres de l'équipe, (5) génération de documentation technique
