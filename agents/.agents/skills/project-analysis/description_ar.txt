أداة تحليل المشروع لتحليل بنية النظام وتدفق بيانات الوحدة لقاعدة التعليمات البرمجية. استخدم هذه المهارة عندما يحتاج المستخدمون إلى فهم بنية المشروع، أو إنشاء مخططات معمارية، أو تحليل تدفق البيانات بين الوحدات، أو إنتاج مخططات تسلسلية. إخراج المخططات المرئية في بناء جملة حورية البحر. السيناريوهات القابلة للتطبيق: (1) مراجعة بنية المشروع، (2) تحليل تبعية الوحدة، (3) تتبع تدفق البيانات، (4) تأهيل أعضاء الفريق الجدد، (5) إنشاء الوثائق الفنية
