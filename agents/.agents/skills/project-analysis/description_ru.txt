Инструмент анализа проекта для анализа системной архитектуры и потока данных модулей кодовой базы. Используйте этот навык, когда пользователям необходимо понять структуру проекта, создать диаграммы архитектуры, проанализировать поток данных между модулями или создать диаграммы последовательности. Выводит визуальные диаграммы в синтаксисе Mermaid. Применимые сценарии: (1) анализ архитектуры проекта, (2) анализ зависимостей модулей, (3) отслеживание потока данных, (4) привлечение новых членов команды, (5) создание технической документации.
