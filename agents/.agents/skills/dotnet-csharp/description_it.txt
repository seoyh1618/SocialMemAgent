Competenza C# di base caricata per ogni percorso di codice .NET. Guida i modelli linguistici (record, corrispondenza di modelli, costruttori primari, C# 8-15), standard di codifica, async/await, DI, LINQ, serializzazione, modellazione di domini, concorrenza, analizzatori Roslyn, globalizzazione, interoperabilità nativa (P/Invoke, LibraryImport, ComWrappers), interoperabilità WASM (JSImport/JSExport) e progettazione del tipo. Comprende 25 argomenti. Non usare per l'architettura dell'endpoint ASP.NET, i modelli del framework dell'interfaccia utente o le indicazioni CI/CD.
