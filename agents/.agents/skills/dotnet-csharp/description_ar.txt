تم تحميل مهارة C# الأساسية لكل مسار كود .NET. يرشد أنماط اللغة (السجلات، مطابقة الأنماط، المنشئات الأساسية، C# 8-15)، معايير الترميز، غير المتزامن/الانتظار، DI، LINQ، التسلسل، نمذجة المجال، التزامن، محللات Roslyn، العولمة، التشغيل المتداخل الأصلي (P/Invoc، LibraryImport، ComWrappers)، التشغيل المتداخل WASM (JSImport/JSExport)، وتصميم النوع. يمتد 25 المواضيع. لا تستخدمه مع بنية نقطة نهاية ASP.NET، أو أنماط إطار عمل واجهة المستخدم، أو إرشادات CI/CD.
