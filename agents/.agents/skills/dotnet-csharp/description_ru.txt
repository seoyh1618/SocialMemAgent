Базовые навыки C# загружаются для каждого пути кода .NET. Управляет языковыми шаблонами (записи, сопоставление шаблонов, первичные конструкторы, C# 8–15), стандартами кодирования, async/await, DI, LINQ, сериализацией, моделированием предметной области, параллелизмом, анализаторами Roslyn, глобализацией, собственным взаимодействием (P/Invoke, LibraryImport, ComWrappers), взаимодействием WASM (JSImport/JSExport) и дизайном типов. Охватывает 25 тем. Не используйте для архитектуры конечных точек ASP.NET, шаблонов инфраструктуры пользовательского интерфейса или рекомендаций CI/CD.
