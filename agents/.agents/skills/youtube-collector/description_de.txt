Eine Fähigkeit, die YouTube-Kanäle registriert, neue Inhalte sammelt und untertitelbasierte Zusammenfassungen generiert. Wird verwendet, wenn der Benutzer (1) die Registrierung/Verwaltung eines YouTube-Kanals anfordert, (2) die Sammlung neuer Videos von registrierten Kanälen anfordert oder (3) eine Zusammenfassung eines YouTube-Videos anfordert. Die Daten werden im YAML-Format im Ordner .reference/ gespeichert.
