Herramientas locales de Node.js para la recopilación y el procesamiento de conocimientos web: busque archivos PDF e imágenes, descargue y valide archivos, convierta archivos PDF a Markdown estructurado y genere voz localmente con Supertone Supertonic ONNX Runtime. Úselo cuando Codex necesite búsqueda sin clave API, descargas seguras de archivos, extracción de texto PDF o TTS en el dispositivo (con una descarga única del modelo en ./models/supertonic-2/).
