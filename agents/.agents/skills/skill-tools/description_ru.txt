Локальные инструменты Node.js для сбора и обработки веб-данных: поиск PDF-файлов и изображений, потоковая загрузка и проверка файлов, преобразование PDF-файлов в структурированный Markdown и локальное генерирование речи с помощью Supertone Supertonic ONNX Runtime. Используйте, когда Кодексу требуется поиск без ключей API, безопасная загрузка файлов, извлечение текста PDF или TTS на устройстве (с однократной загрузкой модели в ./models/supertonic-2/).
