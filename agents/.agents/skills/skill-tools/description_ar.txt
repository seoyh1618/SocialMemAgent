أدوات Node.js المحلية لجمع ومعالجة المعرفة على الويب: البحث في ملفات PDF والصور، وتنزيل الملفات والتحقق من صحتها، وتحويل ملفات PDF إلى Markdown منظم، وإنشاء الكلام محليًا باستخدام Supertone Supertonic ONNX Runtime. يُستخدم عندما لا يحتاج Codex إلى البحث عن مفتاح API، أو تنزيلات الملفات الآمنة، أو استخراج نص PDF، أو TTS على الجهاز (مع تنزيل نموذج لمرة واحدة إلى ./models/supertonic-2/).
