Examine las acciones con brecha posterior a las ganancias para detectar patrones PEAD (desviación del anuncio posterior a las ganancias). Analiza la formación de velas semanales para detectar retrocesos de velas rojas y señales de ruptura. Admite dos modos de entrada: calendario de ganancias FMP (Modo A) o salida JSON del analizador de transacciones de ganancias (Modo B). Úselo cuando el usuario pregunte sobre la evaluación de PEAD, la deriva posterior a las ganancias, el seguimiento de la brecha de ganancias, los patrones de ruptura de velas rojas o las configuraciones del impulso de las ganancias semanales.
