Guide for orchestrating and controlling Claude Code agent teams. Wird verwendet, wenn Sie in Claude Code mit der Koordination mehrerer Agententeams arbeiten oder diese verwalten, einschließlich: (1) Erstellen und Starten von Agententeams, (2) Steuern von Teamkollegen (Anzeigemodi, Aufgabenzuweisung, Delegation), (3) Best Practices für paralleles Arbeiten, (4) Beheben von Teamproblemen. Covers concepts like team lead, teammates, shared task lists, and inter-agent messaging.
