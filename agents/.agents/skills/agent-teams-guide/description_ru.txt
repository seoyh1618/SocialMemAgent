Руководство по организации и контролю команд агентов Claude Code. Используйте при работе с координацией многоагентной команды или управлении ею в Claude Code, включая: (1) Создание и запуск групп агентов, (2) Управление товарищами по команде (режимы отображения, назначение задач, делегирование), (3) Рекомендации по параллельной работе, (4) Устранение неполадок в команде. Охватывает такие понятия, как руководство командой, товарищи по команде, общие списки задач и межагентский обмен сообщениями.
