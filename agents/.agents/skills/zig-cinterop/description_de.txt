Zig C-Interoperabilitätsfähigkeit. Wird verwendet, wenn Sie C aus Zig aufrufen, Zig aus C aufrufen, @cImport und @cInclude verwenden, Translate-c auf C-Headern ausführen, externe Strukturen und gepackte Strukturen definieren, C-ABI-Typen abgleichen oder gemischte C/Zig-Projekte erstellen. Aktiviert bei Abfragen zu @cImport, @cInclude, Translate-C, externer Struktur, gepackter Struktur, Zig C ABI, Aufruf von C aus Zig, Export von Zig nach C oder Bindungsäquivalenten.
