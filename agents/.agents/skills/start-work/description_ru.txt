Ежедневный помощник по запуску работы. Читает почтовый ящик Obsidian и файлы планировщика, напоминает о сегодняшних задачах, спрашивает о планах создания контента, показывает еженедельный прогресс, выбирает актуальные темы. Триггеры: «начать работу», «новый день», «что сделать сегодня». Помогает пользователям быстро войти в продуктивное состояние.
