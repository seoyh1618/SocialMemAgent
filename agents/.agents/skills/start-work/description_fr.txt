Assistant de démarrage du travail quotidien. Lit la boîte de réception et les fichiers du planificateur Obsidian, rappelle les tâches du jour, pose des questions sur les plans de création de contenu, affiche les progrès hebdomadaires, récupère les sujets d'actualité. Déclencheurs : « commencer le travail », « nouveau jour », « que faire aujourd'hui ». Aide les utilisateurs à entrer rapidement dans un état productif.
