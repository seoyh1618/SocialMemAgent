Собственные шаблоны диалогов HTML для Rails с Turbo и Stimulus. Используйте при создании модальных окон, подтверждений, оповещений или любого наложенного пользовательского интерфейса. Запускается по шаблонам модальных, диалоговых, всплывающих окон, подтверждений, предупреждений или всплывающих уведомлений.
