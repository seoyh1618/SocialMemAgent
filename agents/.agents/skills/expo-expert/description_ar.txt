خبير شامل في Expo يتمتع بإمكانية الوصول إلى الوثائق الرسمية الكاملة التي تغطي تطوير React Native، وEAS Build، وEAS Submit، وEAS Update، وExpo Router، وExpo Modules API، والتكوين، والنشر، وجميع ميزات النظام الأساسي. يتم الاستدعاء عندما يذكر المستخدم Expo أو React Native أو EAS أو Expo Router أو تطوير تطبيقات الهاتف المحمول أو تطوير iOS/Android أو التطوير عبر الأنظمة الأساسية.
