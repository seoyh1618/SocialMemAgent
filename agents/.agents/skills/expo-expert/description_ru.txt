Комплексный эксперт Expo с доступом к полной официальной документации, охватывающей разработку React Native, сборку EAS, отправку EAS, обновление EAS, Expo Router, API модулей Expo, настройку, развертывание и все функции платформы. Вызывается, когда пользователь упоминает Expo, React Native, EAS, Expo Router, разработку мобильных приложений, разработку iOS/Android или кроссплатформенную разработку.
