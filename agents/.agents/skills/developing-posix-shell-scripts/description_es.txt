Directrices para escribir, revisar y refactorizar scripts de shell estrictamente compatibles con POSIX (/bin/sh). No se permiten funciones de Bash. Clasifica el script como simple o complejo y luego lo delega al documento de referencia apropiado. Úselo cuando apunte a /bin/sh, Alpine, BusyBox, sistemas integrados o cualquier entorno donde no se pueda asumir Bash.
