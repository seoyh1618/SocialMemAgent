Lignes directrices pour l'écriture, la révision et la refactorisation de scripts shell strictement conformes à POSIX (/bin/sh). Aucune fonctionnalité Bash autorisée. Classifie le script comme simple ou complexe, puis le délègue au document de référence approprié. À utiliser pour cibler /bin/sh, Alpine, BusyBox, les systèmes embarqués ou tout environnement dans lequel Bash ne peut pas être supposé.
