WebMCP — واجهة برمجة تطبيقات من جانب المتصفح تتيح لتطبيقات الويب الكشف عن وظائفها الخاصة كأدوات MCP لوكلاء الذكاء الاصطناعي. يُستخدم عند التصميم أو التكامل مع تطبيقات الويب التي تعرض إجراءات واجهة المستخدم (النماذج والأزرار واستعلامات البيانات) كأدوات وكيل قابلة للاستدعاء. ليس مخصصًا لكشط الويب أو جلب الصفحات الخارجية.
