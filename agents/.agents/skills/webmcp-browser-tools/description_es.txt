WebMCP: API del lado del navegador que permite que las aplicaciones web expongan su propia funcionalidad como herramientas MCP A los agentes de IA. Úselo al diseñar o integrar con aplicaciones web que muestran acciones de la interfaz de usuario (formularios, botones, consultas de datos) como herramientas de agentes invocables. NO para raspado web o recuperación de páginas externas.
