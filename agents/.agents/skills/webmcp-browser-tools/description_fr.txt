WebMCP — API côté navigateur qui permet aux applications Web d'exposer leurs propres fonctionnalités en tant qu'outils MCP AUX agents IA. À utiliser lors de la conception ou de l'intégration d'applications Web qui présentent des actions d'interface utilisateur (formulaires, boutons, requêtes de données) en tant qu'outils d'agent appelables. PAS pour le scraping Web ou la récupération de pages externes.
