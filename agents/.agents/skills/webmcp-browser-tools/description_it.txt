WebMCP: API lato browser che consente alle applicazioni Web di esporre le proprie funzionalità come strumenti MCP agli agenti AI. Da utilizzare durante la progettazione o l'integrazione con app Web che presentano azioni dell'interfaccia utente (moduli, pulsanti, query di dati) come strumenti agente richiamabili. NON per web scraping o recupero di pagine esterne.
