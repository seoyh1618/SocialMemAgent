WebMCP — API на стороне браузера, который позволяет веб-приложениям предоставлять агентам ИИ свои собственные функции в качестве инструментов MCP. Используйте при разработке или интеграции с веб-приложениями, которые отображают действия пользовательского интерфейса (формы, кнопки, запросы данных) как вызываемые инструменты агента. НЕ для парсинга веб-страниц или получения внешних страниц.
