Erstellen Sie durch ausführliche, progressive Interviews ein KI-kollaboratives Persönlichkeitsprofildokument für den Benutzer.
Erstellen Sie eine Beschreibungsdatei „Über mich“, die der Benutzer jedem KI-Agenten übergeben kann, damit die KI den Benutzer wirklich verstehen und proaktiv Hilfe anbieten kann.
Triggerphrasen:
- „Interviewen Sie mich“, „Chatten Sie mit mir“, „Lernen Sie mich kennen“
- „Mein Profil erstellen“, „Eine Beschreibung über mich schreiben“
- „Hilf der KI, mich zu verstehen“, „Schreibe eine Bedienungsanleitung über mich für die KI“
- „Persona-Interview“, „Über mich“
Hinweis: Dies ist kein einfacher Fragebogen oder eine Übung zum Ausfüllen von Formularen. Es handelt sich um ein echtes, ausführliches Interview, das eine schichtweise Erkundung wie ein professioneller Journalist erfordert.
