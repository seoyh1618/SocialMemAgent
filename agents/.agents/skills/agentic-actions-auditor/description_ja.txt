Claude Code Action、Gemini CLI、OpenAI Codex、GitHub AI Inference などの AI エージェント統合におけるセキュリティ脆弱性について GitHub Actions ワークフローを監査します。攻撃者が制御する入力が CI/CD パイプラインで実行されている AI エージェントに到達する攻撃ベクトル (env var 中間パターン、直接式インジェクション、危険なサンドボックス構成、ワイルドカード ユーザー ホワイトリストなど) を検出します。 AI コーディング エージェントを呼び出すワークフロー ファイルを確認する場合、プロンプト インジェクションのリスクについて CI/CD パイプライン セキュリティを監査する場合、またはエージェント アクション構成を評価する場合に使用します。
