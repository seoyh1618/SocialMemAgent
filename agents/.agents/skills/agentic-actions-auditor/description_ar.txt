تدقيق سير عمل GitHub Actions بحثًا عن الثغرات الأمنية في عمليات تكامل وكيل الذكاء الاصطناعي بما في ذلك Claude Code Action وGemini CLI وOpenAI Codex وGitHub AI Inference. يكتشف متجهات الهجوم حيث تصل المدخلات التي يتحكم فيها المهاجم إلى وكلاء الذكاء الاصطناعي الذين يعملون في مسارات CI/CD، بما في ذلك أنماط الوسيط env var، وحقن التعبير المباشر، وتكوينات وضع الحماية الخطيرة، والقوائم المسموح بها لمستخدم أحرف البدل. يُستخدم عند مراجعة ملفات سير العمل التي تستدعي وكلاء ترميز الذكاء الاصطناعي، أو تدقيق أمان خط أنابيب CI/CD لمخاطر الحقن الفوري، أو تقييم تكوينات الإجراء الوكيل.
