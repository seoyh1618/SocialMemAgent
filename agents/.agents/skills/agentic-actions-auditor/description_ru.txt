Аудит рабочих процессов GitHub Actions на наличие уязвимостей безопасности в интеграциях агентов искусственного интеллекта, включая Claude Code Action, Gemini CLI, OpenAI Codex и GitHub AI Inference. Обнаруживает векторы атак, при которых входные данные, контролируемые злоумышленником, достигают агентов ИИ, работающих в конвейерах CI/CD, включая промежуточные шаблоны env var, прямое внедрение выражений, опасные конфигурации песочницы и белые списки пользователей с подстановочными знаками. Используйте при проверке файлов рабочего процесса, которые вызывают агенты кодирования AI, аудите безопасности конвейера CI/CD на предмет рисков быстрого внедрения или оценке конфигураций агентных действий.
