Claude Code Action, Gemini CLI, OpenAI Codex 및 GitHub AI Inference를 포함한 AI 에이전트 통합의 보안 취약성에 대한 GitHub Actions 워크플로를 감사합니다. 환경 변수 중간 패턴, 직접 표현 삽입, 위험한 샌드박스 구성, 와일드카드 사용자 허용 목록을 포함하여 공격자가 제어하는 ​​입력이 CI/CD 파이프라인에서 실행되는 AI 에이전트에 도달하는 공격 벡터를 탐지합니다. AI 코딩 에이전트를 호출하는 워크플로 파일을 검토하거나, 신속한 삽입 위험에 대한 CI/CD 파이프라인 보안을 감사하거나, 에이전트 작업 구성을 평가할 때 사용합니다.
