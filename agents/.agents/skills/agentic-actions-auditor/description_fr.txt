Audite les workflows GitHub Actions pour détecter les vulnérabilités de sécurité dans les intégrations d'agents IA, notamment Claude Code Action, Gemini CLI, OpenAI Codex et GitHub AI Inference. Détecte les vecteurs d'attaque où les entrées contrôlées par l'attaquant atteignent les agents IA exécutés dans les pipelines CI/CD, y compris les modèles intermédiaires de variables d'environnement, l'injection d'expression directe, les configurations de bac à sable dangereuses et les listes autorisées d'utilisateurs génériques. À utiliser lors de l'examen des fichiers de flux de travail qui appellent des agents de codage IA, de l'audit de la sécurité du pipeline CI/CD pour détecter les risques d'injection rapide ou de l'évaluation des configurations d'actions d'agent.
