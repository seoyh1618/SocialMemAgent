Audita los flujos de trabajo de GitHub Actions en busca de vulnerabilidades de seguridad en las integraciones de agentes de IA, incluidas Claude Code Action, Gemini CLI, OpenAI Codex y GitHub AI Inference. Detecta vectores de ataque donde la entrada controlada por el atacante llega a los agentes de IA que se ejecutan en canalizaciones de CI/CD, incluidos patrones intermediarios de env var, inyección directa de expresiones, configuraciones peligrosas de sandbox y listas de usuarios permitidos con comodines. Úselo al revisar archivos de flujo de trabajo que invocan agentes de codificación de IA, auditar la seguridad de la canalización de CI/CD para detectar riesgos de inyección rápida o evaluar configuraciones de acciones de agentes.
