Vérifiez les contrats intelligents sur les explorateurs de blocs compatibles Etherscan. Cette compétence doit être utilisée lorsque l'utilisateur demande de « vérifier le contrat », « vérifier sur etherscan », « vérifier sur chaîne scan ». Gère la vérification standard, l'API Etherscan V2 pour les chaînes non prises en charge sur la fonderie, les modèles de proxy et les contrats créés en usine.
