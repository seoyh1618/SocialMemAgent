Проверьте смарт-контракты в обозревателях блоков, совместимых с Etherscan. Этот навык следует использовать, когда пользователь просит «проверить контракт», «проверить на etherscan», «проверить на сканировании цепочки». Выполняет стандартную проверку, API Etherscan V2 для неподдерживаемых цепочек в литейном производстве, шаблоны прокси и контракты, созданные на заводе.
