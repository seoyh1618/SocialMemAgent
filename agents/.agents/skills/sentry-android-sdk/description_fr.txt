Configuration complète du SDK Sentry pour Android. À utiliser lorsqu'on vous demande « d'ajouter Sentry à Android », « d'installer Sentry-Android », de « configurer Sentry sur Android » ou de configurer la surveillance des erreurs, le traçage, le profilage, la relecture de session ou la journalisation pour les applications Android. Prend en charge les bases de code Kotlin et Java.
