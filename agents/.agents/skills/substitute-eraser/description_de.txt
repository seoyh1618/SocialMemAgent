Diese Fähigkeit sollte verwendet werden, wenn der Benutzer auffordert, „nach TODOs zu suchen“, „Platzhalter zu finden“, „Stubs zu bereinigen“, „temporären Code zu entfernen“, „auf unvollständigen Code zu prüfen“ oder „Ersetzungen aus der Codebasis zu löschen“. Durchsucht vorhandene Dateien nach Platzhaltertokens und generiert einen Behebungsplan.
