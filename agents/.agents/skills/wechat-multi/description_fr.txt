Outil multi-lancement macOS WeChat (Little Green Book). Crée plusieurs instances en dupliquant WeChat.app, en modifiant son ID de bundle et en re-signant. Utilisé lorsque l'utilisateur dit « ouvrir deux WeChat », « WeChat double ouverture », « WeChat multi-ouverture », « ouvrir un autre WeChat », « reconstruire après la mise à jour de WeChat », « réparer WeChat multi-ouverture », « fermer tous les WeChat », « Petit livre vert », etc. macOS uniquement ; script situé sur scripts/wechat-multi.sh.
