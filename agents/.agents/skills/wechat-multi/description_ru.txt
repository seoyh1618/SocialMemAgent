Инструмент мультизапуска macOS WeChat (Маленькая зеленая книга). Достигает нескольких экземпляров путем копирования WeChat.app, изменения идентификатора пакета и повторной подписи. Используйте, когда пользователь говорит: «открыть два WeChat», «двойное открытие WeChat», «множественное открытие WeChat», «открыть еще один WeChat», «перестроить после обновления WeChat», «исправить многократное открытие WeChat», «закрыть все WeChat», «Маленькая зеленая книга». Эксклюзивно для macOS, скрипт находится по адресу scripts/wechat-multi.sh.
