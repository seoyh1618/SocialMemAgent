Verwenden Sie diese Option, wenn Sie eine Bereitstellungsstrategie auswählen, Rollback-Verfahren implementieren, Integritätsprüfungen konfigurieren, Pipelines zur Umgebungsförderung einrichten, eine progressive Bereitstellung mit Feature-Flags planen oder den Explosionsradius während Releases reduzieren.
