Diagnostique y recupere problemas de sesión de Claude Code, incluidas conversaciones perdidas, sesiones dañadas y problemas con archivos de sesión. Úselo siempre que el usuario vea "No se encontraron conversaciones para reanudar", las sesiones aparecen en ubicaciones incorrectas o los archivos de sesión faltan o están dañados. NO lo use para la configuración general de Claude Code o para problemas no relacionados con la persistencia y recuperación de la sesión.
