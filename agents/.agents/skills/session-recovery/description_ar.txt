تشخيص واستعادة مشكلات جلسة Claude Code بما في ذلك المحادثات المفقودة والجلسات التالفة ومشكلات ملفات الجلسة. يُستخدم عندما يرى المستخدم رسالة "لم يتم العثور على محادثات لاستئنافها"، أو تظهر الجلسات في مواقع خاطئة، أو تكون ملفات الجلسة مفقودة أو تالفة. لا تستخدمه لتكوين Claude Code العام أو للمشكلات التي لا علاقة لها باستمرار الجلسة واستعادتها.
