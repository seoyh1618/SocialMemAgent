Создавайте профессиональные еженедельные отчеты о работе на основе истории коммитов Git. Поддерживает создание пакетов на одну или несколько недель, агрегацию нескольких проектов и пользовательские шаблоны (Markdown/Word). Автоматически переводит технические термины на деловой язык и добавляет разделы для планов на следующую неделю/еженедельных сводок/проблем и рисков. Варианты использования: (1) Создание отчетов о работе за текущую неделю из журналов Git (2) Объединение записей о фиксации из нескольких проектов в один еженедельный отчет (3) Создание требуемых руководителем форматов на основе пользовательских шаблонов Word (4) Пакетное создание отчетов за месяц или несколько недель
