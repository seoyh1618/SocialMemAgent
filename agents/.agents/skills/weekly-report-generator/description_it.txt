Genera report di lavoro settimanali professionali dalla cronologia dei commit Git. Supporta la generazione batch di una o più settimane, l'aggregazione di più progetti e modelli personalizzati (Markdown/Word). Traduce automaticamente i termini tecnici nel linguaggio aziendale e aggiunge sezioni per piani/riepiloghi settimanali/problemi e rischi della settimana successiva. Casi d'uso: (1) Genera report di lavoro della settimana corrente dai log Git (2) Combina record di commit da più progetti in un report settimanale (3) Crea formati richiesti dal leader basati su modelli Word personalizzati (4) Genera report in batch per un mese o più settimane
