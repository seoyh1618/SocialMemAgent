AI Task Drill — помощник по разбивке задач, который помогает людям разлагать задачи по типам, определяя, какие задачи должны быть переданы ИИ, а какие должны оставаться под контролем человека. Поддерживает четыре типа задач: прямое решение проблем, прямое создание выходных данных, совместное решение проблем и совместное создание выходных данных, предоставляя подробные рабочие процессы и быстрые стратегии для каждого. Используйте этот навык всякий раз, когда пользователь представляет какую-либо задачу, нуждается в помощи в планировании или запрашивает разбивку работы.
