قم بإنشاء مستند تصميم فني موحد يتضمن أقسامًا كاملة لتصميم قاعدة البيانات وتصميم البنية وتصميم الواجهة والعمليات التجارية.  
استخدم Mermaid لرسم مخططات ER ومخططات انسيابية. ينطبق على: (1) إنشاء خطط فنية للميزات الجديدة، (2) توحيد الوثائق الفنية الموجودة، (3) إنشاء مستندات التصميم التي تلبي معايير الفريق.  
الكلمات الدافعة: الخطة الفنية، وثيقة التصميم، تصميم قاعدة البيانات، تصميم الواجهة، التصميم الفني.
