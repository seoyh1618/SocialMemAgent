Создайте стандартизированный документ технического проекта, включающий полные разделы по проектированию базы данных, проектированию архитектуры, проектированию интерфейса и бизнес-процессам.  
Используйте Mermaid для рисования диаграмм и блок-схем ER. Подходит для: (1) создания технических планов для новых функций, (2) стандартизации существующей технической документации, (3) создания проектной документации, соответствующей стандартам команды.  
Триггерные слова: технический план, проектная документация, проект базы данных, проект интерфейса, технический проект.
