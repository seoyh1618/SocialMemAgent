Создайте гибридные облачные тесты для кодовой базы Sentry. Используйте, когда вас просят «создать тест HC», «создать тест гибридного облака», «записать тест HC», «добавить тест HC», «записать тест RPC», «тестировать службу RPC», «изолированный тест», «межузловой тест», «исходящий тест», «тест шлюза API» или «изолированный тест конечной точки». Охватывает тесты служб RPC, тесты шлюза API, тесты шаблонов исходящих сообщений и тесты конечных точек API с декораторами изолированных модулей.
