Steuern Sie den echten Safari-Browser des Benutzers auf macOS mithilfe von AppleScript und Screencapture. Diese Fähigkeit sollte verwendet werden, wenn der Benutzer mit Safari interagieren, Websites durchsuchen, Webseiten lesen, Browseraufgaben automatisieren, Screenshots von Webinhalten erstellen möchte oder wenn eine andere Aufgabe von der Anzeige oder Interaktion mit dem Browserinhalt profitieren würde. Löst auf Schlüsselwörtern wie „Safari“, „Browser“, „Webseite“, „Tab öffnen“, „Screenshot der Seite“, „Diese Website lesen“, „Durchsuchen“, „Klicken auf“ und „Formular ausfüllen“ aus.
