Управляйте настоящим браузером Safari пользователя на macOS с помощью AppleScript и снимков экрана. Этот навык следует использовать, когда пользователь просит взаимодействовать с Safari, просматривать веб-сайты, читать веб-страницы, автоматизировать задачи браузера, делать снимки экрана веб-контента или когда для любой задачи будет полезно просмотреть или взаимодействовать с тем, что находится в его браузере. Запускается по таким ключевым словам, как «safari», «браузер», «веб-страница», «открыть вкладку», «снимок экрана страницы», «прочитать этот сайт», «просмотреть», «нажать», «заполнить форму».
