# Emoji Translator 🔤➡️😀

A fun skill that converts regular text into emoji representations, making communication more visual and playful by replacing words with their corresponding emoji symbols.

## When to Apply

Reference these guidelines when:
- The user asks to translate text to emojis
- Converting messages or phrases to emoji format
- Making text more visual and expressive
- Creating emoji-heavy social media content
