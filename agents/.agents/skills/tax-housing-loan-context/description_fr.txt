Contexte de base pour le crédit d'impôt sur les prêts au logement (住宅ローン控除) dans le plugin de déclaration de revenus Shinkoku. Contient les conditions d'éligibilité, les limites de crédit, les règles de calcul et l'interaction avec furusato-nozei pour l'année fiscale en cours. Cette compétence n'est pas invocable par l'utilisateur : Claude la charge automatiquement lorsqu'il répond à des questions ou à des calculs de crédit d'impôt pour prêt immobilier.
