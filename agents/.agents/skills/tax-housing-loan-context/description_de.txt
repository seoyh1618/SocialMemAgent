Hintergrundkontext für die Steuergutschrift für Wohnbaudarlehen (住宅ローン控除) im Shinkoku-Steuererklärungs-Plugin. Enthält Berechtigungsvoraussetzungen, Kreditlimits, Berechnungsregeln und Interaktion mit furusato-nozei für das laufende Steuerjahr. Diese Fähigkeit kann nicht vom Benutzer aufgerufen werden – Claude lädt sie automatisch, wenn er auf Fragen oder Berechnungen zur Steuergutschrift für Wohnbaudarlehen antwortet.
