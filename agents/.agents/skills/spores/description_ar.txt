قم بإدارة قوائم تشغيل Spotify، واحفظ العناصر في مكتبتك، وابحث في كتالوج Spotify باستخدام Spores Rust CLI. استخدم هذه المهارة عندما يطلب المستخدم إنشاء قوائم تشغيل Spotify أو إدراجها أو فحصها أو تعديلها، أو إضافة مسارات إلى قوائم التشغيل، أو حفظ المسارات/الألبومات/قوائم التشغيل في المكتبة، أو البحث عن المسارات أو الألبومات أو الفنانين أو قوائم التشغيل على Spotify. مشغلات لطلبات مثل "إنشاء قائمة تشغيل Spotify"، أو "إضافة الأغاني إلى قائمة التشغيل الخاصة بي"، أو "حفظ هذا الألبوم"، أو "البحث في Spotify عن..."، أو "إدراج قوائم التشغيل الخاصة بي"، أو "ما هي المسارات الموجودة في قائمة التشغيل هذه"، أو أي مهمة إدارة قائمة تشغيل/مكتبة Spotify.
