يُستخدم عند إضافة نماذج منبثقة أو تأكيدات أو عروض تفصيلية إلى لوحات معلومات Wix؛ وإنشاء مكونات حوار قابلة لإعادة الاستخدام عبر صفحات لوحة المعلومات؛ عرض معلومات خاصة بالسياق في التراكبات؛ فتح الوسائط من صفحات لوحة المعلومات؛ أو تمرير البيانات بين صفحات لوحة المعلومات والنماذج. لا تستخدمه للمحتوى الثابت (استخدم صفحات لوحة المعلومات بدلاً من ذلك) أو واجهة المستخدم التي تواجه الموقع (استخدم أدوات الموقع/البرامج النصية المضمنة).
