مهارة تصحيح الصدأ لبرمجة الأنظمة. يُستخدم عند تصحيح أخطاء ثنائيات Rust باستخدام GDB أو LLDB، أو تمكين طابعات Rust الجميلة، أو تفسير الذعر والتتبعات الخلفية، أو تصحيح أخطاء async/await باستخدام tokio-console، أو التنقل عبر كود no_std، أو استخدام dbg! وتتبع وحدات الماكرو بشكل فعال. يتم تنشيطه عند الاستعلامات حولrust-gdb، أوrust-lldb، أوRUST_BACKTRACE، أو ذعر الصدأ، أو تصحيح أخطاء الصدأ غير المتزامن، أو tokio-console، أو الطابعات الجميلة.
