Навыки отладки ржавчины для системного программирования. Используйте при отладке двоичных файлов Rust с помощью GDB или LLDB, включении симпатичных принтеров Rust, интерпретации паники и обратных трассировок, отладке async/await с помощью tokio-console, пошаговом выполнении кода no_std или использовании dbg! и эффективное отслеживание макросов. Активируется при запросах о ржавчине-gdb, ржавчине-lldb, RUST_BACKTRACE, панике Rust, асинхронной отладке Rust, tokio-console или Pretty-принтерах.
