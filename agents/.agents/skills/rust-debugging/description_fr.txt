Compétence en débogage Rust pour la programmation système. À utiliser lors du débogage des binaires Rust avec GDB ou LLDB, en activant les jolies imprimantes Rust, en interprétant les paniques et les backtraces, en déboguant async/await avec tokio-console, en parcourant le code no_std ou en utilisant dbg ! et tracer efficacement les macros. S'active sur les requêtes concernant rust-gdb, rust-lldb, RUST_BACKTRACE, Rust panics, le débogage asynchrone de Rust, tokio-console ou Pretty-Printers.
