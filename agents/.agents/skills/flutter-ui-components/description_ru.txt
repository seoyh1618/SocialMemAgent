M3-совместимые компоненты пользовательского интерфейса (кнопки, карточки, формы, поля ввода).
КОГДА ИСПОЛЬЗУЕТСЯ: создание компонентов <300 строк, миграция M3, работа системы проектирования.
НЕ ДЛЯ: полноценных экранов, функций с бизнес-логикой (используйте flutter-developer).
Всегда проверяйте компоненты M3 с помощью инструментов MCP перед созданием собственных.

Примеры:
<пример>
Контекст: необходимо перенести компонент кнопки в M3.
пользователь: «Миграция BukeerButton на Material Design 3»
помощник: «Я воспользуюсь компонентами flutter-ui для миграции BukeerButton на M3».
<commentary>Миграция компонентов пользовательского интерфейса — это специальность flutter-ui-comments.</commentary>
</пример>
<пример>
Контекст: Создайте новое поле формы многократного использования.
пользователь: «Создайте новый входной компонент выбора даты после M3»
помощник: «Я буду использовать компоненты flutter-ui для создания средства выбора даты, совместимого с M3».
<commentary>Отдельные компоненты пользовательского интерфейса должны обрабатываться компонентами flutter-ui.</commentary>
</пример>
