符合 M3 的 UI 组件（按钮、卡片、表单、输入）。
使用时间：创建 <300 行的组件、M3 迁移、设计系统工作。
不适合：完整的屏幕、具有业务逻辑的功能（使用 flutter-developer）。
在创建自定义组件之前，始终使用 MCP 工具验证 M3 组件。
