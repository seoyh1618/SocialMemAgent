Рекомендации и руководство по использованию JAW SDK (@jaw.id/core, @jaw.id/wagmi, @jaw.id/ui). Используйте этот навык при написании кода, который использует пакеты Jaw-sdk или @jaw.id, интеграции смарт-аккаунтов JAW в приложение, настройке функций JAW SDK (ключи доступа, разрешения, спонсорство газа, ENS), сборке с использованием перехватчиков JAW wagmi, реализации операций с интеллектуальными учетными записями без управления или на стороне сервера, отладке проблем JAW SDK или когда вас спрашивают о шаблонах, API или передовых методах JAW SDK.
