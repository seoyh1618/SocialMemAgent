À utiliser lorsque l'utilisateur demande de configurer endorctl, d'installer endorctl, d'exécuter une analyse endorctl, de rechercher des vulnérabilités, d'exécuter une analyse endor ou d'exécuter une analyse Endor Labs ou lorsqu'une commande endorctl échoue avec « commande introuvable », « aucun fichier ou répertoire de ce type », des erreurs d'authentification, « non autorisé », « 403 », « locataire introuvable », une erreur EOF ou des erreurs d'espace de noms/d'accès.
