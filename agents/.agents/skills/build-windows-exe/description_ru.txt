Создает исполняемые файлы Windows (x64 и ARM64) для XerahS с использованием сценария упаковки. Устраняет распространенные проблемы блокировки файлов во время компиляции, завершая процессы, снимая блокировки и при необходимости используя однопоточные сборки. Обеспечивает успешное создание установщиков Inno Setup.
