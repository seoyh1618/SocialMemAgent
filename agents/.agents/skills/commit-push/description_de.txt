Wenn der Benutzer „Code senden“, „festschreiben“, „Code pushen“, „pushen“, „festschreiben und pushen“, „Änderungen festschreiben“, „an Remote senden“, „hilf mir beim Festschreiben“ oder „den Code hochschieben“ sagt, sollte diese Fähigkeit verwendet werden. Es bietet einen standardisierten Git-Commit- und Push-Workflow, analysiert automatisch die Änderungen, generiert eine konforme Commit-Nachricht und führt die entsprechenden Aktionen aus.
