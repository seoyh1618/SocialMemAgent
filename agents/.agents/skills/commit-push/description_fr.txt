Lorsque l'utilisateur dit « soumettre le code », « valider », « pousser le code », « pousser », « valider et pousser », « valider les modifications », « pousser vers la télécommande », « aidez-moi à valider » ou « pousser le code vers le haut », cette compétence doit être utilisée. Il fournit un workflow git commit et push standardisé, analyse automatiquement les modifications, génère un message de validation conforme et exécute les actions correspondantes.
