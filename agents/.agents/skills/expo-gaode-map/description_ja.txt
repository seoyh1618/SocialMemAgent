Expo Amap のコア機能: ネイティブ MapView レンダリング (標準/衛星/夜間/ナビゲーション)、ジェスチャーおよびカメラ制御。位置情報サービス (単一/連続/バックグラウンド位置情報、ブルードット スタイルおよび追跡);オーバーレイ描画 (マーカー/ポリライン/ポリゴン/サークル/ヒートマップ/マルチポイント/クラスター、C++ エンジンによる高性能クラスタリング)。オフラインマップのダウンロードと管理。タイプセーフな TS API とネイティブの幾何学的計算 (距離、最近点スナップ、軌道の細線化、ポリゴン内のポイント、パスの長さ)。 Config Plugin の自動構成と initSDK(webKey/iosKey/androidKey) の初期化をサポートします。 iOS/Android に適応し、プライバシー コンプライアンスと許可処理が組み込まれています。
