Enveloppez les colonnes de base de données basées sur JSON avec des classes de type ActiveModel à l'aide de store_model. À utiliser lors de la création d'objets de configuration, de la gestion d'attributs JSON imbriqués ou de l'ajout de validations aux données JSON. Déclencheurs sur les colonnes JSON, store_model, JSON typé, objets de configuration ou attributs imbriqués.
