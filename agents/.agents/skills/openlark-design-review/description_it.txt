Competenze di progettazione del codice/revisione dell'API pubblica per OpenLark Rust SDK (orientato a crate/modulo). Controlla sistematicamente la progettazione delle voci, il gating delle funzionalità, la coerenza di richieste/servizi/costruttori, il sistema endpoint, la gestione di configurazioni/errori, esportazione e sincronizzazione di documenti, test e controllo degli avvisi e genera un elenco di controllo di correzione in ordine di priorità con piani di refactoring attuabili. Parole chiave di attivazione: revisione del progetto, progettazione del contenitore, progettazione dell'API, API pubblica, indicatore della funzionalità, endpoint, generatore/servizio, coerenza dell'architettura
