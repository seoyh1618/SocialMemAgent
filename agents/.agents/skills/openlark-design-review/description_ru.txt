Навыки разработки кода OpenLark Rust SDK/проверки общедоступного API (уровень ящика/модуля). Систематически проверяет проектирование входной точки, ограничение функций, согласованность запросов/услуг/сборщиков, таксономию конечных точек, обработку конфигураций/ошибок, экспорт и синхронизацию документов, тестирование и контроль проверки, а также выводит контрольный список исправлений с сортировкой по приоритетам и действенные планы рефакторинга. Ключевые слова-триггеры: обзор проекта, дизайн ящика, дизайн API, общедоступный API, флаг функции, конечная точка, строитель/сервис, согласованность архитектуры.
