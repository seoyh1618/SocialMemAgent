قم بإنشاء توصيات البيانات الوصفية المحسنة لـ Apple App Store باستخدام أفضل ممارسات ASO. استخدم هذه المهارة عند تحليل قوائم التطبيقات، أو تحسين البيانات الوصفية (العنوان، أو العنوان الفرعي، أو الوصف، أو الكلمات الرئيسية)، أو إجراء تحليل تنافسي، أو التحقق من صحة متطلبات قائمة متجر التطبيقات. يتم تشغيله عند الاستعلامات حول تحسين متجر التطبيقات، أو مراجعة البيانات التعريفية، أو إستراتيجية لقطة الشاشة.
