# Content Calendar Planner

Create organized, strategic content calendars that ensure consistent publishing and aligned messaging. This skill helps you plan content across channels, balance content types, align with business objectives, and maintain a sustainable publishing cadence.

Consistent content creation is hard without a plan. This skill provides frameworks for theme development, content type balancing, editorial workflow design, and calendar management. Move from reactive content creation to proactive, strategic publishing that builds audience and drives results.

Perfect for content marketers, social media managers, marketing teams, and anyone responsible for regular content publishing.

## Core Workflows

### Workflow 1: Monthly Content Planning
1. **Theme Selection** - Key topics and campaigns for the month
2. **Content Type Mix** - Balance blog, social, email, video
3. **Channel Mapping** - What goes where
4. **Publishing Cadence** - Frequency per channel
5. **Resource Assessment** - Creation capacity check
6. **Key Dates** - Holidays, events, product launches
7. **Calendar Population** - Specific content slots filled

### Workflow 2: Editorial Calendar Setup
1. **Calendar Structure** - Columns, views, workflows
2. **Content Pillars** - Core themes that repeat
3. **Workflow Stages** - Ideation, creation, review, publish
4. **Ownership Assignment** - Who does what
5. **Deadline Framework** - Working back from publish dates
6. **Status Tracking** - Progress visibility
7. **Tool Configuration** - Set up in Notion, Asana, etc.

### Workflow 3: Content Batch Planning
1. **Batch Definition** - Content sets created together
2. **Theme Clustering** - Related content grouped
3. **Asset Bundling** - Graphics, copy, video coordinated
4. **Repurposing Strategy** - One idea, multiple formats
5. **Production Schedule** - Efficient creation blocks
6. **Review Consolidation** - Batch approvals
7. **Publishing Sequencing** - Strategic rollout timing

### Workflow 4: Content Strategy Alignment
1. **Business Goal Mapping** - Content tied to objectives
2. **Buyer Journey Coverage** - Content for each stage
3. **Keyword Integration** - SEO-driven topic selection
4. **Competitive Gaps** - Content differentiation
5. **Seasonal Planning** - Annual content rhythms
6. **Campaign Integration** - Coordinated messaging
7. **Measurement Framework** - Success metrics defined

## Best Practices

- **Plan ahead** - Monthly minimum, quarterly ideal
- **Balance content types** - Educational, entertaining, promotional
- **Buffer time** - Build in flexibility for reactive content
- **Theme consistency** - Content pillars create coherence
- **Repurpose ruthlessly** - One idea, many formats
- **Seasonal awareness** - Plan around key dates
- **Capacity reality** - Plan for what's achievable
- **Batch creation** - Group similar content for efficiency
- **Review built-in** - Time for edits and approval
- **Visible to all** - Calendar transparency helps coordination
- **Flexibility zones** - Leave space for timely content
- **Post-mortem** - Review what worked each month
- **Evergreen balance** - Mix timely and timeless
- **Channel-appropriate** - Not all content fits all channels
- **Update regularly** - Living document, not static plan
