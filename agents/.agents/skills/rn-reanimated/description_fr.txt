Meilleures pratiques de React Native Reanimated 4.x, modèles d'animation à 60 ips, correction de bugs et amélioration du code pour Android et iOS. À utiliser lors de l'écriture, de la révision, de la correction ou de l'optimisation du code d'animation réanimé, des interactions gestuelles, des worklets, des animations de mise en page ou des transitions d'éléments partagés. Déclencheurs sur des tâches impliquant React-Native-Reanimated, useSharedValue, useAnimatedStyle, withSpring, withTiming, withDecay, Gesture.Pan, des animations de mise en page, le débogage de worklet, l'optimisation des performances d'animation ou le dépannage d'animations bancales. À utiliser également lorsque l'utilisateur souhaite corriger des bugs d'animation, améliorer la fluidité de l'animation, revoir le code gestuel ou atteindre 60 ips sur Android et iOS. Se déclenche également lorsque l'utilisateur demande à réviser, auditer, vérifier ou mettre à niveau son code réanimé vers les meilleures pratiques, ou demande « mon code d'animation est-il bon », « vérifier mon code réanimé », « mettre à jour les meilleures pratiques », « réviser mes animations » ou des demandes similaires.
