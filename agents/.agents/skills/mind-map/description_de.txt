Erstellt Mermaid-Mindmap-Diagramme aus Codebasen, Themen, Dateien oder Konversationen. Fasst Quellmaterial visuell zusammen und zeigt Hauptthemen und verwandte Ideen als Verzweigungsdiagramme. Verwenden Sie diese Funktion, wenn Sie aufgefordert werden, eine Mindmap zu erstellen, ein Thema zu visualisieren, eine Codebasis abzubilden, eine Datei als Diagramm zusammenzufassen, eine Concept Map zu erstellen oder eine visuelle Übersicht zu erstellen.
