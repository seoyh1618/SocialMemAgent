Génère des diagrammes mindmap Mermaid à partir de bases de code, de sujets, de fichiers ou de conversations. Résume visuellement le matériel source montrant les sujets principaux et les idées associées sous forme de diagrammes de branchement. À utiliser lorsqu'on vous le demande pour créer une carte mentale, visualiser un sujet, cartographier une base de code, résumer un fichier sous forme de diagramme, générer une carte conceptuelle ou créer un aperçu visuel.
