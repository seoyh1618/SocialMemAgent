Développement Python moderne avec uv (gestionnaire de paquets 10 à 100 fois plus rapide) et ruff (linter/formatter extrêmement rapide). À utiliser lors de la gestion de projets Python, de dépendances, d'environnements virtuels, de l'installation de packages, du linting de code ou du formatage de fichiers Python. Se déclenche sur des expressions telles que "uv install", "ruff check", "python package manager", "format python code" ou l'utilisation de fichiers pyproject.toml.
