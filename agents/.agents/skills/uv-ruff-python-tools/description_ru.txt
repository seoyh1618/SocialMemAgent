Современная разработка на Python с использованием uv (менеджер пакетов в 10-100 раз быстрее) и ruff (чрезвычайно быстрый линтер/форматер). Используйте при управлении проектами Python, зависимостями, виртуальными средами, установке пакетов, проверке кода или форматировании файлов Python. Запускается по таким фразам, как «uv install», «проверка ерша», «менеджер пакетов Python», «форматирование кода Python» или работа с файлами pyproject.toml.
