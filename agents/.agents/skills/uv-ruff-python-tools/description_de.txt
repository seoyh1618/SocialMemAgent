Moderne Python-Entwicklung mit uv (10-100x schnellerer Paketmanager) und ruff (extrem schneller Linter/Formatter). Wird zum Verwalten von Python-Projekten, Abhängigkeiten, virtuellen Umgebungen, zum Installieren von Paketen, zum Flusen von Code oder zum Formatieren von Python-Dateien verwendet. Wird bei Ausdrücken wie „uv install“, „ruff check“, „python package manager“, „format python code“ oder beim Arbeiten mit pyproject.toml-Dateien ausgelöst.
