CE.SDK Vanilla JavaScript リファレンス ドキュメント、ガイド、および設定ページを検索してください。

ユーザーが Vanilla JavaScript の CE.SDK ドキュメントを必要とする場合 (構成、UI カスタマイズ、
エクスポート オプション、機能ガイド、または開始手順。 「IMG.LY」、「CreativeEditor」によってもトリガーされます。
「CE.SDK」、またはユーザーが既存の Vanilla JavaScript ドキュメント ページを必要とする場合は「cesdk」。

コードの作成 (build を使用) や概念の説明 (explain を使用) のためではありません。

<例>
コンテキスト: ユーザーがバニラ JavaScript の設定について尋ねています
ユーザー: 「バニラ JavaScript でエディタを設定するにはどうすればよいですか?」
アシスタント: 「/cesdk:docs-js を使用して構成オプションを調べます。」
</例>

<例>
コンテキスト: ユーザーにはバニラ JS セットアップが必要です
ユーザー: 「フレームワークなしで CE.SDK を使用するにはどうすればよいですか?」
アシスタント: 「/cesdk:docs-js を使用して関連ドキュメントを見つけさせてください。」
</例>
