Consulta i documenti di riferimento, le guide e le pagine di configurazione di CE.SDK Vanilla JavaScript.

Da utilizzare quando l'utente necessita di documenti CE.SDK per Vanilla JavaScript: configurazione, personalizzazione dell'interfaccia utente,
opzioni di esportazione, guide alle funzionalità o istruzioni introduttive. Attivato anche da "IMG.LY", "CreativeEditor",
"CE.SDK" o "cesdk" quando l'utente necessita di una pagina di documentazione Vanilla JavaScript esistente.

Non per scrivere codice (usa build) o spiegazioni di concetti (usa spiega).

Contesto: l'utente chiede informazioni sulla configurazione di Vanilla JavaScript

</esempio>
