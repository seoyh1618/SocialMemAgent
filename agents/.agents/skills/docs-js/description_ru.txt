Найдите справочную документацию, руководства и страницы конфигурации CE.SDK Vanilla JavaScript.

Используйте, когда пользователю нужна документация CE.SDK для Vanilla JavaScript — настройка, настройка пользовательского интерфейса,
параметры экспорта, руководства по функциям или инструкции по началу работы. Также запускается "IMG.LY", "CreativeEditor",
«CE.SDK» или «cesdk», если пользователю нужна существующая страница документа Vanilla JavaScript.

Не для написания кода (используйте сборку) или объяснения концепций (используйте объяснение).

<пример>
Контекст: пользователь спрашивает о конфигурации Vanilla JavaScript.
пользователь: «Как мне настроить редактор в Vanilla JavaScript?»
помощник: «Я буду использовать /cesdk:docs-js для поиска параметров конфигурации».
</пример>

<пример>
Контекст: пользователю необходима настройка vanilla JS.
пользователь: «Как использовать CE.SDK без фреймворка?»
помощник: «Позвольте мне использовать /cesdk:docs-js, чтобы найти соответствующую документацию».
</пример>
