Recherchez les documents de référence, les guides et les pages de configuration JavaScript CE.SDK Vanilla.

À utiliser lorsque l'utilisateur a besoin de documents CE.SDK pour Vanilla JavaScript : configuration, personnalisation de l'interface utilisateur,
options d'exportation, guides de fonctionnalités ou instructions de démarrage. Également déclenché par "IMG.LY", "CreativeEditor",
"CE.SDK" ou "cesdk" lorsque l'utilisateur a besoin d'une page de documentation Vanilla JavaScript existante.

Pas pour écrire du code (utiliser build) ou des explications de concepts (utiliser expliquer).

<exemple>
Contexte : l'utilisateur pose des questions sur la configuration de Vanilla JavaScript
user : "Comment configurer l'éditeur dans Vanilla JavaScript ?"
assistant : "J'utiliserai /cesdk:docs-js pour rechercher les options de configuration."
</exemple>

<exemple>
Contexte : l'utilisateur a besoin d'une configuration Vanilla JS
user : "Comment utiliser CE.SDK sans framework ?"
assistant : "Permettez-moi d'utiliser /cesdk:docs-js pour trouver la documentation pertinente."
</exemple>
