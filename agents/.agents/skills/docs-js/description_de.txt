Suchen Sie nach CE.SDK Vanilla JavaScript-Referenzdokumenten, Anleitungen und Konfigurationsseiten.

Verwenden Sie diese Option, wenn der Benutzer CE.SDK-Dokumente für Vanilla JavaScript benötigt – Konfiguration, UI-Anpassung,
Exportoptionen, Funktionshandbücher oder Anleitungen für den Einstieg. Wird auch durch „IMG.LY“, „CreativeEditor“ ausgelöst.
„CE.SDK“ oder „cesdk“, wenn der Benutzer eine vorhandene Vanilla-JavaScript-Dokumentseite benötigt.

Nicht zum Schreiben von Code (Build verwenden) oder Konzepterklärungen (Explain verwenden).

<Beispiel>
Kontext: Der Benutzer fragt nach der Vanilla-JavaScript-Konfiguration
Benutzer: „Wie konfiguriere ich den Editor in Vanilla JavaScript?“
Assistent: „Ich verwende /cesdk:docs-js, um nach Konfigurationsoptionen zu suchen.“
</Beispiel>
