尋找 CE.SDK Vanilla JavaScript 參考文件、指南和設定頁面。

當使用者需要適用於 Vanilla JavaScript 的 CE.SDK 文件時使用 - 設定、UI 自訂、
匯出選項、功能指南或入門說明。也由「IMG.LY」、「CreativeEditor」觸發，
當使用者需要現有的 Vanilla JavaScript 文件頁面時，稱為「CE.SDK」或「cesdk」。

不適用於編寫程式碼（使用建構）或概念解釋（使用解釋）。

<範例>
情境：使用者詢問 Vanilla JavaScript 配置
使用者：“如何在 Vanilla JavaScript 中配置編輯器？”
助理：“我將使用 /cesdk:docs-js 查找配置選項。”
</範例>
