Busque documentos de referencia, guías y páginas de configuración de CE.SDK Vanilla JavaScript.

Úselo cuando el usuario necesite documentos CE.SDK para Vanilla JavaScript: configuración, personalización de la interfaz de usuario,
opciones de exportación, guías de funciones o instrucciones de introducción. También activado por "IMG.LY", "CreativeEditor",
"CE.SDK" o "cesdk" cuando el usuario necesita una página de documento Vanilla JavaScript existente.

No para escribir código (use compilación) o explicaciones de conceptos (use explicación).

<ejemplo>
Contexto: el usuario pregunta sobre la configuración de Vanilla JavaScript
usuario: "¿Cómo configuro el editor en Vanilla JavaScript?"
asistente: "Usaré /cesdk:docs-js para buscar opciones de configuración".
</ejemplo>
