ابحث عن المستندات والأدلة وصفحات التكوين المرجعية لـ CE.SDK Vanilla JavaScript.

يُستخدم عندما يحتاج المستخدم إلى مستندات CE.SDK لـ Vanilla JavaScript — التكوين، وتخصيص واجهة المستخدم،
خيارات التصدير أو أدلة الميزات أو تعليمات البدء. تم تشغيله أيضًا بواسطة "IMG.LY"، "CreativeEditor"،
"CE.SDK" أو "cesdk" عندما يحتاج المستخدم إلى صفحة مستند Vanilla JavaScript موجودة.

ليس لكتابة التعليمات البرمجية (استخدم البناء) أو شرح المفاهيم (استخدم الشرح).

<مثال>
السياق: يسأل المستخدم عن تكوين Vanilla JavaScript
المستخدم: "كيف يمكنني تكوين المحرر في Vanilla JavaScript؟"
المساعد: "سأستخدم /cesdk:docs-js للبحث عن خيارات التكوين."
</مثال>

<مثال>
السياق: يحتاج المستخدم إلى إعداد Vanilla JS
المستخدم: "كيف يمكنني استخدام CE.SDK بدون إطار عمل؟"
المساعد: "اسمح لي باستخدام /cesdk:docs-js للعثور على الوثائق ذات الصلة."
</مثال>
