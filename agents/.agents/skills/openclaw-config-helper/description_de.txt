OpenClaw-Konfigurationsänderungsassistent. Bevor Sie eine OpenClaw-Konfiguration ändern, müssen Sie zunächst die offizielle Dokumentation konsultieren, um sicherzustellen, dass das Format korrekt ist und Systemabstürze oder Funktionsanomalien vermieden werden. Erzwingen Sie den Prozess: Schema prüfen → Dokumentation prüfen → Bestätigen → Ändern.
