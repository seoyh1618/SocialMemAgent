يراجع الكود المصدري لانتهاكات الجودة الهيكلية وروائح الكود عبر أي لغة برمجة. يُنتج تقرير نتائج للقراءة فقط مع اقتراحات إعادة هيكلة ملموسة. يتم التنشيط عند مراجعة التعليمات البرمجية، أو تدقيق التعليمات البرمجية، أو فحص جودة التعليمات البرمجية، أو روائح التعليمات البرمجية، أو "تنظيف هذا"، أو "جعل هذا أكثر قابلية للصيانة"، أو "تقليل التعقيد".
