Проверяет исходный код на наличие нарушений структурного качества и запахов кода на любом языке программирования. Создает отчет о результатах, доступный только для чтения, с конкретными предложениями по рефакторингу. Активируется при проверке кода, аудите кода, проверке качества кода, обнаружении запаха кода, «очистке», «сделании более удобным в сопровождении» или «уменьшении сложности».
