Utilisez cette compétence lorsque l'utilisateur souhaite interagir avec X/Twitter via la ligne de commande. Les déclencheurs incluent les demandes de tweet, de réponse aux tweets, de lecture de tweets ou de fils de discussion, de recherche sur X/Twitter, de vérification des mentions, d'affichage des signets, d'obtention de sujets tendance, de gestion de listes ou de toute autre opération X/Twitter via CLI. À utiliser également lorsque l'utilisateur mentionne spécifiquement l'outil Bird CLI ou souhaite automatiser les interactions X/Twitter depuis le terminal. Ne PAS utiliser pour les interactions Twitter sur le Web ou les intégrations de l'API Twitter.
