Utilizza questa abilità quando l'utente desidera interagire con X/Twitter tramite riga di comando. I trigger includono richieste di twittare, rispondere a tweet, leggere tweet o thread, cercare X/Twitter, controllare menzioni, visualizzare segnalibri, ottenere argomenti di tendenza, gestire elenchi o qualsiasi altra operazione X/Twitter tramite CLI. Da utilizzare anche quando l'utente menziona specificamente lo strumento CLI Bird o desidera automatizzare le interazioni X/Twitter dal terminale. NON utilizzare per interazioni Twitter basate sul Web o integrazioni API Twitter.
