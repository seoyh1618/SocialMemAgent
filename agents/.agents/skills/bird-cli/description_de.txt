Verwenden Sie diesen Skill, wenn der Benutzer über die Befehlszeile mit X/Twitter interagieren möchte. Zu den Auslösern gehören Anfragen zum Twittern, Beantworten von Tweets, Lesen von Tweets oder Threads, Durchsuchen von X/Twitter, Überprüfen von Erwähnungen, Anzeigen von Lesezeichen, Abrufen von Trendthemen, Verwalten von Listen oder andere X/Twitter-Vorgänge über CLI. Verwenden Sie es auch, wenn der Benutzer das Bird CLI-Tool ausdrücklich erwähnt oder X/Twitter-Interaktionen vom Terminal aus automatisieren möchte. NICHT für webbasierte Twitter-Interaktionen oder Twitter-API-Integrationen verwenden.
