استخدم هذه المهارة عندما يريد المستخدم التفاعل مع X/Twitter عبر سطر الأوامر. تتضمن المشغلات طلبات التغريد أو الرد على التغريدات أو قراءة التغريدات أو المواضيع أو البحث في X/Twitter أو التحقق من الإشارات أو عرض الإشارات المرجعية أو الحصول على موضوعات شائعة أو إدارة القوائم أو أي عمليات X/Twitter أخرى عبر واجهة سطر الأوامر (CLI). يُستخدم أيضًا عندما يذكر المستخدم أداة Bird CLI على وجه التحديد أو عندما يريد أتمتة تفاعلات X/Twitter من الجهاز. لا تستخدمه لتفاعلات Twitter المستندة إلى الويب أو عمليات تكامل Twitter API.
