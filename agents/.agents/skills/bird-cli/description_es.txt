Utilice esta habilidad cuando el usuario quiera interactuar con X/Twitter a través de la línea de comando. Los activadores incluyen solicitudes para twittear, responder a tweets, leer tweets o hilos, buscar X/Twitter, verificar menciones, ver marcadores, obtener temas de actualidad, administrar listas o cualquier otra operación de X/Twitter a través de CLI. Utilícelo también cuando el usuario mencione específicamente la herramienta Bird CLI o quiera automatizar las interacciones X/Twitter desde el terminal. NO lo utilice para interacciones de Twitter basadas en web o integraciones de API de Twitter.
