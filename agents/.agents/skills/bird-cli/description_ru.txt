Используйте этот навык, когда пользователь хочет взаимодействовать с X/Twitter через командную строку. Триггеры включают запросы на твиты, ответы на твиты, чтение твитов или тем, поиск в X/Twitter, проверку упоминаний, просмотр закладок, получение актуальных тем, управление списками или любые другие операции X/Twitter через CLI. Также используйте, когда пользователь конкретно упоминает инструмент Bird CLI или хочет автоматизировать взаимодействие X/Twitter с терминала. НЕ используйте для взаимодействия с Twitter через Интернет или интеграции API Twitter.
