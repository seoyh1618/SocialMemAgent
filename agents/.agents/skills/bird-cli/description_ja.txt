ユーザーがコマンドライン経由で X/Twitter と対話したい場合は、このスキルを使用します。トリガーには、ツイート、ツイートへの返信、ツイートまたはスレッドの読み取り、X/Twitter の検索、メンションの確認、ブックマークの表示、トレンドのトピックの取得、リストの管理、または CLI を介したその他の X/Twitter 操作のリクエストが含まれます。ユーザーが Bird CLI ツールについて具体的に言及した場合、または端末からの X/Twitter インタラクションを自動化したい場合にも使用します。 Web ベースの Twitter 対話または Twitter API 統合には使用しないでください。
