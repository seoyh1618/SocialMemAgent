أنشئ مسارات عمل الذكاء الاصطناعي جاهزة للإنتاج باستخدام Firebase Genkit. يُستخدم عند إنشاء التدفقات أو وكلاء استدعاء الأدوات أو خطوط أنابيب RAG أو أنظمة متعددة الوكلاء أو نشر الذكاء الاصطناعي في Firebase/Cloud Run. يدعم TypeScript وGo وPython مع المكونات الإضافية Gemini وOpenAI وAnthropic وOllama وVertex AI.
