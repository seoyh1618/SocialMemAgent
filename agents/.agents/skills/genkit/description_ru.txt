Создавайте готовые к использованию рабочие процессы искусственного интеллекта с помощью Firebase Genkit. Используйте при создании потоков, агентов вызова инструментов, конвейеров RAG, многоагентных систем или развертывании искусственного интеллекта в Firebase/Cloud Run. Поддерживает TypeScript, Go и Python с плагинами Gemini, OpenAI, Anthropic, Ollama и Vertex AI.
