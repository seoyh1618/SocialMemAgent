Fornisce linee guida complete per la traduzione e la localizzazione delle stringhe di estensione Roo Code. Da utilizzare quando le attività riguardano i18n, traduzione, localizzazione, aggiunta di nuove lingue o aggiornamento di file di traduzione esistenti. Questa competenza copre sia l'estensione principale (src/i18n/locales/) che la localizzazione dell'interfaccia utente WebView (webview-ui/src/i18n/locales/).
