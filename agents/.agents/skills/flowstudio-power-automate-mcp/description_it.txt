Connettiti e gestisci i flussi cloud di Power Automate tramite un server MCP FlowStudio. Utilizzare quando richiesto per: elencare flussi, leggere una definizione di flusso, controllare la cronologia di esecuzione, ispezionare gli output delle azioni, inviare nuovamente un'esecuzione, annullare un flusso in esecuzione, visualizzare connessioni, ottenere un URL di trigger, convalidare una definizione, monitorare l'integrità del flusso o qualsiasi attività che richieda la comunicazione con l'API Power Automate tramite uno strumento MCP. Utilizzare anche per l'individuazione dell'ambiente Power Platform e la gestione della connessione. Richiede un abbonamento MCP FlowStudio o un server compatibile: vedere https://mcp.flowstudio.app
