Stellen Sie über einen FlowStudio MCP-Server eine Verbindung zu Power Automate Cloud Flows her und betreiben Sie diese. Wird verwendet, wenn Sie dazu aufgefordert werden: Flows aufzulisten, eine Flow-Definition zu lesen, den Ausführungsverlauf zu überprüfen, Aktionsausgaben zu überprüfen, eine Ausführung erneut zu senden, einen laufenden Flow abzubrechen, Verbindungen anzuzeigen, eine Trigger-URL abzurufen, eine Definition zu validieren, den Flow-Zustand zu überwachen oder jede Aufgabe auszuführen, die eine Kommunikation mit der Power Automate-API über ein MCP-Tool erfordert. Kann auch für die Erkennung und Verbindungsverwaltung der Power Platform-Umgebung verwendet werden. Erfordert ein FlowStudio MCP-Abonnement oder einen kompatiblen Server – siehe https://mcp.flowstudio.app
