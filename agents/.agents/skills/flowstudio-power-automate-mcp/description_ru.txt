Подключайтесь к облачным потокам Power Automate и управляйте ими через сервер FlowStudio MCP. Используйте, когда вас попросят: составить список потоков, прочитать определение потока, проверить историю выполнения, проверить выходные данные действий, повторно отправить запуск, отменить выполняемый поток, просмотреть соединения, получить URL-адрес триггера, проверить определение, отслеживать работоспособность потока или выполнить любую задачу, требующую взаимодействия с Power Automate API через инструмент MCP. Также используйте для обнаружения среды Power Platform и управления соединениями. Требуется подписка FlowStudio MCP или совместимый сервер — см. https://mcp.flowstudio.app.
