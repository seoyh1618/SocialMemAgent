Conéctese y opere flujos en la nube de Power Automate a través de un servidor FlowStudio MCP. Úselo cuando se le solicite: enumerar flujos, leer una definición de flujo, verificar el historial de ejecución, inspeccionar resultados de acciones, volver a enviar una ejecución, cancelar un flujo en ejecución, ver conexiones, obtener una URL de activación, validar una definición, monitorear el estado del flujo o cualquier tarea que requiera hablar con la API de Power Automate a través de una herramienta MCP. También se utiliza para el descubrimiento de entornos de Power Platform y la administración de conexiones. Requiere una suscripción a FlowStudio MCP o un servidor compatible; consulte https://mcp.flowstudio.app
