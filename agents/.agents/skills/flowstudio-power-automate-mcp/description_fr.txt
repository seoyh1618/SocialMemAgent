Connectez-vous et exploitez les flux cloud Power Automate via un serveur FlowStudio MCP. À utiliser lorsque vous y êtes invité pour : répertorier les flux, lire une définition de flux, vérifier l'historique d'exécution, inspecter les résultats d'une action, soumettre à nouveau une exécution, annuler un flux en cours d'exécution, afficher les connexions, obtenir une URL de déclenchement, valider une définition, surveiller l'état du flux ou toute tâche nécessitant de communiquer avec l'API Power Automate via un outil MCP. À utiliser également pour la découverte de l'environnement Power Platform et la gestion des connexions. Nécessite un abonnement FlowStudio MCP ou un serveur compatible — voir https://mcp.flowstudio.app
