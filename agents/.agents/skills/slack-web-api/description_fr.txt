Conseils complets pour les opérations de l’API Web Slack avec le SDK Go. À utiliser lors de l'envoi de messages, de la publication sur des canaux, de la création ou de la gestion de canaux, de la récupération d'informations utilisateur, du téléchargement de fichiers, de la composition de messages Block Kit ou de l'exécution de toute opération synchrone de l'API Slack.
