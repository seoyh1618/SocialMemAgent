Formulaires interactifs avec champs de saisie et boutons pour les robots. À utiliser lorsque vous avez besoin de : afficher un formulaire, gérer la soumission d'un formulaire, valider les champs d'un formulaire, créer une fenêtre modale, interroger les employés. NE PAS utiliser pour : les boutons normaux dans les messages (→ pachca-messages), les paramètres du bot (→ pachca-bots).
