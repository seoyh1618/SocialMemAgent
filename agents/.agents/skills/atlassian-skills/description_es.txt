Utilidades de Python para la integración de Jira, Confluence y Bitbucket. Proporciona gestión de problemas, búsqueda, flujos de trabajo, gestión de páginas, solicitudes de extracción, historial de confirmaciones y más. Úselo cuando los usuarios necesiten interactuar con productos de Atlassian, como "crear una incidencia de Jira", "buscar páginas de Confluence", "crear una solicitud de extracción", "obtener historial de confirmaciones" o "actualizar el estado del sprint".
