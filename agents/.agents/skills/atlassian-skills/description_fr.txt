Utilitaires Python pour l'intégration de Jira, Confluence et Bitbucket. Fournit la gestion des problèmes, la recherche, les flux de travail, la gestion des pages, les demandes d'extraction, l'historique des validations, etc. À utiliser lorsque les utilisateurs ont besoin d'interagir avec les produits Atlassian, comme « créer un ticket Jira », « rechercher des pages Confluence », « créer une pull request », « obtenir l'historique des validations » ou « mettre à jour le statut du sprint ».
