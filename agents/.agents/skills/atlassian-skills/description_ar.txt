أدوات Python المساعدة لتكامل Jira وConfluence وBitbucket. يوفر إدارة المشكلات والبحث وسير العمل وإدارة الصفحات وطلبات السحب وسجل الالتزام والمزيد. يُستخدم عندما يحتاج المستخدمون إلى التفاعل مع منتجات Atlassian مثل "إنشاء مشكلة Jira"، أو "البحث في صفحات التقاء"، أو "إنشاء طلب سحب"، أو "الحصول على سجل الالتزام"، أو "تحديث حالة السباق".
