Получайте полное содержимое MIME электронной почты и файлы вложений из IMAP по ссылке на сообщение. Используйте, когда последующие шаги уже имеют mail_ref этапа 1 и нуждаются в полных заголовках/тексте/вложениях для конкретного сообщения, с message_id_norm в качестве основного ключа поиска и uid только в качестве запасного.
