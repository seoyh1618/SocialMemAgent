Crée un bloc-notes NotebookLM versionné pour la version actuelle d'OrchestKit. Lit la version de package.json, extrait la section CHANGELOG correspondante, télécharge CHANGELOG + CLAUDE.md + manifests/ork.json en tant que sources, génère un aperçu audio et partage le bloc-notes. À utiliser lors de la découpe d'une version pour créer la base de connaissances de version.
