Erstellt ein versioniertes NotebookLM-Notizbuch für die aktuelle OrchestKit-Version. Liest die Version aus package.json, extrahiert den passenden CHANGELOG-Abschnitt, lädt CHANGELOG + CLAUDE.md + manifests/ork.json als Quellen hoch, generiert eine Audioübersicht und gibt das Notizbuch frei. Verwenden Sie diese Option, wenn Sie ein Release ausschneiden, um die Release-KB zu erstellen.
