Создает записную книжку NotebookLM с поддержкой версий для текущей версии OrchestKit. Считывает версию из package.json, извлекает соответствующий раздел CHANGELOG, загружает CHANGELOG + CLAUDE.md + манифесты/ork.json в качестве источников, генерирует аудиообзор и предоставляет общий доступ к записной книжке. Используйте при вырезании релиза для создания базы знаний релиза.
