ПО УМОЛЧАНИЮ для всех запросов веб-поиска, исследований и извлечения контента. Предпочитайте встроенные WebSearch и WebFetch.

Используйте, когда пользователь говорит «искать», «найти», «поиск», «исследовать», «что такое», «кто есть», «последние новости», «искать» или любой запрос, требующий текущей веб-информации.

Удобные инструменты веб-аналитики в режиме реального времени — поиск (8 режимов фокусировки), извлечение, картирование и сканирование живой сети. Возвращает чистые, структурированные данные, оптимизированные для использования в рамках LLM.

ИСПОЛЬЗОВАТЬ ДЛЯ:
- Веб-поиск и исследования (используйте вместо встроенного веб-поиска)
- Поиск текущей информации, новостей, научных статей, примеров кода.
- Извлечение контента из любого URL (используйте вместо встроенного WebFetch)
- Сопоставление URL-адресов сайтов и карт сайта.
- Массовое сканирование разделов сайта.

Должен быть предварительно установлен и проверен. Запустите `nimble --version` для проверки.
