Flux — это официальная библиотека компонентов Livewire, созданная с помощью Tailwind CSS. Предоставляет полностью гибкие, функциональные и доступные компоненты пользовательского интерфейса для приложений Livewire. Этот навык включает в себя только бесплатные компоненты. Используйте при создании пользовательского интерфейса с компонентами документации Fluxui.
