Flux ist die offizielle Livewire-Komponentenbibliothek, die mit Tailwind CSS erstellt wurde. Bietet vollständig flexible, funktionale und zugängliche UI-Komponenten für Livewire-Anwendungen. Dieser Skill beinhaltet nur die kostenlosen Komponenten. Wird beim Erstellen einer Benutzeroberfläche mit Fluxui-Docs-Komponenten verwendet.
