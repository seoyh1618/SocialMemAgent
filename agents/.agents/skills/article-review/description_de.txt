Verfassen Sie eine ausführliche Kritik/Interpretation basierend auf dem Originaltext. Verwenden Sie diese Fähigkeit, wenn ein Benutzer einen Artikel, einen Blog-Beitrag, einen WeChat-Artikel oder einen längeren Inhalt bereitstellt und eine Bewertung, Interpretation, Reflexion oder abgeleitete Arbeit anfordert. Geeignet für: (1) eingehende Analyse von Fachartikeln, Branchenberichten, Jahresendzusammenfassungen usw.; (2) die Kerngedanken des Textes herausarbeiten und sie in einfacher Sprache wiedergeben; (3) Erstellen sekundärer Inhalte, die für das Teilen in sozialen Medien optimiert sind.
