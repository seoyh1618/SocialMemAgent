استخدم whoo CLI لاسترداد وتفسير البيانات الصحية الخاصة بـ WHOOP: درجة الاسترداد، ومعدل ضربات القلب، وجودة النوم، والإجهاد، وSpO2، وقياسات الجسم. يتم الاستدعاء عندما يسأل المستخدم عن مقاييس WHOOP، أو الاستعداد، أو استعادة اللياقة البدنية، أو أداء النوم، أو البيانات الصحية التي يمكن ارتداؤها، أو يريد سحب بيانات WHOOP أو تحليلها لأي نطاق زمني.
