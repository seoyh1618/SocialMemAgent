Utilice whoo CLI para recuperar e interpretar datos de salud de WHOOP: puntuación de recuperación, VFC, calidad del sueño, tensión, SpO2 y mediciones corporales. Invoque cuando el usuario pregunte sobre sus métricas de WHOOP, preparación, recuperación física, rendimiento del sueño, datos de salud portátiles o quiera extraer o analizar datos de WHOOP para cualquier rango de fechas.
