Conjunto de herramientas completo para validar, identificar, probar y automatizar manuales, roles y colecciones de Ansible. Utilice esta habilidad cuando trabaje con archivos Ansible (.yml, libros de estrategias .yaml, roles, inventarios), valide código de automatización, depure la ejecución del libro de estrategias, realice pruebas de ensayo con el modo de verificación o trabaje con módulos y colecciones personalizados.
