Kit di strumenti completo per la convalida, l'linting, il test e l'automazione di playbook, ruoli e raccolte Ansible. Utilizza questa competenza quando lavori con file Ansible (playbook .yml, .yaml, ruoli, inventari), convalidi il codice di automazione, esegui il debug dell'esecuzione di playbook, esegui test di prova con la modalità di controllo o lavori con moduli e raccolte personalizzate.
