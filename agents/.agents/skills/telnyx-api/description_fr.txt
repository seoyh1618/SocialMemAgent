Utilisez cette compétence lorsque vous travaillez avec les API de communication Telnyx pour la messagerie SMS/MMS, les appels vocaux, la gestion des numéros de téléphone, les profils de messagerie, l'intégration de webhooks, les conversations SMS bidirectionnelles, l'envoi groupé, la planification de messages ou le déploiement en production de fonctionnalités de téléphonie. La v2.0 comprend la documentation officielle de Telnyx, les modèles de production de Twilio-Aldea (validation de signature Ed25519, webhooks indépendants du fournisseur, idempotence) et des exemples de code TypeScript.
