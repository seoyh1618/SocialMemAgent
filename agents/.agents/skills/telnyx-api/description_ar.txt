استخدم هذه المهارة عند العمل مع واجهات برمجة تطبيقات اتصالات Telnyx لرسائل SMS/MMS، أو المكالمات الصوتية، أو إدارة أرقام الهاتف، أو ملفات تعريف المراسلة، أو تكامل خطاف الويب، أو محادثات الرسائل النصية القصيرة ثنائية الاتجاه، أو الإرسال المجمع، أو جدولة الرسائل، أو نشر إنتاج ميزات الهاتف. يتضمن الإصدار 2.0 وثائق Telnyx الرسمية، وأنماط الإنتاج من Twilio-Aldea (التحقق من صحة التوقيع Ed25519، وخطافات الويب المحايدة للموفر، والعجز)، وأمثلة كود TypeScript.
