Используйте этот навык при работе с коммуникационными API Telnyx для обмена SMS/MMS-сообщениями, голосовыми вызовами, управлением номерами телефонов, профилями обмена сообщениями, интеграцией веб-перехватчиков, двусторонними SMS-разговорами, массовой отправкой, планированием сообщений или производственным развертыванием функций телефонии. Версия 2.0 включает официальную документацию Telnyx, производственные шаблоны от Twilio-Aldea (проверка подписи Ed25519, веб-перехватчики, не зависящие от поставщика, идемпотентность) и примеры кода TypeScript.
