Utilice esta habilidad cuando trabaje con las API de comunicación de Telnyx para mensajería SMS/MMS, llamadas de voz, administración de números de teléfono, perfiles de mensajería, integración de webhooks, conversaciones SMS bidireccionales, envío masivo, programación de mensajes o implementación de producción de funciones de telefonía. v2.0 incluye documentación oficial de Telnyx, patrones de producción de Twilio-Aldea (validación de firma Ed25519, webhooks independientes del proveedor, idempotencia) y ejemplos de código TypeScript.
