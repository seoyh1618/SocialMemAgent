Construit, intègre et déploie des backends Convex dans Next.js (App Router par défaut, Pages Router pris en charge). Couvre l'exécution de `npx convex dev`, la définition du schéma/index Convex, l'écriture de requêtes/mutations/actions, le câblage des hooks ConvexReactClient + ConvexProvider (useQuery/useMutation), le préchargement SSR facultatif et les actions serveur via convex/nextjs, les modèles d'authentification et le déploiement (y compris les déploiements Vercel + aperçu). À utiliser lorsque l'utilisateur mentionne Convex, convex.dev, Next.js + Convex, les mises à jour de bases de données en temps réel, les fonctions Convex, les schémas/index, NEXT_PUBLIC_CONVEX_URL, le développement/déploiement convexe ou le dépannage de Convex dans une application Next.js.
