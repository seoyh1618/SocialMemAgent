Exploitez les environnements cryptés dotenc et le contrôle d'accès dans les référentiels qui utilisent dotenc (dépôts d'application utilisant dotenc, pas le référentiel de code source dotenc lui-même). À utiliser lorsque les utilisateurs doivent initialiser dotenc, créer/modifier/répertorier des environnements, exécuter des commandes avec des secrets injectés, gérer des clés publiques, accorder/révoquer l'accès, déconnecter des coéquipiers, guider des choix explicites d'installation/de mise à jour opt-in, installer des intégrations d'agent/éditeur dotenc ou dépanner les flux de travail CLI dotenc.
