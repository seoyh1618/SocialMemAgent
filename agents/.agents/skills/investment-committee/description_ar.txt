قم بتشكيل لجنة استثمار رفيعة المستوى حيث يقوم ثلاثة خبراء افتراضيين على غرار مستثمرين أسطوريين (بافيت، وود، دروكنميلر) بإجراء مناقشات عدائية مستقلة ومتعددة الجولات. حقق تفكيرًا مستقلاً حقًا عبر مكالمات Gemini API المعزولة ماديًا وتوصل إلى قرار نهائي بأغلبية الأصوات. يُستخدم عند تقييم قرارات الاستثمار، أو مراجعة تقارير أبحاث الأسهم، أو البحث عن تحليل متعدد وجهات النظر حول الشركات العامة.
