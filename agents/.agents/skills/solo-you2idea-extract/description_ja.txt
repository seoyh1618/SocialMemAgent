Solograph MCP を介して YouTube 動画からスタートアップのアイデアを抽出します。ビジネス アイデアのビデオ トランスクリプトにインデックスを付け、検索し、分析します。マルチ MCP 調整パターン (YouTube ソース → 分析 → ストレージ)。ユーザーが「YouTube からアイデアを抽出する」、「YouTube ビデオにインデックスを付ける」、「ビデオからスタートアップのアイデアを見つける」、「YouTube でアイデアを分析する」、または「このビデオにどのようなアイデアがあるか」と言うときに使用します。一般的な YouTube の視聴 (スキルは必要ありません) やコンテンツの作成 (/content-gen を使用) には使用しないでください。
