Извлекайте идеи для стартапов из видеороликов YouTube с помощью Solograph MCP — индексируйте, ищите и анализируйте расшифровки видео на предмет бизнес-идей. Схема координации нескольких MCP (источник YouTube → анализ → хранилище). Используйте, когда пользователь говорит «извлечь идеи из YouTube», «индексировать видео YouTube», «найти идеи для стартапа в видео», «проанализировать идеи на YouTube» или «какие идеи содержатся в этом видео». НЕ используйте для обычного просмотра YouTube (навыки не требуются) или создания контента (используйте /content-gen).
