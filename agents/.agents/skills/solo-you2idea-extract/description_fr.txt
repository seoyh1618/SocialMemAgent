Extrayez des idées de démarrage à partir de vidéos YouTube via Solograph MCP : indexez, recherchez et analysez les transcriptions vidéo des idées commerciales. Modèle de coordination multi-MCP (source YouTube → analyse → stockage). À utiliser lorsque l'utilisateur dit "extraire des idées de YouTube", "indexer une vidéo YouTube", "trouver des idées de démarrage dans une vidéo", "analyser des idées sur YouTube" ou "quelles sont les idées dans cette vidéo". Ne PAS utiliser pour regarder YouTube en général (aucune compétence requise) ou pour créer du contenu (utilisez /content-gen).
