На основе навыка создания задач рабочего процесса SpeckKit, используемого для создания действенных задач, упорядоченных по зависимостям, из доступных проектных документов. Используйте этот навык, когда необходимо создать подробный список задач для разработки функций на основе проектных документов, таких как spec.md, plan.md, data-model.md, Contracts/ и т. д. К триггерным словам относятся «задачи Speckit», «генерация задач», «планирование задач», «разбивка задач функций», «создание задач.md» и т. д.
