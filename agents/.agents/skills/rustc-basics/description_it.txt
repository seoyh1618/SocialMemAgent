Abilità nel compilatore Rust per la programmazione di sistemi. Da utilizzare quando si seleziona RUSTFLAGS, si configurano i profili Cargo, si ottimizzano le build di rilascio, si leggono assembly o output MIR, si comprende la monomorfizzazione o si diagnosticano errori di compilazione. Si attiva su query su flag Rust, profili Cargo.toml, opt-level, LTO, unità codegen, target-cpu, emit asm, MIR o prestazioni di build Rust.
