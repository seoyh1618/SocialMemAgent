Авторские определения схемы JSON для использования с проверкой z-схемы. Используйте, когда пользователю нужно написать схему JSON, определить схему для полезных данных API, создать схемы для проверки формы, структурировать схемы с помощью $ref и $defs, выбрать между oneOf/anyOf/if-then-else, разработать схемы объектов с обязательными и дополнительными свойствами, проверить массивы с элементами или prefixItems, добавить ограничения формата, организовать схемы для повторного использования или написать схемы Draft-2020-12.
