Una habilidad que genera metadatos (títulos, descripciones, hashtags) para videos de YouTube que explican tecnología. Aprovecha la psicología de aversión a las pérdidas basada en la teoría de las perspectivas para títulos magnéticos de clics, elabora descripciones fáciles de leer para el espectador (con capítulos, marcas de tiempo y enlaces sociales) y selecciona hashtags sistemáticos. Úselo para solicitudes como "Crear un título de YouTube" o "Generar una descripción de video".
