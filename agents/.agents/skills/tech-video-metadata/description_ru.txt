Навык, который генерирует метаданные (заголовки, описания, хэштеги) для видеороликов YouTube с техническими пояснениями. Он использует теоретическую психологию неприятия потерь для заголовков, притягивающих клики, создает удобные для просмотра описания (с главами, метками времени и социальными ссылками) и выбирает систематические хэштеги. Используйте его для таких запросов, как «Создать заголовок YouTube» или «Создать описание видео».
