Panoramica dell'architettura del modulo Core della pipeline di processo, che copre il CRUD della pipeline, il motore di esecuzione della build, il meccanismo basato sugli eventi e la progettazione dell'architettura a più livelli. Utilizzato quando gli utenti sviluppano le funzionalità principali della pipeline, comprendono il modulo Processo, modificano la logica di creazione o conducono uno sviluppo correlato alla pipeline.
