Présentation de l'architecture du module de base du pipeline de processus, couvrant le pipeline CRUD, le moteur d'exécution de build, le mécanisme piloté par les événements et la conception d'architecture en couches. Utilisé lorsque les utilisateurs développent des fonctionnalités principales du pipeline, comprennent le module Processus, modifient la logique de construction ou effectuent un développement lié au pipeline.
