Applica le migliori pratiche di runtime in stile Vue-3 per wevu nei mini-programmi. Da utilizzare durante l'implementazione di pagine/componenti/negozi con wevu, la definizione di hook del ciclo di vita, la gestione del comportamento diff setData, la progettazione di flussi props/emit e bindModel, l'integrazione con macro JSON SFC weapp-vite o la risoluzione di problemi di compatibilità rispetto a Vue 3.
