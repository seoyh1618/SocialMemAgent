قم بتطبيق أفضل ممارسات وقت التشغيل بأسلوب Vue-3 لـ wevu في البرامج المصغرة. يُستخدم عند تنفيذ الصفحات/المكونات/المتاجر باستخدام wevu، أو تحديد خطافات دورة الحياة، أو التعامل مع سلوك فرق setData، أو تصميم تدفقات الدعائم/الإصدار وbindModel، أو التكامل مع وحدات ماكرو weapp-vite SFC JSON، أو استكشاف أخطاء اختلافات التوافق وإصلاحها مقابل Vue 3.
