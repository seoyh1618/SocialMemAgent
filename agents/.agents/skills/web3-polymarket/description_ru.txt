Интеграция Polymarket для прогнозной рыночной торговли на Polygon. Охватывает аутентификацию (L1 EIP-712, L2 HMAC-SHA256, заголовки конструктора), размещение заказов (GTC/GTD/FOK/FAK, пакетный режим, только после публикации, контрольное сообщение), рыночные данные (Gamma API, API данных, книга заказов, подграф), потоковую передачу WebSocket (каналы рынка/пользователя/спорта), операции CTF (разделение, объединение, погашение, отрицательный риск), мост (депозиты, снятие средств, мультичейн) и безгазовые релейные транзакции. Используйте при создании агентов искусственного интеллекта, автономных маркет-мейкеров, пользовательских интерфейсов рынка прогнозов или любого приложения, интегрируемого с Polymarket на Polygon.
