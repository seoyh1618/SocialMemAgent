Integración de Polymarket para el comercio de mercado de predicción en Polygon. Cubre autenticación (L1 EIP-712, L2 HMAC-SHA256, encabezados de generador), colocación de pedidos (GTC/GTD/FOK/FAK, lote, solo publicación, latido), datos de mercado (Gamma API, Data API, libro de pedidos, subgrafo), transmisión WebSocket (mercado/usuario/canales deportivos), operaciones CTF (división, fusión, canje, riesgo negativo), puente (depósitos, retiros, múltiples cadenas) y sin gas. transacciones de retransmisión. Úselo al crear agentes de IA, creadores de mercado autónomos, interfaces de usuario de mercado de predicción o cualquier aplicación que se integre con Polymarket en Polygon.
