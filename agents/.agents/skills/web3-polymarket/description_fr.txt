Intégration de Polymarket pour le trading de prédiction sur le marché sur Polygon. Couvre l'authentification (L1 EIP-712, L2 HMAC-SHA256, en-têtes de constructeur), le placement des commandes (GTC/GTD/FOK/FAK, batch, post-only, heartbeat), les données de marché (Gamma API, Data API, orderbook, subgraph), le streaming WebSocket (marché/utilisateur/chaînes sportives), les opérations CTF (fractionnement, fusion, rachat, risque négatif), le pont (dépôts, retraits, multi-chaînes) et le relais sans gaz. transactions. À utiliser lors de la création d'agents d'IA, de teneurs de marché autonomes, d'interfaces utilisateur de marché de prédiction ou de toute application s'intégrant à Polymarket sur Polygon.
