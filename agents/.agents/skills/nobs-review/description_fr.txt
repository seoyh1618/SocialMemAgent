Révision de code multimodèle pour les demandes d'extraction et les différences git. À utiliser lorsqu'on vous demande de « réviser mon code », « réviser ce PR », « réviser mes modifications », « trouver des réviseurs » ou « exécuter une révision de code ». Suggère des réviseurs appropriés en fonction des fichiers modifiés, exécute des cycles de révision itératifs sur plusieurs modèles d'IA et suit les résultats au fil des sessions.
