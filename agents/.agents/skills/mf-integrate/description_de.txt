Integrieren Sie die Modulföderation in ein vorhandenes Projekt – fügen Sie die Konfiguration eines Anbieters (macht Module verfügbar) oder eines Verbrauchers (lädt Remote-Module) hinzu. Verwenden Sie diese Option, wenn der Benutzer Module Federation zu einem vorhandenen Rsbuild-/Rspack-/Webpack-/Modern.js-/Next.js-/Vite-Projekt hinzufügen, eine Remote-Verbindung einrichten, eine Host-App erstellen oder schnell den Demo-Anbieter nutzen möchte, um zu sehen, wie MF funktioniert. Die Standardrolle ist Verbraucher.
