Intégrez la fédération de modules dans un projet existant : ajoutez une configuration de fournisseur (expose les modules) ou de consommateur (charge les modules distants). À utiliser lorsque l'utilisateur souhaite ajouter une fédération de modules à un projet Rsbuild / Rspack / Webpack / Modern.js / Next.js / Vite existant, configurer une télécommande, créer une application hôte ou utiliser rapidement le fournisseur de démonstration pour voir MF fonctionner. Le rôle par défaut est consommateur.
