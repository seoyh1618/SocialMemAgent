Integre la Federación de módulos en un proyecto existente: agregue la configuración del proveedor (expone los módulos) o del consumidor (carga los módulos remotos). Úselo cuando el usuario desee agregar Module Federation a un proyecto Rsbuild/Rspack/Webpack/Modern.js/Next.js/Vite existente, configurar un control remoto, crear una aplicación host o consumir rápidamente el proveedor de demostración para ver cómo funciona MF. El rol predeterminado es consumidor.
