Когда пользователь явно просит «написать/доработать реферат предложения NSFC», «создать реферат на китайском и английском языках» или «перевести реферат с китайского на английский», используйте это.

Выведите как китайскую, так и английскую версии: английский должен быть точным переводом китайского (без дополнительной информации, без упущения ключевых ограничений).
Ограничения на длину по умолчанию: китайский ≤400 символов (включая знаки пунктуации), английский язык ≤4000 символов (включая знаки препинания); окончательные ограничения следуют за `skills/nsfc-abstract/config.yaml:limits`.

Метод вывода: запишите результат в файл `NSFC-ABSTRACTS.md` в рабочем каталоге (имя файла для `config.yaml:output.filename`), с одним заголовком `#` для китайского языка и одним заголовком `#` для английского языка.

⚠️ Не применимо в следующих случаях:
- Пользователь хочет перевести только общий текст, не связанный с предложением (перевести напрямую).
- Пользователь хочет написать только обоснование/содержание исследования/фундамент исследования (используйте соответствующий навык серии nsfc)
