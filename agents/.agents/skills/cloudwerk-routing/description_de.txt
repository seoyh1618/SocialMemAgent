Dateibasierte Routing-Konventionen für Cloudwerk, ein Full-Stack-Framework für Cloudflare Workers. Wird beim Erstellen von Seiten, API-Routen, Layouts, Middleware oder Fehlergrenzen verwendet. Löst Aufgaben aus, die Routendateien (page.tsx, route.ts, layout.tsx, middleware.ts, error.tsx, not-found.tsx), dynamische Segmente ([id], [...slug]), Routengruppen ((group)), Loader, Aktionen oder Datenlademuster betreffen.
