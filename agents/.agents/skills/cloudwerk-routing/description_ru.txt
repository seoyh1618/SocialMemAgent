Соглашения о маршрутизации на основе файлов для Cloudwerk, полнофункциональной платформы для Cloudflare Workers. Используйте при создании страниц, маршрутов API, макетов, промежуточного программного обеспечения или границ ошибок. Запускается по задачам, включающим файлы маршрутов (page.tsx,route.ts,layout.tsx, middleware.ts, error.tsx, not-found.tsx), динамические сегменты ([id], [...slug]), группы маршрутов ((group)), загрузчики, действия или шаблоны загрузки данных.
