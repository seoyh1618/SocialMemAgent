اصطلاحات التوجيه المستندة إلى الملفات لـ Cloudwerk، وهو إطار عمل متكامل لـ Cloudflare Workers. يُستخدم عند إنشاء الصفحات أو مسارات واجهة برمجة التطبيقات أو التخطيطات أو البرامج الوسيطة أو حدود الأخطاء. تشغيل المهام التي تتضمن ملفات المسار (page.tsx، وroute.ts، وlayout.tsx، و middleware.ts، وerror.tsx، وnot-found.tsx)، أو المقاطع الديناميكية ([id]، [...slug])، أو مجموعات المسار ((group)) أو أدوات التحميل، أو الإجراءات، أو أنماط تحميل البيانات.
