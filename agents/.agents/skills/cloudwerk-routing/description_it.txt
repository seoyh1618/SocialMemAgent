Convenzioni di routing basate su file per Cloudwerk, un framework full-stack per Cloudflare Workers. Da utilizzare durante la creazione di pagine, percorsi API, layout, middleware o limiti di errore. Si attiva su attività che coinvolgono file di instradamento (page.tsx, route.ts, layout.tsx, middleware.ts, error.tsx, not-found.tsx), segmenti dinamici ([id], [...slug]), gruppi di instradamenti ((gruppo)), caricatori, azioni o modelli di caricamento dei dati.
