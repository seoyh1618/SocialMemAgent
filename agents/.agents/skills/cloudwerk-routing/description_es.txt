Convenciones de enrutamiento basadas en archivos para Cloudwerk, un marco completo para Cloudflare Workers. Úselo al crear páginas, rutas API, diseños, middleware o límites de errores. Activadores de tareas que involucran archivos de ruta (page.tsx, route.ts, layout.tsx, middleware.ts, error.tsx, not-found.tsx), segmentos dinámicos ([id], [...slug]), grupos de rutas ((group)), cargadores, acciones o patrones de carga de datos.
