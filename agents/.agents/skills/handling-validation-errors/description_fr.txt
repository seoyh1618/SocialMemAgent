Inspecte, filtre et mappe les erreurs de validation du schéma Z pour l'utilisation des applications. À utiliser lorsque l'utilisateur doit gérer les erreurs de validation, parcourir les erreurs internes imbriquées à partir des combinateurs anyOf/oneOf/not, mapper les codes d'erreur à des messages conviviaux, filtrer les erreurs avec includeErrors ouexclusErrors, créer des mappeurs d'erreurs de champ de formulaire, utiliser reportPathAsArray, interpréter les champs SchemaErrorDetail comme code/path/keyword/inner, ou déboguer pourquoi la validation a échoué.
