Проверяет, фильтрует и сопоставляет ошибки проверки z-схемы для использования приложением. Используйте, когда пользователю необходимо обрабатывать ошибки проверки, просматривать вложенные внутренние ошибки из комбинаторов AnyOf/oneOf/not, сопоставлять коды ошибок с удобными для пользователя сообщениями, фильтровать ошибки с помощью includeErrors или ignoreErrors, создавать средства сопоставления ошибок полей формы, использовать reportPathAsArray, интерпретировать поля SchemaErrorDetail, такие как код/путь/ключевое слово/внутренний, или отлаживать причину сбоя проверки.
