Arbeiten mit Microsoft Access-Datenbanken, die über VCS (Version Control System for Access) exportiert wurden. Zum Bearbeiten von Access-Formulardefinitionen (.bas), Berichtsdefinitionen, Abfragedateien, VBA-Modulen, verknüpften Tabellendefinitionen oder VCS-Konfigurationsdateien. Behandelt Strukturregeln, häufige Fallstricke und sichere Bearbeitungspraktiken für das binäre Textformat, das von LoadFromText/SaveAsText von Access verwendet wird.
