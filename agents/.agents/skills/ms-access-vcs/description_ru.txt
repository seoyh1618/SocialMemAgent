Работа с базами данных Microsoft Access, экспортированными через VCS (систему контроля версий для Access). Используйте при редактировании определений форм Access (.bas), определений отчетов, файлов запросов, модулей VBA, определений связанных таблиц или файлов конфигурации VCS. Охватывает структурные правила, распространенные ошибки и методы безопасного редактирования двоичного текстового формата, используемого AccessFromText/SaveAsText.
