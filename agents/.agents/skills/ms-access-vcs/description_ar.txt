العمل مع قواعد بيانات Microsoft Access المصدرة عبر VCS (نظام التحكم في الإصدار للوصول). يُستخدم عند تحرير تعريفات نماذج Access (.bas)، أو تعريفات التقارير، أو ملفات الاستعلام، أو وحدات VBA النمطية، أو تعريفات الجداول المرتبطة، أو ملفات تكوين VCS. يغطي القواعد الهيكلية، والأخطاء الشائعة، وممارسات التحرير الآمنة لتنسيق النص الثنائي الذي يستخدمه LoadFromText/SaveAsText في Access.
