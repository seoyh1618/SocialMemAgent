Lavorare con database Microsoft Access esportati tramite VCS (Version Control System for Access). Da utilizzare durante la modifica delle definizioni dei moduli di accesso (.bas), delle definizioni dei report, dei file di query, dei moduli VBA, delle definizioni delle tabelle collegate o dei file di configurazione VCS. Copre regole strutturali, trucchi comuni e pratiche di modifica sicure per il formato di testo binario utilizzato da LoadFromText/SaveAsText di Access.
