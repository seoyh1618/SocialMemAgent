Erstellen und verwalten Sie GitHub-Problemformularvorlagen mithilfe der YAML-Syntax. Verwenden Sie diese Option, wenn Sie benutzerdefinierte Problemformulare für GitHub-Repositorys mit bestimmten Feldtypen, Validierungen, Standardbezeichnungen und Beauftragten entwerfen müssen. Hilft beim Erstellen von Fehlerberichten, Funktionsanfragen, Dokumentationsproblemen und benutzerdefinierten Arbeitsabläufen. Enthält YAML-Syntaxanleitungen, Feldtypreferenzen, Validierungsmuster, Best Practices und gebrauchsfertige Vorlagen.
