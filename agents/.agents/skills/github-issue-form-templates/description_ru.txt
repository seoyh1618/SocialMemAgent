Создавайте и управляйте шаблонами форм задач GitHub, используя синтаксис YAML. Используйте, когда вам нужно разработать собственные формы задач для репозиториев GitHub с определенными типами полей, проверками, метками по умолчанию и правопреемниками. Помогает создавать отчеты об ошибках, запросы функций, проблемы с документацией и настраиваемые рабочие процессы. Включает руководство по синтаксису YAML, справочник по типам полей, шаблоны проверки, рекомендации и готовые к использованию шаблоны.
