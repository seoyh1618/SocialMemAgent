Cree y administre plantillas de formularios de incidencias de GitHub utilizando la sintaxis YAML. Úselo cuando necesite diseñar formularios de incidencias personalizados para repositorios de GitHub con tipos de campos, validaciones, etiquetas predeterminadas y asignados específicos. Ayuda a crear informes de errores, solicitudes de funciones, problemas de documentación y flujos de trabajo personalizados. Incluye guía de sintaxis de YAML, referencia de tipos de campos, patrones de validación, mejores prácticas y plantillas listas para usar.
