Utilisez cette compétence lors de la création d’applications de streaming bidirectionnel en temps réel avec l’API Gemini Live. Couvre le streaming audio/vidéo/texte basé sur WebSocket, la détection d'activité vocale (VAD), les fonctionnalités audio natives, les appels de fonctions, la gestion de session, les jetons éphémères pour l'authentification côté client et toutes les options de configuration de l'API Live. SDK couverts - google-genai (Python), @google/genai (JavaScript/TypeScript).
