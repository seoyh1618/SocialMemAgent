استخدم هذه المهارة عند إنشاء تطبيقات بث ثنائية الاتجاه في الوقت الفعلي باستخدام Gemini Live API. يغطي تدفق الصوت/الفيديو/النص المستند إلى WebSocket، واكتشاف النشاط الصوتي (VAD)، وميزات الصوت الأصلية، واستدعاء الوظائف، وإدارة الجلسة، والرموز المؤقتة للمصادقة من جانب العميل، وجميع خيارات تكوين Live API. حزم SDK المغطاة - google-genai (Python)، @google/genai (JavaScript/TypeScript).
