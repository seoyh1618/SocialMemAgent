Используйте этот навык при создании приложений двунаправленной потоковой передачи в реальном времени с помощью Gemini Live API. Охватывает потоковую передачу аудио/видео/текста на основе WebSocket, обнаружение голосовой активности (VAD), собственные функции звука, вызов функций, управление сеансами, эфемерные токены для аутентификации на стороне клиента и все параметры конфигурации Live API. Охваченные SDK: google-genai (Python), @google/genai (JavaScript/TypeScript).
