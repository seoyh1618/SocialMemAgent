Utilizza questa competenza quando crei applicazioni di streaming bidirezionali in tempo reale con l'API Gemini Live. Copre streaming audio/video/testo basato su WebSocket, rilevamento dell'attività vocale (VAD), funzionalità audio native, chiamata di funzioni, gestione delle sessioni, token temporanei per l'autenticazione lato client e tutte le opzioni di configurazione dell'API Live. SDK coperti: google-genai (Python), @google/genai (JavaScript/TypeScript).
