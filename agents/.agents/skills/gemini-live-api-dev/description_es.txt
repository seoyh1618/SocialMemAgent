Utilice esta habilidad al crear aplicaciones de transmisión bidireccional en tiempo real con la API Gemini Live. Cubre la transmisión de audio/video/texto basada en WebSocket, detección de actividad de voz (VAD), funciones de audio nativas, llamadas de funciones, administración de sesiones, tokens efímeros para autenticación del lado del cliente y todas las opciones de configuración de Live API. SDK cubiertos: google-genai (Python), @google/genai (JavaScript/TypeScript).
