مهارة توليد وتفسير Flamegraph. يُستخدم عند تحويل الأداء أو Valgrind Callgrind أو مخرجات ملفات التعريف الأخرى إلى مخططات Flamegraphs SVG باستخدام أدوات FlameGraph الخاصة بـ Brendan Gregg، أو عند قراءة مخططات اللهب لتحديد اختناقات الأداء. يتم تنشيطه عند الاستعلامات حول Flamegraphs، أو stackcollapse، أو Flamegraph.svg، أو تحديد الإطارات الساخنة، أو الإطارات العريضة مقابل الإطارات الطويلة، أو تصور الأداء.
