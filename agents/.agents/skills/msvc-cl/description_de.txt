MSVC cl.exe und clang-cl-Skill für Windows C/C++-Projekte. Verwenden Sie diese Option, wenn Sie Visual Studio-Builds, MSBuild oder clang-cl als Drop-in-MSVC-Ersatz konfigurieren. Behandelt die Übersetzung von GCC/Clang-Flags in MSVC-Äquivalente, die Auswahl der Laufzeitbibliothek, die Einrichtung des Windows SDK und die Diagnose von MSVC-spezifischen Fehlern. Wird bei Abfragen zu cl.exe, clang-cl, /O-Flags, /MT vs. /MD, PDB-Dateien, Windows ABI oder MSVC-Projekteinstellungen aktiviert.
