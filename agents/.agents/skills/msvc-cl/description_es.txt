Habilidad MSVC cl.exe y clang-cl para proyectos de Windows C/C++. Úselo al configurar compilaciones de Visual Studio, MSBuild o clang-cl como reemplazo directo de MSVC. Cubre la traducción de indicadores GCC/Clang a equivalentes de MSVC, la selección de bibliotecas en tiempo de ejecución, la configuración del SDK de Windows y el diagnóstico de errores específicos de MSVC. Se activa en consultas sobre cl.exe, clang-cl, indicadores /O, /MT vs /MD, archivos PDB, Windows ABI o configuración de proyectos MSVC.
