Compétence MSVC cl.exe et clang-cl pour les projets Windows C/C++. À utiliser lors de la configuration des builds Visual Studio, MSBuild ou clang-cl en remplacement de MSVC. Couvre la traduction des indicateurs GCC/Clang en équivalents MSVC, la sélection de la bibliothèque d'exécution, la configuration du SDK Windows et le diagnostic des erreurs spécifiques à MSVC. S'active sur les requêtes concernant les indicateurs cl.exe, clang-cl, /O, /MT vs /MD, les fichiers PDB, Windows ABI ou les paramètres de projet MSVC.
