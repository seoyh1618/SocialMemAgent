MSVC cl.exe и навыки clang-cl для проектов Windows C/C++. Используйте при настройке сборок Visual Studio, MSBuild или clang-cl в качестве замены MSVC. Охватывает перевод флагов GCC/Clang в эквиваленты MSVC, выбор библиотеки времени выполнения, настройку Windows SDK и диагностику ошибок, специфичных для MSVC. Активируется по запросам о cl.exe, clang-cl, флагах /O, /MT или /MD, файлах PDB, настройках проекта Windows ABI или MSVC.
