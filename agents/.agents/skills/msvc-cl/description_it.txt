Competenza MSVC cl.exe e clang-cl per progetti Windows C/C++. Da utilizzare durante la configurazione di build di Visual Studio, MSBuild o clang-cl come sostituzione diretta di MSVC. Copre la traduzione dei flag GCC/Clang in equivalenti MSVC, la selezione della libreria di runtime, la configurazione di Windows SDK e la diagnosi di errori specifici di MSVC. Si attiva su query su cl.exe, clang-cl, flag /O, /MT vs /MD, file PDB, Windows ABI o impostazioni del progetto MSVC.
