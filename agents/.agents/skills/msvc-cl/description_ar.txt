MSVC cl.exe ومهارة clang-cl لمشاريع Windows C/C++. يُستخدم عند تكوين إصدارات Visual Studio أو MSBuild أو clang-cl كبديل لـ MSVC. يغطي ترجمة إشارات مجلس التعاون الخليجي/Clang إلى مكافئات MSVC، واختيار مكتبة وقت التشغيل، وإعداد Windows SDK، وتشخيص الأخطاء الخاصة بـ MSVC. يتم التنشيط عند الاستعلامات حول علامات cl.exe أو clang-cl أو /O أو /MT vs /MD أو ملفات PDB أو Windows ABI أو إعدادات مشروع MSVC.
