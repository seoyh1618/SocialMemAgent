Rédaction de requêtes devant le Conseil de prud'hommes pour contester un licenciement pour faute grave. Utilice esta habilidad cuando el usuario solicite redactar una solicitud de CPH, concursar una licencia por falta grave, preparar una saisine du CPH para un litigio sobre una licencia, o crear un documento de impugnación de licencia en derecho de trabajo francés.
