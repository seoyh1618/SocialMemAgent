Lösen Sie offene PR-Überprüfungsdiskussionen, indem Sie Anwendbarkeit und Wert bewerten und dann gültiges Feedback mit herkömmlichen Commits korrigieren. Wird verwendet, wenn (1) das Feedback von Prüfern zu einer Pull-Anfrage bearbeitet wird, (2) offene PR-Diskussionen systematisch gelöst werden, (3) PR-Rezensionskommentare auf ihre Umsetzbarkeit geprüft werden, (4) Feedback zur Codeüberprüfung stapelweise korrigiert wird oder (5) ungelöste PR-Gespräche bereinigt werden.
