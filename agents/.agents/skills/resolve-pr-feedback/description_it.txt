Risolvi le discussioni aperte sulla revisione delle PR valutando l'applicabilità e il valore, quindi fissando feedback validi con impegni convenzionali. Da utilizzare quando (1) si risponde al feedback del revisore su una richiesta pull, (2) si risolvono sistematicamente le discussioni PR aperte, (3) si valutano i commenti di revisione PR per valutarne l'operatività, (4) si correggono in batch il feedback di revisione del codice o (5) si puliscono conversazioni PR irrisolte.
