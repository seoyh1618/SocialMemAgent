Решите открытые дискуссии по обзору PR, оценив применимость и ценность, а затем зафиксировав достоверную обратную связь с помощью обычных обязательств. Используйте, когда (1) обрабатываете отзывы рецензентов по запросу на включение, (2) систематически разрешаете открытые PR-дискуссии, (3) сортируете комментарии по PR-обзорам на предмет целесообразности, (4) пакетно исправляете отзывы о проверке кода или (5) очищаете неразрешенные PR-обсуждения.
