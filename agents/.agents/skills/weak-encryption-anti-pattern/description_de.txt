Sicherheits-Anti-Pattern für schwache Verschlüsselung (CWE-326, CWE-327). Wird zum Generieren oder Überprüfen von Code verwendet, der Daten verschlüsselt, Verschlüsselungsschlüssel verarbeitet oder kryptografische Modi verwendet. Erkennt DES, ECB-Modus, statische IVs und benutzerdefinierte Krypto-Implementierungen.
