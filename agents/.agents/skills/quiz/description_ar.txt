يُنشئ اختبارات قاعدة تعليمات برمجية تفاعلية لاختبار فهم البنية والأنماط وتفاصيل التنفيذ. قابلة للتكوين حسب عدد البطاقات (أقل، قياسي، أكثر) والصعوبة (سهل، متوسط، صعب). يُستخدم عندما يُطلب منك إجراء اختبار لي، أو اختبار معرفتي، أو إنشاء بطاقات تعليمية، أو إنشاء أسئلة قاعدة التعليمات البرمجية، أو إجراء اختبار تعليمات برمجية.
