Génère des quiz interactifs sur la base de code pour tester la compréhension de l'architecture, des modèles et des détails d'implémentation. Configurable par nombre de cartes (moins, standard, plus) et difficulté (facile, moyenne, difficile). À utiliser lorsqu'on me demande de m'interroger, de tester mes connaissances, de créer des flashcards, de générer des questions de base de code ou d'exécuter un quiz de code.
