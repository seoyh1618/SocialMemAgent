Создает интерактивные тесты по кодовой базе для проверки понимания архитектуры, шаблонов и деталей реализации. Настраивается по количеству карт (меньше, стандарт, больше) и сложности (легкая, средняя, ​​сложная). Используйте, когда вас попросят задать мне вопрос, проверить мои знания, создать карточки, сгенерировать вопросы по коду или провести тест по коду.
