يستعلم عن مخطط Genesys Cloud Platform API للحصول على تفاصيل نقطة النهاية والأذونات والمعلمات وتنسيقات الاستجابة. يُستخدم عندما يسأل المستخدم عن نقاط نهاية Genesys Cloud API (على سبيل المثال، "/api/v2/assistants/queues")، أو وثائق واجهة برمجة التطبيقات، أو أذونات واجهة برمجة التطبيقات، أو المخططات، أو الإمكانيات، أو عند تنفيذ عمليات Genesys Cloud.
