Crea ed esegui il debug degli agenti AI Pydantic utilizzando le migliori pratiche per dipendenze, prompt di sistema dinamici, strumenti e convalida dell'output strutturato. Utilizzare quando l'utente desidera: (1) Creare un nuovo agente AI Pydantic, (2) Eseguire il debug o correggere un agente esistente, (3) Aggiungere funzionalità come strumenti, validatori o prompt dinamici, (4) Integrare OpenRouter per l'accesso multi-modello, (5) Aggiungere Logfire per debug/osservabilità, (6) Strutturare l'architettura dell'agente con inserimento delle dipendenze.
