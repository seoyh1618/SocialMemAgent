Cree y depure agentes de IA de Pydantic utilizando las mejores prácticas para dependencias, indicaciones dinámicas del sistema, herramientas y validación de resultados estructurados. Úselo cuando el usuario desee: (1) Crear un nuevo agente de Pydantic AI, (2) Depurar o reparar un agente existente, (3) Agregar características como herramientas, validadores o indicaciones dinámicas, (4) Integrar OpenRouter para acceso a múltiples modelos, (5) Agregar Logfire para depuración/observabilidad, (6) Estructurar la arquitectura del agente con inyección de dependencia.
