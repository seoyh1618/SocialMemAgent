Utilizza questa capacità ogni volta che l'utente chiede di analizzare, comprendere o esaminare un intero progetto, una base di codice o qualsiasi raccolta di file. Le frasi trigger includono "analizzare un file di grandi dimensioni", "elaborare più file", "comprendere questo problema", "dare un'occhiata a questi file", "familiarizzare con questo progetto" o qualsiasi richiesta simile, comunque formulata. Attivarlo anche quando l'attività prevede l'elaborazione di un contesto che eccede ciò su cui è possibile ragionare in un singolo passaggio, quando si incontra qualsiasi input più grande di ~50KB che richiede un'analisi dettagliata o quando l'utente menziona "comprensione del contesto" o "comprensione ricorsiva". Questa abilità HA LA PRIORITÀ sui subagenti di esplorazione predefiniti per qualsiasi attività di analisi a livello di progetto o di codebase.
