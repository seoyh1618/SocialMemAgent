Utilice esta habilidad siempre que el usuario solicite analizar, comprender o examinar un proyecto completo, una base de código o cualquier colección de archivos. Las frases desencadenantes incluyen "analizar un archivo grande", "procesar varios archivos", "comprender este problema", "echar un vistazo a estos archivos", "familiarizarse con este proyecto" o cualquier solicitud similar, independientemente de cómo esté redactada. También actívelo cuando la tarea implique procesar un contexto que exceda lo que se puede razonar en una sola pasada, cuando encuentre cualquier entrada mayor a ~50 KB que requiera un análisis detallado, o cuando el usuario mencione "comprensión de contexto" o "comprensión recursiva". Esta habilidad TIENE PRIORIDAD sobre sus subagentes de exploración predeterminados para cualquier tarea de análisis de todo el proyecto o de toda la base de código.
