Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer ein gesamtes Projekt, eine Codebasis oder eine beliebige Dateisammlung analysieren, verstehen oder überblicken möchte. Zu den Auslöserphrasen gehören „Eine große Datei analysieren“, „Mehrere Dateien verarbeiten“, „Dieses Problem verstehen“, „Sehen Sie sich diese Dateien an“, „Machen Sie sich mit diesem Projekt vertraut“ oder jede ähnliche Aufforderung, wie auch immer sie formuliert ist. Aktivieren Sie diese Option auch, wenn die Aufgabe die Verarbeitung von Kontext umfasst, der über das hinausgeht, was in einem einzigen Durchgang begründet werden kann, wenn Sie auf Eingaben stoßen, die größer als ~50 KB sind und eine detaillierte Analyse erfordern, oder wenn der Benutzer „Kontextverständnis“ oder „rekursives Verständnis“ erwähnt. Diese Fertigkeit hat Vorrang vor Ihren standardmäßigen Erkundungs-Subagenten für jede projektweite oder codebasisweite Analyseaufgabe.
