Utilisez cette compétence chaque fois que l'utilisateur demande à analyser, comprendre ou examiner un projet entier, une base de code ou toute collection de fichiers. Les phrases déclencheurs incluent « analyser un fichier volumineux », « traiter plusieurs fichiers », « comprendre ce problème », « jeter un œil à ces fichiers », « vous familiariser avec ce projet » ou toute demande similaire, quelle que soit sa formulation. Activez également lorsque la tâche implique le traitement d'un contexte qui dépasse ce qui peut être raisonné en un seul passage, lorsque vous rencontrez une entrée supérieure à ~ 50 Ko nécessitant une analyse détaillée, ou lorsque l'utilisateur mentionne « compréhension du contexte » ou « compréhension récursive ». Cette compétence A LA PRIORITÉ sur vos sous-agents d'exploration par défaut pour toute tâche d'analyse à l'échelle du projet ou de la base de code.
