Безопасное внедрение навыков с помощью сканирования безопасности цепочки поставок. Обертывает сканер навыков Cisco
проверить навыки перед установкой. Поддерживает репозитории GitHub,kills.sh (npx), торговую площадку Claude.
плагины и локальные пути. Настраиваемая глубина сканирования со статическим и поведенческим анализом по умолчанию.
Использует загрузку архивов GitHub, чтобы избежать рисков выполнения git, с усиленным резервным копированием git.
Решения по безопасности принимаются детерминированным сценарием-оболочкой, а не агентом LLM.
