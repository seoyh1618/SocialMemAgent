Анализируйте структуру статьи и создавайте пакетные иллюстрации с помощью искусственного интеллекта, вызывая API Gemini для создания профессиональных логических/концептуальных диаграмм для каждого абзаца. Срабатывает, когда пользователь говорит «пакетная иллюстрация», «проиллюстрировать статью», «создать изображения статьи» или «добавить изображения в эту статью». Поддерживает два стиля: минималистичный рисованный и архитектурный. Автоматически сохраняет в папку изображений Obsidian и вставляет в документ.
