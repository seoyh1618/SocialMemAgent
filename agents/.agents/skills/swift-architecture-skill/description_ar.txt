Agent Skill لتصميم وتنفيذ بنية Swift، مع أدلة تشغيل خاصة بالهندسة المعمارية وقوائم مراجعة. يُستخدم عند تصميم ميزات جديدة، أو إعادة بناء الوحدات النمطية الحالية، أو مراجعة طلبات السحب، أو تصحيح أخطاء قابلية الصيانة في مشاريع SwiftUI/UIKit وتحتاج إلى إرشادات ملموسة لأنماط MVVM، أو MVI، أو TCA، أو Clean Architecture، أو VIPER، أو التفاعلية.
