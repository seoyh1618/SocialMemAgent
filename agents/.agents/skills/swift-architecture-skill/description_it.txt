Modelli di progettazione e implementazione dell'architettura Agent Skill for Swift, con playbook specifici dell'architettura ed elenchi di controllo di revisione. Utilizzalo durante la progettazione di nuove funzionalità, il refactoring di moduli esistenti, la revisione delle richieste pull o il debug di problemi di manutenibilità nei progetti SwiftUI/UIKit e hai bisogno di indicazioni concrete per MVVM, MVI, TCA, Clean Architecture, VIPER o modelli reattivi.
