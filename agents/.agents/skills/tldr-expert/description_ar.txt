تلخيص موجز وتقطير المعلومات. يغطي الملخصات التنفيذية، وملخصات تغيير التعليمات البرمجية، وأوصاف العلاقات العامة، وملاحظات الاجتماع، والموجزات الفنية، وأنماط الكشف التدريجي.

يُستخدم عند كتابة أوصاف العلاقات العامة، أو تلخيص تغييرات التعليمات البرمجية، أو إنشاء ملخصات تنفيذية، أو استخلاص المناقشات الطويلة، أو إنشاء ملاحظات الإصدار.
