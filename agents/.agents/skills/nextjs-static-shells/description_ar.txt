16 نمطًا معماريًا من Static-first Next.js: الأصداف المخزنة مؤقتًا ذات الفتحات الديناميكية، وجزر الموفر، وحدود "استخدام ذاكرة التخزين المؤقت"، واستراتيجية التحميل المسبق للارتباط. يُستخدم عند إنشاء مسارات Next.js أو إعادة هيكلتها لزيادة العرض الثابت إلى أقصى حد، أو تنفيذ "استخدام ذاكرة التخزين المؤقت" مع التخصيص الديناميكي، أو تقسيم الإدخال مقابل العارضين الثابتين، أو تحديد نطاق موفري العملاء، أو ضبط سلوك الجلب المسبق. يتم تشغيلها على "الغلاف الثابت"، أو "استخدام نمط ذاكرة التخزين المؤقت"، أو "الفتحات الديناميكية"، أو "جزيرة الموفر"، أو "استراتيجية الجلب المسبق"، أو "الثابت أولاً"، أو "حدود ذاكرة التخزين المؤقت"، أو "يتحول المسار إلى ديناميكي بشكل غير متوقع"، أو أي عمل معماري لـ Next.js يتضمن عرضًا ثابتًا/ديناميكيًا مختلطًا.
