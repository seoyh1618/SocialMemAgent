Умение определять уровень теста QA. Он всесторонне оценивает риск, объединяя описания историй, различия кода и диалоги, а затем предлагает соответствующий уровень контроля качества. Используйте в следующих ситуациях:
- Если вы хотите определить уровень качества PR
- Когда нужно решить, нужен ли тестер или перепроверка
- При настройке уровня для проверки обзора точки обзора.
Также может быть вызван с помощью «оценки качества», «уровня тестирования», «оценки рисков» и т. д.
