Capacità di determinare il livello del test QA. Valuta in modo completo il rischio combinando descrizioni di storie, differenze di codice e dialoghi, quindi propone un livello di QA appropriato. Utilizzare nelle seguenti situazioni:
- Quando vuoi determinare il livello di QA di un PR
- Quando è necessario decidere se è necessario un tester o un doppio controllo
- Quando si imposta il livello per una revisione dal punto di vista del test
Può essere richiamato anche con "giudizio QA", "livello di test", "valutazione del rischio", ecc.
