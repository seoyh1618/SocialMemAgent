Compétence pour déterminer le niveau de test QA. Il évalue de manière exhaustive les risques en combinant des descriptions d'histoires, des différences de code et des dialogues, puis propose un niveau d'assurance qualité approprié. Utiliser dans les situations suivantes :
- Lorsque vous souhaitez déterminer le niveau d'assurance qualité d'un PR
- Lorsque vous devez décider si un testeur ou une double vérification est nécessaire
- Lors de la définition du niveau pour une revue de point de vue de test
Peut également être invoqué avec « jugement AQ », « niveau de test », « évaluation des risques », etc.
