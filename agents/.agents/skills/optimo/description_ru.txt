Оптимизируйте и конвертируйте изображения с помощью интерфейса командной строки и API Optimo поверх ImageMagick. Используйте, когда пользователь упоминает уменьшение размера изображения, сжатие изображения, пакетную оптимизацию изображений, преобразование форматов (jpeg/png/webp/avif/heic/jxl), изменение размера в процентах/размерах/максимальном размере или запуск Optimo в сценариях.
