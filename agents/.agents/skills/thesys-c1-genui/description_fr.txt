Fournit des conseils complets pour créer des applications d’interface utilisateur générative basées sur l’IA avec l’API Thesys C1 et le SDK GenUI. Utilisez-le lors du développement de composants d'interface utilisateur interactifs, d'interfaces de discussion, de tableaux de bord ou de toute application bénéficiant d'interfaces React générées dynamiquement à partir d'invites en langage naturel.
