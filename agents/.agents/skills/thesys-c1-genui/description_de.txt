Bietet umfassende Anleitungen zum Erstellen KI-gestützter generativer UI-Anwendungen mit der Thesys C1 API und dem GenUI SDK. Verwenden Sie es, wenn Sie interaktive UI-Komponenten, Chat-Schnittstellen, Dashboards oder jede andere Anwendung entwickeln, die von dynamisch generierten React-Schnittstellen aus Eingabeaufforderungen in natürlicher Sprache profitiert.
