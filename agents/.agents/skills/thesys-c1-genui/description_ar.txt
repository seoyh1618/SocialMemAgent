يوفر إرشادات شاملة لبناء تطبيقات واجهة المستخدم التوليدية التي تعمل بالذكاء الاصطناعي باستخدام Thesys C1 API وGenUI SDK. استخدمه عند تطوير مكونات واجهة المستخدم التفاعلية، أو واجهات الدردشة، أو لوحات المعلومات، أو أي تطبيق يستفيد من واجهات React التي يتم إنشاؤها ديناميكيًا من خلال مطالبات اللغة الطبيعية.
