Analice descripciones de puestos (JD) para extraer información clave, requisitos de habilidades y calificaciones. Identifique las habilidades básicas, las habilidades interpersonales y los requisitos de experiencia. Se utiliza para comparar currículums, preparar entrevistas y analizar la brecha de habilidades. Se activa automáticamente cuando el usuario proporciona texto JD o agrega información de la empresa.
