استكشاف وتحليل تطبيقات TUI لتوثيق ميزاتها للاستنساخ. استخدمه عندما يُطلب منك إجراء هندسة عكسية لواجهة مستخدم طرفية أو تحليلها أو توثيقها أو فهمها مثل Claude Code أو OpenCode أو Codex أو Lazygit أو أي تطبيق قائم على راتاتوي/نكورسيس. تشغيل TUI الهدف في tmux، واستكشاف جميع طرق العرض وارتباطات المفاتيح بشكل منهجي، والتقاط مخططات ASCII لكل شاشة، وكتابة النتائج بشكل متزايد في ملف تخفيض السعر (البقاء على ضغط السياق).
