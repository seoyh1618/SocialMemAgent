Explorez et analysez les applications TUI pour documenter leurs fonctionnalités pour le clonage. À utiliser lorsqu'on vous demande de faire de l'ingénierie inverse, d'analyser, de documenter ou de comprendre une interface utilisateur de terminal comme Claude Code, OpenCode, Codex, lazygit ou toute application basée sur ratatui/ncurses. Lance la TUI cible dans tmux, explore systématiquement toutes les vues et combinaisons de touches, capture les diagrammes ASCII de chaque écran et écrit les résultats de manière incrémentielle dans un fichier markdown (survit au compactage du contexte).
