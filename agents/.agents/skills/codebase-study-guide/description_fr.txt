Générez un guide d’étude pédagogique pour apprendre une base de code inconnue. À utiliser lorsque l'utilisateur souhaite intégrer une base de code, comprendre l'architecture d'un projet, créer du matériel d'apprentissage pour une équipe ou demander des choses telles que « aidez-moi à apprendre cette base de code », « créer un guide d'intégration », « Je suis nouveau dans ce projet », « comment fonctionne ce système », « guide d'étude pour ce dépôt » ou « expliquez-moi cette base de code ». Produit un document structuré qui développe la compréhension du but aux systèmes en passant par les modèles, en utilisant des techniques d'apprentissage fondées sur des preuves (interrogation élaborée, cartographie conceptuelle, concepts de seuil, exemples concrets, divulgation progressive).
