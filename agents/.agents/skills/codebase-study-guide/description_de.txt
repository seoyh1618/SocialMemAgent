Erstellen Sie einen pädagogisch fundierten Studienführer zum Erlernen einer unbekannten Codebasis. Verwenden Sie diese Option, wenn der Benutzer in eine Codebasis einsteigen, die Architektur eines Projekts verstehen, Lernmaterialien für ein Team erstellen möchte oder Dinge fragt wie „Helfen Sie mir, diese Codebasis zu erlernen“, „Erstellen Sie einen Onboarding-Leitfaden“, „Ich bin neu in diesem Projekt“, „Wie funktioniert dieses System“, „Studienleitfaden für dieses Repo“ oder „Erklären Sie mir diese Codebasis“. Erstellt ein strukturiertes Dokument, das mithilfe evidenzbasierter Lerntechniken (ausführliche Befragung, Konzeptzuordnung, Schwellenwertkonzepte, erarbeitete Beispiele, progressive Offenlegung) ein Verständnis vom Zweck über Systeme bis hin zu Mustern aufbaut.
