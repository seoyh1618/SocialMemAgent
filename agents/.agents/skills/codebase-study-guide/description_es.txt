Genere una guía de estudio con base pedagógica para aprender una base de código desconocida. Úselo cuando el usuario quiera incorporarse a un código base, comprender la arquitectura de un proyecto, crear materiales de aprendizaje para un equipo o pedir cosas como "ayúdame a aprender este código base", "crear una guía de incorporación", "soy nuevo en este proyecto", "cómo funciona este sistema", "guía de estudio para este repositorio" o "explicarme este código base". Produce un documento estructurado que desarrolla la comprensión desde el propósito hasta los sistemas y los patrones, utilizando técnicas de aprendizaje basadas en evidencia (interrogatorio elaborado, mapeo conceptual, conceptos umbral, ejemplos resueltos, divulgación progresiva).
