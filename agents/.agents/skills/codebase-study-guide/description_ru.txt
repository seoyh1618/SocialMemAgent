Создайте педагогически обоснованное учебное пособие для изучения незнакомой кодовой базы. Используйте, когда пользователь хочет подключиться к базе кода, понять архитектуру проекта, создать учебные материалы для команды или спросить что-то вроде «помогите мне изучить эту базу кода», «создать руководство по адаптации», «Я новичок в этом проекте», «как работает эта система», «учебное пособие для этого репозитория» или «объясните мне эту кодовую базу». Создает структурированный документ, который строит понимание от целей к системам и шаблонам, используя методы обучения, основанные на фактических данных (подробный опрос, картирование понятий, пороговые понятия, рабочие примеры, постепенное раскрытие).
