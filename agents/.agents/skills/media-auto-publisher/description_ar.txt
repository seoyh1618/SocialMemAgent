أداة عالمية للنشر التلقائي لمقالات الوسائط الذاتية. يدعم سير عمل النشر الآلي لـ Baijiahao وSohu وZhihu. يستخدم أداة Playwright MCP للتنقل عبر النظام الأساسي والدخول إلى صفحات النشر، ويدعم تبديل الحساب عبر إدارة ملفات تعريف الارتباط StorageState. استخدم هذه المهارة عندما يحتاج المستخدمون إلى نشر المقالات على منصات الوسائط الذاتية، أو فتح صفحات نشر النظام الأساسي، أو تبديل الحسابات. سير العمل: 1) التحقق من النظام الأساسي المحدد وفتحه 2) التحقق من حالة تسجيل الدخول 3) إغلاق النوافذ المنبثقة للإعلانات تلقائيًا 4) انتقل إلى صفحة نشر المقالة 5) جمع معلومات النشر 6) الملء التلقائي والنشر.
