Универсальный инструмент для автоматической публикации статей в СМИ. Поддерживает автоматизированные процессы публикации для Baijiahao, Sohu и Zhihu. Использует инструмент Playwright MCP для навигации по платформе и входа на страницы публикации, а также поддерживает переключение учетных записей с помощью управления файлами cookie StorageState. Используйте этот навык, когда пользователям нужно публиковать статьи на платформах для самостоятельного использования, открывать страницы публикации платформы или переключать учетные записи. Рабочий процесс: 1) Проверьте и откройте указанную платформу. 2) Проверьте статус входа. 3) Автоматически закрывайте всплывающие окна с рекламой. 4) Перейдите на страницу публикации статьи. 5) Соберите информацию о публикации. 6) Автоматически заполните и опубликуйте.
