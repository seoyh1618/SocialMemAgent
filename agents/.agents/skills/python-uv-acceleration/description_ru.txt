По умолчанию используйте uv вместо pip для управления зависимостями и создания виртуальной среды в проектах Python, что позволяет ускорить установку в 10–100 раз. Применяйте это при работе с проектами Python, создании виртуальных сред, установке зависимостей или когда пользователи упоминают pip/venv/virtualenv.
