Détectez et installez automatiquement les compétences de recherche ToolUniverse manquantes en vérifiant les répertoires de compétences clients courants et en clonant depuis GitHub en cas d'absence. À utiliser lorsque les compétences spécialisées de ToolUniverse ne sont pas installées, lors de la configuration d'un nouveau projet ou lorsque la compétence de routeur ToolUniverse doit amorcer ses sous-compétences avant le routage.
