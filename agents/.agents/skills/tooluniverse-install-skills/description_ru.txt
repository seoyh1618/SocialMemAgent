Обнаруживайте и автоматически установите недостающие исследовательские навыки ToolUniverse, проверяя общие каталоги навыков клиентов и клонируя их из GitHub, если они отсутствуют. Используйте, когда специальные навыки ToolUniverse не установлены, при настройке нового проекта или когда навыку маршрутизатораtooluniverse необходимо загрузить свои поднавыки перед маршрутизацией.
