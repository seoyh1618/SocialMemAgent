Rileva e installa automaticamente le competenze di ricerca ToolUniverse mancanti controllando le directory delle competenze client comuni e clonandole da GitHub se assenti. Da utilizzare quando le competenze specializzate di ToolUniverse non sono installate, quando si imposta un nuovo progetto o quando la competenza del router di ToolUniverse deve eseguire il bootstrap delle proprie competenze secondarie prima dell'instradamento.
