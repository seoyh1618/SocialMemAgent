اكتشف مهارات البحث المفقودة في ToolUniverse وقم بتثبيتها تلقائيًا عن طريق التحقق من أدلة مهارات العميل الشائعة والاستنساخ من GitHub في حالة غيابها. يُستخدم عندما لا يتم تثبيت المهارات المتخصصة لـ ToolUniverse، أو عند إعداد مشروع جديد، أو عندما تحتاج مهارة جهاز التوجيه الخاص بـ Tooluniverse إلى تمهيد مهاراتها الفرعية قبل التوجيه.
