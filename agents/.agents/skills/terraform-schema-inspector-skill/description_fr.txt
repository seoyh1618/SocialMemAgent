Identifiez la prise en charge du fournisseur Terraform pour les ressources, les sources de données, les actions, les ressources de liste, les ressources éphémères et les fonctions. À utiliser pour vérifier les capacités du fournisseur, demander « quelles ressources le fournisseur X prend-il en charge », « le fournisseur Y a-t-il des actions » ou interroger des fonctionnalités spécifiques du fournisseur.
