استخدم @aptx/api-plugin-auth لتنفيذ البرامج الوسيطة ووحدات التحكم لمصادقة الرمز المميز. يدعم الحقن التلقائي لرأس التفويض، وتحديث الرمز المميز (التعامل مع 401 النشط/السلبي)، ومنع التحديث المكرر، وعمليات الاسترجاعات الفاشلة. المشغل: عندما يطلب المستخدم ميزات المصادقة (تكوين البرامج الوسيطة، أو التعامل مع تحديث 401، أو إدارة الرموز المميزة) أو يتضمن الكود createAuthMiddleware/createAuthController.
