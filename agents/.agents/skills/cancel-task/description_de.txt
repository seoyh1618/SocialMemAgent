Brechen Sie eine Aufgabe auf OpenAnt ab und fordern Sie hinterlegte Gelder zurück. Nur der Ersteller der Aufgabe kann abbrechen. Verwenden Sie diese Option, wenn der Benutzer eine Aufgabe abbrechen, sie entfernen, das Kopfgeld zurückziehen, die Annahme von Anträgen beenden, eine von ihm erstellte Aufgabe beenden oder seine hinterlegten Token wiederherstellen möchte. Diese Fähigkeit wird auch benötigt, wenn der Benutzer Dinge sagt wie „Ich habe es mir anders überlegt“, „diese Aufgabe schließen“, „das Kopfgeld abheben“ oder „das macht mir nichts aus“. Stellen Sie sicher, dass Sie diese Fähigkeit immer dann verwenden, wenn der Benutzer eine von ihm erstellte Aufgabe rückgängig machen oder abbrechen möchte, auch wenn er nicht ausdrücklich „Abbrechen“ sagt.
