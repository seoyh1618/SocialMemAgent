Напишите сценарии тестирования BDD в Gherkin для какой-либо функции. ТРИГГЕР, когда: пользователь просит написать, создать или добавить спецификации, сценарии или функции Gherkin. Примеры: «написать спецификации для...», «создать файл функций», «добавить сценарии BDD для пользовательского экспорта». НЕ ЗАПУСКАЙТЕ, когда: пользователь хочет запустить тесты, написать модульные тесты или реализовать функцию.
