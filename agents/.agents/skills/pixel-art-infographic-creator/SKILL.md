## Quality Checklist

- [ ] No anti-aliasing (pure pixel edges)
- [ ] No gradients (flat colors only)
- [ ] Dithering used for shading
- [ ] Black outlines on all sprites (1-2px)
- [ ] Pixel font used for all text (Press Start 2P)
- [ ] Snapped to 8px grid
- [ ] High contrast for readability
