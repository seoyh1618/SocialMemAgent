Registra y gestiona validadores de formato personalizado en z-schema. Úselo cuando el usuario necesite agregar validación de formato personalizado, crear validadores de formato sincronizados o asincrónicos, registrar formatos globalmente o por instancia, validar correos electrónicos, fechas o números de teléfono o reglas comerciales personalizadas con formato, configurar formatAssertions para un comportamiento consciente del vocabulario, usar la opción customFormats, enumerar formatos registrados, manejar tiempos de espera de formato asincrónico o comprender cómo la validación de formato difiere entre borradores de esquema JSON.
