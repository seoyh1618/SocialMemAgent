z-schema에 사용자 정의 형식 유효성 검사기를 등록하고 관리합니다. 사용자가 사용자 정의 형식 유효성 검사를 추가하고, 동기화 또는 비동기 형식 유효성 검사기를 생성하고, 전역적으로 또는 인스턴스별로 형식을 등록하고, 이메일이나 날짜, 전화번호 또는 형식을 사용하여 사용자 정의 비즈니스 규칙의 유효성을 검사하고, 어휘 인식 동작을 위한 formatAssertions를 구성하고, customFormats 옵션을 사용하고, 등록된 형식을 나열하고, 비동기 형식 시간 초과를 처리하거나, JSON 스키마 초안에서 형식 유효성 검사가 어떻게 다른지 이해해야 할 때 사용합니다.
