يسجل ويدير أدوات التحقق من التنسيق المخصص في مخطط z. يُستخدم عندما يحتاج المستخدم إلى إضافة التحقق من صحة التنسيق المخصص، أو إنشاء أدوات التحقق من صحة التنسيق أو المزامنة، أو تسجيل التنسيقات عالميًا أو لكل مثيل، أو التحقق من صحة رسائل البريد الإلكتروني أو التواريخ أو أرقام الهاتف أو قواعد العمل المخصصة باستخدام التنسيق، أو تكوين تأكيدات التنسيق للسلوك المدرك للمفردات، أو استخدام خيار التنسيقات المخصصة، أو إدراج التنسيقات المسجلة، أو التعامل مع مهلات التنسيق غير المتزامن، أو فهم كيفية اختلاف التحقق من صحة التنسيق عبر مسودات JSON Schema.
