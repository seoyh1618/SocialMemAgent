Recherchez GitHub et installez et configurez automatiquement les outils serveur MCP (Model Context Protocol) dans le fichier de configuration Claude. Cette compétence est déclenchée lorsque l'utilisateur doit installer un outil MCP. Workflow : rechercher des projets MCP sur GitHub → extraire la configuration npx → ajouter à ~/.claude.json → gérer les clés API le cas échéant.
