Найдите GitHub и автоматически установите и настройте серверные инструменты MCP (Model Context Protocol) в файл конфигурации Claude. Этот навык срабатывает, когда пользователю необходимо установить инструмент MCP. Рабочий процесс: поиск проектов MCP на GitHub → извлечение конфигурации npx → добавление в ~/.claude.json → обработка ключей API, если таковые имеются.
