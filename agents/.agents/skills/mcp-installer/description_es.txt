Busque en GitHub e instale y configure automáticamente las herramientas del servidor MCP (Protocolo de contexto modelo) en el archivo de configuración de Claude. Esta habilidad se activa cuando el usuario necesita instalar una herramienta MCP. Flujo de trabajo: busque proyectos MCP en GitHub → extraiga la configuración de npx → agregue a ~/.claude.json → maneje las claves API, si las hay.
