Durchsuchen Sie GitHub und installieren und konfigurieren Sie MCP-Servertools (Model Context Protocol) automatisch in der Claude-Konfigurationsdatei. Dieser Skill wird ausgelöst, wenn der Benutzer ein MCP-Tool installieren muss. Arbeitsablauf: Auf GitHub nach MCP-Projekten suchen → Npx-Konfiguration extrahieren → zu ~/.claude.json hinzufügen → ggf. API-Schlüssel verarbeiten.
