Ejecute un estricto bucle reflejo de múltiples iteraciones con fases (PLANIFICAR, PRODUCIR||PREPARAR, EVALUAR, CRÍTICA, REFINAR) para mejorar un artefacto hasta que pasen los controles de calidad o se alcancen los límites de iteración. Úselo cuando el usuario solicite refinamiento iterativo, generación controlada por calidad o bucles "generar -> criticar -> refinar".
