Führen Sie eine strikte Multi-Iterations-Reflexschleife mit Phasen (PLANEN, PRODUZIEREN || VORBEREITEN, BEWERTEN, KRITIKEN, VERFEINERN) aus, um ein Artefakt zu verbessern, bis die Qualitätstore erfüllt sind oder die Iterationsgrenzen erreicht sind. Verwenden Sie diese Option, wenn der Benutzer eine iterative Verfeinerung, eine qualitätsgesteuerte Generierung oder „Generieren -> Kritik -> Verfeinern“-Schleifen anfordert.
