Utilice XcodeGen CLI para crear, actualizar, inspeccionar y solucionar problemas de proyectos Xcode a partir de especificaciones YAML o JSON. Active esta habilidad cuando los usuarios soliciten ejecutar `xcodegen generate`, `xcodegen dump` o `xcodegen cache`; crear o arreglar `project.yml`; dividir especificaciones con `incluir`; o para diagnosticar fallos de análisis, validación y generación relacionados con rutas.
