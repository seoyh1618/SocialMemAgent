Utilisez XcodeGen CLI pour créer, mettre à jour, inspecter et dépanner des projets Xcode à partir de spécifications YAML ou JSON. Déclenchez cette compétence lorsque les utilisateurs demandent d'exécuter « xcodegen generate », « xcodegen dump » ou « xcodegen cache » ; pour créer ou corriger `project.yml` ; pour diviser les spécifications avec « include » ; ou pour diagnostiquer les échecs d'analyse, de validation et de génération liés au chemin.
