استخدم XcodeGen CLI لإنشاء مشاريع Xcode وتحديثها وفحصها واستكشاف أخطائها وإصلاحها من مواصفات YAML أو JSON. قم بتشغيل هذه المهارة عندما يطلب المستخدمون تشغيل "إنشاء xcodegen" أو "تفريغ xcodegen" أو "ذاكرة التخزين المؤقت xcodegen"؛ لتأليف أو إصلاح `project.yml`؛ لتقسيم المواصفات باستخدام `include`؛ أو لتشخيص فشل التحليل والتحقق من الصحة والإنشاء المرتبط بالمسار.
