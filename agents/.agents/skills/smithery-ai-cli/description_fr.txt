Recherchez, connectez-vous et utilisez les outils et compétences MCP via la CLI Smithery. À utiliser lorsque l'utilisateur recherche de nouveaux outils ou compétences, souhaite découvrir des intégrations, se connecter à un MCP, installer une compétence ou souhaite interagir avec un service externe (e-mail, Slack, Discord, GitHub, Jira, Notion, bases de données, API cloud, surveillance, etc.).
