Encuentre, conecte y utilice herramientas y habilidades de MCP a través de Smithery CLI. Úselo cuando el usuario busca nuevas herramientas o habilidades, quiere descubrir integraciones, conectarse a un MCP, instalar una habilidad o quiere interactuar con un servicio externo (correo electrónico, Slack, Discord, GitHub, Jira, Notion, bases de datos, API en la nube, monitoreo, etc.).
