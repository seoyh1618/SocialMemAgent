Créez des écrans d'interface utilisateur iOS dans Swift (UIKit) à partir de spécifications de conception, de maquettes, de captures d'écran ou de descriptions. Utilisez cette compétence chaque fois que l'utilisateur souhaite créer, implémenter ou recréer une interface iOS, un contrôleur de vue, une vue personnalisée, une vue table/collection, une fenêtre contextuelle, une alerte ou tout écran basé sur UIKit. Utilisez toujours cette compétence lorsque l'utilisateur télécharge une capture d'écran de l'interface utilisateur et demande de l'implémenter dans Swift, ou mentionne SnapKit, SwiftEntryKit, Kingfisher, SDWebImage, SwiftyJSON ou toute tâche de mise en page iOS. Couvre les pleins écrans, les composants individuels, les flux de navigation et les présentations modales.
