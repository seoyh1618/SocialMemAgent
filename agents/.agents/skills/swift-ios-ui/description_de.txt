Erstellen Sie iOS-UI-Bildschirme in Swift (UIKit) aus Designspezifikationen, Modellen, Screenshots oder Beschreibungen. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer eine iOS-Schnittstelle, einen Ansichtscontroller, eine benutzerdefinierte Ansicht, eine Tabellen-/Sammlungsansicht, ein Popup, eine Warnung oder einen beliebigen UIKit-basierten Bildschirm erstellen, implementieren oder neu erstellen möchte. Verwenden Sie diese Fähigkeit immer, wenn der Benutzer einen UI-Screenshot hochlädt und um die Implementierung in Swift bittet oder SnapKit, SwiftEntryKit, Kingfisher, SDWebImage, SwiftyJSON oder eine andere iOS-Layoutaufgabe erwähnt. Deckt Vollbildschirme, einzelne Komponenten, Navigationsflüsse und modale Präsentationen ab.
