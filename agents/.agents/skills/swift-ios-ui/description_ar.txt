أنشئ شاشات واجهة مستخدم iOS في Swift (UIKit) من مواصفات التصميم أو النماذج بالحجم الطبيعي أو لقطات الشاشة أو الأوصاف. استخدم هذه المهارة عندما يريد المستخدم إنشاء أو تنفيذ أو إعادة إنشاء أي واجهة iOS أو عرض وحدة التحكم أو العرض المخصص أو عرض الجدول/المجموعة أو النافذة المنبثقة أو التنبيه أو أي شاشة تعتمد على UIKit. استخدم هذه المهارة دائمًا عندما يقوم المستخدم بتحميل لقطة شاشة لواجهة المستخدم ويطلب تنفيذها في Swift، أو يذكر SnapKit أو SwiftEntryKit أو Kingfisher أو SDWebImage أو SwiftyJSON أو أي مهمة تخطيط iOS. يغطي الشاشات الكاملة والمكونات الفردية وتدفقات التنقل والعروض التقديمية المشروطة.
