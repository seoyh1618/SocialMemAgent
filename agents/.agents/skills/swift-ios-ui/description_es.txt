Cree pantallas de interfaz de usuario de iOS en Swift (UIKit) a partir de especificaciones de diseño, maquetas, capturas de pantalla o descripciones. Utilice esta habilidad siempre que el usuario desee crear, implementar o recrear cualquier interfaz de iOS, controlador de vista, vista personalizada, vista de tabla/colección, ventana emergente, alerta o cualquier pantalla basada en UIKit. Utilice siempre esta habilidad cuando el usuario cargue una captura de pantalla de la interfaz de usuario y solicite implementarla en Swift, o mencione SnapKit, SwiftEntryKit, Kingfisher, SDWebImage, SwiftyJSON o cualquier tarea de diseño de iOS. Cubre pantallas completas, componentes individuales, flujos de navegación y presentaciones modales.
