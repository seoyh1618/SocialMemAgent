デザインスペック、モックアップ、スクリーンショット、または説明から Swift (UIKit) で iOS UI 画面を構築します。ユーザーが iOS インターフェイス、ビュー コントローラー、カスタム ビュー、テーブル/コレクション ビュー、ポップアップ、アラート、または UIKit ベースの画面を作成、実装、または再作成したい場合は常に、このスキルを使用します。ユーザーが UI スクリーンショットをアップロードして Swift での実装を要求する場合、または SnapKit、SwiftEntryKit、Kingfisher、SDWebImage、SwiftyJSON、または iOS レイアウト タスクに言及する場合は、常にこのスキルを使用してください。全画面、個々のコンポーネント、ナビゲーション フロー、モーダル プレゼンテーションをカバーします。
