Агент TuriX Computer Use Agent для автоматизации рабочего стола macOS. Используйте, когда вам нужно выполнить задачи визуального пользовательского интерфейса, для которых отсутствует доступ к CLI или API, например открытие приложений, нажатие кнопок, навигация по графическому интерфейсу или многоэтапные визуальные рабочие процессы.
