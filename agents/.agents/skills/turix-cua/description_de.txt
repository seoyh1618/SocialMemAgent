TuriX Computer Use Agent für die Desktop-Automatisierung von macOS. Verwenden Sie diese Option, wenn Sie visuelle UI-Aufgaben ausführen müssen, für die kein CLI- oder API-Zugriff erforderlich ist, z. B. das Öffnen von Apps, das Klicken auf Schaltflächen, das Navigieren in GUIs oder mehrstufige visuelle Workflows.
