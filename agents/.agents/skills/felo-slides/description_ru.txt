Создавайте PPT/слайды с помощью API задач Felo PPT в Claude Code. Используйте, когда пользователи просят создать/создать/генерировать/экспортировать презентации или наборы слайдов или когда используются явные команды, такие как /felo-slides. Обрабатывает проверку ключей API, создание задач, опрос и окончательный вывод ppt_url.
