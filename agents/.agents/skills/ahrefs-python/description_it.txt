Gestisce l'utilizzo dell'API Ahrefs in Python utilizzando la libreria `ahrefs-python`. Da utilizzare quando si lavora con attività relative alla SEO/marketing o con dati quali backlink, parole chiave, valutazioni di domini, traffico organico, controlli del sito, monitoraggio del posizionamento e monitoraggio del marchio. Copre l'utilizzo di `ahrefs-python` inclusi AhrefsClient / AsyncAhrefsClient, modelli di richiesta/risposta digitati, gestione degli errori e tutte le sezioni API.
