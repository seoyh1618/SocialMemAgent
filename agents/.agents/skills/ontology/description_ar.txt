رسم بياني معرفي مكتوب لذاكرة الوكيل المنظمة والمهارات القابلة للتركيب. يُستخدم عند إنشاء/الاستعلام عن الكيانات (شخص، مشروع، مهمة، حدث، مستند)، أو ربط الكائنات ذات الصلة، أو فرض القيود، أو التخطيط لإجراءات متعددة الخطوات كتحويلات الرسم البياني، أو عندما تحتاج المهارات إلى مشاركة الحالة. قم بتشغيل "تذكر"، أو "ماذا أعرف عنه"، أو "ربط X بـ Y"، أو "إظهار التبعيات"، أو الكيان CRUD، أو الوصول إلى البيانات عبر المهارات.
