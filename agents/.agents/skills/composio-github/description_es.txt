GitHub a través de la API de Composio. Úselo cuando:
(1) Leer el contenido del repositorio, README o descargar repositorios
(2) Crear, actualizar o enumerar problemas
(3) Crear, enumerar o fusionar solicitudes de extracción
(4) Listado de repositorios o sucursales
NO utilice la CLI `gh`: fallará con errores de autenticación. Utilice únicamente la API HTTP de Composio.
