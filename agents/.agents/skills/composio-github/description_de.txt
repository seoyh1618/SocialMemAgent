GitHub über Composio API. Verwenden Sie, wenn:
(1) Repository-Inhalte, README lesen oder Repos herunterladen
(2) Probleme erstellen, aktualisieren oder auflisten
(3) Pull-Requests erstellen, auflisten oder zusammenführen
(4) Auflisten von Repositories oder Branches
Verwenden Sie NICHT die „gh“-CLI – dies schlägt mit Authentifizierungsfehlern fehl. Verwenden Sie nur die Composio-HTTP-API.
