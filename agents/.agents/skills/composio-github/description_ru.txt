GitHub через Composio API. Используйте, когда:
(1) Чтение содержимого репозитория, README или загрузка репозиториев.
(2) Создание, обновление или перечисление проблем
(3) Создание, перечисление или объединение запросов на включение
(4) Листинг репозиториев или филиалов
НЕ ИСПОЛЬЗУЙТЕ CLI `gh` — он завершится ошибкой из-за ошибок аутентификации. Используйте только HTTP API Composio.
