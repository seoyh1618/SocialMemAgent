GitHub عبر Composio API. استخدم عندما:
(1) قراءة محتويات المستودع، أو الملف التمهيدي (README)، أو تنزيل المستودعات
(2) إنشاء المشكلات أو تحديثها أو إدراجها
(3) إنشاء طلبات السحب أو إدراجها أو دمجها
(4) إدراج المستودعات أو الفروع
لا تستخدم `gh` CLI - فسوف يفشل مع وجود أخطاء في المصادقة. استخدم Composio HTTP API فقط.
