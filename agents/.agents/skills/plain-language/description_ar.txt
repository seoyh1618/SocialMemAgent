يراجع نص المشروع (الوثائق، المستندات التمهيدية، النسخة التسويقية، سلاسل واجهة المستخدم، تعليقات التعليمات البرمجية، رسائل الخطأ) للامتثال للغة البسيطة وينتج توصيات تحرير منظمة بناءً على إرشادات اللغة البسيطة للحكومة الفيدرالية الأمريكية (https://digital.gov/guides/plain-language). استخدمه عند مطالبتك بما يلي: مراجعة اللغة، أو التحقق من إمكانية القراءة، أو تدقيق المستندات، أو تحسين وضوح النسخ، أو إجراء فحص لغة بسيطة، أو جعل الملف README أكثر وضوحًا، أو مراجعة جودة الكتابة. تشغيل عبارات مثل "مراجعة بلغة واضحة"، "التحقق من إمكانية القراءة"، "تدقيق النسخة"، "اجعل هذا أكثر وضوحًا".
