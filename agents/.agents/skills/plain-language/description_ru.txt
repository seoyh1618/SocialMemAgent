Проверяет текст проекта (документацию, файлы README, маркетинговые копии, строки пользовательского интерфейса, комментарии к коду, сообщения об ошибках) на предмет соответствия простому языку и выдает структурированные рекомендации по редактированию на основе руководящих указаний федерального правительства США по использованию простого языка (https://digital.gov/guides/plain-language). Используйте, когда вас просят: проверить язык, проверить читабельность, проверить документацию, улучшить ясность текста, проверить простой язык, сделать README более понятным или проверить качество написания. Запускается по таким фразам, как «проверка простым языком», «проверьте читабельность», «проверьте текст», «сделайте это более понятным».
