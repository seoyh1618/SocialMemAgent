Ajoutez des outils locaux personnalisés à ToolUniverse et utilisez-les avec les plus de 1 000 outils intégrés. Utilisez cette compétence lorsqu'un utilisateur souhaite : créer son propre outil pour une API privée ou personnalisée, ajouter un outil local à son espace de travail, intégrer un service interne avec ToolUniverse, ou utiliser un outil personnalisé via le serveur MCP ou l'API Python. Couvre à la fois l'approche de configuration JSON (la plus simple, aucun Python nécessaire) et l'approche de classe Python (contrôle total). Explique également comment vérifier les outils chargés correctement et comment les appeler. Couvre également l'approche du package de plugins pour les ensembles d'outils réutilisables, partageables et installables par pip.
