カスタム ローカル ツールを ToolUniverse に追加し、1,000 以上の組み込みツールと一緒に使用します。このスキルは、ユーザーがプライベート API またはカスタム API 用の独自のツールを作成する場合、ローカル ツールをワークスペースに追加する場合、内部サービスを ToolUniverse と統合する場合、または MCP サーバーまたは Python API 経由でカスタム ツールを使用する場合に使用します。 JSON 構成アプローチ (最も簡単、Python 不要) と Python クラス アプローチ (フル コントロール) の両方をカバーします。ツールが正しくロードされたことを確認する方法とそれらを呼び出す方法についても説明します。また、再利用可能、共有可能、pip インストール可能なツール セットのためのプラグイン パッケージのアプローチについても説明します。
