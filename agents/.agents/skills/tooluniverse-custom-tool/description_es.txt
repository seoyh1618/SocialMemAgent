Agregue herramientas locales personalizadas a ToolUniverse y úselas junto con las más de 1000 herramientas integradas. Utilice esta habilidad cuando un usuario quiera: crear su propia herramienta para una API privada o personalizada, agregar una herramienta local a su espacio de trabajo, integrar un servicio interno con ToolUniverse o usar una herramienta personalizada a través del servidor MCP o la API de Python. Cubre tanto el enfoque de configuración JSON (el más fácil, no se necesita Python) como el enfoque de clases Python (control total). También cubre cómo verificar las herramientas cargadas correctamente y cómo llamarlas. También cubre el enfoque del paquete de complementos para conjuntos de herramientas reutilizables, compartibles e instalables por pip.
