Aggiungi strumenti locali personalizzati a ToolUniverse e usali insieme agli oltre 1000 strumenti integrati. Utilizza questa competenza quando un utente desidera: creare il proprio strumento per un'API privata o personalizzata, aggiungere uno strumento locale al proprio spazio di lavoro, integrare un servizio interno con ToolUniverse o utilizzare uno strumento personalizzato tramite il server MCP o l'API Python. Copre sia l'approccio di configurazione JSON (il più semplice, non è necessario Python) sia l'approccio della classe Python (controllo completo). Spiega inoltre come verificare gli strumenti caricati correttamente e come richiamarli. Copre anche l'approccio del pacchetto plug-in per set di strumenti riutilizzabili, condivisibili e installabili tramite pip.
