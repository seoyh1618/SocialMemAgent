أضف أدوات محلية مخصصة إلى ToolUniverse واستخدمها جنبًا إلى جنب مع أكثر من 1000 أداة مدمجة. استخدم هذه المهارة عندما يريد المستخدم: إنشاء أداة خاصة به لواجهة برمجة تطبيقات خاصة أو مخصصة، أو إضافة أداة محلية إلى مساحة العمل الخاصة به، أو دمج خدمة داخلية مع ToolUniverse، أو استخدام أداة مخصصة عبر خادم MCP أو Python API. يغطي كلا من نهج تكوين JSON (الأسهل، لا حاجة إلى Python) ونهج فئة Python (التحكم الكامل). يغطي أيضًا كيفية التحقق من تحميل الأدوات بشكل صحيح وكيفية الاتصال بها. يغطي أيضًا نهج حزمة البرنامج المساعد لمجموعات الأدوات القابلة لإعادة الاستخدام والقابلة للمشاركة والقابلة للتثبيت.
