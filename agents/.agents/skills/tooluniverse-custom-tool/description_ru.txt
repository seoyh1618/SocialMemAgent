Добавляйте собственные локальные инструменты в ToolUniverse и используйте их вместе с более чем 1000 встроенными инструментами. Используйте этот навык, когда пользователь хочет: создать свой собственный инструмент для частного или пользовательского API, добавить локальный инструмент в свое рабочее пространство, интегрировать внутреннюю службу с ToolUniverse или использовать собственный инструмент через сервер MCP или API Python. Охватывает как подход к настройке JSON (самый простой, Python не требуется), так и подход классов Python (полный контроль). Также рассказывается, как проверить правильность загрузки инструментов и как их вызывать. Также рассматривается подход к пакету плагинов для многоразовых, общих и устанавливаемых с помощью pip наборов инструментов.
