Fügen Sie benutzerdefinierte lokale Tools zu ToolUniverse hinzu und verwenden Sie sie zusammen mit den über 1000 integrierten Tools. Verwenden Sie diese Fähigkeit, wenn ein Benutzer: ein eigenes Tool für eine private oder benutzerdefinierte API erstellen, ein lokales Tool zu seinem Arbeitsbereich hinzufügen, einen internen Dienst mit ToolUniverse integrieren oder ein benutzerdefiniertes Tool über den MCP-Server oder die Python-API verwenden möchte. Deckt sowohl den JSON-Konfigurationsansatz (am einfachsten, kein Python erforderlich) als auch den Python-Klassenansatz (vollständige Kontrolle) ab. Behandelt außerdem, wie man die korrekt geladenen Tools überprüft und wie man sie aufruft. Behandelt auch den Plugin-Paket-Ansatz für wiederverwendbare, gemeinsam nutzbare und per PIP installierbare Tool-Sets.
