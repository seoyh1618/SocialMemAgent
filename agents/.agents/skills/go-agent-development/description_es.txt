Guía de desarrollo de agentes de Go, que cubre el diseño de la arquitectura del agente, el mecanismo de latido, la ejecución de tareas, los informes de registros, el proceso de actualización y la interacción con el módulo de envío. Se utiliza cuando los usuarios desarrollan agentes de máquina de compilación, implementan lógica de ejecución de tareas, manejan la comunicación del agente o desarrollan el lenguaje Go.
