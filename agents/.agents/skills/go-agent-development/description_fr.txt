Go Agent Development Guide, couvrant la conception de l’architecture de l’agent, le mécanisme de pulsation, l’exécution des tâches, les rapports de journaux, le processus de mise à niveau et l’interaction avec le module Dispatch. Utilisé lorsque les utilisateurs développent des agents de machine de construction, implémentent une logique d'exécution de tâches, gèrent la communication des agents ou effectuent du développement en langage Go.
