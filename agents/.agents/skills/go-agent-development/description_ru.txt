Руководство по разработке агента Go, в котором описывается проектирование архитектуры агента, механизм контрольного сигнала, выполнение задач, отчеты журналов, процесс обновления и взаимодействие с модулем Dispatch. Используется, когда пользователи разрабатывают агенты для сборки машины, реализуют логику выполнения задач, обрабатывают связь агента или разрабатывают язык Go.
