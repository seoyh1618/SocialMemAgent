Génère des illustrations académiques prêtes à être publiées à l'aide du pipeline à cinq agents PaperBanana (arXiv : 2601.23265). Le mode Diagramme exécute 5 agents API Gemini avec un apprentissage multimodal en contexte à partir d'images de référence organisées. Le mode Plot génère du code Python exécutable matplotlib/seaborn. À utiliser lorsque l'utilisateur demande : des figures de documents de recherche, des diagrammes académiques, des illustrations de méthodologie, des diagrammes d'architecture, des graphiques statistiques, des visualisations de qualité conférence, des organigrammes pour les articles, des figures NeurIPS/ICML/CVPR ou l'amélioration de figures papier existantes.
