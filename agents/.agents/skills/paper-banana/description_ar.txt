يُنشئ رسومًا توضيحية أكاديمية جاهزة للنشر باستخدام خط أنابيب PaperBanana المكون من خمسة وكلاء (arXiv:2601.23265). يقوم وضع الرسم التخطيطي بتشغيل 5 وكلاء Gemini API مع التعلم متعدد الوسائط في السياق من الصور المرجعية المنسقة. يقوم وضع الرسم بإنشاء رمز Python matplotlib/seaborn القابل للتنفيذ. يُستخدم عندما يطلب المستخدم: أشكال أوراق بحثية، أو مخططات أكاديمية، أو رسوم توضيحية للمنهجية، أو مخططات معمارية، أو مخططات إحصائية، أو تصورات بجودة المؤتمرات، أو مخططات انسيابية للأوراق، أو أشكال NeurIPS/ICML/CVPR، أو تحسين الأشكال الورقية الحالية.
