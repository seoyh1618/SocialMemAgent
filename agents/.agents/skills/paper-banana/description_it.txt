Genera illustrazioni accademiche pronte per la pubblicazione utilizzando la pipeline a cinque agenti PaperBanana (arXiv:2601.23265). La modalità diagramma esegue 5 agenti API Gemini con apprendimento contestuale multimodale da immagini di riferimento selezionate. La modalità Plot genera codice eseguibile Python matplotlib/seaborn. Da utilizzare quando l'utente richiede: figure di documenti di ricerca, diagrammi accademici, illustrazioni di metodologia, diagrammi di architettura, grafici statistici, visualizzazioni di qualità conferenza, diagrammi di flusso per documenti, figure NeurIPS/ICML/CVPR o miglioramento di figure di carta esistenti.
