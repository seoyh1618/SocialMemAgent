PaperBanana 5개 에이전트 파이프라인(arXiv:2601.23265)을 사용하여 출판 가능한 학술 일러스트레이션을 생성합니다. 다이어그램 모드는 선별된 참조 이미지에서 다중 모드 상황 내 학습을 통해 5개의 Gemini API 에이전트를 실행합니다. 플롯 모드는 실행 가능한 Python matplotlib/seaborn 코드를 생성합니다. 사용자가 연구 논문 수치, 학술 다이어그램, 방법론 일러스트레이션, 아키텍처 다이어그램, 통계 플롯, 컨퍼런스 수준의 시각화, 논문 흐름도, NeurIPS/ICML/CVPR 수치 또는 기존 논문 수치 개선을 요청할 때 사용합니다.
