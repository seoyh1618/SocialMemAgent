PaperBanana の 5 つのエージェント パイプライン (arXiv:2601.23265) を使用して、出版用の学術イラストを生成します。ダイアグラム モードでは、厳選された参照画像からのマルチモーダル インコンテキスト学習を使用して 5 つの Gemini API エージェントを実行します。プロット モードでは、実行可能な Python matplotlib/seaborn コードが生成されます。研究論文の図、学術図、方法論の図、アーキテクチャ図、統計プロット、会議品質の視覚化、論文のフローチャート、NeurIPS/ICML/CVPR 図、または既存の論文図の改善をユーザーが要求した場合に使用します。
