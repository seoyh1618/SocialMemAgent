Создает готовые к публикации академические иллюстрации с помощью пятиагентного конвейера PaperBanana (arXiv:2601.23265). В режиме диаграммы запускаются 5 агентов Gemini API с мультимодальным контекстным обучением на основе тщательно подобранных эталонных изображений. В режиме графика генерируется исполняемый код Python matplotlib/seaborn. Используйте, когда пользователь запрашивает: рисунки для исследовательских работ, академические диаграммы, методологические иллюстрации, архитектурные диаграммы, статистические графики, визуализации конференц-класса, блок-схемы для статей, цифры NeurIPS/ICML/CVPR или улучшение существующих бумажных цифр.
