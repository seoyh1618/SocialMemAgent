Genera ilustraciones académicas listas para publicación utilizando el proceso de cinco agentes PaperBanana (arXiv:2601.23265). El modo de diagrama ejecuta 5 agentes API de Gemini con aprendizaje multimodal en contexto a partir de imágenes de referencia seleccionadas. El modo de trazado genera código ejecutable Python matplotlib/seaborn. Úselo cuando el usuario solicite: figuras de artículos de investigación, diagramas académicos, ilustraciones de metodología, diagramas de arquitectura, diagramas estadísticos, visualizaciones con calidad de conferencia, diagramas de flujo para artículos, figuras NeurIPS/ICML/CVPR o mejoras de figuras en papel existentes.
