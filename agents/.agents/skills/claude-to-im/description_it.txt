Questa abilità collega Claude Code alle piattaforme di messaggistica istantanea (Telegram, Discord, Feishu/Lark).
Dovrebbe essere utilizzato quando l'utente desidera avviare un demone in background che inoltri
Messaggi IM alle sessioni di Claude Code o gestisci il ciclo di vita di quel demone.
Trigger su: "claude-to-im", "start bridge", "stop bridge", "bridge status",
"查看日志", "启动桥接", "停止桥接" o qualsiasi menzione della gestione del bridge IM.
Sottocomandi: setup, start, stop, status, logs, reconfigure, doctor.
