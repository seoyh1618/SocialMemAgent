Diese Fähigkeit verbindet Claude Code mit IM-Plattformen (Telegram, Discord, Feishu/Lark).
Es sollte verwendet werden, wenn der Benutzer einen Hintergrunddaemon starten möchte, der weiterleitet
Senden Sie IM-Nachrichten an Claude Code-Sitzungen oder verwalten Sie den Lebenszyklus dieses Daemons.
Auslöser bei: „claude-to-im“, „Brücke starten“, „Brücke stoppen“, „Brückenstatus“,
„查看日志“, „启动桥接“, „停止桥接“ oder jede Erwähnung der IM-Bridge-Verwaltung.
Unterbefehle: Setup, Start, Stopp, Status, Protokolle, Neukonfiguration, Arzt.
