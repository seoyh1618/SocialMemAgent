このスキルは、Claude Code を IM プラットフォーム (Telegram、Discord、Feishu/Lark) に橋渡しします。
ユーザーが転送するバックグラウンド デーモンを起動したい場合に使用する必要があります。
クロード コード セッションへの IM メッセージ、またはそのデーモンのライフサイクルを管理します。
トリガー: 「claude-to-im」、「ブリッジ開始」、「ブリッジ停止」、「ブリッジステータス」、
「查看日志」、「启动桥接」、「停止桥接」、または IM ブリッジ管理に関する言及。
サブコマンド: セットアップ、開始、停止、ステータス、ログ、再構成、ドクター。
