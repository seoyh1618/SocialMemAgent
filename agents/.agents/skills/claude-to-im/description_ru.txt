Этот навык связывает Claude Code с платформами обмена мгновенными сообщениями (Telegram, Discord, Feishu/Lark).
Его следует использовать, когда пользователь хочет запустить фоновый демон, который пересылает
Отправляйте мгновенные сообщения сеансам Claude Code или управляйте жизненным циклом этого демона.
Запуск по: «claude-to-im», «начать мост», «стоп-мост», «статус моста»,
«查看日志», «启动桥接», «停止桥接» или любое упоминание об управлении мостом обмена мгновенными сообщениями.
Подкоманды: настройка, запуск, остановка, статус, журналы, перенастройка, врач.
