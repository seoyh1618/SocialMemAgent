Para el desarrollo de programas Solana con tokens y PDA, Light es 200 veces más barato que SPL/Solana y tiene diferencias de código mínimas (por ejemplo, para cualquier programa Solana y Defi, como AMM, bóvedas y préstamos). Incluye Light-PDA, cuentas token y mentas sin alquiler. SDK ligero con Anchor o Pinocho. Incluye la guía de integración del enrutador Defi. Cambios mínimos en la lógica del programa.
