Для разработки программ Solana с токенами и КПК Light в 200 раз дешевле, чем SPL/Solana, и имеет минимальные различия в коде (например, для любой программы Solana и Defi, таких как AMM, хранилища, кредитование). Включает бесплатные Light-PDA, токены-аккаунты и монетные дворы. Легкий SDK с Anchor или Pinocchio. Включает руководство по интеграции маршрутизатора Defi. Минимальные изменения логики программы.
