Gestion des bots, webhooks entrants/sortants, déploiement des liens. À utiliser lorsque vous avez besoin de : configurer un bot, traiter un webhook, gérer les clics sur les boutons, les résumés périodiques, les alertes, les sondages d'événements, développer un lien. NE PAS utiliser pour : l'envoi de messages depuis le bot (→ pachca-messages), les formulaires interactifs (→ pachca-forms).
