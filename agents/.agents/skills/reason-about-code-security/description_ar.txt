تطوير منطق التهديد المنهجي والتفكير العدائي حول أمان التعليمات البرمجية. يُستخدم عندما يريد المتعلم تحليل التعليمات البرمجية بحثًا عن آثار أمنية، أو فهم أنماط الثغرات الأمنية، أو تطوير تفكير ذي عقلية أمنية. تُعلِّم هذه المهارة نمذجة التهديدات المنهجية، وإظهار الافتراضات، والتفكير الدفاعي - وليس فهرسة الثغرات الأمنية. يطلق عبارات مثل "هل هذا آمن"، أو "الآثار الأمنية"، أو "هل يمكن استغلال ذلك"، أو "تحليل التهديدات"، أو عندما يرغب المتعلم في تطوير مهارات التفكير الأمني.
