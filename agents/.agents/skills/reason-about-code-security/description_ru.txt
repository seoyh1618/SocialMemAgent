Развивайте систематическое обоснование угроз и состязательное мышление в отношении безопасности кода. Используйте, когда учащийся хочет проанализировать код на предмет последствий для безопасности, понять закономерности уязвимостей или развить мышление, ориентированное на безопасность. Этот навык учит систематическому моделированию угроз, выявлению предположений и защитным рассуждениям, а не каталогизации уязвимостей. Запускается по таким фразам, как «это безопасно», «последствия для безопасности», «можно ли это использовать», «анализ угроз» или когда учащийся хочет развить навыки рассуждения о безопасности.
