Руководство по разработке шаблона WinJS HuiPro (корпоративный Интернет). Включает собственный маршрутизатор, Vuex, TabsManagement, контроль разрешений и архитектуру основной подсистемы. Применимо для: (1) Проекты, созданные с помощью шаблона create-win huipro (2) Изменение конфигурации/настроек (3) Поддержка пользовательской маршрутизации src/router (4) Использование TabsManagement для управления вкладками (5) Настройка разрешений/меню (6) Разработка и сборка основной подсистемы (7) Проекты администрирования предприятия Vue 2 + HUI.
