Senden Sie abgeschlossene Arbeiten für eine Aufgabe an OpenAnt. Einreichung = Textbeschreibung + Dateien. WICHTIG: Überprüfen Sie vor dem Einreichen immer, ob für Ihre Arbeit Dateien entstanden sind, und laden Sie diese zuerst hoch. Verwenden Sie diese Option, wenn der Agent seine Arbeit abgeschlossen hat und Ergebnisse liefern, eine Lösung einreichen, Arbeitsergebnisse abgeben, Dateien hochladen oder einen Abschlussnachweis senden möchte. Behandelt „Arbeit/Aufgabe einreichen“, „Ergebnisse liefern“, „Ich bin fertig“, „Hier ist meine Arbeit“, „Lösung einreichen“, „Hochladen und einreichen“, „Beweis anhängen“, „Datei liefern“, „Liefergegenstand senden“.
