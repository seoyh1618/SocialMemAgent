مجموعة إدارة جلسة Claude Code: البحث عن الجلسات حسب الموضوع/المعرف، واستئناف الجلسات المتعطلة، وإنشاء جلسات جديدة (تفاعلية أو مقطوعة الرأس)، ومراقبة الجلسات المباشرة، واكتشاف المشاريع، وتلخيص الجلسات الجديدة تلقائيًا، وتشغيل الإعدادات الجديدة. يجب استخدام هذه المهارة عند البحث عن الجلسات السابقة، أو التحقق من تشغيل الجلسات، أو استئناف العمل، أو إنشاء جلسات جديدة، أو تشغيل جهاز جديد.
