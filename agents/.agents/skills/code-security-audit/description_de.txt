Führen Sie Sicherheitsüberprüfungen für Codeänderungen, Unterschiede oder Verzweigungen durch, um mit hoher Wahrscheinlichkeit ausnutzbare Schwachstellen zu finden. Wird verwendet, wenn Sie aufgefordert werden, „Sicherheit zu prüfen“, „auf Schwachstellen zu überprüfen“, „Sicherheit zu scannen“, „auf Sicherheitsprobleme zu prüfen“, „diese PR zu prüfen“, „diese Änderungen auf Sicherheit zu prüfen“ oder „Schwachstellen in Diff zu finden“. Im Gegensatz zur Sicherheitsüberprüfung (die sichere Codierungsmuster/Checklisten bereitstellt) – diese Fähigkeit prüft aktiv Codeänderungen mithilfe einer strukturierten Methodik mit Falsch-Positiv-Filterung. Enthält Python-Skripte für die GitHub Action CI-Integration und PR-Bewertung.
