コードの変更、差分、またはブランチに対してセキュリティ監査を実行し、悪用可能な信頼性の高い脆弱性を見つけます。 「セキュリティの監査」、「脆弱性の確認」、「セキュリティ スキャン」、「セキュリティの問題の確認」、「この PR の監査」、「セキュリティの変更の確認」、または「diff での脆弱性の検索」を求められた場合に使用します。セキュリティ レビュー (安全なコーディング パターン/チェックリストを提供する) とは異なります。このスキルは、誤検知フィルタリングを備えた構造化された方法論を使用して、コードの変更を積極的に監査します。 GitHub Action CI 統合と PR 評価用の Python スクリプトが含まれています。
