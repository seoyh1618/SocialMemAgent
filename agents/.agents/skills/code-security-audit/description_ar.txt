قم بإجراء عمليات تدقيق أمنية على تغييرات التعليمات البرمجية أو الاختلافات أو الفروع للعثور على ثغرات أمنية عالية الثقة قابلة للاستغلال. استخدم عندما يُطلب منك "تدقيق الأمان"، أو "مراجعة الثغرات الأمنية"، أو "الفحص الأمني"، أو "التحقق من وجود مشكلات أمنية"، أو "تدقيق العلاقات العامة هذه"، أو "مراجعة هذه التغييرات للأمان"، أو "البحث عن الثغرات الأمنية في الاختلافات". تختلف عن المراجعة الأمنية (التي توفر أنماط ترميز/قوائم مرجعية آمنة) — تقوم هذه المهارة بمراجعة تغييرات التعليمات البرمجية بشكل نشط باستخدام منهجية منظمة مع تصفية إيجابية خاطئة. يتضمن نصوص Python لتكامل GitHub Action CI وتقييم العلاقات العامة.
