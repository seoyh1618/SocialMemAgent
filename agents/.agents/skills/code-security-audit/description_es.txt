Realice auditorías de seguridad sobre cambios de código, diferencias o bifurcaciones para encontrar vulnerabilidades explotables de alta confianza. Úselo cuando se le solicite "auditar la seguridad", "revisar las vulnerabilidades", "análisis de seguridad", "verificar problemas de seguridad", "auditar este PR", "revisar estos cambios por seguridad" o "encontrar vulnerabilidades en diferencias". A diferencia de la revisión de seguridad (que proporciona listas de verificación/patrones de codificación seguros), esta habilidad audita activamente los cambios de código utilizando una metodología estructurada con filtrado de falsos positivos. Incluye scripts de Python para la integración de GitHub Action CI y la evaluación de relaciones públicas.
