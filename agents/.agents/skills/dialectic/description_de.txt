Eine Electric Monk-Engine – zwei Subagenten glauben im Namen des Benutzers an vollständig festgelegte Positionen, während der Orchestrator eine Analyse und Synthese struktureller Widersprüche durchführt. Durch die Auslagerung der Glaubensarbeit an Agenten agiert der Benutzer von einer glaubensfreien Position aus, in der er die Struktur des Widerspruchs analysieren kann, anstatt sich auf einer der beiden Seiten zu befinden. Verwenden Sie es, wenn der Benutzer eine Idee einem Stresstest unterziehen, eine echte Spannung lösen, ein tieferes mentales Modell aufbauen oder eine Entscheidung mit hohem Risiko treffen möchte, bei der die Kompromisse unklar sind. Funktioniert in allen Bereichen – technische Architektur, Produktstrategie, Philosophie, persönliche Entscheidungen, Risikoanalyse, Richtlinien, kreative Ausrichtung.
