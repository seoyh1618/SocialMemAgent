Effectuez des révisions de code rigoureuses et contradictoires avec une tolérance zéro pour la médiocrité. À utiliser lorsque les utilisateurs demandent « examiner de manière critique » mon code ou un PR, « critiquer mon code », « trouver des problèmes dans mon code » ou « qu'est-ce qui ne va pas avec ce code ». Identifie les failles de sécurité, les modèles paresseux, les échecs extrêmes et les mauvaises pratiques dans Python, R, JavaScript/TypeScript, SQL et le code frontal. Examine la gestion des erreurs, la sécurité des types, les performances, l’accessibilité et la qualité du code. Fournit des commentaires structurés avec des niveaux de gravité (blocage, obligatoire, suggestions) et des recommandations spécifiques et exploitables.
