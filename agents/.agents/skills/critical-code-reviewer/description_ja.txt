平凡さを一切許さず、厳格で敵対的なコードレビューを実施します。ユーザーが私のコードや PR を「批判的にレビュー」、「私のコードを批判」、「私のコードの問題点を見つけて」、または「このコードのどこが間違っているのか」を求めるときに使用します。 Python、R、JavaScript/TypeScript、SQL、およびフロントエンド コードにわたるセキュリティ ホール、遅延パターン、エッジ ケースの障害、および悪い習慣を特定します。エラー処理、型安全性、パフォーマンス、アクセシビリティ、コード品質を精査します。重大度レベル (ブロック、必須、提案) と具体的で実用的な推奨事項を含む構造化されたフィードバックを提供します。
