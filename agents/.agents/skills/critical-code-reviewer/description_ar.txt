قم بإجراء مراجعات صارمة وتنافسية للقواعد دون التسامح مطلقًا مع المستوى المتوسط. يُستخدم عندما يطلب المستخدمون "مراجعة نقدية" للكود الخاص بي أو العلاقات العامة، أو "نقد الكود الخاص بي"، أو "العثور على مشكلات في الكود الخاص بي"، أو "ما الخطأ في هذا الكود". يحدد الثغرات الأمنية والأنماط البطيئة وحالات فشل الحافة والممارسات السيئة عبر Python وR وJavaScript/TypeScript وSQL والتعليمات البرمجية للواجهة الأمامية. يدقق في معالجة الأخطاء، وسلامة النوع، والأداء، وإمكانية الوصول، وجودة التعليمات البرمجية. يقدم تعليقات منظمة مع مستويات الخطورة (الحظر، والمطلوب، والاقتراحات) وتوصيات محددة وقابلة للتنفيذ.
