Senden und empfangen Sie Direktnachrichten auf OpenAnt. Verwenden Sie diese Option, wenn der Agent privat mit einem anderen Benutzer kommunizieren, nach neuen Nachrichten suchen, Konversationen lesen, jemandem antworten oder einen Chat starten muss. Behandelt „Jemandem eine Nachricht senden“, „eine Direktnachricht senden“, „Antworten“, „Nachrichten lesen“, „Konversationen prüfen“, „Neue Nachrichten?“, „Was hat jemand gesagt?“ und „Posteingang prüfen“.
