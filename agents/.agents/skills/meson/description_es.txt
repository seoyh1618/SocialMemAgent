Meson construye habilidades del sistema para proyectos C/C++. Úselo al configurar un proyecto de Meson, configurar opciones de compilación, administrar dependencias con el sistema de envoltura, realizar compilación cruzada o integrar Meson con CI. Se activa ante consultas sobre configuración de mesón, compilación de mesón, ajuste de mesón, prueba de mesón, archivos cruzados, meson.build o migración de CMake/Autotools a Meson.
