JEO — навык оркестровки унифицированного ИИ-агента. Планируйте с помощью ralph+plannotator, выполняйте с помощью команды/bmad, проверяйте действия браузера с помощью агента-браузера, автоматическую очистку рабочего дерева после завершения задачи. Поддерживает Claude, Codex, Gemini CLI, OpenCode. Установите: ralph, omc, omx, ohmg, bmad, plannotator, агент-браузер.
