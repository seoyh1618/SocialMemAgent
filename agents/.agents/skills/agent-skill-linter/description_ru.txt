Навыки агента Lint для обеспечения соответствия спецификациям и готовности к публикации. Проверяет внешний вид SKILL.md, ЛИЦЕНЗИЮ, значки README, рабочий процесс CI, разделы установки/использования, дедупликацию контента и длину тела. Автоматически исправляет распространенные проблемы с помощью --fix.
