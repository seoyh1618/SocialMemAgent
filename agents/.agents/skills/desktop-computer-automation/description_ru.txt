Автоматизация рабочего стола на основе Vision с помощью Midscene. Управляйте своим рабочим столом (macOS, Windows, Linux) с помощью команд на естественном языке.
Работает полностью на основе снимков экрана — не требуется DOM или метки доступности. Может взаимодействовать со всеми видимыми элементами на экране независимо от технологического стека.

Триггеры: открыть приложение, нажать клавишу, рабочий стол, компьютер, щелчок по экрану, ввести текст, снимок экрана рабочего стола,
запуск приложения, переключение окна, автоматизация рабочего стола, управляющий компьютер, щелчок мышью, сочетание клавиш,
снимок экрана, найти на экране, прочитать экран, проверить окно, закрыть приложение, свернуть окно, развернуть окно

При поддержке Midscene.js (https://midscenejs.com)
