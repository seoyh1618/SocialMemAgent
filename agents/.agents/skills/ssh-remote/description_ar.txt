أنماط الوصول عن بعد عبر SSH وأدوات المساعدة الآمنة. يغطي الاتصالات، وإدارة التكوين، وإنشاء المفاتيح (Ed25519، FIDO2)، والنفق، وإعادة توجيه المنافذ، ونقل الملفات، وتعدد الإرسال.

يُستخدم عند الاتصال بالخوادم، أو إدارة مفاتيح SSH، أو إعداد الأنفاق، أو نقل الملفات عبر SSH، أو تكوين مضيفي الانتقال السريع، أو تقوية الوصول إلى SSH.
