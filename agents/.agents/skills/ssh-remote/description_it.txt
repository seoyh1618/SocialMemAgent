Modelli di accesso remoto SSH e utilità di shell sicura. Copre connessioni, gestione della configurazione, generazione di chiavi (Ed25519, FIDO2), tunneling, port forwarding, trasferimenti di file e multiplexing.

Da utilizzare per la connessione ai server, la gestione delle chiavi SSH, la configurazione di tunnel, il trasferimento di file su SSH, la configurazione di host jump o il rafforzamento dell'accesso SSH.
