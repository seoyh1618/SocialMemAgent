Записывайте контент в Feishu Docs с автоматическими диаграммами Mermaid/PlantUML, отображаемыми в виде досок. Поддерживает расширенные действия, такие как добавление досок, выносок и блоков пакетного обновления. Используйте его, когда пользователи запрашивают создание, запись или обновление документов Feishu. Рекомендуется использовать схемы русалок.
