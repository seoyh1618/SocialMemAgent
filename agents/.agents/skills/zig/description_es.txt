Patrones de lenguaje de programación Zig actualizados para la versión 0.15.2. Úselo al escribir, revisar o depurar código Zig, trabajar con archivos build.zig y build.zig.zon o usar metaprogramación comptime. Es fundamental para evitar patrones obsoletos de los datos de entrenamiento, especialmente las API del sistema de compilación (root_module en lugar de root_source_file, addLibrary con enlace), API de E/S (patrón de escritura en búfer), inicialización de contenedores (.empty/.init), selección de asignador (DebugAllocator), ArrayList ahora no administrado de forma predeterminada, @typeInfo campos en minúsculas (.@"struct" no .Struct) y funciones de lenguaje eliminadas (async/await, usando espacio de nombres).
