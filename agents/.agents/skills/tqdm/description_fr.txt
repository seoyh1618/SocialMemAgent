Une barre de progression rapide et extensible pour Python et CLI. Fait instantanément apparaître vos boucles un indicateur de progression intelligent avec ETA, itérations par seconde et statistiques personnalisables. Frais généraux minimes. À utiliser pour surveiller les boucles de longue durée, les simulations, le traitement des données, la formation ML, les téléchargements de fichiers, les opérations d'E/S, les outils de ligne de commande, les opérations pandas, les tâches parallèles et les barres de progression imbriquées.
