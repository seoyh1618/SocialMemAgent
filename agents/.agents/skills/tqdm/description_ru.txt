Быстрый и расширяемый индикатор выполнения для Python и CLI. Мгновенно заставляет ваши циклы отображать интеллектуальный индикатор прогресса с расчетным временем прибытия, количеством итераций в секунду и настраиваемой статистикой. Минимальные накладные расходы. Используйте для мониторинга длительных циклов, моделирования, обработки данных, обучения машинного обучения, загрузки файлов, операций ввода-вывода, инструментов командной строки, операций pandas, параллельных задач и вложенных индикаторов выполнения.
