OpenAI の Realtime API (スピーチツースピーチ) の音声プロンプトを作成するための専門ガイド。システム プロンプト、会話フロー、ツール定義、またはリアルタイム API 上に構築された音声アシスタントのセッション構成を作成、編集、または確認するときに使用します。プロンプトの構造、性格/口調、音声動作 (速度、言語、多様性)、参考発音、ツール呼び出しパターン (プリアンブル、動作タイプ、言い換えスーパーバイザー)、会話フロー設計 (静的ステート マシン、動的な session.update)、サンプル フレーズ、安全性/エスカレーションについて説明します。タスクに音声 AI プロンプトの作成または改善が含まれる場合は、必ずこのスキルを使用してください。
