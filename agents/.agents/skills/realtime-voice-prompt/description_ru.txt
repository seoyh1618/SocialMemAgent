Экспертное руководство по написанию голосовых подсказок для API OpenAI Realtime (речь в речь). Используйте при создании, редактировании или просмотре системных подсказок, потоков разговоров, определений инструментов или конфигурации сеанса для голосовых помощников, созданных на основе Realtime API. Охватывает структуру подсказок, индивидуальность/тон, поведение голоса (скорость, язык, разнообразие), эталонное произношение, шаблоны вызова инструментов (преамбулы, типы поведения, супервизор перефразирования), структуру потока разговора (статические конечные автоматы, динамический сеанс.обновление), примеры фраз и безопасность/эскалацию. Используйте этот навык всякий раз, когда задача включает в себя написание или улучшение голосовых подсказок искусственного интеллекта.
