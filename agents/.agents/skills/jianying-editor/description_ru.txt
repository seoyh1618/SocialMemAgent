Используйте библиотеку pyJianYingDraft для автоматизации создания, редактирования и управления черновиками видео JianYing. Поддерживает многодорожечное редактирование, анимацию, эффекты, ключевые кадры, замену шаблонов, импорт/экспорт субтитров, автоматический экспорт и запуск инструментов записи экрана.
