Ejecute la fase de verificación previa del flujo de trabajo ITP para crear ADR y diseñar artefactos de especificaciones. Úselo siempre que el usuario solicite crear un ADR, escribir una especificación de diseño, configurar documentación en formato MADR o cuando el flujo de trabajo /itp:go entre en su etapa de verificación previa. NO lo use para redacción de documentación general o formato de rebajas que no esté relacionado con el flujo de trabajo de desarrollo impulsado por ADR.
