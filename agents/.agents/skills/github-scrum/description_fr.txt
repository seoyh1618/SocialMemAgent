Gérez des projets logiciels avec Scrum sur GitHub. Planifiez les MVP, maintenez un backlog produit en tant que problèmes, exécutez des sprints en tant que jalons et automatisez la configuration avec les outils GitHub MCP (de préférence) ou gh CLI (de secours). Adapté aux développeurs solo et aux petites équipes (1 à 3 personnes).
