Recherche de session d'agent de codage - CLI/TUI unifiée pour indexer et rechercher l'historique de l'agent de codage local à partir de Claude Code, Codex, Gemini, Cursor, Aider, ChatGPT, Cline, OpenCode, Amp, Pi-Agent, Factory, etc. À utiliser lors de la recherche de conversations d'agent passées, de l'indexation de l'historique des sessions de codage, de la recherche de solutions précédentes entre agents ou de l'interrogation des journaux de session avec le mode robot CASS CLI.
