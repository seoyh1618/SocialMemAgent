Coding Agent Session Search – einheitliche CLI/TUI zum Indizieren und Durchsuchen des lokalen Coding Agent-Verlaufs von Claude Code, Codex, Gemini, Cursor, Aider, ChatGPT, Cline, OpenCode, Amp, Pi-Agent, Factory und mehr. Verwenden Sie diese Option, wenn Sie vergangene Agentengespräche durchsuchen, den Codierungssitzungsverlauf indizieren, frühere Lösungen für Agenten finden oder Sitzungsprotokolle mit dem CASS-CLI-Robotermodus abfragen.
