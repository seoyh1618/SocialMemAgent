Поиск сеанса агента кодирования — унифицированный CLI/TUI для индексации и поиска локальной истории агента кодирования из Claude Code, Codex, Gemini, Cursor, Aider, ChatGPT, Cline, OpenCode, Amp, Pi-Agent, Factory и других. Используйте при поиске прошлых разговоров с агентами, индексировании истории сеансов кодирования, поиске предыдущих решений между агентами или запросе журналов сеансов с помощью режима робота CASS CLI.
