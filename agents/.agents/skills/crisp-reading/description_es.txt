Lectura CRISP: compañero de lectura profunda de IA. Comprender · Repasar · Interiorizar · Sintetizar · Practicar. Integra la lectura analítica de Adler, la disección de libros TIPS de Fan Deng, el grupo de disección de libros RIA, Zettelkasten, técnica Feynman, Autoexplicación, Steel-Manning. Analiza libros y genera informes de lectura HTML interactivos. Úselo cuando: (1) el usuario dice "lea este libro", "analice este libro", "léalo por mí", "vale la pena leer este libro", "analice este libro", "reseña del libro", "resumen del libro", "notas de lectura", "de qué se trata este libro", "lectura profunda", (2) solicita evaluación del libro, notas de lectura, informes de lectura, (3) menciona CRISP, lectura CRISP, lectura profunda. No apto para: resúmenes puros de documentos (“resúmelos esto para mí” sin intención de lectura), revisiones sistemáticas de literatura de artículos académicos, capacitación en lectura rápida.
