Analyse de l'utilisation des outils de codage d'IA. Suit l'utilisation des jetons dans Claude Code, Codex CLI, OpenCode, Gemini CLI et les outils externes. À utiliser lorsque l'utilisateur souhaite afficher les statistiques d'utilisation de l'IA, synchroniser les données d'utilisation ou configurer le suivi automatique de l'utilisation.
