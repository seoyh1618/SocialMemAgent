À utiliser lorsqu'un binaire se comporte incorrectement sans planter, en diagnostiquant les erreurs de fichier introuvable, les échecs d'autorisation, les problèmes de réseau ou les appels de bibliothèque inattendus en traçant les appels système et les appels de fonctions de bibliothèque. S'active sur les requêtes concernant strace, ltrace, le traçage d'appel système, l'interception de bibliothèque, ENOENT, EPERM, strace -e ou le diagnostic du comportement binaire sans débogueur.
