دليل لبناء التطبيقات على منصة Cloudflare المتطورة. يُستخدم عند تنفيذ وظائف بدون خادم (العمال)، أو قواعد بيانات الحافة (D1)، أو التخزين (R2، KV)، أو تطبيقات الوقت الفعلي (الكائنات المعمرة)، أو ميزات الذكاء الاصطناعي (الذكاء الاصطناعي للعمال، أو بوابة الذكاء الاصطناعي)، أو المواقع الثابتة (الصفحات)، أو أي حلول حوسبة حافة.
