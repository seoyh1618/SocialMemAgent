Руководство по созданию приложений на периферийной платформе Cloudflare. Используйте при реализации бессерверных функций (Workers), пограничных баз данных (D1), хранилища (R2, KV), приложений реального времени (Durable Objects), функций искусственного интеллекта (Workers AI, AI Gateway), статических сайтов (Pages) или любых решений для периферийных вычислений.
