Guida all'architettura del modulo Agent Builder (Go Language), che copre il flusso di avvio dell'agente, il meccanismo di heartbeat, la richiesta e l'esecuzione delle attività, l'upgrade e l'aggiornamento e l'interazione con Dispatch. Utilizzato quando gli utenti sviluppano funzionalità dell'agente, modificano la logica dell'heartbeat, gestiscono l'esecuzione delle attività o implementano gli aggiornamenti dell'agente.
