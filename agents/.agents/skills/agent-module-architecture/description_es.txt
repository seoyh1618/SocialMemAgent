Guía de arquitectura del módulo Agent Builder (lenguaje Go), que cubre el flujo de inicio del agente, el mecanismo de latido, la reclamación y ejecución de tareas, la actualización y la actualización, y la interacción con Dispatch. Se utiliza cuando los usuarios desarrollan funciones del Agente, modifican la lógica de latidos, manejan la ejecución de tareas o implementan actualizaciones del Agente.
