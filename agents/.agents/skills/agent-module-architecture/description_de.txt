Leitfaden zur Architektur des Agent Builder-Moduls (Go-Sprache), der den Startablauf des Agenten, den Heartbeat-Mechanismus, die Anforderung und Ausführung von Aufgaben, Upgrades und Updates sowie die Interaktion mit Dispatch behandelt. Wird verwendet, wenn Benutzer Agent-Funktionen entwickeln, die Heartbeat-Logik ändern, die Aufgabenausführung verwalten oder Agent-Upgrades implementieren.
