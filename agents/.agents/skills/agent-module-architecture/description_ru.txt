Руководство по архитектуре модуля Agent Builder (язык Go), в котором описывается процесс запуска агента, механизм контрольного сигнала, утверждение и выполнение задач, обновление и обновление, а также взаимодействие с Dispatch. Используется, когда пользователи разрабатывают функции агента, изменяют логику контрольных сигналов, обрабатывают выполнение задач или внедряют обновления агента.
