سجل القرارات المستمر الذي ينشئ حلقة تعليقات للقراءة والكتابة: اقرأ القرارات السابقة قبل بدء العمل، واحترمها أو استبدلها، واكتب سجلات بيان Y جديدة أثناء تقدمك. يتم تنشيطه أثناء أي مهمة تتضمن قرارات معمارية أو هيكلية أو تصميمية - اختيار المكتبات، وتصميم المخططات، وإنشاء الوحدات النمطية، وإجراء المفاضلات، واختيار الأنماط. يتم تنشيطه أيضًا عندما يكون DECISIONS.md موجودًا بالفعل في المشروع. لا تقم بالتنشيط لإجراء تغييرات تافهة مثل الأخطاء المطبعية أو إعادة التسمية أو التنسيق.
