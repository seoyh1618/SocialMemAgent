Nutzen Sie die Browserautomatisierung von Playwright, um Notion-Dokumente schnell zu erstellen und zu verfeinern.
Auslöseszenarien: Benutzer erwähnt „Notion“, „Dokument verfeinern“, „Seite erstellen“, „Inhalt füllen“, „Notion-Dokument“.
Anwendbar für: Erstellen neuer Seiten, Ändern von Titeln, Einfügen von Markdown-Inhalten, Formatieren von Dokumenten.
