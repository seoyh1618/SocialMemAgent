Используйте автоматизацию браузера Playwright для быстрого создания и уточнения документов Notion.
Сценарии триггера: пользователь упоминает «Понятие», «Уточнить документ», «Создать страницу», «Заполнить контент», «Документ Notion».
Применимо для: создания новых страниц, изменения заголовков, вставки содержимого Markdown, форматирования документов.
