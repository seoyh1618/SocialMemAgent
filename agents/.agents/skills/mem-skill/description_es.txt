Sistema de acumulación de conocimiento y memoria autoevolutivo para agentes de IA. Actúa como un "segundo cerebro" persistente que recupera automáticamente experiencias pasadas, captura las mejores prácticas y registra proactivamente soluciones exitosas en una base de conocimiento privada. Utilice esta habilidad siempre que inicie una tarea, abra una nueva conversación o active cualquier otra habilidad. Admite motores de memoria (predeterminado: índice JSON/Markdown integrado; opcional: búsqueda semántica QMD). Inicialice con: /mem-skill init [--mem-engine=qmd].
