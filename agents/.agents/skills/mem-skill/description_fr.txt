Système de mémoire et d'accumulation de connaissances auto-évolutif pour les agents d'IA. Agit comme un « second cerveau » persistant qui récupère automatiquement les expériences passées, capture les meilleures pratiques et enregistre de manière proactive les solutions réussies dans une base de connaissances privée. Utilisez cette compétence chaque fois que vous démarrez une tâche, ouvrez une nouvelle conversation ou déclenchez toute autre compétence. Prend en charge les moteurs de mémoire (par défaut : index JSON/Markdown intégré ; facultatif : recherche sémantique QMD). Initialisez avec : /mem-skill init [--mem-engine=qmd].
