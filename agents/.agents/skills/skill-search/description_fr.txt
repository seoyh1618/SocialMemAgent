Recherchez sur des sites officiels comme GitHub et SkillsMP des compétences de haute qualité correspondant à la description de l'utilisateur, présentez-les pour sélection, puis clonez et installez automatiquement la compétence choisie dans le répertoire global ~/.claude/skills/. Déclenchez cette compétence lorsque l'utilisateur doit rechercher ou installer une nouvelle compétence.
