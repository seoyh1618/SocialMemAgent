Найдите на официальных сайтах, таких как GitHub и SkillsMP, высококачественные навыки, соответствующие описанию пользователя, представьте их для выбора, затем автоматически клонируйте и установите выбранный навык в глобальный каталог ~/.claude/skills/. Активируйте этот навык, когда пользователю нужно найти или установить новый навык.
