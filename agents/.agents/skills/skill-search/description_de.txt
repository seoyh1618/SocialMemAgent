Durchsuchen Sie offizielle Websites wie GitHub und SkillsMP nach hochwertigen Fertigkeiten, die der Beschreibung des Benutzers entsprechen, präsentieren Sie sie zur Auswahl und klonen Sie dann automatisch die ausgewählte Fertigkeit und installieren Sie sie im globalen Verzeichnis ~/.claude/skills/. Lösen Sie diesen Skill aus, wenn der Benutzer nach einem neuen Skill suchen oder diesen installieren muss.
