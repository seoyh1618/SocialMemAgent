قم بإعداد بوابة وكيل AI مستمرة على نظام macOS باستخدام جسر حدث Redis ومراقبة نبضات القلب والتوجيه متعدد الجلسات. أسئلة وأجوبة تفاعلية تتوافق مع هدفك - بدءًا من الحد الأدنى (Redis + الامتداد) وحتى الكامل (البرنامج الخفي المضمن + Telegram + watchdog). يُستخدم عندما: "إعداد بوابة"، أو "أريد وكيلي قيد التشغيل دائمًا"، أو "جسر الحدث"، أو "مراقبة نبضات القلب"، أو "إشعارات الوكيل"، أو أي طلب لجعل وكيل الذكاء الاصطناعي دائمًا وقابلاً للوصول.
