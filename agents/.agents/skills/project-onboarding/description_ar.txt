قم بالتحميل بشكل استباقي عند بدء العمل على قاعدة تعليمات برمجية غير مألوفة أو إعداد مشروع جديد. يُستخدم عندما يقول المستخدم "ساعدني في فهم قاعدة التعليمات البرمجية هذه" أو "أنضم إلي" أو "ماذا يفعل هذا المشروع" أو "إعداد بيئتي" أو "رسم خريطة للهندسة المعمارية". يغطي تحليل بنية قاعدة التعليمات البرمجية، ورسم خرائط البنية، وتدقيق التبعية، والكشف عن التقاليد والأنماط، وإعداد بيئة المطور، وتوثيق النتائج للمساهمة الإنتاجية السريعة.
