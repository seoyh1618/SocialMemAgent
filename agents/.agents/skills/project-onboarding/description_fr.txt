Chargez de manière PROACTIVE lorsque vous commencez à travailler sur une base de code inconnue ou lorsque vous configurez un nouveau projet. À utiliser lorsque l'utilisateur dit "aidez-moi à comprendre cette base de code", "intégrez-moi", "que fait ce projet", "configurez mon environnement" ou "cartographiez l'architecture". Couvre l'analyse de la structure de la base de code, le mappage de l'architecture, l'audit des dépendances, la détection des conventions et des modèles, la configuration de l'environnement de développement et la documentation des résultats pour une contribution productive rapide.
