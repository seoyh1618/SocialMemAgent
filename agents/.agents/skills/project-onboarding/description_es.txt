Cargue PROACTIVAMENTE cuando comience a trabajar en una base de código desconocida o configure un nuevo proyecto. Úselo cuando el usuario diga "ayúdame a comprender este código base", "incorpórame", "qué hace este proyecto", "configurar mi entorno" o "mapear la arquitectura". Cubre el análisis de la estructura de la base del código, el mapeo de la arquitectura, la auditoría de dependencias, la detección de convenciones y patrones, la configuración del entorno del desarrollador y la documentación de los hallazgos para una contribución productiva rápida.
