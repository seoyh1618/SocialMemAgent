あらゆるドキュメント、アウトライン、コンテンツを、洗練された自己完結型の HTML スライド プレゼンテーションに変換します。ユーザーがドキュメント、マークダウン ファイル、アウトライン、メモ、またはテキスト コンテンツからプレゼンテーション、スライド デッキ、ピッチ デッキ、またはトークを作成し、その出力を HTML ファイル (PPTX ではない) として希望する場合は、このスキルを使用します。また、ユーザーが「HTML プレゼンテーション」、「Web ベースのスライド」、「reveal.js デッキ」、「ブラウザ プレゼンテーション」を要求した場合、またはドキュメントをブラウザから表示できるスライドに変換したい場合にもトリガーされます。水平スクロール (左/右、Reveal.js 搭載) と垂直スクロール (上から下、キーボード/スクロール ナビゲーション) の 2 つのナビゲーション モードをサポートします。複数のビジュアルテーマ (ダークエディトリアル、ライトミニマル、企業、ハッカーターミナル) が含まれています。まだ指定されていない場合は、テーマ、ナビゲーションの方向、コンテンツ構造についてユーザーに説明を求めることを検討してください。
