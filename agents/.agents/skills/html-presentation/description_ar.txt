قم بتحويل أي مستند أو مخطط تفصيلي أو محتوى إلى عرض شرائح HTML مصقول ومستقل. استخدم هذه المهارة عندما يريد المستخدم إنشاء عرض تقديمي أو مجموعة شرائح أو مجموعة عرض تقديمي أو التحدث من مستند أو ملف تخفيض السعر أو مخطط تفصيلي أو ملاحظات أو أي محتوى نصي ويريد الإخراج كملف HTML (وليس PPTX). يتم تشغيله أيضًا عندما يطلب المستخدم "عرض تقديمي بتنسيق HTML"، أو "شرائح مستندة إلى الويب"، أو "مجموعة reveal.js"، أو "عرض تقديمي في المتصفح"، أو يريد تحويل مستند إلى شرائح يمكنه تقديمها من المتصفح. يدعم وضعين للتنقل - الأفقي (يسار/يمين، مدعوم من Reveal.js) والتمرير العمودي (من الأعلى إلى الأسفل، التنقل عبر لوحة المفاتيح/التمرير). يتضمن موضوعات مرئية متعددة (افتتاحية داكنة، وخفيفة، ومحطة خاصة بالشركة، ومحطة القراصنة). فكر في مطالبة المستخدم بتوضيح الموضوع واتجاه التنقل وبنية المحتوى إذا لم يتم تحديدها بالفعل.
