Convierta cualquier documento, esquema o contenido en una presentación de diapositivas HTML pulida e independiente. Utilice esta habilidad siempre que el usuario quiera crear una presentación, presentación de diapositivas, presentación de propuestas o charla a partir de un documento, archivo de rebajas, esquema, notas o cualquier contenido textual y desee el resultado como un archivo HTML (no PPTX). También se activa cuando el usuario solicita una "presentación HTML", "diapositivas basadas en web", "plataforma reveal.js", "presentación de navegador" o desea convertir un documento en diapositivas que pueda presentar desde un navegador. Admite dos modos de navegación: horizontal (izquierda/derecha, con tecnología Reveal.js) y desplazamiento vertical (de arriba a abajo, navegación con teclado/desplazamiento). Incluye múltiples temas visuales (editorial oscuro, minimalista claro, corporativo, terminal de hackers). Considere pedirle al usuario aclaraciones sobre el tema, la dirección de navegación y la estructura del contenido si aún no se han especificado.
