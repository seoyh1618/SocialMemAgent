إطار تجريبي آمن لوكلاء الذكاء الاصطناعي. ينشئ بيئات معزولة لتجربة الميزات الجديدة واختبار الأساليب واستكشاف الحلول دون تلويث قاعدة التعليمات البرمجية الرئيسية.
متى الاستخدام: يحتاج الوكيل إلى تجربة شيء غير مؤكد، أو استكشاف طرق متعددة، أو اختبار مكتبة جديدة، أو إنشاء نموذج أولي لميزة، أو تشغيل ارتفاع تقني قبل الالتزام بالتنفيذ.
المشغلات الأساسية: "تجربة مع" = إعداد وضع الحماية + تشغيل التجربة "جرب هذا الأسلوب" = تجربة سريعة في وضع الحماية "spike" / "POC" / "النموذج الأولي" = تحقيق فني محدد بالوقت "العبث" / "وضع الترقيع" = أدخل سير عمل التجربة "استكشاف الخيارات" = مقارنة متعددة الأساليب في وضع الحماية
ليس من أجل: تصحيح الأخطاء (استخدم مصحح الأخطاء)، أو الاختبار (استخدم مشغل الاختبار)، أو العمل المميز الملتزم (استخدم فروع git).
أداة التفريق: على عكس فروع git (للتوجيه الملتزم)، فإن الترقيع مخصص للاستكشاف "لا أعرف ما إذا كان هذا سينجح". جرب 5 أشياء في وضع الحماية قبل الالتزام بالفرع. ردود فعل أسرع، بدون تلوث لقاعدة التعليمات البرمجية.
