Безопасная система экспериментов для агентов ИИ. Создает изолированные среды-песочницы для опробования новых функций, подходов к тестированию и изучения решений без загрязнения основной базы кода.
КОГДА ИСПОЛЬЗУЕТСЯ: агенту необходимо попробовать что-то неопределенное, изучить несколько подходов, протестировать новую библиотеку, создать прототип функции или запустить технический пик, прежде чем приступить к реализации.
ОСНОВНЫЕ ТРИГГЕРЫ: «экспериментировать с» = настроить «песочницу» + запустить эксперимент «попробовать этот подход» = быстрый эксперимент в «песочнице» «всплеск» / «POC» / «прототип» = техническое исследование с ограничением по времени «механик» / «режим работы» = войти в рабочий процесс экспериментирования «изучить параметры» = сравнение нескольких подходов в песочнице
НЕ ДЛЯ: Отладки (используйте отладчик), тестирования (используйте средство запуска тестов) или работы с фиксированными функциями (используйте ветки git).
ДИФФЕРЕНЦИАТОР: В отличие от веток git (для четкого направления), работа предназначена для исследования типа «я не знаю, сработает ли это». Попробуйте 5 вещей в песочнице, прежде чем переходить к ветке. Более быстрая обратная связь, нулевое загрязнение кодовой базы.
