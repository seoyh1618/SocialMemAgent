مراقبة تشغيل حلقات الوكيل، وحالات فشل الفرز، والتنظيف بعد الانتهاء، وتحديد متى يتم التدخل. يُستخدم عندما تكون الحلقة قيد التشغيل وتحتاج إلى رعاية، أو عندما تنتهي الحلقة للتو وتحتاج إلى التحقق بعد الدمج، أو عندما يتم تخطي/فشل القصص وتحتاج إلى التشخيص، أو عندما تحتاج عناصر الاختبار القديمة إلى التنظيف. المشغلات: "التحقق من الحلقة"، أو "ما حدث مع الحلقة"، أو "انتهت الحلقة"، أو "التنظيف بعد الحلقة"، أو "لماذا تم تخطي هذه القصة"، أو "حلقة المراقبة"، أو "مربية الحلقة"، أو أي مهمة لإدارة الحلقة بعد البداية. تختلف عن مهارة حلقة الوكيل (التي تتعامل مع حلقات البداية).
