Комплексный набор инструментов для создания, проверки и управления ресурсами Kubernetes YAML. Используйте этот навык при создании манифестов Kubernetes (развертывания, службы, ConfigMaps, StatefulSets и т. д.), работе с пользовательскими определениями ресурсов (CRD) или создании готовых к эксплуатации конфигураций K8s.
