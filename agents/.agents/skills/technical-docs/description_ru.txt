Написание технической документации и создание схем. Используйте при создании документов, синхронизации документации с изменениями кода, построении диаграмм Mermaid, проведении аудита покрытия документации или создании руководств по стилю написания. Используйте для рабочих процессов «документ как код», создания ERD, диаграмм последовательности, анализа пробелов в документации и черчения с помощью искусственного интеллекта.
