كتابة الوثائق الفنية وإنشاء الرسوم البيانية. يُستخدم عند إنشاء المستندات، أو مزامنة الوثائق مع تغييرات التعليمات البرمجية، أو إنشاء مخططات حورية البحر، أو تشغيل عمليات تدقيق تغطية المستندات، أو إنشاء أدلة أسلوب الكتابة. يُستخدم في سير عمل المستند كرمز، وإنشاء ERD، ومخططات التسلسل، وتحليل فجوات التوثيق، والصياغة بمساعدة الذكاء الاصطناعي.
