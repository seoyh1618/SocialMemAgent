Suddividi modifiche di codice di grandi dimensioni in piccole richieste pull in pila utilizzando Vanilla Git e la CLI gh. Attivazione automatica quando si implementa una funzionalità o una modifica che si estende su più passaggi logici, tocca diversi file o supera circa 200 righe modificate. Attiva anche "stack PR", "dividi questo in PR più piccoli", "differenze impilate" o "crea uno stack PR". NON attivare correzioni di file singoli, piccole correzioni di bug o modifiche inferiori a ~200 righe che costituiscono una singola unità logica.
