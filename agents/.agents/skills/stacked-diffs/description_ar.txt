قم بتقسيم تغييرات التعليمات البرمجية الكبيرة إلى طلبات سحب صغيرة ومكدسة باستخدام Vanilla git وgh CLI. يتم التشغيل تلقائيًا عند تنفيذ ميزة أو تغيير يمتد عبر خطوات منطقية متعددة، أو يمس عدة ملفات، أو سيتجاوز حوالي 200 سطر تم تغييره. قم أيضًا بتشغيل "مكدس العلاقات العامة" أو "تقسيم هذا إلى علاقات عامة أصغر" أو "الاختلافات المكدسة" أو "إنشاء مكدس علاقات عامة". لا تقم بتشغيل إصلاحات الملف الفردي، أو إصلاحات الأخطاء الصغيرة، أو التغييرات تحت 200 سطر تقريبًا والتي تمثل وحدة منطقية واحدة.
