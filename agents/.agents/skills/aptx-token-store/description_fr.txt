Utilisez @aptx/token-store pour définir ou implémenter la persistance des jetons. Pour : implémenter l'interface TokenStore, prendre en charge les API sync/async (getToken/setToken/clearToken), les méthodes de métadonnées facultatives, implémenter différents backends de stockage (cookie, localStorage, mini-programme, mémoire) et travailler avec @aptx/api-plugin-auth. Déclenché lorsque le code doit implémenter un TokenStore personnalisé.
