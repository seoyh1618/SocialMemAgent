Используйте @aptx/token-store, чтобы определить или реализовать сохранение токена. Используется для: реализации интерфейса TokenStore, поддержки API-интерфейсов синхронизации/асинхронности (getToken/setToken/clearToken), дополнительных методов метаданных, реализации различных серверных модулей хранения (cookie, localStorage, мини-программа, память) и используется вместе с @aptx/api-plugin-auth. Срабатывает, когда коду необходимо реализовать собственный TokenStore.
