Jotai 국가 관리 라이브러리의 전문가 수준 기술. React 애플리케이션에서 원자 기반 상태 관리를 구현할 때 사용됩니다. 다음과 같은 경우에 이 기술을 적용하십시오.
(1) Jotai Atom 설계 및 구현
(2) 파생된 원자, 비동기 원자 및atomFamily 구현
(3) Jotai 모범 사례를 기반으로 한 리팩토링
(4) 성능 최적화 (selectAtom, SplitAtom 등)
(5) 지속성(localStorage/sessionStorage와의 통합)
(6) TypeScript 유형 정의
(7) 테스트 구현
사용자가 "Jotai", "atom" 또는 "상태 관리"와 관련된 질문을 하거나 구현을 요청할 때 트리거됩니다.
