Consulta y analiza datos utilizando la CLI de DuckDB. Utilícelo cuando el usuario necesite ejecutar consultas SQL, analizar archivos CSV/Parquet/JSON, crear o consultar bases de datos, exportar datos o realizar cualquier análisis de datos ad-hoc desde la línea de comandos. Los desencadenantes incluyen solicitudes para "consultar un archivo", "analizar datos", "ejecutar SQL", "leer un CSV/Parquet/JSON", "crear una base de datos", "exportar a CSV/Parquet" o cualquier tarea de análisis de datos que se beneficie de SQL.
