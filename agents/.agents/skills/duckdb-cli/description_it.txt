Esegui query e analizza i dati utilizzando la CLI DuckDB. Da utilizzare quando l'utente deve eseguire query SQL, analizzare file CSV/Parquet/JSON, creare o eseguire query su database, esportare dati o eseguire qualsiasi analisi dei dati ad hoc dalla riga di comando. I trigger includono richieste di "interrogare un file", "analizzare dati", "eseguire SQL", "leggere un CSV/Parquet/JSON", "creare un database", "esportare in CSV/Parquet" o qualsiasi attività di analisi dei dati che trae vantaggio da SQL.
