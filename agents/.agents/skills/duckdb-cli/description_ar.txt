الاستعلام عن البيانات وتحليلها باستخدام DuckDB CLI. يُستخدم عندما يحتاج المستخدم إلى تشغيل استعلامات SQL، أو تحليل ملفات CSV/Parquet/JSON، أو إنشاء قواعد بيانات أو الاستعلام عنها، أو تصدير البيانات، أو إجراء أي تحليل بيانات مخصص من سطر الأوامر. تتضمن المشغلات طلبات "الاستعلام عن ملف"، أو "تحليل البيانات"، أو "تشغيل SQL"، أو "قراءة ملف CSV/Parquet/JSON"، أو "إنشاء قاعدة بيانات"، أو "التصدير إلى CSV/Parquet"، أو أي مهمة تحليل بيانات تستفيد من SQL.
