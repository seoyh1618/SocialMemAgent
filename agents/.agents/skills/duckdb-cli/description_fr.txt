Interrogez et analysez les données à l'aide de la CLI DuckDB. À utiliser lorsque l'utilisateur doit exécuter des requêtes SQL, analyser des fichiers CSV/Parquet/JSON, créer ou interroger des bases de données, exporter des données ou effectuer toute analyse de données ad hoc à partir de la ligne de commande. Les déclencheurs incluent les requêtes pour « interroger un fichier », « analyser des données », « exécuter SQL », « lire un CSV/Parquet/JSON », « créer une base de données », « exporter vers CSV/Parquet » ou toute tâche d'analyse de données bénéficiant de SQL.
