Úselo cuando el usuario diga "investigar esta idea", "encontrar competidores", "verificar el mercado", "disponibilidad del dominio", "tamaño del mercado" o "analizar oportunidades". NO lo utilice para puntuación de ideas (use /validate) ni para auditorías SEO de páginas existentes (use /seo-audit).
