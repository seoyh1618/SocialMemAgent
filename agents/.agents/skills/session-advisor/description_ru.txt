Мультимодельный советник по управлению SESSION для совместной работы. Автоматически предоставляет предложения по повторному использованию или созданию сеанса перед вызовом Codex или Gemini. Используется, когда пользователи обсуждают многомодельное сотрудничество, управление СЕССИЯМИ или изоляцию контекста.
