Assistant de style code C intégré, basé sur les normes de codage de 51 projets pédagogiques sur les microcontrôleurs. La valeur par défaut est Snake_case, avec camelCase en option. Utilisé pour créer des structures de projet conformes aux normes de développement intégrées, optimiser le style de code et fournir des modèles de pilotes matériels. Convient au développement de projets C embarqués sur 51 microcontrôleurs, STM32 et plus encore.
