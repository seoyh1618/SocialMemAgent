Встроенный помощник по стилю кода C, основанный на стандартах кодирования из 51 проекта обучения микроконтроллерам. По умолчанию используется Snake_case с необязательным CamelCase. Используется для создания структур проекта, соответствующих нормам разработки встроенных систем, оптимизации стиля кода и предоставления шаблонов драйверов оборудования. Подходит для разработки встроенных проектов C на 51 микроконтроллере, STM32 и т. д.
