Assistente in stile codice C integrato, basato sugli standard di codifica di 51 progetti didattici sui microcontrollori. Il valore predefinito è snake_case, con camelCase opzionale. Utilizzato per creare strutture di progetto conformi alle norme di sviluppo integrate, ottimizzare lo stile del codice e fornire modelli di driver hardware. Adatto per lo sviluppo di progetti C integrati su 51 microcontrollori, STM32 e altro.
