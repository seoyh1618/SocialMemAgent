مساعد نمط كود C المضمن، استنادًا إلى معايير الترميز من 51 مشروعًا تعليميًا لوحدات التحكم الدقيقة. الإعدادات الافتراضية هي Snake_case، مع CamelCase الاختياري. يُستخدم لإنشاء هياكل مشروع متوافقة مع معايير التطوير المضمنة وتحسين نمط التعليمات البرمجية وتوفير قوالب برامج تشغيل الأجهزة. مناسب لتطوير مشروع C المضمن على 51 وحدة تحكم دقيقة وSTM32 والمزيد.
