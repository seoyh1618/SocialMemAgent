Asistente de estilo de código C integrado, basado en estándares de codificación de 51 proyectos de enseñanza de microcontroladores. El valor predeterminado es Snake_case, con camelCase opcional. Se utiliza para crear estructuras de proyectos que cumplan con las normas de desarrollo integradas, optimizar el estilo del código y proporcionar plantillas de controladores de hardware. Adecuado para el desarrollo de proyectos C integrados en 51 microcontroladores, STM32 y más.
