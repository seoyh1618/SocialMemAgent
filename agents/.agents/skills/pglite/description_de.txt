PGlite bettet PostgreSQL 17.4 in den Browser und Node.js über WASM ein, unter 3 MB gzipped. Deckt Speicher-Backends, Abfragen, Transaktionen, Erweiterungen (pgvector, pg_trgm, Volltextsuche), Live-Abfragen, React-Integration, Multi-Tab-Worker-Architektur und Electric Sync ab.

Verwenden Sie es, wenn Sie PGlite einrichten, Speicher-Backends auswählen, Abfragen oder Transaktionen schreiben, Erweiterungen konfigurieren, Live-Abfragen implementieren, mit React integrieren, Multi-Tab-Worker verwalten oder mit Electric synchronisieren.
