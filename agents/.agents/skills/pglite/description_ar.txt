يقوم PGlite بتضمين PostgreSQL 17.4 في المتصفح وNode.js عبر WASM، ضمن ملف مضغوط بحجم 3 ميجابايت. يغطي الواجهات الخلفية للتخزين، والاستعلامات، والمعاملات، والإضافات (pgvector، وpg_trgm، والبحث عن نص كامل)، والاستعلامات المباشرة، وتكامل React، والبنية العاملة متعددة علامات التبويب، والمزامنة الكهربائية.

يُستخدم عند إعداد PGlite، أو اختيار واجهات التخزين الخلفية، أو كتابة الاستعلامات أو المعاملات، أو تكوين الامتدادات، أو تنفيذ الاستعلامات المباشرة، أو التكامل مع React، أو إدارة عمال علامات التبويب المتعددة، أو المزامنة مع Electric.
