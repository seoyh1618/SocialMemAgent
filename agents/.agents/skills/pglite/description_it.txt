PGlite incorpora PostgreSQL 17.4 nel browser e Node.js tramite WASM, con 3 MB di gzip. Copre backend di archiviazione, query, transazioni, estensioni (pgvector, pg_trgm, ricerca full-text), query in tempo reale, integrazione React, architettura di lavoro multi-tab e sincronizzazione elettrica.

Da utilizzare durante la configurazione di PGlite, la scelta dei backend di archiviazione, la scrittura di query o transazioni, la configurazione di estensioni, l'implementazione di query in tempo reale, l'integrazione con React, la gestione di lavoratori multi-tab o la sincronizzazione con Electric.
