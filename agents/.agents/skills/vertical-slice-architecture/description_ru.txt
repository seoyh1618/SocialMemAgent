Используйте архитектуру вертикального среза (VSA) при создании приложений на любом языке (Go, .NET/C#, Java, Kotlin, TypeScript, Python и т. д.) и любого типа (веб-API, мобильный сервер, интерфейс командной строки, управляемый событиями). Организуйте код по функциям/вариантам использования, а не по техническим уровням. Каждая функция представляет собой автономный вертикальный срез с единственной точкой входа, которая получает дескриптор маршрутизатора/фреймворка и его зависимости. Используйте, когда пользователь говорит «архитектура вертикального среза», «VSA», «организовать функцию», «архитектура на основе функций», «архитектура среза» или при создании нового приложения или функции, когда проект уже соответствует соглашениям VSA. Также используйте при проверке или рефакторинге кода для соответствия принципам VSA.
