مراجعة شاملة للتغييرات المرحلية في Git لتقييم المخاطر واكتشاف الأخطاء وتحليل التأثير. يُستخدم عندما يريد المستخدم مراجعة التغييرات المرحلية، أو التحقق من الأخطاء قبل التنفيذ، أو تحليل المخاطر في ملفات git المرحلية، أو التحقق من صحة تغييرات التعليمات البرمجية قبل الالتزام، أو يحتاج إلى اقتراحات بشأن التعديلات المرحلية.
