Revisione completa delle modifiche Git graduali per la valutazione del rischio, il rilevamento degli errori e l'analisi dell'impatto. Da utilizzare quando l'utente desidera rivedere le modifiche in fasi, verificare la presenza di errori prima del commit, analizzare i rischi nei file in stage git, convalidare le modifiche al codice prima del commit o ha bisogno di suggerimenti sulle modifiche in stage.
