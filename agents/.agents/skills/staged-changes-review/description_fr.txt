Examen complet des modifications échelonnées de Git pour l’évaluation des risques, la détection des erreurs et l’analyse d’impact. À utiliser lorsque l'utilisateur souhaite examiner les modifications par étapes, vérifier les erreurs avant la validation, analyser les risques dans les fichiers préparés par git, valider les modifications de code avant de les valider ou a besoin de suggestions sur les modifications par étapes.
