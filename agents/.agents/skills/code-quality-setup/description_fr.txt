À utiliser lors de l'installation ou de la configuration d'outils de qualité de code (formatteurs, linters, vérificateurs de type, analyseurs de dépendances) pour un référentiel. À utiliser également lors de l’ajout d’une automatisation de l’accessibilité visuelle ou d’une analyse de base de sécurité. Ne pas utiliser pour le codage général ou lorsque les outils sont déjà configurés.
