Используйте при установке или настройке инструментов качества кода (форматеров, линтеров, проверок типов, сканеров зависимостей) для репозитория. Также используйте при добавлении автоматизации визуальной доступности или базового сканирования безопасности. Не используйте для общего кодирования или когда инструменты уже настроены.
