Функции и шаблоны Java 25 для разработки плагинов Hytale. Охватывает современный синтаксис Java (записи, запечатанные классы, сопоставление с образцом, виртуальные потоки), установку JDK и соглашения Java, специфичные для Hytale. Используйте при написании кода Java для плагинов Hytale или устранении проблем с версией Java.
