Fähigkeit zur automatischen Veröffentlichung von WeChat-Kanälen. Nutzen Sie diese Fähigkeit, wenn Benutzer Videos auf WeChat-Kanälen veröffentlichen müssen. Die Fähigkeit umfasst: Erhalten von Anmeldecookies, Hochladen von Videos, Festlegen von Titeln und Hashtags, Planen von Beiträgen und Originalitätserklärungen.
