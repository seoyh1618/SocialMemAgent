Навык создания файлов Markdown для слайдов презентации с помощью Marp. На основе запросов пользователей он генерирует высококачественные слайды, которые полностью используют возможности Marp — директивы, синтаксис изображений, списки фрагментов, настройки тем и т. д. Используйте его для таких запросов, как «создать слайды», «создать презентационные материалы», «написать слайды в Marp» или «подготовить презентационные документы в Markdown».
