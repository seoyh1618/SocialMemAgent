Compétence pour créer des fichiers Markdown pour les diapositives de présentation à l'aide de Marp. Sur la base des demandes des utilisateurs, il génère des diapositives de haute qualité qui exploitent pleinement les fonctionnalités de Marp : directives, syntaxe d'image, listes de fragments, paramètres de thème, etc. Utilisez-le pour des demandes telles que « créer des diapositives », « créer des supports de présentation », « écrire des diapositives dans Marp » ou « préparer des documents de présentation dans Markdown ».
