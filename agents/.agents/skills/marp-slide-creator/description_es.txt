Habilidad para crear archivos Markdown para diapositivas de presentación usando Marp. Según las solicitudes de los usuarios, genera diapositivas de alta calidad que aprovechan al máximo las funciones de Marp: directivas, sintaxis de imágenes, listas de fragmentos, configuración de temas, etc. Úselo para solicitudes como "crear diapositivas", "crear materiales de presentación", "escribir diapositivas en Marp" o "preparar documentos de presentación en Markdown".
