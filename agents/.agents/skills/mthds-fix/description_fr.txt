Résoudre les problèmes dans les bundles MTHDS. À utiliser lorsque l'utilisateur dit « corriger ce flux de travail », « corriger cette méthode », « réparer les erreurs de validation », « le pipeline est cassé », « réparer le fichier .mthds », après que /mthds-check a trouvé des problèmes ou lorsque la validation signale des erreurs. Applique automatiquement les correctifs et revalide en boucle.
