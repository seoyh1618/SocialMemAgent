قم بتطبيق أنماط الاستخدام التفصيلية وأفضل الممارسات لـ Vue SFC في مشاريع weapp-vite. يُستخدم عند كتابة أو إعادة هيكلة ملفات `.vue` للبرامج الصغيرة، واختيار `<script setup>` واستراتيجية ماكرو JSON، أو تحديد `usingComponents`، أو التعامل مع توافق توجيه القالب (حدود `v-model` / `v-bind`)، أو تنسيق خطافات وقت تشغيل wevu، أو استكشاف أخطاء الترجمة/وقت التشغيل في SFC وإصلاحها.
