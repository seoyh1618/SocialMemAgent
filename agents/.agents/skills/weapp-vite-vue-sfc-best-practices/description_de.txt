Wenden Sie detaillierte Nutzungsmuster und Best Practices für Vue SFC in Weapp-Vite-Projekten an. Wird verwendet, wenn Sie „.vue“-Dateien für Miniprogramme schreiben oder umgestalten, „<script setup>“ und eine JSON-Makrostrategie auswählen, „usingComponents“ definieren, die Kompatibilität von Vorlagendirektiven („v-model“ / „v-bind“-Grenzwerte) handhaben, Wevu-Laufzeit-Hooks koordinieren oder SFC-Kompilierungs-/Laufzeitprobleme beheben.
