Применяйте подробные шаблоны использования и лучшие практики для Vue SFC в проектах weapp-vite. Используйте при написании или рефакторинге файлов `.vue` для мини-программ, выборе `<script setup>` и стратегии макросов JSON, определении `usingComponents`, обработке совместимости директив шаблона (ограничения `v-model` / `v-bind`), координации перехватчиков времени выполнения wevu или устранении проблем компиляции/выполнения SFC.
