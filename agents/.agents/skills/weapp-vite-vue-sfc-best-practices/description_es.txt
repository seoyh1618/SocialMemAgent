Aplique patrones de uso detallados y mejores prácticas para Vue SFC en proyectos weapp-vite. Úselo al escribir o refactorizar archivos `.vue` para miniprogramas, elegir `<script setup>` y la estrategia de macro JSON, definir `usingComponents`, manejar la compatibilidad de directivas de plantilla (límites de `v-model` / `v-bind`), coordinar enlaces de tiempo de ejecución de wevu o solucionar problemas de compilación/tiempo de ejecución de SFC.
