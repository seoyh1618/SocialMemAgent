Applica modelli di utilizzo dettagliati e best practice per Vue SFC nei progetti weapp-vite. Da utilizzare durante la scrittura o il refactoring di file ".vue" per mini-programmi, la scelta di "<script setup>" e la strategia macro JSON, la definizione di "usingComponents", la gestione della compatibilità delle direttive del modello (limiti di "v-model" / "v-bind"), il coordinamento degli hook di runtime wevu o la risoluzione dei problemi di compilazione/runtime di SFC.
