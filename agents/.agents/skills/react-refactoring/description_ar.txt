مهارات التحليل النقدي والتنفيذ لإعادة بناء كود React/TypeScript.  
عندما يوفر المستخدم مسار مكون ومشكلاته، تتم مراجعة كل مشكلة بشكل نقدي من منظور "الكود الجيد"، ويتم إجراء تحليل خطوة بخطوة للتفكير المتسلسل، ويتم إنشاء خطة إعادة البناء وتنفيذها.  

المشغل: يُستخدم عند ذكر كلمات رئيسية مثل "إعادة البناء"، و"أريد التحسين"، و"تغيير بنية التعليمات البرمجية"، و"فصل الخطاف"، و"التماسك"، و"الاقتران" مع مسار مكون React.
