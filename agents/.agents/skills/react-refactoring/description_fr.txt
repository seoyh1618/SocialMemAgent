Compétences critiques en matière d'analyse et d'exécution pour la refactorisation de code React/TypeScript.  
Lorsque l'utilisateur fournit un chemin de composant et ses problèmes, chaque problème est examiné de manière critique du point de vue du « bon code », une analyse étape par étape par pensée séquentielle est effectuée et un plan de refactorisation est établi et exécuté.  

Déclencheur : utilisé lorsque des mots-clés tels que "refactoring", "Je veux m'améliorer", "changement de structure de code", "séparation de hook", "cohésion", "couplage" sont mentionnés avec un chemin de composant React.
