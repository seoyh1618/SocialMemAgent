Habilidades críticas de análisis y ejecución para la refactorización de código React/TypeScript.  
Cuando el usuario proporciona una ruta de componente y sus problemas, cada problema se revisa críticamente desde una perspectiva de "buen código", se realiza un análisis paso a paso de pensamiento secuencial y se establece y ejecuta un plan de refactorización.  

Activador: se utiliza cuando palabras clave como "refactorización", "Quiero mejorar", "cambio de estructura de código", "separación de ganchos", "cohesión", "acoplamiento" se mencionan junto con la ruta de un componente de React.
