Genera, convalida e pubblica diagrammi di flusso di lavoro, sequenza e architettura, utilizzando FlowZap Code DSL. Da utilizzare quando l'utente richiede di creare un flusso di lavoro, un diagramma di flusso, un diagramma di sequenza, una mappa di processo o un diagramma di architettura. Produce codice .fz e URL del parco giochi condivisibili tramite il server MCP FlowZap.
