Обеспечивает синхронизацию сайта GitHub Pages (site-docs/) с исходным кодом и изменениями внутренней документации. Запускается автоматически при изменении файлов в каталогах Synapse/, Plugins/, Profiles/ или Site-docs/, а также при изменении Guides/, docs/ или pyproject.toml. Вызов вручную через /github-pages-sync.
