Соглашение о сообщении об ошибках от многопроцессорных работников TTS главному интерфейсу процесса. Используйте этот навык при изменении, добавлении или отладке рабочих TTS в tts_client.py, чтобы гарантировать, что ошибки подключения, квоты и ограничения API правильно отображают всплывающие уведомления для пользователя, а не срабатывают автоматически.
