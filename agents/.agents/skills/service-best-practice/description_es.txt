Ayude a los desarrolladores a escribir Servicios de acuerdo con las pautas del proyecto, siguiendo las mejores prácticas para la arquitectura tRPC + Servicio + DAO. Proporcione orientación sobre la estructura del servicio, la inyección de dependencias, el manejo de errores, ejemplos de código, plantillas, generación de textos estándar y validación de mejores prácticas. Úselo al crear o refactorizar archivos de servicio en el código base.
