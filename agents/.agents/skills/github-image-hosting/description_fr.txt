Téléchargez des images sur img402.dev pour les intégrer dans les PR, les problèmes et les commentaires de GitHub. Les images de moins de 1 Mo sont téléchargées gratuitement (pas de paiement, pas d'authentification) et persistent pendant 7 jours. À utiliser lorsque l'agent a besoin de partager une image dans un contexte GitHub : captures d'écran, maquettes, diagrammes ou tout autre visuel. Déclencheurs : "faire une capture d'écran", "joindre une image", "ajouter une capture d'écran au PR", "télécharger cette maquette" ou toute tâche produisant une image pour GitHub.
