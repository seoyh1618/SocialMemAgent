مهارة التجميع x86-64 لقراءة وكتابة وتصحيح كود التجميع. استخدمه عند قراءة مخرجات تجميع مجلس التعاون الخليجي/Clang، أو كتابة asm المضمنة في C/C++، أو فهم اصطلاح استدعاء System V AMD64 ABI، أو تصحيح أخطاء التسجيل وحالة المكدس. يتم التنشيط عند الاستعلامات حول تجميع x86-64، أو بناء جملة AT&T مقابل Intel، أو asm المضمن، أو اصطلاحات الاتصال، أو جوهريات SIMD، أو قراءة مخرجات التفكيك من objdump أو GDB.
