يدير محتوى المدونات والكتب والصفحات تلقائيًا لمشاريع إنشاء الموقع الثابت لـ GitSite. استخدم هذه المهارة عندما يحتاج المستخدمون إلى إنشاء منشورات مدونة أو فصول كتاب أو كتب أو صفحات ثابتة. يقوم تلقائيًا بإنشاء بنية الدليل الصحيحة، وإنشاء ملفات قالب Markdown مع مادة أمامية كاملة، وتوفير نصوص Python لإدارة المحتوى. مناسب لمواقع الويب الثابتة التي تم إنشاؤها باستخدام GitSite CLI.
