Gestiona automáticamente el contenido de blogs, libros y páginas para proyectos de generación de sitios estáticos de GitSite. Utilice esta habilidad cuando los usuarios necesiten crear publicaciones de blog, capítulos de libros, libros o páginas estáticas. Crea automáticamente la estructura de directorio correcta, genera archivos de plantilla de Markdown con contenido completo y proporciona scripts de Python para la gestión de contenido. Adecuado para sitios web estáticos creados con GitSite CLI.
