Автоматически управляет содержимым блогов, книг и страниц для проектов генераторов статических сайтов GitSite. Используйте этот навык, когда пользователям нужно создавать публикации в блогах, главы книг, книги или статические страницы. Он автоматически создает правильную структуру каталогов, генерирует файлы шаблонов Markdown с полной информацией и предоставляет сценарии Python для управления контентом. Подходит для статических веб-сайтов, созданных с помощью GitSite CLI.
