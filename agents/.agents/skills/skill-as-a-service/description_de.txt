Führen Sie Programmieragenten (Claude Code, Gemini, Codex) mit Fähigkeiten in der Cloud über eine API aus.
Jede Aufgabe erhält ihre eigene isolierte VM. Verwenden Sie diese Option, wenn Sie Unteraufgaben erstellen müssen.
Delegieren Sie Arbeit an andere Agenten oder führen Sie Aufgaben mit bestimmten Fähigkeiten aus.
Erfordert die Umgebungsvariable REBYTE_API_KEY.
