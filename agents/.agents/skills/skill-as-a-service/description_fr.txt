Exécutez des agents de codage (Claude Code, Gemini, Codex) avec des compétences dans le cloud via API.
Chaque tâche obtient sa propre VM isolée. À utiliser lorsque vous devez générer des sous-tâches,
déléguez le travail à d’autres agents ou exécutez des tâches avec des compétences spécifiques.
Nécessite la variable d'environnement REBYTE_API_KEY.
