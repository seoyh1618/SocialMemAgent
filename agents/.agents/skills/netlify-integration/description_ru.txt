Развертывайте проекты Netlify и управляйте ими с помощью Next.js с бессерверными функциями, переменными среды и непрерывным развертыванием. Используйте этот навык при работе с развертываниями Netlify, настройке netlify.toml, управлении функциями Netlify, отладке веб-перехватчиков, настройке переменных среды или устранении проблем с развертыванием приложений Next.js в Netlify. Необходим для Twilio-Aldea и аналогичных бессерверных проектов SMS/телефонии. Версия 2.0 включает официальную документацию Netlify, производственные шаблоны от Twilio-Aldea (решения по тайм-ауту веб-перехватчика, фоновые функции) и примеры кода TypeScript.
