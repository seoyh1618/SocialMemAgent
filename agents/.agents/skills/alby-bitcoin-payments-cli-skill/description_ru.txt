Интерфейс командной строки для операций с биткойн-кошельком Lightning с использованием Nostr Wallet Connect (NIP-47). Используйте, когда пользователю необходимо отправлять/получать платежи, проверять баланс кошелька, создавать счета, конвертировать между фиатными и спутниковыми деньгами или работать с адресами Lightning.
