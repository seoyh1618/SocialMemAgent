Bootstrap Claude Skill-Synchronisierung in jedem Repository. Wird verwendet, wenn Benutzer die Synchronisierung von „.agents/skills“ mit „.claude/skills“ einrichten, „skills:sync:*“-Skripts generieren, package.json-Skripts/Postinstall-Hooks einfügen oder diesen Synchronisierungsworkflow in einem neuen oder vorhandenen Projekt initialisieren möchten.
