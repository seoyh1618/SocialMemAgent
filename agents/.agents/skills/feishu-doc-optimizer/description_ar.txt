Feishu Cloud Document تحسين المحتوى وتجميل التنسيق. استخدم هذه المهارة عندما يحتاج المستخدمون إلى تحسين تخطيط مستندات Feishu وبنيتها وتنسيقها وجمالياتها. يدعم: (1) قراءة محتوى مستند Feishu، (2) تحسين هيكل المستند وتسلسله الهرمي، (3) مسح محتوى المستند واستبداله، (4) تحرير المستندات عبر أتمتة المتصفح. الكلمات المحفزة: تحسين مستند Feishu، وتجميل المستند، وترتيب تنسيق المستند، وتخطيط المستند.
