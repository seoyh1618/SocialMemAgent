Esegue un'analisi completa della base di codice che copre architettura, qualità del codice, sicurezza, prestazioni, test e manutenibilità. Da utilizzare quando l'utente desidera verificare la qualità del codice, identificare il debito tecnico, individuare problemi di sicurezza, valutare la copertura dei test o ottenere un controllo dello stato della base del codice.
