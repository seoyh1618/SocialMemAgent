يقوم بإجراء تحليل شامل لقاعدة التعليمات البرمجية يغطي البنية وجودة التعليمات البرمجية والأمان والأداء والاختبار وقابلية الصيانة. يُستخدم عندما يريد المستخدم تدقيق جودة التعليمات البرمجية، أو تحديد الديون الفنية، أو العثور على مشكلات أمنية، أو تقييم تغطية الاختبار، أو الحصول على فحص سلامة قاعدة التعليمات البرمجية.
