Выполняет комплексный анализ кодовой базы, охватывающий архитектуру, качество кода, безопасность, производительность, тестирование и удобство сопровождения. Используйте, когда пользователь хочет проверить качество кода, выявить техническую задолженность, найти проблемы безопасности, оценить тестовое покрытие или проверить работоспособность кодовой базы.
