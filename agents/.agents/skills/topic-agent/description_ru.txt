Главный агент системы выбора тем. Координирует сбор горячих точек, создание тем и их проверку, поддерживая итерации до тех пор, пока не будет создана соответствующая тема. Триггеры: (1) «Начать сегодняшние темы» запускает полный рабочий процесс; (2) «Сегодняшние горячие точки искусственного интеллекта» собирают только горячие точки, без создания тем; (3) «У меня есть тема» входит в анализ одной темы; (4) «Рекомендовать хорошие темы» напрямую выводит рекомендации. Результаты сохраняются в хранилище тем Obsidian.
