Utilisez le serveur MCP de Robonet pour créer, backtester, optimiser et déployer des stratégies de trading. Fournit 24 outils spécialisés pour le trading sur le marché des cryptomonnaies et des prédictions : (1) Outils de données pour parcourir les stratégies, les symboles, les indicateurs, les sujets Allora et les résultats de backtest, (2) Outils d'IA pour générer des idées et du code de stratégie, optimiser les paramètres et améliorer les prédictions ML, (3) Outils de backtest pour tester les performances de la stratégie sur des données historiques, (4) Outils de marché de prédiction pour les stratégies de trading Polymarket, (5) Outils de déploiement pour le trading en direct sur Hyperliquid, (6) Outils de compte pour la gestion du crédit. À utiliser lorsque : l'élaboration de stratégies de trading, le backtest de stratégies, le déploiement de robots de trading, l'utilisation d'Hyperliquid ou de Polymarket, ou l'amélioration de stratégies avec les prédictions d'Allora Network ML.
