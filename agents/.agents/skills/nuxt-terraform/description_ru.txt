Scaffold Nuxt + инфраструктура AWS Terraform. Используйте при добавлении преобразователей GraphQL, функций Lambda, инициализации нового проекта с помощью AppSync, DynamoDB, Cognito, написании тестов Terraform или создании/проверке стиля кода Terraform. Срабатывает при: добавлении преобразователя Graphql, создании лямбды, scaffold terraform, инициализации terraform, добавлении преобразователя appsync, добавлении мутации, добавлении запроса, добавлении теста terraform, написании tftest, стиле terraform.
