البنية التحتية لـ Scaffold Nuxt + AWS Terraform. يُستخدم عند إضافة أدوات حل GraphQL، أو وظائف Lambda، أو تهيئة مشروع جديد باستخدام AppSync، أو DynamoDB، أو Cognito، أو كتابة اختبارات Terraform، أو إنشاء/مراجعة نمط كود Terraform. المشغلات: إضافة محلل graphql، وإنشاء lambda، وscaffold terraform، وinit terraform، وإضافة محلل appsync، وإضافة طفرة، وإضافة استعلام، وإضافة اختبار terraform، وكتابة tftest، ونمط terraform.
