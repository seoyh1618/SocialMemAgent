Échafaudage Nuxt + infrastructure AWS Terraform. À utiliser lors de l'ajout de résolveurs GraphQL, de fonctions Lambda, de l'initialisation d'un nouveau projet avec AppSync, DynamoDB, Cognito, de l'écriture de tests Terraform ou de la génération/révision du style de code Terraform. Déclencheurs sur : ajout d'un résolveur graphql, création de lambda, échafaudage terraform, init terraform, ajout d'un résolveur appsync, ajout d'une mutation, ajout d'une requête, ajout d'un test terraform, écriture de tftest, style terraform.
