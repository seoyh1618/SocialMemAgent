يقوم بتصحيح أخطاء تطبيقات Laravel من خلال تكامل Xdebug. يتم التنشيط عند تعيين نقاط التوقف، أو التنقل خلال التعليمات البرمجية، أو فحص المتغيرات، أو تحليل مخرجات dd()، أو تصحيح الأخطاء، أو وحدات التحكم، أو قوائم الانتظار، أو استعلامات Eloquent؛ أو عندما يذكر المستخدم تصحيح الأخطاء أو نقطة التوقف أو الدخول أو فحص المتغيرات أو Xdebug أو استكشاف الأخطاء وإصلاحها.
