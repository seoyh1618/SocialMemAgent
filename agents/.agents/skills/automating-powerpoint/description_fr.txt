Automatise Microsoft PowerPoint via JXA avec la découverte du dictionnaire AppleScript. À utiliser lorsque vous êtes invité à « automatiser les présentations PowerPoint », « créer des diapositives par programme », « scripter JXA PowerPoint » ou « exporter PowerPoint au format PDF ». Couvre les présentations, les diapositives, les formes, le texte, les tableaux, les énumérations d'exportation et l'interopérabilité avec Excel.
