يقوم بأتمتة Microsoft PowerPoint عبر JXA مع اكتشاف قاموس AppleScript. استخدمه عندما يُطلب منك "أتمتة عروض PowerPoint التقديمية" أو "إنشاء شرائح برمجيًا" أو "البرمجة النصية لـ JXA PowerPoint" أو "تصدير PowerPoint إلى PDF". يغطي العروض التقديمية والشرائح والأشكال والنصوص والجداول وتعدادات التصدير والتفاعل مع Excel.
