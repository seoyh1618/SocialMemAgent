Automatizza Microsoft PowerPoint tramite JXA con il rilevamento del dizionario AppleScript. Utilizzare quando viene richiesto di "automatizzare le presentazioni PowerPoint", "creare diapositive a livello di codice", "scripting JXA PowerPoint" o "esportare PowerPoint in PDF". Copre presentazioni, diapositive, forme, testo, tabelle, enumerazioni di esportazione e interoperabilità con Excel.
