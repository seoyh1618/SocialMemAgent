Biblioteca de validación de esquemas basada en TypeScript. Úselo para TODA la validación de entradas en proyectos TypeScript: entradas API, datos de formulario, variables de entorno, archivos de configuración. Compañero esencial para tRPC (obligatorio), React Hook Form y cualquier límite de datos. Elija Zod cuando necesite validación en tiempo de ejecución con inferencia automática de tipos de TypeScript.
