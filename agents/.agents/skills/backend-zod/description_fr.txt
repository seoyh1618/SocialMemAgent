Bibliothèque de validation de schéma TypeScript-first. À utiliser pour TOUTES les validations d'entrée dans les projets TypeScript : entrées API, données de formulaire, variables d'environnement, fichiers de configuration. Compagnon essentiel pour tRPC (obligatoire), React Hook Form et toute limite de données. Choisissez Zod lorsque vous avez besoin d'une validation d'exécution avec une inférence automatique de type TypeScript.
