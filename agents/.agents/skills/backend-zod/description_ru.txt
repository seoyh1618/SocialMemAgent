Библиотека проверки схемы TypeScript-first. Используйте для ВСЕХ входных данных в проектах TypeScript — входных данных API, данных форм, переменных среды, файлов конфигурации. Незаменимый помощник для tRPC (обязательно), React Hook Form и любых границ данных. Выбирайте Zod, если вам нужна проверка во время выполнения с автоматическим определением типа TypeScript.
