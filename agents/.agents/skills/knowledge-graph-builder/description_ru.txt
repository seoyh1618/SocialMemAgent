Реализует графы знаний для реляционных знаний, расширенных искусственным интеллектом. Охватывает проектирование онтологий, выбор графовой базы данных (Neo4j, Neptune, ArangoDB, TigerGraph), извлечение сущностей, гибридную графово-векторную архитектуру, шаблоны запросов и интеграцию искусственного интеллекта.
Используйте при реализации графов знаний, проектировании онтологий, извлечении сущностей и связей, выборе базы данных графов или построении гибридного графово-векторного поиска. Использование для графа знаний, проектирования онтологий, разрешения сущностей, RAG графа, обнаружения галлюцинаций. Для выбора архитектуры и управления используйте навыки менеджера базы знаний. Для конвейеров поиска документов используйте навык rag-implementer.
