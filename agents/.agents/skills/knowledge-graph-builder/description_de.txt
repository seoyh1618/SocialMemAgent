Implementiert Wissensgraphen für KI-gestütztes relationales Wissen. Behandelt Ontologiedesign, Auswahl von Diagrammdatenbanken (Neo4j, Neptune, ArangoDB, TigerGraph), Entitätsextraktion, hybride Diagramm-Vektor-Architektur, Abfragemuster und KI-Integration.
Verwenden Sie es, wenn Sie Wissensgraphen implementieren, Ontologien entwerfen, Entitäten und Beziehungen extrahieren, eine Graphdatenbank auswählen oder eine Hybrid-Graph-Vektor-Suche erstellen. Verwendung für Wissensgraphen, Ontologiedesign, Entitätsauflösung, Graph-RAG, Halluzinationserkennung. Nutzen Sie für die Auswahl und Governance der Architektur die Fähigkeit „Knowledge-Base-Manager“. Verwenden Sie für Pipelines zum Abrufen von Dokumenten die Fertigkeit „rag-implementer“.
