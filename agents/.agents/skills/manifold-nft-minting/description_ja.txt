Manifold のクライアント SDK を使用して NFT ミント エクスペリエンスを構築します。キャンペーンの設定、カスタム ミント Web サイト (React/Next.js)、ミント ボット (Node.js)、および既存のプロジェクトへの SDK の統合についてエージェントをガイドします。 Ethereum、Base、Optimism、Shape、Sepolia、ApeChain 全体で [`Edition`](https://docs.manifold.xyz/client-sdk/sdk/product/edition-product) および [`Blind Mint`](https://docs.manifold.xyz/client-sdk/sdk/product/blind-mint) 製品をサポートします。ミントページ、ミントボットの構築、Manifold NFT 製品の統合、またはユーザーによる Manifold キャンペーンのセットアップを支援するときに使用します。スマート コントラクトの展開、Manifold Studio 設定の管理、または非ミントのブロックチェーン操作には使用できません。
