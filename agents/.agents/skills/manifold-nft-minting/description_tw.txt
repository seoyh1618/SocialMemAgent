使用 Manifold 的 client-sdk 建構 NFT 鑄造體驗。指導代理商完成活動設定、自訂鑄幣網站 (React/Next.js)、鑄幣機器人 (Node.js) 以及將 SDK 整合到現有專案中。支援以太坊、Base、Optimism、Shape、Sepolia 和 ApeChain 的 [`Edition`](https://docs.manifold.xyz/client-sdk/sdk/product/edition-product) 和 [`Blind Mint`](https://docs.manifold.xyz/clipent-sd在建立鑄幣頁面、鑄幣機器人、整合 Manifold NFT 產品或幫助用戶設定 Manifold 活動時使用。不適用於部署智慧合約、管理 Manifold Studio 設定或非鑄幣區塊鏈操作。
