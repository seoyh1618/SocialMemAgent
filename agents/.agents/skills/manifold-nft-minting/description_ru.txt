Создайте возможности майнинга NFT с помощью client-sdk Manifold. Помогает агентам настроить кампанию, настроить веб-сайты для создания монет (React/Next.js), ботов для создания монет (Node.js) и интеграцию SDK в существующие проекты. Поддерживает продукты [`Edition`](https://docs.manifold.xyz/client-sdk/sdk/product/edition-product) и [`Blind Mint`](https://docs.manifold.xyz/client-sdk/sdk/product/blind-mint) в Ethereum, Base, Optimism, Shape, Sepolia и ApeChain. Используйте при создании страниц создания монет, ботов Mint, интеграции продуктов Manifold NFT или помощи пользователям в настройке кампаний Manifold. НЕ для развертывания смарт-контрактов, управления настройками Manifold Studio или операций блокчейна, не связанных с майнингом.
