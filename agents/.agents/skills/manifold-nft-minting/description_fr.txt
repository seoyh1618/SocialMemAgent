Créez des expériences de frappe NFT avec le SDK client de Manifold. Guide les agents dans la configuration des campagnes, les sites Web de création personnalisés (React/Next.js), les robots de création de campagnes (Node.js) et l'intégration du SDK dans les projets existants. Prend en charge les produits [`Edition`](https://docs.manifold.xyz/client-sdk/sdk/product/edition-product) et [`Blind Mint`](https://docs.manifold.xyz/client-sdk/sdk/product/blind-mint) sur Ethereum, Base, Optimism, Shape, Sepolia et ApeChain. À utiliser pour créer des pages de création, créer des robots, intégrer des produits Manifold NFT ou aider les utilisateurs à configurer des campagnes Manifold. PAS pour le déploiement de contrats intelligents, la gestion des paramètres de Manifold Studio ou les opérations de blockchain sans frappe.
