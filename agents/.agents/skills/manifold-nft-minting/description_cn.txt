使用 Manifold 的 client-sdk 构建 NFT 铸造体验。指导代理完成活动设置、自定义铸币网站 (React/Next.js)、铸币机器人 (Node.js) 以及将 SDK 集成到现有项目中。支持以太坊、Base、Optimism、Shape、Sepolia 和 ApeChain 的 [`Edition`](https://docs.manifold.xyz/client-sdk/sdk/product/edition-product) 和 [`Blind Mint`](https://docs.manifold.xyz/client-sdk/sdk/product/blind-mint) 产品。在构建铸币页面、铸币机器人、集成 Manifold NFT 产品或帮助用户设置 Manifold 活动时使用。不适用于部署智能合约、管理 Manifold Studio 设置或非铸币区块链操作。
