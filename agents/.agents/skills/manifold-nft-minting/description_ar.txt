قم ببناء تجارب تعدين NFT باستخدام Client-sdk الخاص بـ Manifold. يرشد الوكلاء من خلال إعداد الحملة، ومواقع التعدين المخصصة (React/Next.js)، وروبوتات التعدين (Node.js)، وتكامل SDK في المشاريع الحالية. يدعم منتجات [`Edition`](https://docs.manifold.xyz/client-sdk/sdk/product/edition-product) و[`Blind Mint`](https://docs.manifold.xyz/client-sdk/sdk/product/blind-mint) عبر Ethereum وBase وOptimism وShape وSepolia وApeChain. يُستخدم عند إنشاء صفحات التعدين، أو روبوتات التعدين، أو دمج منتجات Manifold NFT، أو مساعدة المستخدمين في إعداد حملات Manifold. ليس لنشر العقود الذكية، أو إدارة إعدادات Manifold Studio، أو عمليات blockchain غير المستحدثة.
