Présentation et conseils des principes fondamentaux des tests .NET. Déclenché lorsque les utilisateurs posent des questions de test générales telles que « comment écrire des tests .NET », « tests .NET pour les débutants », « de quels outils de test ai-je besoin », « meilleures pratiques de test » ou « apprendre les tests à partir de zéro ». Il recommande des combinaisons appropriées de sous-compétences basées sur des besoins spécifiques, couvrant 19 compétences fondamentales, notamment les bases des tests, les données de test, les assertions, les simulations et les scénarios spéciaux.
