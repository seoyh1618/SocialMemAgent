Écrivez du code Ruby et Rails dans le style 37signaux distinctif de DHH. Utilisez cette compétence lors de l'écriture de code Ruby, d'applications Rails, de la création de modèles, de contrôleurs ou de tout fichier Ruby. Se déclenche lors de la génération de code Ruby/Rails, des demandes de refactoring ou lorsque l'utilisateur mentionne le style DHH, 37signals, Basecamp, HEY, Fizzy ou Campfire.
