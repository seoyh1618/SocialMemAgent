Fähigkeit zum Erstellen von Tech-Spalten-Umrissen. Verwenden Sie diese Option, wenn der Benutzer Material (Artikel, Tweets usw.) bereitstellt und um eine Analyse und eine Gliederung bittet. Typische Anfragen: „Analysieren Sie diesen Artikel für mich“, „Erstellen Sie eine Gliederung“, „Schreiben Sie eine Kolumne.“ Erstellen Sie zwei bis drei unterschiedliche Gliederungsoptionen, starten Sie Schreibagenten parallel und speichern Sie Entwürfe, aus denen der Benutzer auswählen kann.
