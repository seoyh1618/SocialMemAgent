Documentation de référence sur le wrapper CLI AKShare et l'API. Utilisez cette compétence pour accéder aux données du marché financier chinois via la ligne de commande (actions, contrats à terme, fonds, obligations, forex, indicateurs macro) ou parcourez la documentation complète de l'API avec des tableaux de paramètres, des schémas de sortie et des exemples de code pour toutes les catégories de données prises en charge.
