غلاف AKShare CLI والوثائق المرجعية لواجهة برمجة التطبيقات (API). استخدم هذه المهارة للوصول إلى بيانات السوق المالية الصينية عبر سطر الأوامر (الأسهم والعقود الآجلة والصناديق والسندات والعملات الأجنبية والمؤشرات الكلية) أو تصفح وثائق واجهة برمجة التطبيقات الشاملة مع جداول المعلمات ومخططات الإخراج وأمثلة التعليمات البرمجية لجميع فئات البيانات المدعومة.
