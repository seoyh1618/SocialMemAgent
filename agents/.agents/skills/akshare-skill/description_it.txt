Wrapper della CLI AKShare e documentazione di riferimento dell'API. Utilizza questa competenza per accedere ai dati del mercato finanziario cinese tramite riga di comando (azioni, futures, fondi, obbligazioni, forex, macro indicatori) o sfogliare la documentazione API completa con tabelle di parametri, schemi di output ed esempi di codice per tutte le categorie di dati supportate.
