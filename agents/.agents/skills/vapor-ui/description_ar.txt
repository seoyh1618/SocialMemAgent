مكون نظام تصميم Vapor UI ودليل الأيقونات ومولد نماذج واجهة المستخدم ومحول تصميم Figma. يوفر كتالوج المكونات، والبحث عن الأيقونات، وأنماط الاستخدام، ووثائق الدعائم، ويحول تصميمات Figma إلى كود واجهة المستخدم البخارية الجاهز للإنتاج. يُستخدم عندما يسأل المستخدم عن "مكونات واجهة المستخدم البخارية"، و"أيقونات واجهة المستخدم البخارية"، و"아이콘 찾기"، و"واجهة المستخدم البخارية 사용법"، و"واجهة المستخدم البخارية 사용해서 시안 구현"، و"تحويل الشكل"، و"الشكل إلى رمز"، و"تنفيذ التصميم من الشكل"، ويوفر عنوان URL لـ Figma، أو يذكر مكونات محددة مثل "Button" و"Input" و"Modal".
