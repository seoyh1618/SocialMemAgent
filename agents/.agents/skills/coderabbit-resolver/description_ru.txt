Автоматизирует полный цикл проверки PR CodeRabbit — извлекает комментарии проверки, исправляет код, фиксирует, отправляет, отслеживает CI, разрешает потоки и повторяет до тех пор, пока все проверки не пройдут и потоки не будут решены, затем объединяет и очищает ветки. Поддерживает --bulk для последовательной обработки всех открытых PR. Используйте, когда вас просят исправить отзывы CodeRabbit, разрешить комментарии по связям с общественностью или выполнить полный рабочий процесс проверки-исправления-объединения.
