أتمتة دورة مراجعة CodeRabbit PR الكاملة - يستخرج تعليقات المراجعة، ويصلح التعليمات البرمجية، ويلزم، ويدفع، ويراقب CI، ويحل المواضيع، ويتكرر حتى تمر جميع عمليات التحقق ويتم حل المواضيع، ثم يدمج الفروع وينظفها. يدعم --bulk لمعالجة كافة العلاقات العامة المفتوحة بالتسلسل. يُستخدم عندما يُطلب منك إصلاح مراجعات CodeRabbit، أو حل تعليقات العلاقات العامة، أو التعامل مع سير العمل الكامل للمراجعة والإصلاح والدمج.
