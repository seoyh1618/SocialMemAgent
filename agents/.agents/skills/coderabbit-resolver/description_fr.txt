Automatise le cycle complet de révision de CodeRabbit PR : extrait les commentaires de révision, corrige le code, valide, pousse, surveille le CI, résout les threads et répète jusqu'à ce que toutes les vérifications réussissent et que les threads soient résolus, puis fusionne et nettoie les branches. Prend en charge --bulk pour traiter tous les PR ouverts de manière séquentielle. À utiliser lorsqu'on vous demande de corriger les avis CodeRabbit, de résoudre les commentaires PR ou de gérer le flux de travail complet d'avis, de correction et de fusion.
