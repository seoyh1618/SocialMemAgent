Biblioteca de transiciones de página para crear transiciones fluidas y fluidas entre las páginas del sitio web. Utilice esta habilidad al implementar transiciones de páginas, crear experiencias similares a SPA, agregar cambios de ruta animados o crear sitios web con una navegación fluida. Activa tareas que involucran Barba.js, transiciones de páginas, enrutamiento, administración de vistas, enlaces de transición, integración GSAP o navegación fluida de páginas. Funciona con gsap-scrolltrigger para animaciones de transición.
