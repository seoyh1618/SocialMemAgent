يمكنك الوصول إلى Gmail عبر CLI باستخدام 1Password OAuth. يُستخدم عندما يريد المستخدم قراءة رسائل البريد الإلكتروني، أو البحث في البريد الوارد، أو تصدير الرسائل، أو إنشاء مسودات، أو الإشارة إلى الوصول إلى Gmail. المشغلات - Gmail، والبريد الإلكتروني، وقراءة البريد الإلكتروني، وقائمة رسائل البريد الإلكتروني، والبحث في البريد الوارد، وتصدير رسائل البريد الإلكتروني، وإنشاء مسودة، ومسودة بريد إلكتروني، وإنشاء بريد إلكتروني.
