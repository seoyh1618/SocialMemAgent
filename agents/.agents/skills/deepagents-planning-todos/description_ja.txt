Deep Agent でのタスクの計画と分解には、write_todos ツールを効果的に使用します。ユーザーが、(1) write_todos を使用してタスク計画を実装する、(2) 複雑なタスクをサブタスクに分割する、(3) todo を通じてエージェントの進行状況を追跡する、(4) todo が完了しない理由をデバッグする、(5) さまざまなタスク タイプ (調査、コーディング、分析) の todo 構造を設計する、(6) todo ステータスのライフサイクルとベスト プラクティスを理解する、または (7) LangSmith トレースから todo の進行を視覚化する場合に使用します。
