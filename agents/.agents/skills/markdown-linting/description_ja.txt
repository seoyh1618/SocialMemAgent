markdownlint-cli2 を使用した包括的なマークダウン lint ガイダンス。マークダウン ファイルを実行、実行、確認、検証します。 lint エラー (MD0XX ルール) を修正します。 .markdownlint-cli2.jsonc を構成します (ルールと無視)。 VS Code 拡張機能と GitHub Actions ワークフローをセットアップします。 GitHub Flavored Markdown (GFM) や CommonMark などのマークダウン フレーバーをサポートします。マークダウン ファイルの操作、検証エラーの発生、マークダウン lint の構成、リンティング ワークフローのセットアップ、リンティングの問題のトラブルシューティング、マークダウン品質基準の確立、またはテーブル、タスク リスト、取り消し線のフレーバー固有のルールの構成を行う場合に使用します。
