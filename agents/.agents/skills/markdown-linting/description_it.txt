Guida completa all'linting del markdown utilizzando markdownlint-cli2. Esegui, esegui, controlla e convalida i file di markdown. Correggere gli errori di lanugine (regole MD0XX). Configura .markdownlint-cli2.jsonc (regole e ignora). Configura l'estensione VS Code e i flussi di lavoro di GitHub Actions. Supporta versioni di markdown tra cui GitHub Flavored Markdown (GFM) e CommonMark. Da utilizzare quando si lavora con file di markdown, si riscontrano errori di convalida, si configura markdownlint, si impostano flussi di lavoro di linting, si risolvono problemi di linting, si stabiliscono standard di qualità di markdown o si configurano regole specifiche per i gusti per tabelle, elenchi di attività e barrato.
