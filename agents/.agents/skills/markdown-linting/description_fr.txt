Conseils complets sur le peluchage des démarques à l’aide de markdownlint-cli2. Exécutez, exécutez, vérifiez et validez les fichiers de démarque. Correction des erreurs de peluchage (règles MD0XX). Configurez .markdownlint-cli2.jsonc (règles et ignores). Configurez l’extension VS Code et les workflows GitHub Actions. Prend en charge les versions de démarque, notamment GitHub Flavored Markdown (GFM) et CommonMark. À utiliser lorsque vous travaillez avec des fichiers de démarque, rencontrez des erreurs de validation, configurez markdownlint, configurez des flux de travail de peluchage, résolvez des problèmes de peluchage, établissez des normes de qualité de démarque ou configurez des règles spécifiques à la saveur pour les tableaux, les listes de tâches et le barré.
