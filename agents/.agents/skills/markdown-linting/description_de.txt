Umfassende Markdown-Linting-Anleitung mit markdownlint-cli2. Markdown-Dateien ausführen, ausführen, prüfen und validieren. Linting-Fehler beheben (MD0XX-Regeln). Konfigurieren Sie .markdownlint-cli2.jsonc (Regeln und Ignorierungen). Richten Sie die VS Code-Erweiterung und GitHub Actions-Workflows ein. Unterstützt Markdown-Varianten, einschließlich GitHub Flavored Markdown (GFM) und CommonMark. Verwenden Sie diese Option, wenn Sie mit Markdown-Dateien arbeiten, auf Validierungsfehler stoßen, Markdownlint konfigurieren, Linting-Workflows einrichten, Linting-Probleme beheben, Markdown-Qualitätsstandards festlegen oder geschmacksspezifische Regeln für Tabellen, Aufgabenlisten und Durchstreichungen konfigurieren.
