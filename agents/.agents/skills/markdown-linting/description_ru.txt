Комплексное руководство по анализу уценок с использованием markdownlint-cli2. Запускайте, выполняйте, проверяйте и подтверждайте файлы уценки. Исправьте ошибки линтинга (правила MD0XX). Настройте .markdownlint-cli2.jsonc (правила и игнор). Настройте расширение VS Code и рабочие процессы GitHub Actions. Поддерживает варианты уценки, включая GitHub Flavored Markdown (GFM) и CommonMark. Используйте при работе с файлами уценки, обнаружении ошибок проверки, настройке markdownlint, настройке рабочих процессов проверки, устранении проблем с проверкой, установлении стандартов качества уценки или настройке специальных правил для таблиц, списков задач и зачеркивания.
