Umfassendes Toolkit zum Validieren, Flusen und Testen von GitHub Actions-Workflowdateien, benutzerdefinierten lokalen Aktionen und öffentlichen Aktionen. Verwenden Sie diese Fähigkeit, wenn Sie mit GitHub Actions YAML-Dateien (.github/workflows/*.yml) arbeiten, die Workflow-Syntax validieren, die Workflow-Ausführung mit act testen oder Workflow-Probleme debuggen.
