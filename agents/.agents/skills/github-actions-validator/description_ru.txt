Комплексный набор инструментов для проверки, анализа и тестирования файлов рабочих процессов GitHub Actions, пользовательских локальных и общедоступных действий. Используйте этот навык при работе с файлами YAML действий GitHub (.github/workflows/*.yml), проверке синтаксиса рабочего процесса, тестировании выполнения рабочего процесса с помощью act или отладке проблем рабочего процесса.
