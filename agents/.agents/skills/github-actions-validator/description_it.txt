Kit di strumenti completo per la convalida, l'linting e il test di file di flusso di lavoro di GitHub Actions, azioni locali personalizzate e azioni pubbliche. Utilizza questa competenza quando lavori con file YAML di GitHub Actions (.github/workflows/*.yml), convalidi la sintassi del flusso di lavoro, testi l'esecuzione del flusso di lavoro con act o esegui il debug dei problemi del flusso di lavoro.
