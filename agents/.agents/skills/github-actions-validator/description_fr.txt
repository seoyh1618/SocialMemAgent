Boîte à outils complète pour valider, pelucher et tester les fichiers de workflow GitHub Actions, les actions locales personnalisées et les actions publiques. Utilisez cette compétence lorsque vous travaillez avec des fichiers YAML GitHub Actions (.github/workflows/*.yml), validez la syntaxe du flux de travail, testez l'exécution du flux de travail avec act ou déboguez les problèmes de flux de travail.
