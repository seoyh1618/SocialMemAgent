مجموعة أدوات شاملة للتحقق من صحة ملفات سير عمل GitHub Actions وفحصها واختبارها، والإجراءات المحلية المخصصة، والإجراءات العامة. استخدم هذه المهارة عند العمل مع ملفات GitHub Actions YAML (.github/workflows/*.yml)، أو التحقق من صحة بناء جملة سير العمل، أو اختبار تنفيذ سير العمل باستخدام الفعل، أو تصحيح مشكلات سير العمل.
