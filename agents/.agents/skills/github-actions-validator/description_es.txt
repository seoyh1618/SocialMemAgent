Conjunto de herramientas completo para validar, identificar y probar archivos de flujo de trabajo de GitHub Actions, acciones locales personalizadas y acciones públicas. Utilice esta habilidad cuando trabaje con archivos YAML de GitHub Actions (.github/workflows/*.yml), valide la sintaxis del flujo de trabajo, pruebe la ejecución del flujo de trabajo con act o depure problemas del flujo de trabajo.
