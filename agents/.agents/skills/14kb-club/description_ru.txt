Анализируйте веб-проекты для Клуба 14 КБ — проверяйте, чтобы критический путь рендеринга (HTML + CSS + JS + шрифты) умещался в пределах 14 КБ в сжатом виде gzip для мгновенной первой отрисовки через медленный запуск TCP. Гибридный режим обнаруживает анти-шаблоны в исходном коде и измеряет реальные размеры сжатых файлов на основе выходных данных сборки. Работает с любым фреймворком (Next.js, Vite, Astro, Nuxt, Angular, SvelteKit, Remix, Gatsby, CRA, обычный HTML, PHP). Используйте при проверке веб-производительности, аудите веса страниц, оптимизации первой отрисовки, проверке размера критического пути или когда пользователь просит сделать свой сайт быстрее, легче или проверить, умещается ли он в 14 КБ. Запускается в веб-проектах с файлами HTML/CSS/JS.
