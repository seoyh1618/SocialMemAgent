14KB 클럽을 위한 웹 프로젝트 분석 - TCP 느린 시작을 통해 즉각적인 첫 번째 페인트를 위해 중요한 렌더링 경로(HTML + CSS + JS + 글꼴)가 gzip으로 압축된 14KB 내에 맞는지 감사합니다. 하이브리드 모드는 소스 코드에서 안티 패턴을 감지하고 빌드 출력에서 ​​실제 gzip 크기를 측정합니다. 모든 프레임워크(Next.js, Vite, Astro, Nuxt, Angular, SvelteKit, Remix, Gatsby, CRA, 일반 HTML, PHP)와 작동합니다. 웹 성능 검토, 페이지 무게 감사, 첫 번째 페인트 최적화, 중요 경로 크기 확인 또는 사용자가 사이트를 더 빠르고 가볍게 만들거나 14KB에 맞는지 확인할 때 사용합니다.
