14KB クラブの Web プロジェクトを分析します。重要なレンダリング パス (HTML + CSS + JS + フォント) が、TCP スロー スタート経由でのインスタント ファースト ペイント用に gzip 圧縮された 14KB 以内に収まることを監査します。ハイブリッド モードは、ソース コード内のアンチパターンを検出し、ビルド出力から gzip 圧縮された実際のサイズを測定します。あらゆるフレームワーク (Next.js、Vite、Astro、Nuxt、Angular、SvelteKit、Remix、Gatsby、CRA、プレーン HTML、PHP) で動作します。 Web パフォーマンスのレビュー、ページの重さの監査、最初のペイントの最適化、クリティカル パス サイズのチェック、またはユーザーがサイトの高速化、軽量化、またはサイトが 14 KB に収まるかどうかの確認を要求する場合に使用します。 HTML/CSS/JS ファイルを使用した Web プロジェクトでトリガーします。
