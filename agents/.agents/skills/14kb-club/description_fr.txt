Analysez les projets Web pour le 14KB Club - vérifiez que le chemin de rendu critique (HTML + CSS + JS + polices) correspond à 14 Ko compressés pour une première peinture instantanée via un démarrage lent TCP. Le mode hybride détecte les anti-modèles dans le code source et mesure les tailles réelles compressées à partir de la sortie de build. Fonctionne avec n'importe quel framework (Next.js, Vite, Astro, Nuxt, Angular, SvelteKit, Remix, Gatsby, CRA, plain HTML, PHP). À utiliser lors de l'examen des performances Web, de l'audit du poids de la page, de l'optimisation de la première peinture, de la vérification de la taille du chemin critique ou lorsqu'un utilisateur demande de rendre son site plus rapide, plus léger ou de vérifier s'il tient dans 14 Ko. Déclencheurs sur des projets Web avec des fichiers HTML/CSS/JS.
