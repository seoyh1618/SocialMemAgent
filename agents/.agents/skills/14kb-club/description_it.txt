Analizza progetti web per il Club 14KB: controlla che il percorso di rendering critico (HTML + CSS + JS + font) rientri nei 14KB compressi con gzip per una prima verniciatura istantanea tramite avvio lento TCP. La modalità ibrida rileva gli anti-pattern nel codice sorgente e misura le dimensioni reali del gzip dall'output della build. Funziona con qualsiasi framework (Next.js, Vite, Astro, Nuxt, Angular, SvelteKit, Remix, Gatsby, CRA, HTML semplice, PHP). Da utilizzare durante la revisione delle prestazioni web, il controllo del peso della pagina, l'ottimizzazione della prima verniciatura, il controllo delle dimensioni del percorso critico o quando un utente chiede di rendere il proprio sito più veloce, più leggero o di verificare se rientra in 14 KB. Trigger su progetti web con file HTML/CSS/JS.
