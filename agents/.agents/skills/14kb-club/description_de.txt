Analysieren Sie Webprojekte für den 14-KB-Club – prüfen Sie, ob der kritische Rendering-Pfad (HTML + CSS + JS + Schriftarten) innerhalb von 14 KB passt, die für sofortiges erstes Malen über langsamen TCP-Start komprimiert wurden. Der Hybridmodus erkennt Anti-Patterns im Quellcode und misst echte gzip-Größen anhand der Build-Ausgabe. Funktioniert mit jedem Framework (Next.js, Vite, Astro, Nuxt, Angular, SvelteKit, Remix, Gatsby, CRA, einfaches HTML, PHP). Verwenden Sie diese Funktion, wenn Sie die Webleistung überprüfen, die Seitengewichtung prüfen, die erste Seite optimieren, die kritische Pfadgröße überprüfen oder wenn ein Benutzer seine Website schneller und leichter machen möchte oder prüfen möchte, ob sie in 14 KB passt. Löst auf Webprojekten mit HTML/CSS/JS-Dateien aus.
