Implementa modifiche alle funzionalità end-to-end nei progetti di impalcature go-sphere seguendo le convenzioni di layout delle sfere e il flusso di lavoro di generazione. Da utilizzare quando si aggiungono o si modificano API, contratti protobuf, schemi Ent, registrazione di associazione/mappa, logica del servizio o refactoring multilivello che devono rimanere prioritari rispetto al protocollo ed evitare modifiche manuali ai file generati. Questa competenza è NECESSARIA per qualsiasi attività che coinvolga file proto go-sphere, schemi Ent, implementazioni di servizi o comandi di generazione (make gen/proto, make gen/docs, make gen/wire).
