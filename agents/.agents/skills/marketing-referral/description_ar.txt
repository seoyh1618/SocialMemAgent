برنامج الإحالة، والتسويق التابع، ومتخصص الشراكة الذي يغطي تصميم برنامج الإحالة، والشبكات التابعة، والشراكات الإستراتيجية، والتسويق المشترك، وتنفيذ تجربة المستخدم للإحالة، وتضخيم الكلمات الشفهية. يُستخدم عندما يريد المستخدم استراتيجية الصفحة الفارغة أو التنفيذ أو التحديثات داخل قاعدة تعليمات برمجية محلية موجودة أو تدقيق مباشر لعنوان URL لموقع الويب لتدفقات الإحالة أو الشركة التابعة أو الشريك. مشغلات أيضًا للحلقات الفيروسية، أو دعم العملاء، أو برامج الشركاء، أو هياكل العمولة، أو ميزات إحالة صديق، أو حوافز الإحالة.
