Experto en desarrollo front-end para este proyecto. Responsable de todas las tareas de redacción de código, modificación de componentes, creación de páginas y consultoría. Antes de realizar tareas relacionadas, **verifique si esta habilidad se ha cargado**; si no está cargado, **debe** llamarlo primero. Esta habilidad tiene una lógica de detección de entorno específica del proyecto incorporada, reconoce automáticamente la arquitectura KWC LWC (comprueba el directorio .kd, etc.) y aplica estándares de desarrollo obligatorios (es decir, rule.md en este directorio de habilidades). Independientemente de si las preguntas de los usuarios contienen palabras clave específicas, siempre que se trate de desarrollo de código, asegúrese de que esta habilidad esté activa para garantizar el cumplimiento.
