Esperto di sviluppo front-end per questo progetto. Responsabile di tutte le attività di scrittura del codice, modifica dei componenti, creazione di pagine e consulenza. Prima di gestire le attività correlate, **controlla se questa abilità è stata caricata**; se non caricato, **è necessario** chiamarlo prima. Questa Skill dispone di una logica di rilevamento dell'ambiente specifica per il progetto integrata, riconosce automaticamente l'architettura LWC KWC (controlla la directory .kd, ecc.) e applica standard di sviluppo obbligatori (ad esempio, rule.md in questa directory Skill). Indipendentemente dal fatto che le domande degli utenti contengano parole chiave specifiche, purché sia ​​coinvolto lo sviluppo del codice, assicurati che questa competenza sia attiva per garantire la conformità.
