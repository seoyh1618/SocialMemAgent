Inter-instance communication for OpenClaw. Используйте этот навык, когда требуются сообщения, синхронизация данных или удаленное выполнение команд между несколькими экземплярами OpenClaw. Охватывает обмен сообщениями между агентами, удаленное выполнение nodes.run, связь на уровне файлов и другие методы.
