Используйте, когда вам нужно добавить или оценить зависимости Maven, которые улучшают качество кода — включая аннотации нулевых значений (JSpecify), статический анализ (Error Prone + NullAway), функциональное программирование (VAVR) или тестирование архитектуры (ArchUnit) — и вам нужен консультативный подход, основанный на вопросах, который добавляет только то, что вам действительно нужно.
