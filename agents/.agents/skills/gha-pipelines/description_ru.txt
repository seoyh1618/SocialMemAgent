Создавайте и изменяйте рабочие процессы CI/CD GitHub Actions для репозитория homelab.
Охватывает конвейеры проверки, продвижение артефактов OCI и тестирование инфраструктуры.

Используйте при: (1) создании новых рабочих процессов GitHub Actions, (2) изменении существующих конвейеров CI/CD,
(3) Добавление этапов проверки или тестирования, (4) Отладка сбоев рабочего процесса,
(5) Понимание процесса продвижения OCI.

Триггеры: «действия github», «рабочий процесс», «ci/cd», «конвейер», «gha»,
«артефакт сборки», «рабочий процесс проверки», «конвейер ci»
