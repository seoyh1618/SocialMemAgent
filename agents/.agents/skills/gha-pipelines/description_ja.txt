Homelab リポジトリの GitHub Actions CI/CD ワークフローを作成および変更します。
検証パイプライン、OCI アーティファクトのプロモーション、インフラストラクチャのテストをカバーします。

次の場合に使用します: (1) 新しい GitHub Actions ワークフローを作成する、(2) 既存の CI/CD パイプラインを変更する、
(3) 検証またはテスト段階の追加、(4) ワークフローの失敗のデバッグ、
(5) OCI 推進パイプラインを理解する。
