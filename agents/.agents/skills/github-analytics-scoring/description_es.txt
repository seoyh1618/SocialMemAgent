Fórmulas de puntuación y marcos analíticos para agentes de flujo de trabajo de GitHub. Cubre la puntuación del estado del repositorio (0-100, grados A-F), puntuación de prioridad para problemas/PR/discusiones, niveles de confianza para los hallazgos analíticos, seguimiento delta (fijo/nuevo/persistente/regresado), métricas de velocidad, métricas de contribuyentes, detección de cuellos de botella y clasificación de tendencias. Utilícelo para calcular puntuaciones, realizar un seguimiento del progreso de la solución, crear paneles de control priorizados o detectar cuellos de botella en el flujo de trabajo.
