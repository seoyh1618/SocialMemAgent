Формулы оценки и аналитические платформы для агентов рабочих процессов GitHub. Охватывает оценку работоспособности репозитория (0–100, оценки AF), оценку приоритетов для проблем/запросов/обсуждений, уровни достоверности результатов аналитики, отслеживание дельты (фиксированное/новое/постоянное/регрессивное), метрики скорости, метрики участников, обнаружение узких мест и классификацию тенденций. Используйте при вычислении оценок, отслеживании прогресса исправления, построении информационных панелей с приоритетами или обнаружении узких мест в рабочем процессе.
