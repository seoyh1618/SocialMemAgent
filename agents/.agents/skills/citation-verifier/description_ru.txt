Создавайте и проверяйте записи BibTeX на основе бумажных заметок, записывая `citations/ref.bib` и `citations/verified.jsonl`.  
**Триггер**: цитирование, BibTeX, ref.bib, Verified.jsonl, ссылки, 引用, 参考文献.  
**Используйте, когда**: `papers/paper_notes.jsonl` уже существует, и вам нужны отслеживаемые цитаты (каждая с записью проверки URL/даты/названия) для прозы/LaTeX.  
**Пропустить, если**: еще нет бумажных заметок (или текущий результат не нуждается в цитатах/ссылках).  
**Сеть**: для автоматической проверки обычно требуется сеть; если нет в сети, сначала запишите, затем отметка требует ручной проверки.  
**Guardrail**: каждая запись BibTeX должна иметь соответствующую запись `citations/verified.jsonl`; проза может использовать только ключи цитирования, уже находящиеся в `citations/ref.bib`.
