Используйте, когда пользователь хочет использовать Google Gemini для анализа, больших файлов или баз кода, выполнения в песочнице или мозгового штурма. Использует безголовые сценарии Gemini CLI (без MCP). Триггеры: «использовать Gemini», «анализировать с Gemini», «большой файл», «песочница», «мозговой штурм с Gemini».
