Erhalten Sie Hilfe, Onboarding-Anleitungen und melden Sie Probleme. Verwenden Sie diese Funktion, wenn Sie oder der Benutzer Hilfe mit der Wallet benötigen, Fragen zur Funktionsweise haben, eine exemplarische Vorgehensweise wünschen, zum ersten Mal damit beginnen oder einen Fehler oder ein Problem melden müssen. Behandelt „Wie verwende ich das?“, „Hilf mir den Einstieg“, „Ich bin neu“, „Etwas ist kaputt“, „Fehler melden“ und „Welche Version ist das?“.
