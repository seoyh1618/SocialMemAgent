Zig-Build-Systemkompetenz für Zig-Projekte mit mehreren Dateien. Verwenden Sie es beim Schreiben oder Debuggen von build.zig-Dateien, beim Hinzufügen ausführbarer Dateien oder Bibliotheken, beim Integrieren von C-Quelldateien, beim Konfigurieren von Build-Time-Optionen, beim Ausführen von Tests mit zig build test oder beim Verwalten von build.zig.zon-Paketmanifesten. Wird bei Abfragen zu build.zig, b.addExecutable, addCSourceFiles, b.option, zig build, build.zig.zon oder Zig-Paketverwaltung aktiviert.
