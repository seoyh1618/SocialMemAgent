Compétence système de construction Zig pour les projets Zig multi-fichiers. À utiliser lors de l'écriture ou du débogage de fichiers build.zig, de l'ajout d'exécutables ou de bibliothèques, de l'intégration de fichiers source C, de la configuration des options de build, de l'exécution de tests avec zig build test ou de la gestion des manifestes du package build.zig.zon. S'active sur les requêtes concernant build.zig, b.addExecutable, addCSourceFiles, b.option, zig build, build.zig.zon ou la gestion des packages Zig.
