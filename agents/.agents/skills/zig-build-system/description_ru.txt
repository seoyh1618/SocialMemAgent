Навыки системы сборки Zig для многофайловых проектов Zig. Используйте при написании или отладке файлов build.zig, добавлении исполняемых файлов или библиотек, интеграции исходных файлов C, настройке параметров времени сборки, запуске тестов с помощью теста сборки zig или управлении манифестами пакетов build.zig.zon. Активируется по запросам о build.zig, b.addExecutable, addCSourceFiles, b.option, zig build, build.zig.zon или управлении пакетами Zig.
