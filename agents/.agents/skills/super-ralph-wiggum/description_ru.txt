Super Ralph Wiggum — автономные циклы итераций с шаблонами, поддержкой PRD, отслеживанием прогресса и тестированием браузера. Этот навык следует использовать при запуске Claude Code в автономных циклах для улучшения тестового покрытия, разработки функций на основе PRD, создания документации, создания набора данных, исправления ошибок, очистки кода или миграции инфраструктуры. Сочетает в себе механизм внутрисессионного цикла плагина со специализированными шаблонами и лучшими практиками Джеффри Хантли, Райана Карсона и AI Hero.
