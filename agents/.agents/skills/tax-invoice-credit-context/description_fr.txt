Contexte d'arrière-plan pour les règles de crédit de taxe en amont du système de facturation (仕入税額控除) dans le plug-in de déclaration de revenus Shinkoku. Contient les conditions d'éligibilité, les 6 exceptions permanentes (帳簿のみ保存の恒久特例), les mesures transitoires pour les achats auprès d'entreprises exonérées d'impôt et les exigences de stockage. Cette compétence n'est pas invocable par l'utilisateur — Claude la charge automatiquement lorsqu'il répond aux questions sur le crédit de taxe sur les intrants dans le système de facturation.
