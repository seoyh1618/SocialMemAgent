البنية التحتية كتعليمات برمجية باستخدام OpenTofu (شوكة Terraform مفتوحة المصدر) وPulumi. يغطي بناء جملة OpenTofu HCL، والموفرين، والموارد، ومصادر البيانات، والوحدات النمطية، وإدارة الحالة مع الواجهات الخلفية البعيدة، ومساحات العمل، واستيراد البنية التحتية الحالية، وتخطيط/تطبيق سير العمل، وإدارة المتغيرات، وقيم المخرجات، وموفري الخدمة، وتشفير الحالة (حصريًا لـ OpenTofu). يتضمن Pulumi TypeScript/Python SDKs وإدارة المكدس وموارد المكونات والتكوين/الأسرار والواجهات الخلفية للحالة والسياسة كرمز وواجهة برمجة التطبيقات للأتمتة. الأنماط الشائعة لإعدادات البيئات المتعددة، وتكوين الوحدة، وتكامل CI/CD، واكتشاف الانجراف، والإدارة السرية.

يُستخدم عند كتابة أو مراجعة تكوينات HCL، أو إدارة حالة البنية التحتية السحابية، أو الترحيل من Terraform إلى OpenTofu، أو إنشاء برامج Pulumi في TypeScript أو Python، أو إعداد خطوط أنابيب IaC متعددة البيئات، أو تنفيذ تشفير الحالة.
