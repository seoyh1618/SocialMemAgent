Infraestructura como código con OpenTofu (bifurcación Terraform de código abierto) y Pulumi. Cubre la sintaxis de OpenTofu HCL, proveedores, recursos, fuentes de datos, módulos, administración de estado con backends remotos, espacios de trabajo, importación de infraestructura existente, planificación/aplicación de flujo de trabajo, administración de variables, valores de salida, aprovisionadores y cifrado de estado (exclusivo de OpenTofu). Incluye SDK de Pulumi TypeScript/Python, administración de pila, recursos de componentes, configuración/secretos, backends de estado, políticas como código y API de automatización. Patrones comunes para configuraciones multientorno, composición de módulos, integración CI/CD, detección de deriva y gestión de secretos.

Úselo al escribir o revisar configuraciones de HCL, administrar el estado de la infraestructura de la nube, migrar de Terraform a OpenTofu, crear programas Pulumi en TypeScript o Python, configurar canalizaciones de IaC multientorno o implementar cifrado de estado.
