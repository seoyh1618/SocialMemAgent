Infrastruttura come codice con OpenTofu (fork Terraform open source) e Pulumi. Copre la sintassi HCL di OpenTofu, fornitori, risorse, origini dati, moduli, gestione dello stato con backend remoti, aree di lavoro, importazione di infrastrutture esistenti, flusso di lavoro di pianificazione/applicazione, gestione delle variabili, valori di output, provisioner e crittografia dello stato (esclusiva OpenTofu). Include SDK Pulumi TypeScript/Python, gestione dello stack, risorse dei componenti, configurazione/segreti, backend di stato, policy as code e API di automazione. Modelli comuni per configurazioni multiambiente, composizione dei moduli, integrazione CI/CD, rilevamento della deriva e gestione dei segreti.

Da utilizzare durante la scrittura o la revisione delle configurazioni HCL, la gestione dello stato dell'infrastruttura cloud, la migrazione da Terraform a OpenTofu, la creazione di programmi Pulumi in TypeScript o Python, la configurazione di pipeline IaC multiambiente o l'implementazione della crittografia dello stato.
