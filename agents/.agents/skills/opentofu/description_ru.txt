Инфраструктура как код с OpenTofu (вилка Terraform с открытым исходным кодом) и Pulumi. Охватывает синтаксис OpenTofu HCL, поставщиков, ресурсы, источники данных, модули, управление состоянием с помощью удаленных серверов, рабочие области, импорт существующей инфраструктуры, планирование/применение рабочего процесса, управление переменными, выходные значения, поставщики и шифрование состояния (эксклюзивно для OpenTofu). Включает SDK Pulumi TypeScript/Python, управление стеком, ресурсы компонентов, конфигурацию/секреты, серверные части состояния, политику как код и API автоматизации. Общие шаблоны для настройки нескольких сред, состава модулей, интеграции CI/CD, обнаружения отклонений и управления секретами.

Используйте при написании или проверке конфигураций HCL, управлении состоянием облачной инфраструктуры, переходе с Terraform на OpenTofu, создании программ Pulumi на TypeScript или Python, настройке многосредовых конвейеров IaC или реализации шифрования состояния.
