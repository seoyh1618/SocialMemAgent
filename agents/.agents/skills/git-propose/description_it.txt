Esamina le modifiche complete all'albero di lavoro Git e proponi uno o più messaggi di commit sicuri e rivedibili, oltre all'ordine dei commit. Da utilizzare quando l'utente richiede "git propose", chiede come suddividere le modifiche correnti in commit o desidera messaggi di commit convenzionale da modifiche organizzate, non organizzate e non tracciate.
