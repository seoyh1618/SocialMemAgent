قم بمراجعة التغييرات الكاملة في شجرة عمل Git واقترح واحدة أو أكثر من رسائل الالتزام الآمنة والقابلة للمراجعة بالإضافة إلى طلب الالتزام. يُستخدم عندما يسأل المستخدم عن "اقتراح git"، أو يسأل عن كيفية تقسيم التغييرات الحالية إلى التزامات، أو يريد رسائل الالتزام التقليدية من التغييرات المرحلية وغير المرحلية وغير المتعقبة.
