Nginx または OpenResty/Lua の開発タスクを支援する場合は、このスキルを使用してください。トリガーには、Nginx 構成 (サーバー ブロック、ロケーション ブロック、アップストリーム、プロキシ パス、リライト、マップ、try_files) の書き込みまたはデバッグ、OpenResty/Lua モジュール開発 (コンテンツ ハンドラー、アクセス制御、ヘッダー操作、共有辞書、cosocket の使用)、Nginx パフォーマンス チューニング (バッファ サイズ設定、キープアライブ チューニング、キャッシュ、接続プーリング、ベンチマーク) が含まれます。ユーザーが Web サーバー/リバース プロキシ/CDN のコンテキストで .conf ファイル、nginx.conf、lua モジュール、またはディレクティブ レベルの Nginx の質問に言及する場合にも使用します。 Apache、Caddy、HAProxy、または Nginx に関係のない一般的な Web サーバーに関する質問には使用しないでください。
