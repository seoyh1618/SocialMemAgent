Usa questa competenza quando aiuti con le attività di sviluppo di Nginx o OpenResty/Lua. I trigger includono la scrittura o il debug della configurazione di Nginx (blocchi server, blocchi di posizione, upstream, proxy_pass, riscritture, mappe, try_files), sviluppo di moduli OpenResty/Lua (gestori di contenuti, controllo degli accessi, manipolazione di intestazioni, dizionari condivisi, utilizzo di cosocket) e ottimizzazione delle prestazioni di Nginx (dimensionamento del buffer, ottimizzazione keepalive, caching, pooling di connessioni, benchmarking). Da utilizzare anche quando l'utente menziona file .conf, nginx.conf, moduli lua nel contesto di un server Web/proxy inverso/CDN o qualsiasi domanda Nginx a livello di direttiva. NON utilizzare per domande su Apache, Caddy, HAProxy o server Web generali non correlate a Nginx.
