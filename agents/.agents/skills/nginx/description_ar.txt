استخدم هذه المهارة عند المساعدة في مهام تطوير Nginx أو OpenResty/Lua. تتضمن المشغلات كتابة أو تصحيح تكوين Nginx (كتل الخادم، وكتل الموقع، والمنبع، وproxy_pass، وإعادة الكتابة، والخرائط، وtry_files)، وتطوير وحدة OpenResty/Lua (معالجات المحتوى، والتحكم في الوصول، ومعالجة الرأس، والقواميس المشتركة، واستخدام cosocket)، وضبط أداء Nginx (تحجيم المخزن المؤقت، وضبط البقاء على قيد الحياة، والتخزين المؤقت، وتجميع الاتصال، وقياس الأداء). استخدمه أيضًا عندما يذكر المستخدم ملفات .conf، أو nginx.conf، أو وحدات lua في سياق خادم الويب/الوكيل العكسي/CDN، أو أي أسئلة عن Nginx على مستوى التوجيه. لا تستخدمه مع أسئلة Apache أو Caddy أو HAProxy أو أسئلة خادم الويب العامة التي لا علاقة لها بـ Nginx.
