Utilisez cette compétence lorsque vous participez aux tâches de développement Nginx ou OpenResty/Lua. Les déclencheurs incluent l'écriture ou le débogage de la configuration Nginx (blocs de serveur, blocs d'emplacement, amont, proxy_pass, réécritures, cartes, try_files), le développement de modules OpenResty/Lua (gestionnaires de contenu, contrôle d'accès, manipulation d'en-tête, dictionnaires partagés, utilisation de cosocket) et le réglage des performances de Nginx (dimensionnement du tampon, réglage keepalive, mise en cache, regroupement de connexions, analyse comparative). À utiliser également lorsque l'utilisateur mentionne les fichiers .conf, nginx.conf, les modules lua dans le contexte d'un serveur Web/proxy inverse/CDN, ou toute question Nginx au niveau de la directive. Ne PAS utiliser pour les questions Apache, Caddy, HAProxy ou générales sur le serveur Web sans rapport avec Nginx.
