在協助完成 Nginx 或 OpenResty/Lua 開發任務時使用此技能。觸發器包括編寫或調試 Nginx 配置（服務器塊、位置塊、上游、proxy_pass、重寫、映射、try_files）、OpenResty/Lua 模塊開發（內容處理程序、訪問控制、標頭操作、共享字典、cosocket 使用）以及 Nginx 性能調整（緩衝區大小、保活調整、緩存、連接池、基準測試）。當使用者在 Web 伺服器/反向代理/CDN 上下文中提及 .conf 檔案、nginx.conf、lua 模組或任何指令層級 Nginx 問題時也可使用。請勿用於 Apache、Caddy、HAProxy 或與 Nginx 無關的一般 Web 伺服器問題。
