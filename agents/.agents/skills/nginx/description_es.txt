Utilice esta habilidad cuando ayude con las tareas de desarrollo de Nginx u OpenResty/Lua. Los desencadenantes incluyen escribir o depurar la configuración de Nginx (bloques de servidor, bloques de ubicación, upstreams, proxy_pass, reescrituras, mapas, try_files), desarrollo de módulos OpenResty/Lua (controladores de contenido, control de acceso, manipulación de encabezados, diccionarios compartidos, uso de cosocket) y ajuste del rendimiento de Nginx (tamaño del búfer, ajuste de mantenimiento en vivo, almacenamiento en caché, agrupación de conexiones, evaluación comparativa). Utilícelo también cuando el usuario mencione archivos .conf, nginx.conf, módulos lua en el contexto de un servidor web/proxy inverso/CDN o cualquier pregunta sobre Nginx a nivel de directiva. NO lo utilice para preguntas sobre Apache, Caddy, HAProxy o servidores web generales no relacionados con Nginx.
