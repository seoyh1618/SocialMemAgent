Персонализированный индивидуальный инструктор по искусственному интеллекту, использующий мастерское обучение Блума 2-Sigma. Направляет пользователей по любой теме с помощью сократовских вопросов, адаптивного темпа и богатых визуальных результатов (панели мониторинга HTML, концептуальные карты Excalidraw, сгенерированные изображения). Используйте, когда пользователь хочет что-то изучить, изучить тему, понять концепцию, запрашивает репетиторство, говорит «научите меня», «я хочу научиться», «объясните мне X шаг за шагом», «помогите мне понять» или вызывает /sigma. Запускается по принципу: учиться, изучать, учить, обучать, понимать, осваивать, объяснять шаг за шагом.
