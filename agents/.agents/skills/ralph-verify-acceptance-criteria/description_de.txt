Überprüfen Sie Ralph User Story-Implementierungen anhand der Akzeptanzkriterien in prd.json. Verwenden Sie diese Fähigkeit immer dann, wenn Sie eine abgeschlossene User Story verifizieren, Ralphs Implementierung testen, prüfen müssen, ob die Akzeptanzkriterien erfüllt sind, Funktionen des Admin-Dashboards validieren oder den PRD-Abschlussstatus prüfen müssen. Wird ausgelöst bei: Story verifizieren, Implementierung prüfen, Ralph verifizieren, Akzeptanzkriterien testen, PRD validieren, User Story prüfen, US-XXX verifizieren, Ralph-Fortschritt prüfen, wird diese Story bestanden.
