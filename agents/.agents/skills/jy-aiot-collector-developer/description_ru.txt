На основе платформы плагина Hongyan (jy-aiot) для шахт сгенерируйте код модуля сбора гибридного режима (конфигурация + фрагменты кода) в соответствии с документом протокола сбора данных в формате таблицы Markdown. Используя этот навык, вы можете быстро реализовать модули сбора данных для таких протоколов связи, как HTTP-сервер/клиент, FTP, база данных, MQTT, Kafka, упрощая разработку пользовательских компонентов сбора протоколов.
