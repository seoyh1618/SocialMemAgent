استنادًا إلى إطار عمل البرنامج الإضافي Hongyan (jy-aiot) للمناجم، قم بإنشاء كود وحدة جمع الوضع المختلط (التكوين + مقتطفات الكود) وفقًا لوثيقة بروتوكول جمع البيانات بتنسيق جدول Markdown. باستخدام هذه المهارة، يمكنك تنفيذ وحدات التجميع لبروتوكولات الاتصال بسرعة مثل خادم/عميل HTTP وFTP وقاعدة البيانات وMQTT وKafka، مما يبسط تعقيد تطوير مكونات مجموعة البروتوكول المخصص.
