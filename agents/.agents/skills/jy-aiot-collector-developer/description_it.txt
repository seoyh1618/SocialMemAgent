Basato sul framework del plugin Hongyan (jy-aiot) per le miniere, genera il codice del modulo di raccolta in modalità ibrida (configurazione + snippet di codice) secondo il documento del protocollo di raccolta dati in formato tabella Markdown. Utilizzando questa competenza, puoi implementare rapidamente moduli di raccolta per protocolli di comunicazione come server/client HTTP, FTP, database, MQTT, Kafka, semplificando la complessità di sviluppo dei componenti di raccolta di protocolli personalizzati.
