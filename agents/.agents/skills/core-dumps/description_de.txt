Kernfähigkeit zur Dump-Analyse für die Triage von Produktionsabstürzen. Verwenden Sie diese Option, wenn Sie Kerndateien in GDB oder LLDB laden, die Kernspeicherauszugsgenerierung unter Linux/macOS aktivieren, Symbole mit debuginfo oder debuginfod zuordnen oder Backtraces von Abstürzen extrahieren, ohne das Programm erneut auszuführen. Wird bei Abfragen zu Kerndateien, ulimit, coredumpctl, debuginfod, Absturztriage oder der Analyse von Segfaults aus Produktionsbinärdateien aktiviert.
