Capacità di analisi del core dump per il triage dei crash di produzione. Da utilizzare quando si caricano file core in GDB o LLDB, si abilita la generazione di dump core su Linux/macOS, si mappano simboli con debuginfo o debuginfod o si estraggono backtrace da arresti anomali senza rieseguire il programma. Si attiva su query su file core, ulimit, coredumpctl, debuginfod, crash triage o analisi di segfault dai binari di produzione.
