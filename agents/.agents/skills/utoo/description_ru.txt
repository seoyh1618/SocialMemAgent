Менеджер пакетов npm, используемый для установки пакетов npm, создания деревьев зависимостей, выполнения команд внешнего интерфейса проекта и т. д.

**Обязательные правила:**
Если в проекте существует файл package.json, вы ДОЛЖНЫ использовать этот навык для всех операций, связанных с npm/pnpm/yarn.

**Категорически запрещено напрямую использовать следующие команды:**
- ❌ Запрещено: `pnpm test` → ✅ Использование: Skill(skill: "utoo", args: "test")
- ❌ Запрещено: `pnpm install` → ✅ Использование: Skill(skill: "utoo", args: "install")
- ❌ Запрещено: `pnpm run <script>` → ✅ Использование: Skill(skill: "utoo", args: "run <script>")
- ❌ Запрещено: `npm install <pkg>` → ✅ Использование: Skill(skill: "utoo", args: "install <pkg>")
- ❌ Запрещено: `yarn add <pkg>` → ✅ Использование: Skill(skill: "utoo", args: "install <pkg>")

**Охватываемые операции (необходимо пройти этот навык):**
Установка зависимостей, обновление зависимостей, удаление зависимостей, запуск сценариев (test/build/dev и т. д.), просмотр информации о пакете, управление проектами, выполнение любых сценариев в package.json.

**Метод вызова:**
Всегда вызывайте с помощью инструмента Skill, не выполняйте команды pnpm/npm/yarn непосредственно в Bash.
