npm 패키지 설치, 종속성 트리 생성, 프런트엔드 프로젝트 명령 실행 등에 사용되는 npm 패키지 관리자입니다.

**필수 규칙:**
프로젝트에 package.json 파일이 존재하는 경우 모든 npm/pnpm/yarn 관련 작업에 이 스킬을 사용해야 합니다.

**다음 명령을 직접 사용하는 것은 엄격히 금지됩니다.**
- ❌ 금지: `pnpm test` → ✅ 사용: Skill(skill: "utoo", args: "test")
- ❌ 금지: `pnpm install` → ✅ 사용: Skill(skill: "utoo", args: "install")
- ❌ 금지: `pnpm run <script>` → ✅ 사용: Skill(skill: "utoo", args: "run <script>")
- ❌ 금지됨: `npm install <pkg>` → ✅ 사용: Skill(skill: "utoo", args: "install <pkg>")
- ❌ 금지: `yarn add <pkg>` → ✅ 사용: Skill(skill: "utoo", args: "install <pkg>")

**해당 작업(이 기술을 거쳐야 함):**
종속성 설치, 종속성 업데이트, 종속성 제거, 스크립트 실행(테스트/빌드/개발 등), 패키지 정보 보기, 프로젝트 관리, package.json의 스크립트 실행.

**호출 방법:**
항상 Skill 도구를 통해 호출하고, Bash에서 직접 pnpm/npm/yarn 명령을 실행하지 마세요.
