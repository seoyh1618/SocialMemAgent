npm パッケージのインストール、依存関係ツリーの生成、フロントエンド プロジェクト コマンドの実行などに使用される npm パッケージ マネージャー。

**必須ルール:**
package.json ファイルがプロジェクトに存在する場合、npm/pnpm/yarn 関連のすべての操作にこのスキルを使用する必要があります。

**次のコマンドを直接使用することは固く禁じられています:**
- ❌ 禁止: `pnpm test` → ✅ 使用: Skill(スキル: "utoo"、引数: "test")
- ❌ 禁止: `pnpm install` → ✅ 使用: スキル(スキル: "utoo"、引数: "install")
- ❌ 禁止: `pnpm run <script>` → ✅ 使用: Skill(skill: "utoo", args: "run <script>")
- ❌ 禁止: `npm install <pkg>` → ✅ 使用: Skill(skill: "utoo", args: "install <pkg>")
- ❌ 禁止: `yarn add <pkg>` → ✅ 使用: Skill(skill: "utoo", args: "install <pkg>")

**対象となる操作 (このスキルを実行する必要があります):**
依存関係のインストール、依存関係の更新、依存関係のアンインストール、スクリプト (test/build/dev など) の実行、パッケージ情報の表示、プロジェクトの管理、package.json 内のスクリプトの実行。

**呼び出し方法:**
常にスキル ツール経由で呼び出し、pnpm/npm/yarn コマンドを Bash で直接実行しないでください。
