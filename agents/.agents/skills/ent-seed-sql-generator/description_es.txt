Genere datos semilla INSERT SQL deterministas a partir de esquemas Go Ent y entradas mixtas. Esta habilidad es REQUERIDA siempre que necesite crear SQL semilla para desarrollo o pruebas: maneja la inferencia de entidades, la integridad de las relaciones, los ID estables y la generación de SQL específico de dialecto, incluidos JSON, matrices y tipos complejos. Utilice esta habilidad para cualquier tarea que involucre datos semilla, dispositivos de prueba, inicialización de demostración o llenado de bases de datos a partir de definiciones de esquemas Ent, incluso si el usuario no menciona explícitamente "semilla" o "SQL".
