Genera dati seed deterministici INSERT SQL da schemi Go Ent e input misti. Questa competenza è RICHIESTA ogni volta che è necessario creare seed SQL per lo sviluppo o il test: gestisce l'inferenza di entità, l'integrità delle relazioni, gli ID stabili e la generazione SQL specifica del dialetto, inclusi JSON, array e tipi complessi. Utilizzare questa competenza per qualsiasi attività che coinvolga dati seed, dispositivi di test, inizializzazione demo o popolamento di database dalle definizioni dello schema Ent, anche se l'utente non menziona esplicitamente "seed" o "SQL".
