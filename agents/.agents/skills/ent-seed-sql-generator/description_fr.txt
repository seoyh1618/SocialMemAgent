Générez des données de départ déterministes INSERT SQL à partir de schémas Go Ent et d'entrées mixtes. Cette compétence est REQUISE chaque fois que vous avez besoin de créer du SQL de départ pour le développement ou les tests - elle gère l'inférence d'entité, l'intégrité des relations, les ID stables et la génération SQL spécifique au dialecte, y compris JSON, les tableaux et les types complexes. Utilisez cette compétence pour toute tâche impliquant des données de départ, des montages de test, l'initialisation de démonstration ou le remplissage de bases de données à partir de définitions de schéma Ent, même si l'utilisateur ne mentionne pas explicitement « graine » ou « SQL ».
