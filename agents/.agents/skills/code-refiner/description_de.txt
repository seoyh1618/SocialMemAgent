Tiefgreifende Codevereinfachung, Refactoring und Qualitätsverbesserung. Analysiert strukturelle Komplexität, Anti-Patterns und Lesbarkeitsdefizite und wendet dann gezieltes Refactoring an, um das genaue Verhalten beizubehalten. Sprachunabhängig: Python, Go, TypeScript/JavaScript, Rust. Nutzen Sie diese Fähigkeit, wenn das Ziel eher in der Vereinfachung und Klarheit als in der Fehlersuche besteht. Auslöser bei: „Diesen Code vereinfachen“, „Meinen Code bereinigen“, „Um Klarheit schaffen“, „Komplexität reduzieren“, „Lesbarer machen“, „Codequalitätsüberprüfung“, „Bereinigung technischer Schulden“, „Codeverfeinerung ausführen“, „Letzte Änderungen vereinfachen“, „Dieser Code ist chaotisch“, „zu viel Verschachtelung“, „diese Funktion ist zu lang“, „bereinigen Sie dies, bevor ich es PR mache“, „Räumen Sie meinen Code auf“, zyklomatische Komplexität, kognitive Komplexität, Codegerüche.
