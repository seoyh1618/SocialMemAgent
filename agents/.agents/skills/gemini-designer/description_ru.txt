Делегируйте задачи проектирования Gemini (gemini-2.5-flash-preview) через API aicodemirror. Используйте, когда вам нужен пользовательский интерфейс/веб-дизайн (одностраничный HTML), значки/иллюстрации SVG, цветовые палитры, предложения по типографике, советы по макету или любые рекомендации по визуальному дизайну. Gemini выступает в роли вашего друга-дизайнера — попросите его создать макеты HTML-страниц, разработать значки SVG, предложить системы дизайна или оставить отзыв о дизайне. Запускается по запросам, связанным с дизайном, таким как «создать дизайн страницы», «создать значок», «предложить цвета», «сделать логотип», «макет пользовательского интерфейса», «образец дизайна».
