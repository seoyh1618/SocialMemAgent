تفويض مهام التصميم إلى Gemini (gemini-2.5-flash-preview) عبر واجهة برمجة التطبيقات aicodemirror. استخدمه عندما تحتاج إلى تصميم واجهة المستخدم/الويب (HTML لصفحة واحدة)، أو أيقونات/رسوم توضيحية SVG، أو لوحات الألوان، أو اقتراحات الطباعة، أو نصائح التخطيط، أو أي مرجع للتصميم المرئي. يعمل Gemini كصديقك المصمم - اطلب منه إنشاء نماذج بالحجم الطبيعي لصفحات HTML، أو تصميم أيقونات SVG، أو اقتراح أنظمة تصميم، أو تقديم تعليقات حول التصميم. مشغلات للطلبات المتعلقة بالتصميم مثل "تصميم صفحة"، "إنشاء رمز"، "اقتراح الألوان"، "إنشاء شعار"، "نموذج واجهة المستخدم"، "مرجع التصميم".
