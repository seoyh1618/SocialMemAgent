Обработка изображений для веб-разработки — изменение размера, обрезка, обрезка пробелов, преобразование форматов (PNG/WebP/JPG), оптимизация размера файла, создание миниатюр, создание изображений карточек OG. Использует Pillow (Python) — ImageMagick не требуется. Запускайте с помощью «изменить размер изображения», «конвертировать в webp», «обрезать логотип», «оптимизировать изображения», «создать миниатюру», «создать OG-изображение», «обрезать пробелы», «обработать изображение» или «изображение слишком большое».
