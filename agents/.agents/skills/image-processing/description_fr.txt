Traitez les images pour le développement Web : redimensionnez, recadrez, coupez les espaces, convertissez les formats (PNG/WebP/JPG), optimisez la taille du fichier, générez des vignettes, créez des images de carte OG. Utilise Pillow (Python) – aucun ImageMagick n'est nécessaire. Déclenchez avec « redimensionner l'image », « convertir en WebP », « couper le logo », « optimiser les images », « créer une vignette », « créer une image OG », « recadrer les espaces », « traiter l'image » ou « image trop grande ».
