Metahabilidad para mejorar y optimizar las indicaciones utilizando las mejores prácticas de ingeniería de indicaciones de Anthropic. Proporciona el flujo de trabajo de mejora de 4 pasos (identificación de ejemplos, borrador inicial, refinamiento de la cadena de pensamiento, mejora de ejemplos), registros de palabras clave para búsqueda de documentación y árboles de decisión para estrategias de mejora. Úselo para mejorar indicaciones, optimizar la precisión, agregar razonamiento en cadena de pensamiento, estructurar con etiquetas XML, mejorar ejemplos o iterar sobre la calidad de las indicaciones. Delega la habilidad de gestión de documentos para obtener documentación oficial de ingeniería rápida.
