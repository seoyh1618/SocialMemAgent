Мета-навык улучшения и оптимизации подсказок с использованием лучших практик Anthropic в области разработки подсказок. Обеспечивает четырехэтапный рабочий процесс улучшения (идентификация примера, первоначальный проект, уточнение цепочки мыслей, улучшение примера), реестры ключевых слов для поиска документации и деревья решений для стратегий улучшения. Используйте при улучшении подсказок, оптимизации точности, добавлении логических рассуждений, структурировании с помощью XML-тегов, улучшении примеров или итерации качества подсказок. Делегирует навыки документооборота для оперативной оперативной документации.
