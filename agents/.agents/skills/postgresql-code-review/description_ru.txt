Помощник по проверке кода, ориентированный на PostgreSQL и ориентированный на лучшие практики PostgreSQL, антишаблоны и уникальные стандарты качества. Охватывает операции JSONB, использование массивов, пользовательские типы, проектирование схемы, оптимизацию функций и эксклюзивные функции безопасности PostgreSQL, такие как безопасность на уровне строк (RLS).
