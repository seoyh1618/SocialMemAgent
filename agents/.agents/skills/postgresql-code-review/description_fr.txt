Assistant de révision de code spécifique à PostgreSQL se concentrant sur les meilleures pratiques PostgreSQL, les anti-modèles et les normes de qualité uniques. Couvre les opérations JSONB, l'utilisation des tableaux, les types personnalisés, la conception de schémas, l'optimisation des fonctions et les fonctionnalités de sécurité exclusives à PostgreSQL telles que la sécurité au niveau des lignes (RLS).
