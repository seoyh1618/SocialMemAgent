IRify SAST: pruebas de seguridad de aplicaciones estáticas impulsadas por IA. Compile el código fuente en SSA IR, luego use SyntaxFlow DSL para rastrear el flujo de datos a través de los límites de las funciones, detectar vulnerabilidades (SQLi, RCE, XXE) y responder preguntas como "¿adónde va la entrada del usuario?". Admite 7 idiomas (Java, PHP, JS, Go, Python, C, Yak) y compilación incremental mediante ProgramOverLay. Requiere servidor MCP yaklang: yak mcp -t ssa
