Tool zur Bereinigung von Systemdateien. Scannt und identifiziert große Dateien und Junk-Dateien (temporäre Dateien, Caches, Protokolle, Backups usw.) und bietet Benutzern eine interaktive Bereinigungsoberfläche, über die sie Löschvorgänge auswählen können. Verwenden Sie diese Fähigkeit, wenn Benutzer Speicherplatz bereinigen, Systemdateien organisieren, große Dateien finden, Junk-Dateien löschen oder Speicherplatz freigeben müssen.
