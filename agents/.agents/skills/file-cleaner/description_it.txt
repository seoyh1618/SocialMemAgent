Strumento di pulizia dei file di sistema. Esegue la scansione e identifica file di grandi dimensioni, file spazzatura (file temporanei, cache, registri, backup, ecc.) e fornisce un'interfaccia di pulizia interattiva che consente agli utenti di scegliere le eliminazioni. Utilizza questa capacità quando gli utenti hanno bisogno di pulire spazio su disco, organizzare file di sistema, trovare file di grandi dimensioni, eliminare file spazzatura o liberare spazio di archiviazione.
