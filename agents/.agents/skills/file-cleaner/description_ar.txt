أداة تنظيف ملفات النظام. يقوم بمسح وتحديد الملفات الكبيرة والملفات غير المرغوب فيها (الملفات المؤقتة، وذاكرة التخزين المؤقت، والسجلات، والنسخ الاحتياطية، وما إلى ذلك)، ويوفر واجهة تنظيف تفاعلية للمستخدمين لاختيار عمليات الحذف. استخدم هذه المهارة عندما يحتاج المستخدمون إلى تنظيف مساحة القرص، أو تنظيم ملفات النظام، أو العثور على ملفات كبيرة، أو حذف الملفات غير المرغوب فيها، أو تحرير مساحة تخزين.
