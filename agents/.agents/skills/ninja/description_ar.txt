مهارة بناء نظام النينجا. يُستخدم عند تشخيص حالات فشل بناء Ninja، أو فهم دور Ninja كمنفذ بناء منخفض المستوى تم إنشاؤه بواسطة CMake أو أنظمة بناء التعريف الأخرى، أو ضبط التوازي، أو تفسير مخرجات Ninja، أو العمل مع ملفات build.ninja. يتم تنشيطه عند الاستعلامات حول أخطاء النينجا، أو توازي النينجا، أو إخراج النينجا المطول، أو تنسيق build.ninja، أو النينجا كمولد CMake.
