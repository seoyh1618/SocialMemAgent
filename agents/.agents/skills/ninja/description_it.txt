Abilità del sistema di costruzione Ninja. Da utilizzare per diagnosticare gli errori di build di Ninja, comprendere il ruolo di Ninja come esecutore di build di basso livello generato da CMake o altri sistemi di meta-build, ottimizzare il parallelismo, interpretare l'output di Ninja o lavorare con file build.ninja. Si attiva su query su errori ninja, parallelismo ninja, output verboso ninja, formato build.ninja o ninja come generatore CMake.
