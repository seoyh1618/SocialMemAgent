Полная настройка Sentry SDK для React. Используйте, когда вас просят «добавить Sentry в React», «установить @sentry/react» или настроить мониторинг ошибок, трассировку, воспроизведение сеанса, профилирование или ведение журнала для приложений React. Поддерживает React 16+, React Router v5–v7, TanStack Router, Redux, Vite и webpack.
