Configurazione completa dell'SDK Sentry per React. Da utilizzare quando viene richiesto di "aggiungere Sentry a React", "installare @sentry/react" o configurare il monitoraggio degli errori, la traccia, la riproduzione della sessione, la profilazione o la registrazione per le applicazioni React. Supporta React 16+, React Router v5-v7, TanStack Router, Redux, Vite e webpack.
