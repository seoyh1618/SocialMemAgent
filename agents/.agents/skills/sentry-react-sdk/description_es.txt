Configuración completa de Sentry SDK para React. Úselo cuando se le solicite "agregar Sentry a React", "instalar @sentry/react" o configurar el monitoreo de errores, el seguimiento, la reproducción de sesiones, la creación de perfiles o el registro para aplicaciones de React. Admite React 16+, React Router v5-v7, TanStack Router, Redux, Vite y webpack.
