Vollständiges Sentry SDK-Setup für React. Verwenden Sie diese Option, wenn Sie aufgefordert werden, „Sentry zu React hinzuzufügen“, „@sentry/react zu installieren“ oder Fehlerüberwachung, Nachverfolgung, Sitzungswiedergabe, Profilerstellung oder Protokollierung für React-Anwendungen zu konfigurieren. Unterstützt React 16+, React Router v5-v7, TanStack Router, Redux, Vite und Webpack.
