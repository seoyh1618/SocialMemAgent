Configuration complète du SDK Sentry pour React. À utiliser lorsqu'on vous demande « d'ajouter Sentry à React », « d'installer @sentry/react » ou de configurer la surveillance des erreurs, le traçage, la relecture de session, le profilage ou la journalisation pour les applications React. Prend en charge React 16+, React Router v5-v7, TanStack Router, Redux, Vite et webpack.
