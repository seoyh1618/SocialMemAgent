Komprimieren, komprimieren oder verkleinern Sie Markdown-Dateien, um weniger Token zu verwenden und gleichzeitig alle Informationen und Bedeutung beizubehalten. Verwenden Sie diesen Skill immer dann, wenn der Benutzer die Größe einer Markdown-Datei reduzieren, eine README-Datei verkleinern, eine SKILL.md oder CLAUDE.md komprimieren, die Dokumentation minimieren oder einen Markdown tokeneffizienter gestalten möchte. Wird auch dann ausgelöst, wenn in einer Markdown-Datei nur „Kürzer machen“ oder „Komprimieren“ gesagt wird.
