Compatta, comprimi o minimizza i file di markdown per utilizzare meno token preservando tutte le informazioni e il significato. Utilizza questa abilità ogni volta che l'utente desidera ridurre la dimensione di un file di markdown, ridurre un file README, comprimere SKILL.md o CLAUDE.md, minimizzare la documentazione o rendere qualsiasi markdown più efficiente in termini di token. Attiva anche se dicono semplicemente "rendilo più breve" o "comprimilo" su un file di markdown.
