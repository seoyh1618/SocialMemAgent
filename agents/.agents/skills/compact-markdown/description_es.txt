Compacte, comprima o minimice archivos de rebajas para utilizar menos tokens y al mismo tiempo conservar toda la información y el significado. Utilice esta habilidad siempre que el usuario quiera reducir el tamaño de un archivo de rebajas, reducir un archivo README, comprimir un SKILL.md o CLAUDE.md, minimizar la documentación o hacer que cualquier rebajas sea más eficiente en términos de token. Se activa incluso si solo dicen "acortar esto" o "comprimir esto" en un archivo de rebajas.
