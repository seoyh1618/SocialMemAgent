Diseñe, cree y perfeccione un panel de control de contactos (CCP) personalizado y una configuración completa del centro de contacto de Amazon Connect. Esta habilidad ayuda a crear interfaces de usuario de CCP basadas en web personalizadas, integrarlas con Amazon Connect Streams y modelar perfiles de seguridad, perfiles de enrutamiento, colas, campañas, flujos de contacto, IVR/bots, monitoreo (intrusión/susurro) y métricas para que la experiencia general coincida con las necesidades del negocio.
