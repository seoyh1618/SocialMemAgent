DOIT LIRE avant d'exécuter toute évaluation ADK. Méthodologie d'évaluation ADK : métriques d'évaluation, schéma d'évaluation, LLM en tant que juge, notation de la trajectoire des outils et causes d'échec courantes. À utiliser lors de l'évaluation de la qualité de l'agent, de l'exécution d'adk eval ou du débogage des résultats d'évaluation. Ne PAS utiliser pour les modèles de code API (utilisez adk-cheatsheet), le déploiement (utilisez adk-deploy-guide) ou l'échafaudage de projet (utilisez adk-scaffold).
