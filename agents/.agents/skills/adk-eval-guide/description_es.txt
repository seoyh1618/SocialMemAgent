DEBE LEER antes de ejecutar cualquier evaluación de ADK. Metodología de evaluación de ADK: métricas de evaluación, esquema de conjunto de evaluaciones, LLM como juez, puntuación de trayectoria de herramientas y causas de fallas comunes. Úselo al evaluar la calidad del agente, ejecutar adk eval o depurar resultados de evaluación. NO lo use para patrones de código API (use adk-cheatsheet), implementación (use adk-deploy-guide) o andamiaje de proyectos (use adk-scaffold).
