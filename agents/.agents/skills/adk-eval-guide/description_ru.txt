ОБЯЗАТЕЛЬНО ПРОЧИТАЙТЕ перед запуском какой-либо оценки ADK. Методика оценки ADK — метрики оценки, схема набора оценок, LLM как судья, оценка траектории инструмента и распространенные причины сбоев. Используйте при оценке качества агента, запуске adk eval или отладке результатов оценки. НЕ используйте для шаблонов кода API (используйте adk-cheatsheet), развертывания (используйте adk-deploy-guide) или создания каркасов проекта (используйте adk-scaffold).
