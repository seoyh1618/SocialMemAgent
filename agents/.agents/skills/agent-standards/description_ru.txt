Определяет поведенческие и когнитивные стандарты для старших инженеров ИИ. Используйте при настройке протоколов рассуждения агента, управления памятью или стратегий контекстной инженерии. Используйте для автономного рассуждения, многоуровневых систем памяти, проверяемого выполнения целей, многоагентной оркестрации и оптимизации токенов.
