Détecte la remise à zéro manquante des données sensibles dans le code source et identifie la remise à zéro supprimée par les optimisations du compilateur, avec une analyse au niveau de l'assemblage et une vérification du flux de contrôle. À utiliser pour auditer les secrets, les clés, les mots de passe ou d'autres données sensibles du code C/C++/Rust.
