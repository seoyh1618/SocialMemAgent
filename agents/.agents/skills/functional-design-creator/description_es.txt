Habilidad para crear un documento de diseño funcional basado en el PRD. Si docs/prd.md existe, genera un documento de diseño funcional que incluye la arquitectura del sistema, el modelo de datos, el diseño de componentes, el diseño de algoritmos, etc. Úselo para solicitudes como 「機能設計書を作成して」「PRDから設計書を作って」「diseño funcional.を書いて」.
