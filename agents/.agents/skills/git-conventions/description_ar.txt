اصطلاحات Git للفرق بما في ذلك الالتزامات التقليدية واستراتيجيات التفرع وسير عمل العلاقات العامة واستراتيجيات الدمج وتنسيق رسائل الالتزام. يُستخدم عند كتابة رسائل الالتزام، أو إنشاء الفروع، أو إعداد مهام سير عمل Git، أو عندما يسأل المستخدم عن اصطلاحات Git، أو تنسيقات الالتزام، أو إستراتيجيات التفرع، أو أفضل ممارسات العلاقات العامة.
