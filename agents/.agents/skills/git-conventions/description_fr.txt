Conventions Git pour les équipes, notamment les validations conventionnelles, les stratégies de branchement, les flux de travail de relations publiques, les stratégies de fusion et le formatage des messages de validation. À utiliser lors de la rédaction de messages de validation, de la création de branches, de la configuration de flux de travail Git ou lorsque l'utilisateur pose des questions sur les conventions Git, les formats de validation, les stratégies de branchement ou les meilleures pratiques en matière de relations publiques.
