Преобразует содержимое статьи или краткое изложение в минималистичный рисованный стиль JSON с подсказками для инструментов создания изображений AI. Используйте этот навык всякий раз, когда пользователь хочет создать какой-либо визуальный элемент из текстового контента, включая баннеры, иллюстрации статей, встроенные диаграммы, инфографику или концептуальные визуальные эффекты. Запускается по таким запросам, как «превратить это в визуальное представление», «создать подсказку к изображению», «сделать иллюстрацию для этого», «создать диаграмму из этой статьи», «Мне нужен эскиз для этого раздела» или любому запросу, сочетающему анализ контента с созданием изображения/визуальной подсказки. Всегда используйте этот навык, когда пользователь предоставляет текстовый контент и хочет получить подсказку с изображением, готовым к использованию AI.
