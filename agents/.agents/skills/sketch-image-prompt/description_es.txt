Transforma el contenido de los artículos o resúmenes en mensajes JSON de estilo minimalista dibujados a mano para herramientas de generación de imágenes de IA. Utilice esta habilidad siempre que el usuario desee crear cualquier tipo de imagen a partir de contenido de texto, incluidos pancartas, ilustraciones de artículos, diagramas en línea, infografías o imágenes conceptuales. Active solicitudes como "convertir esto en un elemento visual", "crear un mensaje de imagen", "hacer una ilustración para esto", "generar un diagrama a partir de este artículo", "Necesito un boceto para esta sección" o cualquier solicitud que combine el análisis de contenido con la generación de mensajes visuales o de imágenes. Utilice siempre esta habilidad cuando el usuario proporcione contenido de texto y desee una salida de mensaje de imagen lista para IA.
