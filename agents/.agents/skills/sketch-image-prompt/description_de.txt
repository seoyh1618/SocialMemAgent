Wandelt Artikelinhalte oder Zusammenfassungen in JSON-Eingabeaufforderungen im minimalistischen, handgezeichneten Stil für KI-Bildgenerierungstools um. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer aus Textinhalten beliebige visuelle Elemente erstellen möchte – einschließlich Banner, Artikelillustrationen, Inline-Diagramme, Infografiken oder Konzeptvisualisierungen. Wird bei Anfragen wie „Umwandle dies in ein Bild“, „Erstelle eine Bildaufforderung“, „Erstelle eine Illustration dafür“, „Generiere ein Diagramm aus diesem Artikel“, „Ich benötige eine Skizze für diesen Abschnitt“ oder bei jeder Anforderung, die Inhaltsanalyse mit der Generierung von Bildern/visuellen Aufforderungen kombiniert, ausgelöst. Verwenden Sie diese Fähigkeit immer, wenn der Benutzer Textinhalte bereitstellt und eine KI-fähige Bildausgabe als Eingabeaufforderung wünscht.
