記事のコンテンツまたは要約を、AI 画像生成ツール用のミニマルな手描きスタイルの JSON プロンプトに変換します。ユーザーがバナー、記事のイラスト、インライン図、インフォグラフィック、コンセプトビジュアルなど、テキストコンテンツからあらゆる種類のビジュアルを作成したい場合は常にこのスキルを使用します。 「これをビジュアルに変換する」、「画像プロンプトを作成する」、「このイラストを作成する」、「この記事から図を生成する」、「このセクションのスケッチが必要」などのリクエスト、またはコンテンツ分析と画像/ビジュアル プロンプト生成を組み合わせたリクエストをトリガーします。ユーザーがテキスト コンテンツを提供し、AI 対応の画像プロンプト出力を必要とする場合は、常にこのスキルを使用してください。
