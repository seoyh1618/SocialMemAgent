قم بالتحميل بشكل استباقي عندما تتضمن المهمة تصميم قاعدة البيانات أو المخططات أو الوصول إلى البيانات. يُستخدم عندما يقول المستخدم "إعداد قاعدة البيانات" أو "إنشاء مخطط" أو "إضافة عملية ترحيل" أو "كتابة استعلام" أو "إعداد Prisma". يغطي تصميم المخطط وتطبيعه، وإعداد ORM (Prisma، وDrizzle)، وسير عمل الترحيل، وتجميع الاتصالات، وتحسين الاستعلامات، واستراتيجيات الفهرسة، والبذر، وأنماط المعاملات لـ PostgreSQL، وMySQL، وSQLite، وMongoDB.
