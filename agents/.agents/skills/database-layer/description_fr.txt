Chargez de manière PROACTIVE lorsque la tâche implique la conception de bases de données, les schémas ou l'accès aux données. À utiliser lorsque l'utilisateur dit « configurer la base de données », « créer un schéma », « ajouter une migration », « écrire une requête » ou « configurer Prisma ». Couvre la conception et la normalisation des schémas, la configuration ORM (Prisma, Drizzle), les flux de travail de migration, le regroupement de connexions, l'optimisation des requêtes, les stratégies d'indexation, l'amorçage et les modèles de transactions pour PostgreSQL, MySQL, SQLite et MongoDB.
