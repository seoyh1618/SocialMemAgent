Fornire un flusso di lavoro TTS locale attorno a Qwen3-TTS. Supporta: generazione vocale a frase singola (CustomVoice/VoiceDesign/VoiceClone), generazione di doppiaggio batch di testo lungo (articolo → script di doppiaggio JSON → TTS batch → unione). Scenari applicabili: generazione vocale, doppiaggio di audiolibri, narrazione video, lettura di dialoghi multicarattere, clonazione vocale.
