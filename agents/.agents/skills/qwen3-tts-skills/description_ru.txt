Обеспечьте локальный рабочий процесс TTS вокруг Qwen3-TTS. Поддерживается: генерация голоса из одного предложения (CustomVoice/VoiceDesign/VoiceClone), генерация пакетного дубляжа длинного текста (статья → сценарий дубляжа JSON → пакетный TTS → слияние). Применимые сценарии: генерация голоса, дублирование аудиокниг, видеоповествование, чтение многосимвольных диалогов, клонирование голоса.
