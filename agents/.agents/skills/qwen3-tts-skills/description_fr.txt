Fournissez un flux de travail TTS local autour de Qwen3-TTS. Prend en charge : génération de voix d'une seule phrase (CustomVoice/VoiceDesign/VoiceClone), génération de doublage par lots de texte long (article → script de doublage JSON → batch TTS → fusion). Scénarios applicables : génération de voix, doublage de livres audio, narration vidéo, lecture de dialogues multi-caractères, clonage de voix.
