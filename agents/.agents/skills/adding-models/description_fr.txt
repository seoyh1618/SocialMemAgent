Guide pour ajouter de nouveaux modèles LLM à Letta Code. À utiliser lorsque l'utilisateur souhaite ajouter la prise en charge d'un nouveau modèle, a besoin de connaître les descripteurs de modèle valides ou souhaite mettre à jour la configuration du modèle. Couvre la configuration models.json, la matrice de test CI et la validation des poignées.
