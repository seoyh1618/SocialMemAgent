Implemente el proyecto en la plataforma de alojamiento: lea la pila YAML para obtener la configuración exacta, detecte herramientas CLI locales (vercel, wrangler, supabase, fly, sst), configure la base de datos, inserte código y verifique la implementación en vivo. Úselo cuando el usuario diga "implementarlo", "enviarlo a producción", "configurar alojamiento" o después de que se complete /build. NO lo use antes de que se complete la construcción.
