Déployez le projet sur la plate-forme d'hébergement : lisez la pile YAML pour la configuration exacte, détectez les outils CLI locaux (vercel, wrangler, supabase, fly, sst), configurez la base de données, poussez le code, vérifiez le déploiement en direct. À utiliser lorsque l'utilisateur dit « déployer », « pousser en production », « configurer l'hébergement » ou une fois /build terminé. Ne PAS utiliser avant la fin de la construction.
