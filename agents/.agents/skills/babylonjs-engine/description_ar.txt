مهارة شاملة لمحرك عرض الويب Babylon.js ثلاثي الأبعاد. استخدم هذه المهارة عند إنشاء تجارب ثلاثية الأبعاد في الوقت الفعلي أو ألعاب تعتمد على المتصفح أو تصورات تفاعلية أو تطبيقات ويب غامرة. تشغيل المهام التي تتضمن Babylon.js، أو المشاهد ثلاثية الأبعاد، أو عرض WebGL/WebGPU، أو أنظمة مكونات الكيان، أو عمليات المحاكاة الفيزيائية، أو مواد PBR، أو رسم خرائط الظل، أو تحميل النماذج ثلاثية الأبعاد. بديل لـ Three.js مع تكامل المحرر المدمج وميزات محرك اللعبة.
