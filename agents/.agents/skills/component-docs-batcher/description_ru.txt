Автоматизированный инструмент для пакетного создания и поддержки документации библиотеки компонентов внешнего интерфейса. Разработанный для библиотек компонентов Monorepo (например, атом-ui-mobile), он сравнивает время модификации файлов компонентов и их документации для автоматического создания списков задач и пакетного создания полной документации компонентов, соответствующей стандартам. Поддерживается: (1) периодическое пакетное обновление документов компонентов, (2) ведение однокомпонентных документов, (3) инициализация документов новых компонентов. Автоматически определяет файлы основных компонентов, пропуская вспомогательные файлы, такие как index.ts, store, хуки и т. д.; документы размещаются по адресу index.zh-CN.md внутри каждого каталога компонента.
