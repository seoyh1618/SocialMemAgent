Создавайте и устраняйте неполадки рабочих процессов GitHub Actions. Используйте при редактировании файлов .github/workflows, настройке конвейеров CI/CD, настройке матричных сборок для многоплатформенного тестирования, отладке сбойных рабочих процессов, добавлении кэширования или артефактов, запуске тестов E2E в контейнерах или при запросе «почему мой рабочий процесс не работает» или «как мне протестировать на нескольких операционных системах».
