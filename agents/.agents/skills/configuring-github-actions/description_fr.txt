Créez et dépannez les workflows GitHub Actions. À utiliser lors de la modification de fichiers .github/workflows, de la configuration de pipelines CI/CD, de la configuration de builds matriciels pour des tests multiplateformes, du débogage de workflows défaillants, de l'ajout de mise en cache ou d'artefacts, de l'exécution de tests E2E dans des conteneurs ou de la question « pourquoi mon workflow échoue-t-il » ou « comment tester sur plusieurs systèmes d'exploitation ».
