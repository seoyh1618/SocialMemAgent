Crawler di codice morto basato su Ripgrep che trova moduli JavaScript/CJS/MJS obsoleti, danneggiati o orfani nel framework Agent Studio. Percorre una directory di destinazione, controlla ogni file per i riferimenti esterni e mostra (o elimina) i file senza riferimenti. Previene il gonfiamento del registro e l'interruzione silenziosa degli strumenti da parte dei moduli che nessuno chiama.
