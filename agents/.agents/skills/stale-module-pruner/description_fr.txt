Robot d'exploration de code mort alimenté par Ripgrep qui trouve les modules JavaScript/CJS/MJS obsolètes, cassés ou orphelins dans le framework Agent Studio. Parcourt un répertoire cible, vérifie chaque fichier pour les références externes et fait apparaître (ou supprime) les fichiers non référencés. Empêche la surcharge du registre et les abandons d'outils silencieux des modules que personne n'appelle.
