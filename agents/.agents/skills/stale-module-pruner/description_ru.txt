Сканер мертвого кода на базе Ripgrep, который находит устаревшие, сломанные или потерянные модули JavaScript/CJS/MJS в среде Agent Studio. Обходит целевой каталог, проверяет каждый файл на наличие внешних ссылок и обнаруживает (или удаляет) файлы, на которые нет ссылок. Предотвращает раздувание реестра и автоматическое отключение инструментов из модулей, которые никто не вызывает.
