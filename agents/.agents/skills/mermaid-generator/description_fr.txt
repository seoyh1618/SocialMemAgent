Sélectionnez intelligemment le type de graphique le plus approprié en fonction de la description de l'utilisateur et générez du code Mermaid. Prend en charge tous les types, y compris les organigrammes, les diagrammes de séquence, les diagrammes de classes, les diagrammes ER, les diagrammes de Gantt, les diagrammes d'état, etc., avec des jeux de couleurs éclatants et magnifiques.
