Делайте снимки экрана «до» и «после» для запросов на включение, чтобы документировать визуальные изменения. Поддерживает подходы CLI агент-браузер и Playwright Python. Используйте при создании PR с изменениями пользовательского интерфейса, сравнении визуальных состояний между ветвями, документировании изменений внешнего интерфейса для проверки кода или создании визуальных отчетов о различиях.
