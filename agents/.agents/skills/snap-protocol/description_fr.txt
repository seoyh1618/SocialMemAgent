SNAP (Signed Network Agent Protocol) pour la communication décentralisée d'agent à agent. Couvre : l'identité (adresses Bitcoin P2TR), le format du message, l'authentification (signatures BIP-340 Schnorr), les méthodes, les états de tâche, les cartes d'agent, les codes d'erreur et la découverte basée sur Nostr. À utiliser lorsque l'utilisateur mentionne le protocole SNAP, l'identité de l'agent, les adresses P2TR ou la messagerie agent à agent.
