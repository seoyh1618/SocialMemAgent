SNAP (Signed Network Agent Protocol) für dezentrale Agent-zu-Agent-Kommunikation. Behandelt: Identität (Bitcoin-P2TR-Adressen), Nachrichtenformat, Authentifizierung (BIP-340-Schnorr-Signaturen), Methoden, Aufgabenstatus, Agentenkarten, Fehlercodes und Nostr-basierte Erkennung. Verwenden Sie diese Option, wenn der Benutzer SNAP-Protokoll, Agentenidentität, P2TR-Adressen oder Agent-zu-Agent-Nachrichten erwähnt.
