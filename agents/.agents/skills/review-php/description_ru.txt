Просмотрите код PHP на предмет языковых соглашений и соглашений времени выполнения: строгие типы, обработка ошибок, управление ресурсами, стандарты PSR, пространства имен, нулевая безопасность, генераторы и возможность тестирования. Атомарные навыки, связанные только с языком; вывод представляет собой список результатов.
