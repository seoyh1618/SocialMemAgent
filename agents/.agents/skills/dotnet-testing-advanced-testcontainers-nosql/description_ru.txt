Тестовые контейнеры Полное руководство по интеграционному тестированию NoSQL.
Охватывает операции с документами MongoDB, пять структур данных Redis, шаблон Collection Fixture.
Включает сериализацию BSON, тестирование производительности индексов, стратегии изоляции данных и управление жизненным циклом контейнера.
