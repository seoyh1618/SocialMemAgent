Среда выполнения, сборщик и менеджер пакетов Bun JavaScript. Охватывает HTTP-сервер Bun.serve(), ввод-вывод Bun.file(), SQLite, хеширование паролей, упаковщик Bun.build(), запуск Bun:test и управление пакетами.

Используйте при сборке с помощью Bun API, запуске сценариев с помощью Bun, объединении кода, управлении пакетами с помощью Bun install/add или написании тестов с помощью Bun:test.
