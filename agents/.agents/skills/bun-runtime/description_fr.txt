Runtime Bun JavaScript, bundler et gestionnaire de packages. Couvre le serveur HTTP Bun.serve(), les E/S Bun.file(), SQLite, le hachage de mot de passe, le bundle Bun.build(), bun:test runner et la gestion des packages.

À utiliser lors de la création avec les API Bun, de l'exécution de scripts avec Bun, du regroupement de code, de la gestion de packages avec bun install/add ou de l'écriture de tests avec bun:test.
