Bun JavaScript runtime, bundler e gestore di pacchetti. Copre il server HTTP Bun.serve(), l'I/O Bun.file(), SQLite, l'hashing delle password, il bundler Bun.build(), bun:test runner e la gestione dei pacchetti.

Da utilizzare durante la creazione con API Bun, l'esecuzione di script con Bun, il raggruppamento di codice, la gestione di pacchetti con installazione/aggiunta bun o la scrittura di test con bun:test.
