Riferimento ioredis v5 per il client Redis Node.js: configurazione della connessione, RedisOptions, pipeline, transazioni, Pub/Sub, scripting Lua, Cluster e Sentinel. Utilizzare quando: (1) creare o configurare connessioni Redis (autonome, cluster, sentinel), (2) scrivere comandi Redis con ioredis (get/set, pipeline, multi/exec), (3) impostare Pub/Sub o Streams, (4) configurare retryStrategy, TLS o auto-pipelining, (5) lavorare con le opzioni Redis Cluster (scaleReads, mappatura NAT) o (6) eseguire il debug di problemi di connessione ioredis. Importante: utilizzare il nome import `import { Redis } from 'ioredis'` per i tipi TypeScript corretti con NodeNext.
