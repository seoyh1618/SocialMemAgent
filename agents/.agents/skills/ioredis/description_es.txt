Referencia de ioredis v5 para el cliente Node.js Redis: configuración de conexión, RedisOptions, canalizaciones, transacciones, Pub/Sub, secuencias de comandos Lua, Cluster y Sentinel. Úselo cuando: (1) cree o configure conexiones de Redis (independiente, clúster, centinela), (2) escriba comandos de Redis con ioredis (get/set, pipelines, multi/exec), (3) configure Pub/Sub o Streams, (4) configure retryStrategy, TLS o canalización automática, (5) trabaje con opciones de Redis Cluster (scaleReads, mapeo NAT) o (6) depure problemas de conexión de ioredis. Importante: utilice la importación con nombre `import { Redis } de 'ioredis'` para los tipos correctos de TypeScript con NodeNext.
