Справочник по ioredis v5 для клиента Redis Node.js — настройка соединения, RedisOptions, конвейеры, транзакции, Pub/Sub, сценарии Lua, Cluster и Sentinel. Используйте при: (1) создании или настройке подключений Redis (автономное, кластерное, дозорное), (2) написании команд Redis с помощью ioredis (get/set, конвейеры, multi/exec), (3) настройке Pub/Sub или Streams, (4) настройке retryStrategy, TLS или автоматической конвейерной обработки, (5) работе с параметрами кластера Redis (scaleReads, сопоставление NAT) или (6) отладке ioredis проблемы с подключением. Важно: используйте именованный импорт `import { Redis } from 'ioredis'` для правильных типов TypeScript с NodeNext.
