Référence ioredis v5 pour le client Node.js Redis — configuration de la connexion, RedisOptions, pipelines, transactions, Pub/Sub, scripts Lua, Cluster et Sentinel. À utiliser lors de : (1) la création ou la configuration de connexions Redis (autonome, cluster, sentinel), (2) l'écriture de commandes Redis avec ioredis (get/set, pipelines, multi/exec), (3) la configuration de Pub/Sub ou Streams, (4) la configuration de retryStrategy, TLS ou auto-pipelining, (5) l'utilisation des options du cluster Redis (scaleReads, mappage NAT) ou (6) le débogage des problèmes de connexion ioredis. Important : utilisez l'importation nommée `import { Redis } from 'ioredis'` pour les types TypeScript corrects avec NodeNext.
