مرجع ioredis v5 لعميل Node.js Redis — إعداد الاتصال، وRedisOptions، وخطوط الأنابيب، والمعاملات، وPub/Sub، والبرمجة النصية Lua، وCluster، وSentinel. يُستخدم عندما: (1) إنشاء اتصالات Redis أو تكوينها (مستقلة، أو مجموعة، أو حارسة)، (2) كتابة أوامر Redis باستخدام ioredis (get/set، خطوط الأنابيب، multi/exec)، (3) إعداد Pub/Sub أو Streams، (4) تكوين retryStrategy، أو TLS، أو التوجيه التلقائي، (5) العمل مع خيارات Redis Cluster (scaleReads، أو تعيين NAT)، أو (6) تصحيح أخطاء ioredis مشاكل الاتصال. هام: استخدم الاستيراد المسمى `import { Redis } من 'ioredis'` لأنواع TypeScript الصحيحة مع NodeNext.
