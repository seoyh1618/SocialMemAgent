Node.js Redis クライアントの ioredis v5 リファレンス — 接続セットアップ、RedisOptions、パイプライン、トランザクション、Pub/Sub、Lua スクリプト、クラスター、Sentinel。次の場合に使用します: (1) Redis 接続の作成または構成 (スタンドアロン、クラスター、センチネル)、(2) ioredis を使用した Redis コマンドの作成 (get/set、パイプライン、マルチ/実行)、(3) Pub/Sub または Streams のセットアップ、(4) retryStrategy、TLS、または自動パイプラインの構成、(5) Redis クラスター オプションの操作 (scaleReads、NAT マッピング)、または (6) デバッグioredis 接続の問題。重要: NodeNext で正しい TypeScript 型を得るには、名前付きインポート `import { Redis } from 'ioredis'` を使用してください。
