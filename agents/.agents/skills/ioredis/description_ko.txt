Node.js Redis 클라이언트에 대한 ioredis v5 참조 - 연결 설정, RedisOptions, 파이프라인, 트랜잭션, Pub/Sub, Lua 스크립팅, 클러스터 및 Sentinel. 다음의 경우에 사용하십시오: (1) Redis 연결(독립형, 클러스터, Sentinel) 생성 또는 구성, (2) ioredis로 Redis 명령 작성(get/set, 파이프라인, multi/exec), (3) Pub/Sub 또는 Streams 설정, (4) retryStrategy, TLS 또는 자동 파이프라인 구성, (5) Redis 클러스터 옵션(scaleReads, NAT 매핑) 작업 또는 (6) ioredis 연결 문제 디버깅. 중요: NodeNext에서 올바른 TypeScript 유형을 사용하려면 import `import { Redis } from 'ioredis'`를 사용하세요.
