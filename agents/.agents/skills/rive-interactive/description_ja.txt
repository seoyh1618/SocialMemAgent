ランタイムの対話性と Web 統合を備えたステート マシン ベースのベクター アニメーション。このスキルは、インタラクティブなアニメーション、状態駆動型 UI、ロジックを使用したアニメーション化されたコンポーネント、またはランタイム コントロールを使用してデザイナーが作成したアニメーションを作成するときに使用します。 Rive、ステート マシン、インタラクティブ ベクター アニメーション、入力処理を伴うアニメーション、ViewModel データ バインディング、または React Rive 統合に関連するタスクをトリガーします。ステート マシンと双方向対話性を必要とするアニメーション用の Lottie の代替手段。
