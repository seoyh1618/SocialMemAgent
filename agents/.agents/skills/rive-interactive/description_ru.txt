Векторная анимация на основе конечного автомата с интерактивностью во время выполнения и веб-интеграцией. Используйте этот навык при создании интерактивной анимации, пользовательского интерфейса, управляемого состоянием, анимированных компонентов с логикой или созданных дизайнером анимаций с управлением во время выполнения. Запускается по задачам, связанным с Rive, конечными автоматами, интерактивной векторной анимацией, анимацией с обработкой ввода, привязкой данных ViewModel или интеграцией React Rive. Альтернатива Lottie для анимаций, требующих конечных автоматов и двусторонней интерактивности.
