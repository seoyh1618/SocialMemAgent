الرسوم المتحركة المتجهة القائمة على الآلة مع تفاعل وقت التشغيل وتكامل الويب. استخدم هذه المهارة عند إنشاء رسوم متحركة تفاعلية، أو واجهة مستخدم تعتمد على الحالة، أو مكونات متحركة ذات منطق، أو رسوم متحركة أنشأها المصمم مع التحكم في وقت التشغيل. يتم تشغيل المهام التي تتضمن Rive أو أجهزة الحالة أو الرسوم المتحركة المتجهة التفاعلية أو الرسوم المتحركة مع معالجة الإدخال أو ربط بيانات ViewModel أو تكامل React Rive. بديل لـ Lottie للرسوم المتحركة التي تتطلب أجهزة حالة وتفاعل ثنائي الاتجاه.
