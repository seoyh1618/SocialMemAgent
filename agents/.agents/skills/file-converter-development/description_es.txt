Guía de desarrollo para el paquete base @rytass/file-converter (檔案轉換器開發指南). Úselo al crear nuevos adaptadores de conversión de archivos (adaptador 新增檔案轉換), comprender las interfaces de conversión o crear canalizaciones de procesamiento de imágenes. Cubre la interfaz Converter, ConverterManager y los patrones de integración de Sharp. Palabras clave: convertidor de archivos, 檔案轉換, 圖片處理, procesamiento de imágenes, Sharp, cambiar tamaño, marca de agua, transcodificar
