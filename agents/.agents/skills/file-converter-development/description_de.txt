Entwicklungshandbuch für das Basispaket @rytass/file-converter (檔案轉換器開發指南). Verwenden Sie diese Option, wenn Sie neue Dateikonverteradapter (新增檔案轉換-Adapter) erstellen, Konverterschnittstellen verstehen oder Bildverarbeitungspipelines erstellen. Behandelt die Converter-Schnittstelle, den ConverterManager und Sharp-Integrationsmuster. Schlüsselwörter: Dateikonverter, 檔案轉換, 圖片處理, Bildverarbeitung, Scharf, Größe ändern, Wasserzeichen, Transkodierung
