Руководство по разработке базового пакета @rytass/file-converter (檔案轉換器開發指南). Используйте при создании новых адаптеров конвертеров файлов (адаптер 新增檔案轉換), понимании интерфейсов конвертеров или построении конвейеров обработки изображений. Охватывает интерфейс Converter, ConverterManager и шаблоны интеграции Sharp. Ключевые слова: конвертер файлов, 檔案轉換, 圖片處理, обработка изображений, Sharp, изменение размера, водяной знак, перекодирование.
