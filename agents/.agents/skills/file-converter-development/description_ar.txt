دليل تطوير الحزمة الأساسية @rytass/file-converter (檔案轉換器開發指南). يُستخدم عند إنشاء محولات محولات ملفات جديدة (المحول)، أو فهم واجهات المحول، أو إنشاء مسارات معالجة الصور. يغطي واجهة المحول وConverterManager وأنماط التكامل الحادة. الكلمات الرئيسية: تحويل الملفات، 檔案轉換، 圖片處理، معالجة الصور، حاد، تغيير الحجم، العلامة المائية، تحويل الشفرة
