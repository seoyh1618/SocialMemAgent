Guida allo sviluppo per il pacchetto base @rytass/file-converter (檔案轉換器開發指南). Da utilizzare durante la creazione di nuovi adattatori di conversione file (adattatore 新增檔案轉換), la comprensione delle interfacce del convertitore o la creazione di pipeline di elaborazione delle immagini. Copre l'interfaccia del convertitore, ConverterManager e i modelli di integrazione Sharp. Parole chiave: convertitore di file, 檔案轉換, 圖片處理, elaborazione immagini, Nitido, ridimensionamento, filigrana, transcodifica
