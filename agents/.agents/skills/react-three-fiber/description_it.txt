Costruisci scene 3D dichiarative con React Three Fiber (R3F), un renderer React per Three.js. Da utilizzare quando si creano esperienze 3D interattive in applicazioni React con architettura basata su componenti, gestione dello stato e astrazioni riutilizzabili. Ideale per configuratori di prodotti, portfolio, giochi, visualizzazione di dati ed esperienze web coinvolgenti.
