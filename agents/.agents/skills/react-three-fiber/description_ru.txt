Создавайте декларативные 3D-сцены с помощью React Three Fiber (R3F) — средства рендеринга React для Three.js. Используйте при создании интерактивных 3D-приложений в приложениях React с компонентной архитектурой, управлением состоянием и многократно используемыми абстракциями. Идеально подходит для конфигураторов продуктов, портфолио, игр, визуализации данных и иммерсивного веб-интерфейса.
