قم بإنشاء واجهة مستخدم Flutter للهواتف المحمولة والويب مميزة وذات جودة إنتاجية بجودة تصميم عالية. استخدم هذه المهارة عندما يطلب المستخدم إنشاء شاشات Flutter أو عناصر واجهة المستخدم أو المكونات أو لوحات المعلومات أو التطبيقات الكاملة. يُنشئ رمز Dart/Flutter إبداعيًا ومصقولًا يتجنب جماليات الذكاء الاصطناعي العامة ويتبع أفضل ممارسات Flutter/Material/Cupertino.
