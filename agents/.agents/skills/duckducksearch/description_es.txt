Capacidades de búsqueda web utilizando el motor de búsqueda DuckDuckGo. Úselo cuando: (1) realice búsquedas de texto/página web con filtros, (2) busque imágenes con filtros de tamaño/color/licencia, (3) encuentre videos con opciones de resolución, (4) obtenga resultados de noticias con filtros de tiempo, (5) necesite soporte de proxy o búsquedas específicas de región, (6) use CLI para búsquedas por lotes.
