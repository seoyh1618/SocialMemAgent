Возможности веб-поиска с использованием поисковой системы DuckDuckGo. Используйте, когда: (1) Выполнение поиска по тексту/веб-страницам с помощью фильтров, (2) Поиск изображений с фильтрами размера/цвета/лицензии, (3) Поиск видео с параметрами разрешения, (4) Получение результатов новостей с фильтрами по времени, (5) Требуется поддержка прокси-сервера или поиск по конкретному региону, (6) Использование CLI для пакетного поиска.
