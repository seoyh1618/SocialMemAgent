Funzionalità di ricerca sul Web utilizzando il motore di ricerca DuckDuckGo. Utilizzare quando: (1) eseguire ricerche di testo/pagine Web con filtri, (2) cercare immagini con filtri di dimensione/colore/licenza, (3) trovare video con opzioni di risoluzione, (4) ottenere risultati di notizie con filtri temporali, (5) è necessario supporto proxy o ricerche specifiche per regione, (6) utilizzo della CLI per ricerche batch.
