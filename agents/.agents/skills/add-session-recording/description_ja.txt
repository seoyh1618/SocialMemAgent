Temps SDK を使用して、プライバシーに配慮したセッションの記録と再生を React アプリケーションに追加します。入力マスキング、要素ブロック、GDPR 準拠の同意フローを通じてプライバシーを尊重しながら、再生のためにユーザー インタラクションをキャプチャします。ユーザーが次のことを行う場合に使用します: (1) アプリにセッション記録を追加する、(2) セッション リプレイ機能を実装する、(3) デバッグのためにユーザー セッションを記録する、(4) プライバシーに準拠した画面記録を追加する、(5) ビジュアル リプレイでユーザーの問題をデバッグする、(6) rrweb ベースの記録を実装する、(7) GDPR 準拠のセッション キャプチャを設定する。トリガー: 「セッション記録」、「セッション再生」、「セッション記録」、「ユーザー再生」、「画面記録」、「rrweb」、「セッションキャプチャ」。
