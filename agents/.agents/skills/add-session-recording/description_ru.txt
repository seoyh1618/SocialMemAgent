Добавьте запись и воспроизведение сеанса с учетом конфиденциальности в приложения React с помощью Temps SDK. Фиксирует взаимодействие пользователей для воспроизведения, соблюдая при этом конфиденциальность посредством маскировки ввода, блокировки элементов и потоков согласия, соответствующих требованиям GDPR. Используйте, когда пользователь хочет: (1) добавить запись сеанса в свое приложение, (2) реализовать функцию воспроизведения сеанса, (3) записывать сеансы пользователя для отладки, (4) добавить запись экрана с соблюдением конфиденциальности, (5) отладить проблемы пользователя с помощью визуального воспроизведения, (6) реализовать запись на основе rrweb, (7) настроить захват сеанса в соответствии с GDPR. Триггеры: «запись сеанса», «воспроизведение сеанса», «запись сеанса», «воспроизведение пользователя», «запись экрана», «rrweb», «захват сеанса».
