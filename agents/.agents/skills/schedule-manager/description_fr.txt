(macOS uniquement) Gérez le calendrier et les rappels Apple via osascript et rappels-cli, en suivant la méthodologie GTD. Se déclenche sur « planning », « calendrier », « rappels », « GTD », « todo ». Scénarios de déclenchement : lorsque l'utilisateur dit "planifier une réunion", "créer un rappel", "vérifier mon calendrier", "planifier la semaine prochaine", "ajouter une tâche", "que dois-je faire aujourd'hui", "révision hebdomadaire", "prendre une note", "ne pas oublier".
