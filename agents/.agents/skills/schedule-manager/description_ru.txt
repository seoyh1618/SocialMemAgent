(только для macOS) Управляйте календарем и напоминаниями Apple с помощью osascript и напоминаний-cli, следуя методологии GTD. Триггеры по «расписанию», «календарю», «напоминаниям», «GTD», «todo». Сценарии триггера: когда пользователь говорит «запланировать встречу», «создать напоминание», «проверь мой календарь», «запланировать на следующую неделю», «добавить дело», «что мне сделать сегодня», «еженедельный обзор», «сделать заметку», «не забыть».
