Analizza il CSS, le impostazioni Tailwind e i componenti di un progetto esistente per comprenderne i token di progettazione, quindi migliora visivamente l'interfaccia utente e invia il risultato a Figma tramite il server MCP Figma. Piuttosto che semplici modifiche all’interfaccia utente, garantisce modifiche in linea con il sistema di progettazione del progetto: nessun colore o spaziatura codificata, solo proposte all’interno di token esistenti e regole di denominazione.
