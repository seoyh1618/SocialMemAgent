Analice el CSS, la configuración de Tailwind y los componentes de un proyecto existente para comprender sus tokens de diseño, luego mejore visualmente la interfaz de usuario y envíe el resultado a Figma a través del servidor Figma MCP. En lugar de solo ajustes en la interfaz de usuario, garantiza cambios alineados con el sistema de diseño del proyecto: sin colores ni espacios codificados, solo propuestas dentro de tokens y reglas de nomenclatura existentes. Úselo cuando: – “Mire esta interfaz de usuario y envíela a Figma” / “Mejore el diseño de esta página y sincronícela con Figma” – “Analice la interfaz de usuario y cree una propuesta de diseño en Figma” – Siempre que necesite refinar la interfaz de usuario de una aplicación web y sincronizar el diseño con Figma – Cuando el usuario diga “ui-to-figma”, “enviar UI a Figma” o “mejorar el diseño y enviar a Figma” Requisitos previos: Dramaturgo (habilidad de prueba de aplicaciones web) y el servidor Figma MCP deben estar disponibles.
