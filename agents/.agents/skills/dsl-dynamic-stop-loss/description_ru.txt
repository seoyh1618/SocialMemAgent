Автоматический трейлинг-стоп для бессрочных позиций с кредитным плечом на Hyperliquid. Отслеживает цену через cron, фиксирует минимальные уровни прибыли с помощью настраиваемых уровней и автоматически закрывает позиции при нарушении через Mcporter — без вмешательства агента на критическом пути. Работает для ДЛИННЫХ и КОРОТКИХ. Триггеры уровня на основе ROE (рентабельности маржи), которые автоматически учитывают кредитное плечо. Используйте при защите открытой перп-позиции Hyperliquid, настройке трейлинг-стопа, управлении уровнями прибыли или автоматизации выхода из позиции в случае нарушения.
