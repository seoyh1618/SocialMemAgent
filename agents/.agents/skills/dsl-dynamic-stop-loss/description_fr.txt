Stop loss suiveur automatisé pour les positions perpétuelles à effet de levier sur Hyperliquide. Surveille les prix via cron, augmente les niveaux de profit via des niveaux configurables et ferme automatiquement les positions en cas de violation via mcporter – aucune intervention d'agent pour le chemin critique. Fonctionne pour LONG et COURT. Déclencheurs de niveau basés sur le ROE (retour sur marge) qui tiennent automatiquement compte de l'effet de levier. À utiliser pour protéger une position ouverte de perp Hyperliquid, configurer des stop suiveurs, gérer les niveaux de profit ou automatiser les sorties de position en cas de violation.
