Fornire indicazioni sull'implementazione per le notifiche di avviso popup Sileo (pacchetto npm "sileo") nelle app React. Da utilizzare quando si aggiunge la vista Toaster, si attivano toast globali tramite sileo.success/error/warning/info/show/action, si gestiscono flussi asincroni con sileo.promise, si ignorano/cancellano toast, si configurano posizioni/offset, si creano temi con modalità chiaro/scuro/di sistema e si personalizza lo stile tramite riempimento, sostituzioni di stili, pilota automatico, rotondità, variabili CSS e selettori di attributi dati. Utilizza questa abilità anche per il comportamento di scorrimento per eliminare, icone personalizzate, descrizioni JSX, supporto per movimenti ridotti, esportazioni di tipo TypeScript e comprensione dell'architettura di morphing appiccicosa SVG interna di Sileo. Assicurati di utilizzare questa abilità ogni volta che l'utente menziona sileo, notifiche di avviso popup in React, toast appiccicosi, notifiche basate sulla fisica o desidera aggiungere qualsiasi tipo di sistema di avviso di avviso/notifica utilizzando sileo, anche se non nomina esplicitamente il pacchetto.
