Proporcionar orientación de implementación para notificaciones de brindis de Sileo (paquete npm "sileo") en aplicaciones React. Úselo al agregar la ventana gráfica Toaster, activar brindis globales a través de sileo.success/error/warning/info/show/action, manejar flujos asíncronos con sileo.promise, descartar/borrar tostadas, configurar posiciones/desplazamiento, crear temas con modos claro/oscuro/sistema y personalizar el estilo mediante relleno, anulaciones de estilos, piloto automático, redondez, variables CSS y selectores de atributos de datos. Utilice también esta habilidad para el comportamiento de deslizar para descartar, íconos personalizados, descripciones JSX, soporte de movimiento reducido, exportaciones de tipo TypeScript y comprensión de la pegajosa arquitectura de transformación SVG interna de Sileo. Asegúrate de usar esta habilidad siempre que el usuario mencione sileo, notificaciones de brindis en React, brindis pegajosos, notificaciones basadas en física o quiera agregar cualquier tipo de sistema de notificación/tost usando sileo, incluso si no nombran explícitamente el paquete.
