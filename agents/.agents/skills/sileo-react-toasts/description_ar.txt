توفير إرشادات التنفيذ لإشعارات Sileo (حزمة npm "sileo") في تطبيقات React. يُستخدم عند إضافة منفذ عرض Toaster، وإطلاق الخبز المحمص الشامل عبر sileo.success/error/warning/info/show/action، والتعامل مع التدفقات غير المتزامنة باستخدام sileo.promise، ورفض/مسح الخبز المحمص، وتكوين المواضع/الإزاحة، وتخصيص الأنماط عبر أوضاع التعبئة، وتجاوز الأنماط، والطيار الآلي، والاستدارة، ومتغيرات CSS، ومحددات سمات البيانات. استخدم أيضًا هذه المهارة لسلوك التمرير السريع للرفض، والأيقونات المخصصة، وأوصاف JSX، ودعم الحركة المنخفضة، وتصدير أنواع TypeScript، وفهم بنية تحويل SVG الداخلية لـ Sileo. تأكد من استخدام هذه المهارة عندما يذكر المستخدم sileo، أو إشعارات الخبز المحمص في React، أو الخبز المحمص اللزج، أو الإشعارات المستندة إلى الفيزياء، أو يريد إضافة أي نوع من نظام الخبز المحمص/الإشعارات باستخدام sileo، حتى لو لم يذكر اسم الحزمة بشكل صريح.
