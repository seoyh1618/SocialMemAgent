Bereitstellung von Implementierungsanleitungen für Sileo-Toastbenachrichtigungen (npm-Paket „sileo“) in React-Apps. Wird verwendet, wenn Sie das Toaster-Ansichtsfenster hinzufügen, globale Toasts über sileo.success/error/warning/info/show/action auslösen, asynchrone Flows mit sileo.promise verarbeiten, Toasts verwerfen/löschen, Positionen/Versatz konfigurieren, Themen mit Hell-/Dunkel-/Systemmodi festlegen und den Stil über Füllung, Stilüberschreibungen, Autopilot, Rundheit, CSS-Variablen und Datenattributselektoren anpassen. Nutzen Sie diese Fähigkeit auch für das Wischen zum Verwerfen, benutzerdefinierte Symbole, JSX-Beschreibungen, Unterstützung für reduzierte Bewegungen, Exporte von TypeScript-Typen und das Verständnis der internen SVG-Morphing-Architektur von Sileo. Stellen Sie sicher, dass Sie diesen Skill immer dann verwenden, wenn der Benutzer Sileo, Toast-Benachrichtigungen in React, klebrige Toasts oder physikbasierte Benachrichtigungen erwähnt oder jede Art von Toast-/Benachrichtigungssystem mit Sileo hinzufügen möchte, auch wenn er das Paket nicht explizit nennt.
