Предоставить рекомендации по реализации всплывающих уведомлений Sileo (пакет npm «sileo») в приложениях React. Используйте при добавлении области просмотра Toaster, запуске глобальных всплывающих уведомлений через sileo.success/error/warning/info/show/action, обработке асинхронных потоков с помощью sileo.promise, отклонении/очистке всплывающих уведомлений, настройке позиций/смещения, оформлении тем с помощью светлых/темных/системных режимов и настройке стилей с помощью заливки, переопределений стилей, автопилота, округлости, переменных CSS и селекторов атрибутов данных. Также используйте этот навык для поведения смахивания для закрытия, пользовательских значков, описаний JSX, поддержки замедленного движения, экспорта типов TypeScript и понимания внутренней архитектуры морфинга SVG Sileo. Обязательно используйте этот навык всякий раз, когда пользователь упоминает sileo, всплывающие уведомления в React, липкие тосты, уведомления на основе физики или хочет добавить любую систему тостов/уведомлений с использованием sileo, даже если он явно не называет пакет.
