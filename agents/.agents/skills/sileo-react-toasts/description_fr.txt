Fournir des conseils de mise en œuvre pour les notifications toast Sileo (package npm « sileo ») dans les applications React. À utiliser lors de l'ajout de la fenêtre Toaster, du déclenchement de toasts globaux via sileo.success/error/warning/info/show/action, de la gestion des flux asynchrones avec sileo.promise, du rejet/effacement des toasts, de la configuration des positions/décalage, du thème avec les modes clair/sombre/système et de la personnalisation du style via le remplissage, les remplacements de styles, le pilote automatique, la rondeur, les variables CSS et les sélecteurs d'attributs de données. Utilisez également cette compétence pour le comportement de balayage pour rejeter, les icônes personnalisées, les descriptions JSX, la prise en charge des mouvements réduits, les exportations de type TypeScript et la compréhension de l'architecture interne de morphing gluant SVG de Sileo. Assurez-vous d'utiliser cette compétence chaque fois que l'utilisateur mentionne sileo, les notifications toast dans React, les toasts gluants, les notifications basées sur la physique ou souhaite ajouter tout type de système toast/notification à l'aide de sileo, même s'il ne nomme pas explicitement le package.
