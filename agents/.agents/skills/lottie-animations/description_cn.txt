After Effects animation rendering for web and React applications.在实现 Lottie 动画、JSON 矢量动画、交互式动画图标、微交互或加载动画时使用此技能。触发涉及 Lottie、lottie-web、lottie-react、dotLottie、After Effects JSON 导出、bodymovin、动画 SVG 替代品或设计师创建的动画的任务。 Complements GSAP ScrollTrigger and Framer Motion for scroll-driven and interactive animations.
