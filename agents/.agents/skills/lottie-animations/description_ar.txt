عرض الرسوم المتحركة After Effects لتطبيقات الويب وReact. استخدم هذه المهارة عند تنفيذ رسوم Lottie المتحركة، أو رسوم متحركة متجهة JSON، أو أيقونات متحركة تفاعلية، أو تفاعلات دقيقة، أو تحميل رسوم متحركة. يتم تشغيلها عند تنفيذ المهام التي تتضمن Lottie، أو lottie-web، أو lottie-react، أو dotLottie، أو تصدير After Effects JSON، أو bodymovin، أو بدائل SVG المتحركة، أو الرسوم المتحركة التي أنشأها المصمم. يكمل GSAP ScrollTrigger وFramer Motion للرسوم المتحركة التفاعلية والمعتمدة على التمرير.
