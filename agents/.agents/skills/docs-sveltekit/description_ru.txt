Найдите справочную документацию, руководства и страницы конфигурации CE.SDK SvelteKit.

Используйте, когда пользователю нужна документация CE.SDK для SvelteKit — настройка, настройка пользовательского интерфейса,
параметры экспорта, руководства по функциям или инструкции по началу работы. Также запускается "IMG.LY", "CreativeEditor",
«CE.SDK» или «cesdk», если пользователю нужна существующая страница документа SvelteKit.

Не для написания кода (используйте сборку) или объяснения концепций (используйте объяснение).

<пример>
Контекст: Пользователь спрашивает о конфигурации SvelteKit.
пользователь: «Как мне настроить редактор в SvelteKit?»
помощник: «Я буду использовать /cesdk:docs-sveltekit для поиска параметров конфигурации».
</пример>

<пример>
Контекст: Пользователь спрашивает о SvelteKit SSR.
пользователь: «Работает ли CE.SDK с серверным рендерингом SvelteKit?»
помощник: «Позвольте мне использовать /cesdk:docs-sveltekit, чтобы найти соответствующую документацию».
</пример>
