Suchen Sie nach CE.SDK SvelteKit-Referenzdokumenten, Anleitungen und Konfigurationsseiten.

Verwenden Sie diese Option, wenn der Benutzer CE.SDK-Dokumente für SvelteKit benötigt – Konfiguration, UI-Anpassung,
Exportoptionen, Funktionshandbücher oder Anleitungen für den Einstieg. Wird auch durch „IMG.LY“, „CreativeEditor“ ausgelöst.
„CE.SDK“ oder „cesdk“, wenn der Benutzer eine vorhandene SvelteKit-Dokumentseite benötigt.

Nicht zum Schreiben von Code (Build verwenden) oder Konzepterklärungen (Explain verwenden).

<Beispiel>
Kontext: Der Benutzer fragt nach der SvelteKit-Konfiguration
Benutzer: „Wie konfiguriere ich den Editor in SvelteKit?“
Assistent: „Ich verwende /cesdk:docs-sveltekit, um nach Konfigurationsoptionen zu suchen.“
</Beispiel>
