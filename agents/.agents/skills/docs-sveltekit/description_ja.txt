CE.SDK SvelteKit リファレンス ドキュメント、ガイド、構成ページを検索してください。

ユーザーが SvelteKit の CE.SDK ドキュメントを必要とする場合 (構成、UI カスタマイズ、
エクスポート オプション、機能ガイド、または開始手順。 「IMG.LY」、「CreativeEditor」によってもトリガーされます。
「CE.SDK」、またはユーザーが既存の SvelteKit ドキュメント ページを必要とする場合は「cesdk」。

コードの作成 (build を使用) や概念の説明 (explain を使用) のためではありません。

<例>
コンテキスト: ユーザーが SvelteKit の構成について尋ねる
ユーザー: 「SvelteKit でエディタを設定するにはどうすればよいですか?」
アシスタント: 「/cesdk:docs-sveltekit を使用して構成オプションを調べます。」
</例>

<例>
コンテキスト: ユーザーが SvelteKit SSR について質問する
ユーザー: 「CE.SDK は SvelteKit サーバー側レンダリングで動作しますか?」
アシスタント: 「関連ドキュメントを見つけるには /cesdk:docs-sveltekit を使用させてください。」
</例>
