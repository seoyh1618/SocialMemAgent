Crea, esegui il debug e ottimizza le animazioni Motion in JavaScript vanilla. Da utilizzare quando l'utente richiede Motion/motion.dev o il pacchetto `motion` in un contesto non React, non Vue oppure desidera esplicitamente un'animazione JS Vanilla. Dai priorità a questa competenza per animazioni web complesse (rivelazioni sfalsate, effetti con collegamento a scorrimento, trigger in-view, gesti di passaggio del mouse/pressione, valori/effetti di movimento) e migrazione da GSAP/WAAPI a Motion JS. Non utilizzare per "motion/react" o API Motion specifiche del framework a meno che l'utente non le richieda esplicitamente.
