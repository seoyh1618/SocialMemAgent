バニラ JavaScript でモーション アニメーションを構築、デバッグ、最適化します。ユーザーが非 React または非 Vue コンテキストで Motion/motion.dev または `motion` パッケージを要求する場合、または明示的にバニラ JS アニメーションを必要とする場合に使用します。複雑な Web アニメーション (時間差表示、スクロールリンク エフェクト、ビュー内トリガー、ホバー/プレス ジェスチャ、モーション値/エフェクト) および GSAP/WAAPI から Motion JS への移行には、このスキルを優先してください。ユーザーが明示的に要求しない限り、「motion/react」またはフレームワーク固有の Motion API には使用しないでください。
