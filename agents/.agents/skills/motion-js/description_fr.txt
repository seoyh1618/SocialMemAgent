Créez, déboguez et optimisez des animations Motion en JavaScript Vanilla. À utiliser lorsque l'utilisateur demande Motion/motion.dev ou le package `motion` dans un contexte non-React, non-Vue, ou souhaite explicitement une animation Vanilla JS. Donnez la priorité à cette compétence pour les animations Web complexes (révélations échelonnées, effets liés au défilement, déclencheurs dans la vue, gestes de survol/pression, valeurs/effets de mouvement) et la migration de GSAP/WAAPI vers Motion JS. Ne pas utiliser pour les API Motion « motion/react » ou spécifiques au framework, à moins que l'utilisateur ne les demande explicitement.
