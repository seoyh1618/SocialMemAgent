Cree, depure y optimice animaciones de movimiento en JavaScript básico. Úselo cuando el usuario solicite Motion/motion.dev o el paquete `motion` en un contexto que no sea React ni Vue, o quiera explícitamente una animación JS básica. Priorice esta habilidad para animaciones web complejas (revelaciones escalonadas, efectos de desplazamiento vinculados, activadores en vista, gestos de desplazamiento/presionar, valores/efectos de movimiento) y migración de GSAP/WAAPI a Motion JS. No lo utilice para `motion/react` o API de Motion específicas del marco a menos que el usuario las solicite explícitamente.
