Ежедневный помощник по программированию, который автоматически запускается при написании/изменении кода и предоставляет основной контрольный список.

✅ Триггерные сценарии:
- Внедрение новых функций, добавление кода, изменение существующего кода.
- Пользовательские запросы "написать...", "внедрить...", "добавить...", "изменить..."
- Любая задача кодирования, включающая инструменты редактирования/записи.

❌ Не вызывает:
- Чистое чтение/понимание кода (без намерений модификации)
- Уже обладает специальными навыками (обнаружение ошибок, архитектурное проектирование, tdd-руководство)
- Внесение изменений в конфигурационный файл, написание документации.
