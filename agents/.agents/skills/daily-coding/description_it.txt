Assistente di codifica giornaliero che si attiva automaticamente durante la scrittura/modifica del codice, fornendo una lista di controllo fondamentale.

✅ Scenari scatenanti:
- Implementazione di nuove funzionalità, aggiunta di codice, modifica di codice esistente
- L'utente richiede "scrivi un...", "implementa...", "aggiungi...", "modifica..."
- Qualsiasi attività di codifica che coinvolga strumenti di modifica/scrittura

❌ Non attiva:
- Codice di pura lettura/comprensione (nessun intento di modifica)
- Già ricoperte da competenze specialistiche (bug-detective, architecture-design, tdd-guide)
- Modifiche ai file di configurazione, scrittura della documentazione
