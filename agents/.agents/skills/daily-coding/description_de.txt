Täglicher Codierungsassistent, der beim Schreiben/Ändern von Code automatisch auslöst und eine Kerncheckliste bereitstellt.

✅ Triggerszenarien:
- Implementierung neuer Funktionen, Hinzufügen von Code, Ändern vorhandener Codes
- Benutzeranfragen „Schreibe ein…“, „Implementiere…“, „Hinzufügen…“, „Ändern…“
- Jede Codierungsaufgabe, die Bearbeitungs-/Schreibwerkzeuge umfasst

❌ Löst nicht aus:
- Reines Lesen/Verstehen von Code (keine Änderungsabsicht)
- Bereits durch Fachkompetenzen abgedeckt (Bug-Detective, Architektur-Design, TDD-Guide)
- Änderungen an Konfigurationsdateien, Verfassen von Dokumentationen
