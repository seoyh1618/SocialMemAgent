Verwenden Sie Chrome CDP, um jede URL zu extrahieren und in Markdown zu konvertieren. Unterstützt zwei Modi: automatisches Scraping nach dem Laden der Seite oder Warten auf ein Benutzersignal (geeignet für Seiten, die eine Anmeldung erfordern). Verwenden Sie diese Option, wenn der Benutzer darum bittet, eine Webseite als Markdown zu speichern.
