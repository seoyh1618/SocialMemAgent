Используйте Chrome CDP, чтобы очистить любой URL-адрес и преобразовать его в Markdown. Поддерживает два режима — автоматическое парсинг после загрузки страницы или ожидание сигнала пользователя (подходит для страниц, требующих входа в систему). Используйте, когда пользователь просит сохранить веб-страницу в формате Markdown.
