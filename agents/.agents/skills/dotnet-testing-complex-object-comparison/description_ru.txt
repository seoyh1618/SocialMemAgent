Специализированные навыки для сравнения сложных объектов и проверки глубокого уровня.
Используйте, когда необходимо глубокое сравнение объектов, необходимо исключить определенные свойства, обрабатывать циклические ссылки или требуется проверка DTO/Entity.
Охватывает правила BeEquiваленто, Исключение, Включение и пользовательские правила сопоставления.
