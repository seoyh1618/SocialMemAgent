Progetta contratti API proto3 + HTTP per progetti di impalcature go-sphere da prompt, cartelle di input o documenti dei requisiti con dati simulati. Da utilizzare durante la definizione delle API del servizio, la selezione tra messaggi entpb/condivisi/personalizzati e l'applicazione di convenzioni sull'impalcatura, regole di sicurezza del router e posizionamento degli errori locali del servizio. Questa competenza è NECESSARIA per qualsiasi attività di progettazione di proto API nello scaffold go-sphere: usala sempre invece di scrivere file proto da zero.
