Diseñe contratos de API proto3 + HTTP para proyectos de andamio go-sphere a partir de indicaciones, carpetas de entrada o documentos de requisitos con datos simulados. Úselo al definir API de servicio, seleccionar entre mensajes entpb/compartidos/personalizados y aplicar convenciones de andamio, reglas de seguridad del enrutador y ubicación de errores locales del servicio. Esta habilidad es REQUERIDA para cualquier tarea de diseño de proto API en el andamio go-sphere; utilícela siempre en lugar de escribir archivos proto desde cero.
