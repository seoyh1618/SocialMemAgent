قم بتصميم عقود proto3 + HTTP API لمشاريع سقالة go-sphere من المطالبات أو مجلدات الإدخال أو مستندات المتطلبات مع بيانات وهمية. يُستخدم عند تعريف واجهات برمجة تطبيقات الخدمة، والاختيار بين الرسائل entpb/المشتركة/المخصصة، وفرض اصطلاحات السقالة، وقواعد سلامة جهاز التوجيه، ووضع الخطأ المحلي للخدمة. هذه المهارة مطلوبة لأي مهمة تصميم واجهة برمجة تطبيقات أولية في go-sphere scaffold - استخدمها دائمًا بدلاً من كتابة ملفات أولية من البداية.
