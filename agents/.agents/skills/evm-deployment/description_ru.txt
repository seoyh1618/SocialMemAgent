Развертывание контрактов Sablier EVM (Controller, ERC20 Faucet, Flow, Lockup, Airdrops) с полной автоматизацией рабочих процессов. Этот навык следует использовать, когда пользователь запрашивает «развертывание», «развертывание протокола», «развертывание в цепочку» или упоминает задачи, связанные с развертыванием. Управляет развертыванием контрактов, проверкой проводника, обновлениями SDK и созданием образцов данных с помощью сценария Init.
