Strumento di automazione del browser intelligente basato sull'intelligenza artificiale. Utilizza LLM per comprendere le pagine ed eseguire attività in modo automatico, in modo più intelligente ed efficiente in termini di token rispetto al drammaturgo tradizionale. Ideale per interazioni complesse, pagine dinamiche e operazioni del browser che richiedono decisioni intelligenti. Preferibile il browser Chrome.
