Herramienta de automatización de navegador inteligente impulsada por IA. Utiliza LLM para comprender páginas y realizar tareas automáticamente, de forma más inteligente y con mayor eficiencia simbólica que el Playwright tradicional. Ideal para interacciones complejas, páginas dinámicas y operaciones de navegador que requieren decisiones inteligentes. Preferiblemente el navegador Chrome.
