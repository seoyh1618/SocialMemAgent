أدوات مساعدة Python للقراءة فقط لتكامل Jira وConfluence وBitbucket. يوفر إمكانية الوصول للقراءة إلى المشكلات والبحث ومسارات العمل والصفحات وطلبات السحب وسجل الالتزام والمزيد. يُستخدم عندما يحتاج المستخدمون إلى الاستعلام عن منتجات Atlassian مثل "الحصول على مشكلة Jira"، أو "البحث في صفحات التقاء"، أو "عرض تفاصيل طلب السحب"، أو "الحصول على سجل الالتزام". يستثني هذا المتغير جميع عمليات الكتابة لكفاءة الرمز المميز وسلامته.
