Утилиты Python только для чтения для интеграции Jira, Confluence и Bitbucket. Предоставляет доступ для чтения к проблемам, поиску, рабочим процессам, страницам, запросам на включение, истории коммитов и многому другому. Используйте, когда пользователям нужно запросить продукты Atlassian, например «получить проблему с Jira», «поиск на страницах Confluence», «просмотреть подробности запроса на включение» или «получить историю коммитов». Этот вариант исключает все операции записи для повышения эффективности и безопасности токена.
