Utilidades de Python de solo lectura para la integración de Jira, Confluence y Bitbucket. Proporciona acceso de lectura a problemas, búsquedas, flujos de trabajo, páginas, solicitudes de extracción, historial de confirmaciones y más. Úselo cuando los usuarios necesiten consultar productos de Atlassian como "obtener un problema de Jira", "buscar páginas de Confluence", "ver detalles de la solicitud de extracción" u "obtener el historial de confirmaciones". Esta variante excluye todas las operaciones de escritura para lograr eficiencia y seguridad del token.
