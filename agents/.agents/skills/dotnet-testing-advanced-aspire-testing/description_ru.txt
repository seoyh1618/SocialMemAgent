Полное руководство по инфраструктуре интеграционного тестирования .NET Aspire Testing.
Охватывает настройку проекта AppHost, DistributedApplicationTestingBuilder, управление жизненным циклом контейнера.
Включает миграцию из тестовых контейнеров, мультисервисную оркестровку, настройку Respawn и разработку возможности тестирования во времени.
