استخدم هذه المهارة عندما يريد المستخدم أن يبدو الكلام أكثر إنسانية، أو يشبه الرفيق، أو معبرًا عاطفيًا. تتضمن المشغلات: أي ذكر لـ "قل مثلًا"، أو "تحدث مثل"، أو "تحدث مثل"، أو "صوت الرفيق"، أو "أريحني"، أو "ابتهجني"، أو "ابدو أكثر إنسانية"، أو "صوت ليلة سعيدة"، أو "صوت صباح الخير"، أو طلبات إضافة مواد مالئة أو عاطفة أو شخصية إلى الكلام الذي يتم إنشاؤه. يُستخدم أيضًا عندما يريد المستخدم تقليد صوت شخصية معينة، أو تطبيق الإعدادات المسبقة لأسلوب التحدث (ليلة سعيدة، صباح، راحة، احتفال، دردشة)، ضبط المعلمات العاطفية مثل الدفء أو الحنان، أو جعل إخراج تحويل النص إلى كلام (TTS) يبدو وكأنه شخص حقيقي يتحدث. إذا طلب المستخدم "رسالة صوتية"، أو "صوت مصاحب"، أو "صوت الشخصية"، أو أراد خطابًا يتنهد، أو يضحك، أو يتردد، أو يبدو دافئًا حقًا، فاستخدم هذه المهارة. لا تستخدم لتحويل النص إلى كلام عادي بدون شخصية أو إنشاء موسيقى أو مؤثرات صوتية أو مهام ترميز عامة لا علاقة لها بالكلام التعبيري.
