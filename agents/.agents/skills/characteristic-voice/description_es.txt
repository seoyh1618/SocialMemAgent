Utilice esta habilidad siempre que el usuario desee que el habla suene más humana, parecida a la de un compañero o emocionalmente expresiva. Los desencadenantes incluyen: cualquier mención de "decir como", "hablar como", "hablar como", "voz de compañero", "consolarme", "animarme", "suena más humano", "buenas noches", "buenos días" o solicitudes para agregar rellenos, emociones o personalidad al discurso generado. Úselo también cuando el usuario quiera imitar la voz de un personaje específico, aplicar ajustes preestablecidos de estilo de habla (buenas noches, días, comodidad, celebración, charla), ajustar parámetros emocionales como calidez o ternura, o hacer que la salida TTS se sienta como una persona real hablando. Si el usuario solicita un "mensaje de voz", "audio complementario", "voz de personaje" o quiere un discurso que suspire, ría, dude o suene genuinamente cálido, utilice esta habilidad. NO lo use para conversión de texto a voz sin personalidad, generación de música, efectos de sonido o tareas generales de codificación no relacionadas con el habla expresiva.
