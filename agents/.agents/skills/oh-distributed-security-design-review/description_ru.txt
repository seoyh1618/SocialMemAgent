Специализированные навыки проверки кода безопасности в распределенных системах OpenHarmony. Запускается, когда пользователь запрашивает «проверку реализации безопасности», «проверку кода безопасности», «проверку безопасного кода» или аналогичную проверку безопасности кода распределенной системы. Этот навык предоставляет подробное руководство по обзору 18 правил проектирования безопасности распределенного бизнеса OpenHarmony, охватывающих контроль авторизации, конечные автоматы, передачу данных, управление разрешениями, доверительные отношения и другие области безопасности. Используйте этот навык для проведения целевых проверок безопасности распределенных систем OpenHarmony помимо общих правил кибербезопасности.
