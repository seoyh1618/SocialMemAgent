Habilidad especializada para revisión de código de seguridad en sistemas distribuidos OpenHarmony. Se activa cuando el usuario solicita "revisar la implementación de seguridad", "revisión del código de seguridad", "revisión de código seguro" o una inspección de seguridad de código de sistema distribuido similar. Esta habilidad proporciona una guía de revisión detallada para 18 reglas de diseño de seguridad de negocios distribuidos de OpenHarmony, que cubren control de autorización, máquinas de estado, transmisión de datos, gestión de permisos, relaciones de confianza y otros dominios de seguridad. Utilice esta habilidad para realizar revisiones de seguridad específicas para sistemas distribuidos OpenHarmony además de las reglas generales de ciberseguridad.
