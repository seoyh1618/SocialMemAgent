Generieren Sie iterativ OpenSCAD (SCAD)-Code, rendern Sie PNG-Vorschauen, bewerten Sie das Ergebnis visuell und verfeinern Sie es in einer engen Schleife, bis das Modell übereinstimmt. Verwenden Sie diese Option, wenn der Benutzer ein 3D-Modell erstellen, ein Teil entwerfen, eine Halterung/ein Gehäuse/eine Halterung erstellen, in OpenSCAD rendern, STL exportieren oder eine beliebige Anforderung stellen möchte, die als parametrische CSG-Geometrie ausgedrückt werden kann. Erfordert die OpenSCAD MCP-Server-Tools (render_scad_png, export_scad_stl).
