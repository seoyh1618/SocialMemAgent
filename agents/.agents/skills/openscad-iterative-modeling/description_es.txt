Genere de forma iterativa código OpenSCAD (SCAD), renderice vistas previas de PNG, critique visualmente el resultado y refine en un bucle cerrado hasta que el modelo coincida. Úselo cuando el usuario solicite crear un modelo 3D, diseñar una pieza, hacer un soporte/cerramiento/montaje, renderizar en OpenSCAD, exportar STL o cualquier solicitud que pueda expresarse como geometría CSG paramétrica. Requiere las herramientas del servidor OpenSCAD MCP (render_scad_png, export_scad_stl).
