تحليل سيناريوهات الخصومة ونمذجة التهديدات لعقود Solidity الذكية. يُستخدم عند تحليل العقود من منظور المهاجم، أو تحديد نواقل الهجوم متعددة الخطوات، أو تنفيذ نماذج التهديدات. يغطي هجمات القروض السريعة، والتلاعب في Oracle، وMEV/التشغيل الأمامي، واستغلال الحوكمة، وسيناريوهات إعادة الدخول، وتجاوزات التحكم في الوصول، واستغلال المنطق الاقتصادي، ومخاطر التركيب عبر العقود. يتم تشغيل المهام التي تتضمن تحليل الخصومة، أو نمذجة التهديدات، أو سيناريوهات الهجوم، أو نواقل الهجوم، أو تحليل الاستغلال، أو مراجعة الفريق الأحمر.
