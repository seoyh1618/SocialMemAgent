Análisis de escenarios adversarios y modelado de amenazas para contratos inteligentes de Solidity. Úselo al analizar contratos desde la perspectiva de un atacante, identificar vectores de ataque de varios pasos o realizar modelos de amenazas. Cubre ataques de préstamos flash, manipulación de Oracle, MEV/front-running, exploits de gobernanza, escenarios de reentrada, omisiones de control de acceso, exploits de lógica económica y riesgos de componibilidad entre contratos. Activadores de tareas que involucran análisis adversario, modelado de amenazas, escenarios de ataque, vectores de ataque, análisis de exploits o revisión del equipo rojo.
