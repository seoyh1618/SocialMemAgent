Comprobación del estado de SEO para cualquier URL: analiza metaetiquetas, OG, JSON-LD, mapa del sitio, robots.txt, posiciones SERP y puntuaciones de 0 a 100. Úselo cuando el usuario diga "verificar SEO", "auditar esta página", "puntuación SEO", "verificar metaetiquetas" o "posición SERP". NO lo use para generar contenido de destino (use /landing-gen) o publicaciones en redes sociales (use /content-gen).
