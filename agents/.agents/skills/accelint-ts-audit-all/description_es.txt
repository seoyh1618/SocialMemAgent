Sistema integral de auditoría de archivos TypeScript. Habilidad de solo comando (sin desencadenantes naturales). Acepta rutas de archivos o directorios para realizar auditorías sistemáticas mediante las habilidades accelint-ts-testing, accelint-ts-best-practices, accelint-ts-rendimiento y accelint-ts-documentación. Mantiene el seguimiento del progreso en todas las sesiones con aprobación de cambios interactiva. Utiliza árboles de trabajo de Git aislados para permitir auditorías paralelas sin conflictos.
