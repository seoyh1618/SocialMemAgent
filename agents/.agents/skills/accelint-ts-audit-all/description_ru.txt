Комплексная система аудита файлов TypeScript. Навык, предназначенный только для командования (без естественных триггеров). Принимает путь к файлу или каталогу для систематического аудита посредством тестирования Accelint-TS, лучших практик Accelint-TS, Accelint-TS-производительности и навыков документирования Accelint-TS. Поддерживает отслеживание прогресса между сеансами с интерактивным утверждением изменений. Использует изолированные рабочие деревья git для обеспечения параллельного аудита без конфликтов.
