Sistema completo di controllo dei file TypeScript. Abilità di solo comando (nessun trigger naturale). Accetta il percorso di file o directory per eseguire controlli sistematici tramite le competenze di accelint-ts-testing, accelint-ts-best-practices, accelint-ts-performance e accelint-ts-documentation. Mantiene il monitoraggio dei progressi tra le sessioni con l'approvazione interattiva delle modifiche. Utilizza alberi di lavoro Git isolati per abilitare audit paralleli senza conflitti.
