Herramienta automatizada de exploración y visualización de datos que ofrece una solución EDA completa, desde la carga de datos hasta la generación de informes profesionales. Admite múltiples tipos de gráficos, diagnóstico de datos inteligente, evaluación de modelos y generación de informes HTML. Adecuado para proyectos de análisis de datos en salud, finanzas, comercio electrónico y otros campos.
