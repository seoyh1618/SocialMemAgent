Komponentenreferenz für das Wix Design System. Verwenden Sie diese Option, wenn Sie eine Benutzeroberfläche mit @wix/design-system erstellen, Komponenten auswählen oder Requisiten und Beispiele überprüfen. Löst auf „Welche Komponente“, „Wie mache ich“, „WDS“, „Zeige mir Requisiten“ oder Komponentennamen wie Schaltfläche, Karte, Modal, Box, Text aus.
