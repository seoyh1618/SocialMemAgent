Mitigue la complejidad incidental del código cuando el flujo de control está enredado, el anidamiento es profundo, los nombres son difíciles de analizar o el razonamiento requiere saltos entre archivos. Úselo cuando una revisión se estanca en la legibilidad, necesita un plan de refactorización de análisis primero antes de las ediciones, o desea veredictos esenciales versus incidentales, pasos de simplificación clasificados, un pequeño bosquejo estructural y una evaluación TRACE (solo análisis; sin ediciones).
