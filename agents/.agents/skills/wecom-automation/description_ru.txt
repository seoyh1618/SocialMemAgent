Прямая автоматизация личного кабинета Enterprise WeChat. Создан на базе Wechaty для получения сообщений WeCom, автоматического принятия запросов на добавление в друзья, предоставления вопросов и ответов из базы знаний и отправки оповещений о передаче управления человеком. Идеально подходит для персональных ботов WeCom и помощников по автоматизации.
