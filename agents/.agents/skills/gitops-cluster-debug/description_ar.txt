تصحيح الأخطاء واستكشاف الأخطاء وإصلاحها وتشخيص وفحص والتحقيق في خطوط أنابيب GitOps على مجموعات Kubernetes المباشرة. استخدم هذه المهارة عندما يسأل المستخدمون عن حالة Flux، أو فشل HelmRelease، أو أخطاء التخصيص، أو مشكلات التسوية، أو سجلات pod، أو مشكلات وحدة التحكم، أو تصحيح أخطاء المجموعة - حتى لو لم يقولوا "تصحيح الأخطاء" صراحةً.
