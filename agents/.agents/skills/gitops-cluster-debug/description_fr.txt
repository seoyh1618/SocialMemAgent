Déboguez, dépannez, diagnostiquez, inspectez et étudiez les pipelines GitOps sur les clusters Kubernetes en direct. Utilisez cette compétence lorsque les utilisateurs posent des questions sur l'état de Flux, les échecs de HelmRelease, les erreurs de personnalisation, les problèmes de réconciliation, les journaux de pod, les problèmes de contrôleur ou le débogage du cluster, même s'ils ne disent pas explicitement « débogage ».
