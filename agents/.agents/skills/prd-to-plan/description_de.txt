Verwandeln Sie einen PRD in einen mehrphasigen Implementierungsplan mithilfe vertikaler Markierungsscheiben, die als lokale Markdown-Datei in ./plans/ gespeichert werden. Verwenden Sie diese Option, wenn der Benutzer eine PRD aufschlüsseln, einen Implementierungsplan erstellen, Phasen aus einer PRD planen oder „Markierungsgeschosse“ erwähnen möchte.
