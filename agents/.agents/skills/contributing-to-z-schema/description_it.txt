Guida i contributori attraverso la codebase dello schema z, il flusso di lavoro PR e le attività di sviluppo comuni. Utilizzare quando l'utente desidera contribuire a z-schema, aggiungere una nuova funzionalità o parola chiave, aggiungere un codice di errore, aggiungere un validatore di formato, modificare opzioni, scrivere test, eseguire la suite di test, correggere un test non riuscito, comprendere la pipeline di convalida, esplorare l'architettura del codice sorgente o inviare una richiesta pull. Da utilizzare anche quando qualcuno menziona il contributo, i PR, il codice sorgente z-schema o l'integrazione di JSON Schema Test Suite.
