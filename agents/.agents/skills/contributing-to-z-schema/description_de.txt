Führt Mitwirkende durch die Z-Schema-Codebasis, den PR-Workflow und allgemeine Entwicklungsaufgaben. Verwenden Sie diese Option, wenn der Benutzer zum Z-Schema beitragen, eine neue Funktion oder ein neues Schlüsselwort hinzufügen, einen Fehlercode hinzufügen, einen Formatvalidator hinzufügen, Optionen ändern, Tests schreiben, die Testsuite ausführen, einen fehlgeschlagenen Test beheben, die Validierungspipeline verstehen, durch die Quellcodearchitektur navigieren oder eine Pull-Anfrage senden möchte. Verwenden Sie es auch, wenn jemand Beiträge, PRs, den Z-Schema-Quellcode oder die JSON Schema Test Suite-Integration erwähnt.
