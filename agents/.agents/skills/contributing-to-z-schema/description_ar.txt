يرشد المساهمين من خلال قاعدة تعليمات Z-Schema، وسير عمل العلاقات العامة، ومهام التطوير المشتركة. يُستخدم عندما يريد المستخدم المساهمة في مخطط z، أو إضافة ميزة أو كلمة رئيسية جديدة، أو إضافة رمز خطأ، أو إضافة مدقق تنسيق، أو تعديل الخيارات، أو كتابة الاختبارات، أو تشغيل مجموعة الاختبار، أو إصلاح اختبار فاشل، أو فهم مسار التحقق من الصحة، أو التنقل في بنية التعليمات البرمجية المصدر، أو إرسال طلب سحب. استخدمه أيضًا عندما يذكر شخص ما المساهمة أو العلاقات العامة أو كود مصدر مخطط z أو تكامل JSON Schema Test Suite.
