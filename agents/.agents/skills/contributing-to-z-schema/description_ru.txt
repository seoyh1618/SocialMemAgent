Помогает участникам разобраться с кодовой базой z-схемы, рабочим процессом PR и общими задачами разработки. Используйте, когда пользователь хочет внести свой вклад в z-схему, добавить новую функцию или ключевое слово, добавить код ошибки, добавить средство проверки формата, изменить параметры, написать тесты, запустить набор тестов, исправить неудачный тест, понять конвейер проверки, перемещаться по архитектуре исходного кода или отправить запрос на включение. Также используйте, когда кто-то упоминает вклад, PR, исходный код z-схемы или интеграцию JSON Schema Test Suite.
