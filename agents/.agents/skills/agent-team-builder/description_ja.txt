マルチエージェント チームを設計して実行し、反復的なコラボレーション、品質ゲート、改良ループを通じて複雑なタスクを達成します。ユーザーが、順番または並行して作業する専門エージェントの恩恵を受ける重要なタスクを達成したい場合に使用します。記事の執筆、ソフトウェア機能の構築、調査の実施、マーケティング キャンペーンの作成、システムの設計、教育コンテンツの作成、または調査 → 計画 → 実行 → レビュー → 改良の各段階に自然に分解されるタスク。 「...のためにチームを構築してください」、「...のためにエージェントを使用します」、「...のためにエージェントを調整します」などのフレーズ、またはタスクが十分に複雑で単一のエージェントがスペシャリストに分解することで恩恵を受ける場合にトリガーされます。
