Entwirft und führt Multi-Agenten-Teams aus, um komplexe Aufgaben durch iterative Zusammenarbeit, Quality Gates und Verfeinerungsschleifen zu erfüllen. Verwenden Sie diese Option, wenn ein Benutzer eine nicht triviale Aufgabe ausführen möchte, die von der Arbeit spezialisierter Agenten nacheinander oder parallel profitieren würde – z. B. Schreiben eines Artikels, Erstellen einer Softwarefunktion, Durchführen von Recherchen, Produzieren einer Marketingkampagne, Entwerfen eines Systems, Erstellen von Bildungsinhalten oder jede andere Aufgabe, die sich natürlich in die Phasen Recherche → Planung → Ausführung → Überprüfung → Verfeinerung zerlegen lässt. Wird bei Formulierungen wie „Bilden Sie mir ein Team, um ...“, „Agenten einsetzen, um ...“, „Agenten orchestrieren für …“ ausgelöst oder wenn eine Aufgabe so komplex ist, dass ein einzelner Agent von der Zerlegung in Spezialisten profitieren würde.
