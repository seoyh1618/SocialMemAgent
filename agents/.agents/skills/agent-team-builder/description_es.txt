Diseña y ejecuta equipos de múltiples agentes para realizar tareas complejas a través de colaboración iterativa, puertas de calidad y bucles de refinamiento. Úselo cuando un usuario quiera realizar cualquier tarea no trivial que se beneficiaría de agentes especializados trabajando en secuencia o en paralelo, por ejemplo. escribir un artículo, crear una función de software, realizar una investigación, producir una campaña de marketing, diseñar un sistema, crear contenido educativo o cualquier tarea que naturalmente se descomponga en etapas de investigación → planificación → ejecución → revisión → refinamiento. Se desencadenan frases como "constrúyeme un equipo para...", "usar agentes para...", "orquestar agentes para...", o cuando una tarea es lo suficientemente compleja como para que un solo agente se beneficiaría de su descomposición en especialistas.
