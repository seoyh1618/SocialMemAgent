Разрабатывает и управляет многоагентными командами для выполнения сложных задач посредством итеративного сотрудничества, контроля качества и циклов уточнения. Используйте, когда пользователь хочет выполнить какую-либо нетривиальную задачу, для которой было бы полезно, чтобы специализированные агенты работали последовательно или параллельно - например. написание статьи, создание функции программного обеспечения, проведение исследования, проведение маркетинговой кампании, проектирование системы, создание образовательного контента или любая задача, которая естественным образом распадается на этапы исследования → планирования → выполнения → анализа → доработки. Срабатывает при таких фразах, как «соберите мне команду, чтобы…», «используйте агентов, чтобы…», «организуйте агентов для…», или когда задача достаточно сложна, чтобы один агент мог получить выгоду от разложения на специалистов.
