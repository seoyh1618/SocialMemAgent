Una habilidad de diálogo metacognitivo para una autoexploración profunda. Ayuda a estructurar pensamientos dispersos, descubrir patrones personales y extraer ideas para escribir. Utiliza preguntas basadas en hipótesis (no sólo "¿por qué?") para descubrir valores ocultos, direcciones de crecimiento y puntos ciegos. Ideal para la autorreflexión, indicaciones para llevar un diario y la ideación de artículos. Una habilidad conversacional que ayuda al pensamiento metacognitivo. Organiza pensamientos dispersos, profundiza la autocomprensión y descubre ideas adecuadas para escribir. Úselo cuando el usuario diga cosas como "Ayúdame a organizar mis pensamientos", "Quiero entenderme a mí mismo" o "Estoy planeando escribir un artículo".
