Навык метакогнитивного диалога для глубокого самоисследования. Помогает структурировать разрозненные мысли, обнаруживать личные закономерности и извлекать идеи для написания. Использует вопросы, основанные на гипотезах (а не только «почему?»), чтобы выявить скрытые ценности, направления роста и «слепые пятна». Идеально подходит для самоанализа, подсказок для ведения дневника и разработки идей для статей. Разговорный навык, который помогает метакогнитивному мышлению. Он организует разрозненные мысли, углубляет самопонимание и открывает идеи, пригодные для письма. Используйте его, когда пользователь говорит что-то вроде «Помогите мне организовать мои мысли», «Я хочу разобраться в себе» или «Я планирую написать статью».
