Cette compétence guide l'écriture d'un nouveau code Ruby suivant la syntaxe Ruby 3.x moderne, les 4 règles pour les développeurs de Sandi Metz et les meilleures pratiques idiomatiques de Ruby. À utiliser lors de la création de nouveaux fichiers Ruby, de l'écriture de méthodes Ruby ou de la refactorisation du code Ruby pour garantir le respect des normes de clarté, de simplicité et de maintenabilité.
