Questa abilità guida la scrittura del nuovo codice Ruby seguendo la moderna sintassi Ruby 3.x, le 4 regole per gli sviluppatori di Sandi Metz e le migliori pratiche idiomatiche di Ruby. Da utilizzare durante la creazione di nuovi file Ruby, la scrittura di metodi Ruby o il refactoring del codice Ruby per garantire il rispetto degli standard di chiarezza, semplicità e manutenibilità.
