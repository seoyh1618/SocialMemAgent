قم بالتحميل بشكل استباقي عندما تتضمن المهمة ميزات الذكاء الاصطناعي أو LLM أو التعلم الآلي. يُستخدم عندما يقول المستخدم "أضف دردشة AI"، أو "نفذ استجابات البث"، أو "أنشئ خط أنابيب RAG"، أو "أضف التضمينات"، أو "دمج OpenAI". يغطي واجهات الدردشة، والبث باستخدام Vercel AI SDK، والتوليد المعزز للاسترجاع، والبحث عن المتجهات، وخطوط أنابيب التضمين، واستدعاء الأدوات/الوظائف، وتجريد الموفر لنماذج OpenAI، والإنسانية، والمحلية.
