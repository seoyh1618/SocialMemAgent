Da utilizzare quando l'utente dice "crealo", "inizia a costruire", "esegui il piano", "implementa attività", "spediscilo" o fa riferimento a un ID traccia. NON utilizzare per la pianificazione (use /plan) o per impalcature (use /scaffold).
