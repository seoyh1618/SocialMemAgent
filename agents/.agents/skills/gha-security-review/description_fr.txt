Examen de la sécurité de GitHub Actions pour les vulnérabilités d’exploitation des flux de travail. À utiliser lorsqu'on vous demande de « examiner les actions GitHub », « d'auditer les workflows », « vérifier la sécurité CI », « la sécurité GHA », « examiner la sécurité des workflows » ou examiner .github/workflows/ pour les requêtes pwn, l'injection d'expression, le vol d'informations d'identification et les attaques de la chaîne d'approvisionnement. Axé sur l’exploitation avec des scénarios PoC concrets.
