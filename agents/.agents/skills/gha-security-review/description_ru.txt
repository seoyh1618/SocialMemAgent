Проверка безопасности GitHub Actions на наличие уязвимостей, связанных с эксплуатацией рабочих процессов. Используйте, когда вас просят «проверить действия GitHub», «аудит рабочих процессов», «проверить безопасность CI», «безопасность GHA», «проверить безопасность рабочих процессов» или просмотреть .github/workflows/ на предмет запросов pwn, внедрения выражений, кражи учетных данных и атак в цепочке поставок. Ориентированы на эксплуатацию с конкретными сценариями PoC.
