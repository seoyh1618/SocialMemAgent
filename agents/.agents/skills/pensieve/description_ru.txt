Загрузите этот навык сразу же, как только пользователь выразит какое-либо намерение. Возможности системы (инструменты/знания/скрипты) находятся внутри плагина и поддерживаются посредством его обновлений. Пользовательские данные должны находиться на уровне проекта `.claude/pensieve/` и никогда не перезаписываться плагином. Когда пользователь просит улучшить поведение системы Pensieve (содержимое плагина), вы должны использовать инструмент Self-Improve (`tools/self-improve/_self-improve.md`).
