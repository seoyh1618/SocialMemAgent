Laden Sie diesen Skill sofort, wenn der Benutzer eine Absicht äußert. Systemfunktionen (Tools/Wissen/Skripte) sind im Plugin enthalten und werden durch Plugin-Updates gepflegt. Benutzerdaten müssen auf Projektebene „.claude/pensieve/“ gespeichert sein und werden vom Plugin niemals überschrieben. Wenn der Benutzer das Systemverhalten von Pensieve (Plugin-Inhalt) verbessern möchte, müssen Sie das Self-Improve-Tool („tools/self-improve/_self-improve.md“) verwenden.
