Fournissez des bases de données Postgres temporaires instantanées via Claimable Postgres by Neon (pg.new) sans connexion, inscription ou carte de crédit. À utiliser lorsque les utilisateurs demandent un environnement Postgres rapide, une DATABASE_URL jetable pour le prototypage/les tests, ou "donnez-moi simplement une base de données maintenant". Les déclencheurs incluent : "postgres rapides", "postgres temporaires", "pas de base de données d'inscription", "pas de base de données de cartes de crédit", "instant DATABASE_URL".
