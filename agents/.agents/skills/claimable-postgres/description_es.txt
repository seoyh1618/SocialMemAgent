Aprovisione bases de datos temporales instantáneas de Postgres a través de Claimable Postgres by Neon (pág. nuevo) sin necesidad de iniciar sesión, registrarse ni usar tarjeta de crédito. Úselo cuando los usuarios soliciten un entorno Postgres rápido, un DATABASE_URL desechable para creación de prototipos/pruebas, o "simplemente dame una base de datos ahora". Los activadores incluyen: "postgres rápido", "postgres temporal", "sin base de datos de registro", "sin base de datos de tarjetas de crédito", "DATABASE_URL instantánea".
