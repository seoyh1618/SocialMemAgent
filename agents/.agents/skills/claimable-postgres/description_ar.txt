توفير قواعد بيانات Postgres المؤقتة الفورية عبر Postgres القابلة للمطالبة بواسطة Neon (pg.new) بدون تسجيل الدخول أو الاشتراك أو بطاقة الائتمان. يُستخدم عندما يطلب المستخدمون بيئة Postgres سريعة، أو عنوان DATABASE_URL سريع للنماذج الأولية/الاختبارات، أو "فقط أعطني قاعدة بيانات الآن". تتضمن المشغلات: "postgres سريعة"، و"postgres مؤقتة"، و"لا توجد قاعدة بيانات للاشتراك"، و"لا توجد قاعدة بيانات لبطاقات الائتمان"، و"DATABASE_URL الفوري".
