Stellen Sie sofortige temporäre Postgres-Datenbanken über Claimable Postgres by Neon (pg.new) bereit, ohne dass Sie sich anmelden, registrieren oder eine Kreditkarte benötigen. Verwenden Sie es, wenn Benutzer eine schnelle Postgres-Umgebung, eine wegwerfbare DATABASE_URL für Prototyping/Tests oder „Geben Sie mir jetzt einfach eine Datenbank“ wünschen. Zu den Auslösern gehören: „schnelles Postgres“, „temporäres Postgres“, „keine Anmeldedatenbank“, „keine Kreditkartendatenbank“, „instant DATABASE_URL“.
