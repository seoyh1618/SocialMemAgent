Une compétence pour générer et éditer des images à l'aide de Nano Banana 2 (API Google Gemini). Utilisé lorsque des requêtes telles que « générer une image », « créer une illustration », « dessiner une image de ○○ », « créer une image », « modifier cette image », « créer ○○ basé sur cette image », « générer une image », « créer une image », « modifier cette image », etc. Prend en charge la génération à partir de texte, la génération à partir d’images de référence, l’édition d’images et la génération d’images reflétant les dernières informations via Google Search Grounding. Peut également traiter des demandes telles que « dernières ○○ », « refléter les tendances », « informations en temps réel », etc.
