Навык создания и редактирования изображений с помощью Nano Banana 2 (API Google Gemini). Используется при таких запросах, как «создать изображение», «создать иллюстрацию», «нарисовать изображение ○○», «создать изображение», «отредактировать это изображение», «сделать ○○ на основе этого изображения», «создать изображение», «создать изображение», «отредактировать это изображение» и т. д. Поддерживает создание из текста, создание из эталонных изображений, редактирование изображений и создание изображений, отражающих последнюю информацию с помощью Google Search Grounding. Также может обрабатывать такие запросы, как «последние ○○», «отражать тенденции», «информация в реальном времени» и т. д.
