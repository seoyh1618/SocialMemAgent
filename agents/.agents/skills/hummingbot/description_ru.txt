Навыки агента, которые точно воспроизводят команды интерфейса командной строки Hummingbot (подключение, балансировка, создание, запуск, остановка, статус, история) через API Hummingbot. V1 фокусируется на основных торговых процессах. Для стратегий DEX/LP на Solana вместо этого используйте lp-agent.
