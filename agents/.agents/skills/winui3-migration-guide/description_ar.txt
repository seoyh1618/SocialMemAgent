مرجع ترحيل UWP-to-WinUI 3. يقوم بتعيين واجهات برمجة تطبيقات UWP القديمة لتصحيح مكافئات Windows App SDK مع مقتطفات التعليمات البرمجية قبل/بعد. يغطي تغييرات مساحة الاسم، والترابط (CoreDispatcher إلى DispatcherQueue)، والنوافذ (CoreWindow إلى AppWindow)، ومربعات الحوار، والمنتقيات، والمشاركة، والطباعة، ومهام الخلفية، وأخطاء إنشاء كود Copilot الأكثر شيوعًا.
