Référence de migration UWP vers WinUI 3. Mappe les anciennes API UWP pour corriger les équivalents du SDK d’application Windows avec des extraits de code avant/après. Couvre les modifications d'espace de noms, le threading (CoreDispatcher à DispatcherQueue), le fenêtrage (CoreWindow à AppWindow), les boîtes de dialogue, les sélecteurs, le partage, l'impression, les tâches en arrière-plan et les erreurs de génération de code Copilot les plus courantes.
