Riferimento per la migrazione da UWP a WinUI 3. Mappa le API UWP legacy per correggere gli equivalenti dell'SDK dell'app Windows con frammenti di codice prima/dopo. Copre le modifiche allo spazio dei nomi, il threading (da CoreDispatcher a DispatcherQueue), il windowing (da CoreWindow a AppWindow), le finestre di dialogo, i selettori, la condivisione, la stampa, le attività in background e gli errori più comuni di generazione del codice Copilot.
