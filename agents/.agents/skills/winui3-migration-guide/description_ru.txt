Справочник по переходу с UWP на WinUI 3. Сопоставляет устаревшие API-интерфейсы UWP для исправления эквивалентов Windows App SDK с помощью фрагментов кода до и после. Описывает изменения пространства имен, многопоточность (CoreDispatcher to DispatcherQueue), работу с окнами (CoreWindow to AppWindow), диалоговые окна, средства выбора, совместное использование, печать, фоновые задачи и наиболее распространенные ошибки генерации кода Copilot.
