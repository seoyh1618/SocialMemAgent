Linee guida e flussi di lavoro per la creazione, la revisione e la protezione delle app desktop Tauri (frontend React/Vite). Da utilizzare durante l'implementazione di comandi IPC Tauri, accesso API nativo, integrazione di file system o finestre di dialogo, comportamento di finestre/menu/vassoi, autorizzazioni/liste consentite/CSP, ciclo di vita delle app, aggiornamento automatico o attività di creazione di pacchetti/rilasci.
