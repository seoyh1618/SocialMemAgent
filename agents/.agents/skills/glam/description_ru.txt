Управление хранилищем Solana через протокол GLAM. Триггеры: glam, glam-cli, glam-sdk, создание/управление хранилищем, токенизированное хранилище, общий класс, хранилище DeFi, казначейство, управление активами, контроль доступа, права делегирования, обмен Jupiter, бессрочные контракты Drift/спот/хранилища, кредитование/заимствование/хранилища/фермы Kamino, стейкинг (Marinade/native/SPL/Sanctum/LST), кросс-чейн USDC (CCTP), временная блокировка, подписка/погашение, цены NAV, передача токенов. Поддерживает CLI и TypeScript SDK.
