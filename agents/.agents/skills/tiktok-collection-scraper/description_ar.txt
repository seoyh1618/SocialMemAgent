قم بكشط مجلدات مجموعة مستخدمي TikTok وروابط الفيديو الخاصة بهم مع البيانات التعريفية الكاملة (عمليات التشغيل والإعجابات والتعليقات والمشاركات). يُستخدم عندما يطلب المستخدم الحصول على مجموعات TikTok أو المفضلة أو مقاطع الفيديو المحفوظة أو الإشارات المرجعية أو محتويات مجلد المجموعة أو استخراجها أو جلبها أو استخراجها أو تنزيلها لأي حساب TikTok. يقبل أي معرف مستخدم - اسم المستخدم، @username، أو عنوان URL للملف الشخصي، أو عنوان URL للفيديو، أو الرابط القصير (vm.tiktok.com)، أو user_id، أو secUid. لا يوجد أي اعتماد على واجهة برمجة التطبيقات الخارجية، ولا حاجة إلى خدمة مدفوعة، ويتطلب فقط curl_cffi. يعمل بدون ملفات تعريف الارتباط لتسجيل الدخول للمجموعات العامة (تغطية تصل إلى 80%). استخدمه أيضًا لتحليل منافسي TikTok، أو البحث عن المحتوى، أو اكتشاف الفيديو الفيروسي، أو إنشاء قواعد بيانات الفيديو.
