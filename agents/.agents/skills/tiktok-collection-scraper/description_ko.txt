TikTok 사용자 컬렉션 폴더와 해당 비디오 링크를 전체 메타데이터(재생, 좋아요, 댓글, 공유)로 스크랩합니다. 사용자가 TikTok 계정에 대한 TikTok 컬렉션, 즐겨찾기, 저장된 비디오, 북마크 또는 컬렉션 폴더 콘텐츠를 가져오고, 긁고, 가져오고, 추출하거나 다운로드하도록 요청할 때 사용합니다. 사용자 이름, @username, 프로필 URL, 비디오 URL, 짧은 링크(vm.tiktok.com), user_id 또는 secUid 등 모든 사용자 식별자를 허용합니다. 외부 API 종속성이 없으며 유료 서비스가 필요하지 않으며 단지 컬_cffi만 필요합니다. 공개 컬렉션의 경우 로그인 쿠키 없이 작동합니다(~80% 적용 범위). 또한 TikTok 경쟁사 분석, 콘텐츠 연구, 바이러스성 비디오 검색 또는 비디오 데이터베이스 구축에도 사용됩니다.
