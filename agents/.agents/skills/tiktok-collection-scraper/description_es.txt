Elimine las carpetas de colección de usuarios de TikTok y sus enlaces de videos con metadatos completos (reproducciones, me gusta, comentarios, recursos compartidos). Úselo cuando el usuario solicite obtener, extraer, extraer o descargar colecciones de TikTok, favoritos, videos guardados, marcadores o contenidos de carpetas de colecciones para cualquier cuenta de TikTok. Acepta cualquier identificador de usuario: nombre de usuario, @nombre de usuario, URL de perfil, URL de video, enlace corto (vm.tiktok.com), user_id o secUid. Cero dependencia de API externa, no se necesita ningún servicio pago, solo requiere curl_cffi. Funciona sin cookies de inicio de sesión para colecciones públicas (~80% de cobertura). Úselo también para análisis de la competencia de TikTok, investigación de contenido, descubrimiento de videos virales o creación de bases de datos de videos.
