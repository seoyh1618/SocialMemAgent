TikTok ユーザーのコレクション フォルダーとそのビデオ リンクを完全なメタデータ (再生、いいね、コメント、共有) とともにスクレイピングします。ユーザーが TikTok アカウントの TikTok コレクション、お気に入り、保存されたビデオ、ブックマーク、またはコレクション フォルダーの内容を取得、スクレイピング、フェッチ、抽出、またはダウンロードするように要求する場合に使用します。任意のユーザー識別子 (ユーザー名、@ユーザー名、プロフィール URL、ビデオ URL、短縮リンク (vm.tiktok.com)、user_id、または secUid) を受け入れます。外部 API への依存性はなく、有料サービスは不要で、curl_cffi のみが必要です。パブリック コレクションの場合はログイン Cookie なしで動作します (カバー率約 80%)。 TikTok競合他社の分析、コンテンツ調査、バイラルビデオの発見、またはビデオデータベースの構築にも使用できます。
