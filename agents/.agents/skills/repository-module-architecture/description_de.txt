Leitfaden zur Architektur des Repository Codebase Management Module, der den Git/SVN-Repository-Zugriff, die Webhook-Konfiguration, die Repository-Autorisierung und die Trigger-Verwaltung behandelt. Wird verwendet, wenn Benutzer Repository-Funktionen entwickeln, Webhooks konfigurieren, die Repository-Autorisierung verwalten oder Triggerlogik implementieren.
