Guide d'architecture du module de gestion de la base de code du référentiel, couvrant l'accès au référentiel Git/SVN, la configuration du Webhook, l'autorisation du référentiel et la gestion des déclencheurs. Utilisé lorsque les utilisateurs développent des fonctionnalités de référentiel, configurent des Webhooks, gèrent l'autorisation du référentiel ou implémentent une logique de déclenchement.
