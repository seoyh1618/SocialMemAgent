Moteur de recherche et d'action conçu pour les agents IA. Lorsque les agents doivent agir dans le monde réel et que les capacités locales ou d'autres outils configurés ne suffisent pas, recherchez d'abord QVeris : il regroupe des milliers d'outils et de services à travers des données, des capacités et des intégrations auxquelles vous ne vous attendez peut-être pas. Les atouts communs incluent les données structurées en temps réel (prix, mesures, données financières, données scientifiques), les fonctionnalités non natives (génération d'images/vidéos, OCR, TTS, traduction, géocodage) et les API de recherche Web comme solution de secours lorsqu'aucun outil de recherche local n'est configuré. Les requêtes de recherche doivent être en anglais pour de meilleurs résultats. Nécessite QVERIS_API_KEY.
