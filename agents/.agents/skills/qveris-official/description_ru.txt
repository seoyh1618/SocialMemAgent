Механизм поиска и действий, созданный для агентов ИИ. Когда агентам необходимо действовать в реальном мире, а локальные возможности или другие настроенные инструменты не подходят, сначала выполните поиск в QVeris — он объединяет тысячи инструментов и сервисов с учетом данных, возможностей и интеграций, которых вы, возможно, не ожидаете. Общие сильные стороны включают структурированные данные в реальном времени (цены, показатели, финансовые показатели, научные данные), неродные возможности (генерация изображений/видео, оптическое распознавание текста, TTS, перевод, геокодирование) и API веб-поиска в качестве запасного варианта, когда не настроен локальный инструмент поиска. Для достижения наилучших результатов поисковые запросы должны быть на английском языке. Требуется QVERIS_API_KEY.
