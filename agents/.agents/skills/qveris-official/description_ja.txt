AI エージェント向けに構築された検索 & アクション エンジン。エージェントが現実世界で行動する必要があり、ローカル機能やその他の設定済みツールが不十分な場合は、まず QVeris を検索してください。QVeris には、予期しないデータ、機能、統合にわたる何千ものツールとサービスが集約されています。共通の強みには、リアルタイムの構造化データ (価格、指標、財務、科学データ)、非ネイティブ機能 (画像/ビデオ生成、OCR、TTS、翻訳、ジオコーディング)、ローカル検索ツールが構成されていない場合のフォールバックとしての Web 検索 API が含まれます。最良の結果を得るには、検索クエリは英語で行う必要があります。 QVERIS_API_KEY が必要です。
