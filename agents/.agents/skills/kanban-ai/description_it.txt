Gestisci una bacheca Kanban basata su Markdown utilizzando i file delle carte in una directory kanban/ (incluso kanban/archived/ per le carte completate). Da utilizzare quando l'utente chiede di creare, spostare, visualizzare, elencare o gestire attività o schede su una bacheca kanban o quando si tengono traccia degli elementi di lavoro attraverso stati come arretrati, cose da fare, in corso, completati o archiviati.
