قم بإعداد خطة مقاييس PostHog باستخدام مسار تحويل الأحداث ومعايير مؤشرات الأداء الرئيسية وحدود قرار القتل/التكرار/القياس. يُستخدم عندما يقول المستخدم "إعداد المقاييس"، أو "تتبع مؤشرات الأداء الرئيسية"، أو "أحداث PostHog"، أو "تحليل مسار التحويل"، أو "متى يجب التوقف أو القياس"، أو "مقاييس النجاح". لا تستخدمه لمقاييس تحسين محركات البحث (استخدم /seo-audit).
