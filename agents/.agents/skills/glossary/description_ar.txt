قم بإنشاء وإدارة مسرد لغوي واسع الانتشار على مستوى المشروع - وهو عبارة عن مفردات مشتركة يستخدمها المطورون ومديرو المنتجات ومصممو تجربة المستخدم ومهندسو ضمان الجودة ومحللو الأعمال باستمرار. استخدم هذه المهارة عندما يذكر المستخدم "المسرد"، أو "المصطلحات"، أو "اللغة واسعة الانتشار"، أو "ماذا يعني X"، أو "تحديد مصطلح"، أو "إضافة مصطلح"، أو "مصطلح البحث"، أو "سلامة المسرد"، أو "المصطلحات القديمة"، أو يريد إنشاء مفردات مشروع مشتركة أو الاستعلام عنها أو الحفاظ عليها. استخدمه أيضًا عند كتابة التعليمات البرمجية أو الاختبارات أو المواصفات والمصطلحات المتسقة المهمة - المسرد هو المصدر الوحيد للحقيقة لما تعنيه المصطلحات في هذا المشروع.
