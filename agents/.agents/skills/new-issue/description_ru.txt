Создавайте проблемы GitHub с помощью обнаружения дубликатов и анализа базы кода. Используйте, когда вы обнаружили ошибку, улучшение или проблему, которая не была в исходной области действия/спецификации во время разработки. Запускается по таким фразам, как «создать проблему», «заявить об ошибке», «это следует отследить», «мы должны исправить это позже», «проблема, выходящая за рамки», «обнаружена проблема», «обнаружена ошибка».
