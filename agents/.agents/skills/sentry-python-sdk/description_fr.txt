Configuration complète du SDK Sentry pour Python. À utiliser lorsqu'on vous demande d'« ajouter Sentry à Python », « d'installer Sentry-sdk », de « configurer Sentry dans Python » ou de configurer la surveillance des erreurs, le traçage, le profilage, la journalisation, les métriques, les crons ou la surveillance de l'IA pour les applications Python. Prend en charge Django, Flask, FastAPI, Celery, Starlette, AIOHTTP, Tornado et plus encore.
