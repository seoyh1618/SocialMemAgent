Cette compétence doit être utilisée lorsque l'utilisateur souhaite produire sa déclaration de revenus par voie électronique via le 確定申告書等作成コーナー (NTA Tax Return Preparation Corner) à l'aide de Claude dans Chrome, du sous-agent Antigravity Browser ou de Playwright CLI (repli). Il guide la saisie par navigateur des données fiscales calculées. Les expressions déclencheurs incluent : "e-Tax提出", "電子申告", "e-Taxで申告", "作成コーナーに入力", "確定申告書等作成コーナー", "作成コーナー", "申告書を提出".
