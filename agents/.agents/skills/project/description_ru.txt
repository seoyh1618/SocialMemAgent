Клонируйте и отслеживайте внешние репозитории. Используйте, когда пользователь делится URL-адресом GitHub для изучения или разработки или говорит «найти репозиторий», «найти репозиторий», «где находится [проект]». Действия - изучить (клонировать для изучения), инкубировать (клонировать для разработки), искать/найти (поиск репозиториев), список (показать отслеживаемое).
