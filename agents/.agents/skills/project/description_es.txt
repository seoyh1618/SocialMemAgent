Clona y rastrea repositorios externos. Úselo cuando el usuario comparte la URL de GitHub para estudiar o desarrollar, o dice "buscar repositorios", "buscar repositorio", "dónde está [proyecto]". Acciones: aprender (clonar para estudiar), incubar (clonar para desarrollo), buscar/encontrar (buscar repositorios), enumerar (mostrar seguimiento).
