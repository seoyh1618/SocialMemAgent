Clonez et suivez les dépôts externes. À utiliser lorsque l'utilisateur partage l'URL GitHub pour étudier ou développer, ou dit « rechercher des dépôts », « trouver un dépôt », « où se trouve [projet] ». Actions - apprendre (cloner pour étude), incuber (cloner pour développement), rechercher/trouver (rechercher dans les dépôts), lister (afficher le suivi).
