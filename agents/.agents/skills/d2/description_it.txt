Da utilizzare durante la scrittura, la generazione, la revisione o il miglioramento del codice del diagramma D2. Si attiva quando l'utente chiede di creare diagrammi di architettura, diagrammi di flusso, diagrammi di sequenza, diagrammi ER, diagrammi di classi o qualsiasi diagramma utilizzando la sintassi D2. Si attiva anche quando si lavora con file .d2 o quando è disponibile lo strumento MCP d2_render.
