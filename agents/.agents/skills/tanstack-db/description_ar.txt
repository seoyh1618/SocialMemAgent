قاعدة بيانات TanStack DB التفاعلية من جانب العميل مع استعلامات حية وطفرات متفائلة. يُستخدم عند إنشاء واجهات مستخدم تفاعلية باستخدام البيانات المحلية أولاً، أو محركات المزامنة، أو التحديثات الإيجابية. يستخدم لـ tanstack-db، والاستعلامات المباشرة، والطفرات المتفائلة، ومحرك المزامنة، وقاعدة البيانات التفاعلية، والمحلية أولاً.
