Esegue il debug di applicazioni Windows e Linux/macOS (native, .NET/CLR, modalità mista) con WinDbg MCP (crash dump, !analyze, !syncblk, !dlk, !runaway, !dumpheap, !gcroot, BSOD), dotnet-dump, lldb con SOS, createump e diagnostica del contenitore (Docker, Kubernetes). Diagnosi di blocchi/deadlock, valutazione elevata della CPU, analisi delle perdite di memoria, debug del kernel e monitoraggio dotnet per la produzione. Copre 17 aree tematiche. Non utilizzare per la profilazione di routine di .NET SDK, la progettazione di benchmark o il debug dei test CI.
