Отладка приложений Windows и Linux/macOS (собственные, .NET/CLR, смешанный режим) с помощью WinDbg MCP (аварийные дампы, !analyze, !syncblk, !dlk, !runaway, !dumpheap, !gcroot, BSOD), dotnet-dump, lldb с SOS, созданный дамп и диагностика контейнеров (Docker, Kubernetes). Диагностика зависаний/взаимных блокировок, высокая сортировка ЦП, расследование утечек памяти, отладка ядра и dotnet-monitor для рабочей среды. Охватывает 17 тематических областей. Не используйте для рутинного профилирования .NET SDK, разработки тестов или отладки CI-тестов.
