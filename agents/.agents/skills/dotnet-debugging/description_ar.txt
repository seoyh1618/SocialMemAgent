يصحح أخطاء تطبيقات Windows وLinux/macOS (الأصلي، .NET/CLR، الوضع المختلط) باستخدام WinDbg MCP (مخلفات الأعطال، !analyze، !syncblk، !dlk، !runaway، !dumpheap، !gcroot، BSOD)، dotnet-dump، lldb مع SOS، createump، وتشخيصات الحاويات (Docker، Kubernetes). تشخيص حالة التوقف/التوقف التام، وفرز وحدة المعالجة المركزية بشكل عالي، والتحقيق في تسرب الذاكرة، وتصحيح أخطاء kernel، ومراقبة dotnet للإنتاج. يمتد إلى 17 مجالًا للموضوع. لا تستخدمه في ملفات تعريف .NET SDK الروتينية أو التصميم المعياري أو تصحيح أخطاء اختبار CI.
