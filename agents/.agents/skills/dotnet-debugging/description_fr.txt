Débogue les applications Windows et Linux/macOS (natives, .NET/CLR, mode mixte) avec WinDbg MCP (crash dumps, !analyze, !syncblk, !dlk, !runaway, !dumpeap, !gcroot, BSOD), dotnet-dump, lldb avec SOS, Createdump et diagnostics de conteneur (Docker, Kubernetes). Diagnostic de blocage/blocage, triage élevé du processeur, enquête sur les fuites de mémoire, débogage du noyau et moniteur dotnet pour la production. Couvre 17 domaines thématiques. Ne pas utiliser pour le profilage de routine du SDK .NET, la conception de référence ou le débogage de tests CI.
