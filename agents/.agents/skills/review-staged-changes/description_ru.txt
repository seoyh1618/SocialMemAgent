Просматривает поэтапные изменения git на предмет качества кода, удобства сопровождения, читабельности и потенциальных регрессов. Проверяет, что изменения имеют смысл в контексте, улучшают удобство обслуживания, улучшают читабельность и не вызывают побочных эффектов. Используйте при просмотре поэтапных изменений, изучении различий git или когда пользователь просит просмотреть изменения перед их фиксацией.
