Conseils d'experts pour le développement JavaScript/TypeScript avec le runtime Bun. Couvre la configuration du projet, la gestion des packages, les serveurs HTTP, les API intégrées, les tests, le regroupement et la migration depuis Node.js. À utiliser lors du démarrage de projets Bun, à l'aide des API Bun (Bun.serve, sql/SQL, s3, redis, Bun.$), lors de la migration depuis Node.js ou lors du dépannage du comportement spécifique à Bun.
