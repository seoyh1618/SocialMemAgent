Kompetente Anleitung für die JavaScript-/TypeScript-Entwicklung mit der Bun-Laufzeitumgebung. Behandelt Projekteinrichtung, Paketverwaltung, HTTP-Server, integrierte APIs, Tests, Bündelung und Migration von Node.js. Verwenden Sie diese Option, wenn Sie Bun-Projekte starten, Bun-APIs (Bun.serve, sql/SQL, s3, redis, Bun.$) verwenden, von Node.js migrieren oder Bun-spezifisches Verhalten beheben.
