Ricerca semantica nativa Exa AI tramite API Composio. Utilizzare quando:
(1) Ricerca sul Web con query in linguaggio naturale
(2) Ottenere risposte supportate da citazioni alle domande di ricerca
(3) Trovare pagine simili a un determinato URL
(4) Recupero del contenuto completo dai risultati di ricerca
