أفضل ممارسات عميل NextDNS CLI لتثبيت وتكوين وإدارة وكيل DNS-over-HTTPS. يجب استخدام هذه المهارة عند تثبيت NextDNS على أجهزة التوجيه أو الخوادم أو أجهزة العميل، أو تكوين إعدادات DNS، أو إدارة البرنامج الخفي، أو استكشاف مشكلات DNS وإصلاحها. يتم تشغيلها في المهام التي تتضمن تثبيت NextDNS CLI، أو التحكم في البرنامج الخفي، أو تكوين النظام، أو إعداد وكيل DNS.
