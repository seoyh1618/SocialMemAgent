Рекомендации по использованию клиента NextDNS CLI для установки, настройки и управления прокси-сервером DNS-over-HTTPS. Этот навык следует использовать при установке NextDNS на маршрутизаторы, серверы или клиентские устройства, настройке параметров DNS, управлении демоном или устранении неполадок DNS. Запускается при выполнении задач, связанных с установкой NextDNS CLI, управлением демоном, конфигурацией системы или настройкой DNS-прокси.
