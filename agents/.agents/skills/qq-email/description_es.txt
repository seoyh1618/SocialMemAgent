IMAP para recibir y SMTP para enviar correos electrónicos para QQ Mail; La cuenta y el código de autorización se leen de las variables de entorno QQ_EMAIL_ACCOUNT y QQ_EMAIL_AUTH_CODE. Úselo cuando el usuario desee enviar correos electrónicos de QQ, recibir correos electrónicos, consultar correos electrónicos, enviar correos electrónicos en nombre de otros o configurar QQ Mail.
