IMAP pour la réception et SMTP pour l'envoi d'e-mails via QQ Mail ; le compte et le code d'autorisation sont lus à partir des variables d'environnement QQ_EMAIL_ACCOUNT et QQ_EMAIL_AUTH_CODE. À utiliser lorsque l'utilisateur doit envoyer, recevoir, vérifier ou relayer des e-mails QQ ou configurer QQ Mail.
