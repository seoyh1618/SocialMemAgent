Compétence de validation de la couverture de l'API OpenLark. Utilisé pour vérifier le nombre et la couverture des API implémentées par chaque caisse, en comparant les implémentations réelles de code basées sur le script tools/validate_apis.py et api_list_export.csv. Mots-clés déclencheurs : validation de l'API, couverture de l'API, vérification du nombre d'API, vérification de la mise en œuvre de l'API, statistiques de l'API
