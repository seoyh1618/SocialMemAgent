Навыки проверки покрытия OpenLark API. Используется для проверки количества реализаций API и покрытия каждого ящика, сравнения фактической реализации кода со сценарием Tools/validate_apis.py и api_list_export.csv. Ключевые слова-триггеры: проверка API, покрытие API, проверка количества API, проверка реализации API, статистика API.
