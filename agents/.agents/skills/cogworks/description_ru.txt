Кодирует знания по теме в навыки, которые можно вызвать из URL-адресов и файлов. В качестве вспомогательных навыков требуются cogworks-encode и cogworks-learn. Создает каталоги и файлы в качестве побочных эффектов, поэтому вызывается только тогда, когда пользователь явно вводит команду «cogworks» (например, «cogworks encode», «cogworks Learn», «cogworks автоматизировать»). Общие слова, такие как «учиться», «кодировать» или «автоматизировать», сами по себе не указывают на намерение пользователя создавать файлы навыков.
