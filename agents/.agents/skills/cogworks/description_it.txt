Codifica la conoscenza dell'argomento in competenze richiamabili da URL e file. Richiede cogworks-encode e cogworks-learn come competenze di supporto. Crea directory e file come effetti collaterali, quindi richiamalo solo quando l'utente digita esplicitamente un comando 'cogworks' (ad esempio 'cogworks encode', 'cogworks learn', 'cogworks automatite'). Parole generiche come "imparare", "codificare" o "automatizzare" da sole non indicano l'intenzione dell'utente di creare file di abilità.
