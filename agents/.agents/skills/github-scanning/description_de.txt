GitHub-Datenerfassungsmuster für Workflow-Agenten. Behandelt die Erstellung von Suchanfragen nach Absicht, die Handhabung von Datumsbereichen, die Eingrenzung des Repository-Bereichs, die Integration von Preferences.md, Cross-Repo-Intelligenz, das Modell der parallelen Stream-Sammlung und die automatische Wiederherstellung für leere Ergebnisse. Verwenden Sie diese Option, wenn Sie Agenten erstellen, die GitHub nach Problemen, PRs, Diskussionen, Releases, Sicherheitswarnungen oder CI-Status durchsuchen.
