Шаблоны сбора данных GitHub для агентов рабочих процессов. Охватывает построение поисковых запросов по намерению, обработку диапазона дат, сужение области репозитория, интеграцию с Preferences.md, анализ перекрестных репозиториев, модель сбора параллельных потоков и автоматическое восстановление для пустых результатов. Используйте при создании агентов, которые ищут в GitHub проблемы, запросы на запрос, обсуждения, выпуски, предупреждения безопасности или статус CI.
