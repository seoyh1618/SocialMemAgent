Patrones de recopilación de datos de GitHub para agentes de flujo de trabajo. Cubre la construcción de consultas de búsqueda por intención, manejo de rango de fechas, reducción del alcance del repositorio, integración de preferencias.md, inteligencia entre repositorios, modelo de recopilación de flujo paralelo y recuperación automática de resultados vacíos. Úselo cuando cree agentes que busquen en GitHub problemas, relaciones públicas, discusiones, lanzamientos, alertas de seguridad o estado de CI.
