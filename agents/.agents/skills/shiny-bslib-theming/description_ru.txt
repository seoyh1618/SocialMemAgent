Расширенное оформление тем для приложений Shiny с использованием bslib и Bootstrap 5. Используйте при настройке внешнего вида приложения с помощью bs_theme(), тем Bootswatch, пользовательских цветов, типографики, интеграции Brand.yml, переменных Bootstrap Sass, пользовательских правил Sass/CSS, темного режима и цветовых режимов, динамического переключения тем, оформления тем в реальном времени, проверки тем или создания графиков R, соответствующих теме приложения с помощью тематических.
