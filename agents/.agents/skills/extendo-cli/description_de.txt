Kommunizieren Sie mit einem menschlichen Benutzer über Extendo – senden Sie Nachrichten, warten Sie auf Antworten, erstellen Sie strukturierte Entscheidungen (Ja/Nein, Multiple Choice, Checkliste, Rangfolge, Kategorisierung, Dokumentenprüfung, DAG, Fortschrittstabelle) und bauen Sie menschliche Entscheidungspunkte in Agenten-Workflows ein. Der Benutzer sieht eine umfangreiche Benutzeroberfläche auf seinem Telefon und antwortet. Löst aus bei „Nachricht an Benutzer senden“, „Benutzer fragen“, „Genehmigung einholen“, „Entscheidung erstellen“, „Artefakt“, „Erweitern“, „Menschliches Tor“, „Gate auf Benutzer“, „Blockieren, bis Benutzer entscheidet“, „Genehmigungs-Gate“, „Extendo-Gate“, „Dag“, „Fortschrittsraster“, „Statusraster“ oder jede Aufgabe, die eine menschliche Eingabe über mobile Push-Benachrichtigungen erfordert.
