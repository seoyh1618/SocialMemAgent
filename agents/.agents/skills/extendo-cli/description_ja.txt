Extendo を介して人間のユーザーと通信します。メッセージを送信し、返信を待ち、構造化された意思決定 (はい/いいえ、複数選択、チェックリスト、ランキング、分類、文書レビュー、DAG、進行状況グリッド) を作成し、エージェントのワークフローに人間の意思決定ゲートを組み込みます。ユーザーは携帯電話でリッチ UI を確認し、応答します。 「ユーザーにメッセージを送信」、「ユーザーに質問」、「承認の取得」、「決定の作成」、「アーティファクト」、「拡張」、「ヒューマン ゲート」、「ユーザーのゲート」、「ユーザーが決定するまでブロック」、「承認ゲート」、「拡張ゲート」、「DAG」、「進行状況グリッド」、「ステータス グリッド」、またはモバイル プッシュ通知を介して人間による入力を必要とするタスクをトリガーします。
