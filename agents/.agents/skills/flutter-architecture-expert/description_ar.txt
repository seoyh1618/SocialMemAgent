إرشادات معمارية لتطبيقات Flutter باستخدام مجموعة إنشاءات Flutter_it (get_it، watch_it، Command_it، الاستماع_it). يغطي بنية الرفرفة العملية (PFA) مع الخدمات/المديرين/طرق العرض، وبنية المشروع القائمة على الميزات، ونمط المدير، ونمط الوكيل مع التحديثات المتفائلة وحقول التجاوز، وDataRepository مع العد المرجعي، والخدمات المحددة النطاق، وتفاصيل عناصر واجهة المستخدم، والاختبار، وأفضل الممارسات. يُستخدم عند تصميم بنية التطبيق، أو هيكلة مشاريع Flutter، أو مديري التنفيذ أو الوكلاء، أو تخطيط تنظيم الميزات.
