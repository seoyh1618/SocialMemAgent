مدير إصدار أداة متعدد اللغات Mise (RTX سابقًا)، ومدير متغير البيئة، ومشغل المهام. يغطي إدارة إصدار الأداة (التثبيت والاستخدام والتثبيت عبر Node/Python/Rust/Go/Java والمزيد)، وتعريفات مهام Mise.toml (البرامج النصية، والتبعيات، ووضع المراقبة، والبنيات التزايدية)، وإدارة متغيرات البيئة (تكامل .env، وenv لكل دليل، والتنقيح)، وتثبيت الأداة على مستوى المشروع، ونظام الواجهة الخلفية (asdf، npm، pipx، البضائع، إصدارات GitHub)، والمقارنة مع البدائل (asdf، nvm، pyenv، Makefiles).

يُستخدم عند تكوين إصدارات أداة التطوير، أو تحديد مهام المشروع في mise.toml، أو إدارة متغيرات البيئة لكل مشروع، أو إعداد بيئات تطوير متعددة اللغات، أو الترحيل من asdf أو nvm، أو أتمتة سير عمل البناء/الاختبار/lint/النشر من خلال تشغيل mise.
