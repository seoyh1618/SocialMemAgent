Feishu All-in-One-Toolbox: Tabellenkalkulation, Kalender, Aufgabenverwaltung, Gruppenchat-Verwaltung, Whiteboard-Vorgänge, PlantUML-Diagramme, Dateiverwaltung, Asset-Upload/Download, Dokumentkommentare, Wissensdatenbank, Suche, Benutzerverzeichnis. Wird verwendet, wenn der Benutzer Feishu-Blätter bedienen, Kalender anzeigen, Aufgaben erstellen, Gruppenchats verwalten, Whiteboards bedienen, PlantUML generieren, Dateien verwalten, Assets hochladen, Kommentare anzeigen, Wissensdatenbanken anzeigen, Nachrichten durchsuchen, Benutzerinformationen nachschlagen oder Abteilungen abfragen möchte. Deckt alle Funktionen von Feishu-Cli ab, mit Ausnahme des Lesens/Schreibens/Importierens/Exportierens von Dokumenten und der Berechtigungsverwaltung.
