Caja de herramientas todo en uno de Feishu: hoja de cálculo, calendario, gestión de tareas, gestión de chat grupal, operaciones de pizarra, diagramas PlantUML, gestión de archivos, carga/descarga de activos, comentarios de documentos, base de conocimientos, búsqueda, directorio de usuarios. Se utiliza cuando el usuario solicita operar hojas Feishu, ver calendarios, crear tareas, administrar chats grupales, operar pizarras, generar PlantUML, administrar archivos, cargar activos, ver comentarios, ver la base de conocimientos, buscar mensajes, buscar información del usuario o consultar departamentos. Cubre todas las funciones de feishu-cli excepto lectura/escritura/importación/exportación de documentos y gestión de permisos.
