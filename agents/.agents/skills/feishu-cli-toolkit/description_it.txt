Casella degli strumenti tutto in uno Feishu: foglio di calcolo, calendario, gestione delle attività, gestione delle chat di gruppo, operazioni sulla lavagna, diagrammi PlantUML, gestione dei file, caricamento/download di risorse, commenti sui documenti, base di conoscenza, ricerca, directory degli utenti. Utilizzato quando l'utente richiede di utilizzare fogli Feishu, visualizzare calendari, creare attività, gestire chat di gruppo, utilizzare lavagne, generare PlantUML, gestire file, caricare risorse, visualizzare commenti, visualizzare knowledge base, cercare messaggi, cercare informazioni utente o interrogare dipartimenti. Copre tutte le funzioni di feishu-cli tranne la lettura/scrittura/importazione/esportazione dei documenti e la gestione dei permessi.
