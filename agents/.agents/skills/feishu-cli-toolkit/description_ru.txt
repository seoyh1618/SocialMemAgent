Универсальный набор инструментов Feishu: электронные таблицы, календарь, управление задачами, управление групповым чатом, операции с виртуальной доской, диаграммы PlantUML, управление файлами, загрузка/загрузка ресурсов, комментарии к документам, база знаний, поиск, каталог пользователей. Используется, когда пользователь запрашивает работу с таблицами Feishu, просмотр календарей, создание задач, управление групповыми чатами, работу с виртуальными досками, создание PlantUML, управление файлами, загрузку ресурсов, просмотр комментариев, просмотр базы знаний, поиск сообщений, поиск информации о пользователе или запрос отделов. Охватывает все функции feishu-cli, кроме чтения/записи/импорта/экспорта документов и управления разрешениями.
