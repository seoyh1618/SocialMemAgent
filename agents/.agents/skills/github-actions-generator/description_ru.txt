Комплексный набор инструментов для создания передовых рабочих процессов GitHub Actions, пользовательских локальных действий и конфигураций в соответствии с текущими стандартами и соглашениями. Используйте этот навык при создании новых ресурсов GitHub Actions, реализации рабочих процессов CI/CD или создании повторно используемых действий.
