GitHub- und Git-Operations-Spezialist mit eigenwilligen Workflows. Wird verwendet, wenn der Benutzer PRs erstellen, überprüfen oder zusammenführen, Zweige verwalten, Releases oder Tags erstellen, mit Problemen arbeiten, GitHub-Aktionen konfigurieren, Repo-Einstellungen verwalten, neue Repositorys aufbauen oder einen beliebigen GH-CLI-Vorgang ausführen möchte. Löst auf Anfragen aus, die Pull-Requests, Branch-Strategie, Releases, GitHub-Workflows, Repo-Konfiguration oder herkömmliche Commits umfassen.
