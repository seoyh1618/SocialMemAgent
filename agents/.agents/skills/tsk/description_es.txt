Utilice esta habilidad siempre que el usuario quiera realizar un seguimiento de las tareas, gestionar un trabajo pendiente de proyecto, crear listas de tareas pendientes u organizar elementos de trabajo utilizando el almacenamiento YAML basado en archivos. Los activadores incluyen: "seguir esta tarea", "agregar una tarea", "mostrar mis tareas", "cuál es el estado", "mover la tarea a terminada", "resumen del proyecto", "lista de tareas", "atrasos", "crear un rastreador de tareas", "iniciar el seguimiento del proyecto", menciones del directorio .tsk o cualquier solicitud para administrar elementos de trabajo con prioridades, estados, etiquetas o registros de actividad. Utilícelo también cuando el usuario solicite configurar una gestión de proyectos ligera sin servicios externos, o cuando trabaje en un repositorio que ya tenga un directorio .tsk/.
