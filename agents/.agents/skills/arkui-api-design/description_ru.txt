Этот навык следует использовать, когда пользователь просит «разработать API ArkUI», «добавить свойство компонента», «создать метод модификатора», «пересмотреть API ArkUI», «отменить API», «написать JSDOC для ArkUI» или упоминает стандарты проектирования API OpenHarmony. Предоставляет комплексное руководство по проектированию API компонентов ArkUI в соответствии с рекомендациями по кодированию OpenHarmony, включая статическую/динамическую синхронизацию интерфейса, компиляцию SDK и проверку.
