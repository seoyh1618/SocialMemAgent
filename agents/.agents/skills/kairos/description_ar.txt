دمج Kairos API لاكتشاف المنتجات المدعومة بالذكاء الاصطناعي وتقديم التوصيات. يُستخدم عند إنشاء وكلاء يحتاجون إلى التوصية بالمنتجات أو اكتشاف الصفقات أو مساعدة المستخدمين في العثور على عناصر للشراء. يتم تشغيل المهام التي تتضمن "التوصية بالمنتجات"، و"العثور على الصفقات"، و"اكتشاف المنتج"، و"مساعد التسوق"، و"واجهة برمجة تطبيقات المنتج المتكاملة". لا يتطلب أي إعداد - لا يوجد تسجيل، ولا يوجد مفتاح API، فقط أرسل طلبًا واحصل على توصيات.
