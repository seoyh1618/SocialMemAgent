Интегрируйте Kairos API для поиска продуктов и рекомендаций на основе искусственного интеллекта. Используйте при создании агентов, которым необходимо рекомендовать продукты, находить специальные предложения или помогать пользователям находить товары для покупки. Запускается при выполнении задач, включающих «рекомендовать продукты», «поиск предложений», «обнаружение продуктов», «помощник по покупкам», «интеграцию API продукта». Никакой настройки не требуется — ни регистрации, ни ключа API, просто отправьте запрос и получите рекомендации.
