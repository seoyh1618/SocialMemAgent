Интеллектуальный инструмент для создания материалов для заявок на авторские права. Автоматически анализирует исходный код проекта и генерирует спецификации программного обеспечения и документацию по исходному коду, соответствующую требованиям приложений по защите авторских прав на программное обеспечение. Поддерживает поиск по ключевым словам, интеллектуальный анализ исходного кода, форматированный вывод и экспорт PDF.
