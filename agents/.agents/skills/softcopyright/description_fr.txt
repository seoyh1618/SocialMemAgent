Outil intelligent de génération de matériel d'application de droits d'auteur de logiciel. Analyse automatiquement le code source du projet et génère des spécifications logicielles et une documentation sur le code source qui répondent aux exigences des applications de droits d'auteur sur les logiciels. Prend en charge la recherche par mot clé, l'analyse intelligente du code source, la sortie formatée et l'exportation PDF.
