Herramienta de generación de material de aplicación de derechos de autor de software inteligente. Analiza automáticamente el código fuente del proyecto y genera especificaciones de software y documentación del código fuente que cumplen con los requisitos de la aplicación de derechos de autor del software. Admite búsqueda de palabras clave, análisis inteligente de código fuente, salida formateada y exportación de PDF.
