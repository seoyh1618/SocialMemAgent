Si attiva anche su: "aggiorna il sito", "scrivi un post", "correggi il blog", "joelclaw.com", "aggiorna pagina di rete", "aggiungi una pagina", "modifica l'intestazione" o qualsiasi attività che coinvolga l'app Web rivolta al pubblico.
